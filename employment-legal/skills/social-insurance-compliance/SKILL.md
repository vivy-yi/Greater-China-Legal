# 社会保险缴纳合规咨询 (Social Insurance Compliance)

## 1. 目的与范围

### 1.1 核心目的

为用人单位提供社会保险（含养老、医疗、工伤、失业、生育保险）缴纳合规咨询服务，识别社保缴纳中的法律风险并提供整改建议。

### 1.2 适用范围

- 城镇职工社会保险的缴纳基数、比例、时限合规审查
- 社保基数调整的正确操作流程
- 社保补缴（含历史欠缴）的合规性分析
- 跨地区用工的社保缴纳地认定
- 新业态灵活就业人员社保问题
- 社保违规的行政风险评估

### 1.3 本Skill不涵盖

- 住房公积金缴纳合规（见公积金合规专项）
- 商业保险配置建议
- 社保待遇计算与发放争议
- 跨省社保转移接续的具体操作（仅提供原则性指引）

## 2. 法定缴纳框架

### 2.1 强制缴纳义务

| 险种 | 依据 | 缴费义务起始时点 |
|------|------|----------------|
| 养老保险 | 社会保险法第10条 | 用工之日起30日内 |
| 医疗保险 | 社会保险法第第23条 | 用工之日起30日内 |
| 工伤保险 | 社会保险法第33条 | 用工之日（不设等待期） |
| 失业保险 | 社会保险法第44条 | 用工之日起 |
| 生育保险 | 社会保险法第第53条 | 用工之日起 |

### 2.2 缴费基数确定规则

**正常缴费基数：**

- 上限：职工工资收入超过当地上年度职工平均工资300%的，按300%计算
- 下限：职工工资收入低于当地上年度职工平均工资60%的，按60%计算
- 区间外部分：不得按区间外基数申报

**试用期社保：**

- 试用期属于劳动合同期限的一部分，必须依法缴纳社保
- 试用期内如工资低于当地社平工资60%，按60%为基数缴纳

**提成/奖金类工资：**

- 应计入社保缴费基数的工资总额包括：计时工资、计件工资、奖金、津贴补贴、加班工资、特殊情况下支付的工资
- 部分地方口径要求将业务提成年均后计入基数，需核实当地规定

### 2.3 缴费比例（全国基本标准，各地可能有浮动）

| 险种 | 单位缴费比例 | 个人缴费比例 |
|------|------------|------------|
| 养老保险 | 16%（2022年后） | 8% |
| 医疗保险 | 约8%（各地不同） | 2% |
| 工伤保险 | 0.2%-1.4%（按行业风险类别） | 不缴费 |
| 失业保险 | 0.7% | 0.3% |
| 生育保险 | 约0.5%-1%（与医保合并后由医保基金支付） | 不缴费 |

## 3. 常见违规情形与风险

### 3.1 基数申报违规

| 违规类型 | 表现形式 | 法律后果 |
|---------|---------|---------|
| 基数偏低申报 | 按低于实际工资收入基数申报 | 补缴+每日万分之五滞纳金；情节严重可处1-3倍罚款 |
| 虚构基数 | 伪造工资表虚报基数 | 涉嫌社保欺诈，可能移送司法 |
| 区间外申报 | 按超过300%或低于60%基数申报 | 稽核纠正；历史期间可追溯处理 |

### 3.2 人员漏缴

- 试用期不缴：违法，试用期必须缴纳社保
- 实习生不缴：原则上不强制，但建议商业保险替代
- 退休返聘不缴：已依法领取养老金的返聘人员可不缴工伤外的社保
- 外国籍员工：需核查双边社保协定
- 兼职人员：非全日制用工可单独参加工伤保险；其他情况应依法缴纳

### 3.3 补缴合规要求

**可补缴情形：**

- 劳动关系存续期间未缴纳（可追溯，但有2年追诉期限制，详见3.4）
- 基数差错的纠正性补缴
- 员工自愿放弃缴纳的协议因违反强制性规定而无效，仍须补缴

**补缴计算标准：**

- 按应申报年度的社平工资及实际工资基数核定
- 需缴纳本金+每日万分之五的滞纳金
- 超过2年追溯期的，用人单位已建立劳动关系期间未缴纳的，劳动监察部门可处罚款

### 3.4 追诉时效与执法权限

| 情形 | 时效/权限 | 说明 |
|------|----------|------|
| 劳动监察投诉 | 2年（一般违规）；5年（涉及工资、社保） | 自违法行为终了之日起算 |
| 社保稽核 | 无硬性时效限制，但受"发现之日起2年内"自查纠正期约束 | 稽核权与监察权存在交叉 |
| 司法救济 | 劳动争议申请仲裁60日；诉讼3年 | 从知道或应当知道权益被侵害之日起算 |

## 4. 合规审查清单

### 4.1 入职阶段核查

```
□ 劳动合同是否在用工之日起30日内签订
□ 社保增员申报是否在用工之日起30日内完成
□ 试用期工资是否低于社平工资60%，如是须按60%为基数申报
□ 工资构成中哪些项目应计入社保基数（核实当地口径）
□ 特殊人员类型（外籍、港澳台、兼职、实习生）适用哪种缴纳规则
```

### 4.2 年度基数调整核查

```
□ 是否在每年规定时间内完成基数调整申报
□ 调整后基数是否覆盖上年度全年工资收入
□ 年终奖、业务提成等是否正确分摊至各月并计入基数
□ 跨年度补差情形是否已正确计算并申报
□ 基数调整文件是否已存档备查
```

### 4.3 离职阶段核查

```
□ 社保减员是否在解除/终止后30日内完成
□ 是否存在当月减员导致当月社保断缴情形
□ 离职当月工资是否含社保个人部分扣款
□ 失业保险缴费年限是否满足领取条件
□ 是否已告知员工社保转移接续的办理流程与时限
```

## 5. 风险量化评估参考

### 5.1 补缴成本估算公式

```
补缴本金 = Σ(应缴基数_月 × 费率 × 欠缴月数)
滞纳金 = 本金 × 0.0005 × 欠缴天数
罚款（可能）= 本金 × (0.5~3)  [需视情节]
总成本 ≈ 本金 + 滞纳金 + 罚款
```

### 5.2 风险等级判断

| 风险等级 | 违规人数 | 欠缴时长 | 金额规模 | 处理建议 |
|---------|---------|---------|---------|---------|
| 低 | ≤10人 | ≤6个月 | ≤10万元 | 自查自纠，主动补缴 |
| 中 | 10-50人 | 6-24个月 | 10-50万元 | 制定补缴计划，可申请分期 |
| 高 | >50人或 | >24个月 | >50万元 | 律师介入，制定整体方案，准备行政应对 |

## 6. 应对行政检查流程

### 6.1 社保稽核与劳动监察分工

- **社保经办机构（稽核）**：核查缴费基数、人数、报表真实性，有权调阅账册
- **人力资源和社会保障部门（监察）**：可处罚未登记、迟延登记、拒缴等行为
- 两者可同时启动，企业应分别准备材料

### 6.2 检查应对要点

```
1. 收到检查通知后，确认检查主体与事项范围
2. 准备近3年劳动台账、工资发放记录、社保申报表
3. 如存在争议金额，准备书面说明与证据
4. 不应销毁、隐匿任何资料
5. 涉及补缴的，争取在检查结论前主动纠正
6. 检查结论有异议的，可在60日内申请行政复议或诉讼
```
