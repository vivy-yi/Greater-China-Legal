# work-injury-compensation

工伤认定与赔偿咨询技能。辅助分析工伤认定、赔偿计算、劳动能力鉴定等问题。

---

## 法域假设

默认中国大陆法域，适用《工伤保险条例》《工伤认定办法》等法规。

---

## 信息提取

用户提供信息后，提取以下要素：

| 信息项 | 说明 |
|--------|------|
| 当事人角色 | 用人单位 / 员工 |
| 劳动关系 | 已签劳动合同 / 未签 / 实习期 |
| 用人单位参保情况 | 已参加工伤保险 / 未参加 |
| 事故经过 | 何时、何地、因何受伤 |
| 伤情诊断 | 初步诊断或已确诊的伤情 |
| 月工资标准 | 员工本人月工资（元） |
| 事故类型 | 典型工伤 / 上下班途中交通事故 / 职业病 / 抢险救灾受伤 |
| 是否已做劳动能力鉴定 | 已鉴定等级 / 未鉴定 |

---

## 工作流程

### 第一步：判断是否属于工伤

#### 法定认定情形

**应当认定工伤（《工伤保险条例》第14条）：**

| 情形 | 法条 |
|------|------|
| 工作时间、工作场所内，因工作原因受伤 | 第14条第1款 |
| 工作时间前后，在工作场所内，从事与工作有关的预备性或收尾性工作受伤 | 第14条第2款 |
| 工作时间、工作场所内，因履行工作职责受到暴力等意外伤害 | 第14条第3款 |
| 患职业病 | 第14条第4款 |
| 因工外出期间，因工作原因受伤或失踪 | 第14条第5款 |
| 上下班途中，受到非本人主要责任的交通事故伤害 | 第14条第6款 |
| 法律、行政法规规定应当认定工伤的其他情形 | 第14条第7款 |

**视同工伤（《工伤保险条例》第15条）：**

| 情形 | 法条 |
|------|------|
| 工作时间、工作岗位，突发疾病48小时内死亡 | 第15条第1款 |
| 抢险救灾等维护国家利益、公共利益活动中受伤 | 第15条第2款 |
| 退役军人旧伤复发 | 第15条第3款 |

**不得认定工伤或视同工伤：**
- 故意犯罪
- 醉酒或吸毒
- 自残或自杀
- 法律、行政法规规定的其他排除情形

#### 特殊情形处理

**未签劳动合同：** 可通过工资支付记录、工作证、考勤记录、证人证言等证明事实劳动关系，仍可申请工伤认定。

**用人单位未参保：** 由用人单位按《工伤保险条例》规定的标准支付全部工伤保险待遇。

**劳务派遣：** 被派遣员工发生工伤，由劳务派遣单位依法申请认定，派遣单位和用工单位承担连带赔偿责任。

### 第二步：工伤认定程序

#### 申请主体与时效

| 申请主体 | 时效要求 |
|---------|---------|
| 用人单位 | 事故发生之日起30日内提出（特殊情况可适当延长） |
| 员工或其近亲属、工会组织 | 事故发生之日起1年内提出 |
| 职业病诊断机构 | 按诊断之日起30日内提出 |

#### 申请材料（一般要求）

1. 工伤认定申请表
2. 劳动关系的证明材料（劳动合同、事实劳动关系证明等）
3. 医疗机构诊断证明或职业病诊断证明
4. 事故经过描述材料
5. 证人证言及身份证明（如有）
6. 公安交通管理部门出具的事故责任认定书（上下班途中交通事故需提供）

#### 认定流程

```
申请提交 → 材料补正（如需）→ 社会保险行政部门受理 → 调查核实 → 作出认定决定 → 送达
```

- 受理时限：材料齐全情况下，15日内作出是否受理决定
- 认定时限：受理之日起60日内作出认定决定
- 结论形式：工伤认定决定书 / 不予认定决定书

---

### 第三步：劳动能力鉴定

#### 鉴定时机

- 伤情相对稳定后
- 停工留薪期满（含延长）
- 存在残疾、影响劳动能力的

#### 鉴定程序

1. 向设区的市级劳动能力鉴定委员会提出申请
2. 提交：工伤认定决定书、医疗机构诊断证明、身份证明
3. 鉴定委员会组织专家进行鉴定
4. 作出劳动能力鉴定结论

#### 伤残等级（1-10级）

| 等级 | 劳动能力丧失程度 | 典型情形 |
|------|----------------|---------|
| 一级 | 完全丧失劳动能力 | 生活完全不能自理 |
| 二级 | 高度丧失劳动能力 | 生活大部分不能自理 |
| 三级 | 高度丧失劳动能力 | 生活部分不能自理 |
| 四级 | 中度丧失劳动能力 | 生活需要他人扶助 |
| 五级 | 中度丧失劳动能力 | 器官部分缺损 |
| 六级 | 中度丧失劳动能力 | 器官部分缺损 |
| 七级 | 轻度丧失劳动能力 | 器官大部分缺损 |
| 八级 | 轻度丧失劳动能力 | 器官轻微缺损 |
| 九级 | 轻度丧失劳动能力 | 器官部分缺损 |
| 十级 | 轻度丧失劳动能力 | 器官轻微缺损 |

#### 复查鉴定

- 自鉴定结论作出之日起1年后，伤残情况变化可申请复查鉴定

---

### 第四步：赔偿计算

#### 赔偿项目总览

| 赔偿项目 | 支付主体 | 说明 |
|---------|---------|------|
| 医疗待遇 | 工伤保险基金 | 符合工伤保险诊疗目录的费用 |
| 住院伙食补助费 | 工伤保险基金 | 统筹地区标准 |
| 交通、食宿费 | 工伤保险基金 | 需经医疗机构出具证明，统筹地区标准 |
| 辅助器具费 | 工伤保险基金 | 经劳动能力鉴定委员会确认配置 |
| 停工留薪期工资 | 用人单位 | 不超过12个月，严重伤情可延长至24个月 |
| 护理费 | 用人单位 | 停工留薪期需要护理的 |
| 一次性伤残补助金 | 工伤保险基金 | 按伤残等级计算 |
| 伤残津贴 | 工伤保险基金 | 五级、六级按月支付（难以安排工作的） |
| 一次性工伤医疗补助金 | 工伤保险基金 | 解除或终止劳动关系时支付 |
| 一次性伤残就业补助金 | 用人单位 | 解除或终止劳动关系时支付 |
| 工亡补助金 | 工伤保险基金 | 上年度全国城镇居民人均可支配收入20倍 |
| 丧葬补助金 | 工伤保险基金 | 6个月统筹地区上年度职工月平均工资 |
| 供养亲属抚恤金 | 工伤保险基金 | 按月计发，符合供养条件 |

#### 停工留薪期工资

计算公式：`停工留薪期工资 = 月工资 × 停工留薪月数`

- 标准：员工本人月工资
- 最长期限：一般不超过12个月，伤情严重或情况特殊经市级劳动能力鉴定委员会确认可延长至24个月
- 不扣减全勤奖、绩效工资等按月发放的正常工资收入

#### 一次性伤残补助金

计算公式：`一次性伤残补助金 = 本人月工资 × 月数`

| 伤残等级 | 支付月数 |
|---------|---------|
| 一级 | 27个月 |
| 二级 | 25个月 |
| 三级 | 23个月 |
| 四级 | 21个月 |
| 五级 | 18个月 |
| 六级 | 16个月 |
| 七级 | 13个月 |
| 八级 | 11个月 |
| 九级 | 9个月 |
| 十级 | 7个月 |

#### 伤残津贴（月发，五至六级）

| 等级 | 标准（本人工资比例） |
|------|---------------------|
| 五级 | 70% |
| 六级 | 60% |

> 注：难以安排工作的，由用人单位按月发放至符合退休条件

#### 一次性工伤医疗补助金与一次性伤残就业补助金

解除或终止劳动合同时支付。两项合并计算参考（以多数省份标准为例，实际以本地规定为准）：

| 伤残等级 | 医疗补助金 | 就业补助金 |
|---------|-----------|-----------|
| 五级 | 约30个月统筹地区月平均工资 | 约30个月 |
| 六级 | 约25个月 | 约25个月 |
| 七级 | 约15个月 | 约15个月 |
| 八级 | 约10个月 | 约10个月 |
| 九级 | 约6个月 | 约6个月 |
| 十级 | 约4个月 | 约4个月 |

> 具体标准各省不同，须参照当地规定。

#### 工亡待遇

| 项目 | 标准 |
|------|------|
| 丧葬补助金 | 6个月统筹地区上年度职工月平均工资 |
| 供养亲属抚恤金 | 配偶：本人月工资40%/人；其他亲属：本人月工资30%/人 |
| 工亡补助金 | 上年度全国城镇居民人均可支配收入20倍 |

> 2024年度全国城镇居民人均可支配收入约54,128元，工亡补助金约108万元。

### 第五步：策略建议与升级决策门

#### 员工维权路径

```
工伤认定申请 → 劳动能力鉴定 → 工伤保险待遇协商 → 劳动仲裁（如协商不成）
```

#### 用人单位处理要点

1. 事故发生后30日内主动申请工伤认定
2. 未参保的，用人单位承担全部赔偿责任
3. 配合调查、提供相关证据材料
4. 对认定结论有异议可在60日内申请行政复议或提起行政诉讼

#### 升级决策门

> ⚠️ **辅助分析，不构成法律意见。** 工伤认定须经社会保险行政部门作出决定；劳动能力鉴定须经劳动能力鉴定委员会作出结论；劳动争议须先经过劳动仲裁程序。建议委托具备劳动法律师资格的律师代理。

以下情形建议升级至专业律师或仲裁诉讼：

| 情形 | 说明 |
|------|------|
| 用人单位拒绝申请工伤认定 | 员工可自行在1年内申请，或寻求法律帮助 |
| 对工伤认定结论不服 | 可申请行政复议或提起行政诉讼 |
| 对劳动能力鉴定结论有异议 | 可在15日内向省级劳动能力鉴定委员会申请再次鉴定 |
| 赔偿金额争议较大 | 建议通过劳动仲裁解决 |
| 用人单位注销或无力赔偿 | 可申请工伤保险基金先行支付 |

```
# 工伤认定与赔偿分析

## 基本信息
| 要素 | 内容 |
|------|------|
| 当事人角色 | [员工/用人单位] |
| 劳动关系 | [已签/未签/劳务派遣等] |

| 伤情 | [诊断结果] |
| 月工资 | [元] |

## 工伤认定分析
- 是否符合认定条件：[是/否/需进一步核实]
- 适用法条：[具体条款]
- 申请主体与时效：[说明]

## 赔偿预估
| 项目 | 金额（元）| 支付主体 |
|------|----------|---------|
| 停工留薪期工资 | [计算结果] | 用人单位 |
| 一次性伤残补助金 | [计算结果] | 工伤保险基金 |
| 其他 | [...] | [...] |

## 建议后续行动
- [ ] 收集并保留所有医疗单据、诊断证明
- [ ] 确认用人单位是否在30日内申请工伤认定
- [ ] 如用人单位拒绝，准备自行在1年内申请
- [ ] 伤情稳定后申请劳动能力鉴定
- [ ] 赔偿协商不成时申请劳动仲裁
- [ ] 委托劳动法律师代理仲裁程序

## 风险提示
- 工伤认定申请有时效限制（1年），逾期可能丧失认定资格
- 劳动能力鉴定结论是计算赔偿的重要依据
- 未签订劳动合同不影响工伤认定，但需证明事实劳动关系
- 用人单位未参加工伤保险的，由用人单位支付全部待遇
```

## 本技能不涵盖

- **代理劳动仲裁或诉讼**
- **工伤认定决定的效力确认** — 由社会保险行政部门依法确认
- **劳动能力鉴定结论的效力确认** — 由劳动能力鉴定委员会依法确认
- **跨法域（港澳台地区）工伤处理**
