# 试用期合规管理与试用期解除咨询 (Probation Period Advisor)

## 1. 目的与范围

### 1.1 核心目的

为用人单位与劳动者提供试用期合规管理的法律支持，涵盖试用期约定、考核标准设定、解除条件判断与程序合规等事项，确保试用期用工合法、规范，最大限度降低争议风险。

### 1.2 适用范围

- 试用期劳动合同的订立与条款审查
- 试用期期限与工资标准合规判断
- 试用期考核标准的制定与评估
- 试用期解除条件的认定与程序
- 试用期延长与缩短的操作指引
- 试用期竞业限制与服务期的处理
- 违法解除试用期的法律后果

### 1.3 本Skill不涵盖

- 正式劳动合同的签订与履行（见employment-contract-advisor）
- 劳动仲裁代理与诉讼代理
- 集体劳动合同的谈判与签订
- 劳务派遣与外包用工的试用期处理
- 公务员与事业单位人员的试用期管理

## 2. 试用期基础规则

### 2.1 试用期期限规定

```plaintext
劳动合同期限与试用期对应关系：

│ 劳动合同期限          │ 最长试用期  │
│---------------------│----------│
│ 不满三个月            │ 不得约定   │
│ 三个月以上不满一年      │ 不超过1个月  │
│ 一年以上不满三年        │ 不超过2个月  │
│ 三年以上              │ 不超过6个月  │
│ 派遣制/非全日制        │ 不得约定   │
└─────────────────────┴──────────┘
```

**注意事项：**
- 试用期包含在劳动合同期限内，试用期不另设劳动合同
- 同一用人单位与同一劳动者只能约定一次试用期
- 试用期不得延长；需延长者须重新签订劳动合同（不推荐）
- 以完成一定工作任务为期限的劳动合同，不得约定试用期

### 2.2 试用期工资标准

```plaintext
试用期工资合规要求：

要求一：不得低于以下任一标准（取较高者）
  ├─ 本单位相同岗位最低档工资的80%
  ├─ 劳动合同约定工资的80%
  └─ 当地最低工资标准

要求二：不得低于本单位同岗正式员工工资的70%
  （部分地区要求，具体以当地规定为准）

要求三：试用期内享有同等社会保险和福利
```

### 2.3 试用期社保缴纳

- 用人单位必须自用工之日起30日内为劳动者缴纳社会保险
- 试用期内不得以任何理由中止或缓缴社保
- 社保缴费基数按试用期内实际工资计算
- 违法不缴社保的法律后果：由社会保险费征收机构责令限期补缴，并自欠缴之日起加收万分之五滞纳金；劳动者可据此提出解除劳动合同并主张经济补偿

## 3. 试用期考核标准

### 3.1 合规考核标准要素

用人单位在试用期解除劳动合同，必须能够证明劳动者不符合录用条件。合规的考核标准应具备以下特征：

```plaintext
合格录用条件判断标准：

┌─────────────────────────┐
│      明确性标准          │
│ 岗位说明书/录用条件已书面约定│
│ 劳动者签字确认知悉         │
└─────────────────────────┘
           │
           ▼
┌─────────────────────────┐
│      可评估性标准         │
│ 考核指标量化或具体化       │
│ 考核方式明确              │
│ 考核时间节点清晰           │
└─────────────────────────┘
           │
           ▼
┌─────────────────────────┐
│      客观性标准          │
│ 以客观事实为依据          │
│ 排除主观随意评价          │
│ 证据可留存可追溯          │
└─────────────────────────┘
```

### 3.2 常见违规考核问题

| 违规情形 | 法律风险 |
|---------|---------|
| 录用条件未书面化或未告知劳动者 | 无法证明"不符合录用条件"，构成违法解除 |
| 考核标准过于主观（如"态度不佳"） | 仲裁/诉讼中难以举证 |
| 考核结果未书面记录或未告知劳动者 | 劳动者否认知晓解除理由 |
| 考核材料未经劳动者签字确认 | 证据效力存疑 |
| 延长试用期以"再给机会" | 违反"同一单位与同一劳动者只能约定一次试用期" |

### 3.3 考核标准制定建议

```plaintext
岗位录用条件框架示例：

基本条件：
  ├─ 学历/专业资质符合岗位要求
  ├─ 持有岗位所需的资格证书（有效期内）
  └─ 无犯罪记录/竞业限制冲突

工作能力条件：
  ├─ 试用期（月）内完成工作任务量 ≥ 约定指标的80%
  ├─ 工作质量合格（由直接上级评定，附具体说明）
  └─ 按时完成试用期考核，无重大延误

出勤与纪律条件：
  ├─ 试用期出勤率 ≥ 95%（事假不超过X天）
  └─ 无严重违反公司规章制度行为

特别约定条件（须在劳动合同中列明）：
  ├─ 销售岗位：试用期销售额达到 ¥XX
  ├─ 技术岗位：试用期完成 XX 项目并通过评审
  └─ 管理岗位：试用期团队管理指标达标
```

## 4. 试用期解除类型与条件

### 4.1 解除类型概览

```plaintext
试用期解除类型：

│ 解除方    │ 解除类型  │ 法定条件              │ 是否需提前通知 │
│---------│---------│---------------------│────────────│
│ 用人单位   │ 即时通知解除 │ 不符合录用条件            │ 无需提前通知   │
│ 用人单位   │ 预告解除   │ 胜任工作（须支付补偿）        │ 提前30日或支付代通知金│
│ 劳动者    │ 预告解除   │ 任意理由均可               │ 提前3日通知   │
│ 劳动者    │ 即时解除   │ 单位过错（欠薪/不缴社保/强令违章）│ 无需提前通知   │
│ 双方协议   │ 协商解除   │ 协商一致                 │ 按协议约定    │
└─────────┴─────────┴──────────────────────┴────────────┘
```

### 4.2 用人单位解除——不符合录用条件

**法定依据：**《劳动合同法》第三十九条第一款：劳动者在试用期内被证明不符合录用条件的，用人单位可以解除劳动合同。

**操作步骤：**

```plaintext
步骤一：录用条件确认
  ├─ 劳动合同/岗位说明书中已列明录用条件
  ├─ 劳动者入职时签字确认知悉
  └─ 录用条件应具体、可量化、可操作

步骤二：考核与证据收集
  ├─ 按约定的考核方式与时间节点进行考核
  ├─ 考核过程有书面记录
  ├─ 考核结果有直接上级签字
  └─ 重要考核节点及时告知劳动者

步骤三：解除前评估
  ├─ 法律顾问或HR复核解除理由与证据链
  ├─ 确认解除理由具体、充分
  └─ 确认不存在禁止解除情形（见4.4）

步骤四：解除通知与送达
  ├─ 向劳动者出具书面解除劳动合同通知书
  ├─ 解除理由载明"不符合录用条件"及具体事实
  ├─ 通知工会（建立工会者须履行告知程序）
  └─ 有效送达（直接送达/邮寄送达/公告送达）

步骤五：离职手续与资料归档
  ├─ 出具解除劳动合同证明
  ├─ 办理社保减员与档案转移
  └─ 解除通知与考核材料归档备查
```

**通知书必备要素：**
```
解除劳动合同通知书（要素清单）
  ├─ 劳动者基本信息（姓名、身份证号、岗位）
  ├─ 劳动合同签订日期与试用期期限
  ├─ 解除理由："不符合录用条件"
  ├─ 具体事实陈述（考核不达标的具体情形）
  ├─ 解除日期
  └─ 用人单位签章与日期
```

### 4.3 用人单位解除——提前通知（胜任工作）

试用期用人单位也可按《劳动合同法》第四十条，以劳动者不能胜任工作为由解除，但须：
- 证明劳动者不能胜任工作（经考核）
- 经过培训或调整工作岗位后仍不能胜任
- 提前30日书面通知或支付1个月工资代通知金
- 支付经济补偿（n个月工资，n=工作年限，不满半年按0.5个月计算）

**注意：** 此方式成本高、程序复杂，实务中"不符合录用条件解除"更为常用。

### 4.4 禁止解除情形

试用期解除同样适用《劳动合同法》第四十二条的禁止性规定，以下情形不得以"不符合录用条件"为由解除：

```plaintext
禁止解除情形（试用期亦适用）：

┌──────────────────────────────────┐
│ 从事接触职业病危害作业的劳动者     │
│ 未进行离岗职业健康检查             │
└──────────────────────────────────┘
                │
┌──────────────────────────────────┐
│ 疑似职业病病人或职业病病人          │
│ 在诊断或医学观察期间               │
└──────────────────────────────────┘
                │
┌──────────────────────────────────┐
│ 在本单位患职业病或因工负伤          │
│ 确认丧失或部分丧失劳动能力          │
└──────────────────────────────────┘
                │
┌──────────────────────────────────┐
│ 患病或非因工负伤                   │
│ 规定的医疗期内                    │
└──────────────────────────────────┘
                │
┌──────────────────────────────────┐
│ "三期"女职工                      │
│ （孕期、产期、哺乳期）              │
└──────────────────────────────────┘
                │
┌──────────────────────────────────┐
│ 距法定退休年龄不足5年              │
│ （含试用期在内）                  │
└──────────────────────────────────┘
                │
┌──────────────────────────────────┐
│ 疑似烈性传染病患者                 │
│ 在隔离治疗期间                    │
└──────────────────────────────────┘
```

## 5. 试用期解除常见争议与风险防范

### 5.1 常见争议类型

| 争议类型 | 常见情形 | 用人单位风险 |
|---------|---------|------------|
| 违法解除赔偿金 | 录用条件约定不明，解除被认定违法 | 2n经济赔偿金 |
| 工资差额 | 试用期工资低于约定工资80% | 补发工资差额 |
| 社保缴纳 | 试用期内未缴纳社保 | 补缴+滞纳金 |
| 试用期违法延长 | 约定1个月后延长至2个月 | 重新计算试用期（超期部分视为正式期） |
| 重复约定试用期 | 合同变更重新约定试用期 | 违法，须支付赔偿金 |
| 解除程序瑕疵 | 未通知工会/未书面通知 | 程序违法，须赔偿 |

### 5.2 风险防范检查清单

```plaintext
试用期解除合规检查清单：

□ 劳动合同已签订并生效
□ 劳动合同已明确约定试用期期限
□ 录用条件已书面化并经劳动者签字确认
□ 考核标准已量化或具体化，可操作
□ 考核周期明确（试用期内完成考核）
□ 考核过程有书面记录并存档
□ 考核结果已告知劳动者并取得签字确认
□ 解除理由具体、充分，有证据支撑
□ 不存在禁止解除情形
□ 已书面通知工会（建立工会者）
□ 已向劳动者出具书面解除通知书
□ 解除通知书已有效送达
□ 离职手续已办理（社保减员/档案转移）
□ 解除材料已归档备查
□ 工资与补偿金已足额支付
```

### 5.3 违法解除的法律后果

```plaintext
违法解除试用期劳动合同法律后果：

一、经济赔偿金
  用人单位违法解除 → 2n×月工资标准
  （n为工作年限，试用期工作年限不满半年按0.5计算）

二、劳动关系恢复
  劳动者要求继续履行劳动合同 → 用人单位须恢复劳动关系
  恢复后须支付违法解除期间的工资损失

三、工资损失赔偿
  劳动者不要求继续履行或无法继续履行 → 支付工资损失赔偿
  标准：劳动关系解除前后工资差额 × 期间

四、社保与公积金补缴
  试用期内未缴纳 → 补缴试用期间社保/公积金
  滞纳金由用人单位承担
```

## 6. 试用期常见实务问题

### 6.1 试用期能否"延长"

**结论：不得延长。**

- 试用期期限由劳动合同约定，用人单位单方延长属于变更劳动合同，须双方协商一致
- 实务中若延长被认定为"重新约定试用期"，违反《劳动合同法》第十九条第二款
- 用人单位径行延长试用期，构成违法，须按转正后工资标准支付超期工资，并可能承担赔偿金

**替代方案：**
- 在试用期内加强考核，及时作出解除决定
- 试用期届满前，若需进一步观察，可通过延长工作时间或调整岗位的方式，但不得单方延长试用期

### 6.2 试用期能否"重新约定"

**结论：不得重复约定。**

- 同一用人单位与同一劳动者只能约定一次试用期（劳动合同法第十九条第二款）
- 劳动合同续订、岗位变更或合同到期后续签，均不得再次约定试用期
- 违反者须按转正后工资标准支付工资，并可能承担赔偿金

### 6.3 试用期能否"中止"

**结论：一般不得中止，但可协商解决。**

- 试用期系劳动合同期限的一部分，一般不得因请假等原因中止计算
- 劳动者请长病假/产假等，可与劳动者协商延长试用期（须书面协商一致）
- 协商不成者，用人单位不得单方扣除试用期

### 6.4 试用期解除是否需要"赔偿"

| 解除类型 | 是否支付补偿/赔偿 |
|---------|----------------|
| 不符合录用条件解除 | 无补偿金（但须结清工资） |
| 协商一致解除 | 按工作年限支付经济补偿 |
| 违法解除 | 2n赔偿金 |
| 劳动者即时解除（单位有过错）| 经济补偿金 |

### 6.5 试用期解除后竞业限制的效力

- 试用期内解除，竞业限制条款是否生效取决于约定
- 若劳动合同约定"试用期届满后竞业限制生效"，则试用期解除后竞业限制不生效
- 若未作区分约定，竞业限制自劳动合同解除之日起生效
- 用人单位应在竞业限制期间按月支付经济补偿，否则劳动者可请求解除
