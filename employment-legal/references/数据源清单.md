## 二、商业法律数据库（YD / WKL / BD）

### 2.1 元典（YD）— yuandian-mcp

| 字段 | 值 |
|---|---|
| 标注 | `[YD]` |
| 工具 | yuandian MCP |
| 覆盖 | 法律法规 + 案例 + 实务文章 |
| 优势 | 案例数量大，地方裁判口径较全 |
| 路由 | 见 §三、查询路径 |

**YD 标注使用场景：**
- 引用案例时：`[YD]（2023）京02民终1234号`
- 引用法条时：`[YD] 劳动合同法第39条（2012修正）`
- 引用地方规定时：`[YD] 北京市工资支付规定`

### 2.2 无讼（WKL）— weiken-mcp

| 字段 | 值 |
|---|---|
| 标注 | `[WKL]` |
| 工具 | weiken MCP |
| 覆盖 | 综合法律检索 |
| 优势 | 跨库聚合，可对比多源 |

**WKL 标注使用场景：**
- 综合检索：`[WKL] 关键词: 经济补偿金基数`
- 类案对比：`[WKL] 检索 5 个类案，最高/最低赔偿额对比`

### 2.3 北达（BD）— beidalu API

| 字段 | 值 |
|---|---|
| 标注 | `[BD]` |
| 工具 | beidalu API |
| 覆盖 | 法规原文精准检索 |
| 优势 | 法规版本管控、时效标注 |

**BD 标注使用场景：**
- 法规原文：`[BD] 劳动合同法（2012修正）现行有效`
- 法规版本对比：`[BD] 劳动合同法2008版 vs 2012版 修订对照`

### 2.4 中国裁判文书网（GOV/WEB）

| 字段 | 值 |
|---|---|
| 标注 | `[GOV]` 或 `[web]` |
| 网址 | https://wenshu.court.gov.cn |
| 覆盖 | 公开裁判文书 |
| 注意 | 2021 年起公开数量大幅下降，部分案例需通过 [YD]/[WKL] 检索 |

## 三、典型判例库（场景化）

employment-legal 27 个 skill 中以下场景需要高频查阅判例库：

### 3.1 解除合法性（核心）

| 争议类型 | 类案检索关键词 | 数据源 |

| 违法解除 2N | "违法解除劳动合同"、"2N赔偿" | [YD] |

## 六、来源标注规范（强制）

每个结论/数字必须挂载来源标签：

| 标注 | 含义 | 触发条件 |
|---|---|---|
| `[YD]` | 元典检索结果 | yuandian-mcp 实际返回了该结果 |
| `[WKL]` | 无合力检索结果 | weiken-mcp 实际返回了该结果 |
| `[BD]` | 北达检索结果 | beidalu 实际返回了该结果 |
| `[GOV]` | 政府平台数据 | 官方网址直接访问的结果 |
| `[web]` | 网络搜索结果 | 通过 web search 工具获取 |
| `[model]` | 模型知识（须核实） | 默认 fallback |
| `[user]` | 用户提供的文件 | 用户直接粘贴/上传 |
| `[settled — last confirmed YYYY-MM-DD]` | 已验证的稳定引用 | 曾在 primary source 确认过，且在有效期内 |

**规则：**
- 标注在数字/结论**行末**，不在段落末尾（Anthropic CONTRIBUTING.md: "Attach provenance tags to numbers, not to paragraphs"）
- 关键结论须 2 个独立数据源
- 未连接的 connector 标注 `not connected — cites from training knowledge, verify before relying`
