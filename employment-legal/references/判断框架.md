### 2.2 核心判断节点（所有 skill 通用）

任何劳动法律问题的处理都经过这 5 个节点：

```
[节点 1] 法律关系性质识别
  ├─ 是否构成劳动关系？
  │   ├─ 是 → 进入劳动法体系
  │   └─ 否（劳务/承揽/合伙/实习）→ 进入民法体系
  ├─ 是否涉外（含港澳台）？
  │   ├─ 是 → 涉外劳动法专项判断
  │   └─ 否 → 大陆劳动法
  └─ 是否特殊群体（孕期/工伤/医疗期/高龄）？
      ├─ 是 → 触发 §五、高风险情形核查
      └─ 否 → 普通判断路径

[节点 2] 法律依据识别
  ├─ 劳动合同法 / 劳动争议调解仲裁法 / 工伤保险条例 / 职工带薪年休假条例
  ├─ 配套司法解释（最高院）
  ├─ 各省市工资支付条例 / 劳动合同条例
  └─ 集体合同 / 行业规范

[节点 3] 各省市差异匹配
  ├─ 见 §四、各省市裁判差异表
  └─ 数据源：[YD] + 各省市劳动人事争议仲裁委员会

[节点 4] 程序合规性检查
  ├─ 解除/裁员前是否通知工会？
  ├─ 制度是否经过民主程序 + 公示？
  ├─ 是否在法定期限内完成？
  └─ 证据是否充分？

[节点 5] 经济补偿/赔偿计算
  ├─ N / N+1 / 2N 适用情形
  ├─ 月工资基数（见 §六、计算框架）
  └─ 高收入员工三倍上限 + 12 年封顶
```

## 三、解除合法性判断（核心子流程）

劳动合同法第 36/39/40/41 条对应不同解除路径：

```
解除类型
├── 协商解除（第36条）
│   ├─ 用人单位提出 → 支付经济补偿金 N
│   └─ 劳动者提出 → 无经济补偿
│
├── 过失性解除（第39条）
│   ├─ 试用期不符合录用条件 → 录用条件须明确约定+考核标准客观
│   ├─ 严重违反规章制度 → 制度民主程序+公示+事实清楚
│   ├─ 严重失职/营私舞弊 → 损失金额可量化+因果关系
│   ├─ 同时与其他单位建立劳动关系 → 须书面通知后拒不改正
│   └─ 被追究刑事责任 → 生效法律文书
│   后果：用人单位无需支付经济补偿
│
├── 非过失性解除（第40条）
│   ├─ 医疗期满不能工作 → 医疗期届满 + 不能从事原工作也不能从事另行安排工作
│   ├─ 不能胜任工作 → 培训或调岗后仍不能胜任
│   └─ 客观情况重大变化 → 未能就变更合同达成协议
│   程序：提前30日书面通知 或 额外支付一个月工资（代通知金）
│   后果：支付经济补偿 N
│
├── 经济性裁员（第41条）
│   ├─ 人数门槛：20人以上 或 不足20人但占职工总数10%以上
│   ├─ 程序：提前30日向工会说明情况 + 向劳动行政部门报告
│   ├─ 优先留用：固定期限/长期合同/家庭无就业人员
│   └─ 后果：支付经济补偿 N
│
└── 违法解除（违反第39-41条程序/实质要件）
    └─ 后果：赔偿金 2N + 可能继续履行
```

**判断清单：**

1. ✅ 解除原因是否有法律依据（§36/§39/§40/§41）？
2. ✅ 是否有充分证据支持解除事实？
3. ✅ 程序是否合规（通知工会/提前30日/报告行政部门）？
4. ✅ 是否触碰第42条禁止解除情形（特殊群体）？
5. ✅ 经济补偿是否计算准确（基数/年限/上限）？

## 五、高风险情形（触发律师复核）

劳动合同法第 42 条 + 第 41 条 + 第 43 条 + 第 84 条：

- [ ] 第42条 接触职业病危害作业未进行离岗前职业健康检查
- [ ] 第42条 疑似职业病病人在诊断或医学观察期间
- [ ] 第42条 因工负伤并被确认丧失或部分丧失劳动能力
- [ ] 第42条 患病或非因工负伤，在规定的医疗期内
- [ ] 第42条 女职工在孕期、产期、哺乳期
- [ ] 第42条 在本单位连续工作满15年，且距法定退休年龄不足5年
- [ ] 第41条 经济性裁员（20人以上 或 10%以上）
- [ ] 第39条 严重违反规章制度（须证据确凿，程序合法）
- [ ] 第43条 用人单位单方解除应事先将理由通知工会
- [ ] 第84条 扣押证件/收取押金
- [ ] 试用期违法解除（第21条：不符合录用条件须证明）
- [ ] 涉外/港澳台劳动关系（须专项判断）

**触发任一项 → 升级至律师复核，不可直接执行。**

## 六、经济补偿/赔偿计算框架

### 6.1 计算公式

| 项目 | 计算 | 法律依据 |
|---|---|---|
| **N**（经济补偿） | 每满一年支付一个月工资；六个月以上不满一年按一年；不满六个月支付半个月 | 劳动合同法第47条 |
| **N+1** | N + 代通知金（一个月工资） | 劳动合同法第40条 |
| **2N**（违法解除赔偿） | N × 2 | 劳动合同法第87条 |
| **三倍上限** | 月工资 > 用人单位所在地上年度职工月平均工资三倍 → 按三倍支付 | 劳动合同法第47条 |
| **12年封顶** | 按三倍计算的年限最高不超过12年 | 劳动合同法第47条 |

### 6.2 月工资基数

- 指劳动合同解除或终止前**12个月**平均工资
- 包括基本工资、奖金、津贴、补贴等货币性收入
- **不包括**：非货币性福利、社保费、公积金个人部分、个税

### 6.3 各省市差异

| 地区 | "上年度职工月平均工资"取值 |
|---|---|
| 全国 | 用人单位所在地市级 |
| 北京 | 区级（实务通常较高） |
| 上海 | 市级 |
| 广东 | 市级（深圳有独立标准） |
| 江苏 | 市级（部分区域补充规定） |

## 七、争议处理流程（必经路径）

```
争议发生
  ↓
协商（非必经，但推荐优先）
  ↓
调解（企业内部调解委员会 / 基层人民调解组织）
  ↓
劳动仲裁（必经程序，**不得直接起诉**）
  ├─ 时效：1年（劳动争议调解仲裁法第27条）
  ├─ 管辖：用人单位所在地 或 合同履行地仲裁委
  ├─ 审理：45日内，复杂可延长15日
  └─ 费用：免费
  ↓
一裁终局（小额/标准明确的劳动争议）
  └─ 劳动者可起诉，用人单位仅在法定情形可申请撤销
  ↓
诉讼（不服仲裁裁决，15日内起诉）
```

**关键点：**
- 仲裁前置（除最高法第 168 条规定外）
- 仲裁时效可中止/中断（劳动争议调解仲裁法第27条）
- 一裁终局制度（劳动争议调解仲裁法第47条）
