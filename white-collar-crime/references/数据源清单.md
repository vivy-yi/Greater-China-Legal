## 五、数据源使用指南

### 5.1 优先级推荐

| 使用场景 | 推荐数据源 |
|---------|-----------|
| 查法条原文 | 国家法律法规数据库 |
| 查司法解释 | 最高人民法院官网 |
| 查典型案例 | 人民法院案例库 |
| 查裁判文书 | 中国裁判文书网 |
| 查合规指引 | 国资委/证监会官网 |
| 做学术研究 | 北大法宝+知网 |

### 5.2 时效性检查

| 检查频率 | 法规类型 | 说明 |
|---------|---------|-----|
| 使用前必查 | 所有法规 | 确认现行有效 |
| 每6个月 | 司法解释 | 可能有新解释 |
| 每1年 | 立案追诉标准 | 可能修订 |
| 实时关注 | 最高检/最高法指导性案例 | 持续更新 |

### 5.3 数据质量评估

| 评估维度 | 说明 |
|---------|-----|
| 权威性 | 官方来源 > 商业数据库 > 民间整理 |
| 时效性 | 确认最后更新日期 |
| 完整性 | 是否涵盖全部相关法规 |
| 可追溯性 | 是否有原文链接 |

## 六、引用规范

### 6.1 法条引用格式

```markdown
示例：中华人民共和国刑法（2023年修正）第266条
或：[法典名]第[条]条（[年份]年修正/修订）
```

### 6.2 案例引用格式

```markdown
指导性案例：张三受贿案，指导性案例XX号
裁判文书：（202X）最高法刑终XX号
典型案例：最高人民法院发布典型案例（XXXX年X月X日）
```

### 6.3 司法解释引用格式

```markdown
示例：最高人民法院、最高人民检察院关于办理贪污贿赂刑事案件适用法律若干问题的解释（2016年4月18日）
```
