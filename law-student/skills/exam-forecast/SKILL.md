# /exam-forecast — China Mainland

## CN法考考前重点预测

### 民法高频考点
1. 合同效力（有效/无效/可撤销）
2. 物权变动（登记/交付）
3. 担保物权（抵押/质押/留置）
4. 侵权责任（用人单位责任/产品责任）
5. 婚姻家庭（夫妻共同财产/离婚）

### 刑法高频考点
1. 故意犯罪形态（预备/未遂/中止）
2. 共同犯罪
3. 正当防卫/紧急避险
4. 刑罚裁量（自首/坦白/立功）
5. 分则重点罪名（诈骗/盗窃/抢劫）

### 民诉高频考点
1. 管辖（地域管辖/级别管辖）
2. 当事人（原告/被告/第三人）
3. 证据（举证责任/证据种类）
4. 诉讼时效
