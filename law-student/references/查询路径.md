# 法律学习查询路径 — 中国大陆

> 本文档提供法律学习场景实际检索入口和操作流程，按数据类型分类整理。

## 一、法条原文检索

### 1.1 中央法规

```
查询目标：民法典/刑法/民诉法 等

步骤 1：[GOV] 国家法律法规数据库
  URL: https://flk.npc.gov.cn
  检索字段：标题（如"中华人民共和国民法典"）
  输出：现行有效版本 + 历次修正记录

步骤 2：[BD] 版本时效验证
  工具：beidalu MCP（如果已连接）
  验证项：是否有最新修订/部分失效条款
  重点检查：
    - 民法典是否仍是当前版本（2021版）
    - 公司法（2024 修订）
    - 民事诉讼法（2023 修正）

步骤 3：[YD] 交叉验证
  检索同一法条，确认与 [GOV]/[BD] 一致
  不一致 → 标注 ⚠️ 来源冲突
```

### 1.2 司法解释检索

```
查询目标：最高法司法解释（学习重点）

URL: https://www.court.gov.cn → 司法解释 → 民事/刑事/行政
或: [YD] 检索"司法解释 + {学科}"

学习重点司法解释清单：
  - 民法典总则编解释（2022.02.25 施行）
  - 民法典合同编通则解释（2023.05.25 施行）
  - 民法典物权编解释（2021.01.01 施行）
  - 民法典人格权编解释（2021.10.01 施行）
  - 民法典侵权责任编解释（2021.10.01 施行）
```

### 1.3 法律法规英文版（JD 适用）

```
查询目标：英译法律（JD/比较法学习）

URL: 
  - 全国人大英文版 https://www.npc.gov.cn → English
  - 国务院法制办英文版 http://www.npc.gov.cn/englishnpc

[web] 检索："{法规名} English translation"
注意：英译本可能与中文版有时差，引用须标注版本
```

## 二、案例检索（学习场景特化）

### 2.1 最高人民法院指导案例

```
查询目标：指导性案例（必学）

URL: 人民法院案例库 https://rmfyalk.court.gov.cn
检索字段：关键词（如"侵权责任"）+ 案由
筛选：指导性案例 > 公报案例 > 普通案例

数量：
  - 指导案例：~200+（累计）
  - 公报案例：~2000+（累计）

学习价值：
  - 必学：指导案例（影响法官裁判）
  - 重要：公报案例
  - 选学：典型案例
```

### 2.2 类案检索（IRAC 训练）

```
查询目标：相似案例（用于案例分析训练）

[YD] 检索："{争议焦点} + 案由"
[WKL] 综合检索

操作流程：
  1. 问题定性（IRAC Issue）
  2. 关键事实提取
  3. 关键词组合
  4. 数据库检索
  5. 相似度评估（同案由+同地域+类似事实）

输出格式：
  | 案例 | 裁判日期 | 法院 | 裁判要旨 |
  | --- | --- | --- | --- |
  |（2023）最高法民申 XX 号 | 2023-06 | 最高法 | 要旨 |
```

### 2.3 英美判例对照（JD 适用）

```
查询目标：英美法经典判例（比较法学习）

URL:
  - Westlaw https://www.westlaw.com（需订阅）
  - HeinOnline https://heinonline.org（学术机构订阅）
  - Justia https://www.justia.com（免费）

[JD 经典判例]：
  - 民法：Palsgraf v. Long Island R. Co.（侵权因果关系）
  - 刑法：People v. McManus（一般要件）
  - 合同：Hawkins v. McGee（损害赔偿期待利益）
  - 宪法：Marbury v. Madison（司法审查）

[web] 检索："{案例名} case brief"

输出：case brief 格式
  Facts / Issue / Rule / Application / Conclusion
```

## 八、检索效率技巧（学习场景）

### 8.1 类案检索四步法

```
1. 问题定性（IRAC Issue）
   例：民事侵权 → 因果关系 → 近因认定

2. 关键事实提取
   例：损害事实、加害行为、时间、地点、因果链

3. 关键词组合
   规范性：侵权责任构成要件
   非规范性：行为与结果的关联

4. 数据库检索
   [YD] + [WKL] + [GOV] 交叉
   地域/时间/审级范围确定
   相似度评估（同案由+同地域+类似事实）
```

### 8.2 真题训练三步法

```
1. 限时模拟（练习时间管控）
   - 客观题：每题 ≤ 90 秒
   - 主观题：每题 ≤ 30-40 分钟

2. 解析对照
   - 官方解析 vs 商业解析
   - 不一致时：回归法条原文

3. 错题归档
   - 记录错题原因（知识点遗漏/审题错误/时间不够）
   - 针对性强化训练
```

### 8.3 学术写作（学硕/JD）

```
1. 文献检索
   - [BD] CNKI + 万方
   - 限定：核心期刊 + 近 5 年

2. 文献筛选
   - 高被引 = 影响力大
   - 学界权威 = 立场主流

3. 文献管理
   - Zotero https://www.zotero.org
   - NoteExpress

4. 写作规范
   - GB/T 7714（参考文献）
   - 法学引注手册
```

## 九、应急检索（考试季）

### 9.1 法考冲刺期

```
考前 2 周：
  1. 重做近 3 年真题（计时模拟）
  2. 主观点默写（每科高频考点）
```

## 十、数据源故障回退

| 故障 | 回退方案 |
|---|---|
| [YD] yuandian-mcp 未连接 | 标注 `not connected` + [BD] 知网兜底 |
| [BD] 知网不可访问 | [BD] 万方替代 + 学校图书馆数据库 |
| [GOV] 政府网站不可访问 | [web] 替代 + 时效核验 |
| 全部数据库不可用 | 标注 `[model]` + 建议核实主流教材 |

**学习场景特别提示**：
- 学习用旧版教材（如 2023 版 vs 2024 版）的**核心条款通常不会变**
- 但**新增司法解释**（如民法典系列解释）必须用最新版
- 引用前标注**版本日期 + 来源**
