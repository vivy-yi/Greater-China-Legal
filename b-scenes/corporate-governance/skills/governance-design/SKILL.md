# /governance-design — 公司治理结构设计

## 有限责任公司治理结构

### 三会一层

| 机构 | 组成 | 法定职权 |
|---|---|---|
| 股东会 | 全体股东 | 修改章程/选举董事监事/利润分配/解散 |
| 董事会 | 3-13人（可设副董事长） | 召集股东会/执行决议/决定经营计划 |
| 监事会 | ≥3人（股东代表+职工代表） | 检查财务/监督董事高管/提议召开股东会 |
| 经理层 | 总经理由董事会聘任 | 主持经营/组织实施决议 |

### 表决权设计

| 表决事项 | 有限责任公司 | 股份有限公司 |
|---|---|---|
| 普通决议 | 过半数表决权 | 出席过半 |
| 重大决议 | 2/3表决权 | 出席2/3 |
| 公司章程修改 | 2/3表决权 | 2/3表决权 |

## 控制权设计工具

### AB股（同股不同权）

**有限责任公司：**
- 章程可约定不按出资比例行使表决权
- 可约定某些股东享有特殊表决权

**股份有限公司（A股不允许，科创板/创业板允许）：**
- A类股：每股1票
- B类股：每股10票（或更多）
- B类股须符合同股不同权条件（仅限科创板/创业板上市公司）

### 一致行动人

**定义：** 多个股东约定就特定事项一致行使表决权

**用途：**
- 创始团队绑定表决权
- 抱团对抗投资人

### 表决权委托

**创始人可要求投资人将表决权委托给创始人**
- 投资人放弃表决权
- 创始人保持控制

## 股东协议核心条款

| 条款 | 说明 |
|---|---|
| 优先认购权 | 新股发行时原股东优先购买 |
| 优先购买权 | 股权转让时原股东优先购买 |
| 拖售权 | 投资人可强制其他股东一起出售 |
| 跟随权 | 投资人出售时其他股东可一同出售 |
| 知情权 | 定期获得财务报表/重大事项通知 |
| 竞业禁止 | 股东不得从事竞争业务 |
| 保密义务 | 股东须对商业秘密保密 |
