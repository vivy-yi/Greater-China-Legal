# 数据源清单 — [YD]/[GOV]/[WKL] 覆盖范围

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [YD] | 法律精灵综合检索 | 实时法规+判例+律师分析聚合 |
| [GOV] | 政府官方数据 | 裁判文书网/人社局官网/官方文件 |
| [WKL] | 律师实务经验 | 律师分析文章/法律科技产品 |
| [BD] | 内部数据 | 企业历史案件/合同/仲裁记录 |
| [model] | 法律推理 | LLM基于法律知识的推理 |

---

## [YD] 法律精灵

**覆盖范围：**
- 劳动法/劳动合同法全条款及司法解释
- 各省市高院/中院的劳动争议判例
- 各地仲裁委发布的典型案例集（部分）
- 律师实务分析文章（部分）
- 最新法规动态（实时更新）

**不覆盖：**
- 地方人社局的具体补贴标准（须去[GOV]查）
- 最新判例可能滞后3-6个月
- 部分地方仲裁委内部掌握的裁量标准

**使用建议：**
```
搜索格式：[城市] + [争议类型] + [具体问题]
时间筛选：优先近2年内的判例
来源筛选：优先"判例"类型结果
```

---

## [GOV] 裁判文书网

**覆盖范围：**
- 全国各级法院公开的民事判决书（劳动争议类）
- 中院/高院的二审判决（权威性高）
- 最高院指导案例（全国参考价值）

**不覆盖：**
- 仲裁委裁决书（大部分不公开）
- 一审未上诉的基层法院判决（部分公开）
- 2021年之前的部分历史判例
- 涉及当事人隐私的案件（部分不公开）

**使用建议：**
```
检索路径：wenshu.court.gov.cn
关键词：[城市名] + 劳动争议 + [争议类型]
法院级别：中院/高院优先
时间：近3年内优先
案号：如能获取案号，可直接查询
```

---

## [GOV] 各地人社局官网

**覆盖范围：**
- 最低工资标准（每年更新）
- 高温津贴标准（各省市明确）
- 社保缴费基数上下限（每年调整）
- 劳动关系官方指导文件
- 仲裁委联系方式和地址

**不覆盖：**
- 具体的裁判口径（这是法院/仲裁委的职能）
- 仲裁时效的具体适用
- 裁量性问题的官方表态

**常用入口：**

| 城市 | 官网 | 特色数据 |
|---|---|---|
| 北京 | rsj.beijing.gov.cn | 劳动关系指导案例集 |
| 上海 | 12333sh.gov.cn | 仲裁时效解答 |
| 广东 | hrss.gd.gov.cn | 劳动标准规定 |
| 浙江 | zjhrss.gov.cn | 典型案例汇编 |
| 江苏 | jshrss.gov.cn | 劳动争议审判指南 |
| 四川 | rst.sc.gov.cn | 羌族地区特色规定 |
| 重庆 | cqhrss.gov.cn | 直辖市统一口径 |

## [WKL] 律师分析

**覆盖范围：**
- 律师实务经验总结
- 特定城市的深度分析
- 新型争议的法律分析
- 对裁判趋势的预判

**不覆盖：**
- 不具有法律约束力（仅参考）
- 可能存在立场偏见（代表某一方当事人）
- 时效性不稳定

**使用建议：**
```
使用场景：
- [YD]/[GOV]均查不到本地数据时
- 需要了解新型争议的处理方式时
- 需要预判裁判趋势时

注意事项：
- 须核实发表时间和作者背景
- 须与其他来源交叉验证
```

## [BD] 内部数据

**覆盖范围：**
- 企业历史劳动仲裁案件
- 内部和解/调解记录
- 员工手册和规章制度
- 劳动合同模板及历史版本

**使用场景：**
```
场景1：评估诉讼风险
→ 查询企业历史同类案件的处理结果
→ 判断胜诉/败诉概率

场景2：制定调解策略
→ 分析企业历史调解让步幅度
→ 评估对方当事人的谈判风格

场景3：规章制度合规性
→ 对照企业历史被认定为违规的条款
→ 评估现行规章制度是否需要修改
```

**⚠️ 须注意：内部数据仅用于企业内部，不应作为法律依据对外引用。**

## 数据源组合使用

### 常规查询（补偿金/工资基数）
```
[YD]法律精灵 → [GOV]裁判文书网（交叉验证）→ [WKL]律师分析（补充）
```

### 法规依据查询
```
[YD]法律精灵 → [GOV]人社局官网（确认最新标准）
```

### 裁量标准查询（无明确法规时）
```
[GOV]裁判文书网（高院/中院判例）→ [YD]法律精灵（综合）→ [WKL]律师分析（补充）
```

### 趋势预判
```
[WKL]律师分析（近期文章）→ [YD]法律精灵（最高院指导案例）→ [GOV]裁判文书网（新判例）
```
