# /compensation-calculator — 经济补偿金/赔偿金计算引擎

## 计算公式速查

| 解除类型 | 计算公式 | 上限 |
|---|---|---|
| 协商解除（单位提出） | N × 月工资基数 | 3倍×12年（即封顶36个月） |
| 员工非过错解除 | N × 月工资基数 | 3倍×12年 |
| 经济性裁员 | N × 月工资基数 | 3倍×12年 |
| 合同到期不续签（单位不续） | N × 月工资基数 | 无上限 |
| 违法解除 | 2N × 月工资基数 | 6倍×12年 |

**N = 工作年限（6个月以上按1年计算，不满6个月按0.5年计算）**

---

## 第一步：确定月工资基数

### 工资构成（含以下项目）

- 基本工资
- 岗位工资/职务工资
- 绩效工资/提成（离职前12个月平均）
- 奖金/年终奖（离职前12个月平均）
- 补贴（交通补贴/通讯补贴/住房补贴等）

### ⚠️ 加班费/奖金是否计入基数——必须查询当地裁判口径

**各地口径不同，必须通过 `region-adjudicator` 查询目标城市的最新裁判口径。**

常见口径（参考，非固定）：
- **北京/上海/广东/江苏/浙江/四川/重庆**：加班费**不计入**基数，奖金**不计入**基数
- **全国一般标准**：加班费和奖金通常计入月工资基数

**本skill仅提供一般规则，实际计算须以当地仲裁口径为准。**

### 工资构成（不含以下项目）

- 单位代缴的社保个人部分（属于用人单位社保费）
- 单位代缴的公积金个人部分（属于用人单位住房公积金费）
- 防暑降温费（福利，不计入工资）
- 工作服/体检等福利待遇

### 三倍社平工资封顶

**全国上年度职工月平均工资3倍（具体以各地为准）。本数据为一般参考。**

**⚠️ 重要：各省市上年度职工月平均工资标准不同，须调用 `region-adjudicator` skill 查询目标城市的最新数据。**

超过3倍的部分不计入缴费年限（最多计算12年）

**计算示例（超过3倍情形）：**
- 月工资 50,000元，工作15年
- 基数封顶 = 3倍社平工资
- 补偿年限 = 12年（超出3倍部分不计入）
- 补偿金额 = 12 × 36,000 = 432,000元（而非 15 × 50,000 = 750,000元）

---

## 第二步：确定工作年限

### 年限计算规则

| 在职时间 | 计算结果 |
|---|---|
| 满6个月 | 按1年计算 |
| 不满6个月 | 按0.5年计算 |
| 零散时间合并计算 | 6个月以上/以下分别合并 |

**注意：** 关联企业工作年限合并计算（母子公司/同一实际控制人），须有证据证明

### 特殊情形

| 情形 | 年限计算 |
|---|---|
| 军龄 | 军龄视同本单位工作年限 |
| 下乡/知青 | 视同工作年限（须有档案记录） |
| 待业期间 | 不计算（除非单位安排待岗） |
| 病假/产假期间 | 计算年限 |

---

## 第三步：计算经济补偿金

### 标准公式

```
经济补偿金 = N × 月工资基数

其中：
- N = 工作年限
- 月工资基数 = 离职前12个月平均工资（应发，含加班费）
```

### N的确定（年限）

| 工作时间 | N值 |
|---|---|
| 1年以下（含试用期末） | 0.5（不满6个月按0.5） |
| 1年以上 | 实际年限 |

---

## 第四步：协商解除的特殊计算

### N+1（代通知金）

- 适用：员工非过错解除（劳动合同法第40条）
- "1" = 一个月工资（代通知金，上月工资标准）
- 是否支付：可协商，实践中常见

### 代通知金计算基数

= 上一个月正常出勤工资（不含加班费）

---

## 第五步：2N赔偿金

### 适用情形

违法解除劳动合同（见termination-legality判断）

### 计算公式

```
赔偿金 = 2N × 月工资基数
```

### 2N与继续履行的选择

| 选项 | 适用情形 |
|---|---|
| 要求继续履行 | 劳动合同还能履行（岗位存在/公司仍运营） |
| 要求2N赔偿金 | 劳动合同已无法履行（岗位撤销/公司注销） |
| 同时主张工资损失+赔偿金 | 单位拖延不安排工作期间的工资损失 |

---

## 地区差异：各省市裁判口径

### 经济补偿金基数差异（核心差异）

| 省市 | 加班费是否计入基数 | 年终奖/提成是否计入 | 备注 |
|---|---|---|---|
| 北京 | 不含加班费 | 不含 | 有明确裁判意见 |
| 上海 | 约定工资不含加班费 | 含（按实际发放） | 以约定工资为基数 |
| 广东 | 含加班费 | 含 | 广东省高院指导意见 |
| 浙江 | 含加班费 | 含 | — |
| 江苏 | 含加班费 | 含（特殊情形） | — |
| 四川 | 含加班费 | 含 | — |

### 高温津贴是否计入基数

| 省市 | 是否计入 |
|---|---|
| 大多数省市 | 不计入（属于福利，不属于工资） |
| 特殊情形 | 部分仲裁委可能计入 |

### 违法解除赔偿金基数地区差异

**绝大多数省市：** 2N计算基数不含加班费（以应发工资为基数，但加班费争议较大）

**建议：** 查询当地仲裁委/法院口径，以实际判例为准

## 综合计算示例

### 示例1：正常解除（协商解除）

```
工作年限：5年3个月 → N=5.5
月工资基数：15,000元/月（含加班费）
解除类型：协商解除（单位提出）
地区：上海

计算：
- N = 5.5
- 基数 = 15,000元
- 补偿金额 = 5.5 × 15,000 = 82,500元

上海口径：加班费不含在基数内
实际基数（扣除加班费后）：12,000元/月
补偿金额 = 5.5 × 12,000 = 66,000元
```

### 示例2：违法解除（2N）

```
工作年限：8年
月工资基数：30,000元（含加班费、奖金）
解除类型：违法解除
地区：北京

上限检查：
- 社平工资3倍 = 约36,000元/月
- 30,000 < 36,000，不超上限

计算：
- 2N = 2 × 8 = 16
- 赔偿金 = 16 × 30,000 = 480,000元

北京口径：加班费不计入基数
重新计算基数：22,000元/月（扣除加班费后）
赔偿金 = 16 × 22,000 = 352,000元
```
