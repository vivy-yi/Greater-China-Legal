# /negotiation-advisor — 劳动争议调解策略

## 协商筹码评估框架

### 员工方筹码

| 筹码 | 强度 | 说明 |
|---|---|---|
| 法律依据充分（违法解除证据确凿） | ⭐⭐⭐ | 最强筹码 |
| 劳动仲裁记录影响单位后续招调人员 | ⭐⭐ | 单位一般不希望有仲裁记录 |
| 媒体/舆论压力（大厂/知名企业） | ⭐⭐ | 社会形象考量 |
| 继续在公司工作意愿 | ❌ | 如坚持工作则无筹码 |
| 仲裁时效即将届满 | ⭐⭐ | 倒逼单位协商 |

### 单位方筹码

| 筹码 | 强度 | 说明 |
|---|---|---|
| 诉讼成本（时间/律师费） | ⭐⭐ | 拖字诀 |
| 一裁终局（小额，员工无法上诉） | ⭐⭐ | 适用小额时 |
| 举证能力（证据充分） | ⭐⭐⭐ | 关键筹码 |
| 预算限制（国企/上市公司有预算约束） | ⭐ | 可能被财务限制 |

## 协商区间计算

### 理论值 vs 协商区间

```
理论应得金额 = 根据法规计算的金额
协商下限 = 理论金额 × [0.6-0.8]（取决于证据充分程度）
协商上限 = 理论金额（单位一般不超过理论值）

协商区间：
[协商下限, 理论应得金额] — 员工让步空间
```

### 示例

```
违法解除，经济补偿金理论值 = 10万元

员工筹码评估：
- 有解除通知书（书面）→ 证据充分
- 单位无考核记录 → 程序违规
- 综合筹码：强

协商区间：
- 下限：6万元（60%）
- 上限：10万元
- 建议目标：7-8万元

单位评估：
- 单位希望快速解决
- 走仲裁可能输
- 底线：8万元

最终协商结果：7.5万元
```

## 调解时机选择

| 时机 | 优点 | 缺点 |
|---|---|---|
| 仲裁前（立案前） | 快速、保密、成本低 | 无仲裁背书 |
| 仲裁中（开庭前） | 仲裁员可促成调解 | 需出庭 |
| 开庭后 | 仲裁员已知双方立场 | 时间成本 |
| 裁决前 | 双方立场明确 | 接近裁决结果 |

## 各争议类型的协商策略

### 违法解除（最强立场）

**员工最优策略：**
1. 先向单位发书面解除通知（固定证据）
2. 同步申请仲裁（触发60日时效）
3. 仲裁中看单位态度再决定是否调解
4. 协商底线：理论金额70%以上

### 协商解除（平等谈判）

**员工最优策略：**
1. 要求单位书面说明解除理由
2. 评估补偿方案（与N或2N对比）
3. 重点谈N+1（协商解除+代通知金）
4. 注意协议措辞（避免"放弃一切权利"）

### 拖欠工资（立场明确）

**员工最优策略：**
1. 发送书面催告（固定时间节点）
2. 同步申请仲裁（时效无限制，但建议尽快）
3. 工资争议协商空间小，基本按实际欠发金额

## 仲裁员调解偏好

### 仲裁员的角色

- **促成调解是仲裁员的职责之一**
- 仲裁员通常会给出一个"建议裁决额"供参考
- 仲裁员一般会压低期望值（各打五十大板）

### 利用仲裁员

**策略：**
- 开庭前主动联系仲裁员说明情况
- 让仲裁员了解自己的底线
- 请仲裁员帮忙"压"对方
