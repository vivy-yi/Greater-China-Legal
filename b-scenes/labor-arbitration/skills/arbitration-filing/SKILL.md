# /arbitration-filing — 劳动仲裁申请材料生成

## 材料清单

### 基础材料

1. 仲裁申请书（一式两份，仲裁委留存一份）
2. 身份证复印件
3. 用人单位企业信息（国家企业信用信息公示系统打印）
4. 证据材料（按清单整理）

### 仲裁申请书要素

**必须包含：**
- 申请人（劳动者）：姓名、身份证号、地址、电话
- 被申请人（单位）：名称、统一社会信用代码、地址、电话
- 仲裁请求（分项列明金额）
- 事实与理由（简明扼要）
- 证据清单

## 仲裁请求撰写规范

### 常见仲裁请求模板

**违法解除赔偿金（2N）：**
```
请求：被申请人支付违法解除劳动合同赔偿金人民币[X]元。
依据：劳动合同法第48条、第87条。
```

**经济补偿金（N）：**
```
请求：被申请人支付经济补偿金人民币[X]元。
依据：劳动合同法第46条、第47条。
```

**代通知金：**
```
请求：被申请人支付未提前30日通知的代通知金人民币[X]元。
依据：劳动合同法第40条。
```

**拖欠工资：**
```
请求：被申请人支付[年]年[月]至[年]月工资人民币[X]元。
依据：劳动法第50条。
```

**加班费：**
```
请求：被申请人支付在职期间加班费人民币[X]元。
依据：劳动法第44条 / 工资支付暂行规定第13条。
```

**未休年休假工资：**
```
请求：被申请人支付[年度]年度未休年休假工资人民币[X]元。
依据：职工带薪年休假条例第5条。
```

**工资差额：**
```
请求：被申请人支付[年份]月工资差额人民币[X]元。
依据：劳动合同法第35条（变更劳动合同须协商一致）。
```

## 证据材料整理

### 证据清单模板

```
证据1：劳动合同
证明内容：劳动关系存续期间
证据来源：[单位提供/本人保存]

证据2：工资发放记录（银行流水）
证明内容：工资标准、发放金额
证据来源：[银行打印]

证据3：社保缴费记录
证明内容：工作年限、缴费基数
证据来源：[社保局打印]

证据4：解除通知书/辞退通知
证明内容：解除原因、时间
证据来源：[单位出具]

证据5：[其他证据]
...
```

### 证据提交要求

- 提交复印件（仲裁委核实原件）
- 证据须有页码
- 超过10页须附目录
- 电子证据须打印并注明来源

## 仲裁时效

| 情形 | 时效 | 起算点 |
|---|---|---|
| 解除/终止争议 | 60日 | 知道或应当知道权利被侵害之日 |
| 拖欠工资（劳动关系存续） | 无限制 | — |
| 拖欠工资（劳动关系解除后） | 1年 | 劳动关系解除之日 |
| 追缴社保 | 无时效 | 向劳动监察投诉，不受仲裁时效限制 |

**⚠️ 超过60日时效：丧失胜诉权（除非有正当理由）**

正当理由：不可抗力/无民事行为能力/限制民事行为能力人的法定代理人未确定/其他正当理由

## 仲裁委管辖

### 管辖规则

| 情形 | 管辖仲裁委 |
|---|---|---|
| 注册地与实际经营地不同 | 实际经营地仲裁委也可受理 |
| 劳务派遣 | 合同履行地或派遣单位所在地 |
| 建筑工地工伤 | 项目所在地 |

### 查询方法

```
百度：[区/市]劳动人事争议仲裁委员会 地址电话
```

## 特殊程序

### 一裁终局（小额诉讼）

**适用条件：**
- 标的额≤当地月最低工资标准×12个月
- 追索劳动报酬/工伤医疗费/经济补偿金

**效力：**
- 单位：不得起诉（只能中级法院申请撤销）
- 员工：可以起诉

### 调解优先

**仲裁前调解（各省市有劳动争议调解中心）：**
- 优点：快速（15天内）、免费、不伤和气
- 缺点：协议无强制执行力（须仲裁确认）

**仲裁中调解：**
- 仲裁员会主动提出调解
- 调解协议有强制执行力（一方不履行可申请法院执行）
