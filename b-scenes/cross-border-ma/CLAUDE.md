## 交易结构选择

### 股权收购 vs 资产收购

| 对比维度 | 股权收购 | 资产收购 |
|---|---|---|
| 税务负担 | 较高（股权转让所得税） | 较低（资产转让增值税/所得税） |
| 债务承担 | 继承全部债务 | 可选择资产范围 |
| 员工 | 继承劳动关系 | 须重新签署合同 |
| 审批 | 股权变更须审批 | 资产过户通常无需审批 |
| 时间 | 较长（尽职调查） | 较短 |

### 境内架构选择

| 架构 | 适用场景 | 说明 |
|---|---|---|
| 直接持股 | 非限制行业 | 最简单 |
| WFOE | 限制行业 | 外商独资企业 |
| VIE | 负面清单行业 | 协议控制，存在法律风险 |
| 合资企业 | 限制行业须合资 | 双方共同出资 |

## 尽职调查范围

### 必调项目

| 调项 | 内容 |
|---|---|
| 法律尽职调查 | 目标公司设立/股权/资产/重大合同/诉讼 |
| 财务尽职调查 | 财务报表/资产负债表/现金流 |
| 税务尽职调查 | 历史税务合规/转让定价/欠税 |
| 劳动尽职调查 | 劳动合同/社保合规/集体合同 |
| 环境尽职调查 | 环评/排污许可/土地污染 |

### 并购后整合事项

```
员工整合：劳动关系继承/薪酬体系调整/管理架构
税务整合：合并纳税/转让定价政策统一/税务档案
合规整合：牌照/许可/资质继承
文化整合：管理制度/沟通机制
```

## 审批流程

### 中国境内企业收购境外企业（ODI备案）

| 步骤 | 机关 | 时间 |
|---|---|---|
| 发改委备案/核准 | 发改委 | 约20工作日 |
| 商务部备案/核准 | 商务部 | 约15工作日 |
| 外管局外汇登记 | 外管局 | 约5工作日 |
| 反垄断审查 | 市监总局 | 约30工作日（须申报情形） |

### 须进行经营者集中申报的情形

```
申报条件（2022年修订）：
- 参与集中的经营者上一年度全球营业额>100亿元
- 或中国境内营业额>20亿元
- 且另一方在中国境内营业额>4亿元

豁免情形：
- 同一经营者持有全部股份的全资子公司之间的集中
- 特定情形的基金参与集中
```

## 交易文件清单

### 核心文件

| 阶段 | 文件 |
|---|---|
| 意向书阶段 | LOI/MOU（不具约束力） |
| 尽职调查阶段 | NDA/信息备忘录 |
| 交易阶段 | SPA/股权收购协议/资产收购协议 |
| 交割阶段 | 交割确认书/股份转让书/董事会决议 |
| 交割后 | 变更登记文件/员工安置方案 |

### SPA核心条款

```
必备条款：
- 交易结构和价格（估值调整机制）
- 陈述与保证（Reps & Warranties）
- 交割条件（Conditions）
- 赔偿责任上限（Cap）
- 锁定期（Lock-up）
- 竞业禁止（Non-compete）
- 管理权保留（如须）

中国特色条款：
- 对赌协议（九民纪要后效力有限）
- 回购权（须合法设计）
- 优先清算权（须合法设计）
```
