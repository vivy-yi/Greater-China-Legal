# /dd-checker — 法律尽职调查

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取调查范围/审批要求。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取调查范围/风险等级
2. 制定尽职调查清单
3. 分模块核查（设立/股权/资产/合同/诉讼/劳动）
4. 识别法律风险
5. 输出尽调报告

## 六维核查框架

### 维度一：公司设立

| 核查项 | 风险 |
|---|---|
| 营业执照 | 虚假设立/超范围经营 |
| 出资是否到位 | 出资不足/虚假出资 |
| 设立批准/许可证 | 无证经营/许可证过期 |
| 外商投资企业批准证书 | 外资合规性 |

### 维度二：股权结构

| 核查项 | 风险 |
|---|---|
| 股东名册 | 代持/股权争议 |
| 股权转让历史 | 瑕疵转让/优先购买权未保障 |
| 股权质押/查封 | 权利受限 |
| 实际控制人 | VIE结构/穿透核查 |

### 维度三：资产权属

| 核查项 | 风险 |
|---|---|
| 不动产 | 无产权证/抵押/查封 |
| 知识产权 | 权属争议/未缴费失效/侵权 |
| 设备/车辆 | 抵押/融资租赁 |
|应收账款| 坏账风险 |

### 维度四：重大合同

| 核查项 | 风险 |
|---|---|
| 采购/销售合同 | 不正常交易/潜在损失 |
| 借款/担保合同 | 隐性负债 |
| 关联交易 | 利益输送 |
| 限制性条款 | 竞业禁止/保密 |

### 维度五：诉讼与行政处罚

```
查询路径：
- 中国裁判文书网：https://wenshu.court.gov.cn/
- 执行信息公开网：http://zxgk.court.gov.cn/
- 国家企业信用信息公示系统：https://www.gsxt.gov.cn/
- 市场监管行政处罚文书网
```

### 维度六：劳动合规

| 核查项 | 风险 |
|---|---|
| 劳动合同签署 | 未签/过期/不完整 |
| 社保/公积金缴纳 | 未足额缴纳 |
| 特殊员工 | 三期/工伤/职业病 |
| 集体合同 | 群体性事件风险 |
