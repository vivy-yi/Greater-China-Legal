# /fdi-odi-filing — FDI/ODI备案指引

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取备案机关/流程/材料清单。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取备案机关/材料清单
2. 判断是FDI还是ODI
3. 梳理办理流程
4. 准备材料
5. 输出指引

## FDI vs ODI

| 类型 | 含义 | 投资方向 | 备案机关 |
|---|---|---|---|
| FDI | 外来直接投资 | 境外→中国境内 | 商务部门/市场监管部门 |
| ODI | 对外直接投资 | 中国境内→境外 | 发改委/商务部/外管局 |

## ODI备案流程

### 第一步：发改委备案/核准

```
事项：境外投资项目备案/核准
机关：国家发改委（敏感项目）/省级发改委（非敏感项目）
时间：约20工作日
条件：须有真实境外投资项目
材料：项目申请报告/项目备案表/投资主体营业执照/财务报表
```

### 第二步：商务部备案/核准

```
事项：境外企业设立备案/核准
机关：商务部（敏感）/省级商务部门（非敏感）
时间：约15工作日
材料：对外投资设立企业证书申请表/境外公司章程/董事会决议/可行性研究报告
```

### 第三步：外汇登记

```
机关：开户银行或外管局
事项：ODI外汇登记
时间：约5工作日
材料：发改委批文/商务部批文/营业执照
注意：须在ODI备案完成后方可办理外汇登记
```

## FDI设立流程

### 外商投资法后流程（2020年后）

```
第一步：商务部门审批/备案
  → 《外商投资准入负面清单》内须审批，负面清单外须备案
  → 商务部门出具《外商投资企业批准证书》或《备案回执》

第二步：工商设立登记
  → 提交材料：公司章程/投资者主体资格证明/批准证书/法定代表人任职文件
  → 领取营业执照

第三步：税务/社保登记
  → 税务登记/社保登记/公积金登记

第四步：外汇登记（开立外汇资本金账户）
```
