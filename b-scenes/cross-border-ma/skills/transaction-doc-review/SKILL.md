# /transaction-doc-review — 并购交易文件审查

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取条款效力判断标准/九民纪要。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取对赌/回购/优先清算条款规则
2. 逐条审查核心条款
3. 识别法律风险
4. 给出修改建议
5. 输出审查报告

## 核心条款审查

### 对赌条款（九民纪要2019第5条）

| 条款类型 | 效力 | 风险点 |
|---|---|---|
| 目标公司股东对赌 | ✅ 通常有效 | — |
| 目标公司对赌 | ✅ 通常有效 | 须目标公司有利润可分配 |
| 固定收益回购 | ⚠️ 可能无效 | 可能被认定"名股实债" |
| 担保/抵押 | ✅ 有效 | — |

### 优先清算权

| 条款类型 | 效力 | 说明 |
|---|---|---|
| 优先清算权（国内公司法） | ⚠️ 效力有争议 | 清算剩余财产分配属法定 |
| 优先分红权 | ✅ 有效 | 章程可约定 |

### 反稀释条款

| 类型 | 风险点 |
|---|---|
| 完全棘轮 | 最保护投资方，对创始人最不利 |
| 加权平均 | 较公平，按比例调整 |

### 拖带权（Drag-along）

```
含义：多数股东可强制少数股东一起出售股权
注意：须约定合理的价格机制和条件
```

### 跟随权（Tag-along）

```
含义：少数股东可跟随多数股东一起出售
作用：保护小股东变现权
```

## 赔偿责任条款

| 条款 | 审查要点 |
|---|---|
| 陈述与保证（Reps） | 范围是否过宽 |
| 赔偿上限（Cap） | 通常为交易对价的百分比 |
| 追诉时效 | 通常为交割后12-24个月 |
|  deductibles | 通常有免赔额 |
