# /evidence-guide — IP侵权证据收集指引

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取证据规则/公证要求。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取证据类型/公证要求
2. 分析侵权场景
3. 确定所需证据类型
4. 给出证据收集指引
5. 提示证据固定方式

## 证据类型清单

### 商标侵权证据

| 证据类型 | 收集方式 |
|---|---|
| 侵权产品实物 | 购买公证（线上/线下） |
| 侵权产品销售页面截图 | 网页公证 |
| 进货凭证 | 发票/合同 |
| 侵权对比材料 | 我方注册商标证+侵权物对比 |
| 损害证明 | 销售下降数据/广告投放记录 |

### 专利侵权证据

| 证据类型 | 收集方式 |
|---|---|
| 被控侵权产品 | 购买/现场取样 |
| 被控侵权方法 | 视频录制/证人证言 |
| 专利权利要求对比 | 技术分解表 |
| 损害证明 | 专利许可费/市场份额下降 |

### 著作权侵权证据

| 证据类型 | 收集方式 |
|---|---|
| 原作品 | 著作权登记证书/创作手稿 |
| 侵权作品 | 公证购买/网页公证 |
| 接触可能性 | 原作品发表记录 |
| 实质性相似 | 专家鉴定意见 |

## 证据固定方式

| 方式 | 适用场景 | 优缺点 |
|---|---|---|
| 公证 | 线上侵权/现场取证 | 证明力强，须及时 |
| 时间戳认证 | 电子数据 | 成本低，须配合其他证据 |
| 工商查处 | 生产/销售窝点 | 强制力强，须掌握地址 |
| 鉴定 | 技术对比/损害评估 | 专业性强，须有资质 |

## 工商查处申请

```
申请流程：
1. 准备举报材料（营业执照/商标证/侵权证据）
2. 向侵权行为地工商部门举报
3. 工商部门立案查处
4. 查封/扣押侵权商品
5. 罚款+赔偿调解

所需材料：
- 权利人身份证明
- 注册商标证书
- 侵权证据（须公证或工商现场取证）
- 索赔金额依据
```
