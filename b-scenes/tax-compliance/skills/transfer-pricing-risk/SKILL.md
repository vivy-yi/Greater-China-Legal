# /transfer-pricing-risk — 转让定价风险评估

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取转让定价规则/同期文档要求。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取转让定价方法/文档要求
2. 识别关联交易类型
3. 评估是否满足独立交易原则
4. 检查同期文档是否齐备
5. 输出风险评估

## 关联交易类型

| 类型 | 风险点 |
|---|---|
| 有形资产购销 | 价格偏离市场价 |
| 劳务提供 | 费用不合理 |
| 资金融通 | 利率偏离市场 |
| 无形资产转让使用 | 许可费不合理 |
| 股权收益分配 | 利润转移 |

## 转让定价方法

| 方法 | 适用场景 |
|---|---|
| 可比非受控价格法 | 有市场价的交易 |
| 再销售价格法 | 买方将货物再销售 |
| 成本加成法 | 制造业/服务业 |
| 利润分割法 | 高价值无形资产 |
| 交易净利润法 | 难以找到可比方 |

## 风险信号

| 信号 | 说明 |
|---|---|
| 关联交易占比高 | 利润向低税地转移 |
| 长期亏损 | 可能有利润转移 |
| 无正当商业理由 | 仅以税务为目的 |
| 利率/费率异常 | 偏离市场水平 |
| 无可比交易 | 难以验证合理性 |

## 同期文档要求

| 企业类型 | 须准备文档 |
|---|---|
| 年度关联交易>2000万元 | 主体文档 |
| 特定关联交易>400万元且利润为负 | 本地文档 |
| 融资关联交易>2亿元 | 资本弱化特殊事项文档 |
