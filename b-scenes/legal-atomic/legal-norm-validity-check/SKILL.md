# 法律规范效力核查

## 定位

**本skill是所有场景的"法条质检门"。**

任何GCL场景skill在输出法条引用前，必须先通过本门。

```
引用法条 → [legal-norm-validity-check] → ✅有效/⚠️部分有效/❌失效 → 场景skill继续处理
```

## 效力状态速查表

| 状态 | 标记 | 含义 |
|---|---|---|
| 现行有效 | ✅ VALID | 已生效且未被修改或废止 |
| 已修正/修订 | 🔄 AMENDED | 原文本已被替代，必须引用最新版本 |
| 部分失效 | ⚠️ PARTIAL | 整体有效，但特定条款已被修改或废止 |
| 已废止 | ❌ REPEALED | 已被明确废止，不再具有法律效力 |
| 已失效 | ❌ EXPIRED | 因期限届满或对象消失而自然失效 |
| 尚未生效 | ⏳ PENDING | 已公布但未到达施行日期 |
| 效力存疑 | ❓ UNCERTAIN | 无法确定，需进一步核实 |

## 三维度核查框架

```
维度1：时间效力    → 是否现行有效？有无修订/废止？
维度2：层级效力    → 制定主体是否有权？层级是否正确？
维度3：冲突检查    → 与上位法/同位法是否矛盾？
```

### 维度1：时间效力检查（优先级最高，阻断性）

```
2.1 检查施行日期
    ├── 尚未到达施行日期 → ⏳ PENDING，终止
    └── 已到达 → 继续

2.2 检查有无有效期限
    ├── 期限已届满 → ❌ EXPIRED，终止
    └── 未届满/无期限 → 继续

2.3 检查是否被修正/修订
    ├── 已修正（个别条款）→ 🔄 AMENDED，确认引用条款是否在修改范围内
    ├── 已修订（全面修改）→ 🔄 AMENDED，必须用新版本
    └── 未修改 → 继续

2.4 检查是否被废止
    ├── 明示废止（新法列清单）→ ❌ REPEALED
    ├── 默示废止（新法与旧法矛盾）→ ⚠️ PARTIAL，需分析
    └── 未废止 → ✅ VALID
```

**高频废止清单（必须立即更新引用）：**

| 已废止 | 废止日期 | 替代 |
|---|---|---|
| 《合同法》 | 2021-01-01 | 《民法典》合同编 |
| 《民法通则》 | 2021-01-01 | 《民法典》总则编 |
| 《物权法》 | 2021-01-01 | 《民法典》物权编 |
| 《担保法》 | 2021-01-01 | 《民法典》合同编 |
| 《婚姻法》《继承法》《收养法》 | 2021-01-01 | 《民法典》婚姻家庭编/继承编 |
| 《侵权责任法》 | 2021-01-01 | 《民法典》侵权责任编 |
| 《公司法》旧版（2018修正） | 2024-07-01 | 《公司法》2023修订 |

### 维度2：层级效力检查

```
层级体系：
第一层：宪法
第二层：法律（全国人大/全国人大常委会）
第三层：行政法规（国务院）
第四层：地方性法规 ←→ 部门规章（同层级）
第五层：地方政府规章
其他：司法解释、规范性文件、指导性案例
```

**立法权限红线：**
- 犯罪与刑罚、限制人身自由、税种设立 → 只能制定法律（绝对保留）
- 部门规章不得超越执行性立法范围，不得增设行政许可

### 维度3：冲突检查

| 冲突类型 | 适用规则 | 依据 |
|---|---|---|
| 下位法 vs 上位法 | 上位法优先 | 《立法法》第99条 |
| 特别法 vs 一般法 | 特别法优先 | 《立法法》第103条 |
| 新法 vs 旧法（同机关） | 新法优先 | 《立法法》第103条 |
| 地方性法规 vs 部门规章 | 国务院提出意见 | 《立法法》第104条 |

## 置信度体系

| 置信度 | 触发条件 |
|---|---|
| 🟢 HIGH | 权威来源（A级）+ 效力明确 + 无矛盾信号 |
| 🟡 MEDIUM | B级来源 / 近期有修正 / 冲突有明确解决规则 |
| 🔴 LOW | 无法确认修正情况 / 冲突无法通过规则解决 / 规范性文件约束力不明 |

**置信度对后续操作的影响：**
- HIGH → 可直接用于法律推理
- MEDIUM → 可用于推理，但应注明保留意见
- LOW → **不得**作为确定性结论依据，必须建议用户进一步核实

## 各场景法条核查要点

### 劳动仲裁场景
```
重点法条：劳动合同法、劳动争议调解仲裁法、最高人民法院劳动争议司法解释
核查重点：
  - 劳动合同法修正版（2012）vs 2008旧版：经济补偿金计算规则有调整
  - 劳动仲裁时效：60日（特殊）vs 民事诉讼时效3年，注意衔接
  - 地方劳动仲裁规则差异（北上广深各有细则）
```

### 合同审查场景
```
重点法条：民法典合同编、最高人民法院合同编司法解释
核查重点：
  - 《合同法》已废止：引用必须更新为《民法典》第577条（原107条）
  - 合同编司法解释（一）（二）：2023年发布，部分条款替代旧司法解释
  - 民间借贷利率：2020年8月20日LPR4倍上限替代旧固定利率
```

### 数据合规场景
```
重点法条：PIPL、数据安全法、个人信息保护合规审计办法
核查重点：
  - PIPL：2021年11月1日施行，境外处理规则需注意
  - CAC执法：2023年后的执法力度和标准变化快，引用时注意时效
  - 网信办指南：属于规范性文件，不是法规，需标注
```

### 公司治理场景
```
重点法条：公司法2023修订（2024年7月1日施行）
核查重点：
  - 2024年7月1日前适用旧版公司法（2018修正）
  - 过渡期规则：存量公司2年调整期
  - 司法解释（一）至（五）：部分与新公司法不一致，需核查
```

### 税务合规场景
```
重点法条：增值税暂行条例、消费税法、所得税法、税务规范性文件管理办法
核查重点：
  - 部分税种仍以"暂行条例"形式存在，具有法律效力
  - 税务总局公告：属于规范性文件，不可作为裁判依据引用
  - 税收协定 vs 国内法：协定优先（但中国声明保留的条款除外）
```

### 跨境并购场景
```
重点法条：外商投资法及实施条例、ODI备案办法、证券法（VIE架构核查）
核查重点：
  - 外商投资法（2020）替代旧三资法：过渡期安排
  - VIE架构：证监会2021年限VIE境外上市规定，注意时效
  - 国家安全审查：外商投资安全审查办法（2021）
```

## 常见错误防范

| 错误 | 后果 | 防范 |
|---|---|---|
| E01：引用已废止法条 | ⭐⭐⭐⭐⭐致命 | 引用前必须通过本门；建立废止清单 |
| E02：引用旧版本 | ⭐⭐⭐⭐⭐致命 | 确认最新修正日期；对比新旧条文 |
| E03：层级错误 | ⭐⭐⭐⭐严重 | 明确标注层级；冲突时严格适用上位法优先 |
| E04：忽视冲突 | ⭐⭐⭐⭐严重 | 对核心依据执行冲突检查 |
| E05：规范性文件误当法规 | ⭐⭐⭐重大 | 区分法规/规章/规范性文件 |
