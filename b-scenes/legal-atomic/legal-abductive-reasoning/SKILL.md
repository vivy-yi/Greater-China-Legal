# /legal-abductive-reasoning — 法律溯因推理

## 核心概念

```
溯因推理：从结果推测原因
"观察到现象 → 提出最合理的解释（假设）"

关键特征：
- 可废止性：假设可被新证据推翻（非单调推理）
- 创造性：需要想象力和创造性思维
- 解释性：关注"为什么"而非"是什么"
```

### 溯因 vs 演绎 vs 归纳

| 类型 | 方向 | 结论确定性 |
|---|---|---|
| 演绎推理 | 一般→特殊 | 确定 |
| 归纳推理 | 特殊→一般 | 或然性 |
| 溯因推理 | 结果→原因 | 可废止性 |

## 米尔五法：假设生成与检验工具

### 求同法（Method of Agreement）— 找必要条件

```
案例1：因素A、B、C → 现象x发生
案例2：因素A、D、E → 现象x发生
─────────────────────────────
∴ A 是 x 的必要条件（无A则无x）
```

### 求异法（Method of Difference）— 找充分条件

```
案例1：因素A、B、C → 现象x发生
案例2：    B、C   → 现象x不发生
─────────────────────────────
∴ A 是 x 的充分条件（有A则有x）
```

### 求同求异并用法（Joint Method）— 必要+充分

同时应用求同法和求异法，强化因果推断。

### 剩余法（Method of Residues）— 解释剩余部分

```
已知：A+B+C → x+y+z
已知：B → y，C → z
────────────────────
∴ A → x（剩余部分由新原因解释）
```

### 共变法（Method of Concomitant Variation）— 变量关系

```
A增强 → x增强
A减弱 → x减弱
────────────────────
∴ A 与 x 存在因果或共同原因关系
```

## 工作流程：五步溯因分析法

### Step 1: 事实梳理

```
目标：清晰列出已知事实，识别信息缺口

操作：
1. 区分已证实事实与待证事实
2. 标注有利事实与不利事实
3. 识别模糊之处和多种可能性

输出：
## 已知事实
- [事实1]：[描述]（证据来源/强度）

## 信息缺口
- [缺口1]：[缺失的关键信息]

## 模糊之处
- [模糊点1]：[可多种解读的事实]
```

### Step 2: 假设生成

```
目标：基于米尔五法，生成多个竞争性解释

强制要求：至少生成3个假设，涵盖不同角度

假设类型：
- 事实假设：事件经过的不同版本
- 意图假设：行为动机和目的的不同解释
- 法律假设：构成要件是否满足的不同判断

输出：
## 竞争性假设

### 假设1：[简要描述]
- 基于方法：[求同/求异/并用/剩余/共变]
- 核心主张：[一句话概括]
- 支持证据：[列出支持的事实]
- 潜在弱点：[列出不利的事实]
- 可废止条件：[什么证据会推翻此假设]
```

### Step 3: 假设评估（PEIRCE标准）

| 标准 | 含义 | 评估问题 |
|---|---|---|
| 解释力 | 覆盖度 | 能否解释所有已知事实？ |
| 简洁性 | 奥卡姆剃刀 | 是否引入不必要实体？ |
| 可检验性 | 实证可能 | 能否被验证或证伪？ |
| 背景契合 | 先例一致性 | 与类似案例是否一致？ |

```
评估矩阵：
| 假设 | 解释力 | 简洁性 | 可检验性 | 背景契合 | 综合 |
|------|--------|--------|----------|----------|------|
| H1   | 高/中/低 | 高/中/低 | 高/中/低 | 高/中/低 | ⭐⭐⭐⭐⭐ |
| H2   | ...    | ...    | ...      | ...      | ...    |
```

### Step 4: 最佳解释推断

```
决策：
- 首选假设：综合评分最高
- 备选假设：评分相近的保留选项
- 关键不确定性：影响判断的信息缺口
```

### Step 5: 验证与修正

```
操作：
1. 列出可进一步收集的证据
2. 预测新证据对各假设的影响
3. 制定假设修正的触发条件
```
