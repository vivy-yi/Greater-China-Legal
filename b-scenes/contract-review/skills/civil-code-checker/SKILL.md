# /civil-code-checker — 民法典合同编合规检查

## 法定权利义务checklist

### 买卖合同（民法典第595-647条）

**卖方义务：**
- 交付标的物（数量/质量/方式）
- 转移标的物所有权
- 权利瑕疵担保（第三方不能主张权利）
- 物的瑕疵担保（质量符合约定/国家标准）

**买方义务：**
- 支付价款
- 验收标的物（数量/外观）
- 通知义务（收到标的物发现数量/质量不符合约定应及时通知）

**风险提示：**
- 风险转移规则（交付转移，动产vs不动产有差异）
- 检验期限（约定<合理期限，未约定须在合理期间内通知）

---

### 租赁合同（民法典第703-734条）

**出租人义务：**
- 交付符合约定用途的租赁物
- 维修义务（当事人约定除外）
- 权利瑕疵担保
- 附属设施/物的瑕疵维修

**承租人义务：**
- 支付租金
- 正常使用（不擅自转租/改动）
- 返还租赁物

**风险提示：**
- 最长租赁期限20年（超期部分无效）
- "买卖不破租赁"（租赁物转让不影响租赁合同效力）
- 优先购买权（出租人卖房须提前通知承租人）

---

### 承揽合同（民法典第770-787条）

**定作人权利：**
- 任意解除权（解除合同但须赔偿承揽人实际损失）

**承揽人义务：**
- 按约定工作（材料/技术要求）
- 通知义务（发现定作人提供的图纸/技术要求有问题须及时通知）
- 交付工作成果
- 缺陷担保（工作成果须符合约定）

**风险提示：**
- 定作人任意解除权的赔偿范围（已工作部分+可得利益损失）

---

### 委托合同（民法典第919-936条）

**委托人义务：**
- 支付报酬
- 赔偿受托人损失

**受托人义务：**
- 按指示处理委托事务
- 报告义务（报告委托事务处理情况）
- 转移处理成果/所得利益

**风险提示：**
- 任意解除权（随时可解除，但须赔偿损失）
- 损失的赔偿范围：直接损失+可预期利益损失

---

### 借款合同（民法典第667-680条）

**利率限制：**
```
民间借贷：不超过合同成立时4倍LPR（2024年约14.2%）
金融借贷：浮动利率，原则上不受4倍LPR限制
```

**高风险条款：**
- 砍头息（预先扣除利息，本金按实际到账计算）
- 复利（利滚利，须在借款合同中明确约定）

## 效力性强制性规定识别

### 常见无效条款

| 条款类型 | 风险 | 法条依据 |
|---|---|---|
| 违反工程建设资质要求 | 建设工程合同无效 | 建筑法第26条 |
| 违反特许经营限制 | 合同无效 | 行政许可法第53条 |
| 违反金融准入规定 | 借款合同部分无效 | 银行业监管规定 |
| 恶意串通损害他人 | 合同无效 | 民法典第154条 |
| 违反公序良俗 | 合同无效 | 民法典第153条 |
| 无权处分的买卖合同 | 效力待定 | 民法典第597条 |

### 提示和说明义务（格式条款）

**格式条款提供方的义务：**
- 合理方式提示对方注意免除/限制其责任的条款
- 说明格式条款的含义和后果

**无效的格式条款：**
- 造成对方人身损害的免责条款
- 故意/重大过失造成对方财产损失的免责条款
- 排除对方主要权利的条款

## 合同效力checklist

| 检查项 | 结果 | 风险 |
|---|---|---|
| 主体资格 | ✅/❌ | 无营业执照/资质不够 |
| 意思表示真实 | ✅/❌ | 欺诈/胁迫/重大误解 |
| 不违反强制性规定 | ✅/❌ | 违反则可能无效 |
| 不违反公序良俗 | ✅/❌ | 违反则无效 |
| 标的可能/合法 | ✅/❌ | 违法标的合同无效 |
| 形式符合要求 | ✅/❌ | 应采用书面形式的合同 |
