# /dispute-handler — 合同履约争议处理路径

## 四种争议解决路径

| 路径 | 成本 | 时间 | 效力 | 适用情形 |
|---|---|---|---|---|
| 协商和解 | 低 | 快（1-2周） | 无强制执行力 | 双方关系尚可，争议金额不大 |
| 人民调解 | 低 | 快（2-4周） | 司法确认后有执行力 | 须双方自愿，有调解组织参与 |
| 仲裁 | 中高 | 快（1-4个月） | 一裁终局 | 有仲裁条款，合同金额较大 |
| 诉讼 | 高 | 长（6-24个月） | 两审终审 | 无法仲裁/金额极大/须上诉 |

---

## 路径选择判断树

### 第一步：是否有仲裁条款

**有仲裁条款：**
```
→ 原则上必须仲裁
→ 选择哪个仲裁委（CIETAC/SIAC/HKIAC/各地方仲裁委）
→ 仲裁比诉讼快，但费用较高
```

**无仲裁条款：**
```
→ 可直接诉讼
→ 选择管辖法院（被告住所地/合同履行地）
```

---

### 第二步：争议金额判断

| 金额区间 | 建议路径 | 备注 |
|---|---|---|
| <5万 | 协商优先 | 诉讼成本不划算 |
| 5-50万 | 调解/仲裁 | 仲裁效率高 |
| 50-500万 | 仲裁/诉讼 | 视证据充分程度 |
| >500万 | 诉讼（一审+二审） | 金额大，审级保障 |

---

### 第三步：时效检查

| 争议类型 | 诉讼时效 | 备注 |
|---|---|---|
| 合同违约（要求继续履行） | 3年 | 履行请求权 |
| 合同违约（要求解除+赔偿） | 3年 | 损害赔偿请求权 |
| 合同无效（要求返还原物） | 3年 | 返还原物请求权 |
| 工程质量纠纷 | 3年 | 实践中多认定为违约 |
| 租赁合同纠纷 | 1年（身体/财产损害3年） | 租赁合同适用特殊时效 |

**⚠️ 超过诉讼时效：丧失胜诉权（对方不提出则法院不主动审查）**

## 常见履约争议处理

### 拖欠货款/服务费

**第一步：发送书面催告**
```
要求对方在[X]日内支付欠款，否则将采取法律措施。
```

**第二步：协商调解（1-2周）**
- 如对方有诚意，可分期付款+追加违约金
- 签署还款协议

**第三步：申请支付令（快速通道）**
```
条件：债权人持有明确的债权凭证（合同+发票+催告函）
优点：15日内发出，费用低（100元）
缺点：对方提出异议则支付令失效
```

**第四步：起诉（无法调解时）**
- 准备证据：合同/发票/送货单/聊天记录/催告函
- 起诉金额含：本金+违约金+律师费（如合同约定）

---

### 质量问题争议

**第一步：固定证据**
- 拍照/录像
- 第三方检测报告
- 专家论证意见

**第二步：判断是否超过检验期限**
- 约定了检验期：在检验期内通知
- 未约定检验期：收到标的物后及时检验（合理期限内）
- 隐蔽瑕疵：在合理期限内/两年内通知

**第三步：处理路径**
- 轻微瑕疵 → 要求减价/修复
- 严重违约 → 要求解除合同+赔偿损失
- 质量问题争议 → 法院委托鉴定（耗时2-4个月）

---

### 工期延误争议

**第一步：判断责任归属**

| 延误原因 | 责任方 | 免责抗辩 |
|---|---|---|
| 承包方原因 | 承包方 | — |
| 发包方原因（设计变更/付款延迟） | 发包方 | 有权顺延工期 |
| 不可抗力 | 各自承担 | 须及时通知+证明 |

**第二步：索赔计算**
- 工期延误损失：实际发生的窝工费/设备租赁费
- 须有证据证明（付款记录/发票/照片）
