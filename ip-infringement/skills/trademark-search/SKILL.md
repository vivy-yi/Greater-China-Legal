# /trademark-search — 商标查询与近似判断

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取商标法规/判断标准。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取尼斯分类/判断标准
2. 确定商品类别（尼斯分类）
3. 查询注册状态（CNIPA官网）
4. 评估是否近似（音形义三维度）
5. 输出混淆可能性评估

## 尼斯分类（商品类别）

| 类别 | 内容 |
|---|---|
| 第9类 | 科学仪器/电子产品/软件 |
| 第35类 | 广告/商业经营/网站 |
| 第38类 | 通讯服务/互联网服务 |
| 第41类 | 教育/娱乐/文体服务 |
| 第42类 | 技术服务/软件开发 |

**注：互联网产品通常涉及第9/35/38/41/42类**

## 查询路径

```
CNIPA官网：https://www.cnipa.gov.cn/
  商标查询：http://sbj.cpaa.org.cn/
  检索：商标名称 + 类别 + 申请人

[GOV] 中国商标网：https://www.ct.cn/
```

## 近似判断三维度

| 维度 | 判断方法 |
|---|---|
| 音 | 读音相同或近似 |
| 形 | 字形相似/整体外观相似 |
| 义 | 含义相同或容易产生联想 |

## 混淆可能性评估

| 评估因素 | 说明 |
|---|---|
| 商标显著性和知名度 | 越知名，保护范围越大 |
| 商品关联性 | 商品类似度越高，越容易混淆 |
| 销售渠道 | 是否在同一渠道销售 |
| 消费者注意程度 | 专业买家/普通消费者 |
