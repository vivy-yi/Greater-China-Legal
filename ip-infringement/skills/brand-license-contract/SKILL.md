# 商标许可使用合同审查

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取场景级配置（法域足迹/数据源/置信度规则）。

---

## 工作流程

```
1. 读取 ../CLAUDE.md → 获取IP类型/赔偿计算规则/维权路径
2. 识别合同基本信息 → 许可类型/品牌/期限/费用结构
3. 许可类型判断 → 独占许可/排他许可/普通许可
4. 核心条款审查 → 许可范围/质量控制/费用/终止后续处理
5. 风险评估 → 低/中/高风险标识
6. 条款完善建议 → 问题条款+修改建议
7. 升级决策 → 是否移交外部律师
```

### 三种许可类型对比表

| 类型 | 法律效果 | 许可方权利 | 被许可方独占性 | 适用场景 |
|------|----------|------------|----------------|----------|
| 独占许可 | 最高保护 | 完全让渡 | ✅ 完全独占 | 品牌深度合作/子公司 |
| 排他许可 | 中等保护 | 保留使用权 | ⚠️ 排除第三方 | 战略合作伙伴 |
| 普通许可 | 基础保护 | 保留所有权利 | ❌ 不独占 | 广泛授权/渠道分销 |

### 许可费计算量化表

| 许可费类型 | 计算公式 | 行业参考区间 | 风险点 |
|------------|----------|--------------|--------|
| 固定费用 | 年费=基础金额×年限 | 10-50万元/年 | 市场变化风险 |
| 销售额提成 | 许可费=销售额×提成比例 | 1%-15% | 核查困难 |
| 使用量计费 | 许可费=使用量×单件费率 | 0.5-5元/件 | 计量争议 |
| 最低保证金 | 保底销售额×提成比例 | 确保收益下限 | 超额结算复杂 |

## 二、核心条款审查判断树

### 2.1 许可范围审查

```
                    ┌─────────────────────────────┐
                    │     许可范围是否明确？        │
                    └─────────────┬───────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐
        │                         │                         │
        ▼                         ▼                         ▼
     ┌──────┐               ┌──────────┐              ┌──────────┐
     │ 模糊 │               │  较清晰  │              │  非常    │
     │ ⚠️   │               │  ⚠️     │              │ 清晰 ✅  │
     └──────┘               └──────────┘              └──────────┘
```

**许可范围审查清单：**

| 审查项 | 必备内容 | 常见问题 | 风险等级 |
|--------|----------|----------|----------|
| 商品/服务类别 | 尼斯分类具体类别 | 仅写"服装"未写具体类别 | ⚠️ 中 |
| 使用方式 | 生产/销售/广告/包装 | 使用方式约定不明 | ⚠️ 中 |
| 地域范围 | 国家/省份/城市级别 | 地域范围过广或过窄 | 🔴 高 |
| 期限 | 起止日期+续展条件 | 自动续展条款不清 | ⚠️ 中 |
| 子许可权 | 是否允许再授权 | 未约定则默认不允许 | 🔴 高 |

### 2.2 质量控制条款审查

```
                    ┌─────────────────────────────┐
                    │     质量控制条款是否完善？     │
                    └─────────────┬───────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐
        │                         │                         │
        ▼                         ▼                         ▼
     ┌──────┐               ┌──────────┐              ┌──────────┐
     │ 缺失 │               │  部分    │              │  完整    │
     │ 🔴   │               │  具备 ⚠️ │              │ ✅      │
     └──────┘               └──────────┘              └──────────┘
```

**质量控制核心要求（商标法第43条）：**

| 要素 | 法律要求 | 操作建议 |
|------|----------|----------|
| 质量标准 | 许可人应当监督被许可人使用其注册商标的商品质量 | 明确质量标准+检验频率 |
| 合格证书 | 被许可产品须符合的质量认证 | 要求提供CCC/ISO等认证 |
| 检验权 | 许可方有权定期检验 | 约定检验方式/费用 |
| 整改义务 | 不合格时的整改期限和措施 | 明确整改失败的处理 |
| 监督费用 | 检验费用由谁承担 | 建议约定由被许可方承担 |

### 2.3 许可费支付审查

```
                    ┌─────────────────────────────┐
                    │     许可费支付条款是否完善？   │
                    └─────────────┬───────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐
        │                         │                         │
        ▼                         ▼                         ▼
     ┌──────┐               ┌──────────┐              ┌──────────┐
     │ 缺失 │               │  基本    │              │  完整    │
     │ 🔴   │               │  具备 ⚠️ │              │ ✅      │
     └──────┘               └──────────┘              └──────────┘
```

**许可费支付量化审查表：**

| 审查项 | 标准要求 | 常见缺陷 | 修改建议 |
|--------|----------|----------|----------|
| 付款期限 | 约定具体日期或期间 | "收到发票后付款"过模糊 | 明确"收到发票后30日内" |
| 付款方式 | 银行转账/支票/电子支付 | 未约定账户信息 | 明确账户+账号+开户行 |
| 最低保证金 | 约定年度最低销售额 | 无最低保证可能损失 | 约定最低保证金及补足机制 |
| 核查权 | 许可方有权核查账簿 | 无核查权则无法核实 | 约定审计权+提供报表义务 |
| 滞纳金 | 逾期付款的违约金 | 滞纳金约定不明或过高 | 按日万分之五或LPR4倍 |
| 税务 | 税费承担约定 | 税务责任不清 | 明确税费承担方 |

```
                    ┌─────────────────────────────┐
                    │     许可终止后续处理是否约定？ │
                    └─────────────┬───────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐

     ┌──────┐               ┌──────────┐              ┌──────────┐

     │ 🔴   │               └──────────┘              └──────────┘
     └──────┘
```

**许可终止后续处理审查表：**

| 情形 | 必须约定内容 | 常见问题 | 建议处理 |
|------|--------------|----------|----------|
| 合同到期终止 | 库存商品处理期限/方式 | 无处理期限争议 | 约定30-60日处理期 |
| 提前解除 | 通知期限/违约责任 | 通知期限过短 | 提前3-6个月通知 |
| 主体资格消灭 | 商标归属/变更手续 | 无约定导致纠纷 | 明确注销后处理 |
| 续展优先权 | 被许可方优先续展权 | 未约定优先权丧失 | 约定优先续展权 |
| 竞品限制 | 许可期间/后竞品限制 | 竞品限制过严/不明 | 明确范围+期限+地域 |
| 保密义务 | 商业秘密/客户信息 | 终止后无保密义务 | 约定2-5年保密期 |

## 三、风险分级量化标准

### 3.1 风险等级判断矩阵

| 风险因素 | 低风险 ✅ | 中风险 ⚠️ | 高风险 🔴 |
|----------|----------|-----------|----------|
| 许可类型 | 普通许可 | 排他许可 | 独占许可 |
| 许可费结构 | 固定费用 | 混合模式 | 纯浮动/核查困难 |
| 地域范围 | 单一省份 | 2-5省份 | 全国/跨境 |
| 许可期限 | ≤2年 | 2-5年 | >5年/自动续展 |
| 合同金额 | <50万 | 50-500万 | >500万 |
| 行业敏感度 | 普通商品 | 食品/服装 | 药品/医疗器械 |

### 3.2 综合风险评分计算

```
风险评分 = 许可类型权重×分值 + 许可费权重×分值 + 地域权重×分值 + 期限权重×分值 + 金额权重×分值

评分标准：
- 许可类型：普通=1, 排他=2, 独占=3
- 许可费：固定=1, 混合=2, 浮动=3
- 地域：单省=1, 多省=2, 全国/跨境=3
- 期限：≤2年=1, 2-5年=2, >5年=3
- 金额：<50万=1, 50-500万=2, >500万=3

权重分配：类型30% + 费25% + 地域20% + 期限15% + 金额10%

风险等级：
- 1.0-1.5分 → LOW（低风险）→ 可快速审核
- 1.5-2.0分 → MEDIUM（中风险）→ 建议完善条款
- 2.0-3.0分 → HIGH（高风险）→ 必须外部律师审核
```

### 3.3 高风险条款识别

**以下条款触发高风险预警（必须修改）：**

| 序号 | 高风险条款 | 风险描述 | 建议修改 |
|------|-----------|----------|----------|
| 1 | "许可方保留一切权利" | 与独占许可冲突 | 明确限定保留范围 |
| 2 | "许可费按实际发生额结算" | 无最低保证 | 增加最低保证金条款 |
| 3 | "被许可方可自由转授权" | 未限制子许可 | 须许可方书面同意 |
| 4 | "许可期限自动续展" | 永久授权风险 | 约定最长续展期限 |
| 5 | "质量控制由被许可方自行管理" | 违反商标法要求 | 许可方保留监督权 |
| 6 | "争议解决仅适用境外仲裁" | 跨境执行困难 | 约定中国法院管辖 |
| 7 | "被许可方承担一切侵权责任" | 显失公平 | 区分责任归属 |
| 8 | "许可方随时终止权" | 无正当理由终止 | 约定终止的正当理由 |

## 四、合同审查具体步骤

### 步骤一：基本信息提取（5分钟）

```
□ 许可方信息：名称/注册地/商标注册证号
□ 被许可方信息：名称/注册地/主营业务
□ 许可商标：商标名称/注册号/类别
□ 许可类型：独占/排他/普通
□ 许可期限：起止日期/续展条件
□ 许可地域：国家/省份/城市
□ 许可费：金额/计算方式/支付方式
□ 合同金额估算
□ 争议解决条款
□ 适用法律
```

### 步骤二：许可范围审查（10分钟）

**审查要点：**
- 商品/服务类别是否与注册商标一致
- 使用方式是否在授权范围内
- 地域范围是否有边界限制
- 是否有子许可授权条款

**量化判断：**
- 尼斯分类类别匹配度：完全匹配=✅ / 部分匹配=⚠️ / 不匹配=🔴
- 地域范围与实际业务匹配度

### 步骤三：质量控制审查（10分钟）

**审查要点：**
- 是否有明确的质量标准
- 许可方是否有检验权
- 不合格产品的处理方式
- 质量监督费用承担

**量化判断：**
- 质量标准明确度：具体可执行=✅ / 模糊需解释=⚠️ / 无标准=🔴
- 检验权可执行性：可执行=✅ / 受限制=⚠️ / 无权=🔴

### 步骤四：许可费审查（10分钟）

**审查要点：**
- 费用计算是否清晰
- 支付时间和方式是否明确
- 是否有最低保证金
- 是否有核查权条款

**量化判断：**
- 费用结构清晰度：完全明确=✅ / 基本明确=⚠️ / 不明确=🔴
- 核查权可执行性：可执行=✅ / 受限=⚠️ / 无权=🔴

### 步骤五：终止条款审查（10分钟）

**审查要点：**
- 终止条件和程序是否明确
- 终止后续处理是否有约定
- 库存商品处理期限
- 竞业限制条款

**量化判断：**
- 终止条款完整性：完整=✅ / 部分=⚠️ / 缺失=🔴
- 后续处理可行性：可行=✅ / 需补充=⚠️ / 无约定=🔴

### 步骤六：风险评估汇总（5分钟）

**输出风险矩阵：**

| 审查维度 | 风险等级 | 主要问题 | 修改建议优先级 |
|----------|----------|----------|----------------|
| 许可范围 | [等级] | [问题描述] | [高/中/低] |
| 质量控制 | [等级] | [问题描述] | [高/中/低] |
| 许可费 | [等级] | [问题描述] | [高/中/低] |
| 终止条款 | [等级] | [问题描述] | [高/中/低] |
| 争议解决 | [等级] | [问题描述] | [高/中/低] |
| **综合** | **[综合等级]** | — | — |

## 六、数据源与法律依据

### 核心法规

| 法规名称 | 生效日期 | 关键条款 | 用途 |
|----------|----------|----------|------|
| 民法典 | 2021-01-01 | 第863-887条（知识产权合同） | 合同效力/违约责任 |
| 商标法 | 2019-11-01 | 第43条（许可使用） | 许可有效性/备案要求 |
| 商标法实施条例 | 2002-09-15 | 第71-73条 | 许可备案程序 |
| 商标民事纠纷司法解释 | 2002-10-16 | 第17-24条 | 许可费/侵权责任 |

### 关键法律要点

**商标许可使用法定要求：**
1. 许可方应当监督被许可人使用其注册商标的商品质量
2. 被许可人应当保证使用该注册商标的商品质量
3. 许可使用注册商标必须签订许可合同
4. 许可人应当自商标许可使用合同签订之日起30日内报商标局备案

**许可费法定标准：**
- 实际损失 → 侵权获利 → 参照许可费 → 法定赔偿（500万元以下）
- 许可费可作为侵权赔偿的参照标准

## 升级决策门

### 必须移交专业律师的情形

| 情形 | 条件 | 移交原因 |
|------|------|----------|
| 许可类型 | 独占许可 | 权利范围最广，纠纷风险最高 |
| 合同金额 | >500万元 | 金额重大，需专业把关 |
| 行业特殊 | 食品/药品/医疗器械 | 监管要求严格 |
| 跨境因素 | 涉及境外主体或境外使用 | 法律适用复杂 |
| 期限超长 | >5年或自动续展 | 长期约束需谨慎 |
| 主体特殊 | 上市公司/国有企业 | 合规要求更高 |

### 快速升级判断

```
以下任一条件满足 → 必须移交外部律师：
□ 许可类型 = 独占许可 且 合同金额 > 100万元
□ 许可类型 = 独占许可 且 行业 = 特殊监管行业
□ 合同金额 > 500万元
□ 涉及跨境使用或多个法域
□ 许可期限 > 5年（含自动续展条款）
□ 被许可方为上市公司或拟上市公司
□ 涉及国有资产或国有参股企业
□ 涉及药品/医疗器械/食品许可
```
