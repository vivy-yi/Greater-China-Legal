# /clearance — China Mainland

## Purpose

> **本技能仅提供初步检索，不是注册意见。** 商标注册意见须由专业商标检索律师完成。"未发现明显冲突"仅表示初步检索未发现问题，不表示商标可注册。已有客户因通过初步检索但未做专业检索而卷入商标纠纷。

---

## CN商标检索特殊说明

**中国商标申请体系：**
- 主管机关：国家知识产权局（CNIPA）— 中国商标网 (wssx.cna.gov.cn)
- 申请基础：注册在先 + 申请在先原则
- 商品分类：尼斯分类（45类）
- 审查周期：约6-9个月（含驳回-复审）

---

## Intake（询问）

在单次批量中完成询问：

> 1. **提案商标** — 文字、图形、或文字+图形组合
> 2. **商品/服务** — 实际销售的产品或服务（一句话描述）
> 3. **类别** — 尼斯分类类别（不确定则描述，我方建议）
> 4. **管辖区域** — 中国大陆/香港/澳门/台湾/新加坡/其他
> 5. **实际使用方式** — 包装/广告/互联网/其他

## 绝对理由审查（中国《商标法》第10-12条）

在数据库检索前，先审查绝对理由（无需检索即知）：

| 驳回理由 | 含义 | 标志情形 |
|---|---|---|
| **通用名称** | 标志仅为本类商品通用名称 | 标志=商品名称 |
| **描述性** | 直接描述商品功能/品质/原料 | 消费者无需想象即可理解含义 |
| **欺骗性** | 欺骗性描述商品质量/产地/功能 | 标志误导消费者且影响购买决策 |
| **违反公序良俗** | 有害于社会主义道德风尚 | 标志有不良影响 |
| **地理标志** | 误导公众对产地的认知 | 标志为地理标志但商品非来自该地 |
| **仅为姓氏** | 标志仅为姓氏（须证明与自然人有对应关系）| 标志=姓氏且非独创性组合 |
| **政府标志** | 国徽/国旗/国际组织标志/虚假原产地标记 | 标志含官方标志 |

**输出：** 每类绝对理由标注"无问题"或具体标志 + 理由。

## 近似商标检索

**中国商标数据库检索：**
- [WKL] 中国商标网（wssx.cna.gov.cn）— 近似文字检索
- [WKL] 商标评审委员会裁定（site.uspto.gov.cn/w无效宣告/驳回复审）
- [YD] 法律精灵—商标侵权案例

**检索策略：**
1. 文字近似：字形/读音/含义任一相似
2. 图形近似：构图/颜色/整体视觉相似
3. 商品类似：尼斯分类同一类别或类似商品

## CN混淆可能性因素

参照《最高人民法院关于审理商标民事纠纷案件适用法律若干问题的解释》：

| 因素 | 说明 |
|---|---|
| 商标显著性和知名度 | 独创性越高/知名度越大，保护范围越宽 |
| 商标近似程度 | 字形/读音/含义/视觉相似度 |
| 商品/服务类似程度 | 同类别或关联类别 |
| 相关公众认知程度 | 目标消费群体的认知水平 |
| 实际混淆证据 | 是否有实际混淆的证据 |
| 使用意图 | 是否有搭便车的故意 |

## 近似商标发现时的处理

对于每个发现的近似商标，记录：
- **商标**（文字/图形）
- **注册号**（如有）
- **类别/商品**
- **注册人**
- **状态**（已注册/申请中/异议中/无效）
- **冲突评估**

---

## 邻近商标族补充（必做）

> 仅检索精确匹配和近似匹配会遗漏竞争对手因你方商标已被注册而采用的替代商标。

**中文商标邻近族：**
- 同音字替代（如：`华为`→`华威`/`华微`）
- 拼音替代（如：`Huawei`→`Huawi`）
- 含义相近词（如：`云`→`云端`/`云驰`）
- 数字+汉字组合

**须确认：** 邻近族列表是否完整。
