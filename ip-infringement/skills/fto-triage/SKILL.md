# /fto-triage — China Mainland

## Purpose

> **本技能不是自由实施意见。** 正式FTO意见须由注册专利代理师完成全面检索、权利要求解读和逐项侵权分析。专利侵权为严格责任；明知专利存在仍继续实施可构成故意侵权，增加赔偿倍数（中国《专利法》第71条）。

## Intake（询问）

> 1. **产品/工艺/功能** — 具体描述（技术方案）
> 2. **技术细节** — 实现方式、技术特征
> 3. **目标市场** — 中国大陆/香港/台湾/其他
> 4. **已知专利** — 是否有已知的相关专利？
> 5. **上市时间计划**

---

## CN专利侵权判断框架

**法律依据：**《专利法》第11条、第64条

| 判断步骤 | 内容 |
|---|---|
| 1. 确定权利要求保护范围 | 独立权利要求1的技术特征 |
| 2. 识别被诉产品技术特征 | 被诉产品是否包含相同/等同技术特征 |
| 3. 全面覆盖原则 | 是否覆盖所有必要技术特征 |
| 4. 等同原则 | 是否有基本相同的方式实现基本相同的功能达到基本相同的效果 |

## 初步专利检索策略

1. **关键词检索** — 产品名称+技术领域
2. **分类号检索** — IPC国际专利分类
3. **申请人/发明人检索** — 主要竞争对手
4. **同族专利检索** — 失效专利/在审专利

## CN特殊风险提示

**故意侵权风险（中国）：**
- 明知专利存在仍实施，可能增加赔偿倍数
- 收到警告函后继续实施 = 故意侵权
- 建议在产品上市前完成FTO分析

**实用新型/外观设计稳定性：**
- 实用新型无实质审查，稳定性较低（可能被无效宣告）
- 外观设计与发明专利同样保护
- 须关注竞争对手的无效宣告程序
