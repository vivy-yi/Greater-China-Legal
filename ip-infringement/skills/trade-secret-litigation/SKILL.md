# 商业秘密侵权诉讼

## 目的

分析商业秘密侵权诉讼的诉因、举证责任和赔偿计算，辅助制定诉讼或刑事报案策略。覆盖：商业秘密认定、侵权行为类型、举证责任分配、损害赔偿计算、刑事报案路径。

本技能为分析辅助，不代理诉讼或刑事案件。

## 法域假设

默认中国大陆法域，适用《反不正当竞争法》第9条、《刑法》第219条和最高院商业秘密司法解释 `[SME 核查]`。

## ⚠️ 重要提示

商业秘密案件举证难度大，权利人须证明：1. 商业秘密存在；2. 对方实施了侵权行为；3. 损失或侵权获利。建议尽早委托知识产权律师。

## 加载信息

- 商业秘密内容（技术信息或经营信息）
- 泄露方式（员工带走/竞争对手获取/合作方泄露）
- 当事人角色（权利人或嫌疑人）
- 已有证据

## 工作流程

### 第一步：商业秘密认定 `[SME 核查]`

商业秘密须同时满足：

| 要件 | 说明 | 是否满足 |
|------|------|---------|
| 秘密性 | 不为公众所知悉；不能从公开渠道轻易获取 | 是/否/存疑 |
| 价值性 | 具有商业价值；能给权利人带来竞争优势 | 是/否/存疑 |
| 保密措施 | 采取了合理保密措施 | 是/否/存疑 |

**常见商业秘密类型**：
- 技术信息：配方、工艺、设计图纸、源代码、算法
- 经营信息：客户名单、定价策略、供应商渠道、营销计划

### 第二步：侵权行为类型 `[SME 核查]`

| 行为类型 | 说明 | 举证重点 |
|---------|------|---------|
| 盗窃 | 以盗窃、欺诈、胁迫等不正当手段获取 | 手段不正当性 |
| 欺诈 | 以欺诈、贿赂、虚假陈述等方式获取 | 手段不正当性 |
| 违反保密义务 | 违反保密义务披露、使用商业秘密 | 保密协议+泄露行为 |
| 利诱 | 教唆、引诱他人违反保密义务 | 第三方行为证据 |
| 二次侵权 | 明知商业秘密来自上述途径仍获取、使用 | 明知+来源 |

### 第三步：举证责任分配 `[SME 核查]`

权利人举证责任（初步）：
1. 商业秘密的具体内容
2. 采取了保密措施
3. 对方有接触商业秘密的条件

对方举证责任（抗辩）：
- 商业秘密为公众所知悉
- 对方独立研发/合法获取

### 第四步：损害赔偿计算 `[SME 核查]`

| 计算方式 | 适用场景 |
|---------|---------|
| 实际损失 | 权利人因侵权受到的损失 |
| 侵权获利 | 侵权人因侵权所获利益 |
| 许可费 | 参考正常许可费的倍数（1-5倍）|

最低赔偿：500万元（2019年修法后新增惩罚性赔偿门槛，对恶意侵权可判赔5倍）

### 第五步：刑事报案路径 `[SME 核查]`

侵犯商业秘密罪（刑法219条）：

| 情节 | 量刑 |
|------|------|
| 给商业秘密权利人造成重大损失 | 三年以下有期徒刑或拘役，并处或单处罚金 |
| 造成特别严重后果 | 三到七年有期徒刑，并处罚金 |

**立案标准**（公安理解）：

- 直接经济损失≥30万元
- 或侵权获利≥50万元
- 或商业秘密丧失非公知性

### 第六步：升级决策门

> "商业秘密案件的举证难度较大，建议尽早委托知识产权律师。涉及技术秘密的案件通常需要申请技术鉴定。刑事报案须达到立案标准，且需要完整证据链支撑。这不构成法律意见。"
