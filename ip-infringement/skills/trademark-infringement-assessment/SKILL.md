# 商标侵权认定评估

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取场景级配置（法域足迹/数据源/置信度规则）。

---

## 工作流程

```
1. 读取 ../CLAUDE.md → 获取IP类型/赔偿计算规则/维权路径
2. 提取权利基础信息 → 权利商标信息/注册类别/权利稳定性
3. 分析涉嫌侵权行为 → 使用方式/商品类别/是否突出使用
4. 进入侵权判断树 → 依次判断：相同相同→类似混淆→正当使用抗辩
5. 平行进口专项（如涉及）
6. 输出侵权认定结论 + 赔偿估算 + 维权路径建议
7. 升级决策 → 是否移交专业律师
```

## 一、权利基础审查

### 1.1 商标权有效性审查

**审查步骤：**

```
第一步：核实注册证
├─ 商标注册号：[号码]
├─ 注册人：[名称]
├─ 核定使用商品/服务类别：[尼斯分类类别]
├─ 专用权期限：[起止日期]
└─ 有效期状态：[有效/有效期届满/续展中/被撤销]

第二步：核实权利稳定性
├─ 是否处于异议/无效宣告程序 → ⚠️ 权利不稳定
├─ 是否被撤销三年未使用 → ⚠️ 可能被撤销
├─ 是否涉及转让/许可备案 → ⚠️ 需核实备案状态
└─ 是否为防御商标/联合商标 → ✅ 扩大保护范围
```

**权利稳定性量化表：**

| 状态 | 风险等级 | 说明 |
|------|----------|------|
| 有效注册+稳定状态 | ✅ 低风险 | 可直接主张权利 |
| 有效注册+存在争议程序 | ⚠️ 中风险 | 需等待程序终结 |
| 有效期届满未续展 | 🔴 高风险 | 权利已消灭 |
| 被撤销三年未使用 | 🔴 高风险 | 权利可能被撤销 |
| 异议/无效宣告进行中 | ⚠️ 中风险 | 视程序进展判断 |

## 二、侵权判断树

### 2.1 总体判断框架

```
                    ┌─────────────────────────────┐
                    │     商标侵权判断起点         │
                    └─────────────┬───────────────┘
                                  │
                    ┌─────────────▼───────────────┐
                    │  是否为商标法保护的对象？     │
                    │  （文字/图形/组合/三维/声音） │
                    └─────────────┬───────────────┘
                                  │
              ┌───────────────────┴───────────────────┐
              │ 是                                         │ 否
              ▼                                           ▼
    ┌─────────────────┐                        ┌─────────────────┐
    │ 进入侵权判断流程 │                        │ 不构成商标侵权   │
    └────────┬────────┘                        │ （但可能构成其他）│
             │                                  └─────────────────┘
             ▼
    ┌─────────────────────────────────────────────────────────┐
    │                   侵权判断三步法                          │
    ├─────────────────────────────────────────────────────────┤
    │  第一步：相同商品 + 相同商标 → 假冒注册商标（直接侵权）    │
    │  第二步：类似商品 + 相同商标 → 容易导致混淆 → 侵权         │
    │  第三步：正当使用抗辩审查（描述性使用/指示性使用/平行进口）│
    └─────────────────────────────────────────────────────────┘
```

### 2.2 第一步：相同商品 + 相同商标判断

**判断标准（商标法第57条第1项）：**

```
┌─────────────────────────────────────────────────────────────┐
│  同一种商品                                                │
│  ↓                                                         │
│  使用与注册商标相同的商标                                   │
│  ↓                                                         │
│  → 直接认定侵权（无需考虑混淆可能性）                       │
│  → 构成假冒注册商标行为                                    │
└─────────────────────────────────────────────────────────────┘
```

**"相同"的认定标准（2002年商标民事纠纷司法解释第9条）：**

| 认定要素 | 相同 ✅ | 基本相同 ⚠️ | 不同 ❌ |
|----------|--------|-------------|---------|
| 文字 | 完全一样 | 字体/颜色微小差异 | 读音相同但字形不同 |
| 图形 | 完全一样 | 局部细微变化 | 构图不同 |
| 组合 | 完全一样 | 排列顺序变化 | 要素替换 |
| 颜色 | 完全一样 | 深浅色变化 | 完全不同色 |
| 三维标志 | 完全一样 | 角度差异 | 形状不同 |

**量化判断：**

```
相同商品 + 相同商标 → 侵权认定：🔴 成立（直接侵权）
相同商品 + 基本相同商标 → 侵权认定：⚠️ 可能成立（需综合判断）
相同商品 + 近似商标 → 进入第二步判断
```

### 2.3 第二步：类似商品 + 相同商标（或类似商品 + 类似商标）

**判断标准（商标法第57条第2项）：**

```
┌─────────────────────────────────────────────────────────────┐
│  类似商品                                                  │
│  ↓                                                         │
│  使用与注册商标相同或近似的商标                             │
│  ↓                                                         │
│  容易导致混淆                                              │
│  ↓                                                         │
│  → 认定侵权                                               │
└─────────────────────────────────────────────────────────────┘
```

**类似商品/服务判断依据：**

```
商品是否类似，依据以下因素综合判断：
1. 商品的功能/用途（是否相同或互补）
2. 商品的原材料/生产部门/消费对象
3. 商品的销售渠道/销售渠道
4. 商品之间是否存在特定联系（配套/替代/竞争）
5. 尼斯分类表中同一类似群组
6. 相关公众的通常认知
```

**类似商品判断量化表：**

| 判断因素 | 构成类似 ✅ | 可能类似 ⚠️ | 不类似 ❌ |
|----------|------------|-------------|----------|
| 功能用途 | 相同/互补 | 部分重叠 | 完全不同 |
| 消费对象 | 相同群体 | 部分重叠 | 不同群体 |
| 销售渠道 | 相同渠道 | 部分重叠 | 完全分离 |
| 配套关系 | 配套使用 | 可能配套 | 无关联 |
| 尼斯分类 | 同类似群 | 相邻群组 | 远距离群组 |

**混淆可能性的判断因素：**

```
混淆可能性判断六要素（2002年商标民事纠纷司法解释第10条）：

1. 注册商标的显著性和知名度
   ├─ 显著性越强 → 混淆可能性越高
   └─ 知名度越高 → 混淆可能性越高（跨类保护扩大）

2. 被控侵权商标的使用方式
   ├─ 突出使用（作为商标使用）→ 混淆可能性高
   └─ 叙述性使用（描述性使用）→ 混淆可能性低

3. 商品/服务的类似程度
   ├─ 同一种商品 → 混淆可能性高
   └─ 类似商品 → 需综合判断

4. 相关公众的注意程度
   ├─ 普通消费者 → 注意程度一般
   └─ 专业买家 → 注意程度较高

5. 实际混淆的证据
   ├─ 有实际混淆案例 → 直接证明混淆可能性
   └─ 无实际混淆案例 → 需推定混淆可能性

6. 被告的主观意图
   ├─ 故意攀附 → 混淆可能性高（恶意）
   └─ 无意使用 → 混淆可能性相对低
```

**混淆类型区分：**

```
┌─────────────────────────────────────────────────────────────┐
│  直接混淆（Direct Confusion）                                │
│  ├─ 定义：消费者误认为侵权商品来自权利人                     │
│  ├─ 判断标准：普通消费者施以普通注意是否会混淆               │
│  └─ 适用场景：同一种或类似商品 + 相同或近似商标              │
├─────────────────────────────────────────────────────────────┤
│  售后混淆（Post-sale Confusion）                             │
│  ├─ 定义：购买者以外的人（转售者/旁观者）产生混淆            │
│  ├─ 判断标准：非购买者的相关公众是否产生混淆                 │
│  └─ 适用场景：奢侈品/高价商品（购买者注意程度高）            │
├─────────────────────────────────────────────────────────────┤
│  初始兴趣混淆（Initial Interest Confusion）                  │
│  ├─ 定义：搜索阶段产生混淆，吸引消费者进入侵权网站           │
│  ├─ 判断标准：使用他人商标作为关键词是否导致混淆             │
│  └─ 适用场景：网络搜索/竞价排名/域名抢注                     │
└─────────────────────────────────────────────────────────────┘
```

**混淆可能性量化判断表：**

| 因素 | 高混淆可能性 | 中混淆可能性 | 低混淆可能性 |
|------|-------------|-------------|-------------|
| 商标近似度 | 高度近似 | 中度近似 | 低度近似 |
| 商品类似度 | 同一种商品 | 类似商品 | 不类似商品 |
| 权利商标知名度 | 知名/驰名 | 一般知名 | 不知名 |
| 被告主观意图 | 恶意攀附 | 无意使用 | 巧合 |
| 实际混淆证据 | 有大量混淆案例 | 有个别混淆 | 无混淆案例 |
| **综合认定** | **→ 侵权成立** | **→ 视情况而定** | **→ 侵权不成立** |

## 三、正当使用抗辩审查

### 3.1 正当使用抗辩类型总览

```
                    ┌─────────────────────────────┐
                    │     正当使用抗辩审查         │
                    └─────────────┬───────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐
        │                         │                         │
        ▼                         ▼                         ▼
┌───────────────┐       ┌───────────────┐       ┌───────────────┐
│  描述性使用   │       │  指示性使用   │       │  说明性使用   │
│ (Descriptive) │       │ (Nominative) │       │ (Generic)    │
└───────────────┘       └───────────────┘       └───────────────┘
```

### 3.2 描述性使用抗辩（商标法第59条第1款）

**适用条件：**

```
描述性使用成立条件（须同时满足）：
1. 使用目的是描述商品的性质/功能/用途/原料/产地等
2. 使用方式为说明性使用（非作为商标使用）
3. 使用范围合理（仅说明所需最小范围）
4. 使用者与商标权人无竞争关系或无攀附故意
```

**典型场景：**

| 场景 | 是否构成描述性使用 | 说明 |
|------|-------------------|------|
| 服装网店标注"纯棉T恤" | ✅ 是 | 说明商品原料 |
| 食品包装标注"巧克力味" | ✅ 是 | 说明商品风味 |
| 手机店标注"华为手机维修" | ⚠️ 视情况 | 可能构成指示性使用 |
| 在同种商品上使用与他人注册商标相同的标识 | ❌ 否 | 不构成描述性使用 |

**量化判断标准：**

```
描述性使用判定：
├─ 是否为商品通用名称/图形 → ✅ 通用名称不可注册
├─ 是否为描述商品特点的必要使用 → ✅ 仅说明特点
├─ 是否突出使用（字体/颜色/位置） → ❌ 突出使用构成侵权
├─ 是否超出说明必要范围 → ❌ 超出必要范围构成侵权
└─ 消费者是否仍能识别来源 → ✅ 能识别则不混淆
```

### 3.3 指示性使用抗辩（Nominative Fair Use）

**适用条件：**

```
指示性使用成立条件：
1. 必须使用他人的商标才能指示商品/服务的真实来源
2. 使用方式合理且必要（不超过所需范围）
3. 使用者与商标权人存在真实关联（配套/兼容/服务）
4. 使用者无攀附故意或造成来源混淆
```

**典型场景：**

| 场景 | 是否构成指示性使用 | 说明 |
|------|-------------------|------|
| 汽车配件标注"适用于奔驰" | ✅ 是 | 说明商品兼容性 |
| 打印机耗材标注"HP墨盒专用" | ✅ 是 | 说明商品兼容性 |
| 维修店标注"苹果手机专业维修" | ⚠️ 视情况 | 需区分修理与翻新 |
| 二手商品标注"二手LV包" | ✅ 是（前提：确实是二手） | 说明商品来源 |
| 平行进口商品标注"原装进口" | ✅ 是 | 说明商品来源 |

**量化判断标准：**

```
指示性使用判定：
├─ 是否无法用其他方式替代 → ✅ 必须使用该商标
├─ 使用范围是否最小化 → ✅ 仅说明所需信息
├─ 是否造成来源混淆 → ❌ 不能让消费者误认来源
├─ 是否暗示与权利人存在虚假关联 → ❌ 不得暗示授权/赞助
└─ 被告主观是否善意 → ✅ 仅为说明真实情况
```

### 3.4 平行进口抗辩

**法律依据：**

```
平行进口合法性判断（中国）：
├─ 原则：权利用尽原则（商标权已用尽）
├─ 条件：合法取得 + 未改变注册商标 + 未损害商标功能
└─ 限制：须经权利人许可（如有独占许可则限制）
```

**平行进口合法性判断树：**

```
                    ┌─────────────────────────────┐
                    │     平行进口商品审查         │
                    └─────────────┬───────────────┘
                                  │
        ┌─────────────────────────┼─────────────────────────┐
        │                         │                         │
        ▼                         ▼                         ▼
┌───────────────┐       ┌───────────────┐       ┌───────────────┐
│ 合法取得来源  │       │ 商标使用状态  │       │ 独占许可限制  │
│ ✅ 可证明     │       │ ✅ 未改变     │       │ ⚠️ 需审查     │
└───────────────┘       └───────────────┘       └───────────────┘
        │                         │                         │
        ▼                         ▼                         ▼
┌───────────────┐       ┌───────────────┐       ┌───────────────┐
│ 商品与国内    │       │ 是否改变商标  │       │ 独占许可人   │
│ 版本是否一致  │       │ （包装/标签） │       │ 是否同意     │
└───────────────┘       └───────────────┘       └───────────────┘
```

**平行进口合法性量化表：**

| 判断因素 | 合法 ✅ | 违法 ❌ |
|----------|--------|--------|
| 取得方式 | 合法渠道购买 | 非法渠道/仿品 |
| 商标状态 | 未改变原注册商标 | 改变/去除/覆盖商标 |
| 商品品质 | 与国内版本实质相同 | 品质差异（可能损害商标声誉） |
| 包装状态 | 保持原包装完整性 | 重新包装/分装 |
| 独占许可 | 非独占许可或经许可 | 独占许可人明确反对 |
| 信息标注 | 标注产地/经销商 | 虚假标注产地 |

## 四、侵权责任承担方式

### 4.1 民事责任类型

```
┌─────────────────────────────────────────────────────────────┐
│  民事责任承担方式（民法典第179条 + 商标法第60条）            │
├─────────────────────────────────────────────────────────────┤
│  1. 停止侵害                                                │
│     ├─ 停止使用侵权商标                                     │
│     ├─ 销毁侵权商品/包装/标识                               │
│     └─ 屏蔽/关闭侵权网站/店铺                               │
│                                                             │
│  2. 消除影响                                                │
│     ├─ 在指定媒体刊登声明消除影响                           │
│     └─ 要求公开赔礼道歉                                     │
│                                                             │
│  3. 赔偿损失                                                │
│     ├─ 实际损失（权利人实际减少的收入/增加的支出）          │
│     ├─ 侵权获利（侵权人因侵权获得的利益）                    │
│     ├─ 许可使用费的合理倍数（参照商标许可费）               │
│     └─ 法定赔偿（500元-500万元，由法院酌定）                │
│                                                             │
│  4. 惩罚性赔偿（故意侵权+情节严重）                         │
│     └─ 赔偿数额 = 上述计算方式的1-5倍                       │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 赔偿数额计算量化表

| 计算方式 | 适用条件 | 举证要求 | 参考金额 |
|----------|----------|----------|----------|
| 实际损失 | 权利人可证明损失 | 提供财务证据 | 可主张 |
| 侵权获利 | 侵权人账目清晰 | 提供侵权人财务记录 | 可主张 |
| 许可费倍数 | 有可参照许可费 | 提供许可合同 | 通常1-3倍 |
| 法定赔偿 | 以上均难以确定 | — | 500元-500万元 |
| 惩罚性赔偿 | 故意+情节严重 | 证明恶意和情节 | 1-5倍 |

## 五、具体评估步骤

### 步骤一：权利基础核实（10分钟）

```
□ 核实商标注册证（注册号/注册人/类别/期限）
□ 核实权利稳定性（是否有撤销/无效程序）
□ 确定核定使用的商品/服务范围
□ 评估权利商标的显著性和知名度
□ 收集权利商标使用证据（证明知名度）
```

### 步骤二：涉嫌侵权行为分析（10分钟）

```
□ 提取涉嫌侵权商标信息（标识/文字/图形）
□ 确认涉嫌侵权商品/服务类别
□ 分析使用方式（是否突出使用）
□ 核实销售渠道（线上/线下/跨境）
□ 收集涉嫌侵权证据（截图/购买实物/公证）
□ 判断使用时间/规模/主观意图
```

### 步骤三：进入侵权判断树（15分钟）

```
第一步判断：相同商品 + 相同商标
├─ 是 → 直接认定侵权（假冒注册商标）
└─ 否 → 进入第二步

第二步判断：类似商品 + 相同/近似商标
├─ 评估商品类似度
├─ 评估商标近似度
├─ 评估混淆可能性（六要素）
└─ 综合判断侵权是否成立

第三步判断：正当使用抗辩审查
├─ 是否为描述性使用
├─ 是否为指示性使用
├─ 是否为平行进口（如涉及）
└─ 抗辩是否成立
```

### 步骤四：混淆可能性量化评估（10分钟）

```
混淆可能性六要素打分：

1. 商标近似度（权重：25%）
   ├─ 高度近似（易混淆）→ 3分
   ├─ 中度近似（可能混淆）→ 2分
   └─ 低度近似（不易混淆）→ 1分

2. 商品类似度（权重：25%）
   ├─ 同一种商品 → 3分
   ├─ 类似商品 → 2分
   └─ 非类似商品 → 1分

3. 权利商标知名度（权重：20%）
   ├─ 知名/驰名商标 → 3分
   ├─ 一般知名 → 2分
   └─ 不知名 → 1分

4. 被告主观意图（权重：15%）
   ├─ 恶意攀附 → 3分
   ├─ 无意使用 → 2分
   └─ 巧合/不知 → 1分

5. 实际混淆证据（权重：10%）
   ├─ 有大量实际混淆 → 3分
   ├─ 有个别混淆 → 2分
   └─ 无混淆证据 → 1分

6. 销售渠道重叠度（权重：5%）
   ├─ 完全相同渠道 → 3分
   ├─ 部分重叠 → 2分
   └─ 不同渠道 → 1分

加权总分 = Σ（得分 × 权重）
判断标准：
├─ 2.5-3.0分 → 🔴 高混淆可能性 → 侵权成立
├─ 2.0-2.5分 → ⚠️ 中混淆可能性 → 视情况而定
└─ 1.0-2.0分 → ✅ 低混淆可能性 → 侵权不成立
```

### 步骤五：法律后果评估（5分钟）

```
□ 评估应承担的民事责任类型
□ 估算赔偿数额区间
□ 判断是否涉及行政处罚（工商查处）
□ 判断是否涉及刑事追诉（假冒注册商标罪）
□ 评估诉讼时效（3年，从知道或应当知道侵权之日起算）
```
