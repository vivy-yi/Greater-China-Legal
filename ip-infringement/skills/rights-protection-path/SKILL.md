# /rights-protection-path — 维权路径选择

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取维权路径规则/刑事立案标准。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取各路径条件/优劣对比
2. 评估侵权严重程度
3. 评估证据充分性
4. 选择最优维权路径
5. 给出具体步骤建议

## 四种维权路径对比

| 路径 | 速度 | 成本 | 效果 | 适用场景 |
|---|---|---|---|---|
| 协商和解 | 快 | 低 | 不确定 | 损失小/对方有诚意 |
| 行政投诉 | 中 | 中 | 查处+有限赔偿 | 须固定证据/生产销售窝点 |
| 民事诉讼 | 慢 | 高 | 确定赔偿 | 证据充分/金额较大 |
| 刑事报案 | 中 | 中 | 威慑力大 | 情节严重/跨境有影响力 |

## 维权路径选择树

```
第一步：评估侵权严重程度
  → 批量生产/恶意/造成重大损失 → 刑事报案优先
  → 单次/轻微 → 民事诉讼或协商

第二步：评估证据充分性
  → 证据充分 → 直接民事诉讼
  → 证据不足 → 先行政投诉固定证据

第三步：评估时效
  → 民事诉讼时效3年
  → 刑事追诉时效：商标/专利10年，著作权无限制

第四步：选择路径
  → 刑事报案（须同时满足：严重情节+证据充分）
  → 民事诉讼（须：权属证明+侵权证明+损害证明）
  → 行政投诉（须：权属证明+侵权证据）
```

## 刑事立案标准

### 商标侵权（情节严重）

```
标准（最高检公安部立案标准）：
- 非法经营数额≥5万元
- 违法所得数额≥3万元
- 制造、销售假冒注册商标标识≥2万件（套）
- 两年内因同种商标侵权受过行政处罚又实施侵权
```

### 专利侵权（情节严重）

```
标准：
- 假冒专利违法所得≥10万元
- 假冒专利给权利人造成损失≥50万元
- 造成重大社会影响
```

### 著作权侵权（情节严重）

```
标准（最高检公安部立案标准）：
- 非法经营数额≥5万元
- 违法所得数额≥3万元
- 复制品数量≥500份（音像制品/软件）
```
