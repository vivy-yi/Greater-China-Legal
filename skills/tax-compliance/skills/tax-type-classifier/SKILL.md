# /tax-type-classifier — 税种类型判断

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取税种覆盖/适用税率。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取企业类型/税种覆盖
2. 分析企业类型和业务
3. 判断涉及的税种
4. 确定适用税率
5. 输出税种清单

## 税种分类

| 税种 | 纳税人 | 税率/计算方式 |
|---|---|---|
| 企业所得税 | 居民企业/非居民企业 | 25%/15%/20% |
| 增值税 | 一般纳税人/小规模 | 13%/9%/6%/3% |
| 个人所得税 | 个人所得者 | 七级超额累进 |
| 消费税 | 特定商品生产者 | 定额/从价 |
| 关税 | 进口货物收发货人 | 关税税率表 |
| 土地增值税 | 转让土地使用权/房产 | 超率累进30%-60% |
| 印花税 | 应税凭证立账人 | 0.03%-1% |

## 增值税税率判断

| 行业/商品类型 | 税率 |
|---|---|
| 销售货物/加工修理修配 | 13% |
| 建筑/运输/基础电信/邮政 | 9% |
| 增值电信/金融/现代服务/生活服务 | 6% |
| 小规模纳税人 | 3%（2023年后1%） |

## 企业所得税优惠税率

| 情形 | 税率 |
|---|---|
| 一般企业 | 25% |
| 高新技术企业 | 15% |
| 技术先进型服务企业 | 15% |
| 小型微利企业 | 20%（实际税负2.5%-5%） |
| 西部大开发企业 | 15% |
