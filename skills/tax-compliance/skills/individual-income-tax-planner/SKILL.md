# /individual-income-tax-planner — China Mainland

## 法律背景与法规框架

2018年修订的《个人所得税法》建立了"综合所得+分类所得"的混合税制。综合所得（工资薪金、劳务报酬、稿酬、特许权使用费）实行按月预扣预缴+年度汇算清缴；分类所得（经营所得、财产租赁、财产转让、股息红利、偶然所得）分别按各自规则计税。综合所得适用3%-45%的七级超额累进税率，经营所得适用5%-35%的五级超额累进税率。

个税筹划的核心是在合法的前提下，合理利用税法给予的扣除、优惠和差异安排来降低税负。可合法利用的工具有：专项附加扣除（子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人、3岁以下婴幼儿照护）、年终奖优惠计税政策（延续至2027年底）、个人养老金递延纳税、外籍个人免税补贴等。但所有筹划安排必须具备合理商业目的，不得构成避税或逃税。

## 工作流程

### 第一步：收入分类与结构分析

```
收入结构分析
├─ 综合所得（3%-45%累进税率）
│   ├─ 工资薪金（受雇）
│   ├─ 劳务报酬（20%费用扣除，年度汇算）
│   ├─ 稿酬（30%费用扣除优惠）
│   └─ 特许权使用费（20%费用扣除）
├─ 经营所得（5%-35%累进税率）
│   ├─ 个体工商户
│   ├─ 个人独资企业/合伙企业
│   └─ 承包经营/承租经营
├─ 分类所得（固定税率）
│   ├─ 财产租赁：20%（住房租赁10%）
│   ├─ 财产转让：20%
│   ├─ 利息股息红利：20%（持股>1年A股免税）
│   └─ 偶然所得：20%
└─ 免税收入
    ├─ 国债利息
    ├─ 保险赔款
    ├─ 国家统一津贴补贴
    └─ 按规定缴付的社保和公积金
```

**筹划切入点：**
- 收入类型之间税率有差异（如经营所得可扣除成本，最高税率35%但可抵扣项目多）
- 不同收入类型的扣除方式不同
- 收入时间安排可影响年度税负

### 第二步：专项附加扣除最大化

**七项专项附加扣除（2026年标准）：**

| 扣除项目 | 扣除标准 | 适用条件 |
|---------|---------|---------|
| 子女教育 | 每个子女每月2000元 | 3岁-博士全日制教育 |
| 继续教育 | 学历教育每月400元；职业资格证书3600元/年 | 在学期间或取证当年 |
| 大病医疗 | 超过15000元部分，上限80000元/年 | 医保报销后个人负担 |
| 住房贷款利息 | 每月1000元（首套房） | 最长240个月 |
| 住房租金 | 800-1500元/月（按城市分三档） | 无自有住房 |
| 赡养老人 | 每月3000元（独生子女）；非独生子女分摊 | 父母年满60岁 |
| 婴幼儿照护 | 每个3岁以下婴幼儿每月2000元 | 2022年起 |

**筹划建议：**
- 夫妻双方合理分配扣除项目（高税率方多扣）
- 独生子女与非独生子女的赡养老人扣除分配
- 租房vs贷款利息（不可同时享受，择优选择）
- 大病医疗扣除的年度安排

### 第三步：年终奖优惠计税筹划

年终奖优惠计税政策延续至2027年底：全年一次性奖金不并入综合所得，单独按除以12后的金额确定适用税率。

```
年终奖最优发放区间分析：

应纳税额 = 奖金 ÷ 12 → 找税率 → 奖金 × 税率 - 速算扣除数

注：存在"税率跳档"区间（多发1元导致税负大幅增加）
无效奖金区间（多发奖金少于多缴税款）：
├─ 36001-38567元（年终奖从3%跳到10%）
├─ 144001-160500元（从10%跳到20%）
├─ 300001-318333元（从20%跳到25%）
├─ 420001-447500元（从25%跳到30%）
├─ 660001-706538元（从30%跳到35%）
├─ 960001-1120000元（从35%跳到45%）

筹划策略：在无效区间内，应选择少发（避免高税率）或将超额部分转至综合所得或其他方式发放。
```

**年终奖与综合所得的分配优化：**
- 基本思路：总收入在"综合所得"和"年终奖"之间分配
- 综合所得应纳税额 + 年终奖应纳税额之和最小化
- 常用做法：综合所得适用税率高于10%时，多用年终奖分配；低于10%时，少用年终奖

### 第四步：其他合法筹划工具

**个人养老金（税延型）：**
- 每年缴纳上限：12000元
- 在综合所得/经营所得中税前扣除
- 领取时按3%的税率缴纳个税（不并入综合所得）
- 适合：适用税率10%及以上的纳税人

**外籍个人免税补贴（至2027年底）：**
- 住房补贴、语言训练费、子女教育费（实报实销）
- 须凭合法凭证报销
- 外籍个人可以选择享受免税补贴或专项附加扣除（二选一）

**股权激励个税优化：**
- 见 equity-incentive-tax-advisor 详情
- 上市公司股权激励单独计税（不并入综合所得）
- 非上市公司递延纳税（转让时按20%）

**自由职业者/灵活用工：**
- 劳务报酬与经营所得的边界（避免被重新定性）
- 经营所得可以扣除实际发生的成本费用
- 利用税收优惠园区政策（须真实经营）

## 升级决策门

以下情况须移交专业税务律师处理：

- 涉及通过设立个体工商户/个人独资企业将劳务报酬转化为经营所得（可能被"穿透"认定）
- 涉及跨境收入的个税处理（如境外股票期权/境外房产出租）
- 涉及非居民个人身份认定或居民身份冲突
- 涉及高净值人士的复杂资产交易（如房产置换/股权转让）
- 涉及税务机关已启动个税稽查或调查
- 涉及税收优惠政策适用条件存在疑问
- 涉及遗产、赠与或代持安排的个税影响
