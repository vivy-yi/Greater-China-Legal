# 企业债券/公司债券/可转债 法律场景

## 场景定义

Corporate-bond = 企业债(发改委)/公司债(证监会)/可转债

本场景涵盖中国资本市场中企业债券、公司债券及可转换债券的发行、交易、信息披露等全流程法律合规事务。

## 关键法律要点

### 发行主体资格
- 公司债：股份有限公司、有限责任公司
- 企业债：具有法人资格的企业
- 可转债：上市公司

### 发行条件（公司债）
1. 股份有限公司净资产 ≥ 6000万元
2. 有限责任公司净资产 ≥ 6000万元
3. 最近三年平均可分配利润足以支付债券一年利息
4. 募集资金投向符合国家产业政策
5. 债券余额不超过净资产40%

### 信息披露要求
- 发行前：募集说明书披露
- 存续期：年度报告、中期报告
- 重大事项：临时公告
