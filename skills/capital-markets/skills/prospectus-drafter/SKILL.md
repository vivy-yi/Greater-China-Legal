## 章节结构（A股IPO招股说明书）

| 章节 | 核心内容 | 注意事项 |
|---|---|---|
| 第一节 释义 | 专业术语定义 | 与正文保持一致 |
| 第二节 概览 | 发行概况、主业、发行价格区间 | 简明扼要 |
| 第三节 风险因素 | 行业风险/经营风险/财务风险/政策风险 | 重大性原则 |
| 第四节 发行人的基本情况 | 股本演变、股东、实际控制人 | 股权清晰 |
| 第五节 业务与技术 | 所属行业、业务模式、市场地位、竞争优势 | 引用数据来源 |
| 第六节 同业竞争与关联交易 | 竞争方情况、交易金额、定价公允性 | 核查充分性 |
| 第七节 董事监事高级管理人员 | 任职资格、兼职、薪酬 | 核查合规性 |
| 第八节 公司治理 | 三会运作、内控制度 | 有效性说明 |
| 第九节 财务会计信息 | 审计意见、财务报表、报表附注 | 与审计报告一致 |
| 第十节 管理层讨论与分析 | 盈利能力、偿债能力、运营效率 | 逻辑清晰 |
| 第十一节 募集资金运用 | 募投项目备案/环评/用地 | 合规性核查 |
| 第十二节 未来发展与规划 | 战略目标、与现有业务衔接 | 审慎表述 |

---

## 债券募集说明书关键章节

| 章节 | 核查要点 |
|---|---|
| 发行条款 | 票面利率、期限、还本付息方式 |
| 募集资金用途 | 是否符合产业政策、是否有变相占用 |
| 发行人基本情况 | 主体评级、净资产、资产负债率 |
| 财务状况 | 最近3年审计数据、偿债能力指标 |
| 担保/信用增进 | 担保人资质、担保合同效力 |
| 风险揭示 | 利率风险/流动性风险/信用风险 |

---

## 质量核查清单

- [ ] 法条引用准确，表述与法规原文一致
- [ ] 财务数据与审计报告一致
- [ ] 不存在重大遗漏（对照披露准则逐项核对）
- [ ] 风险因素按重大性排序，不含空洞表述
- [ ] 避免"最"、"第一"等绝对化表述
- [ ] 涉及预测性陈述有合理依据

## 升级决策门

**以下情形须向律师升级：**
- 涉及未公开重大信息
- 行业数据无法核实来源
- 存在潜在重大诉讼或行政处罚未披露
- 涉及国有资产特殊问题
- 涉及外商投资准入问题

## 本 Skill 不涵盖

- 审计报告真实性核查（须持证会计师事务所）
- 行业准入具体审批流程
- 税务处理（请咨询税务顾问）
- 发行保荐业务（须持证保荐代表人）
