# 资本市场 — 判断框架

## 一、A股IPO发行条件核查框架

### 主体资格（硬性条件）

| 条件 | 法条依据 | 核查结果 |
|---|---|---|
| 依法设立且存续满3年 | 《首发办法》第8条 | [✅/❌] |
| 注册资本已足额缴纳 | 《首发办法》第9条 | [✅/❌] |
| 生产经营合法合规 | 《首发办法》第10条 | [✅/❌] |
| 最近3年主业、董事、高管无重大变化 | 《首发办法》第12条 | [✅/❌] |
| 实际控制人最近3年未变更 | 《首发办法》第12条 | [✅/❌] |
| 股权清晰 | 《首发办法》第13条 | [✅/❌] |

### 财务指标

| 指标 | 科创板（预计市值10亿+） | 主板（最近一期净利润>0） | 核查结果 |
|---|---|---|---|
| 最近3年净利润 | 均为正且累计>3000万 | 最近3年均为正且累计>1.5亿 | [✅/❌] |
| 最近3年经营活动现金流 | 累计>1亿或最近3年收入>3亿 | — | [✅/❌] |
| 最近3年营业收入 | 累计>3亿 | 最近3年均为正 | [✅/❌] |
| 预计市值 | 满足适用标准 | — | [✅/❌] |

### 合规性核查

| 核查项 | 法条依据 | 核查结果 |
|---|---|---|
| 最近3年无重大违法行为 | 《首发办法》第10条 | [✅/❌] |
| 报告期不存在重大不确定性诉讼/仲裁 | 《首发办法》第20条 | [✅/❌] |
| 关联交易定价公允 | 《首发办法》第20条 | [✅/❌] |
| 同业竞争已解决或承诺解决 | 《首发办法》第20条 | [✅/❌] |

## 二、公司债发行核查框架

### 公开发行公司债（证监会）

| 条件 | 标准 | 核查结果 |
|---|---|---|
| 净资产（股份有限公司>3000万，有限>6000万） | 《公司债发行与交易管理办法》第16条 | [✅/❌] |
| 累计债券余额不超过净资产40% | 同上 | [✅/❌] |
| 最近3年平均可分配利润足以支付债券1年利息 | 同上 | [✅/❌] |
| 募集资金投向符合国家产业政策 | 同上 | [✅/❌] |
| 债券利率不超过国务院限定水平 | 同上 | [✅/❌] |

### 非公开发行公司债

| 条件 | 标准 | 核查结果 |
|---|---|---|
| 净资产要求参照公开发行 | 同上 | [✅/❌] |
| 发行对象不超过200人 | 《公司债发行与交易管理办法》第26条 | [✅/❌] |
| 不得采用广告/公开劝诱形式 | 同上 | [✅/❌] |

## 三、ABS（资产支持证券）核查框架

| 核查项 | 要点 | 核查结果 |
|---|---|---|
| 基础资产合规性 | 不得为法律禁止的负面清单资产 | [✅/❌] |
| 基础资产真实性 | 债权真实存在，权属清晰 | [✅/❌] |
| SPV设立 | 特殊目的载体设立合规 | [✅/❌] |
| 现金流归集 | 现金流真实、完整、可预测 | [✅/❌] |
| 信用增级安排 | 内部增信/外部增信措施 | [✅/❌] |
| 信息披露 | 计划说明书完备 | [✅/❌] |

## 四、升级决策门

**以下情形须向律师升级：**
- 主体资格存在瑕疵但有解释空间
- 财务指标勉强达标
- 存在潜在重大诉讼
- 同业竞争解决方案不明确
- 关联交易金额重大且定价不公允
