# 资本市场 — 数据源清单

## 一、内部数据（发行人提供）

| 数据类型 | 内容 | 核查要点 |
|---|---|---|
| 工商登记资料 | 营业执照、公司章程、股权结构 | 存续状态、股权清晰度 |
| 财务报表 | 近3年审计报告 | 收入/利润/现金流真实性 |
| 重大合同 | 金额占收入5%以上合同 | 经营稳定性 |
| 诉讼仲裁 | 涉诉金额/涉诉事项 | 是否为重大不确定性 |
| 关联方清单 | 关联交易对手方 | 同业竞争/关联交易核查 |
| 知识产权 | 专利/商标/著作权清单 | 资产完整性 |

## 二、公开数据源

### 法规与政策

| 数据源 | 说明 |
|---|---|
| 证监会官网 | 现行法规、部门规章、规范性文件 |
| 交易所官网 | 科创板/主板/创业板规则 |
| 北大法宝 | 法律法规检索（中英文） |
| Westlaw China | 国际法律数据库 |

### 发行人背景核查

| 数据源 | 说明 |
|---|---|
| 国家企业信用信息公示系统 | 工商登记信息 |
| 中国裁判文书网 | 涉诉信息 |
| 全国法院被执行人信息查询 | 被执行信息 |
| 知识产权局 | 专利/商标登记信息 |

### 市场数据

| 数据源 | 说明 |
|---|---|
| Wind | 资本市场数据、债券行情 |
| 彭博 | 全球市场数据 |
| 巨潮资讯 | 上市公司公告、披露文件 |

## 三、外部接口

| 接口 | 用途 |
|---|---|
| 证监会行政许可查询 | 批文进度 |
| 交易所审核进度查询 | 在审项目状态 |
| 企业信用评级（联合评级/中诚信） | 债券评级 |

## 四、数据质量判断

| 数据类型 | 判断标准 |
|---|---|
| 审计报告 | 出具机构是否具有证券从业资格 |
| 工商信息 | 是否与发行人提供文件一致 |
| 诉讼信息 | 是否存在隐瞒重大诉讼 |
| 知识产权 | 权属证书是否在有效期内 |
