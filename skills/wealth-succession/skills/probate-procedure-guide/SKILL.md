## 一、遗嘱执行启动程序

### 1.1 遗嘱执行人确认
| 情形 | 执行人确定方式 | 优先顺序 |
|-----|--------------|---------|
| 遗嘱指定执行人 | 遗嘱指定 | 第一优先 |
| 单位担任执行人 | 继承人委托 | 第二优先 |
| 亲属担任执行人 | 继承人推选 | 第三优先 |
| 民政部门/村委会 | 无继承人或继承人均放弃 | 最后保障 |

**遗嘱执行人资格要求：**
- 自然人须为完全民事行为能力人
- 不得为继承人本人（除非遗嘱另有指定）
- 不得为与继承人存在利益冲突的人

### 1.2 遗嘱效力确认
```
确认流程图：
┌─────────────────────────────────────┐
│ 1. 提交遗嘱原件及死亡证明           │
│          ↓                         │
│ 2. 判断遗嘱类型                     │
│    ┌────┬────┬────┬────┐         │
│    ↓    ↓    ↓    ↓    ↓         │
│   公证 自书 代书 录音 口头         │
│    ↓    ↓    ↓    ↓    ↓         │
│ 3. 形式要件审查                     │
│    ↓    ↓    ↓    ↓    ↓         │
│ 4. 公证机构出具遗嘱继承权公证书     │
│          ↓                         │
│ 5. 启动遗产清点程序                 │
└─────────────────────────────────────┘
```

### 1.3 公证遗嘱继承程序
**公证机构选择：**
- 被继承人死亡时住所地公证机构
- 遗产主要所在地公证机构
- 继承人住所地公证机构（须上述机构同意）

**公证遗嘱继承优势：**
1. 公证书具有最高证明力，减少继承人争议
2. 可凭公证书直接办理财产过户
3. 公证机构承担初步审查责任，降低执行人风险

## 二、遗产清点与保全

### 2.1 遗产清点范围
| 资产类别 | 清点内容 | 价值确定方式 |
|---------|---------|------------|
| 银行存款 | 账户信息、余额 | 金融机构查询 |
| 不动产 | 房产证、土地使用证 | 不动产登记信息或评估 |
| 股权 | 工商登记信息 | 公司净资产评估或市价 |
| 车辆 | 行驶证、车辆登记信息 | 市场询价 |
| 有价证券 | 账户信息、持仓情况 | 交易价格 |
| 贵重物品 | 珠宝、古董、艺术品 | 专业评估 |
| 债权债务 | 借条、合同、往来凭证 | 凭证核实 |

### 2.2 遗产保全措施
**紧急保全申请（继承开始后，遗产分配前）：**
```
触发条件：
- 继承人或利害关系人有转移、隐匿遗产嫌疑
- 遗产面临灭失、损毁风险
- 继承人之间存在重大争议

申请程序：
1. 向被继承人死亡时住所地或遗产所在地人民法院申请
2. 提交：申请书、身份证明、遗嘱复印件、财产线索
3. 法院裁定后，发出财产保全裁定书
4. 通知相关财产保管人/金融机构协助执行
```

**保全措施类型：**
- 查封（不动产、车辆）
- 冻结（银行账户、股权）
- 扣押（贵重物品）
- 限制交易（特定财产）

### 2.3 债务清偿程序
```
清偿顺序（《民法典》第1161条）：
┌────────────────────────────────────────────┐
│ 第一顺位：继承费用（丧葬费、遗嘱执行费）   │
├────────────────────────────────────────────┤
│ 第二顺位：被继承人债务（生前借款、合同）   │
├────────────────────────────────────────────┤
│ 第三顺位：税款（遗产变现环节）             │
├────────────────────────────────────────────┤
│ 第四顺位：剩余财产按遗嘱/法定继承分配     │
└────────────────────────────────────────────┘
```

**债务清偿原则：**
- 以遗产实际价值为限清偿债务（限定继承原则）
- 继承人放弃继承的，对被继承人债务不承担责任
- 无遗嘱执行人时，继承人负责债务清偿

## 三、遗产分配与程序完结

### 3.1 遗产分配方案制定
**分配原则：**
1. 遗嘱优先于法定继承（遗嘱自由原则）
2. 必留份保障（《民法典》第1141条）
3. 胎儿预留份（《民法典》第1155条）
4. 均等分配为原则，但可依据贡献、扶养义务等调整

**分配协议签订：**
```
参与主体：全体继承人
协议内容：
- 遗产范围及清单确认
- 各继承人应得份额
- 财产分配方式（实物分割/折价补偿/变现分割）
- 债务清偿方案
- 违约责任

协议效力：
- 经全体继承人签名后生效
- 可公证以增强效力
- 异议人可在法定期限内主张权利
```

### 3.2 财产过户时间表
| 财产类型 | 过户/变更手续 | 主管部门 | 典型时限 |
|---------|-------------|---------|---------|
| 银行存款 | 继承公证书→银行 | 各商业银行 | 5-15工作日 |
| 房产 | 继承公证书→不动产登记 | 自然资源部门 | 15-30工作日 |
| 车辆 | 继承公证书→车管所 | 公安交警部门 | 5-10工作日 |
| 股权 | 继承公证书→工商变更 | 市场监督管理部门 | 10-20工作日 |
| 证券 | 继承公证书→券商/登记公司 | 证券公司/登记公司 | 10-20工作日 |

### 3.3 遗嘱执行报告
**报告内容：**
- 遗嘱执行概况
- 遗产清点结果
- 债务清偿情况
- 遗产分配明细
- 执行费用结算
- 遗留问题说明

**报告提交对象：**
- 全体继承人
- 公证机构（若由公证机构参与）
- 人民法院（若有诉讼）

## 升级决策门

### 触发条件（满足任一即升级）
1. **继承人间争议**：继承人之间就遗产分配无法达成一致，存在诉讼风险
2. **遗嘱效力争议**：任何继承人或利害关系人对遗嘱效力提出质疑
3. **遗产范围争议**：无法确定全部遗产范围，或存在疑似隐匿财产
4. **跨境遗产**：被继承人留有境外财产，或继承人为境外人士
5. **债务金额重大**：被继承人生前债务金额超过遗产总额的30%
6. **执行人职责争议**：遗嘱执行人职责履行存在争议或受到质疑
7. **遗赠效力争议**：受遗赠人权利主张与继承人主张冲突
8. **遗产被不当处分**：发现遗产被部分继承人擅自处分

### 升级路径
```
升级决策门触发 → 停止自动生成程序建议 →
输出"本案件存在[具体情形]，建议通过以下途径解决"通知 →
提供继承诉讼律师查询路径（参见 references/查询路径.md）
或建议向公证机构申请调解
```

### 升级后本skill责任终止情形
- 进入诉讼程序后，本skill不提供诉讼策略建议
- 涉及遗嘱效力认定的司法鉴定程序，不属于本skill范围
- 执行人为法人时的具体操作，不属于本skill范围

## 本Skill不涵盖

### 明确排除事项
1. **遗嘱见证与公证代理**：见证遗嘱订立、代理申请公证手续
2. **继承诉讼代理**：代理继承人提起或应对继承纠纷诉讼
3. **遗产评估**：不动产、股权等资产的评估（须聘请有资质评估机构）
4. **财产过户代理**：代替继承人前往各部门办理具体过户手续
5. **税务申报代理**：遗产变现环节的各项税务申报
6. **境外遗产程序**：涉及外国遗嘱认证程序（Probate）的跨境事务
7. **被继承人债务核实**：对债务真实性、金额的司法认定
8. **遗嘱执行人赔偿责任**：执行人失职导致的赔偿责任认定
9. **跨境遗产执行**：涉及外汇管制、跨境资金流动的遗产执行
10. **遗产税（如开征）申报**：遗产税的计算与申报代理
