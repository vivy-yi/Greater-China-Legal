# 技能文件一：ODI登记实务操作技能

## 技能概述

本技能用于指导律师或企业境外投资部门人员独立完成境外直接投资（ODI）的核准/备案及外汇登记实务操作。

---

## 一、前置准备工作

### 1.1 确定管理机关

根据投资主体情况确定受理层级：

| 投资主体 | 受理机关 |
|---------|---------|
| 中央企业 | 商务部 |
| 地方企业 | 省级商务主管部门 |
| 金融企业 | 商务部（会同金融主管部门） |

### 1.2 行业判断

首先判断是否属于敏感行业：

**敏感行业（须商务部核准）**：
- 武器装备研制生产
- 土地成片开发
- 竹鼠养殖
- 新闻媒体
- 法律法规限制的其他行业

**敏感国家/地区（须商务部核准）**：
- 未建交国家
- 战争、内乱国家
- 国际条约限制国家

**非敏感行业+非敏感地区** → 地方商务主管部门备案

### 1.3 材料预审清单

```
□ 投资主体营业执照（副本复印件，加盖公章）
□ 经审计的近三年财务报表
□ 可行性研究报告（含市场分析、收益测算、风险评估）
□ 资金来源说明（自有资金/银行融资/其他，需明确比例）
□ 拟投资项目的董事会决议或股东会决议
□ 如为并购：提供并购协议或合作意向书
□ 如涉及国有资产：国有资产监督管理机构批准文件
□ 境外投资备案表（商务部门提供格式）
```

---

## 二、核准程序（敏感项目）

### 2.1 提交材料

向商务部提交一式一套申请材料：

1. **请示文**（红头文件，含项目名称、投资主体、投资规模、目的地）
2. **《境外投资申请表》**
3. **附件**：
   - 营业执照复印件
   - 审计报告（近三年）
   - 可行性研究报告
   - 资金来源说明
   - 并购合同/增资决定
   - 投资环境分析（针对敏感国家和地区）

### 2.2 商务部审查

- **法定时限**：30个工作日
- **延期**：可延长30个工作日（须书面通知）
- **补正**：材料不完整5个工作日内通知补正

### 2.3 证书领取

核准通过后，领取《企业境外投资证书》，有效期**2年**，需展期提前申请。

---

## 三、备案程序（非敏感项目）

### 3.1 提交材料

向省级商务主管部门提交：

1. 《境外投资备案表》（在线系统填报）
2. 营业执照复印件
3. 董事会决议
4. 资金来源说明

### 3.2 备案时限

- **材料齐备**：当场或3个工作日内出具备案回执
- **在线系统**：商务部境外投资管理系统（OIAS）

---

## 四、发改委核准/备案

### 4.1 判断标准

| 项目类型 | 管理方式 |
|---------|---------|
| 敏感行业+中方投资额≥3亿美元 | 核准 |
| 敏感行业+中方投资额<3亿美元 | 备案 |
| 非敏感行业+中方投资额≥3亿美元 | 备案（中央层面） |
| 非敏感行业+中方投资额<3亿美元 | 省级发改委备案 |

### 4.2 敏感行业范围（发改委）

- 武器装备
- 跨境水资源
- 新闻传媒
- 房地产（受限）
- 其他需核准行业

### 4.3 项目申请报告大纲

```
一、项目名称
二、项目建设必要性
三、项目内容（建设规模/并购内容）
四、项目投资情况（规模、资金来源）
五、社会经济效益分析
六、风险分析
七、附件：董事会决议、审计报告、资信证明
```

## 五、外汇登记程序

### 5.1 登记节点

完成商务部门ODI核准/备案 → 银行办理外汇登记 → 资金汇出

### 5.2 银行端操作

**第一步：业务开通**
- 在具备ODI外汇业务资格的银行开立外汇账户
- 签署外汇业务协议

**第二步：提交材料至银行**
```
□ 资本项目外汇业务申请表
□ ODI证书（商务部门颁发）
□ 发改委核准/备案文件
□ 投资主体的营业执照
□ 经审计的财务报表
□ 资金来源说明
□ 资金使用计划
```

**第三步：银行端操作**
- 银行登录资本项目外汇业务系统
- 录入企业信息、项目信息
- 生成"资本项目信息表"登记凭证

**第四步：资金汇出**
- 凭外汇登记凭证办理购汇
- 跨境汇款（电汇/票汇）

### 5.3 37号文特殊登记

**适用情形**：境内居民（含自然人）在境外设立特殊目的公司（SPV），拟返程投资

**登记时机**：
- 创始人出境前
- 或返程投资前

**补登记**：历史上未登记的SPV，向外汇局申请补登记

## 六、变更与注销

### 6.1 变更事项

以下事项须在30日内向原核准/备案机关报告：
- 投资主体变更
- 投资金额变动（增减资）
- 股权结构变化
- 撤回投资
- 合并、分立

### 6.2 注销ODI证书

撤回投资时，需：
1. 撤回投资清算报告
2. 外汇注销登记
3. 交回ODI证书原件
4. 向商务部门备案注销

## 七、常见问题排查

| 问题 | 原因 | 解决方案 |
|-----|-----|---------|
| 银行拒绝办理外汇登记 | ODI证书行业与实际不符 | 向商务部门申请变更说明 |
| 资金来源审核不通过 | 财务报表与资金说明不符 | 提供详细资金来源证明 |
| 并购合同未经法律审核 | 商务部门要求合同附件 | 补充法律审查意见 |
| 备案被退回 | 材料签字/盖章不完整 | 补充完整材料重新提交 |
| 证书过期 | 未关注有效期 | 申请展期（提前30日） |

---

## 八、实操注意事项

1. **时效管理**：ODI证书有效期2年，项目未开工应及时申请展期
2. **资金链路**：确保资金路径清晰，全程可追溯
3. **行业判断**：敏感行业判断是核心，提前与商务部门沟通
4. **银行选择**：选择具备资本项目业务资格的银行
5. **档案留存**：全套材料复印件留档备查
