# 交易结构设计技能 (Deal Structurer)

## 技能定位

本技能用于设计和优化并购交易结构，在法律合规框架内实现客户商业目标最大化。

## 核心能力

### 1. 结构类型分析

#### 1.1 股权收购 vs 资产收购
```
股权收购特征：
- 优点：承继全部权利义务，交易简便，整体评估
- 缺点：承继潜在负债，延续历史风险
- 适用：规范运营的公司，股东整体退出

资产收购特征：
- 优点：风险隔离，可选择性接收，税务灵活
- 缺点：税负较重，需逐项交割，许可续展
- 适用：资产清晰的公司，资产包转让
```

#### 1.2 现金交易 vs 股份交易
```
现金交易：
- 优点：交易确定性高，卖方无持续风险
- 缺点：卖方即时税负，卖方参与意愿
- 适用：纯财务投资人，卖方需流动性

股份交易：
- 优点：税务递延，卖方分享未来收益
- 缺点：结构复杂，买方风险延续
- 适用：战略收购，卖方愿意留资
```

#### 1.3 一手交钱一手交货 vs 分期付款
```
即时交割：
- 优点：交易确定性，买方快速控制
- 缺点：卖方风险转移，买方尽调压力大
- 适用：成熟公司，规范运营

分期付款（Earn-out）：
- 优点：风险共担，激励卖方持续经营
- 缺点：交易复杂，未来争议风险
- 适用：增长型企业，未来业绩可预期
```

### 2. 境内架构设计

#### 2.1 直接持股架构
```
适用场景：
- 境内实体直接持股
- 简单直接的产权关系
- 无特殊税务规划需求

设立要求：
□ 符合外商投资产业政策
□ 资金跨境路径清晰
□ 未来退出路径明确
```

#### 2.2 间接持股架构
```
适用场景：
- 多个项目统一管理
- 未来资本运作平台
- 家族财富管理

中间层公司考量：
□ 注册地选择（税收优惠/协定）
□ 公司治理便利性
□ 信息披露要求
□ 退出便利性
```

#### 2.3 国有产权转让特殊考量
```
审批层级：
- 国有资本运营公司：内部决策
- 国有资产监管机构：分级授权

评估要求：
- 净资产评估为基本原则
- 进场交易（产权交易所）
- 职工安置方案审批
- 债权债务处理方案
```

### 3. 跨境架构设计

#### 3.1 红筹架构
```
典型结构：
境内运营实体 → WFOE → 境外持股公司 → 上市主体

适用场景：
- 美元基金投资
- 境外上市目标
- VIE协议控制（限制性行业）

主要关注点：
□ 37号文登记
□ 境外投资备案（ODI）
□ 返程投资合规
□ 外汇进出境管理
```

#### 3.2 跨境并购融资架构
```
资金路径：
境外买方 → 境外融资 → ODI审批 → 境内支付

担保安排：
□ 境外资产担保
□ 境内母公司担保
□ 备用信用证
□ 账户质押

监管审批：
□ 发改委境外投资备案
□ 商务部境外投资证书
□ 外汇登记
□ 跨境资金池（如适用）
```

### 4. 特殊结构安排

#### 4.1 业绩对赌（Earn-out）
```
核心要素：
- 衡量指标：净利润、营业收入、市场份额
- 对赌期限：通常1-3年
- 调整机制：双向调整、封顶、保底

法律要点：
□ 会计政策一致性
□ 干扰行为禁止
□ 信息披露义务
□ 争议解决机制
```

#### 4.2 优先清算权（Participating Preferred）
```
权利内容：
- 优先分配权：优先于普通股获得分配
- 参与分配权：优先分配后继续参与分配
- 清算折价：按一定折扣优先分配

谈判要点：
□ 触发事件定义
□ 参与方式选择
□ 上限设定
```

#### 4.3 拖售权/跟随权（Drag-along / Tag-along）
```
拖售权（买方主导）：
- 条件：达到约定价格/条件
- 效力：强制其他股东一同出售
- 保护：合理价格保护

跟随权（卖方保护）：
- 条件：第三方收购触发
- 效力：其他股东有权按同等条件参与出售
- 限制：仅在特定条件下触发
```

### 5. 税务结构优化

#### 5.1 股权收购税务优化
```
特殊性税务处理（59号文）：
- 适用条件：股权支付比例≥85%
- 递延效果：当期不确认所得/损失
- 后续处理：递延确认，按后续处置计算

一般性税务处理：
- 即时确认所得/损失
- 成本基础调整
- 现金流影响评估
```

#### 5.2 资产收购税务考量
```
资产转让税负：
- 增值税：可能涉及
- 企业所得税：按资产处置收益征税
- 土地增值税：土地使用权转让可能涉及
- 契税：受让方可能免征

税负转嫁安排：
- 净价条款约定
- 税前抵扣安排
- 税收优惠继承
```

### 6. 结构设计决策树

```
第一步：确定交易目的
├── 战略收购 → 关注长期协同效应
├── 财务投资 → 关注退出路径
└── 资产剥离 → 关注税务效率

第二步：评估转让方情况
├── 国有企业 → 评估进场交易要求
├── 上市公司 → 关注信息披露、要约收购
└── 民营企业 → 相对灵活，注意代际传承

第三步：识别监管要求
├── 反垄断审查 → 评估是否达到申报标准
├── 行业监管 → 确定行业准入要求
└── 外商投资 → 确定是否受限

第四步：税务优化
├── 股权 vs 资产 → 税负对比
├── 支付方式 → 现金 vs 股份
└── 架构选择 → 直接 vs 间接

第五步：风险分配
├── 或有负债 → 赔偿机制设计
├── 交割后事项 → 托管/保证金
└── 陈述保证 → 范围与期限
```

## 升级决策门

**以下情形须向律师升级：**
- 涉及国有资产转让特殊审批
- 外商投资准入存在争议
- 反垄断审查门槛临界
- 交易结构涉及跨境架构
- 存在重大潜在负债未在尽调中发现
- 交易对方存在资信风险
- 涉及特殊监管行业（金融/医药/教育）

## 本 Skill 不涵盖

- 财务审计与估值建模
- 目标公司日常经营管理
- 税务申报与税务筹划
- 劳动仲裁代理与诉讼
- 涉及境外法律的跨境问题
