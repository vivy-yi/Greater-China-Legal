# 交割后整合技能 (Post-Closing Integration)

## 技能定位

本技能用于管理并购交易交割后的整合工作，确保交易价值实现，协同效应最大化。

## 核心能力

### 1. 整合计划制定

#### 1.1 整合时间线
```
整合阶段划分：

交割日（D日）：
□ 工商变更完成
□ 印鉴交接
□ 证照移交
□ 管理层任命
□ 内部公告

D+30日（第一个月）：
□ 组织架构调整
□ 人员沟通
□ 制度对接启动
□ 财务系统对接
□ IT系统整合

D+90日（第三个月）：
□ 核心业务流程整合
□ 供应商/客户关系梳理
□ 资产整合
□ 文化整合启动

D+180日（第六个月）：
□ 整合完成评估
□ 协同效应验证
□ 持续优化
□ 整合报告
```

#### 1.2 整合工作组
```
组织架构：
整合指导委员会（Steering Committee）：
- 双方高管组成
- 重大事项决策
- 争议解决

整合执行组（Integration Task Force）：
- 各职能负责人
- 日常整合工作
- 进度跟踪汇报

专项小组：
- 财务整合组
- HR整合组
- IT整合组
- 法务合规组
- 业务整合组
```

## 注意事项

1. **沟通优先**：及时透明的沟通是整合成功的基础
2. **持续跟踪**：整合是持续过程，需长期跟踪
3. **灵活调整**：根据实际情况调整整合计划
4. **价值保护**：确保交割前确定的交易价值不被损害
5. **合规底线**：任何整合措施不得突破合规底线

## 升级决策门

**以下情形须向律师升级：**
- 涉及国有资产转让特殊审批
- 外商投资准入存在争议
- 反垄断审查门槛临界
- 交易结构涉及跨境架构
- 存在重大潜在负债未在尽调中发现
- 交易对方存在资信风险
- 涉及特殊监管行业（金融/医药/教育）

## 本 Skill 不涵盖

- 财务审计与估值建模
- 目标公司日常经营管理
- 税务申报与税务筹划
- 劳动仲裁代理与诉讼
- 涉及境外法律的跨境问题
