# 跨境并购 (Cross-Border M&A) 场景导航

## 场景概述

跨境并购是指中国境内企业通过新设、并购或其他方式在境外设立或取得既有企业或资产的控制权，并涉及资金跨境流动的综合性法律业务。

本场景聚焦以下核心领域：
- **境外直接投资 (ODI)** 审批与备案
- **红筹架构** 搭建与拆除
- **资金出境** 路径合规
- **国家安全审查** 申报
- **外汇管理** 合规

---

## 核心法规体系

| 法规名称 | 效力层级 | 关键要点 |
|---------|---------|---------|
| 《境外投资管理办法》 | 商务部规章 | ODI审批/备案程序、禁止类目录 |
| 《外汇管理条例》 | 行政法规 | 外汇登记、资金跨境汇出管理 |
| 《企业境外投资管理办法》 | 发改委规章 | 核准/备案、敏感行业限制 |
| 《国家安全法》 | 法律 | 安全审查启动条件 |
| 《反垄断法》 | 法律 | 经营者集中申报标准 |

## 业务场景分类

### A. 新设投资场景
```
境内主体 → ODI审批 → 设立境外子公司 → 资金汇出
```

### B. 股权并购场景
```
境内主体 → ODI审批 → 收购境外公司股权 → 交割过户
```

### C. 红筹架构场景
```
创始人 → BVI → Cayman → 境内运营公司（VIE）
         ↓
      上市/融资
```

### D. 返程投资场景
```
境外资金 → 外汇登记 → 入境 → 持有境内资产
```

---

## 典型工作流

1. **尽调阶段**：目标公司法律结构、资质证照、劳动合规、诉讼风险
2. **架构设计**：税务优化、股权层级、协议控制(VIE)安排
3. **ODI申报**：商务部/发改委审批、外汇登记
4. **交割执行**：股权变更登记、资金跨境汇划
5. **合规运营**：年度ODI报告、外汇收支申报

## 风险提示

⚠️ **敏感地区/行业**：若投资流向受联合国制裁国家或涉及传媒、互联网等行业，需额外关注。

⚠️ **资金来源证明**：外汇管理部门要求明确资金来源，需提前准备。

⚠️ **返程投资**：VIE架构拆除时需关注外汇登记的补登记问题。
