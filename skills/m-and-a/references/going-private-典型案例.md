# 私有化参考文件2：典型案例分析

## 案例一：万达商业私有化（2016年）

### 基本情况
- **时间**：2016年
- **背景**：万达商业地产在港交所上市后估值偏低
- **方案**：大股东提出要约收购全部H股
- **价格**：每股52.80港元

### 法律程序
1. 大连万达集团提出私有化建议
2. 董事会审议通过
3. 股东大会表决通过
4. 港交所审批
5. 要约收购实施

### 关键法律问题
- 关联交易认定（大连万达与万达商业）
- 独立财务顾问意见
- 股东踯跃率（超过98%）

## 案例二：学大教育私有化（2016年）

### 基本情况
- **时间**：2016年
- **背景**：中概股回归
- **方案**：创始人提出私有化要约
- **价格**：每ADS 5.50美元

### 法律程序
1. 收到私有化提议
2. 特别委员会审议
3. 尽职调查
4. 最终协议达成
5. 股东大会批准

## 案例三：哈尔滨电器私有化

### 法律关注点
- 国有股权转让程序

1. 合理的收购价格
2. 充足的融资安排
3. 良好的公司治理
4. 充分的投资者沟通
5. 有效的审批程序
