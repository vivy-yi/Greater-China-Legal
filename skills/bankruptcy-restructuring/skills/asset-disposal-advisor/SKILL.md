## 一、核心判断框架

### 1.1 资产分类体系

|| 资产类型 | 变现方式 | 处置时限 | 受偿对象 |
|---|---|---|---|---|
| 货币资金 | 直接分配 | 随时 | 按债权顺位 |
| 应收账款 | 催收/转让/核销 | 6-12个月 | 普通债权 |
| 存货 | 变卖/折价分配 | 3-6个月 | 普通债权 |
| 固定资产 | 拍卖/变卖 | 6-12个月 | 担保债权/普通债权 |
| 无形资产 | 评估转让/拍卖 | 6-12个月 | 普通债权 |
| 长期股权投资 | 评估转让/拍卖 | 12个月以上 | 普通债权 |

### 1.2 资产处置程序

```破产财产变价程序：
│
├── 编制变价方案
│   ├── 资产分类清单
│   ├── 评估方法确定
│   └── 处置方式选择
│
├── 变价方案通过
│   ├── 债权人会议表决（如需）
│   └── 法院批准（如涉及担保物）
│
├── 实施变价
│   ├── 评估机构确定
│   ├── 拍卖机构确定（公开拍卖）
│   └── 意向买家征集
│
└── 变现与分配
    ├── 变现款项入账
    └── 按顺位分配
```

### 1.3 处置方式选择

```
资产处置方式决策树：
│
├── 资产类型？
│   ├── 不动产 → 司法拍卖为主
│   ├── 动产 → 拍卖或协议转让
│   ├── 股权 → 评估转让或拍卖
│   └── 无形资产 → 评估转让
│
├── 处置时限要求？
│   ├── 紧急（担保债权到期）→ 加速处置
│   └── 正常 → 公开拍卖
│
├── 市场流动性？
│   ├── 高 → 协议转让
│   └── 低 → 拍卖
│
└── 价值考量？
    ├── 价值稳定 → 拍卖
    └── 价值波动 → 尽快处置
```

## 三、分析流程

### 3.1 资产处置分析步骤

```
第一步：资产分类 → 区分货币类/应收账款/存货/固定资产/无形资产/股权
第二步：权属确认 → 确认有无担保物权属负担
第三步：价值评估 → 分析评估价值与快速变现价值
第四步：变现时序 → 制定处置先后顺序
第五步：方案设计 → 给出各类资产的处置方式建议
第六步：受偿测算 → 计算对各债权人的受偿影响
```

### 3.2 处置优先序分析

|| 优先级 | 资产类型 | 处置理由 |
|---|---|---|---|
| 第一 | 货币资金 | 无需变现，直接分配 |
| 第二 | 易腐存货 | 价值快速贬损 |
| 第三 | 担保物（担保债权） | 优先保障担保债权人 |
| 第四 | 市场流动性好的资产 | 变现成本低 |
| 第五 | 其他资产 | 按序处置 |

### 3.3 快速变现折扣参考

|| 资产类型 | 快速变现折扣 | 说明 |
|---|---|---|---|
| 不动产 | 评估价值70-80% | 市场折价 |
| 上市公司股权 | 市场价格80-90% | 流动性较好 |
| 非上市股权 | 评估价值50-70% | 流动性差 |
| 机动车 | 评估价值60-80% | 折旧快 |
| 机器设备 | 评估价值40-60% | 专业性强者折扣大 |
| 应收账款 | 账龄1年内80%/1-3年50%/3年以上20% | 回收风险 |

## 升级决策门

以下情形应向上级 Skill 升级：

- 涉及担保资产处置（担保物权人同意变价程序复杂）
- 涉及国有资产处置（需履行评估与进场程序）
- 涉及上市公司资产处置（信息披露要求）
- 涉及知识产权、探矿权等特殊资产
- 涉及关联方资产转让（利益冲突审查）
- 涉及资产处置价格显失公平争议

## 本Skill不涵盖

- 资产变价合同的具体条款起草
- 拍卖程序的操作与监督
- 资产处置中的税务规划
- 特定不动产的权属纠纷处理
- 资产处置过程中的诉讼应对
- 破产程序终结后的未变现资产管理
## 本 Skill 不涵盖

- 资产的司法鉴定与评估
- 资产拍卖的具体程序
- 知识产权等专业资产的定价
- 资产处置合同的起草
