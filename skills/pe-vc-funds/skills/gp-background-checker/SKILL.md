## CN GP背景尽职调查

本Skill面向潜在LP（有限合伙人）提供GP尽职调查服务，覆盖GP主体资格、团队背景、历史业绩、合规记录、利益冲突和声誉风险等关键维度。数据来源：中基协AMBERS系统 `[GOV]`、国家企业信用信息公示系统 `[GOV]`、中国裁判文书网 `[WKL]`、企查查/天眼查 `[WKL]`、见微数据 `[BD]`。

## 工作流程

### 第一步：GP主体资格与中基协登记状态核查

**中基协登记状态核查 `[GOV]`：**
- 登记编码、登记日期、登记状态（未登记/已登记/被注销/被撤销/暂停）
- 机构类型（私募股权/私募证券/其他）
- 业务类型（私募股权投资基金/创业投资基金/并购基金/其他）
- 实缴资本（须≥人民币1,000万元）
- 登记高管人数（法定代表+合规风控+至少1名投资经理）
- 法律意见书状态（是否正常）
- 最近一次产品备案时间（如长期未备案新基金——可能已在"僵尸管理人"名单）

**工商信息核查：**
- 法定代表人、注册地址、经营范围（须含"私募投资基金管理"）
- 股权结构（实际控制人穿透到自然人/最终法人）
- 对外投资情况（管理人控股子公司/参股实体）
- 经营异常状态（是否被标记为经营异常、是否列入严重违法名单）
- 股东变更记录（近3年高管的变动频率——如法定代表人、合规风控负责人频繁变更，可能为高风险信号）

**行政处罚/自律处分记录：**
- 中基协自律处分（公开谴责/暂停备案/撤销登记）
- 证监会/证监局行政处罚（包括行政监管措施——如警示函、监管谈话、责令改正）
- 各地金融监督管理局的行政处罚
- 法院判决/仲裁裁决中的负面认定（裁判文书网 `[WKL]`）

### 第二步：GP团队背景与投资能力评估

**核心人员的履历审查：**
- 核心创始人/行业合伙人的投资行业背景（是否与基金投资方向匹配）
- 投资决策委员会（投委会）成员的行业从业年限和投资经验
- 合规风控负责人的从业资格（须持有基金从业资格，且有3年以上合规运营经验）
- 投研团队的规模和人员配置（团队总人数、行业覆盖度——10人以下团队的基金通常投资能力有限）

**历史业绩审查：**
- 已管理基金的数量和总认缴规模
- 已退出项目的数量、IRR、DPI、TVPI、MOIC等核心指标 `[model]`
- 不同基金或不同"vintage year"的业绩一致性
- 明星投资项目（Top Quartile Deals）和亏损项目的分布
- GP自有资金跟投记录（是否对每个项目投入一定比例的GP自有资金——利益绑定信号）

**GP出资能力验证：**
- GP在目标基金中的认缴出资比例（中基协要求GP出资不低于基金认缴总额的**0%**——但实务中常见1-5%）
- GP的自有资本真实来源（是否借贷/是否涉及非法集资）
- 如GP为有限责任公司——其实缴资本能否覆盖潜在的无限连带责任风险

### 第三步：利益冲突与关联交易排查

**潜在的关联交易风险：**
- GP是否同时管理多支投资策略重叠的基金（同一GP旗下基金间的co-investment安排可能存在利益冲突）
- GP高管是否在被投企业中持有个人股权（未披露的话可能涉及老鼠仓）
- GP是否向被投企业收取或接受顾问费/董事费/服务费——如未归入基金利益可能构成利益冲突
- GP与管理人之间的关联交易（如管理人与GP之间的服务合同定价公允性）

**基金之间的交易：**
- GP是否在管理的不同基金之间交叉交易（cross-trading）——即A基金卖出被投企业股份，B基金买入
- 基金之间共享投资机会（allocation of investment opportunities）——是否有公平分配机制

**GP的关联方排查：**
- GP实控人关联的其他实体（可能从事与基金存在竞争或利益冲突的业务）
- GP被投企业之间的相互担保/关联贷款

### 第四步：投资者关系与LP评估

**现有LP的反馈采集：**
- 查看是否可获取现有LP对GP的投资者关系评价（信披质量、沟通频次、LP信息权行使的顺畅度）
- 是否召开年度LP大会（AGM）、季度投资报告会议
- LP咨询事项的回复速度和质量

**LP权利条款审查：**
- LP咨询委员会（LPAC）的组成和决策范围
- LP的信息权（有权查阅基金财务报表、投资组合名单、费用明细）
- 关键人物条款（Key Person Clause）——如GP核心人员离职，LP是否有权暂停出资或终止基金
- LP的退出条款（LP转让合伙权益的条件、GP回购义务等）

**GP对LP的限制：**
- 保密条款（LP不得向第三方披露基金投资组合细节——实务中的冲突点）
- 竞业禁止（LP是否被限制对被投企业的投资或竞争）

### 第五步：声誉风险与舆情监测

**公开信息检索：**
- 执法记录：裁判文书网诉讼记录、证监会处罚记录
- 媒体报道：涉及GP的负面报道、投资者维权报道
- 社交媒体：知乎/微博上的LP维权或投诉
- 行业组织：是否被列入行业协会/仲裁机构/交易所的黑名单、失信被执行人名单

**创投自律排名参考：**
- 清科/投中排名 `[model]`：GP近年在相关赛道中的排名（仅作参考——排名与投资能力不一定正相关）
- Zero2IPO/Preqin等第三方机构对GP的评级

**商业调查：**
- GP是否运营与投资业务无关的商业活动（降低专注度）
- GP的场地情况（是否在新设的共享办公空间运营——可能为资金紧张的信号）

### 第六步：尽职调查报告产出

综合以上五个维度的发现，产出打分矩阵：

| 维度 | 评分（1-5） | 说明 | 风险等级 |
|------|-----------|------|---------|
| 主体资格与合规 |  |  | 🟢/🟠/🔴 |
| 团队与投资能力 |  |  | 🟢/🟠/🔴 |
| 利益冲突 |  |  | 🟢/🟠/🔴 |
| 投资者关系 |  |  | 🟢/🟠/🔴 |
| 声誉风险 |  |  | 🟢/🟠/🔴 |

**总分评级：90分+强力推荐 → 70-89分谨慎推荐 → 50-69分建议增加保护条款 → <50分不建议投资**

## 升级决策门

以下情形须移交专业律师：
- GP存在未披露的重大行政处罚/诉讼/仲裁（历史合规问题）
- GP的实控人/高管有刑事犯罪纪录或在失信被执行人名单
- GP存在严重的未披露利益冲突（如A基金与B基金的利益输送）
- GP同时管理竞争性基金（"盲池"问题——LP无法判断GP的资源分配）
- GP的基金备案被中基协退回3次以上或管理人申请材料被认定为虚假/遗漏
- GP已被列入中基协"异常机构"名单（如未按时报送信息）
- LP已对GP就其信披问题提出书面的异议或争议
