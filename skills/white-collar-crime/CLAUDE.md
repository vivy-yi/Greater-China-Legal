# 白领犯罪法律咨询域 (White-Collar Crime Legal Advisory)

## 一、域概述

### 1.1 域定位
本域专注于中国法项下白领犯罪的预防、识别、应对与程序指引，覆盖企业合规管理及相关刑事法律服务。

### 1.2 子域架构

| 子Skill | 触发指令 | 核心功能 |
|---------|---------|---------|
| fraud-crime-checker | /white-collar-crime:fraud-check | 诈骗罪构成要件分析与风险评估 |
| bribery-crime-advisor | /white-collar-crime:bribery-advisor | 行贿受贿犯罪法律咨询 |
| money-laundering-advisor | /white-collar-crime:ml-advisor | 洗钱犯罪预防与应对 |
| tax-crime-dispatcher | /white-collar-crime:tax-crime | 涉税犯罪案件分流与指引 |
| corruption-compliance-advisor | /white-collar-crime:compliance | 企业腐败合规体系建设 |
| criminal-procedure-guide | /white-collar-crime:procedure | 刑事诉讼程序指引 |

### 1.3 法域说明
本域法律依据为中国大陆法域，适用于中国大陆境内发生的白领犯罪行为及相关合规管理活动。

```

│   ├── 诈骗罪（刑法第266条）

│   ├── 逃税罪（刑法第201条）
```

## 三、工作流程

### 3.1 案件评估流程

```
初步评估
├── 事实收集（行为、金额、主体、后果）
├── 罪名初步匹配
├── 法定刑档次判断
└── 程序阶段识别
        ↓
风险评估（按各子Skill）
        ↓
升级决策（是否需要律师介入）
```

### 3.2 合规咨询流程

```
合规需求
├── 制度建设（制度设计/修订）
├── 风险评估（风险识别/分级）
├── 培训宣导（教育/测试）
└── 调查应对（内部调查/外部应对）
```

## 四、质量控制

### 4.1 升级决策门
以下情形必须升级给执业律师：
1. 涉及具体刑事指控的认定
2. 涉及具体案件的法律策略
3. 涉及正式法律文件的出具
4. 涉及与司法机关的正式沟通
5. 涉及重大经济利益的法律判断

### 4.2 免责声明
本域输出的法律分析仅供参考，不构成正式法律意见。涉及具体法律判断时必须由执业律师复核。
