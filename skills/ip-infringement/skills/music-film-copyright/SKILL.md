## 第一部分：作品与权利类型识别

### 音乐作品侵权类型

| 侵权类型 | 行为描述 | 适用法条 |
|---------|---------|---------|
| 擅自表演 | 未经授权公开表演音乐作品（商场播放/背景音乐） | 著作权法第10条（表演权） |
| 擅自复制 | 未经授权复制音乐作品（刻录/下载/制作铃声） | 著作权法第10条（复制权） |
| 擅自发行 | 未经授权发行音乐制品（盗版CD/数字专辑） | 著作权法第10条（发行权） |
| 擅自信息网络传播 | 未经授权通过网络传播音乐（网盘分享/在线播放） | 著作权法第10条（信息网络传播权） |
| 翻唱侵权 | 主播翻唱/短视频翻唱未获授权 | 著作权法第37条（表演者权） |
| 改编侵权 | 擅自改编音乐作品（remix/二次创作） | 著作权法第10条（改编权） |
| 录音侵权 | 擅自录制他人词曲（未向词曲作者获取授权） | 著作权法第44条 |

### 影视作品侵权类型

| 侵权类型 | 行为描述 | 适用法条 |
|---------|---------|---------|
| 盗版传播 | 未经授权通过网络传播影视作品（盗版网站/网盘） | 著作权法第10条（信息网络传播权） |
| 切条剪辑 | 将影视作品剪辑成短视频传播（西瓜视频/抖音切条） | 著作权法第10条（改编权/汇编权） |
| 盗录 | 电影院偷拍/录屏 | 著作权法第10条（复制权）+反不正当竞争法 |
| 擅自转发 | 未经授权转发影视节目（微信分享/论坛分享） | 著作权法第10条（信息网络传播权） |
| 弹幕侵权 | 视频网站弹幕功能涉及侵权 | 存在争议，须个案分析 |
| 解析侵权 | 技术手段解析VIP内容（视频解析软件） | 著作权法第10条+反不正当竞争法 |
| 盗版周边 | 仿制影视周边（道具/服装/海报） | 著作权法第54条/商标法第57条 |

## 第二部分：权利基础审查

### 2.1 音乐作品权利审查

```
审查顺序：
┌────────────────────────────────────────────┐
│ Step 1：确认词曲作者 → 原始著作权人         │
│   - 自然人：终身+死后50年                  │
│   - 法人：首次发表后50年（合作作品同上）   │
│   - 是否已超过保护期？                    │
├────────────────────────────────────────────┤
│ Step 2：确认权利归属 → 法人作品/职务作品   │
│   - 法人作品：著作权归单位                │
│   - 职务作品：一般归作者，合同约定除外    │
├────────────────────────────────────────────┤
│ Step 3：确认授权链条 → 词→曲→录音→发行    │
│   词曲授权 → 录音制作者授权 → 平台授权    │
│   注意：各环节须分别授权                  │
└────────────────────────────────────────────┘
```

**音乐版权授权链条示例：**
```
词曲权利人 →（词曲授权）→ 唱片公司/录音制作者 →（录音授权）→ 音乐平台
                                                      ↓
                                              用户/主播（使用）
```

### 2.2 影视作品权利审查

```
审查顺序：
┌────────────────────────────────────────────┐
│ Step 1：确认原始权利人 → 出品方/制片方      │
│   - 著作权归制片者（一般为出品方）         │
│   - 保护期：首次发表后50年                 │
├────────────────────────────────────────────┤
│ Step 2：确认权利内容 → 著作权登记证书       │
│   - 著作权登记证书（可选，非必须）         │
│   - 授权链条：出品→发行方→平台→用户        │
├────────────────────────────────────────────┤
│ Step 3：信息网络传播权专项审查             │
│   - 是否取得信息网络传播权授权             │
│   - 授权期限/地域/平台范围                │
│   - 是否限制下载/录屏                     │
└────────────────────────────────────────────┘
```

### 2.3 常见权利瑕疵

| 权利瑕疵类型 | 风险描述 | 影响 |
|-------------|---------|------|
| 授权链条断裂 | 某一环节未获授权 | 侵权风险 |
| 超出授权范围 | 超出约定用途/平台/期限 | 侵权风险 |
| 权利归属不明 | 合作作品/委托作品归属不清 | 诉讼障碍 |
| 已过保护期 | 作品进入公有领域 | 无需授权 |

## 第三部分：侵权判断标准（接触+实质性相似）

### 3.1 著作权侵权判定两步法

```
第一步：接触可能性（Access）
  → 被控侵权人是否有合理机会接触原作品？
  → 公开播放/网络传播/出版物 → 推定有接触可能
  → 内部作品/未发表作品 → 需证明接触可能

第二步：实质性相似（Substantial Similarity）
  → 受著作权保护的部分是否实质性相似？
  → 思想/事实/通用表达 → 不受保护（思想二分理论）
  → 独创性表达（旋律/编曲/剧本/台词）→ 受保护
```

### 3.2 音乐侵权判定要点

| 判断要素 | 内容 | 举证要点 |
|---------|------|---------|
| 接触可能 | 被控侵权人是否有机会接触原曲 | 原曲发布时间早/知名度高→推定接触 |
| 实质性相似 | 旋律/和弦进行/编曲是否相似 | 专家鉴定/听感对比 |
| 独创性表达 | 受保护的是音符而非思想 | 相似部分是否为独创性表达 |
| 比例问题 | 相似部分占原曲的比例 | 片段相似也可能构成侵权 |

**注意：** 同一旋律不同编曲、改变速度/调性仍可能构成侵权（接触+实质性相似）。

### 3.3 影视侵权判定要点

| 判断要素 | 内容 | 适用场景 |
|---------|------|---------|
| 内容相似 | 剧情/人物关系/台词/场景是否相似 | 剧本/小说改编 |
| 切条侵权 | 将长视频剪辑成短视频传播 | 短视频平台 |
| 盗录 | 电影院偷拍/录屏 | 盗版源头 |
| 深度链接 | 设链行为是否超出合理范围 | 视频网站 |

### 3.4 侵权判断树

```
┌─────────────────────────────────────────────────────────┐
│ 开始：权利人主张被侵权                                    │
├─────────────────────────────────────────────────────────┤
│ Step 1：权利基础审查                                     │
│   ├─ 有著作权登记/授权文件 → 权利基础✅                 │
│   └─ 无登记/授权文件 → 补充举证（创作痕迹/发表记录）     │
├─────────────────────────────────────────────────────────┤
│ Step 2：接触可能性                                       │
│   ├─ 原作品已公开发布/传播 → 推定接触可能✅             │
│   └─ 未公开发布 → 需举证证明接触可能                    │
├─────────────────────────────────────────────────────────┤
│ Step 3：实质性相似                                       │
│   ├─ 独创性表达高度相似 → 侵权可能性高                  │
│   ├─ 思想/通用表达相似 → 侵权可能性低                   │
│   └─ 相似部分比例极低 → 不构成侵权                      │
├─────────────────────────────────────────────────────────┤
│ 结论：                                                   │
│   权利基础✅ + 接触可能✅ + 实质性相似✅ → 构成侵权     │
│   权利基础❌ 或 接触可能❌ → 不构成侵权                  │
└─────────────────────────────────────────────────────────┘
```

## 第四部分：证据收集与固定

### 4.1 音乐侵权证据

| 证据类型 | 固定方式 | 证明内容 |
|---------|---------|---------|
| 原版作品 | 正版专辑/官方发布记录 | 作品内容/发表时间 |
| 侵权作品 | 录音/录像/截图 | 侵权行为 |
| 侵权获利 | 平台数据/销售记录 | 赔偿计算依据 |
| 接触可能 | 作品知名度/发布时间 | 侵权故意 |

### 4.2 影视侵权证据

| 证据类型 | 固定方式 | 证明内容 |
|---------|---------|---------|
| 正版来源 | 平台会员证明/购买记录 | 授权合法 |
| 侵权平台 | 录屏/截图/时间戳 | 侵权行为/传播量 |
| 侵权截图 | 公证处公证/时间戳认证 | 证据效力 |
| 技术鉴定 | 委托鉴定机构 | 实质性相似判断 |

### 4.3 公证取证注意事项

```
证据固定优先顺序：
┌──────────────────────────────────────────────────────┐
│ 1. 公证取证（最强证据效力）                           │
│    → 当地公证处申请                                   │
│    → 适用于网络侵权/实体侵权                         │
│    → 费用：1000-5000元/件                            │
├──────────────────────────────────────────────────────┤
│ 2. 时间戳认证（成本低、速度快）                       │
│    → 联合信任时间戳服务中心                           │
│    → 适用于电子证据                                  │
│    → 费用：约100元/件                                │
├──────────────────────────────────────────────────────┤
│ 3. 电子存证（区块链/云存证）                         │
│    → 第三方平台存证                                   │
│    → 便捷但效力存在争议                               │
└──────────────────────────────────────────────────────┘
```

## 第五部分：维权路径选择

### 5.1 四种维权路径对比

| 路径 | 成本 | 时间 | 效果 | 适用情形 |
|------|------|------|------|---------|
| 协商和解 | 低 | 快（1-2周） | 弱 | 侵权轻微、金额不大 |
| 行政投诉 | 低~中 | 快（2-4周） | 中 | 著作权侵权、侵权明确 |
| 民事诉讼 | 高 | 长（6-18个月） | 强 | 损失较大、须赔偿 |
| 刑事报案 | 中高 | 中（3-6个月） | 最强 | 情节严重、达到追诉标准 |

### 5.2 维权路径选择树

```
┌─────────────────────────────────────────────────────────────┐
│ 第一步：评估侵权严重程度                                      │
│   ├─ 批量传播/恶意侵权/营利目的 → 刑事报案优先               │
│   └─ 单次/轻微/非营利 → 民事诉讼或协商                       │
├─────────────────────────────────────────────────────────────┤
│ 第二步：评估证据充分性                                        │
│   ├─ 证据充分（公证+侵权截图+获利证明）→ 直接诉讼             │
│   └─ 证据不足 → 先行政投诉/公证固定证据                       │
├─────────────────────────────────────────────────────────────┤
│ 第三步：评估时效                                              │
│   ├─ 民事诉讼时效：3年（知道或应当知道起算）                 │
│   └─ 刑事追诉时效：依罪名不同（参见刑法第87条）               │
├─────────────────────────────────────────────────────────────┤
│ 第四步：评估赔偿可行性                                        │
│   ├─ 侵权获利明确 → 选择侵权获利赔偿                         │
│   ├─ 实际损失明确 → 选择实际损失赔偿                         │
│   └─ 两者均难证明 → 参照许可费或法定赔偿                     │
└─────────────────────────────────────────────────────────────┘
```

```

│ 1. 准备材料                                 │
│    - 权利证明（著作权登记证/授权文件）      │
│    - 侵权证据（公证文书/录屏截图）          │
│    - 投诉申请书                             │
├────────────────────────────────────────────┤
│ 2. 提交投诉                                 │
│    - 线上：版权局官网/全国12315平台         │
│    - 线下：侵权行为地/结果地版权局           │
├────────────────────────────────────────────┤
│ 3. 行政处理                                 │
│    - 责令停止侵权                           │

├────────────────────────────────────────────┤
│ 4. 如不服行政处理 → 行政复议或行政诉讼      │
└────────────────────────────────────────────┘
```

```
│ 3. 诉讼程序                                 │
│    - 举证期（30日）                         │
│    - 开庭审理                               │
│    - 调解（如愿意）                         │
├────────────────────────────────────────────┤
│ 4. 判决执行                                 │
│    - 胜诉后申请强制执行                     │
│    - 执行不能→债权凭证                     │
└────────────────────────────────────────────┘
```

### 5.5 刑事报案流程

```

  - 非法经营额≥5万元

┌────────────────────────────────────────────┐
│ 1. 准备材料                                 │
│    - 权利证明                               │
│    - 侵权证据（公证+审计报告）              │
│    - 审计报告（侵权经营额/违法所得）         │
├────────────────────────────────────────────┤
│ 2. 向公安机关报案                           │
│    - 侵权行为地/结果地公安机关               │
│    - 提交报案材料                           │
├────────────────────────────────────────────┤
│ 3. 立案侦查                                 │
│    - 公安机关审查                           │
│    - 立案后侦查                             │
├────────────────────────────────────────────┤
│ 4. 移送检察院审查起诉 → 审判               │
└────────────────────────────────────────────┘
```

## 第六部分：赔偿计算

### 6.1 著作权侵权赔偿顺序（著作权法第54条）

| 顺序 | 计算方式 | 说明 |
|------|---------|------|
| 1 | 实际损失 | 权利人实际损失 |
| 2 | 侵权获利 | 侵权人获得的利益 |
| 3 | 参照许可费 | 参照许可费的倍数（通常1-3倍） |
| 4 | 法定赔偿 | 500元～500万元（2021年修法后） |

### 6.2 赔偿计算要点

```
实际损失计算：
  - 音乐：专辑销量减少×单价 / 许可费损失
  - 影视：点播量减少×单价 / 会员流失损失

侵权获利计算：
  - 平台：广告收入+会员转化
  - 盗版网站：广告收入（需审计）

法定赔偿适用条件：
  - 无法证明实际损失
  - 无法证明侵权获利
  - 也无法参照许可费
```
