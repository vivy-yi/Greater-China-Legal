# /trademark-search-report — 商标注册前检索分析

## 加载上下文

首次使用时，读取 `../CLAUDE.md` 获取商标侵权判断标准和维权路径选择逻辑。

## 核心概念

商标注册前的近似检索是降低驳回风险的关键步骤。中国商标局对近似商标的审查标准主要依据《商标审查及审理标准》，从音（呼叫）、形（外观）、义（含义）三个维度综合判断。尼斯分类的选择直接影响保护范围——选择过窄可能出现保护盲区，选择过宽可能遇到在先权利障碍或面临三年不使用撤销风险。

## 工作流程

### 第一步：商标信息确认与初步分析

**商标要素分析**：

| 要素 | 审查要点 | 注册难度 |
|------|---------|---------|
| 文字商标（中文）| 汉字组合、含义、呼叫 | 组合越独特越易注册 |
| 文字商标（英文）| 拼写、含义、与中文音译/意译关系 | 非通用词汇更易注册 |
| 图形商标 | 图形构成、线条、色彩（黑白注册保护彩色使用）| 越具独创性越易注册 |
| 组合商标 | 文字+图形的整体近似判断 | 整体判断，但各部分单独审查 |
| 立体商标 | 三维形状是否具有显著性 | 难度较高，须获得第二含义 |
| 声音商标 | 声音是否具有识别性 | 难度高，须提供实际使用证据 |

**绝对驳回理由分析**（商标法第10-12条）：

| 驳回理由 | 法律依据 | 可克服性 |
|---------|---------|---------|
| 缺乏显著特征 | 第9条/第11条 | 可通过使用获得第二含义克服 |
| 通用名称 | 第11条第1款第1项 | 不可克服 |
| 仅直接表示商品质量/原料等 | 第11条第1款第2项 | 可通过使用获得第二含义 |
| 违反公序良俗 | 第10条第1款 | 不可克服 |
| 与国徽/国旗等相同/近似 | 第10条第1款第1项 | 不可克服 |
| 带有欺骗性 | 第10条第1款第7项 | 不可克服 |

### 第二步：近似商标检索

**近似判断标准**（商标审查及审理标准）：

| 判断维度 | 审查标准 | 近似示例 |
|---------|---------|---------|
| 音（呼叫）| 呼叫相同或近似 | "ABC" vs "ABBC" |
| 形（外观）| 整体外观近似，视觉印象混淆 | "茅台" vs "矛台"（近似字）|
| 义（含义）| 含义相同或关联性强 | "苹果" vs "APPLE"（中英含义相同）|

**商品/服务类似判断**：
- 尼斯分类共45类（1-34类为商品，35-45类为服务）
- 同类别内的商品/服务不必然类似——须根据《类似商品和服务区分表》判断
- 跨类别保护须商标达到驰名程度

**检索流程**：
```
第一步：精确检索
  ├── 检索完全相同商标（同一类别）
  ├── 检索近似商标（音/形/义）
  └── 检索类似商品/服务上的近似商标

第二步：扩展检索
  ├── 检索跨类别的相同/近似商标
  └── 检索相近主体名称的商标

第三步：法律状态核实
  ├── 确认在先商标是否有效（续展期内）
  ├── 确认在先商标是否已被撤销/无效
  └── 确认在先商标是否处于异议/争议中
```

### 第三步：驳回风险评估

**驳回风险等级**：

| 风险等级 | 发现 | 成功率 | 建议 |
|---------|------|-------|------|
| 高风险（>60%）| 存在相同或高度近似的在先注册商标 | <40% | 修改商标或增加显著要素 |
| 中风险（30-60%）| 存在部分近似的在先商标 | 40-70% | 通过限定商品/服务范围降低风险 |
| 低风险（<30%）| 无相同或高度近似商标 | >70% | 按计划申请 |

**常见驳回情形**：

| 驳回类型 | 说明 | 应对策略 |
|---------|------|---------|
| 相对驳回（在先权利冲突）| 与在先商标构成近似+商品类似 | ①提交使用证据证明市场区分 ②限定商品范围 ③与在先商标权人协商共存协议 ④申请驳回复审 |
| 绝对驳回（缺乏显著特征）| 描述性词汇/通用名称 | ①申请驳回复审+提交使用证据 ②修改商标增加独创性元素 ③重新申请组合商标 |

### 第四步：商标布局建议

**防御性注册策略**：

| 布局策略 | 说明 | 适用情形 |
|---------|------|---------|
| 核心类别注册 | 在主营业务对应的尼斯分类中注册 | 所有申请人 |
| 关联类别注册 | 在上下游产品或互补服务类别中注册 | 有一定规模和品牌知名度的企业 |
| 防御类别注册 | 在可能产生混淆的全部类别中注册（全类注册）| 知名品牌/驰名商标权利人 |
| 联合商标注册 | 注册与主商标近似的多个商标（如"老干妈""老干爹"等）| 防止他人恶意注册近似商标 |
| 国际商标注册 | 通过马德里体系或单一国家注册 | 有跨境经营计划的企业 |

**建议申请方案**：

| 企业类型 | 建议申请类别 | 预算参考 |
|---------|------------|---------|
| 初创企业 | 核心类别1-2类 | 1,000-3,000元 |
| 成长型企业 | 核心+关联3-5类 | 3,000-8,000元 |
| 品牌型企业 | 核心+关联+防御5-10类 | 8,000-20,000元 |
| 大型知名企业 | 全类45类+联合商标 | 50,000元以上 |

### 第五步：申请策略与时间线

**商标注册流程时间线**：

```
提交申请 → 形式审查（1个月）→ 实质审查（6-9个月）
         ↓                    ↓
     补正或不受理          驳回或初步审定
                                 ↓
                             公告期（3个月）
                                 ↓
                             无人异议→核准注册→颁发商标证（约12个月）
                              ↓
                             有人异议→异议程序（12-18个月）
```

**策略建议**：
- 使用前先检索：在投入商业使用前完成检索，避免投入使用后发现侵权
- 尽早申请：中国采用"先申请"原则，晚一天可能丧失权利
- 保留使用证据：申请时可能不需要，但在驳回复审中至关重要

### 第六步：升级决策门

> "商标检索分析和申请策略涉及商标法审查标准、近似判断和商业布局的综合判断。以下情形建议委托专业商标代理机构：(1)涉及商标驳回复审或异议程序；(2)在先权利冲突涉及知名商标或驰名商标保护；(3)商标布局涉及全类注册或防御商标策略设计；(4)申请商标包含地名/人名/外国名称等特殊元素可能触发绝对驳回；(5)申请人有跨境商标保护需求（马德里/单一国家）。本分析为初步检索评估框架，不构成正式法律意见。最终商标注册的可行性和审查结果须以商标局审查决定为准。"
