# /portfolio — China Mainland

## CN IP续展时间线

### 商标（商标法第38-40条）

| 时间节点 | 行动 |
|---|---|
| 到期日前12个月 | 可提出续展注册申请 |
| 到期日前6个月 | 宽展期（须支付延迟费）|
| 到期日 | 注销 |
| 注销后6个月内 | 可恢复（须有正当理由）|

**有效期：** 10年，自核准注册之日起计算

---

### 专利（专利法第43-44条）

**发明专利：**
| 年费 | 缴纳时间 |
|---|---|
| 1-3年 | 每年申请日前 |
| 4-6年 | 每年申请日前 |
| 7-9年 | 每年申请日前 |
| 10-20年 | 每年申请日前 |

**实用新型专利（10年）：** 每年申请日前缴纳年费

**外观设计专利（15年）：** 每年申请日前缴纳年费

---

## CN年费标准（参考）

| 年度 | 发明专利 | 实用新型/外观设计 |
|---|---|---|
| 1-3年 | 900元/件 | 600元/件 |
| 4-6年 | 1200元/件 | 900元/件 |
| 7-9年 | 2000元/件 | 1200元/件 |
| 10年+ | 4000元/件 | 2000元/件 |

---

## 使用声明（商标法第49条）

**撤销风险：**
- 注册商标连续3年无正当理由不使用，任何人可申请撤销
- 须保留使用证据（合同/发票/包装/广告）

**使用证据类型：**
- 合同+发票
- 产品包装
- 广告材料
- 展会参展记录
