# /patent-validity-checker — 专利有效性核查

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取专利法规/无效宣告规则。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取专利类型/有效期规则
2. 核查专利基本信息（号/类型/权利人）
3. 核查专利有效性（有效期/年费/无效宣告）
4. 评估是否可被无效
5. 输出有效性评估

## 专利类型与有效期

| 类型 | 保护期 | 审查方式 |
|---|---|---|
| 发明专利 | 20年 | 实质审查 |
| 实用新型专利 | 10年 | 形式审查 |
| 外观设计专利 | 15年（2021年修法） | 形式审查 |

## 有效性核查清单

```
第一步：专利基本信息
  → 专利号/专利名称/类型/权利人/申请日

第二步：是否在保护期内
  → 发明：申请日+20年
  → 实用新型：申请日+10年
  → 外观设计：申请日+15年
  → 超期则无效

第三步：年费是否缴纳
  → 未缴年费 → 专利权终止
  → 查询国家知识产权局专利年费缴纳记录

第四步：是否被无效
  → 查询无效宣告程序记录
  → 生效的无效决定 → 专利无效

第五步：可无效事由（如须反诉）
  → 缺乏新颖性/创造性
  → 说明书不支持权利要求
  → 修改超出原范围
  → 不具备实用性
```

## 查询路径

```
[GOV] 国家知识产权局：https://www.cnipa.gov.cn/
  专利检索：https://pss-system.cponline.cnipa.gov.cn/

[GOV] 中国专利公布公告：http://epub.sipo.gov.cn/
```

## 无效宣告程序

| 要点 | 说明 |
|---|---|
| 提起方式 | 向国家知识产权局专利复审委员会提交无效宣告请求 |
| 举证责任 | 请求方负举证责任 |
| 常见理由 | 新颖性/创造性/说明书不支持 |
| 效力 | 无效决定生效后，专利权视为自始不存在 |
