# /takedown — China Mainland

## 两种模式

- `--send` — 起草侵权通知（主张权利）
- `--respond` — 分析收到的侵权通知（防御方）
- `--counter` — 起草反通知（被投诉方）

---

## CN著作权侵权通知制度

**法律依据：**《信息网络传播权保护条例》（2006年，2013年修订）

**通知要素：**
1. 权利人身份证明
2. 侵权作品/内容的具体信息
3. 构成侵权的初步证明材料
4. 通知人真实身份信息

**平台义务：** 接到通知后及时删除/屏蔽侵权内容，并通知用户

---

## CN平台类型与通知路径

| 平台类型 | 通知路径 |
|---|---|
| 电商平台（淘宝/京东）| 知识产权保护平台 |
| 视频平台（优酷/腾讯视频）| 侵权投诉入口 |
| 社交媒体（微信/微博）| 侵权投诉入口 |
| 应用商店（App Store/华为应用市场）| 知识产权投诉 |
| 搜索引擎（百度）| 侵权投诉入口 |

## Send模式：起草侵权通知

### Step 1: 确认权利基础

询问：
- 作品类型：文字/图片/视频/音乐/软件
- 登记号（如有）：[编号]
- 首次发表时间：[日期]
- 侵权内容URL：[链接]

### Step 2: 侵权事实

- 侵权内容：[描述]
- 发现方式：[平台名称]
- 发现日期：[日期]

### Step 3: 起草通知

**CN侵权通知须包含：**
1. 权利人姓名/名称 + 联系方式
2. 侵权内容具体信息（URL/截图）
3. 构成侵权的初步证明（作品截图+发布时间）
4. 通知人签字/盖章

---

## Respond模式：分析侵权通知

**分析维度：**
1. **权利有效性** — 通知方是否为著作权人或其代理人？
2. **侵权指控是否成立** — 是否确实构成侵权？
3. **通知是否合规** — 是否缺少法定要素？
4. **抗辩理由** — 合理使用/已获授权/不存在侵权

**合理使用抗辩（《著作权法》第24条）：**
- 为个人学习、研究或欣赏
- 为介绍、评论某一作品或说明某一问题
- 为报道时事新闻
- 国家机关为执行公务在合理范围内使用

---

## Counter模式：起草反通知

**前提条件：**
- 收到通知后内容已被删除/屏蔽
- 确信是错误删除（未侵权）
- 准备应诉或接受法院管辖

**CN反通知须包含：**
1. 反通知人姓名/名称 + 联系方式
2. 被删除/屏蔽的内容及其位置
3. 不构成侵权的初步证明
4. 反通知人真实身份信息
