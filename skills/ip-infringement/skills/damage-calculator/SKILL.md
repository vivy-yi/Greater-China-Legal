# /damage-calculator — IP侵权损害赔偿计算

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取赔偿计算规则/法定赔偿标准。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取赔偿计算规则/各IP类型标准
2. 收集已知数据
3. 按四种方式依次计算
4. 确定合理赔偿金额
5. 输出计算过程

## 赔偿计算四步法

### 方式一：实际损失

```
公式：权利人损失 = 减少的销售量 × 单件利润
条件：须证明侵权与损失之间的因果关系
注意：须有完整销售数据支持
```

### 方式二：侵权获利

```
公式：侵权获利 = 侵权产品销售量 × 单件利润
条件：须提供侵权人财务数据（须申请证据保全）
替代：侵权人财务报表/税务数据/电商平台后台数据
```

### 方式三：参照许可费

```
公式：赔偿金额 = 合理许可费 × 侵权期间倍数
条件：须有相同/类似许可协议作为参照
倍数：通常1-3倍，考虑侵权情节
```

### 方式四：法定赔偿（兜底）

| IP类型 | 2023年前 | 2023年后 |
|---|---|---|
| 商标 | 300万元以下 | 500万元以下 |
| 专利 | 100万元以下 | 500万元以下 |
| 著作权 | 50万元以下 | 500万元以下 |
| 商业秘密 | 300万元以下 | 500万元以下 |

## 各IP类型赔偿特点

| 类型 | 特点 |
|---|---|
| 商标 | 考虑品牌知名度/侵权持续时间 |
| 专利 | 发明专利通常高于实用新型/外观设计 |
| 著作权 | 赔偿较低，法定赔偿上限500万 |
| 商业秘密 | 须证明实际损失，难以量化时参照许可费 |

## 惩罚性赔偿（2019年修法）

```
适用条件（须同时满足）：
1. 故意侵权
2. 情节严重
3. 权利人请求

计算：实际损失/侵权获利/许可费的1-5倍

举证：须证明故意+情节严重
```
