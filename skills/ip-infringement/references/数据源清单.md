## 三、案例库（IP 侵权场景特化）

### 3.1 最高人民法院指导案例 / 公报案例

```
URL: 人民法院案例库 https://rmfyalk.court.gov.cn
[Y D] 检索："指导案例 + IP 类型"

IP 必学指导案例：
  - 商标近似判断（多个指导案例）
  - 专利等同原则
  - 著作权合理使用
  - 商业秘密构成要件
  - 损害赔偿计算
  - 标准必要专利（FRAND 原则）
```

## 七、多源交叉验证

**IP 关键结论须 2 个数据源确认**：

| 结论类型 | 必须验证的源 |
|---|---|
| 商标可注册性 | [GOV]CNIPA + [BD]商标数据库 |
| 专利有效性 | [GOV]CNIPA + [BD]商用数据库 |
| 侵权认定 | [YD]案例 + [WKL]综合 |
| 赔偿金额 | [YD]案例 + 法院判赔标准 |
| 开源许可证 | SPDX + OSI + 项目 LICENSE 文件 |
| 商业秘密构成 | [YD]案例 + [GOV]反不正当竞争法司法解释 |

**不一致时输出**：
> ⚠️ 来源冲突：[YD] 说是 A，[WKL] 说是 B。建议知识产权律师复核。

## 八、来源标注规范

每个结论/数字必须挂载来源标签：

| 标注 | 含义 | IP 场景示例 |
|---|---|---|
| `[YD]` | 元典检索结果 | `[YD] 商标法第 57 条` |
| `[WKL]` | 无合力检索结果 | `[WKL] 类案 XX 号` |
| `[BD]` | 北达/商用数据库 | `[BD] PatSnap/申请号 XX` |
| `[GOV]` | 政府平台数据 | `[GOV] CNIPA 注册号 XX` |
| `[web]` | 网络搜索结果 | `[web] WIPO 国际检索` |
| `[model]` | 模型知识（须核实） | 不允许直接引用，须律师审核 |
| `[user]` | 用户提供的文件 | `[user] 客户提供的专利证书` |
| `[settled — last confirmed YYYY-MM-DD]` | 已验证的稳定引用 | 已审核引用 |

**IP 场景特殊规则**：
- 专利/商标号：`[GOV] CNIPA + 申请号/注册号`
- 开源许可证：`SPDX + 许可证 ID`
- 案例引用：`[YD]（年份）法院代码 + 案号 + 裁判要旨`
- 学术观点：`[BD] 知网 + 卷期 + 页码`

## 十、IP 风险等级说明

ip-infringement 风险等级主要由 skill 内部设定：

| skill | 风险等级 | 原因 |
|---|---|---|
| infringement-triage / infringement-detector | high | 涉及侵权认定 |
| infringement-evidence-collection / evidence-guide | medium | 证据收集 |
| rights-protection-path | high | 维权路径涉及诉讼/刑事 |
| damage-calculator | high | 赔偿金额重大 |
| clearance / trademark-search | medium | 注册前评估 |
| brand-license-contract / ip-clause-review | medium | 合同审查 |
| oss-review | high | GPL 传染风险 |
| patent-claim-analysis / patent-validity-checker | high | 涉及专利效力 |
| trade-secret-litigation | high | 刑事风险 |
| portfolio | low | 资产管理 |
| takedown | medium | 平台投诉 |

**触发升级**：
- 涉外 IP → 律师专项
- 标准必要专利 → 律师专项
- 商业秘密 → 律师专项
- 刑事风险 → 律师专项
- 涉及上市公司 → 律师 + 合规专项
