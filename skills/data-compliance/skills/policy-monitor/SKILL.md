# /policy-monitor — China Mainland

## 监控范围

### 核心法规

| 法规 | 主管机关 | 监控关键词 |
|---|---|---|
| 个人信息保护法（PIPL）| 国家网信办 | 个人信息保护法、实施条例、司法解释 |
| 数据安全法（DSL）| 国家网信办 | 数据安全法、数据安全管理办法 |
| 网络安全法（CSL）| 公安部/网信办 | 网络安全法、网络安全审查 |
| GB/T 35273-2020 | 网信办参考 | 个人信息安全规范、更新 |
| 个人信息跨境规定 | 国家网信办 | 个人信息出境、标准合同、认证 |
| App违法违规收集使用个人信息 | 工信部/网信办 | App监管、违规通报、下架 |
| 汽车数据安全管理 | 网信办等 | 智能网联汽车、数据境内存储 |

## 工作流程

### 定期检查

**默认检查周期：** 最近30天

**检查内容：**
1. 国家网信办官网最新公告
2. 工信部App监管通报
3. 公安部网络安全通报
4. 相关国家标准发布（GB/T）
5. 地方网信办规范性文件（如上海市网信办、广东省网信办）

### 关键词监控

| 关键词 | 触发场景 |
|---|---|
| 个人信息保护法 修订 | PIPL重大修订 |
| 个人信息出境 安全评估 | 跨境传输合规 |
| App 违规 通报 | App合规检查 |
| 数据安全 管理办法 | DSL配套规定 |
| 汽车 数据 境内存储 | 智能汽车合规 |
| 生成式AI 服务 管理 | AI数据合规 |
| 人脸识别 收集 | 生物识别合规 |

## 本技能不做什么

- 不提供法律建议。只汇总公开信息。
- 不保证信息完整性。须结合官方渠道核实。
- 不替代专业法律咨询。
