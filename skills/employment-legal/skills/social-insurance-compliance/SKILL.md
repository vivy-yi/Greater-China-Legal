# 社会保险缴纳合规咨询

## 目的

本技能为用人单位提供社会保险缴纳的合规指导，涵盖五险（养老/医疗/工伤/失业/生育）一金（住房公积金）的基数确定、比例计算、合规要求和补缴路径。

> 社会保险是中国劳动法体系中的强制性制度。用人单位不得以任何形式约定不缴或少缴社保——即使员工主动要求不缴社保，该约定也因违反法律强制性规定而无效。

## 法域假设

默认中国大陆法域，适用《社会保险法》（2011）、《住房公积金管理条例》、《社会保险费征缴暂行条例》及相关规定 `[SME 核查]`。各地社保基数上下限和比例存在差异，须以当地社保部门公布数据为准。

## 加载信息

- 用人单位所在地（省/市/区）
- 员工类型（城镇职工/农村户籍/外籍人员/灵活就业人员）
- 员工上一年度月平均工资
- 当地上年度职工月平均工资（用于确认基数上下限）
- 当前社保缴纳基数和比例
- 是否与员工有"不缴社保"的约定（如有，须特别关注）

## 工作流程

### 第一步：社保基数确认 `[SME 核查]`

**基数确定规则**（社会保险法第12条、第35条）：

```
缴纳基数 = 员工本人上一年度月平均工资
          （包括基本工资+奖金+津贴+补贴等所有货币性收入）

基数调整：
- 基数 < 当地社平工资60% → 按60%执行（下限）
- 基数 > 当地社平工资300% → 按300%执行（上限）
- 基数在60%-300%之间 → 按实际工资执行
```

**各地基数上下限示例**（以2025-2026年度典型城市为例，实际以当地公布为准 `[GOV]`）：

| 地区 | 社平工资（月） | 60%下限 | 300%上限 |
|------|--------------|---------|---------|
| 上海 | 约12,183元 | 约7,310元 | 约36,549元 |
| 北京 | 约11,000元 | 约6,600元 | 约33,000元 |
| 深圳 | 约10,000元 | 约6,000元 | 约30,000元 |
| 成都 | 约7,500元 | 约4,500元 | 约22,500元 |

### 第二步：社保比例确认 `[GOV]`

**全国参考比例**（各地略有差异，以当地社保部门数据为准）：

| 险种 | 单位比例（约） | 个人比例（约） | 说明 |
|------|-------------|-------------|------|
| 基本养老保险 | 16% | 8% | 全国基本统一 |
| 基本医疗保险 | 8%-10% | 2%+3元 | 各地差异较大 |
| 失业保险 | 0.5%-0.8% | 0.2%-0.5% | 视地区浮动 |
| 工伤保险 | 0.2%-1.9% | 0% | 按行业风险分类 |
| 生育保险 | 0.5%-1% | 0% | 部分地区已并入医保 |

**住房公积金**（公积金管理中心规定）：
- 单位缴存比例：5%-12%
- 个人缴存比例：5%-12%
- 缴存基数上下限：同社保原则

> **注意**：自2020年起，多地已将生育保险和基本医疗保险合并实施，但依然分别计算缴费基数。

### 第三步：合规评估 `[SME 核查]`

**常见不合规情形**：

| 情形 | 是否合规 | 法律风险 |
|------|---------|---------|
| 试用期不缴社保 | ❌ 违法 | 劳动合同法第38条——员工可要求解除合同并主张经济补偿 |
| 按最低基数缴纳（实际工资高于最低基数）| ❌ 少缴 | 社保稽核可责令补缴差额+滞纳金 |
| 员工书面同意不缴社保 | ❌ 无效约定 | 违反强制性规定，无效，仍须补缴 |
| 只缴部分险种 | ❌ 违法 | 五险须同时缴纳（部分群体除外） |
| 按实际工资足额缴纳 | ✅ 合规 | — |
| 按上限缴纳（工资超上限）| ✅ 合规 | 超上限部分不计缴，社保待遇也受上限限制 |

**试用期社保义务**：
- 劳动关系自用工之日起建立，试用期包含在劳动合同期内
- 用人单位须自用工之日起30日内办理社保登记
- 试用期不缴社保属于违法行为
- 部分地区允许在次月补缴（如上海），但须在法定期限内

### 第四步：未缴/少缴的法律后果 `[SME 核查]`

| 后果类型 | 具体影响 |
|---------|---------|
| 员工解除权 | 员工可依据劳动合同法第38条以"未依法缴纳社会保险"为由解除合同，并主张经济补偿金（N） |
| 社保稽核补缴 | 社保部门可要求补缴少缴/未缴的全部社保费用，按日加收万分之五滞纳金 |
| 行政处罚 | 劳动监察部门可责令限期改正并处罚款 |
| 工伤/医疗待遇 | 未缴社保期间发生工伤/医疗费用的，由用人单位承担全部工伤/医疗待遇 |
| 公积金补缴 | 住房公积金管理中心可责令补缴，并可申请法院强制执行 |
| 信用影响 | 社保违规可能影响企业信用记录、招投标资格、上市合规 |

**补缴路径**：

| 路径 | 适用情形 | 程序 |
|------|---------|------|
| 补缴（3个月以内）| 短期漏缴/迟缴 | 提供补缴申请+劳动关系证明 |
| 补缴（1年以内）| 少缴基数 | 社保稽核申请+工资证明 |
| 补缴（1年以上）| 长期未缴 | 须经社保稽核，可能须提供完整劳动关系证明 |
| 投诉举报 | 用人单位拒不补缴 | 向社保部门/劳动监察投诉 |

### 第五步：特殊情形

**外籍员工社保**：
- 2011年《在中国境内就业的外国人参加社会保险暂行办法》生效
- 外籍员工须参加养老、医疗、工伤、失业四险
- 与母国签有社保双边协议的可申请免缴

**农村户籍员工**：
- 养老保险：农村户籍与城镇户籍统一适用
- 失业保险：部分农村户籍人员享受失业待遇有特殊规定

**灵活就业人员**：
- 可自愿参加养老和医疗保险（不能参加工伤/失业/生育保险）
- 基数由个人选择，可在上下限范围内自定

### 第六步：升级决策门

> "社保缴纳是用人单位法定义务，不合规缴纳的法律后果包括员工解除权、补缴滞纳金、行政处罚和民事赔偿。本分析为合规指导，不构成法律意见。以下情形建议委托专业律师：(1)社保稽核和行政处罚应对；(2)员工因社保问题解除合同的经济补偿争议；(3)工伤待遇纠纷中涉及未缴社保的赔付问题；(4)社保基数调整方案设计；(5)涉及外籍员工社保双边协议的适用。"
