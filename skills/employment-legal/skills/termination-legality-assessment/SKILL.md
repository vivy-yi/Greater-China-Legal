# 劳动合同解除合法性评估

## 目的

本技能为用人单位提供劳动合同解除的全流程合法性评估。劳动解除是中国劳动争议中风险最高、金额最大、败诉率最高的领域。程序瑕疵、事实依据不足、法律适用错误都可能导致违法解除认定。

> 北京和上海等地的司法实践中，用人单位的解除行为一旦被认定为违法解除，面临向员工支付2N赔偿金或恢复劳动关系的双重风险。程序合规比实体正确更为重要——程序瑕疵足以推翻实体正确。

## 法域假设

默认中国大陆法域，适用《劳动合同法》（2012年修正）及最高院司法解释一 `[SME 核查]`。各省市裁判口径存在差异，具体操作中须结合当地实践。

## 加载信息

- 当事人角色（用人单位/员工）
- 解除原因（协商/过失性/非过失性/经济性裁员）
- 员工基本信息（入职日期、岗位、月平均工资、是否签有无固定期限合同）
- 是否存在高风险情形（孕期/产期/哺乳期、医疗期、工伤、连续工作满15年等）
- 用人单位规章制度是否经过民主程序并公示
- 是否已通知工会

## 工作流程

### 第一步：确认解除类型与法律依据 `[SME 核查]`

**五大解除类型对照表**：

| 解除类型 | 法律依据 | 是否需提前通知 | 经济补偿 | 适用场景 |
|---------|---------|--------------|---------|---------|
| 协商解除（单位提出）| 第36条 | 否 | N | 双方自愿，最常见 |
| 协商解除（员工提出）| 第36条 | 是（30日）| 0 | 员工主动辞职 |
| 过失性解除 | 第39条 | 否 | 0 | 员工严重违纪等 |
| 非过失性解除 | 第40条 | 是（30日或代通知金）| N | 不能胜任/医疗期满等 |
| 经济性裁员 | 第41条 | 是（30日）| N | 经营困难等 |

**注意**：员工依据第38条（用人单位过错）提出的解除，用人单位须支付N。

### 第二步：高风险情形核查 `[SME 核查]`

**绝对禁止解除情形**（劳动合同法第42条）：

| 情形 | 法律依据 | 适用解除类型 |
|------|---------|------------|
| 从事接触职业病危害作业未做离岗前健康检查 | 第42条第1项 | 非过失性/经济性裁员 |
| 疑似职业病在诊断或医学观察期间 | 第42条第1项 | 非过失性/经济性裁员 |
| 在本单位患职业病或因工负伤丧失/部分丧失劳动能力 | 第42条第2项 | 非过失性/经济性裁员 |
| 患病或非因工负伤在医疗期内 | 第42条第3项 | 非过失性/经济性裁员 |
| 女职工在孕期、产期、哺乳期 | 第42条第4项 | 非过失性/经济性裁员 |
| 在本单位连续工作满15年且距退休不足5年 | 第42条第5项 | 非过失性/经济性裁员 |

**特别注意**：上述情形**仅禁止非过失性解除和经济性裁员**，不禁止过失性解除（第39条）和协商解除（第36条）。

### 第三步：解除要件逐项审查

#### 过失性解除（第39条）审查清单：

| 解除理由 | 核心要件 | 常见败诉原因 |
|---------|---------|------------|
| 试用期不符合录用条件 | ①书面录用条件 ②已公示 ③有证据证明不符合 | 录用条件不明、未公示、证据不足 |
| 严重违反规章制度 | ①制度经民主程序 ②已公示 ③违纪事实清楚 ④严重程度合理 | 制度未公示、违纪严重性认定不当 |
| 严重失职造成重大损害 | ①有明确职责定义 ②有损失证据 ③因果关系明确 | 损害认定不清、因果关系不成立 |
| 员工同时与其他单位建立劳动关系影响本单位工作 | ①有证据证明 ②单位提出后拒不改正 | 未提出改正、影响程度不足 |

#### 非过失性解除（第40条）审查清单：

| 解除理由 | 前置程序 | 常见误区 |
|---------|---------|---------|
| 医疗期满不能从事原工作也不能从事另行安排的工作 | ①医疗期计算准确 ②已安排另行工作 ③劳动能力鉴定（如需）| 医疗期计算错误、未安排另行工作 |
| 不能胜任工作 | ①有明确岗位要求 ②经培训或调岗 ③仍不能胜任 | 未经培训/调岗直接解除、考核标准不合理 |
| 客观情况重大变化 | ①变化确实重大 ②协商变更不成 | 变化程度不"重大"、未充分协商 |

#### 经济性裁员（第41条）审查清单：

| 要件 | 要求 |
|------|------|
| 人数门槛 | 20人以上或不足20人但占职工总数10%以上 |
| 实体条件 | ①依破产法重整 ②生产经营严重困难 ③转产/技术革新后富余（经变更合同不成）④客观经济情况重大变化 |
| 程序要求 | 提前30日向工会/职工说明情况+听取意见+向劳动行政部门报告 |
| 优先留用 | ①无固定期限合同 ②较长期限合同 ③家庭无其他就业人员 |
| 优先招用 | 裁员后6个月内重新招用人员的，优先通知被裁减人员 |

### 第四步：经济补偿金计算

| 解除类型 | 补偿标准 | 计算方式 |
|---------|---------|---------|
| 协商解除（单位提出）| N | 工作年限×月均工资 |
| 非过失性解除 | N+1(代通知金可选) | 同上+提前30日通知则只需N |
| 经济性裁员 | N | 同上 |
| 违法解除 | 2N 或 继续履行 | 2×工作年限×月均工资 |
| 员工依据第38条解除 | N | 同上 |

**计算公式细则**：
- N：每满一年=1个月工资，6个月以上不满一年=1年，不满6个月=0.5年
- 月平均工资：解除前12个月平均工资（含奖金、津贴等）
- 上限：月平均工资>当地社平工资3倍时，按3倍计算，最高12年

### 第五步：发出前核查清单

```
发出解除通知前，逐一确认：
□ 解除类型选择正确
□ 不具备禁止解除情形（第42条）
□ 事实依据充分且证据确凿
□ 实体法律依据正确（对应法条准确）
□ 程序合规（已通知工会/已履行民主程序）
□ 经济补偿金计算准确
□ 解除通知书面形式，内容明确
□ 送达方式合规（当面送达签字/EMS邮寄/短信+公告）
□ 离职证明已准备（15日内出具）
□ 社保/档案转移手续已安排
```

### 第六步：升级决策门

> "劳动合同解除是中国劳动争议中败诉率最高的领域。程序瑕疵、证据不足、法律适用错误都可能导致违法解除认定。以下情形必须移交专业律师：(1)涉及第42条禁止解除情形的任何解除决定；(2)经济性裁员方案整体设计；(3)涉及高级管理人员或高薪员工的解除（补偿金计算复杂）；(4)涉及违法解除后的恢复劳动关系或2N赔偿金争议。本分析为合规指导，不构成法律意见，所有解除决定须经劳动法律师最终确认方可执行。"
