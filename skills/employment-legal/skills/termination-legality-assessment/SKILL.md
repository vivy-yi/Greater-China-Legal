# 劳动合同解除合法性评估与经济补偿咨询 (Termination Legality Assessment)

## 1. 目的与范围

### 1.1 核心目的

为用人单位与劳动者提供劳动合同解除合法性的法律评估框架，明确违法解除的认定标准，系统计算经济补偿金/赔偿金，最大限度降低劳动争议风险，保障双方合法权益。

### 1.2 适用范围

- 用人单位单方解除劳动合同的合法性判断
- 劳动者主动解除劳动合同的权利与补偿
- 经济补偿金（N）、赔偿金（2N）及代通知金的分析计算
- 违法解除劳动合同的法律后果与救济途径
- 特殊群体（孕期女职工、工伤员工、医疗期员工）解除保护
- 协商解除与过失性解除的边界认定
- 劳动合同终止（到期不续签）的补偿义务

### 1.3 本Skill不涵盖

- 劳动仲裁代理与诉讼代理（见 labor-arbitration-filing）
- 集体劳动合同的谈判与签订
- 劳务派遣与外包用工的解除特殊规定
- 公务员与事业单位人员的聘用制管理
- 竞业限制纠纷（见 non-compete-enforcement）

## 2. 劳动合同解除类型体系

### 2.1 解除类型总览

```plaintext
劳动合同解除
├── 用人单位单方解除
│   ├── 过失性解除（第39条）
│   ├── 非过失性解除（第40条）
│   └── 经济性裁员（第41条）
├── 劳动者主动解除
│   ├── 主动辞职（第37条）
│   └── 被迫辞职（第38条）
└── 协商一致解除
    ├── 用人单位提议（第36条）
    └── 劳动者提议（一般无补偿）
```

## 3. 用人单位单方解除合法性判断

### 3.1 过激性解除（第39条）

#### 适用情形

| 序号 | 解除情形 | 关键证据要求 |
|------|----------|-------------|
| 1 | 试用期不符合录用条件 | 录用条件明确约定+考核标准客观化+解除前已通知工会 |
| 2 | 严重违反用人单位规章制度 | 制度经过民主程序制定+已向员工公示+违规事实清楚 |
| 3 | 严重失职，营私舞弊，给用人单位造成重大损害 | 损失金额需具体可量化+因果关系明确 |
| 4 | 劳动者同时与其他用人单位建立劳动关系，对完成本单位工作任务造成严重影响，或经用人单位提出拒不改正 | 存在竞业限制义务或工作任务影响+书面通知后仍未改正 |
| 5 | 劳动合同无效（欺诈、胁迫、乘人之危） | 以欺诈手段签订劳动合同+证据充分 |

#### 合规要点

- **程序要求**：解除前应书面通知工会（建立工会组织者），工会提出意见后研究处理并将处理结果书面通知工会
- **举证责任**：用人单位对解除理由的客观事实承担完全举证责任
- **时效**：应在知道或应当知道违规事实之日起合理期限内作出解除决定

#### 常见违法情形

```plaintext
过失性解除高频违法风险点：

1. 录用条件未明确约定或无法量化
   → 解除违法，风险：赔偿金2N

2. 规章制度未经民主程序或未公示
   → 制度对员工不发生效力，解除违法

3. 解除前未通知工会
   → 程序瑕疵，但部分地区认定不影响解除效力（需结合当地裁判口径）

4. 违规事实证据不足或证据链断裂
   → 事实不清，解除违法

5. 解除决定超出行使期限
   → 丧失行使权利，解除可能被认定无效
```

### 3.2 非过失性解除（第40条）

#### 适用条件

劳动者有下列情形之一的，用人单位提前三十日以书面形式通知劳动者本人或者额外支付劳动者一个月工资后，可以解除劳动合同：

```plaintext
情形一：劳动者患病或非因工负伤
├── 医疗期满不能从事原工作
└── 不能从事另行安排的工作

情形二：劳动者不能胜任工作
├── 经培训或调整工作岗位后仍不能胜任
└── 需先经过培训或调岗程序

情形三：客观情况发生重大变化
├── 劳动合同订立时所依据的客观情况发生重大变化
├── 致使劳动合同无法履行
└── 经协商未能就变更劳动合同达成协议
```

#### 额外支付一个月工资 = 代通知金

即上一个月工资标准（不含加班费、奖金等浮动部分）

#### 违法解除的认定

```plaintext
非过失性解除高频违法风险点：

1. 医疗期计算错误
   → 医疗期期限依据工龄计算，不同于治疗期

2. 未进行培训或调岗即直接解除
   → 程序缺失，解除违法

3. 客观情况变化证据不足
   → 常见于部门裁撤、业务调整等情形

4. 未提前30日书面通知或未支付代通知金
   → 程序瑕疵，但补偿金可补正，不必然导致2N

5. 协商变更程序缺失
   → 未与劳动者协商即单方解除，程序违法
```

### 3.3 经济性裁员（第41条）

#### 启动条件（须同时满足）

```plaintext
经济性裁员法定条件：

情形一：濒临破产进行法定整顿期间
情形二：生产经营发生严重困难
情形三：企业转产、重大技术革新或经营方式调整
情形四：其他因劳动合同订立时所依据的客观经济情况发生重大变化

另须满足：
  用人单位需要裁减人员二十人以上
  或裁减不足二十人但占企业职工总数百分之十以上
```

#### 程序要求

```plaintext
经济性裁员程序步骤：

第一步：提前30日向工会或全体职工说明情况
        ↓
第二步：听取工会或职工意见
        ↓
第三步：向劳动行政部门报告裁减人员方案
        ↓
第四步：公布裁减人员方案并予以实施
```

#### 优先留用与优先招用

```plaintext
优先留用人员（同一条件下）：
  1. 与本单位订立较长期限的固定期限劳动合同
  2. 与本单位订立无固定期限劳动合同
  3. 家庭无其他就业人员，有需要扶养的老人或未成年人

优先招用被裁减人员：
  用人单位裁减人员后，在6个月内重新招用人员的，
  应当通知被裁减人员，并在同等条件下优先招用。
```

#### 违法认定

```plaintext
经济性裁员高频违法风险点：

1. 裁减人数未达法定标准（20人/10%以上）
   → 不构成经济性裁员，按一般解除处理

2. 未履行报告程序或报告内容不实
   → 程序违法，解除可能被撤销

3. 未优先留用应当保留的人员
   → 违法解除

4. 裁减人员选择不公平（如仅裁撤年龄较大员工）
   → 存在就业歧视风险

5. 优先招用义务未履行
   → 违反法定责任，可能承担赔偿责任
```

## 4. 劳动者解除劳动合同

### 4.1 主动辞职（第37条）

```plaintext
劳动者提前通知解除劳动合同：

一般劳动者：提前30日书面通知用人单位，30日期满后劳动关系解除
试用期员工：提前3日书面通知，3日期满后劳动关系解除

法律效果：
  → 用人单位无需支付经济补偿金
  → 工资结算至最后工作日
  → 用人单位应在解除时开具离职证明
```

### 4.2 被迫辞职（第38条）

#### 法定被迫解除情形

```plaintext
用人单位有下列情形之一的，劳动者可以解除劳动合同：

1. 未按照劳动合同约定提供劳动保护或劳动条件
2. 未及时足额支付劳动报酬
3. 未依法为劳动者缴纳社会保险费
4. 用人单位的规章制度违反法律、法规的规定，损害劳动者权益
5. 因欺诈、胁迫或乘人之危致使劳动合同无效
6. 法律、行政法规规定劳动者可以解除劳动合同的其他情形

另有（司法解释一第第45条）：
  以暴力、威胁或非法限制人身自由等手段强迫劳动
  违章指挥、强令冒险作业危及劳动者人身安全
```

#### 被迫解除的法律效果

```plaintext
被迫解除 ≠ 主动放弃权利

法律效果：
  → 用人单位须支付经济补偿金（N）
  → 劳动者可主张未足额支付的工资/报酬
  → 可同时主张被迫解除的补偿责任与拖欠工资的法律责任

注意：
  劳动者须书面说明被迫解除的具体理由
  口头通知可能存在举证困难
```

## 5. 协商解除劳动合同（第36条）

### 5.1 用人单位提议协商解除

```plaintext
类型：用人单位向劳动者提出解除劳动合同协商
结果：经双方协商一致解除

法律效果：
  → 用人单位应支付经济补偿金（N）
  → 补偿年限从入职之日起算

示例：
  员工入职5年3个月，月工资15000元（高于当地平均工资3倍）
  经济补偿金 = 15000 × 5.5 = 82500元
  （6个月封顶：15000 × 6 = 90000元）
```

### 5.2 劳动者提议协商解除

```plaintext
类型：劳动者主动向用人单位提出解除劳动合同协商
结果：经双方协商一致解除

法律效果：
  → 用人单位无需支付经济补偿金
  → 但实际协商中用人单位可能自愿给予补偿

实操建议：
  劳动者应保留协商过程相关证据
  避免在协商记录中出现"本人主动提出辞职"等表述
```

## 6. 经济补偿金与赔偿金计算

### 6.1 经济补偿金（N）

#### 计算公式

```plaintext
经济补偿金 = 劳动者在本单位工作年限 × 劳动者解除前12个月平均工资

工作年限计算规则：
  6个月以上不满1年的，按1年计算
  不满6个月的，按半年计算（即0.5年）
  最高补偿年限：12年（工资超过当地平均工资3倍的）
```

#### 工资计算标准

```plaintext
工资定义（劳动合同法第47条）：
  劳动者在劳动合同解除前12个月的平均工资
  包括：基本工资、岗位工资、绩效工资、加班费、奖金、津贴、补贴等

不包括：
  用人单位一次性或偶然性支付的款项
  未连续发放的款项

特殊情形：
  劳动者工作不满12个月的，按实际工作月数计算平均工资
  工资低于当地最低工资标准的，按最低工资标准计算
  工资超过当地上年度职工月平均工资3倍的，按3倍封顶计算
```

#### 年限封顶规则

```plaintext
普通劳动者：年限无封顶，最多补偿12个月
高薪员工（工资超过当地上年度职工月平均工资3倍）：
  补偿年限最高不超过12年
  补偿基数按当地平均工资3倍封顶

示例（假设当地上年度职工月平均工资为8000元）：
  员工月薪50000元（超过3倍），工作年限15年
  经济补偿金 = 8000 × 3 × 12 = 288000元（双封顶）
  （年限12年 + 基数8000×3，双重封顶）
```

### 6.2 赔偿金（2N）

#### 适用情形

```plaintext
用人单位违法解除或终止劳动合同，须支付赔偿金：

赔偿金 = 2 × 经济补偿金（ 即2N）

常见违法解除情形：
  1. 解除理由不符合法定情形
  2. 解除程序存在重大瑕疵
  3. 解除理由证据不足
  4. 违反对特殊群体（孕期女职工、工伤员工、医疗期员工）的保护性规定
  5. 劳动合同到期应当续签而用人单位不续签（符合订立无固定条件时）
```

#### 2N与N+1的区分

```plaintext
2N    = 违法解除的惩罚性赔偿（用人单位有过错）
N+1   = 非过失性解除的补偿（合法解除但程序瑕疵，用代通知金补正）
N     = 协商解除、被迫解除、合法非过失解除

注意：N+1 ≠ 2N，劳动者不能同时主张
      仲裁/诉讼中可根据案情变更请求
```

### 6.3 代通知金（+1）

#### 适用情形

```plaintext
非过失性解除（第40条）时，用人单位可选择：

方案A：提前30日书面通知  → 无需支付代通知金
方案B：额外支付1个月工资 → 立即解除

代通知金计算标准：
  劳动者上一个月正常出勤应发工资（不含加班费、一次性奖金）
  即标准工资，不低于当地最低工资标准
```

### 6.4 计算示例汇总

```plaintext
【示例一】协商解除（用人单位提议），正常工资
  入职时间：2018年3月1日
  解除时间：2024年8月15日
  解除前12个月平均工资：12000元/月
  工作年限：6年5个月 → 按6.5年计算
  补偿金 = 12000 × 6.5 = 78000元

【示例二】协商解除（用人单位提议），高薪员工
  入职时间：2015年7月1日
  解除时间：2024年6月30日
  解除前12个月平均工资：35000元/月
  当地上年度职工月平均工资：9000元
  工作年限：8年11个月 → 按9年计算（但受12年封顶）
  封顶后年限：9年（未超12年）
  封顶后工资：35000 vs 27000（9000×3）→ 按27000计算
  补偿金 = 27000 × 9 = 243000元

【示例三】违法解除（无合法理由单方解除）
  入职时间：2020年1月10日
  解除时间：2024年9月1日
  解除前12个月平均工资：8000元/月
  工作年限：4年7个月 → 按5年计算
  经济补偿金 = 8000 × 5 = 40000元
  赔偿金 = 40000 × 2 = 80000元（2N）

【示例四】非过失性解除（未提前30日通知）
  入职时间：2019年5月1日
  解除时间：2024年10月31日
  解除前12个月平均工资：10000元/月
  工作年限：5年5个月 → 按5.5年计算
  经济补偿金 = 10000 × 5.5 = 55000元
  代通知金 = 10000元
  合计 = 65000元（N+1）

【示例五】被迫解除（用人单位拖欠工资）
  入职时间：2021年2月1日
  解除时间：2024年7月31日
  解除前12个月平均工资：9000元/月
  工作年限：3年5个月 → 按3.5年计算
  经济补偿金 = 9000 × 3.5 = 31500元（N）
  + 可同时主张拖欠工资
```

## 7. 特殊群体解除保护

### 7.1 孕期、产期、哺乳期女职工（第42条）

```plaintext
对"三期"女职工的特殊保护：

适用情形：
  用人单位不得依据第40条（非过失性解除）
  或第41条（经济性裁员）解除"三期"女职工劳动合同

但以下情形仍可解除：
  1. 协商一致解除（第36条）
  2. 劳动者提出解除（第37条）
  3. 过失性解除（第39条）— 如严重违纪
  4. 劳动者达到法定退休年龄
  5. 劳动合同期满（须续签至哺乳期满）

违法后果：
  → 违法解除须继续履行劳动合同
  → 或支付赔偿金（2N）+ 产假工资损失
```

### 7.2 工伤与职业病员工

```plaintext
工伤员工解除保护（根据伤残等级）：

1-4级伤残：保留劳动关系，退出劳动岗位
           用人单位不得解除或终止劳动合同

5-6级伤残：保留劳动关系
           劳动者提出解除可获补偿
           用人单位不得提出解除

7-10级伤残：劳动合同期满可终止
            用人单位提出解除须支付一次性伤残就业补助金
            违法解除须支付赔偿金

注意：工伤员工离职后，一次性伤残就业补助金由用人单位支付
      一次性工伤医疗补助金由工伤保险基金支付
```

### 7.3 医疗期员工

```plaintext
医疗期计算规则：

  实际工作年限    │ 本单位工作年限  │ 医疗期月数
  ─────────────┼──────────────┼──────────
  10年以下       │ 5年以下         │ 3个月
  10年以下       │ 5年以上         │ 6个月
  10年以上       │ 5年以下         │ 6个月
  10年以上       │ 5-10年          │ 9个月
  10年以上       │ 10-15年         │ 12个月
  10年以上       │ 15-20年         │ 18个月
  20年以上       │ 20年以上         │ 24个月

医疗期内：
  用人单位不得依据第40条（不能胜任工作）
  解除劳动合同
  但可依据第39条（过失性解除）解除

医疗期满后：
  劳动者不能从事原工作也不能从事另行安排工作的
  用人单位可依法解除，但须支付经济补偿金
```

## 8. 违法解除的法律后果与救济

### 8.1 劳动者的救济权利

```plaintext
违法解除后，劳动者可选择：

选项一：要求继续履行劳动合同
  适用情形：劳动者有继续履行意愿+劳动合同有履行可能
  法律效果：用人单位须安排原岗位或同等岗位工作
           须支付违法解除期间的工资损失

选项二：要求支付赔偿金
  适用情形：劳动合同已无法继续履行（如岗位已招新）
  法律效果：赔偿金 = 2N

选项三：要求支付经济补偿金及损失赔偿
  适用情形：用人单位解除存在瑕疵但非完全违法
  法律效果：结合具体案情确定补偿项目
```

### 8.2 工资损失计算

```plaintext
违法解除期间工资损失计算：

计算起点：解除之日至裁判生效之日

计算标准：原劳动合同约定的工资标准
          （不含加班费，按正常工作时间计算）

最长计算期间：一般不超过12个月
              （各地仲裁时效与裁判口径存在差异）

示例：
  违法解除时间：2024年3月1日
  仲裁受理时间：2024年5月15日
  仲裁裁决时间：2024年8月30日
  工资损失 = 原工资 × 6个月
```

## 9. 常见争议场景分析

### 9.1 场景一：部门撤销，岗位取消

```plaintext
问题：用人单位以组织架构调整、部门撤销为由解除员工
      是否构成违法解除？

分析：
  1. 须判断是否符合"客观情况发生重大变化"
     → 部门撤销是否属于客观情况？（司法解释一第26条细化界定）
     → 须区分主观经营决策与客观情况

  2. 是否履行协商变更程序
     → 必须先与劳动者协商变更劳动合同
     → 未能达成协议是解除的前提条件

  3. 是否支付经济补偿金或代通知金

裁判口径：
  单纯组织架构调整通常不构成"客观情况重大变化"
  须结合具体情形、规模、原因综合判断
```

### 9.2 场景二：绩效考核不合格解除

```plaintext
问题：用人单位以绩效考核不合格为由解除员工
      是否合法？

分析：
  1. 绩效考核制度是否经过民主程序制定
     → 须经职工代表大会或全体职工讨论

  2. 考核标准是否明确、客观、可量化
     → 不能仅凭主观评价

  3. 是否给予培训或调岗机会
     → 不能胜任工作须先培训或调岗

  4. 考核程序是否公正透明
     → 员工是否有申辩机会

  5. 解除前是否通知工会

高频风险：
  → 考核标准未经民主程序 → 制度无效 → 解除违法
  → 未进行培训调岗即解除 → 程序缺失 → 解除违法
  → 考核结果无客观依据 → 事实不清 → 解除违法
```

### 9.3 场景三：员工不辞而别

```plaintext
问题：员工突然不来上班，不辞而别，用人单位如何处理？

正确处理步骤：

第一步：核实情况
  → 联系员工确认原因（电话、短信、邮件）
  → 留存联系记录

第二步：发出返岗通知
  → 书面催告员工返岗工作
  → 明确逾期不返岗的法律后果

第三步：依据规章制度处理
  → 连续旷工达到规章制度规定的解除天数
  → 依据第39条第2项（严重违反规章制度）解除

注意事项：
  → 不能直接按"自动离职"处理
  → 须履行书面通知工会的程序
  → 须出具解除劳动合同证明
  → 不得扣押员工档案或不出具离职证明
```

## 10. 操作建议与合规清单

### 10.1 用人单位解除合规清单

```plaintext
解除劳动合同前须确认：

□ 解除类型：属于哪种法定解除情形？
□ 事实依据：解除的事实是否清楚、证据是否充分？
□ 程序合规：是否提前通知工会？（建立工会组织者）
□ 程序合规：是否提前30日书面通知或支付代通知金？（第40条）
□ 特殊保护：是否为三期女职工、工伤员工、医疗期员工？
□ 补偿准备：经济补偿金/赔偿金金额是否核算清楚？
□ 文书准备：解除通知书是否书面出具？内容是否准确？
□ 档案管理：解除相关材料是否归档保存？
□ 离职手续：是否开具离职证明、办理社保转移？
```

### 10.2 劳动者维权建议

```plaintext
收到解除通知后须确认：

□ 解除理由：用人单位出具的解除理由是什么？
□ 合法性判断：该解除理由是否符合法定情形？
□ 程序合规：解除是否提前30日通知或支付代通知金？
□ 补偿核算：经济补偿金/赔偿金计算是否正确？
□ 特殊保护：自己是否属于特殊保护群体（孕期、工伤等）？
□ 证据固定：工资条、社保缴纳记录、劳动合同是否保存？
□ 时效注意：劳动仲裁时效为一年（从知道或应当知道之日起算）
□ 协商空间：如接受协商解除，补偿金额是否有谈判余地？
```
