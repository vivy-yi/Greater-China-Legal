# 劳动仲裁申请程序与材料准备

## 目的

本技能为用户提供劳动仲裁申请的全流程指导，从争议识别、管辖确定、时效核查到材料准备和请求设计。旨在帮助用户高效完成仲裁申请前的准备工作，降低因程序瑕疵导致申请被驳回的风险。

> 劳动仲裁是劳动法争议的**必经前置程序**，未经仲裁不得直接向人民法院提起诉讼。仲裁阶段的结果对后续诉讼有实质性影响。

## 法域假设

默认中国大陆法域，适用《劳动争议调解仲裁法》（2008）、《劳动合同法》（2012年修正）及相关司法解释 `[SME 核查]`。

## 加载信息

- 当事人角色（用人单位/员工）
- 争议类型（拖欠工资/违法解除/经济补偿/工伤赔偿/社保争议/竞业限制等）
- 涉及金额
- 劳动关系存续期间
- 最后一次主张权利或争议发生的时间
- 已有证据情况

## 工作流程

### 第一步：确认争议类型与仲裁管辖

| 争议类型 | 是否属劳动仲裁受案范围 | 说明 |
|---------|----------------------|------|
| 拖欠工资/加班费 | 是 | 核心劳动仲裁事项 |
| 违法解除劳动合同 | 是 | 要求支付2N赔偿金或继续履行 |
| 经济补偿金/赔偿金 | 是 | N/N+1/2N争议 |
| 未签书面合同二倍工资 | 是 | 须在入职后满一个月起算 |
| 社保补缴 | **否**（仲裁不受理） | 须向劳动监察/税务部门投诉 |
| 公积金补缴 | **否**（仲裁不受理） | 须向公积金管理中心投诉 |
| 竞业限制争议 | 是 | 违约金/补偿金争议 |
| 工伤赔偿 | 是 | 但须先完成工伤认定+劳动能力鉴定 |

**管辖规则**（劳动争议调解仲裁法第21条）：
- 劳动合同履行地仲裁委优先管辖
- 用人单位所在地仲裁委补充管辖
- 双方同时申请仲裁的，由劳动合同履行地管辖

### 第二步：时效核查 `[SME 核查]`

| 争议类型 | 仲裁时效 | 起算点 |
|---------|---------|-------|
| 一般劳动争议 | 1年 | 知道或应当知道权利被侵害之日 |
| 拖欠劳动报酬 | 1年（在职期间不受限） | 劳动关系终止之日起算 |
| 未签合同二倍工资 | 1年 | 知道或应当知道之日 |
| 在职期间工资争议 | 1年（不受限，至离职起算） | 劳动关系终止之日起算 |

**时效中断情形**（可重新计算）：
- 向对方当事人主张权利（须有证据）
- 向有关部门请求权利救济
- 对方同意履行义务

**注意**：超过时效且无中断事由 → 仲裁委将驳回请求。但时效抗辩须由对方主动提出，仲裁委不主动适用。

### 第三步：仲裁请求设计与材料准备

**仲裁请求示例格式**：
```
1. 请求裁决被申请人支付拖欠工资人民币[___]元（[期间]）
2. 请求裁决被申请人支付违法解除劳动合同赔偿金人民币[___]元（[计算依据]）
3. 请求裁决被申请人支付未休年休假折算工资人民币[___]元
4. 请求裁决被申请人出具解除劳动合同证明并办理社保转移手续
```

**申请材料清单**：

| 材料名称 | 要求 | 份数 |
|---------|------|------|
| 劳动仲裁申请书 | 写明请求事项、事实与理由、证据清单 | 按被申请人人数+1份 |
| 身份证明 | 身份证复印件（员工）/营业执照复印件（单位） | 1份 |
| 授权委托书 | 如有代理人 | 1份 |
| 证据材料 | 按证据清单排序，编页码 | 按被申请人人数+1份 |
| 证据清单 | 列明证据名称、证明目的、页码 | 按被申请人人数+1份 |

**常见证据类型**：
- 劳动合同/协议（证明劳动关系）
- 工资单/银行流水（证明工资标准）
- 考勤记录/加班审批（证明加班事实）
- 解除通知/解除协议（证明解除事实）
- 微信聊天记录/邮件（辅助证明）
- 录音/录像（须核实合法性）

### 第四步：证据准备与策略建议 `[SME 核查]`

**举证责任分配**：

| 争议事项 | 举证责任方 |
|---------|-----------|
| 劳动关系存在 | 员工（提供劳动合同/工资记录/工作证等） |
| 工资标准 | 用人单位（保留工资台账和考勤记录） |
| 加班事实 | 员工（提供初步证据后转由用人单位举证） |
| 解除合法性 | 用人单位（对解除的合法性和程序合规性负举证责任） |
| 经济补偿计算 | 双方各自提供依据 |

**证据保全建议**：
- 对电子证据（微信记录、邮件、系统截图）提前做公证保全
- 关键录音须提供完整版本，不得剪辑
- 证人证言须本人出庭（书面证言效力有限）

### 第五步：升级决策门

> "劳动仲裁是法律程序，申请材料的质量直接影响案件结果。本分析为辅助指导，不构成正式法律意见。建议在以下情形委托专业劳动法律师代理：(1)案件金额较大；(2)涉及竞业限制、工伤等复杂争议；(3)双方证据对抗激烈；(4)涉及违法解除等高风险认定。对于社保和公积金补缴事项，仲裁委不受理，须向劳动监察或公积金管理部门投诉。"

## 本技能不涵盖

- **代理劳动仲裁或诉讼** — 须委托具有执业资格的律师或法律工作者
- **确认法律效力** — 仲裁请求是否成立须经仲裁委裁决 `[SME 核查]`
- **非中国大陆法域的劳动争议处理**
- **社保/公积金补缴—仲裁不受理事项，须向行政部门投诉**
