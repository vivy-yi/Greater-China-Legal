# 试用期管理与解除合规咨询

## 目的

本技能为用户提供试用期设定、管理和解除的全流程合规指导。试用期是中国劳动法中最常见的争议领域之一，核心问题包括：期限约定不合法、工资低于法定标准、解除程序不合规、录用条件设定不明确等。

> 试用期的法律逻辑是给予双方相互考察期，但法律对试用期设定了严格的限制。用人单位违反试用期规定，可能面临行政处罚和员工索赔的双重风险。

## 法域假设

默认中国大陆法域，适用《劳动合同法》（2012年修正）及其实施条例 `[SME 核查]`。

## 加载信息

- 当事人角色（用人单位/员工）
- 劳动合同期限
- 试用期起止时间
- 试用期工资标准
- 录用条件设定（书面/口头）
- 试用期解除原因（如适用）
- 是否存在试用期延长/重复试用情况

## 工作流程

### 第一步：试用期期限合规审查 `[SME 核查]`

**法定期限上限**（劳动合同法第19条）：

| 合同期限 | 试用期上限 |
|---------|-----------|
| 3个月以下 | 不得约定试用期 |
| 3个月以上不满1年 | 1个月 |
| 1年以上不满3年 | 2个月 |
| 3年以上及无固定期限 | 6个月 |

**重要规则**：
- 同一用人单位与同一劳动者**只能约定一次试用期**
- 以完成一定工作任务为期限的合同**不得约定试用期**
- 试用期包含在劳动合同期限内
- 仅约定试用期的，试用期不成立，该期限为劳动合同期限

**常见违规情形**：
- 超过法定上限 → 超出部分按转正工资支付（劳动合同法第83条）
- 重复约定试用期 → 无效，已履行的按转正标准赔偿
- 试用期后延长试用期 → 违反"一次试用期"原则

### 第二步：试用期工资审查 `[SME 核查]`

**法定最低标准**（劳动合同法第20条）：
```
试用期工资 ≥ 以下三者中的最高者：
  ① 本单位相同岗位最低档工资的80%
  ② 劳动合同约定工资的80%
  ③ 用人单位所在地最低工资标准
```

| 情形 | 结论 |
|------|------|
| 试用期工资≥约定工资80%且≥当地最低工资 | 合规 |
| 试用期工资<约定工资80% | 违法，员工可要求补足差额 |
| 试用期工资<当地最低工资 | 违法，须补足差额并可能被行政处罚 |

### 第三步：试用期解除合规分析 `[SME 核查]`

**可以解除的情形**（劳动合同法第39条第1项）：
1. 劳动者在试用期间**不符合录用条件**
2. 严重违反用人单位规章制度
3. 严重失职，给用人单位造成重大损害
4. 劳动者同时与其他用人单位建立劳动关系（对完成本单位工作任务造成严重影响）
5. 劳动者以欺诈手段使单位在违背真实意思下订立合同
6. 被依法追究刑事责任

**试用期解除必须具备的**（劳动合同法第21条）：
```
前提条件：
1. 有明确、书面、可量化的录用条件
2. 录用条件已向员工公示（员工签字确认）
3. 有证据证明员工不符合录用条件
4. 解除决定在试用期内作出并送达
5. 说明解除理由
6. 通知工会（如有）
```

**程序不合规的后果**：
- 缺少上述任何一项 → 可能被认定为违法解除
- 违法解除后果：继续履行合同或支付2N赔偿金（劳动合同法第87条和第48条）

### 第四步：转正条件与程序

**转正评估合规要点**：

| 要点 | 合规要求 |
|------|---------|
| 考核周期 | 试用期内完成评估，不得推迟至试用期满后 |
| 考核标准 | 须与录用条件一致，不得在试用期内随意变更标准 |
| 考核记录 | 须有书面记录，员工签字确认 |
| 合格标准 | 须事先明确，不得以"表现良好"等模糊标准拒绝转正 |

**试用期届满的法律后果**：
- 用人单位不表示不录用 → 视为自动转正
- 用人单位未在试用期内作出解除决定 → 员工自动转为正式员工
- 转正后解除须符合非过失性解除或过失性解除的法定条件

### 第五步：升级决策门

> "试用期管理涉及周期计算、工资标准、录用条件设定和解除程序等多重合规要求。其中试用期解除是最容易引发争议的环节，程序瑕疵即使事实成立也可能导致违法解除认定。本分析为辅助指导，不构成法律意见。建议在以下情形委托律师：(1)试用期解除决定须提前做好证据固定和程序审查；(2)试用期约定明显不合法的情形（超期/重复/无固定期限合同6个月上限等）；(3)涉及竞业限制或保密义务的试用期员工管理。"
