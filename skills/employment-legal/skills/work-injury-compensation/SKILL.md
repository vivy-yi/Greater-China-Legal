# 工伤认定与赔偿咨询

## 目的

本技能为用户提供工伤认定与赔偿的全流程指导，包括工伤认定范围判断、申请程序指引、劳动能力鉴定、赔偿金额计算以及争议解决路径。

> 工伤是中国劳动争议中涉及金额较高的领域之一，一次性赔偿可能高达数十万至上百万元（视伤残等级和本人工资而定）。用人单位未依法缴纳工伤保险的，全部工伤待遇由用人单位承担。

## 法域假设

默认中国大陆法域，适用《工伤保险条例》（2010年修订）、《工伤认定办法》、《劳动能力鉴定标准》及相关规定 `[SME 核查]`。

## 加载信息

- 事故经过（时间、地点、原因、现场情况）
- 劳动关系证明材料
- 伤情诊断证明（医院出具的诊断书、病历等）
- 员工受伤前12个月月平均工资
- 用人单位是否依法缴纳工伤保险
- 是否已向社保部门申请工伤认定（如有，提供认定结果）

## 工作流程

### 第一步：工伤认定范围评估 `[SME 核查]`

**应当认定为工伤的情形**（工伤保险条例第14条）：

| 情形 | 示例 | 认定关键 |
|------|------|---------|
| 工作时间和工作场所内，因工作原因受到事故伤害 | 操作机器受伤 | 三要素须同时满足 |
| 工作时间前后在工作场所内，从事与工作有关的预备性或收尾性工作受到事故伤害 | 上班前调试设备受伤 | 时间关联性 |
| 工作时间和工作场所内，因履行工作职责受到暴力等意外伤害 | 保安制止打斗受伤 | 职责关联性 |
| 患职业病 | 噪音性耳聋/尘肺病 | 须有职业病诊断证明 |
| 因工外出期间，由于工作原因受到伤害或发生事故下落不明 | 出差路上车祸 | 工作关联性 |
| 上下班途中，受到非本人主要责任的交通事故/城市轨道交通/客运轮渡/火车事故伤害 | 下班被车撞伤 | ①途中②非本人主责 |
| 法律、行政法规规定应当认定为工伤的其他情形 | 见专项规定 | — |

**视同工伤的情形**（第15条）：

| 情形 | 说明 |
|------|------|
| 工作时间和工作岗位，突发疾病死亡或在48小时内抢救无效死亡 | 关键：48小时时限 |
| 在抢险救灾等维护国家利益/公共利益活动中受到伤害 | 不限工作时间和地点 |
| 职工原在军队服役，因战/因公负伤致残，已取得革命伤残军人证，到用人单位后旧伤复发 | 须有军人伤残证 |

**不得认定为工伤的情形**（第16条）：

| 情形 | 说明 |
|------|------|
| 故意犯罪 | 须刑事判决认定 |
| 醉酒或吸毒 | 须有血液检测证明 |
| 自残或自杀 | 须有证据排除工伤 |

**工伤认定的几个常见争议点**：
- 上下班途中：合理路线+合理时间的判断因案而异
- 48小时死亡：以医疗机构出具的死亡证明时间为准
- 工作原因：须排除个人原因（如打闹、私事）
- 职业病：须由有资质的职业病诊断机构出证

### 第二步：工伤认定申请程序

**申请主体与时限**：

| 申请主体 | 申请时限 | 逾期后果 |
|---------|---------|---------|
| 用人单位 | 事故伤害发生之日或被诊断/鉴定为职业病之日起**30日内** | 逾期期间的工伤待遇由用人单位承担 |
| 职工/近亲属/工会 | 事故伤害发生之日起**1年内** | 逾期一般不予受理 |
| 社保部门 | 收到申请后**60日内**作出决定（事实清楚15日内）| — |

**申请材料清单**：
1. 工伤认定申请表（含事故经过、诊断结论等）
2. 与用人单位存在劳动关系的证明材料（劳动合同/工资记录/工作证等）
3. 医疗机构出具的受伤后诊断证明或职业病诊断证明书
4. 事故现场证明（照片/录像/证人证言等）
5. 用人单位营业执照复印件（如用人单位拒不配合）

**认定后的救济**：
- 对工伤认定决定不服 → 可提起行政复议或行政诉讼
- 复议和诉讼期间，工伤认定决定不停止执行

### 第三步：劳动能力鉴定

**鉴定顺序与程序**：

```
工伤认定完成 → 伤情相对稳定后 → 申请劳动能力鉴定
                          ↓
                   设区的市级劳动能力鉴定委员会
                          ↓
                       初次鉴定
                          ↓
           不服→省级劳动能力鉴定委员会→最终鉴定
```

**伤残等级**（10级，一级最重，十级最轻）：

| 等级范围 | 特征 | 待遇影响 |
|---------|------|---------|
| 一级至四级（完全丧失劳动能力）| 生活/工作能力基本丧失 | 保留劳动关系+退出工作岗位+按月领取伤残津贴 |
| 五级至六级（大部分丧失劳动能力）| 可从事简单工作 | 保留劳动关系+安排适当工作或按月领取伤残津贴 |
| 七级至十级（部分丧失劳动能力）| 可正常工作 | 一次性伤残补助金+一次性工伤医疗补助金+伤残就业补助金（解除合同时）|

### 第四步：工伤保险待遇计算

**各项待遇一览表**：

| 待遇项目 | 计算标准 | 支付方 |
|---------|---------|-------|
| 医疗费 | 工伤保险诊疗项目目录内的合理费用 | 工伤保险基金 |
| 住院伙食补助费 | 统筹地区规定标准×住院天数 | 工伤保险基金 |
| 交通食宿费 | 合理交通+食宿费用 | 工伤保险基金 |
| 辅助器具费 | 因日常生活或就业需要，经鉴定配置 | 工伤保险基金 |
| 停工留薪期待遇 | 原工资福利待遇不变，最长12个月（可延长至24个月）| 用人单位 |
| 护理费 | 生活完全不能自理→统筹地区上年度社平工资50%；大部分→40%；部分→30% | 工伤保险基金/单位 |

**一次性补助金**：

| 等级 | 一次性伤残补助金（月）| 伤残津贴（月，1-4级）| 标准基数 |
|------|--------------------|-------------------|---------|
| 一级 | 27个月 | 本人工资的90% | 本人受伤前12个月平均工资 |
| 二级 | 25个月 | 85% | 同上 |
| 三级 | 23个月 | 80% | 同上 |
| 四级 | 21个月 | 75% | 同上 |
| 五级 | 18个月 | 70%（单位安排不了工作）| 同上 |
| 六级 | 16个月 | 60%（单位安排不了工作）| 同上 |
| 七级 | 13个月 | — | 同上 |
| 八级 | 11个月 | — | 同上 |
| 九级 | 9个月 | — | 同上 |
| 十级 | 7个月 | — | 同上 |

**工亡待遇**：

| 项目 | 标准 |
|------|------|
| 一次性工亡补助金 | 上一年度全国城镇居民人均可支配收入×20倍 |
| 丧葬补助金 | 统筹地区上年度职工月平均工资×6个月 |
| 供养亲属抚恤金 | 配偶每月40%；其他亲属每月30%；孤寡老人/孤儿上浮10% |

> **注意**：一次性工亡补助金全国统一标准，2025年约为103万元（基于2024年全国城镇居民人均可支配收入约5.16万元×20倍，实际以国家统计局公布数据为准 `[GOV]`）。

### 第五步：争议处理路径

| 争议事项 | 处理路径 |
|---------|---------|
| 工伤认定争议 | 行政复议→行政诉讼 |
| 劳动能力鉴定争议 | 省级再次鉴定（最终结论）|
| 工伤保险待遇争议 | 劳动仲裁（必经程序）→民事诉讼 |
| 单位未缴社保的工伤待遇 | 全部由单位承担，劳动仲裁→民事诉讼 |

**举证责任**：
- 职工主张工伤成立：提供初步证据（劳动关系+事故+伤情）
- 用人单位主张不构成工伤：对排除情形（醉酒/自残等）负举证责任
- 单位未缴工伤保险：对工伤待遇承担全部责任

### 第六步：升级决策门

> "工伤认定和赔偿涉及程序复杂、金额较大。以下情形必须移交专业律师：(1)工伤认定被社保部门驳回后申请行政复议或行政诉讼；(2)单位未缴纳工伤保险导致全部工伤待遇须由单位承担；(3)工亡案件一次性补助金和抚恤金计算；(4)劳动能力鉴定争议；(5)涉及第三人侵权竞合（如交通事故+工伤竞合）的赔偿方案设计。本分析为辅助指导，不构成法律意见，工伤认定决定和赔偿金额以社保部门核定为准。"
