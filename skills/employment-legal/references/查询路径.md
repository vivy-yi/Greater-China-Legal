# 劳动法查询路径 — 中国大陆

> 本文档提供劳动法领域实际检索入口和操作流程，按数据类型分类整理。
> 数据源标注规则见 `references/数据源清单.md`。

## 一、法律法规检索

### 1.1 中央法规

```
查询目标：劳动合同法/劳动争议调解仲裁法/工伤保险条例 等

步骤 1：[GOV] 访问国家法律法规数据库
  URL: https://flk.npc.gov.cn
  检索字段：标题（如"中华人民共和国劳动合同法"）
  输出：现行有效版本 + 历次修正记录

步骤 2：[BD] 通过 beidalu API 验证版本时效
  工具：beidalu MCP（如果已连接）
  验证项：是否有最新修订/部分失效条款

步骤 3：交叉验证
  [YD] 检索同一法条，确认与 [GOV]/[BD] 一致
  不一致 → 标注 ⚠️ 来源冲突
```

### 1.2 司法解释

```
查询目标：最高法关于劳动争议的司法解释

URL: https://www.court.gov.cn → 司法解释 → 民事类
或: [YD] 检索"劳动争议司法解释"
或: 人民法院案例库 https://rmfyalk.court.gov.cn
```

### 1.3 各省市工资支付规定 / 劳动合同条例

```
查询目标：北京/上海/广东/江苏/浙江 等地工资支付规定

URL 模板：
  {省份}市人社局官网 → 政策法规 → 工资支付
  例：https://rsj.beijing.gov.cn → 政策法规

[YD] 检索："{省份} 工资支付规定"
[web] 时效核验：发布日期 + 是否有修订
```

## 二、案例检索

### 2.1 最高人民法院指导案例 / 公报案例

```
URL: 人民法院案例库 https://rmfyalk.court.gov.cn
检索字段：关键词（如"违法解除"）+ 案由（劳动争议）
筛选：指导性案例 > 公报案例 > 普通案例

[YD] 检索："劳动争议 指导案例"
```

### 2.2 普通案例（最高法判决 / 高院判决）

```
URL: 中国裁判文书网 https://wenshu.court.gov.cn
注：2021 年起公开数量下降，部分案例通过 [YD]/[WKL] 检索

[YD] 检索："{争议类型} + 地域 + 时间"
例："违法解除 上海 2023"
```

### 2.3 地方典型案例

```
目标：识别某省市裁判口径

[YD] 检索："{省市} + {争议类型}"
[WKL] 综合检索，对比多源

输出格式：
  | 案例 | 裁判日期 | 法院 | 裁判要旨 |
  | --- | --- | --- | --- |
  |（2023）京02民终1234号 | 2023-06 | 北京二中院 | 程序违法即构成违法解除 |
```

### 2.4 仲裁案例（不公开但部分有摘要）

```
[YD] 检索："劳动仲裁 + 案由 + 地域"
注意：仲裁裁决不公开（中国裁判文书网），依赖 [YD]/[WKL] 案例库
```

## 三、地方数据查询（高频更新）

### 3.1 各省市最低工资标准

```
查询目标：北京/上海/广东/江苏/浙江 等地最低工资

URL 模板：
  {省市}人社局官网 → 信息公开 → 最低工资标准
  例：https://rsj.beijing.gov.cn → 信息公开

[web] 时效核验：最低工资通常每年 1 月或 7 月调整
[GOV] 验证：以最新政府文件为准

输出格式：
  | 地区 | 月最低工资 | 小时最低工资 | 生效日期 |
  | --- | --- | --- | --- |
  | 北京 | 2420元 | 24.4元 | 2024-09-01 |
  | 上海 | 2690元 | 24元 | 2024-07-01 |
```

### 3.2 社保缴费基数上下限

```
查询目标：上年度职工月平均工资 → 社保基数上下限

URL 模板：
  {省市}人社局 → 社保缴费 → 缴费基数
  国家税务总局 https://www.chinatax.gov.cn → 社保费

[web] 时效核验：每年 7 月左右调整
```

### 3.3 公积金缴存基数

```
查询目标：住房公积金缴存上下限

URL 模板：
  {省市}住房公积金管理中心官网
  例：北京住房公积金管理中心 https://www.bjzgjj.gov.cn

[web] 年度核验（每年 7 月调整）
```

### 3.4 各省市劳动人事争议仲裁委员会

```
查询目标：争议案件管辖仲裁委

URL 模板：
  {省市}人社局 → 劳动人事争议仲裁
  例：北京市劳动人事争议仲裁委员会

管辖规则：
  用人单位所在地 → 仲裁委
  或：合同履行地 → 仲裁委
```

## 四、特定数据查询

### 4.1 工伤认定标准

```
查询目标：工伤/视同工伤/不得认定情形

[GOV] 工伤保险条例 https://www.gov.cn
[YD] 检索："工伤认定 + {情形}"

关键情形：
  - 工作时间工作场所因工作原因受伤
  - 上下班途中非本人主要责任的交通事故
  - 患职业病
  - 视同工伤（工作时间工作岗位突发疾病 48 小时内死亡等）
```

### 4.2 工伤保险待遇标准

```
查询目标：伤残等级对应待遇

URL：各省市人社局 → 工伤保险 → 待遇标准

[YD] 检索："{伤残等级} + 工伤待遇 + {地区}"

查询项：
  - 一次性伤残补助金（1-10级）
  - 伤残津贴（1-4级）
  - 一次性工伤医疗补助金（5-10级解除合同时）
  - 一次性伤残就业补助金（5-10级解除合同时）
```

### 4.3 经济补偿金/赔偿金计算

```
查询目标：N / N+1 / 2N 计算

[YD] 检索："经济补偿金计算 + 月工资基数"

计算工具：
  - 用人单位所在地上年度职工月平均工资（[GOV] 各省市人社局）
  - 劳动者前12个月平均工资（公司工资台账）
  - 服务年限（公司人事记录）

公式：
  N = 月平均工资 × 服务年限
  2N = N × 2（违法解除）
  三倍上限：月平均工资 > 当地三倍平均 → 按三倍
  12年封顶：按三倍计算最高12年
```

### 4.4 竞业限制补偿/违约金

```
查询目标：合理补偿标准 + 违约金酌减规则

[YD] 检索："竞业限制 + 补偿金 + {地区}"

参考标准（CLAUDE.md §各省市差异）：
  北京：≥20% 工资
  上海：≥30% 工资
  其他：参照约定（合理性审查）

违约金酌减：[YD] 检索"违约金酌减 + 竞业限制"
  司法实践：通常按实际损失 + 合理预期调整
```

## 五、争议处理程序查询

### 5.1 仲裁申请流程

```
查询目标：仲裁申请书模板 + 提交材料清单

URL：{省市}劳动人事争议仲裁委员会官网
[Y D] 检索："劳动仲裁申请书模板"

申请材料：
  - 仲裁申请书（写明申请人/被申请人/请求/事实理由）
  - 身份证
  - 劳动关系证明（劳动合同/工资流水/工牌等）
  - 证据清单
  - 用人单位工商信息（企查查/天眼查）

时效：1年（劳动争议调解仲裁法第27条）
```

### 5.2 仲裁→诉讼路径

```
查询目标：一裁终局 + 起诉规则

[YD] 检索："一裁终局 + 劳动争议"

一裁终局情形（劳动争议调解仲裁法第47条）：
  - 追索劳动报酬、工伤医疗费、经济补偿或赔偿金
  - 不超过当地月最低工资标准 12 个月的争议
  - 因执行国家劳动标准在工作时间、休息休假、社会保险
    等方面发生的争议

起诉规则：
  - 劳动者不服：15日内向法院起诉
  - 用人单位：仅在法定情形下申请撤销（第49条）
```

## 六、企业人力资源合规自查

### 6.1 规章制度民主程序

```
查询目标：制度民主程序 + 公示要求

URL：{省市}人社局 → 政策法规 → 工资支付/劳动合同
[Y D] 检索："规章制度民主程序 + {地区}"

参考：
  劳动合同法 第4条（规章制度协商 + 公示）
  各地工资支付规定：公示方式（签字/邮件/培训）

[YD] 检索："规章制度 + 未公示 + 劳动者主张"
```

### 6.2 内部调查合规

```
查询目标：内部调查程序 + 取证合法性

[Y D] 检索："内部调查 + 工会通知 + 职代会"

参考：
  - 工会法 + 劳动合同法 第43条
  - 取证不得违反法律禁止性规定
  - 调查报告须客观、证据充分
```

```
查询目标：境外派遣 / 来华就业合规

URL：商务部 https://www.mofcom.gov.cn → 对外投资合作

## 七、应急检索（争议时效紧迫时）

```

[Y D] 检索："劳动仲裁时效 + 中断"

## 八、检索效率技巧

### 8.1 类案检索四步法

```
1. 问题定性：劳动争议案由
   例：劳动争议 → 确认劳动关系 → 解除合法性 → 经济补偿

2. 信息收集：争议事实 + 法律关系
   例：服务年限、解除原因、月工资

3. 关键词提取：
   规范性关键词：违法解除、经济补偿金
   非规范性关键词：打卡记录、年终奖发放条件

4. 数据库检索：
   [YD] + [WKL] + [GOV] 交叉
   地域/时间/审级范围确定
   效力分级（指导案例 > 公报案例 > 普通案例）
```

### 8.2 类案相似度评估

```
高相似度：
  - 相同案由 + 相同地域 + 同类事实
  - 法院级别相同或更高级

中相似度：
  - 相同案由 + 不同地域 + 类似事实
  - 需对比各省市裁判口径

低相似度（参考价值有限）：
  - 不同案由 + 相同地域
```

### 8.3 效力层级判断

```
指导性案例：各级法院"应当参照"（最高法规定）
公报案例：参考价值高
典型案例：参考价值中
普通案例：事实认定参考

引用规则：
  - 关键主张 → 引用指导案例
  - 辅助论据 → 引用公报/典型案例
  - 一般事实 → 普通案例
```
