# 工程量与造价纠纷咨询

## 一、基本信息

### 1.1 核心目的
处理建设工程结算阶段的工程量争议与造价纠纷，覆盖变更签证效力认定、造价鉴定程序、审计结论对抗、索赔时效等核心争议点。

### 1.2 适用范围
- 工程量清单争议（计量范围/计量方法/漏项/错项）
- 工程变更签证效力争议（签字人权限/签证时效）
- 材料调差/人工费调整争议
- 工程索赔（工期索赔/费用索赔）争议
- 政府审计结论与合同约定结算价的冲突
- 造价鉴定机构选取与鉴定结论质证

## 二、分析框架

### 2.1 工程量争议核心类型

| 争议类型 | 典型情形 | 法律依据 | 处理原则 |
|---------|---------|---------|---------|
| 工程量计算争议 | 清单量与实际量不符 | GB50500第4.1.2条 | 按实际完成量计量 |
| 工程量漏项 | 招标清单漏项 | 司法解释第19条 | 有约从约，无约按定额 |
| 变更计量争议 | 变更范围/计价方式存疑 | 合同约定+司法解释 | 发包人确认+变更计价约定 |
| 隐蔽工程计量 | 未经发包人验收即隐蔽 | 司法解释第13条 | 按发包人认可量或鉴定量 |
| 超设计工程量 | 超出设计图纸范围的施工 | 司法解释第19条 | 发包人认可或视为认可 |

### 2.2 工程签证效力审查要素

| 效力要件 | 审查要点 | 常见瑕疵 |
|---------|---------|---------|
| 签证主体 | 签字人是否具有授权 | 现场代表无授权签字 |
| 形式要件 | 是否为单位盖章+负责人签字 | 仅有个人签字无公章 |
| 时限要件 | 是否在合同约定时限内提出 | 逾期视为放弃 |
| 内容要件 | 变更范围/工程量/价款是否明确 | 描述模糊无法计价 |
| 真实要件 | 变更事实是否真实发生 | 虚假签证/倒签 |

**签证效力争议的裁判规则**：
```
发包人代表在授权范围内签署的签证 → 对发包人有约束力
现场监理签字（超出监理权限）→ 无效（但可证明事实）
发包人收到签证后未在约定时限内提出异议 → 视为认可
签证单上发包人签字人虽无授权但多次签署发包人未反对 → 表见代理
```

### 2.3 工程造价鉴定程序

```
造价鉴定启动
├── 申请主体：发包人/承包人
├── 申请阶段：诉前/诉中
├── 鉴定事项：工程量/工程造价/材料价差/索赔金额
├── 鉴定机构：列入司法鉴定名册的造价咨询企业
└── 鉴定依据：合同+图纸+签证+隐蔽记录+现场勘验

鉴定程序
├── 鉴定机构选定（调解→摇号→指定）
├── 鉴定材料质证（双方确认）
├── 现场勘验（鉴定人+双方代表）
├── 初步鉴定意见出具
├── 双方质证（书面异议+专家辅助人）
├── 补充鉴定/重新鉴定
└── 正式鉴定报告出具

法院对鉴定意见的审查
├── 鉴定机构和鉴定人资质
├── 鉴定依据的真实性/合法性/关联性
├── 鉴定方法的科学性
├── 鉴定结论的明确性
├── 是否存在程序违法
└── 质证后的采纳/部分采纳/不采纳
```

### 2.4 政府审计结论vs合同约定结算价

| 情形 | 裁判规则 | 依据 |
|------|---------|------|
| 合同约定以审计结论为准 | 审计结论有约束力 | 意思自治原则 |
| 合同未约定审计条款 | 审计结论不对施工方有约束力 | 最高院民一庭意见（2001） |
| 审计结论明显不合理 | 法院可委托重新鉴定 | 司法解释原则 |
| 政府投资项目强制审计 | 审计不影响合同双方的结算约定 | 全国人大常委会法工委意见（2017） |
| 审计长期拖延 | 发包人不得以审计未完成为由拒绝结算 | 司法解释第21条 |

## 三、操作流程

### 3.1 工程造价纠纷处理六步法

**Step 1：结算文件审查**
- [ ] 竣工结算书/结算报告完整性
- [ ] 结算方式（固定总价/可调总价/单价合同）
- [ ] 合同约定的结算程序和时限
- [ ] 已支付的工程进度款汇总

**Step 2：工程量争议梳理**
- [ ] 清单内vs清单外（漏项/变更/新增）
- [ ] 实际完成量vs结算申报量
- [ ] 签证/变更/索赔文件的效力审查
- [ ] 隐蔽工程的验收记录

**Step 3：计价争议分析**
- [ ] 单价适用（合同单价/新组价/定额计价）
- [ ] 材差调整（投标基准日vs结算期价格）
- [ ] 人工费/机械费调整的合同依据
- [ ] 措施费/规费/税金的计算争议

**Step 4：费用索赔识别与评估**
- [ ] 工期延误导致的费用索赔
- [ ] 停工/窝工损失的索赔
- [ ] 加速施工/赶工费索赔
- [ ] 索赔的程序合规性（28天通知要求）

**Step 5：造价鉴定策略**
- [ ] 是否需要申请造价鉴定
- [ ] 鉴定范围（全部vs部分争议）
- [ ] 鉴定机构推荐和回避申请
- [ ] 鉴定材料准备和质证策略

**Step 6：谈判/调解/诉讼**
- [ ] 协商谈判（背靠背调解/面对面谈判）
- [ ] 建设主管部门调解
- [ ] 仲裁/诉讼中的造价鉴定质证
- [ ] 调解协议/和解协议的司法确认

### 3.2 发包人延迟结算的法律后果

| 迟延情形 | 发包人后果 | 承包人权利 |
|---------|-----------|----------|
| 收到结算书后28天不答复 | 视为认可结算（合同有约定） | 按报送结算主张 |
| 收到结算书后28天不答复（合同无约定） | 不当然视为认可 | 可催告+按约主张利息 |
| 迟延支付结算款 | 逾期付款利息 | 中止施工+优先受偿权主张 |
| 以审计未完成为由拒付 | 对抗无效（非约定审计条款时） | 按合同约定结算+起诉 |
| 以鉴定未完成为由拒付 | 构成拖延 | 申请法院指定鉴定机构 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **造价鉴定结论存在重大程序瑕疵** — 鉴定人未出庭/鉴定依据未经质证
- **政府审计结论与合同约定严重冲突** — 审计核减比例超过30%
- **涉及黑白合同/多份合同效力争议** — 中标合同vs实际履行合同
- **施工合同被认定无效** — 结算参照合同约定还是定额
- **发包人破产/重组** — 结算债权在破产程序中的确认
- **涉及虚假签证/伪造结算资料** — 需启动刑事报案
- **结算金额巨大（超过5000万元）且争议面广** — 多个争议点交织
- **涉及业主方/代建方/施工方三角结算关系** — 多份合同交叉

### 4.2 升级咨询模板

```
【升级咨询】

主要争议：[工程量/单价/材差/签证效力/审计/其他]
当前阶段：[结算报送/对账/谈判/鉴定/诉讼]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. 工期延误索赔的详细分析和量化计算（工期延误分析专业内容）
2. 国际工程结算争议（适用FIDIC合同条件的项目）
3. 合伙/联营施工体的内部结算争议
4. BT/BOT/PPP项目的特殊结算规则
5. 施工合同无效后的工程款折价补偿争议（特殊法律适用）
6. 工程造价咨询企业/鉴定机构的质量责任诉讼
7. 建设工程优先受偿权的诉讼程序（由专业律师代理）
