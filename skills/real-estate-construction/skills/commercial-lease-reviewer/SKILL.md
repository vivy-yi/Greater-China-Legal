# 商业租赁合同审查

## 一、基本信息

### 1.1 核心目的
为商业物业出租方/承租方提供租赁合同全条款审查与谈判支持，覆盖租金机制、租赁期限、装修免租期、转租限制、续租优先权等核心商业安排。

### 1.2 适用范围
- 写字楼/办公楼租赁
- 商业综合体/购物中心租赁
- 产业园区/工业园区租赁
- 商铺/街铺租赁
- 仓储/物流物业租赁

## 二、分析框架

### 2.1 商业租赁合同核心条款审查清单

| 条款类别 | 审查要点 | 风险评级 | 谈判建议 |
|---------|---------|---------|---------|
| 租赁标的 | 物业权属/面积实测/规划用途/消防验收 | 🔴 高 | 核实不动产权证+产调 |
| 租期 | 起租日/到期日/续租期/提前解约权 | 🟡 中 | 明确续租条件和通知期限 |
| 租金 | 月/日/年单方+付款周期+递增机制 | 🟡 中 | 明确递增率和调整触发条件 |
| 押金/保证金 | 金额（通常2-3个月租金）/退还条件 | 🟡 中 | 退还时间条款（通常30天内） |
| 免租期/装修期 | 时长/起算时点/提前解约退还 | 🔴 高 | 按实际装修工期核定 |
| 转租/分租 | 是否允许/出租方同意条件/收益分成 | 🟡 中 | 出租方不得无理拒绝 |
| 装修改造 | 审批流程/消防/结构安全/复原要求 | 🟡 中 | 界定可拆除和不可拆除部分 |
| 维修责任 | 房屋主体/设施设备/公共区域/日常维修 | 🟡 中 | 按物业部位划分权责 |
| 违约责任 | 欠租/提前解约/逾期交房/迟延开业的违约金 | 🔴 高 | 违约金不超过实际损失30% |
| 优先权 | 优先续租权/优先购买权/优先承租权 | 🟡 中 | 列举续租条件和拒绝情形 |
| 不可抗力 | 防疫/政府征收/自然灾害/情势变更 | 🟡 中 | 明确减免租金和租期的处理 |

### 2.2 营业提成租金机制设计

| 提成模式 | 适用场景 | 风险控制 |
|---------|---------|---------|
| 固定租金+营业额提成（两者取高） | 购物中心主力店 | 设定最低固定租金底线 |
| 纯营业额提成 | 百货商场联营 | 设定销售目标+保底条款 |
| 阶梯式提成（按档位递增） | 知名品牌旗舰店 | 明确销售数据审计权（商场有权审计租户销售） |
| 封顶提成 | 高毛利业态（珠宝/化妆品） | 设定提成上限保护租户 |

**营业额定义审查重点**：
- 是否包含线上销售/外卖/团购
- 是否剔除退货金额
- 增值税是否计入营业收入
- 关联交易价格是否公允
- 销售审计权（第三方审计权+审计成本承担）

### 2.3 租赁合同解除权条款分析

```
出租方解除权
├── 承租方欠租达一定期限（通常15-30天）
├── 未经同意转租/分租
├── 擅自改变房屋用途/结构
├── 违法经营导致行政处罚
├── 营业执照长期无法办理
└── 其他严重违约行为

承租方解除权
├── 出租方逾期交房（通常超过30天）
├── 房屋存在质量/安全/消防重大瑕疵
├── 无法办理营业执照/经营许可
├── 出租方无产权证或权属争议
├── 租赁物被查封/征收
└── 疫情/政府管控导致无法经营（情势变更）
```

## 三、操作流程

### 3.1 商业租赁合同审查五步法

**Step 1: 商业条款核实**
- [ ] 标的物业基本信息（地址/面积/楼层/车位配比）
- [ ] 租赁用途确认（办公/零售/餐饮/仓储/联合办公）
- [ ] 起始日/免租期/终止日/续租期
- [ ] 租金单价+递增率+付款周期
- [ ] 押金/物业费/水电燃气/停车费等

**Step 2: 产权与合规审查**
- [ ] 不动产权证书（核对权利人/面积/用途/他项权利）
- [ ] 建设工程消防验收备案/验收意见书
- [ ] 出租方营业执照与授权文件（如为转租）
- [ ] 房屋租赁登记备案（住建部门）
- [ ] 物业是否已抵押（抵押权人同意函）

**Step 3: 条款逐条审查**
- [ ] 租金及押金条款（递增率/支付方式/逾期罚息）
- [ ] 免租期/装修期条款（装修验收标准/恢复原状范围）
- [ ] 转租/分租条款（同意条件/收益分配）
- [ ] 维修/维护条款（房屋主体vs设备设施）
- [ ] 违约责任条款（违约金标准/解除条件）
- [ ] 不可抗力/情势变更条款
- [ ] 争议解决条款（管辖法院/仲裁机构）

**Step 4: 风险标注与谈判策略**
- [ ] 不可接受条款（标注红色）
- [ ] 建议修改条款（标注黄色+建议修改方案）
- [ ] 可接受条款（标注绿色）
- [ ] 优先谈判点（标记P0-P3优先级）

**Step 5: 合同签署与备案**
- [ ] 签署前最终版本核对
- [ ] 授权代表签字权核实
- [ ] 合同备案（租赁合同登记备案）

### 3.2 常见争议解决方案

| 争议类型 | 典型场景 | 解决路径 |
|---------|---------|---------|
| 租金递增争议 | 租金涨幅超过市场接受度 | 协商调整+比价周边同类物业 |
| 免租期退还 | 提前解约退免租期租金 | 约定按比例退还/按实际租期折算 |
| 转租收益分配 | 承租方转租溢价收益归属 | 明确转租利润分配比例 |
| 续租条件 | 市场租金上涨后的续租价格 | 约定第三方评估或固定涨幅+优先续租 |
| 装修改造复原 | 退租时装修范围拆除范围 | 列明永久性装修不得拆除/可拆除范围 |
| 新冠疫情/管控 | 政府部门要求关停 | 约定不可抗力/情势变更的处理机制 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **物业权属存在重大瑕疵** — 无产权证/违建/权属存争议
- **涉及抵押权的租赁** — 租赁权与抵押权顺位冲突（先抵后租/先租后抵）
- **涉及征收/拆迁的租赁** — 征收补偿分配争议
- **长租期（超过10年）的特殊条款** — 超过20年部分无效
- **涉外租赁** — 境外承租方/跨境担保/外汇支付
- **群体性租户纠纷** — 运营方面临批量退租
- **物业管理权/收费权转让** — 商业项目整体租赁后转租
- **REITs/CMBS物业租赁合同审查** — 资产支持专项计划特殊要求

## 五、本Skill不涵盖

1. 居住租赁合同审查（民用住宅租赁受专营法规管制）
2. 保障性租赁住房/人才公寓的特殊规则
3. 物业买卖交易中的租赁关系处理（买卖不破租赁）
4. 商业租赁税务处理（增值税/房产税/印花税）
5. 酒店/民宿/短租的特殊监管
6. 农业用地租赁（土地管理法特别规定）
7. 物业管理服务合同审查
