# 诉讼文书章节起草

## 目的

辅助起草诉讼/仲裁文书的各个章节，包括：事实陈述、法律分析、证据清单、诉讼请求。提供各章节的写作要点和结构指引，帮助非律师用户准备诉讼材料。

本技能辅助起草，不替代律师撰写完整文书。

## 法域假设

默认中国大陆法域，适用《民事诉讼法》文书格式和《最高人民法院关于民事诉讼证据的若干规定》`[SME 核查]`。

## 各章节写作指引

### 事实陈述（事实与理由章节）

**写作要点**：
- 按时间顺序叙述，条理清晰
- 每个事实点需有证据支撑（旁注证据编号）
- 事实与法律分析分开叙述
- 语言客观，不夸大，不情绪化

**结构示例**：
```
二、事实与理由

（一）合同签订背景
[时间]，[当事人]与[对方当事人]签订[合同名称]（证据X）。
合同约定：[核心条款]。

（二）合同履行情况
[时间]，[我方]按约履行了[义务]（证据X）。
[时间]，[对方]未按约[行为]，构成违约（证据X）。

（三）损害结果
因[对方]违约，[我方]遭受如下损失：[损害描述]（证据X）。
```

### 法律分析

**写作要点**：
- 每个法律论点对应一个事实
- 引用具体法条和司法解释 `[SME 核查]`
- 回应对方可能的抗辩

**结构示例**：
```
（一）对方构成[违约/侵权]，依据[法条]
[事实描述]符合[法律要件]，理由如下：
1. [要件1]：[事实+证据]
2. [要件2]：[事实+证据]

（二）我方诉请有法律依据
依据[法条]，[诉求内容]于法有据。
```

### 证据清单

**结构**：
```
证据清单：

证据一：[证据名称]
证明内容：[简述]
形式：原件/复印件
证据编号：[自编证据编号]

证据二：[...]
```

### 诉讼请求

**写作要点**：
- 诉讼请求须具体、明确
- 金额类请求须列明计算方式
- 诉讼请求须与事实和法律依据对应

**示例**：
```
诉讼请求：

一、判令被告支付货款人民币[___]元，及逾期付款利息人民币[___]元（自[起始日期]至实际支付日止，按同期LPR计算）；

二、判令被告承担本案诉讼费用。
```

## 升级决策门

> "诉讼文书的起草质量直接影响案件结果。重要案件的一审起诉状、答辩状、上诉状，建议由诉讼律师审核或撰写。这不构成法律意见。"

## 本技能不涵盖

- **撰写完整诉讼文书**
- **确认法律依据的准确性** `[SME 核查]`
- **代理诉讼或仲裁**
