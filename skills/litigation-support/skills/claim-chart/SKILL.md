# /claim-chart — China Mainland

## Purpose

> **本图表仅为律师分析草稿，不是诉状或法律意见。** 每个映射都是律师须验证的线索。要素来自模式 Jury instructions / Restatement / 权利要求语言——管辖法院的权威依据可能不同，始终以权威依据为准。

## CN民事要素图表

### 合同纠纷要素（民法典第577条）

| 要素 | 说明 |
|---|---|
| 合同成立 | 存在有效合同 |
| 合同义务 | 对方有合同义务 |
| 违约事实 | 对方未履行义务 |
| 因果关系 | 违约与损失有因果关系 |
| 损失 | 存在实际损失 |

---

### 侵权纠纷要素（民法典第1165条）

| 要素 | 说明 |
|---|---|
| 加害行为 | 存在侵权行为 |
| 主观过错 | 故意或过失 |
| 损害事实 | 存在实际损害 |
| 因果关系 | 侵权行为与损害有因果关系 |
| 违法性 | 行为具有违法性 |

---

### 劳动争议解除合法性要素

| 要素 | 说明 |
|---|---|
| 解除理由 | 用人单位有法定解除理由 |
| 程序合规 | 已履行通知工会/通报等程序 |
| 经济补偿 | 已支付经济补偿金 |
| 社保转移 | 已办理社保转移 |

## CN专利权利要求图表

### 侵权判断框架（专利法第11条、第64条）

| 步骤 | 内容 |
|---|---|
| 1. 确定保护范围 | 独立权利要求1的所有技术特征 |
| 2. 识别被诉产品特征 | 被诉产品的技术方案 |
| 3. 全面覆盖原则 | 是否覆盖所有必要技术特征 |
| 4. 等同原则 | 是否以基本相同的方式实现基本相同的功能达到基本相同的效果 |

## 缺口检测

**民事案件缺口列表：**
- 缺少证据支持的要素
- 事实不清楚的要素
- 须进一步取证的要素

**专利案件缺口列表：**
- 缺乏证据支持的权利要求映射
- 须进一步现有技术检索的要素
- 须权利要求解释确认的术语
