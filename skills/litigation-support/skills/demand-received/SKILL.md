# 律师函 / 催告函分析

## 目的

分析收到的律师函、催告函或其他法律函件，识别对方诉求、评估法律依据、寻找谈判和抗辩空间。本技能辅助决策是否回应和如何回应，不代理谈判或诉讼。

## 法域假设

默认中国大陆法域，适用《民法典》和相关程序法 `[SME 核查]`。

## 加载信息

- 函件原文
- 发送方信息
- 截止日期（如有）
- 合同关系背景

## 工作流程

### 第一步：提取函件核心信息

| 信息项 | 内容 |
|--------|------|
| 发函方 | [___] |
| 发函依据 | [合同/侵权/其他] |
| 诉求内容 | [___] |
| 截止日期 | [如有] |
| 要求金额 | [如有] |

### 第二步：分析法律依据

- **合同依据** — 是否有合同关系，条款是否支持对方诉求
- **法律依据** — 适用什么法条，是否在诉讼时效内 `[SME 核查]`
- **事实基础** — 事实是否充分，举证责任如何分配

### 第三步：识别抗辩空间

| 抗辩类型 | 是否适用 | 说明 |
|---------|---------|------|
| 诉讼时效抗辩 | 是/否 | 已过诉讼时效可作为抗辩 `[SME 核查]` |
| 对方违约在先 | 是/否 | 如对方先违约则其诉求受阻 |
| 情势变更 | 是/否 | 合同基础发生重大变化 |
| 责任限制条款 | 是/否 | 合同约定了责任上限 |
| 违约金过高 | 是/否 | 可申请法院酌减违约金 |

### 第四步：评估是否需要回应

| 情况 | 建议 |
|------|------|
| 有截止日期且在时效内 | 须在期限内书面回应 |
| 无截止日期 | 建议书面回应以保留谈判空间 |
| 诉求明显无理 | 可不回应或直接拒绝 |
| 涉及诉讼时效 | 无论如何须书面回应以免失权 |

### 第五步：升级决策门

> "律师函的回应涉及诉讼时效和权利保留，在期限届满前建议由律师审核回应内容。重要的法律函件不应随意回应或忽视，应对策略建议由专业律师制定。这不构成法律意见。"

## 本技能不涵盖

- **代理谈判或起草正式回应函**
- **确认诉讼时效** — 须由律师确认 `[SME 核查]`
