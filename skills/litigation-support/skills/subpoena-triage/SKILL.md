# /subpoena-triage

1. 读取传票/调令原文，确认类型和发出机构。
2. 核实收到方与案件的关系，确认是否有配合义务。
3. 识别抗辩事由（特权保护/文件不存在/范围过宽）。
4. 评估不配合的法律后果。
5. 生成响应策略和行动时间表。
6. 升级决策门。

# 传票 / 调查令分拣

## 目的

分析收到的传票、调查令或律师取证请求，识别响应策略和风险。覆盖：文件出示要求、作证要求、跨境证据调取、保密特权抗辩、不配合的法律后果。

本技能不代理诉讼，但辅助准备响应策略。

## 法域假设

默认中国大陆法域，适用《民事诉讼法》证据章和司法协助规定 `[SME 核查]`。涉及境外法院传票的，适用《海牙取证公约》或双边司法协助协定 `[SME 核查]`。

## 加载信息

- 传票/调令原文（用户提供或路径）
- 发出机构（法院/仲裁机构/对方律师）
- 要求的内容和时间范围
- 截止日期

## 工作流程

### 第一步：确认传票类型和性质

| 类型 | 说明 | 配合义务 |
|------|------|---------|
| 法院调查令 | 法院主动调取证据 | 法定配合义务 `[SME 核查]` |
| 当事人申请的文件调取 | 对方律师向法院申请后发出 | 可提出异议 |
| 仲裁取证请求 | 仲裁程序中的文件请求 | 按仲裁规则处理 |
| 律师函附带取证要求 | 诉讼前的取证试探 | 通常无强制力 |

### 第二步：识别抗辩事由

评估以下抗辩是否适用：

- **保密特权** — 律师-客户特权、医生-患者特权、商业秘密保护
- **文件不存在** — 特定文件从未被创建或已销毁（注意法律保留义务）
- **范围过宽** — 要求过于宽泛，超出案件争议范围 `[SME 核查]`
- **第三方权利** — 文件涉及第三方隐私或商业秘密
- **境外限制** — 文件在境外，受出口管制或当地法律限制 `[SME 核查]`

### 第三步：评估不配合后果

- 法院调查令不配合：可能面临罚款、拘留，或法院采纳对方主张
- 律师取证请求不配合：对方可向法院申请强制令
- 仲裁程序：按仲裁规则处理，可能影响仲裁裁决的承认

### 第四步：生成响应策略

```
响应选项：
A. 完全配合 — 按要求提供文件/出庭
B. 部分配合 — 提出范围异议，只提供相关部分
C. 拒绝配合 — 援引特权保护，提交书面说明
D. 申请延期 — 以时间紧迫为由申请延期

建议时间表：
- [截止日期-X天]：准备响应文件
- [截止日期-Y天]：内部审核
- [截止日期]：提交响应或提出异议
```

### 第五步：升级决策门

> "传票和调查令具有法律强制力，不配合可能导致罚款、拘留或败诉风险。在提交任何文件或出庭前，建议由诉讼律师审核响应策略。这不构成法律意见。"

**以下情形须立即升级：**
- 涉及法院调查令（强制配合）
- 涉及刑事侦查相关取证
- 涉及跨境证据调取

## 本技能不涵盖

- **代理诉讼或仲裁**
- **向境外法院提交文件**（涉及国际司法协助）`[SME 核查]`

- **确认法律效力** — 所有结论标注 `[SME 核查]`
