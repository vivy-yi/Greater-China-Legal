# 诉讼/仲裁费用估算

## 目的

估算中国大陆诉讼或仲裁的大致费用和周期，辅助评估是否值得诉讼。覆盖：法院诉讼费、律师费（市场参考）、保全费、鉴定费、仲裁费等各类费用，以及案件审理周期预估。

本技能提供费用估算参考，实际费用因案件复杂度和地区差异可能不同。

## 法域假设

默认中国大陆法域。涉及诉讼费用计算适用《诉讼费用交纳办法》，律师费参考《律师服务收费管理办法》（市场调节价为主）。

## 加载信息

- 争议金额
- 案件类型（合同纠纷/侵权纠纷/婚姻继承等）
- 审理法院（一审/二审/再审；基层/中级/高级/最高院）
- 是否需要申请保全
- 是否需要仲裁（部分纠纷适用仲裁前置）

## 工作流程

### 第一步：计算法院诉讼费 `[SME 核查]`

财产类案件受理费按比例计算：

| 争议金额区间 | 收费标准 |
|-------------|---------|
| 1万元以下 | 50元 |
| 1万-10万元 | 2.5% - 200元 |
| 10万-50万元 | 2% + 2300元 |
| 50万-100万元 | 1.5% + 10300元 |
| 100万-200万元 | 1% + 15300元 |
| 200万-500万元 | 0.9% + 25300元 |
| 500万-1000万元 | 0.8% + 40300元 |
| 1000万以上 | 0.7% + 70300元 |

**其他费用**：
- 保全申请费：保全金额的0.5%，最高5000元
- 执行申请费：不收费（执行阶段由被执行人承担）

### 第二步：估算律师费（市场参考）`[SME 核查]`

律师费实行市场调节价，以下为常见参考区间：

| 服务类型 | 收费标准 |
|---------|---------|
| 全程代理（一审+二审+执行）| 争议金额的3%-10% |
| 一审代理 | 争议金额的2%-5% |
| 二审代理 | 一审收费的50%-80% |
| 单次咨询 | 1000-5000元/小时 |

**风险代理**：部分案件可采用"胜诉后按结果收费"模式，费率可达执行金额的20%-30%。

### 第三步：估算其他费用

| 费用类型 | 估算 |
|---------|------|
| 保全费 | 保全金额×0.5%（上限5000元）|
| 鉴定费 | 5000-50000元（视鉴定类型）|
| 公证费 | 200-2000元 |
| 公告费 | 几百元 |
| 仲裁费 | 仲裁机构按标的金额比例收取 |

### 第四步：估算案件周期 `[SME 核查]`

| 审理阶段 | 正常周期 |
|---------|---------|
| 一审 | 6个月（简易程序）/ 12个月（普通程序）|
| 二审 | 3个月 |
| 再审 | 6-12个月 |
| 执行 | 6个月内（有财产可执行时）|

### 第五步：输出总成本估算

```
费用估算汇总（争议金额[___]元）：
| 费用类型 | 金额 |
|---------|------|
| 法院受理费 | [___]元 |
| 律师费（一审）| [___]元（参考）|
| 保全费 | [___]元 |
| 鉴定费 | [___]元（参考）|
| **合计** | **[___]元（参考）** |

周期估算：
- 一审：[___]个月
- 二审（如有）：[___]个月
- 执行（如有）：[___]个月
- **总周期：[___]个月（参考）**
```

## 本技能不涵盖

- **精确律师费报价**（各律所差异大）
- **诉讼费减免申请**
- **法律援助申请**
- **确认具体费用** — 以法院/仲裁机构/律所实际收费为准 `[SME 核查]`
