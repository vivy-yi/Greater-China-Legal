## 对赌协议判断规则

### 效力判断（九民纪要2019第5条）

| 情形 | 效力 | 说明 |
|---|---|---|
| 目标公司与投资方对赌 | ✅ 有效 | 约定业绩目标，通常有效 |
| 目标公司股东与投资方对赌 | ✅ 通常有效 | 实践中一般认定有效 |
| 固定收益回报（名投资实借贷） | ❌ 可能无效 | 可能被认定为借贷关系 |
| 违反法律强制性规定 | ❌ 无效 | 按具体条款判断 |

### 调整机制建议

```
建议：设置估值调整上限（如"补偿不超过投资额的X倍"）
避免：固定回报回购条款（可能被认定无效）
```

## 外商投资准入判断

### 负面清单查询

```
[GOV] 国家发改委：http://www.ndrc.gov.cn/
[GOV] 商务部：https://www.mofcom.gov.cn/
关键词："外商投资准入特别管理措施（负面清单）"
```

### 行业分类

| 分类 | 说明 | VIE是否可行 |
|---|---|---|
| 禁止类 | 绝对禁止外资 | VIE为实践做法（但法律上存在风险） |
| 限制类 | 外资比例受限或须合资 | 可考虑合资或VIE |
| 鼓励类 | 一般无限制 | 直接外资即可 |
| 未列入负面清单 | 允许外资 | 直接外资即可 |

## ODI备案流程

### 须办理ODI备案的情形

| 情形 | 是否须备案 |
|---|---|
| 境内企业向境外设立公司 | ✅ 须备案 |
| 境内企业向境外汇出投资款 | ✅ 须外汇登记 |
| 37号文登记 | 须（个人境外投资）或可不（企业境外投资） |

### 主管机关

| 机关 | 职责 |
|---|---|
| 商务部/发改委 | ODI备案/核准 |
| 外管局 | 外汇登记 |
| 36号文登记（个人） | 个人境外投资备案 |

## 股权激励规则

### 中国法律下的股权激励

| 类型 | 法律依据 | 说明 |
|---|---|---|
| 员工期权 | 无直接法律依据 | 通常通过持股平台或代持实现 |
| 限制性股票 | 无直接法律依据 | 实践中有各种变通做法 |
| 虚拟股 | 无直接法律依据 | 内部激励，非真正股权 |

### 税务考虑

```
行权时：按"工资薪金所得"计征个人所得税
转让时：按"财产转让所得"计征个人所得税
建议：咨询专业税务律师
```
