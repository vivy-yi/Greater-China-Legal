# 股权转让限制与优先购买权咨询 (Transfer Restriction & ROFR Advisor)

## 1. 目的与范围

### 1.1 核心目的
就有限责任公司股权/股份转让限制及优先购买权提供法律咨询，辅助识别风险、设计条款结构和评估争议解决方案。

### 1.2 适用范围
- 有限责任公司股权转让限制条款
- 股份有限公司股份转让限制安排
- 股东优先购买权（ROFR/ROFO）设计与行使
- 公司章程转让限制条款审核
- 私募/风投协议中的转让限制条款
- 员工持股平台股份转让限制

### 1.3 本Skill不涵盖
- 商业条款谈判（价格、估值）
- 国有资产转让特殊程序（见regulatory-approval-tracker skill）
- 上市公司重大资产重组
- 税务规划（见estate-tax-planner skill）
- 跨境ODI/返程投资架构

## 2. 法律框架

### 2.1 有限责任公司股权转让限制

**法律依据：公司法（2023修订）第84条**

| 情形 | 法律规定 | 章程另有规定 |
|-----|---------|-------------|
| 公司内部转让 | 自由转让，无需其他股东同意 | 可以限制 |
| 对外转让 | 应经其他股东过半数同意 | 可以限制，但不得禁止 |
| 优先购买权 | 其他股东在同等条件下享有优先购买权 | 可以扩大优先购买权范围 |
| 强制执行转让 | 法院强制执行参照对外转让处理 | 可以约定 |

**司法解释要点（公司法司法解释四第11条-第19条）：**
- 优先购买权行使期限：章程规定或书面通知后30日内
- "同等条件"认定：应综合考虑价格、支付方式、期限等因素
- 损害赔偿：未通知或侵害优先购买权需赔偿因此造成的损失
- 继承/赠与：章程可以规定是否适用优先购买权

### 2.2 股份有限公司股份转让限制

| 公司类型 | 转让限制 | 法律依据 |
|---------|---------|---------|
| 发起人股份 | 发起人持股锁定期1年 | 公司法第141条 |
| 董事/监事/高管 | 在职期间每年转让不超过25%，离职后6个月内不得转让 | 公司法第141条、第142条 |
| 上市公司股份 | 按证券法及减持规则执行 | 上市公司股东减持股份实施细则 |
| 限售股 | 按招股说明书/协议约定执行 | 相关协议 |

### 2.3 优先购买权类型

```
优先购买权类型：
│
├── 优先购买权（ROFR = Right of First Refusal）
│   ├── 转让股东拟对外转让时触发
│   ├── 其他股东可按同等条件优先购买
│   └── 最常见的优先购买权类型
│
├── 优先认购权（ROFO = Right of First Offer）
│   ├── 转让股东拟对外转让前须先通知其他股东
│   ├── 其他股东可选择按同等条件收购
│   └── 若其他股东不行使，转让股东可转让给第三方
│
└── 随售权（Tag-Along）与拖售权（Drag-Along）
    ├── 随售权：控股股东转让时，少数股东可参与
    └── 拖售权：满足条件时，控股股东可强制少数股东一起出售
```

## 3. 条款设计框架

### 3.1 转让限制条款核心要素

```表格：转让限制条款要素

| 要素 | 常见约定 | 注意事项 |
|-----|---------|---------|
| 限制对象 | 全部股东/特定股东/特定持股比例以上 | 避免过度限制导致股权僵局 |
| 触发条件 | 对外转让/内部转让/特定情形 | 明确"对外转让"定义 |
| 审批程序 | 股东会决议/董事会决议/书面通知 | 符合公司法规定 |
| 限制期限 | 永久/期限约定/特定事件发生前 | 考虑锁定期与退出需求 |
| 豁免情形 | 继承/赠与/内部重组/司法执行 | 明确豁免条件 |
| 违约后果 | 转让无效/违约金/强制回购 | 避免约定转让无效的无效后果 |
```

### 3.2 优先购买权条款设计

**标准ROFR条款结构：**

```
【优先购买权条款模板】

一、触发条件
1. 任何股东（下称"拟转让股东"）拟将其持有的公司股权转让予任何第三方（下称"拟受让方"）时，应事先按本条规定程序进行。

二、通知义务
1. 拟转让股东应提前[30]日书面通知公司及其他股东（下称"优先购买权人"），说明拟转让股权的数量、价格、受让方主要情况及主要交易条件。

三、优先购买权行使
1. 优先购买权人有权在收到通知后[30]日内以书面形式表示同意按同等条件购买拟转让的全部或部分股权。
2. 如有多名优先购买权人行使优先购买权，各方可按其持股比例分配拟转让股权。

四、同等条件认定
1. "同等条件"是指与拟受让方所获交易条件实质相同，综合考虑价格、支付方式、支付期限、附随义务等因素。
2. 转让股东应向优先购买权人提供完整的交易条件信息。

五、放弃与后续转让
1. 优先购买权人未在规定期限内行使优先购买权的，视为放弃。
2. 放弃后，转让股东可在[90]日内按不优于向优先购买权人提出的条件完成转让。
3. 如实际交易条件发生重大变更，应重新通知。

六、例外情形
以下转让不适用优先购买权：
（1）公司内部股东之间的转让；
（2）因继承、离婚财产分割等法定原因导致的股权转让；
（3）经全体股东事先书面同意的特定转让；
（4）司法机关强制执行程序中的转让。

七、违约责任
1. 违反本条约定进行转让的，违约股东应向公司支付违约金人民币[_____]元，并赔偿因此造成的全部损失。
2. 违约转让不影响转让行为的效力，但不影响违约方承担赔偿责任。
```

### 3.3 风险评估矩阵

```
股权转让限制条款风险评估：

条款类型          │ 风险等级 │ 常见问题 │ 建议
----------------│--------│--------│------
限制过严（禁止转让）│ 🔴红色   │ 可能被认定无效 │ 仅在法律允许范围内限制
限制期限过长     │ 🟠橙色   │ 影响股东退出 │ 建议设置合理期限（如3-5年）

```

## 4. 争议解决指引

### 4.1 优先购买权纠纷类型

```
优先购买权常见争议：

一、通知瑕疵争议
   ├── 未发出书面通知
   ├── 通知内容不完整（缺少价格/条件）
   └── 通知期限不足

二、同等条件认定争议
   ├── 价格评估方法分歧
   ├── 附随条件认定差异
   └── 信息披露不充分

三、行使程序争议
   ├── 超过法定期限行使
   ├── 部分行使优先购买权
   └── 连续多次转让情形处理

四、章程条款效力争议
   ├── 禁止转让条款效力
   ├── 加重转让限制条款效力
   └── 与合同条款冲突处理
```

### 4.2 司法裁判要点

**公司法司法解释四第16-19条核心规则：**

| 争议类型 | 裁判规则 |
|---------|---------|
| 未通知优先购买权人 | 转让股东应赔偿其他股东因此受到的损失 |
| 优先购买权人放弃后条件变更 | 应重新通知，条件重大变变更应重新给予优先购买权 |
| 部分行使优先购买权 | 拟转让股东可拒绝部分行使，要求全部行使或全部放弃 |
| 继承与优先购买权冲突 | 章程有规定从章程；章程无规定，继承人可主张权利 |
| 强制执行中的优先购买权 | 法院应通知其他股东，其他股东可申请优先购买权执行 |

### 4.3 争议解决方案

```
【优先购买权争议解决方案】

方案一：协商调解
适用情形：转让条件存在争议，双方有继续合作意愿
步骤：1. 明确各方核心诉求
     2. 委托第三方评估价格
     3. 协商调整转让条件
     4. 达成书面和解协议

方案二：诉讼/仲裁
适用情形：一方明确违约，无法协商解决
步骤：1. 评估胜诉可能性
     2. 评估诉讼成本与时间
     3. 提起诉讼或申请仲裁
     4. 申请财产保全

方案三：方案三：替代交易安排
适用情形：转让确有必要，但优先购买权存在争议
步骤：1. 探索其他交易结构
     2. 调整受让方范围
     3. 重新设计转让路径
     4. 确保合规前提下完成交易
```

## 5. 上市公司特殊规则

### 5.1 股份减持规则要点

| 减持方式 | 限制规定 | 适用对象 |
|---------|---------|---------|
| 集中竞价 | 90日内减持比例不超过1% | 大股东、董监高 |
| 大宗交易 | 90日内减持比例不超过2% | 大股东 |
| 协议转让 | 单次转让比例不低于5% | 特定股东 |
| 定向增发 | 锁定期12个月 | 特定对象 |

### 5.2 上市公司ROFR条款注意事项

- 上市公司股份转让需遵守证券法及减持规则
- 股份转让限制不得与减持规则冲突
- 违规转让可能触发监管措施
- 信息披露要求需同步考虑

## 6. 升级决策门

### 6.1 必须升级的情形

1. **章程与协议冲突**
   - 公司章程条款与股东协议约定不一致
   - 可能影响股东权利实现

2. **限制条款效力存疑**
   - 拟约定禁止或变相禁止股权转让
   - 可能违反公司法强制性规定

3. **特殊主体股权转让**
   - 涉及国有股权、外资股权
   - 涉及金融、证券、保险等特殊监管行业

4. **优先购买权纠纷**
   - 其他股东主张优先购买权被侵害
   - 涉及较大金额或重要股东关系

5. **上市公司合规问题**
   - 涉及信息披露违规风险
   - 可能触发监管调查
