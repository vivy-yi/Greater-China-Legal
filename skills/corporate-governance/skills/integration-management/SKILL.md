# /integration-management — China Mainland

## CN整合工作流的特殊要求

### 国有企业并购特殊程序

**党委前置程序（《公司法》第69条、第121条）：**
- 国有企业重大经营事项须经党组织研究讨论
- 前置程序：党委会审议 → 董事会决议 → 股东会决议
- 涉及国有资产：须履行评估、备案、进场交易程序（《企业国有资产法》）

**职工安置：**
- 须职代会审议通过（涉及劳动关系重大变更）
- 经济补偿方案须符合《劳动合同法》
- 特殊岗位（井下/有毒有害/高空）须劳动能力鉴定

### 外商投资企业并购特殊程序

**审批要求（《外商投资法》）：**
- 外资并购须取得商务部门审批或备案
- 涉及经营者集中须申报反垄断审查
- 外资，实业投资须符合负面清单

## 分阶段工作计划

### Day 1（交割后7日内）

| 事项 | 负责方 | 备注 |
|---|---|---|
| 营业执照变更登记 | 法律部/工商代理 | 股东、法定代表人变更 |
| 公章/财务章移交 | 综合管理部 | 交接记录双方签字 |
| 银行预留印鉴更新 | 财务部 | 同步更新网银权限 |
| 社保公积金账户变更 | 人力资源部 | 劳动关系转移后 |
| 党委/工会组织关系 | 党群工作部 | 如适用 |

### Day 30（交割后30日内）

| 事项 | 负责方 | 备注 |
|---|---|---|
| 董事会/监事会改选 | 法律部 | 依据章程和出资比例 |
| 章程修订备案 | 法律部 | 工商变更登记 |
| 重大合同主体变更通知 | 业务部门 | 合同相对方书面同意 |
| 知识产权转让登记 | 知识产权部 | 商标/专利/著作权 |
| 银行贷款合同变更 | 财务部 | 需取得银行同意 |

### Day 90（交割后90日内）

| 事项 | 负责方 | 备注 |
|---|---|---|
| 业务整合方案确定 | 业务部门 | 战略+运营+财务 |
| 信息系统对接 | IT部门 | 数据迁移+权限管理 |
| 员工劳动关系整合 | 人力资源部 | 统一薪酬福利体系 |
| 税务合规整合 | 财务部 | 税目/税率/优惠申请 |
| 内控制度统一 | 法律部/审计部 | 治理制度合规对接 |

### Day 180（交割后180日内）

| 事项 | 负责方 | 备注 |
|---|---|---|
| 品牌整合 | 市场部/品牌部 | 商标许可协议 |
| 供应商/客户合同重签 | 采购/销售部 | 合同主体变更 |
| 资产处置 | 资产管理部 | 闲置/重复资产 |
| 企业文化整合 | 人力资源部 | 沟通机制建立 |
| 全面合规审计 | 法律部/审计部 | 交割后100天审计 |

## CN合同移交重点

### 须通知相对方的合同

| 合同类型 | 是否须通知/取得同意 | 法律依据 |
|---|---|---|
| 房屋租赁合同 | 须书面通知+新主体重新签署 | 民法典第702条 |
| 银行借款合同 | 须取得银行书面同意 | 银行业监管规定 |
| 担保合同 | 须担保人书面同意 | 民法典第695条 |
| 劳动人事合同 | 须劳动者同意（工作年限合并计算）| 《劳动合同法》第34条 |
| 供应/采购框架协议 | 须相对方书面同意 | 民法典第551条 |
| 知识产权许可 | 须许可方书面同意 | 知识产权相关法律 |
