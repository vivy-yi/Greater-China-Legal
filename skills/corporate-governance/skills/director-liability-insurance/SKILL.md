# 董事高管责任险咨询 (Director & Officer Liability Insurance)

## 1. 目的与范围

### 1.1 核心目的

为上市公司及非上市公众公司的董事、监事、高级管理人员提供责任风险保障方面的
咨询参考，协助法务部门、合规团队或董事会秘书理解D&O保险的基本框架、投保要点
及常见免责情形。

### 1.2 适用范围

- A股上市公司、H股上市公司
- 非上市公众公司（如新三板精选层公司）
- 拟上市公司（上市前过渡期安排）
- 大型有限责任公司（尤其是引入外部投资者或有多层股权结构的公司）

### 1.3 本Skill不涵盖

- 保险公司具体产品的比价与推荐（应通过持牌保险经纪进行）
- 董监高个人的税务规划
- 公司本身的产品责任险、雇主责任险等（见其他专项技能）
- 海外上市主体的D&O保险安排（需另行研究适用司法管辖区的法律）

## 2. 法律框架

### 2.1 相关法律依据

**《公司法》（2023年修订）**
- 第180条：董事、监事、高级管理人员对公司负有忠实义务和勤勉义务
- 第181条至第188条：董事、高管的禁止行为（包括关联交易未披露、同业竞争等）
- 第189条：董事高管对公司的损害赔偿责任
- 第192条：关联交易中董事高管的赔偿责任

**《证券法》（2020年修订）**
- 第78条、第79条：信息披露义务及违规处罚
- 第85条：信息披露违法行为的民事赔偿责任（包括董事高管的连带责任）
- 第120条至第125条：证券服务机构责任

**《刑法》（2023年修正）**
- 第169条之一：背信损害上市公司利益罪（董事、高管将公司利益转移至他人，致使上市公司利益遭受重大损失的）

**《上市公司治理准则》（2018年修订）**
- 第56条至第61条：董事、监事、高级管理人员的行为规范
- 第72条：鼓励上市公司建立董事责任保险制度

**《上市公司信息披露管理办法》（2021年修订）**
- 董事、监事、高级管理人员对信息披露真实性、准确性、完整性的保证义务

### 2.2 司法实践要点

- 董事高管因未尽勤勉义务导致的民事赔偿，法院通常结合具体案情和过错程度综合认定赔偿范围
- 证券民事赔偿诉讼中，法院可判决董事高管与上市公司承担连带赔偿责任（《证券法》第85条）
- 刑事追责中，个人主观故意或重大过失是认定罪责的核心要素

## 3. D&O保险产品概述

### 3.1 基本概念

董事高管责任险（Directors & Officers Liability Insurance，简称D&O保险）：
承保董事、监事及高级管理人员在履行职务过程中因过失、疏忽、误导性陈述等
行为导致第三方（通常为股东、债权人、监管机构等）遭受损失，依法应承担
民事赔偿责任时，由保险公司代为赔付的保险产品。

### 3.2 保障类型

**Side A：个人保障**
- 保障董监高个人因履职行为导致的索赔
- 当公司无力偿还时，直接赔付个人
- 典型场景：上市公司资不抵债时

**Side B：公司补偿保障**
- 当公司依法对董监高进行补偿时，保险公司向公司补偿
- 典型场景：公司主动补偿被诉董监高

**Side C：证券赔偿保障**
- 保障公司因证券欺诈或误导性陈述导致的索赔
- 主要针对上市公司的公开披露文件

### 3.3 常见承保范围

| 保障事项 | 典型情形 |
|---|---|
| 决策过失 | 投资决策失误、并购决策不当 |
| 披露不实 | 年度报告、招股说明书重大遗漏或误导性陈述 |
| 违反信义义务 | 关联交易未披露、利用职务便利牟利 |
| 监管调查应对 | 证监会立案调查期间的法律费用 |
| 股东派生诉讼 | 股东代表公司提起诉讼 |
| 竞业禁止违约 | 董事、高管违反竞业禁止义务 |

### 3.4 常见免责事项（重点提示）

以下情形通常不属于D&O保险赔付范围（以具体保险合同条款为准）：

- **故意违法行为**：董事高管已知行为违法仍为之（如内幕交易）
- **身体伤害与有形财产损失**：属于财产险或责任险范畴
- **污染/环境损害**：须附加环境责任险
- **合同约定责任**：保险合同明确排除的合同违约责任
- **罚款与罚金**：多数D&O保险不赔付监管机构的行政处罚罚款（部分产品可附加）
- **欺诈或故意犯罪**：保险公司可在刑事判决生效后拒赔
- **已知索赔**：投保时已知的潜在索赔（须如实告知）

## 4. 投保与尽调要点

### 4.1 投保前准备

**公司层面**
1. 梳理公司章程、董事会议事规则中关于董监高责任的规定
2. 评估公司所处行业及监管环境（如金融、医疗、教育等高风险行业）
3. 审查公司历史涉诉或监管记录
4. 确定投保额度（Coverage Limit）和免赔额（Deductible）

**董监高个人层面**
1. 逐一审阅各董监高的提名文件和任职条件
2. 评估各人过往行为是否存在潜在风险点
3. 确认是否已持有其他相关保险（如职业责任险E&O）

### 4.2 投保时如实告知义务

依据《保险法》第16条，投保人在投保时对保险公司提出的有关问题必须如实告知。
未尽如实告知义务，可能导致保险公司在理赔时解除合同或拒绝赔付。

常见告知事项：
- 公司近3-5年涉诉情况
- 监管调查或行政处罚记录
- 重大并购、重组或融资事项
- 财务报表重大调整或重述
- 董事高管的犯罪记录及重大民事诉讼

### 4.3 保单关键条款审查

**通知条款（Notice）**
- 出险后须在约定时间内通知保险公司，否则可能影响赔付

**抗辩费用（Defense Costs）**
- 多数D&O保险将律师费等抗辩费用计入赔付范围（不占用保障额度）

**赔偿限额与分项限额（Aggregate Limit & Sublimits）**
- 主险：年度或单次赔付上限
- 附加险：特定情形（如SEC调查费用）可能有单独上限

**等待期（Discovery Period）**
- 保险期限届满后发现行为的发现期（通常可付费延长）

## 5. 理赔流程与注意事项

### 5.1 一般理赔流程

```
出险 → 立即通知保险公司 → 保险公司指派律师 → 协同抗辩/和解谈判
→ 提供损失证明 → 保险公司核定赔付 → 赔付到位
```

### 5.2 出险后关键动作

1. **第一时间通知保险公司**
   - 不要自行与原告和解或承认责任
   - 通知后由保险公司协调律师介入

2. **保全证据**
   - 保留董事会会议记录、决策文件、财务凭证
   - 重大事项的决策链和授权文件

3. **配合保险公司调查**
   - 保险公司有权对索赔进行调查
   - 故意隐瞒或毁灭证据可能导致拒赔

4. **不得私自承诺赔付**
   - 未征得保险公司同意，公司或个人不得向索赔方承诺赔偿

### 5.3 常见理赔争议

- **免责条款适用争议**：保险公司主张属于免责事项
- **赔偿金额核定争议**：损失认定与保险合同约定不符
- **故意违法行为认定争议**：民事赔偿责任与刑事判决的关联性
- **如实告知争议**：投保时未告知事项是否影响理赔

## 6. 上市公司特殊考量

### 6.1 信息披露要求

根据《上市公司信息披露管理办法》及相关监管规则：

- 购买D&O保险的议案属于董事会决议事项，上市公司应当进行信息披露
- 保险合同的主要条款（保费、保险金额、承保范围摘要）应在年度报告中披露
- 保险公司为国有控股上市公司提供保险服务，可能涉及国有资产管理要求

### 6.2 上市公司治理准则的鼓励性规定

《上市公司治理准则》第72条明确鼓励上市公司建立董事责任保险制度。
实践中，A股上市公司D&O保险覆盖率逐年提升，尤其在以下情形：
- 公司正处于监管调查期间
- 公司曾发生重大合规风险事件
- 公司近期完成IPO或再融资

### 6.3 科创板、创业板特殊要求

科创板、创业板上市公司因其高成长性和不确定性，D&O保险覆盖率较高。
部分保荐机构在持续督导阶段也会建议公司完善D&O保险安排。

## 7. 常见问题（FAQ）

**Q1：D&O保险保费大概是多少？**
保费取决于公司规模、行业风险、董监高人数、赔偿限额及历史索赔记录。
A股上市公司年保费通常在50万至300万人民币不等（具体以保险经纪询价结果为准）。

**Q2：独立董事是否在D&O保险保障范围内？**
是。独立董事与其他董事享有同等保障，但部分保单对独立董事故意违法行为的免责约定可能不同。

**Q3：公司已为员工购买了雇主责任险，是否还需要D&O保险？**
需要。雇主责任险保障员工因工作遭受的人身伤害，D&O保险保障董监高履职过程中的民事赔偿责任，两者覆盖风险不同。

**Q4：保险公司拒绝承保时怎么办？**
可尝试更换保险公司或调整投保方案（如降低赔偿限额、减少附加险）；
部分高风险公司可能需要通过专属保险（Captive Insurance）或再保险安排。

**Q5：D&O保险能否覆盖刑事罚款？**
原则上D&O保险不赔付刑事罚金。若董事高管面临刑事追责，应另行安排个人资产保护措施。

**Q6：董监高离任后，保险是否仍然有效？**
部分D&O保险提供"发现期"（Discovery Period），在保险期限届满后一定时间内发现的索赔仍可赔付。
若离任董监高希望延续保障，可在投保时与保险公司协商加长发现期或单独购买延续险。
