## 三、案例库（公司治理场景）

### 3.1 最高人民法院指导案例 / 公报案例

```
URL: 人民法院案例库 https://rmfyalk.court.gov.cn
[Y D] 检索："指导案例 + 公司法"

公司治理必学案例：
- 股东会决议撤销（程序瑕疵）
- 异议股东回购请求权
- 派生诉讼
- 公司僵局（司法解散）
- 关联交易损害公司利益
- 股东出资纠纷
- 股权转让效力
- 股权代持效力
- 公司人格否认（揭开公司面纱）
```

### 3.2 各省市典型案例

```
[Y D] 检索："{省市} + 公司纠纷 + {类型}"

常见争议焦点：
- 股东会/董事会决议瑕疵

- 关联交易公允性
```
