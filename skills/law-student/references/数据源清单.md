# 法律学习数据源清单 — 中国大陆

> 本清单整理法律学习场景常用数据源，按 [YD]/[WKL]/[GOV]/[BD]/[model] 五级标注分类。
> 学习场景与实务场景的关键差异：法条**不要求最新修订**（教材稳定），但**真题/案例要求高质量**。

## 六、多源交叉验证（学习场景）

**关键结论须 2 个数据源确认**：

| 结论类型 | 必须验证的源 |
|---|---|
| 法条引用准确性 | [YD] + [GOV]国家法律法规数据库 |
| 司法解释有效 | [GOV]最高法 + [YD] |
| 案例事实 | [GYL]案例库 + 原文裁判文书 |
| 学派观点 | [BD]知网 + 教材 |

**学术观点特别提醒**：
- 引用学者观点须**核实原始文献**，避免二手转述
- 注明学派归属（如"张明楷教授主张客观主义立场"）
- 争议性观点标注"少数说"或"通说"

## 九、学习场景风险等级说明

law-student 全部 skill 风险等级为 `low`，因为：
1. 学习活动不直接产生法律后果
2. 输出仅供学习参考，不替代专业法律意见
3. 学生对结论有自主判断能力

**但下列情形需要警惕**：
- 引用过时法规（如民法典施行前已被替代的旧条款）
- 引用失效司法解释（部分司法解释被新解释替代）
- 引用地方性文件（地区差异可能导致结论不准确）
- 引用未经核实的学者观点（避免误导）
