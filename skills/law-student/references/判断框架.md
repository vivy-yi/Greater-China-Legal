# 法律学习判断框架 — 中国大陆

> 本框架用于 law-student 13 个 skill 的场景识别与判断逻辑，作为学习场景的统一依据。
> 不同于 employment-legal 的"工作流判断"，law-student 是"学习方法路由"。
> 各 skill 内部已有场景识别段，本文件描述跨 skill 的整体决策路由。

## 三、CN 法学体系地图（核心知识结构）

### 3.1 部门法与考试占比

| 部门法 | 核心法规 | 法考占比 | 学硕占比 | 特征 |
|---|---|---|---|---|
| 民法 | 民法典（2021） | ~25% | ~25% | 体系庞杂，重在体系理解 |
| 刑法 | 刑法典 | ~20% | ~20% | 理论+判例并重 |
| 行政法 | 行政诉讼法/行政处罚法 | ~15% | ~15% | 程序性强 |
| 民事诉讼法 | 民诉法（2023修正） | ~10% | ~10% | 流程繁琐 |
| 刑事诉讼法 | 刑诉法 | ~10% | ~10% | 与民诉对比 |
| 商法 | 公司法/证券法/破产法 | ~10% | ~10% | 与民法交叉 |
| 经济法 | 反垄断/消费者保护 | ~10% | ~10% | 多为规章 |

### 3.2 法考主观题命题规律

| 年份 | 民法 | 刑法 | 民诉 | 刑诉 | 商法 | 行政 |
|---|---|---|---|---|---|---|
| 2024 | 5/6 题 | 4/5 题 | 1-2 题 | 1-2 题 | 1 题 | 0-1 题 |
| 2023 | 4/5 题 | 4 题 | 2 题 | 1 题 | 1-2 题 | 1 题 |
| 2022 | 5 题 | 4 题 | 1 题 | 1 题 | 1-2 题 | 1 题 |

**判断**：民法+刑法合计约 50% 主观题分数，主观题重点在这两科。

### 3.3 三大诉讼法对比

| 维度 | 民诉 | 刑诉 | 行诉 |
|---|---|---|---|
| 主管 | 法院 | 公安/检察/法院 | 法院 |
| 当事人 | 原告/被告 | 公诉/被害人/被告人 | 原告/被告/第三人 |
| 举证责任 | 谁主张谁举证 | 控方证明 | 被告负举证 |
| 时效 | 3 年 | 不限 | 6 个月 |
| 调解 | 着重调解 | 公诉案件不可调解 | 着重调解 |

## 四、考试类型核心判断节点

### 4.1 法考 vs 学硕 vs JD（识别节点）

```
[节点 1] 目标考试？
├─ 法律职业资格考试（法考）→ S1 路径
│   ├─ 客观题（每年9月）
│   └─ 主观题（每年10月）
├─ 法学硕士/博士入学考试（学硕）→ S2 路径
│   ├─ 初试（每年12月）
│   │   ├─ 政治（100分）
│   │   ├─ 英语（100分）
│   │   ├─ 专业基础（150分，民法+刑法）
│   │   └─ 专业综合（150分，法理+宪法+法制史）
│   └─ 复试（次年3-4月）
└─ 中国 JD（法律硕士·法学）→ S3 路径
    ├─ 各校自主命题
    ├─ 案例教学法为主
    └─ 类似美国 1L 训练

[节点 2] 距考试时间？
├─ 12个月+ → 系统复习期
├─ 6-12个月 → 强化期
├─ 3-6个月 → 冲刺期
└─ < 3个月 → 重点突破期

[节点 3] 学生基础？
├─ 零基础 → 通识先修
├─ 已学完一轮 → 强化训练
└─ 多轮复习 → 模拟实战

[节点 4] 院校目标（学硕/JD）？
├─ 顶尖院校（北大/清华/人大等）→ 高难度
├─ 211 院校 → 中高难度
└─ 普通院校 → 中等难度
```

### 4.2 学习阶段判断

| 阶段 | 特征 | 主要 skill |
|---|---|---|
| 通识先修 | 部门法基础概念 | outline-builder / flashcards |
| 一轮复习 | 系统掌握学科框架 | outline-builder / case-brief |
| 强化训练 | 重点难点突破 | irac-practice / exam-forecast |
| 冲刺模拟 | 模拟考试 + 时间管控 | bar-prep-questions / session |
| 课程学习 | 学期内同步 | cold-call-prep / socratic-drill / case-brief |
| 论文写作 | 学术写作能力 | legal-writing |

## 五、知识框架判断（学科学习）

### 5.1 学科学习三层结构

```
法律学科学习
├── 概念层（concepts）
│   ├─ 基本概念（如"民事权利能力"）
│   └─ 法律术语（如"善意取得"）
│
├── 制度层（institutions）
│   ├─ 法律制度（如"物权变动"）
│   ├─ 构成要件（如"侵权责任四要件"）
│   └─ 法律效果
│
└── 理论层（theories）
    ├─ 学说争议（如"客观说 vs 主观说"）
    └─ 价值判断（如"意思自治 vs 公序良俗"）
```

### 5.2 大纲构建逻辑

```
Step 1：识别学科
  ├─ 8 大部门法任选

Step 2：选择深度
  ├─ 概念级（仅术语）
  ├─ 制度级（含要件）
  └─ 理论级（含学说）

Step 3：目标对齐
  ├─ 法考（侧重制度级）
  ├─ 学硕（侧重理论级）
  └─ JD（侧重案例分析 + 英美判例对照）

Step 4：交叉学科
  ├─ 民法 ↔ 商法（公司法、证券法）
  ├─ 民法 ↔ 民事诉讼法（程序衔接）
  └─ 刑法 ↔ 刑事诉讼法（程序衔接）
```

## 六、案例分析判断树（IRAC）

```
[输入] 案例描述
  ↓
[Issue] 争议焦点识别
  ├─ 单焦点 → 直接分析
  └─ 多焦点 → 排序（核心争点 → 次级争点）
  ↓
[Rule] 法律规则检索
  ├─ 法条依据（如《民法典》第 XXX 条）
  ├─ 司法解释（如最高院《民法典总则解释》）
  └─ 学说观点（参考价值）
  ↓
[Application] 法律适用
  ├─ 涵摄（subsumption）
  ├─ 利益衡量
  └─ 价值判断
  ↓
[Conclusion] 结论
  ├─ 明确结论
  ├─ 引用法条
  └─ 抗辩回应
```

**IRAC 训练要点**（law-student/irac-practice skill 内部判断）：
- 每个结论须有法条支持
- 分析须结合案件事实
- 逻辑清晰，层次分明
- 主观题作答须结构化（按 IRAC 段落）

## 七、跨场景通用判断

### 7.1 学生画像（cold-start-interview）

```
[节点 1] 你是哪类学生？
├─ 法学本科一年级 → 课程学习为主
├─ 法学本科二三年级 → 法考准备启动
├─ 法学本科四年级 → 法考冲刺 / 学硕准备

├─ 期中 → 课程+考试准备
├─ 期末 → 复习冲刺

[节点 3] 主攻方向？
├─ 通过法考 → S1
├─ 学硕考研 → S2
├─ JD → S3
└─ 课程学习 → S4

```

├─ 中文 → 默认

## 九、与其他 plugin 的差异

law-student 与 employment-legal（律师实务）、legal-clinic（法律援助）的根本差异：

| 维度 | law-student | employment-legal | legal-clinic |
|---|---|---|---|
| 目标用户 | 学生 | 律师/HR | 法援学生/律师 |
| 输出类型 | 知识传递 + 训练 | 法律意见 | 法律服务 |
| 数据需求 | 教材 + 真题 + 案例库 | 法规 + 实务案例 + 地方差异 | 法规 + 案件档案 + 当事人信息 |
| 时效要求 | 中（教材稳定） | 高（法规更新） | 高（时效紧迫） |
| 风险等级 | 低（学习不直接产生法律后果） | 高（影响实际案件） | 中-高（影响当事人） |

**注意**：law-student 的所有 skill 风险等级为 `low`，因为学习活动不直接产生法律后果。
