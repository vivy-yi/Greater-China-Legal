# 不良资产收购咨询

## 一、基本信息

### 1.1 核心目的
为不良资产投资者（AMC/AIC/秃鹫基金/私募基金/信托）提供资产包收购全流程法律支持，覆盖债权尽调、转让合规、诉讼时效排查、担保效力审查及退出路径设计。

### 1.2 适用范围
- 银行不良资产包（对公/零售/个贷）
- 信托/券商/基金子公司非标不良资产
- 融资租赁/保理不良资产
- 信用卡/消费金融不良资产
- 网贷/P2P不良资产（处置特殊规则）

### 1.3 数据源与判断框架引用

| 引用来源 | 用途 |
|---------|------|
| [YD] 元典MCP | 债务人涉诉/执行信息、担保物权登记查询 |
| [WKL] 裁判文书网 | 破产案例、债权确认案例、担保物权效力判例 |
| [GOV] 政府平台 | 企业信用信息公示、不动产登记、动产融资统一登记 |
| [web] 网络搜索 | AMC交易公告、不良资产包信息 |
| [model] 模型推理 | 回收率预测、诉讼时效中断计算（须核实） |

### 1.4 核心法规依据
- 《民法典》合同编（债权转让）、担保制度司法解释
- 《金融企业不良资产批量转让管理办法》（财金〔2012〕6号）

## 二、分析框架

### 2.1 不良资产收购全流程

```
资产包尽调
├── 债权真实性审查（借款合同/借据/对账单/催收记录）
├── 担保有效性审查（抵押登记/质押登记/保证人资质）
├── 诉讼时效排查（催收通知/还款承诺/部分履行）
├── 保证期间排查（保证合同约定的期间）
├── 查封/执行状态核查（首封/轮候/查封期限）
└── 债务人状态排查（在营/注销/吊销/破产）

         ↓

转让合规性审查
├── 转让主体资格（受让方为AMC/符合资质的社会投资者）
├── 批量转让户数限制（对公≤100户/个贷不限）
├── 公告程序（省级以上报刊/AMC官网）
├── 转让定价（不低于评估价/竞标程序）
└── 优先购买权排查（共有人/承租人）

         ↓

交割与诉权转移
├── 债权转让协议签署
├── 债权凭证移交（借款合同/担保合同/判决书等）
├── 债权转让通知（债务人）
├── 担保登记变更（如需）
├── 申请变更申请执行人（如有生效判决）
└── 资金交割
```

### 2.2 债权转让通知效力——核心审查点

| 通知方式 | 效力认定 | 操作要点 |
|---------|---------|---------|
| 书面通知（挂号信/EMS） | 有效（最常用） | 保留邮寄凭证+签收回执 |
| 公告通知（报纸/AMC官网） | 有效（金融不良资产批量转让适用） | 省级以上报刊或AMC官网 |
| 诉讼中通知（起诉状/举证通知） | 法院认可 | 须在法庭陈述中明确 |
| 受让人自行通知 | 有效（民法典第546条） | 须证明转让事实 |
| 未通知债务人 | 效力：不对债务人发生效力 | 债务人可向原债权人履行 |

**最高额抵押权转让特别处理**：
- 最高额抵押担保的债权确定前，部分债权转让的，最高额抵押权不得转让
- 批量转让中需确保主债权确定（决算期届满/债权确定事由发生）

### 2.3 诉讼时效排查矩阵

| 债权状态 | 时效状况 | 补救措施 |
|---------|---------|---------|
| 借款到期未超3年+有催收记录 | 时效未届满 | 直接执行 |
| 借款到期超3年+有催收记录 | 时效中断/重新计算 | 确认中断效力 |
| 借款到期超3年+无催收记录 | 时效届满（丧失胜诉权） | ①催收后债务人签收确认 ②债务人作出还款承诺 ③部分履行 |
| 分期还款/循环贷 | 逐笔计算/最后还款日起算 | 核对每笔到期时间 |
| 保证责任 | 保证期间已过=保证责任免除 | 确认主债权可执行即有价值 |

**诉讼时效中断的常见证据形式**：
- 债务人签章的催收通知/对账单
- 债务人部分还款的银行流水
- 债务人出具的还款计划/承诺函
- 债务重组协议签署
- 诉讼/仲裁立案受理通知书
- 债权转让通知的债务人回复

### 2.4 资产包定价参考框架

| 资产类型 | 正常回收率区间 | 关键定价因素 |
|---------|--------------|-------------|
| 有抵押对公贷款 | 30%-70% | 抵押物类型/位置/价值稳定性/首封法院 |
| 无抵押对公贷款 | 5%-20% | 债务人经营状况/追加被执行人可能性 |
| 个人住房抵押贷款 | 50%-80% | 房产价值/首套房/居住人清退难度 |
| 信用卡不良 | 1%-5% | 债务人还款能力/委外催收效果 |
| 汽车贷款 | 20%-40% | 车辆折价率/GPS可追溯性 |
| 小微企业贷 | 10%-30% | 担保人连带责任/联保体追偿 |

## 三、操作流程

### 3.1 资产包尽调标准流程

**Phase 1：审前准备（1周）**
- [ ] 获取资产包清单/债权明细
- [ ] 分类：有抵押/无抵押/有判决/有执行/纯债权
- [ ] 确定尽调范围（样本/全量）
- [ ] 组建尽调团队（法律+评估+信贷）

**Phase 2：逐户尽调（2-6周，按资产包规模）**
- [ ] 每户债权真实性审查（合同+借据+担保文件+催收记录）
- [ ] 诉讼时效/保证期间排查
- [ ] 抵押物现场勘察（拍照+估值+瑕疵记录）
- [ ] 保证人资信调查
- [ ] 涉诉/执行信息查询
- [ ] 工商/税务/信用信息查询

**Phase 3：回收预测与定价（1-2周）**
- [ ] 逐户回收率测算（高/中/低三档）
- [ ] 加权平均回收率计算
- [ ] 处置成本扣除（诉讼/评估/拍卖/律师）
- [ ] 时间成本折算（年化IRR计算）
- [ ] 建议定价区间

**Phase 4：转让实施（2-4周）**
- [ ] 参与竞标/协议谈判
- [ ] 合规性法律意见书出具
- [ ] 债权转让协议签署
- [ ] 交割清单核对
- [ ] 债权转让通知执行

### 3.2 特殊类型不良资产业务要点

| 类型 | 特殊风险 | 处置策略 |
|------|---------|---------|
| 个贷不良 | 个人信息保护、催收合规 | 合规催收+委外+核销 |
| 重整中的不良债权 | 调整后的债权受偿率 | 参与重整计划+债转股 |
| 破产清算中债权 | 分配方案不明 | 评估有限分配+收购折扣极大 |
| 网贷/P2P不良 | 债权真实性存疑 | 出借人穿透+底层资产确认 |
| 信用卡不良 | 金额小/户数多 | 批量化处置+委外催收 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **债权转让存在国资审批瑕疵** — 债权转让未经国有资产管理部门评估/报批
- **涉及最高额抵押转让** — 主债权未确定时的最高额抵押权转移
- **个人破产/债务集中清理试点** — 深圳/浙江等试点地区特殊处理
- **保证人权利主张** — 保证人以转让未通知/保证期间届满抗辩
- **外资AMC参与转让** — 外商投资负面清单限制
- **债权的优先购买权争议** — 共有人/承租人主张优先购买
- **虚假债权/伪造文件** — 发现债权凭证真实性存疑
- **债务人涉及刑事犯罪** — 集资诈骗/非法吸收存款/洗钱

### 4.2 升级咨询模板

```
【升级咨询】

```

7. 不良资产处置中的税务筹划与优化
