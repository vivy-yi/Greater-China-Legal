# 担保物执行与拍卖

## 一、基本信息

### 1.1 核心目的
为抵押权人（银行/AMC/信托/私募基金）提供担保物执行全流程法律支持，覆盖担保物权的确认、执行启动、资产处置及优先受偿。

### 1.2 适用范围
- 不动产抵押（土地使用权、房屋、在建工程）
- 动产抵押（机器设备、原材料、存货）
- 权利质押（股权、应收账款、基金份额、知识产权）
- 最高额抵押/质押
- 浮动抵押

| 引用来源 | 用途 |
|---------|------|
| [YD] 元典MCP | 《民法典》担保制度司法解释、担保物权案例 |
| [WKL] 裁判文书网 | 担保物执行异议案例、优先权顺位纠纷 |

### 1.4 核心法规依据
- 《民法典》第394-424条（抵押权）、第425-446条（质权）

## 二、分析框架

### 2.1 担保物执行路径选择

```
执行路径
├── 实现担保物权特别程序（民事诉讼法第196-197条）
│   ├── 优势：无需诉讼，时间短（1-2个月）
│   ├── 适用：担保物权关系明确、无实质性争议
│   └── 流程：申请 → 法院审查 → 裁定拍卖/变卖
│
├── 赋予强制执行效力的公证债权文书
│   ├── 优势：跳过诉讼，直接申请执行
│   ├── 适用：经公证处出具执行证书
│   └── 流程：公证执行证书 → 法院立案 → 执行
│
├── 诉讼取得生效判决后执行
│   ├── 优势：能覆盖复杂争议
│   ├── 适用：担保物权存在争议/债务人抗辩
│   └── 流程：起诉 → 判决 → 申请执行 → 拍卖
│
└── 破产程序中别除权行使
    ├── 优势：破产管理人在先处置
    ├── 适用：债务人已进入破产程序
    └── 流程：债权申报 → 别除权确认 → 管理人处置
```

### 2.2 担保物权效力审查清单

| 审查要素 | 不动产抵押 | 动产抵押 | 股权质押 | 应收账款质押 |
|---------|-----------|---------|---------|------------|
| 合同生效要件 | 书面形式 | 书面形式 | 书面形式 | 书面形式 |
| 登记/公示 | 登记设立 | 登记对抗 | 登记设立 | 登记设立 |
| 登记机构 | 不动产登记中心 | 人民银行征信中心 | 市场监督管理局 | 人民银行征信中心 |
| 最高额担保 | 决算期登记 | 同左 | 同左 | 同左 |
| 担保范围 | 约定→主债权+利息+费用 | 同左 | 同左 | 同左 |

### 2.3 优先顺位规则

```
第一顺位：建设工程价款优先受偿权（优于抵押权）
  ↓
第二顺位：已登记的抵押权（按登记时间先后）
  ↓
第三顺位：留置权（动产，优于抵押权和质权）
  ↓
第四顺位：已登记的浮动抵押（结晶前优于一般债权）
  ↓
第五顺位：一般债权（按执行措施先后）
```

**注意**：购房消费者（已支付全部或大部分房款）的物权期待权优于建设工程价款优先权，进而也优于抵押权。

## 三、操作流程

### 3.1 担保物执行业务流程

**Step 1：担保物权状态核查**
- [ ] 核实担保合同有效性及担保范围
- [ ] 确认登记/公示是否完成且有效
- [ ] 核查是否存在重复抵押/质押（顺位关系）
- [ ] 核查担保物现状（是否存在毁损/灭失/查封）
- [ ] 确认主债权是否存在时效中断/届满

**Step 2：执行启动**
- [ ] 选择执行路径（特别程序/公证债权文书/诉讼）
- [ ] 准备申请材料（担保合同、登记证明、催收记录）
- [ ] 申请财产保全（防止转移担保物）
- [ ] 向有管辖权的法院提交执行申请

**Step 3：担保物评估与拍卖**
- [ ] 法院委托评估机构对担保物价值评估
- [ ] 评估结果异议期（7日内可提出异议）
- [ ] 确定拍卖保留价（不低于评估价70%）
- [ ] 网络司法拍卖（一拍/二拍/变卖）
- [ ] 流拍后协议折价或以物抵债

**Step 4：案款分配**
- [ ] 制作参与分配方案
- [ ] 优先权人债权确认
- [ ] 案款划付
- [ ] 剩余案款退还被执行人

### 3.2 常见执行异议及应对

| 异议类型 | 常见理由 | 应对策略 |
|---------|---------|---------|
| 案外人异议 | 主张所有权/租赁权在先 | 审查真实性和登记时间 |
| 优先权异议 | 主张建设工程价款优先权 | 审查工程款真实性及优先范围 |
| 租赁权异议 | 主张"买卖不破租赁" | 审查租赁合同签订时间与租金支付 |
| 评估异议 | 评估价值明显偏低/偏高 | 申请重新评估 |
| 清偿顺位异议 | 主张税款/职工债权优先 | 按破产法/税法规定审查 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **担保物权存在重大效力瑕疵** — 登记无效/担保范围约定不明/交叉担保
- **多轮查封与多顺位担保冲突** — 首封法院与其他债权人冲突
- **建设工程价款优先权与抵押权顺位争议** — 工程款真实性存疑
- **被执行企业进入破产清算** — 需在破产程序中行权
- **跨境担保物执行** — 担保物位于境外或涉及跨境担保
- **上市公司股权质押执行** — 涉及信息披露义务/控制权变更
- **涉及刑事查封/追缴** — 担保物被公安机关查封
- **虚假租赁/恶意占有担保物** — 需排除妨害诉讼

## 五、本Skill不涵盖

1. 担保合同纠纷诉讼代理
2. 担保物权确认之诉
3. 破产重整程序中的担保物处置决策
4. 不良资产ABS中担保物转让合规
5. 跨境担保外汇登记合规（如内保外贷）
6. 融资租赁物取回程序（适用《融资租赁合同司法解释》）
7. 非典型担保（让与担保/所有权保留/保理）的执行路径
