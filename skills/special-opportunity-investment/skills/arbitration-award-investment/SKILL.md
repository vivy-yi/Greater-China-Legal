# 仲裁裁决投资

## 一、基本信息

### 1.1 核心目的
为特殊机会投资者提供仲裁裁决债权投资的可行性分析与全流程法律支持，覆盖境内仲裁裁决执行与境外仲裁裁决在中国法院的承认与执行。

### 1.2 适用范围
- 中国内地仲裁机构作出的裁决（贸仲/北仲/上仲等）
- 香港/澳门仲裁裁决（依据《内地与香港仲裁裁决相互执行安排》《内地与澳门仲裁裁决相互执行安排》）
- 纽约公约成员国仲裁裁决
- 临时仲裁裁决（认可范围限于特定自贸区实验）

### 1.3 数据源与判断框架引用

## 二、分析框架

### 2.1 裁决可执行性评估四步流程

```
Step 1: 裁决状态核查
├── 裁决是否已生效
├── 是否存在撤销/不予执行程序
├── 仲裁协议效力是否已被质疑
└── 裁决送达确认

Step 2: 司法审查风险预判
├── 仲裁协议有效性（书面形式、意思表示真实）
├── 仲裁庭组成与程序合规（仲裁规则适用是否正确）
├── 裁决事项是否超出仲裁范围（可仲裁性判断）
├── 公共政策保留审查（社会公共利益/强制性规定）
└── 送达程序瑕疵（正当程序审查）

Step 3: 被执行人偿付能力评估
├── 被执行人主体信息与资产状况
├── 是否存在多个债权人及执行顺位
├── 是否存在破产/清算程序
└── 财产线索可追溯性

Step 4: 投资回报测算
├── 裁决本金 + 利息 + 仲裁费用
├── 法院执行费用与处置周期
├── 汇率波动风险（跨境裁决）
└── 预期回收率评估
```

### 2.2 纽约公约裁决特别审查清单

| 审查事项 | 中国法院实践要点 |
|---------|----------------|
| 互惠原则 | 推定互惠（2018年起放松，不以条约对等为唯一标准） |
| 仲裁地法律 | 仲裁协议效力依仲裁地法律 |
| 正当程序 | 被执行人须被适当通知并有申辩机会 |
| 公共政策 | 严格解释——限于违背我国法律基本原则、社会公共利益 |
| 驳回执行申请 | 可重新申请（无"一事不再理"限制） |

### 2.3 香港/澳门裁决特别流程

```
内地认可和执行香港裁决
├── 路径：向被执行人住所地/财产所在地中院申请
├── 材料：裁决书正本 + 仲裁协议 + 经公证认证的翻译件
├── 期限：裁决作出后2年内申请
└── 审查：参照纽约公约标准

内地认可和执行澳门裁决
├── 路径：类似香港路径
├── 审查：澳门法院确认裁决为终局裁决的证明
└── 期限：裁决生效后2年内
```

## 三、操作流程

### 3.1 裁决投资尽职调查清单

**第一层：基础文件审核**
- [ ] 仲裁裁决书全文（中英文版本，境外裁决需公证认证）
- [ ] 仲裁协议/仲裁条款
- [ ] 仲裁送达回执与程序性文书
- [ ] 原合同及相关附件
- [ ] 争议双方主体资格文件

**第二层：执行障碍排查**
- [ ] 裁决作出后是否有撤销申请（查询仲裁委 + 中院立案系统）
- [ ] 被执行人是否有财产（不动产/银行账户/股权/应收款）
- [ ] 被执行人是否进入破产清算程序
- [ ] 裁决是否涉公共利益或强制性规定
- [ ] 被执行人是否有跨境资产

**第三层：投资结构设计**
- [ ] 债权转让协议（裁决债权可转让性审查）
- [ ] 通知债务人程序（债权转让通知效力）
- [ ] 收益分配条款设计
- [ ] 退出机制（协商、执行、再次转让）

### 3.2 执行推进时间表

| 阶段 | 预计时间 | 关键动作 |
|------|---------|---------|
| 执行立案 | 7-14天 | 提交申请材料至有管辖权的法院 |
| 财产查控 | 30-60天 | 法院通过执行系统查控被执行人名下资产 |
| 评估拍卖 | 60-120天 | 查封资产后委托评估机构，资产进入司法拍卖 |
| 案款分配 | 30-90天 | 案款到账后制作分配方案，优先债权确认 |
| 跨境协助 | 3-12个月 | 境外资产执行需通过司法协助途径 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **撤销仲裁裁决申请** — 被执行人已向法院提起撤销仲裁裁决程序
- **涉及公共政策审查** — 裁决事项可能涉及中国强制性规定或社会公共利益
- **被执行人破产** — 债务人已进入破产程序，需确认债权申报义务
- **仲裁协议效力争议** — 仲裁协议的真实性、效力存在争议
- **财产线索缺失** — 被执行人无明显可执行财产
- **跨境执行复杂** — 涉及境外送达、境外资产查控或多法域裁决执行
- **裁决金额特别巨大** — 单笔裁决本金超过人民币5000万元

### 4.2 升级咨询模板

```
【升级咨询】
裁决编号：[仲裁委 + 案号]
裁决类型：[境内/香港/澳门/纽约公约]
裁决本金：[金额]
执行障碍：[核心问题概述]
已核查事项：[已完成的尽职调查]
建议行动：[需律师确认事项]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. 仲裁案件代理（由执业律师出庭）
2. 仲裁过程中临时措施申请
3. 涉外商事仲裁实体争议（反请求/抗辩策略）
4. 劳动仲裁裁决执行（适用劳动争议特别规定）
5. 税务/海关等行政仲裁
6. 仲裁员回避或仲裁庭组成异议程序
