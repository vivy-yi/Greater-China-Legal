# 股东协议谈判技能 (SHA Negotiator)

## 技能定位

本技能用于股东协议（Shareholders Agreement）谈判，站在买方或卖方立场，在合法框架内争取最优条款。

## 核心能力

### 1. 核心条款谈判

#### 1.1 股权结构条款
```
持股比例与认购权：
- 各方持股比例
- 新增资本认购比例（pro rata）
- 优先认购权（Right of First Refusal）

反稀释保护：
- 优先认购权（Preferred Subscription）
- 优先购买权（ROFP）
- 优先购买权触发条件
- 价格调整机制（Weighted Average / Broad-based）

谈判要点：
□ 稀释计算基准
□ 完全棘轮（Full Ratchet）vs 加权平均（Weighted Average）
□ 反稀释适用例外（员工期权、融资折扣等）
□ 新增股份上限
```

#### 1.2 公司治理条款
```
董事会组成：
- 董事席位分配
- 董事长指定权
- 独立董事选任

股东会表决：
- 普通决议事项（简单多数）
- 特别决议事项（特定多数）
- 保护性事项（否决权）

保护性条款示例：
- 重大资产处置需全体同意
- 关联交易超一定金额需批准
- 增减资、合并、分立需批准
- 董事会成员任免
- 高管薪酬
- 对外担保

谈判要点：
□ 保护性事项范围设定
□ 否决权触发金额门槛
□ 决策效率与保护平衡
□ 中国法律项下合规性
```

#### 1.3 退出机制条款
```
随售权（Tag-along）：
- 触发条件：主要股东转让股权
- 参与范围：按转让比例
- 价格条件：同等条件
- 期限：通常30-90天

拖售权（Drag-along）：
- 触发条件：特定多数股东同意出售
- 效力：强制少数股东同价出售
- 价格保护：最低价格限制
- 适用例外：员工持股

优先购买权（ROFR）：
- 适用情形：股东拟转让股权
- 优先购买期限：通常15-30天
- 价格确定机制：协商/评估/公式
- 卖方义务：通知义务

优先清算权（Liquidation Preference）：
- 优先清算金额：1x / 2x / non-participating
- 参与分配权：参与型/非参与型
- 清算事件定义
- 实际控制人变更是否触发

谈判要点：
□ 触发条件合理性
□ 价格机制公允性
□ 期限合理性
□ 例外情形设定
```

### 2. 财务条款谈判

#### 2.1 利润分配条款
```
分红政策：
- 分红比例（固定比例/剩余分配）
- 分红条件（累计利润要求）
- 分红形式（现金/股票）

强制分红：
- 特定情形下强制分红义务
- 违规分红责任

谈判要点：
□ 分红比例与公司发展阶段匹配
□ 分红条件可实现性
□ 董事会裁量权范围
```

#### 2.2 估值调整条款（对赌条款）
```
对赌指标：
- 净利润
- 营业收入
- 市场份额
- 用户数

对赌方式：
- 现金补偿
- 股权调整
- 双向对赌

计算公式示例（现金补偿）：
补偿金额 = 承诺估值 - 实际估值 × 持股比例

法律合规性（中国）：
- 2020年《全国法院民商事审判工作会议纪要》
- 对赌协议原则上有效
- 但需区分与公司对赌 vs 与股东对赌

谈判要点：
□ 指标选择可量化性
□ 承诺期限合理性
□ 会计政策一致性
□ 信息披露义务
```

#### 2.3 托管条款（Escrow）
```
托管安排：
- 托管金额比例
- 托管期限
- 托管账户开立
- 托管银行指定

释放条件：
- 交割日后X日内无重大索赔
- 特定事项成就/未成就
- 争议解决程序完成

扣留/释放机制：
- 单方扣留情形
- 争议扣留处理
- 最终释放安排

谈判要点：
□ 托管金额与风险敞口匹配
□ 托管期限合理性
□ 释放条件确定性
□ 争议解决机制
```

### 3. 保护性条款

#### 3.1 陈述与保证
```
卖方陈述保证：
- 公司合法存续
- 股权无瑕疵
- 财务报表真实性
- 无重大诉讼
- 知识产权有效性
- 劳动合规
- 环境保护
- 税务合规

买方陈述保证：
- 收购授权完整
- 资金来源合法
- 非出于规避责任目的

保证期限：
- 一般陈述：交割后2-3年
- 重大陈述（股权、税务）：交割后5-7年
- 披露函（Schedule）列明事项除外

谈判要点：
□ 陈述保证范围与尽调发现匹配
□ 买方要求vs卖方承受力
□ 披露函作用（Carve-out）
□ 违反后果（赔偿/终止）
```

#### 3.2 赔偿条款（Indemnification）
```
赔偿范围：
- 违反陈述保证
- 交割前义务
- 或有负债
- 税务责任
- 环境污染
- 员工索赔

赔偿限制：
- 免赔额（Deductible）
- 赔偿上限（Cap）
- 追诉时效（Time Bar）
- 例外情形

赔偿计算：
- 直接损失（Direct Loss）
- 间接损失（Consequential Loss）是否包含
- 损失金额计算方法

谈判要点：
□ 赔偿上限与交易价格挂钩
□ 免赔额合理性
□ 重大事项单独设置上限
□ 税务/环境等特殊事项处理
```

#### 3.3 竞业禁止
```
竞业限制：
- 限制主体：卖方创始人/管理层
- 限制期限：通常2-3年
- 限制地域：约定范围
- 限制业务：同业竞争
- 补偿：通常包含

竞业补偿：
- 固定补偿
- 按月补偿
- 与业绩挂钩

法律限制：
- 《劳动合同法》：员工竞业限制不超过2年
- 经济补偿标准
- 违约金上限

谈判要点：
□ 竞业范围合理性
□ 补偿充分性
□ 违约金可执行性
□ 与劳动关系协调
```

### 4. 争议解决条款

#### 4.1 适用法律
```
中国法：
- 国际商事仲裁
- 国内仲裁/诉讼
- 选择机构：贸仲/北仲/上仲

外国法：
- 香港法（常用）
- 英国法
- 新加坡法

注意：
- 外商投资企业适用中国法
- 涉外因素认定
- 仲裁条款有效性
```

#### 4.2 争议解决方式
```
仲裁 vs 诉讼：
仲裁优点：私密性、专家裁判、国际执行
诉讼优点：公开、成本较低、上诉机制

仲裁机构：
- 中国国际经济贸易仲裁委员会（贸仲）
- 北京仲裁委员会（北仲）
- 香港国际仲裁中心（HKIAC）
- 国际商会仲裁院（ICC）
- 新加坡国际仲裁中心（SIAC）

仲裁规则考量：
- 简易程序适用条件
- 仲裁语言
- 仲裁地 vs 开庭地
- 多方仲裁机制
```

### 5. 谈判策略

#### 5.1 立场确定
```
买方优先条款：
- 完整的信息披露权
- 严格陈述保证
- 宽泛赔偿范围
- 优先购买权
- 随售权
- 重大事项否决权

卖方优先条款：
- 最小化陈述保证范围
- 有限赔偿期限和上限
- 限制买方干预权
- 保护性拖售权
- 优先清算权
- 竞业禁止补偿
```

#### 5.2 谈判优先级
```
核心条款（必须争取）：
- 股权比例
- 董事会席位
- 保护性事项范围
- 退出机制

重要条款（尽量争取）：
- 优先清算权
- 随售/拖售权
- 竞业禁止范围
- 赔偿上限

一般条款（可接受妥协）：
- 仲裁语言
- 通知方式
- 文本份数
```

#### 5.3 常见让步模式
```
典型让步：
- 陈述保证期限：3年 → 2年
- 赔偿上限：20% → 15%
- 竞业期限：3年 → 2年
- 优先购买权期限：30天 → 15天
- 随售比例：100% → 50%

交换筹码：
- 更多董事会席位 ↔ 更少否决权
- 更高估值 ↔ 更多保护条款
- 现金支付 ↔ 股份支付
```

## 风险提示

1. **条款冲突**：注意与公司章程、投资协议的一致性
2. **可执行性**：确保条款在中国法律框架下可执行
3. **税负影响**：部分条款可能产生税务后果
4. **灵活性**：为未来融资、并购预留空间
