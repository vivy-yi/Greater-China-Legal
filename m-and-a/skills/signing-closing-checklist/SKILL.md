# 签约与交割清单技能 (Signing-Closing Checklist)

## 技能定位

本技能用于管理并购交易从签署到交割的全流程，确保各项先决条件满足，交易顺利闭合。

## 核心能力

### 1. 签约前准备

#### 1.1 文件审查
```
必备文件清单：
□ 并购协议/股权转让协议（已签署）
□ 股东协议（如涉及多方）
□ 董事会决议
□ 股东会决议
□ 授权委托书

审查要点：
□ 签章完整性
□ 日期准确性
□ 附件完备性
□ 附录与正文一致性
```

#### 1.2 先决条件（Conditions Precedent）梳理
```
监管审批类：
□ 反垄断审查批准
□ 外商投资审批
□ 行业监管审批
□ 上市公司审核（如适用）

第三方同意类：
□ 银行债务人同意
□ 担保权人同意
□ 租赁方同意转让
□ 合作伙伴优先购买权放弃

其他条件：
□ 买方资金落实证明
□ 卖方陈述保证持续有效
□ 无重大不利变化（MAC）
□ 交割前审计完成
```

### 2. 签约管理

#### 2.1 签约流程
```
签约前72小时：
□ 确认签约文本无修改
□ 确认各方授权完整
□ 确认签约代表证件
□ 确认签约场地安排
□ 确认见证/公证安排（如需）

签约当日：
□ 文本交换核对
□ 签章见证
□ 签署页收集
□ 副本交接
□ 签约确认函
```

#### 2.2 签约仪式管理
```
场地准备：
- 签约厅布置
- 各方座位安排
- 媒体/摄影安排（如需）
- 安保（如涉密项目）

人员确认：
- 签约代表
- 见证人
- 律师
- 翻译（如跨境）

文件准备：
- 签约用文本（正本数量）
- 副本
- 附录附件
- 骑缝章文本
```

### 3. 交割前准备

#### 3.1 交割先决条件核查清单
```
监管审批完成：
□ 反垄断审查决定书
□ 发改委境外投资备案/批复
□ 商务部境外投资证书
□ 外管局外汇登记
□ 行业监管批复

内部授权：
□ 买方董事会/股东会决议（更新）
□ 卖方董事会/股东会决议（更新）
□ 授权签字文件更新

第三方同意：
□ 银行同意函
□ 担保注销/转让文件
□ 租赁转让同意函
□ 许可证变更申请

其他条件：
□ 工商查档无异常
□ 税务查档无异常
□ 法院查封已解除
```

#### 3.2 交割文件准备
```
股权转让类：
□ 股权转让协议
□ 股东会决议
□ 新章程
□ 董事监事变更决议
□ 法定代表人变更决议

变更登记类：
□ 公司登记（备案）申请书
□ 营业执照正副本
□ 法定代表人信息表
□ 股权转让凭证明细表

财务类：
□ 交割日资产负债表
□ 往来款清理确认
□ 印鉴交接清单
□ 证照移交清单
```

### 4. 交割执行

#### 4.1 交割日流程
```
上午：
□ 验证文件完整性
□ 确认各项批准已取得
□ 确认变更登记材料齐备
□ 确认转让价款已足额到位/托管

下午：
□ 递交工商变更申请
□ 预留取件凭证
□ 确认印鉴交接
□ 确认证照移交
□ 各方签署交割确认书
```

#### 4.2 交割确认书要素
```
确认事项：
- 交割日期
- 交割条件满足情况
- 转让标的描述
- 转让价款确认
- 文件物品移交清单
- 各方声明

签署方：
- 买方
- 卖方
- 交割代理人（如有）
```

### 5. 交割后事项

#### 5.1 变更登记
```
工商变更：
□ 股权变更登记
□ 董事监事变更登记
□ 法定代表人变更登记
□ 章程备案

其他变更：
□ 税务登记变更
□ 银行账户变更
□ 社保公积金账户变更
□ 知识产权权利人变更
```

#### 5.2 公告与通知
```
内部公告：
□ 员工通知
□ 内部部门协调

外部通知：
□ 客户通知（如需）
□ 供应商通知（如需）
□ 监管报告（如需）
```

#### 5.3 后续跟踪
```
交割后30日：
□ 工商变更完成确认
□ 新营业执照领取
□ 银行印鉴变更
□ 重要合同通知相对方

交割后90日：
□ 证照变更完成确认
□ 税务自查
□ 整合计划启动
```

### 6. 时间节点管理

```
关键时间节点：

签约日（T日）：
- 签署并购协议
- 支付定金/诚意金
- 约定交割先决条件完成期限

先决条件期限：
- 通常签约后90-180日
- 每月核查进展
- 逾期可终止情形

交割窗口期：
- 先决条件满足后确定交割日
- 提前5日发出交割通知
- 准备好所有交割文件

尾款支付（如适用）：
- 交割确认后X日内
- 托管释放条件成就后
```

## 风险管理

### 常见交割延误原因
```
审批延误：
- 反垄断审查时间超预期
- 行业监管审批延迟
- 跨部门协调时间

文件问题：
- 第三方同意未取得
- 文件瑕疵需补充
- 签字瑕疵需重新签署

资金问题：
- 融资款不到位
- 换汇延迟
- 托管资金争议

解决方案：
□ 设置先决条件满足的合理期限
□ 约定逾期终止权和终止费
□ 设置分期交割机制
□ 预留交割调整机制
```
