# M&A 法律工作空间

本目录为并购交易法律工作专用空间，包含参考文件框架和技能系统。

## 目录结构

```
m-and-a/
├── CLAUDE.md                    # 本说明文件
├── references/                  # 参考文件目录
│   ├── 判断框架.md              # 交易判断与分析框架
│   ├── 查询路径.md              # 法律研究与查询路径
│   └── 数据源清单.md            # 数据源与信息清单
└── skills/                      # 技能目录
    ├── due-diligence-checker/   # 尽职调查技能
    ├── deal-structurer/         # 交易结构设计技能
    ├── signing-closing-checklist/ # 签约交割清单技能
    ├── regulatory-approval-tracker/ # 监管审批追踪技能
    ├── sha-negotiator/          # 股东协议谈判技能
    └── post-closing-integration/ # 交割后整合技能
```

## 使用说明

1. **尽职调查**: 使用 `due-diligence-checker` 技能进行目标公司法律尽职调查
2. **交易结构**: 使用 `deal-structurer` 技能设计和优化交易结构
3. **签约交割**: 使用 `signing-closing-checklist` 技能管理签约至交割流程
4. **监管审批**: 使用 `regulatory-approval-tracker` 技能追踪各类监管审批
5. **股东协议**: 使用 `sha-negotiator` 技能进行股东协议谈判
6. **交割后整合**: 使用 `post-closing-integration` 技能协调交割后整合工作

## 核心原则

- 所有法律分析需引用具体法条依据
- 涉及商业判断时需标注法律边界
- 监管审批需注意时效和条件限制
- 跨境交易需考虑多司法辖区要求
