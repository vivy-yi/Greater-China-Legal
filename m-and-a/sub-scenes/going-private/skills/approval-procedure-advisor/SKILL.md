# 私有化技能3：审批程序与实施流程管理

## 技能描述
本技能用于协助用户完成私有化的审批程序与管理实施全流程。

## 核心功能
1. 审批流程规划
2. 时间节点管理
3. 文件准备清单
4. 实施风险监控

## 审批流程总览

### 内部审批
1. 董事会审议
   - 审议时间：通常5-10个工作日
   - 所需文件：私有化方案、财务顾问意见
   
2. 股东大会审议
   - 通知时限：股东大会召开前15日
   - 表决要求：出席股东所持表决权2/3以上通过

### 外部审批
1. 证监会审批（如涉及发行股份）
   - 审批时限：30个工作日
   - 申报材料：收购报告书、财务顾问意见等

2. 交易所审批
   - 终止上市申请
   - 审查时间：通常10-15个工作日

## 时间节点示例（要约收购路径）
```
第1周：聘请中介机构
第2-4周：尽职调查与方案制定
第5周：董事会审议
第6-7周：披露公告
第8周：发出要约
第9-12周：要约踯跃期
第13周：踯跃结果确认
第14周：股份交割
第15周：终止上市
```

## 风险监控要点
1. 踯跃率不达标的应急处理
2. 竞争性要约的应对
3. 股价异常波动监测
4. 媒体舆情管理

## 升级决策门

以下情况须移交专业律师：
- 涉及监管审批
- 存在法律风险
