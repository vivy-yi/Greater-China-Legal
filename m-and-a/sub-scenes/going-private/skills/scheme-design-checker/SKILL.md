# 私有化技能1：私有化方案设计与合规审查

## 技能描述
本技能用于协助用户进行上市公司私有化方案的制定与合规性审查。

## 核心功能
1. 私有化路径选择建议
2. 法规合规性检查
3. 方案风险评估
4. 披露文件审核

## 操作指南

### 输入信息
- 上市公司基本信息
- 股权结构
- 私有化动因
- 初步定价意向

### 输出内容
1. 私有化路径建议
2. 合规要点清单
3. 风险提示
4. 建议下一步行动

## 关键法规引用
- 《上市公司收购管理办法》第三十条至第四十条
- 《证券法》第七十一条至第八十六条
- 《上市公司信息披露管理办法》

## 合规检查清单
- [ ] 收购人资格核查
- [ ] 收购资金来源合法性
- [ ] 要约收购条件合规性
- [ ] 信息披露完整性
- [ ] 审批程序完备性

## 常见问题
Q：私有化是否必须聘请财务顾问？
A：根据《上市公司收购管理办法》，收购人应当聘请财务顾问，但存在例外情形。

Q：要约期多久？
A：收购要约约定的收购期限不得少于30日，并不得超过60日。

## 升级决策门

以下情况须移交专业律师：
- 涉及监管审批
- 存在法律风险
