# 私有化参考文件3：实务操作指南

## 私有化方案设计

### 两种主要路径

#### 路径一：要约收购私有化
```
步骤：
1. 聘请财务顾问和法律顾问
2. 制定要约收购方案
3. 董事会审议
4. 披露要约收购报告书
5. 证监会审批（如需）
6. 发出要约
7. 等待踯跃期届满
8. 股份交割
9. 终止上市
```

#### 路径二：吸收合并私有化
```
步骤：
1. 制定合并方案
2. 董事会审议
3. 债权人通知
4. 股东会表决
5. 审批程序
6. 合并实施
7. 注销上市资格
```

## 收购价格确定

### 定价方法
1. **市场法**：二级市场价格
2. **收益法**：现金流折现
3. **资产基础法**：净资产评估

### 定价考虑因素
- 历史股价表现
- 行业平均估值水平
- 公司成长性
- 控股溢价

## 信息披露要求

### 重要事项披露
1. 私有化方案及理由
2. 收购价格及定价依据
3. 财务顾问意见
4. 独立董事意见
5. 后续经营计划

### 时间节点
- 董事会决议公告
- 要约收购报告书摘要
- 要约收购报告书全文
- 踯跃结果公告
- 终止上市公告

## 税务考量
- 企业所得税
- 个人所得税（股息所得）
- 股权转让所得
- 土地增值税（涉及不动产）
