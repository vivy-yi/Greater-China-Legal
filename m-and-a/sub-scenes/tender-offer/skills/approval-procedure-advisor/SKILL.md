# 要约收购技能3：收购完成与后续整合

## 技能描述
本技能用于协助用户完成要约收购后的股份交割及后续整合工作。

## 核心功能
1. 交割程序管理
2. 股份登记变更
3. 整合计划制定
4. 持续合规管理

## 股份交割程序

### 交割条件确认
1. 要约期满
2. 踯跃数量达标
3. 证监会审批（如需）
4. 资金准备就绪

### 交割流程
```
第1步：确认踯跃结果
   ├── 计算有效踯跃数量
   ├── 确认踯跃条件满足
   └── 出具交割确认函

第2步：资金划转
   ├── 资金划至指定账户
   ├── 证券公司协助办理
   └── 资金安全确认

第3步：股份变更登记
   ├── 中国结算办理
   ├── 股份过户手续
   └── 股东名册更新

第4步：后续公告
   ├── 踯跃结果公告
   ├── 股份交割公告
   └── 收购完成公告
```

## 合并报表与财务处理

### 合并报表起点
- 收购完成日为合并起点
- 评估公允价值确认
- 识别可辨认资产/负债

### 商誉计算
```
商誉 = 收购成本 - 被收购方净资产公允价值
```

## 后续整合计划

### 业务整合
1. 战略整合
   - 业务定位调整
   - 协同效应发挥
   - 资源配置优化

2. 运营整合
   - 采购整合
   - 销售整合
   - 人员整合

### 治理整合
1. 董事会改组
   - 董事席位分配
   - 独立董事选任
   - 专门委员会设立

2. 管理层调整
   - 高管选聘
   - 考核激励机制
   - 责任分工

### 人员整合
- 员工安置方案
- 薪酬体系调整
- 企业文化融合

## 持续合规管理

### 信息披露持续义务
- 收购人持股变动披露
- 后续计划进展披露
- 财务报告关联披露

### 锁定期承诺
- 收购人股份锁定期
- 一致行动人股份锁定期
- 相关人员股份变动限制

## 常见问题
Q：踯跃不足怎么办？
A：可延长期限或宣布要约失败。失败后12个月内不得再收购。

Q：收购完成后需要持续披露多久？
A：通常持续至收购人持股降至30%以下或相关承诺履行完毕。

## 升级决策门

以下情况须移交专业律师：
- 涉及监管审批
- 存在法律风险
