# /procedure-timeline — 诉讼程序时间线

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取程序规则/时效规定。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取程序规则/各阶段时限
2. 确定案件类型和管辖法院
3. 梳理各程序阶段时间线
4. 输出时间线

## 一审程序

| 阶段 | 普通程序 | 简易程序 |
|---|---|---|
| 立案 | 7日内决定是否受理 | 7日内（当场可立案） |
| 送达被告 | 5日内（被告15日内答辩） | 同上 |
| 开庭通知 | 开庭3日前通知 | 同上 |
| 审理期限 | 6个月（可延长） | 3个月（不可延长） |
| 判决 | 审理期限内 | 审理期限内 |

## 二审程序

| 阶段 | 时限 |
|---|---|
| 上诉期限 | 判决书送达15日内 |
| 上诉状提交 | 上诉期届满后3日内提交原审法院 |
| 原审法院移送上 | 5日内 |
| 二审立案 | 收到上诉状后5日内 |
| 二审审理 | 3个月内审结（可延长） |

## 再审程序

| 阶段 | 时限 |
|---|---|
| 申请再审 | 判决生效6个月内 |
| 法院审查 | 3个月内（决定是否再审） |
| 再审审理 | 按一审/二审程序（视案件性质） |

## 特殊程序

| 程序 | 时限 |
|---|---|
| 公示催告 | 公告期60日（票据丧失） |
| 特别程序 | 30日内审结（选民资格除外） |
| 督促程序 | 支付令发出15日后无异议则生效 |

## 送达时间

```
直接送达：签收日
邮寄送达：回执注名日期起算
留置送达：完成留置动作日
公告送达：公告期60日届满次日
```
