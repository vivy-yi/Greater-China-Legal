# /appeal-path-checker — 上诉/再审路径核查

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取上诉/再审条件/成功率评估。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取上诉/再审条件
2. 分析一审判决
3. 判断上诉/再审可行性
4. 评估成功率
5. 输出路径建议

## 上诉路径

### 上诉条件

```
条件：
1. 上诉人须为一审当事人（原告/被告/第三人）
2. 上诉须在判决书送达之日起15日内提出
3. 须提交上诉状（正本+副本）

注意事项：
- 判决书送达须签收，签收日期起算15日
- 上诉期最后一日为节假日的，顺延至第一个工作日
- 上诉状须明确上诉请求和理由
```

### 二审改判条件

```
二审法院改判/发回重审的情形：
1. 一审认定事实错误（证据不足/新证据）
2. 一审适用法律错误
3. 一审程序违法（须发回重审）
4. 一审遗漏当事人/超出诉讼请求

注：二审法院以"事实不清"发回重审须同时说明具体理由
```

### 上诉成功率评估

| 情形 | 成功率 |
|---|---|
| 一审事实认定清楚，法律适用正确 | 低（<10%） |
| 一审事实认定错误（新证据） | 中（30-50%） |
| 一审法律适用错误 | 中（30-50%） |
| 一审程序违法 | 高（>60%，通常发回重审） |

## 再审路径

### 再审条件（民事诉讼法第207条）

```
法定再审条件（须满足其一）：
1. 有新证据，足以推翻原判决
2. 原判决认定的基本事实缺乏证据证明
3. 原判决认定事实的主要证据是伪造/篡改/伪造
4. 原判决适用法律确有错误
5. 审判组织组成不合法/违法剥夺当事人辩论权
6. 审判人员有贪污受贿/徇私舞弊/枉法裁判
7. 送达不合规，剥夺当事人出庭权
8. 一审漏判/超出诉讼请求
```

### 再审申请时效

```
时效：判决生效后6个月内
起算：判决书送达之日（生效之日）
注意：6个月为除斥期间，不中断不延长
```

### 再审与上诉的区别

| 对比 | 上诉 | 再审 |
|---|---|---|
| 时效 | 15日 | 6个月 |
| 次数 | 一次 | 一次 |
| 条件 | 无须特定理由 | 须满足法定条件 |
| 效果 | 二审全面审查 | 再审法院有限审查 |
| 成功难度 | 相对较低 | 难度高 |

## 执行异议

```
执行异议：
- 当事人/利害关系人认为执行行为违反法律规定
- 向执行法院提出书面异议
- 执行法院15日内审查

执行异议之诉：
- 对执行异议裁定不服
- 15日内提起执行异议之诉
```
