# 行为保全申请评估

## 目的

评估是否申请诉前或诉中行为保全（财产保全、证据保全、行为禁令），包括必要性分析、担保金额计算、错误保全风险评估。本技能辅助决策，不代理申请。

## 法域假设

默认中国大陆法域，适用《民事诉讼法》保全章和《最高人民法院关于人民法院办理财产保全案件若干问题的规定》`[SME 核查]`。

## 加载信息

- 案件类型和争议金额
- 保全目的（防止判决无法执行/防止证据灭失/防止侵权持续）
- 被申请人财产线索

## 工作流程

### 第一步：确认保全类型

| 类型 | 适用场景 | 紧迫程度 |
|------|---------|---------|
| 诉前财产保全 | 情况紧急+执行风险高 | 高（30日内须起诉）|
| 诉中财产保全 | 判决可能无法执行 | 中 |
| 诉前证据保全 | 证据可能灭失 | 高 |
| 行为禁令 | 侵权持续/不可逆损害 | 高 |

### 第二步：评估保全必要性

| 因素 | 评估 |
|------|------|
| 胜诉可能性 | 高/中/低 |
| 被申请人执行能力 | 强/弱/不明 |
| 判决可执行性 | 好/差/不确定 |
| 不保全的风险 | 高/中/低 |

**保全必要性判断**：`胜诉可能性高 + 执行能力弱/不明 + 不保全风险高` → 建议申请

### 第三步：计算担保金额 `[SME 核查]`

| 保全类型 | 担保要求 |
|---------|---------|
| 诉前财产保全 | 通常为请求金额的30% |
| 诉中财产保全 | 通常为请求金额的10-30% |
| 错误保全赔偿 | 担保金额可覆盖错误保全损失 |

### 第四步：错误保全风险 `[SME 核查]`

错误保全须承担赔偿责任：
- 赔偿被申请人因保全所受的直接损失
- **错误保全风险高**时（案件事实不清、证据不足），应审慎申请

### 第五步：升级决策门

> "财产保全涉及担保和错误赔偿风险，申请前建议由诉讼律师评估必要性和担保金额。诉前保全须在30日内起诉，时间节点很重要。这不构成法律意见。"

## 本技能不涵盖

- **代理保全申请**
- **起草保全申请文书**
- **确认担保金额** — 须由法院裁定 `[SME 核查]`
