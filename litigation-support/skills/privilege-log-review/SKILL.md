# /privilege-log-review — China Mainland

## CN特权保护要点

**律师-当事人特权：**
- 通信须明确标记"律师保密特权"
- 仅在与律师通信中使用
- 不得在公开渠道传输

**工作成果保护：**
- 律师准备的分析、备忘录受保护
- 须明确标记"律师工作成果"

**CN特殊情况：**
- 企业法律顾问的特权保护范围有限（参考最高法判例）
- 须区分内部法务与外部律师通信

- 特权放弃风险
- 跨境数据传输的特权问题
