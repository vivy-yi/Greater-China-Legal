# /lawsuit-doc-generator — 诉讼文书生成

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取文书格式/必备要素。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取文书格式/注意事项
2. 收集案件信息
3. 按文书类型生成
4. 输出规范文书

## 起诉状要素（民事诉讼法第124条）

```
一、当事人信息
  原告：姓名/性别/年龄/住所/联系方式
  被告：同上

二、诉讼请求
  具体写明请求法院判令的事项

三、事实与理由
  事实经过
  法律依据

四、证据和证据来源
  证人姓名和住所
  书证、物证须说明

五、附件清单
```

## 答辩状要素

```
一、答辩人基本信息

二、对原告诉讼请求的答辩
  逐条回应原告的诉讼请求

三、事实与理由
  对原告主张的事实和理由进行反驳
  提出自己的事实主张

四、证据

五、附件
```

## 上诉状要素

```
一、上诉人/被上诉人基本信息

二、上诉请求
  具体写明要求二审法院改判/撤销的内容

三、上诉理由
  一审判决的错误所在
  法律依据

四、附件：一审判决书副本
```

## 常见错误

| 错误类型 | 后果 |
|---|---|
| 诉讼请求不明确 | 法院要求补正，可能驳回起诉 |
| 被告信息不完整 | 无法送达，可能驳回起诉 |
| 事实描述不具体 | 影响举证和审理 |
| 法律依据引用错误 | 不影响立案，但影响判决 |
| 证据目录不规范 | 可能影响证据采纳 |
