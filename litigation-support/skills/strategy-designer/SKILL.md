# /strategy-designer — 诉讼策略制定 + 论证链条构建

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取诉讼程序/策略选择规则。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取程序规则/胜诉因素
2. 分析案件事实和证据
3. 确定诉讼地位（原告/被告）
4. **构建论证链条（THUYRan方法论）** ← 新增
5. 设计诉讼策略
6. 输出策略建议 + 论证方案

## 第一部分：论证链条构建（THUYRan方法论）

> 以下方法论参考 THUYRan/Legal-Skills-Chinese/argument-chain-construction

### 论证结构类型

根据案件复杂度选择：

```
结构A：线性链式（ Sequential Chain）
  适用：单一请求权基础、因果关系清晰
  结构：A → B → C → 最终结论

结构B：并列支撑（ Parallel Support）
  适用：多个独立理由支撑同一结论
  结构：理由1 ─┐
         理由2 ─┼→ 最终结论
         理由3 ─┘

结构C：层级递进（ Hierarchical Progression）
  适用：复杂法律关系、多层次分析
  结构：第一层 → 第二层 → 第三层 → 最终结论

结构D：预防性防御（ Defensive Architecture）
  适用：需要应对对方反驳
  结构：主论证链 ──→ 结论
              ├── 预设反驳1 → 回应1
              └── 预设反驳2 → 回应2
```

### 论证节点标准格式

每个论证节点包含三个要素：

```
┌─────────────────────────────────────┐
│ 论证节点：[节点名称]                 │
├─────────────────────────────────────┤
│ 大前提（Major Premise）              │
│ → 法律规范/法条依据（精确到条款项）  │
│                                     │
│ 小前提（Minor Premise）              │
│ → 案件事实/证据（精确到证据编号）    │
│                                     │
│ 涵摄推理                            │
│ → 事实如何归入法律规范               │
│                                     │
│ 节点结论                            │
│ → 法律效果/法律评价                 │
└─────────────────────────────────────┘
```

### 论证构建五步法

```
步骤1：明确论证目标
  □ 我要证明什么？（最终结论）
  □ 目标受众是谁？（法官/仲裁员）
  □ 对方可能的反驳是什么？

步骤2：盘点论证素材
  - 法律规范（法条）
  - 案件事实（证据）
  - 裁判规则（类案）

步骤3：设计论证结构
  - 选择结构类型（A/B/C/D）
  - 绘制论证蓝图

步骤4：构建论证节点
  - 逐一构建节点（大前提→小前提→结论）
  - 检查涵摄是否成立
  - 连接节点形成链条

步骤5：构建防御层
  - 列出对方可能的反驳（至少3个）
  - 为每个反驳准备回应
```

### 逻辑谬误检查

```
□ 肯定后件：把结果当原因
□ 否定前件：把无因当无果
□ 循环论证：用结论证前提
□ 滑坡谬误：不当推导极端后果
□ 稻草人谬误：歪曲对方观点后反驳
□ 以偏概全：从个案推导一般规则
□ 偷换概念：同一术语不同含义
□ 因果谬误：时间先后等同于因果
```

## 第二部分：诉讼策略设计

### 原告策略（攻击型）

```
核心目标：胜诉 + 获得最大赔偿/实现最大诉求

策略要点：
1. 诉讼请求设计：最大化但合理（避免被驳回）
2. 论证结构：采用结构A或结构C（线性/递进）
3. 证据组织：完整证据链，避免证据突袭
4. 管辖选择：如有选择，选择对我方有利的法院
5. 诉讼时机：注意时效，及时起诉
6. 保全申请：及时申请财产保全
7. 防御层：预设被告的三种主要反驳并准备回应
```

### 被告策略（防御型）

```
核心目标：驳回/减少责任/反诉

策略要点：
1. 答辩要点：逐条反驳原告主张
2. 论证结构：采用结构D（防御型）+ 结构B（并列反驳）
3. 论证重心：攻击原告论证链的薄弱节点
4. 反诉：如存在反诉理由，考虑反诉
5. 时效抗辩：如诉讼时效届满，提出时效抗辩
6. 责任减轻：提出免责/减责事由
7. 备选论证：如主论证失败，准备退守方案
```

## 各场景论证重点

### 合同纠纷（层级递进结构）
```
合同成立 → 合同生效 → 合同履行 → 违约认定 → 责任承担
```

### 侵权纠纷（线性链式结构）
```
侵权行为 → 损害后果 → 因果关系 → 过错认定 → 责任分配
```

### 劳动争议（并列支撑 + 备选论证）
```
主张1：解除违法 → 赔偿2N
主张2：工资拖欠 → 支付工资
备选：如2N不被支持 → 主张N+1
```

### IP侵权（层级递进 + 类比论证）
```
权利基础 → 侵权比对 → 损害计算
类比论证：援引类似判例支撑
```
