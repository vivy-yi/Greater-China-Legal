# 行政处罚决定书审查

## 目的

审查行政处罚决定书的合法性，识别可抗辩事由，辅助制定复议/诉讼策略。覆盖：执法主体、事实认定、法律适用、法定程序、处罚幅度、证据充分性。

本技能为审查辅助，不代理复议或诉讼。

## 法域假设

默认中国大陆法域，适用《行政处罚法》（2021年修订）和各行业监管法规 `[SME 核查]`。

## 加载信息

- 处罚决定书原文（用户提供路径或直接提供）
- 违规事实描述
- 罚款金额和处罚类型

## 工作流程

### 第一步：提取处罚决定书核心信息

| 信息项 | 内容 |
|--------|------|
| 执法机关 | [___] |
| 当事人 | [___] |
| 违规类型 | [___] |
| 违反法条 | [___] |
| 处罚类型 | [警告/罚款/没收/吊销许可证/其他] |
| 罚款金额 | [___]元 |
| 处罚日期 | [YYYY-MM-DD] |

### 第二步：五维度合法性审查

#### 1. 执法主体是否适格 `[SME 核查]`
- 执法机关是否具有管辖权
- 执法人员是否持有合法证件
- 是否在法定授权范围内执法

#### 2. 事实认定是否清楚
- 违规事实是否有充分证据支撑
- 证据是否合法取得（取证程序是否合规）
- 事实描述是否具体明确

#### 3. 法律适用是否正确 `[SME 核查]`
- 引用法条是否与违规行为对应
- 法条是否在有效期内
- 是否符合法定处罚种类和幅度

#### 4. 程序是否合法 `[SME 核查]`
- 是否依法告知陈述申辩权
- 是否告知听证权利（达到听证标准）
- 是否保障了当事人的合法权利
- 集体讨论程序是否履行（重大处罚）

#### 5. 处罚幅度是否合理
- 罚款金额是否在法定幅度内
- 是否考虑了从重/从轻/减轻情节
- 是否符合当地裁量基准

### 第三步：识别可抗辩事由

| 抗辩类型 | 说明 | 是否适用 |
|---------|------|---------|
| 主体不适格 | 执法机关无权管辖 | 是/否 |
| 事实不清 | 证据不足或事实认定错误 | 是/否 |
| 法律适用错误 | 引用法条不当 | 是/否 |
| 程序违法 | 未告知/未保障权利 | 是/否 |
| 幅度不当 | 罚款过重或过轻 | 是/否 |

### 第四步：升级决策门

> "行政处罚决定书的复议期限为收到决定书后60日，时效性很强。在提交复议申请前，建议由行政诉讼律师审核决定书的合法性和抗辩空间。这不构成法律意见。"

## 本技能不涵盖

- **代理复议或行政诉讼**
- **起草复议申请书**
