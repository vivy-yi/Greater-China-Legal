# /portfolio-status — China Mainland

## 一、技能说明

本技能用于生成中国内地诉讼案件组合的总览视图，输出内容包括：

1. **风险分级案件列表**：按高/中/低风险三级分类，含案件名称、风险标识、敞口金额、下一关键截止日期
2. **本周/本月截止事项**：时间轴视角的截止事项清单，帮助识别优先级
3. **汇总指标**：总案件数、总敞口金额、本月预计法律费用、已消耗预算等
4. **组合健康度评分**：基于风险分布、截止日期分布、费用消耗率等维度的综合评分
5. **建议行动**：根据组合状态自动生成的优先级建议

## 三、具体步骤

### 步骤1：识别请求类型与参数解析

**操作说明：**
- 解析用户输入中的触发短语和参数标记
- 识别请求范围（全量/筛选/日期范围）
- 确定输出格式（详细报告/汇总表格/仅指标）

**判断规则：**

| 输入模式 | 触发条件 | 输出范围 |
|---------|---------|---------|
| `--all` | 显式要求全量 | 所有活跃案件 |
| `--filter 高` | 指定高风险 | 仅🔴高风险案件 |
| `--filter 中` | 指定中风险 | 仅🟠中风险案件 |
| `--filter 低` | 指定低风险 | 仅🟡低风险案件 |
| `--date 本周` | 本周截止 | 7日内截止事项 |
| `--date 本月` | 本月截止 | 30日内截止事项 |
| `--summary` | 仅需汇总 | 指标+评分，无案件列表 |
| 无参数 | 默认全量 | 所有活跃案件 |

### 步骤2：数据筛选与分类

**操作说明：**
- 从案件数据库中提取所有活跃案件
- 按风险等级（高/中/低）对每件案件进行分类
- 收集每件案件的关键信息：案件名称slug、敞口金额、下一截止日期

**风险等级判定标准（民法典相关）：**

| 风险等级 | 判定维度 | 量化标准 |
|---------|---------|---------|
| 🔴 高风险 | 涉案金额≥500万元 且 案件难度高 或 对方当事人有执行难记录 | 金额≥500万 或 涉及群体诉讼/刑民交叉 |
| 🟠 中风险 | 涉案金额100万-500万元 且 案件事实较复杂 | 金额100-500万 且 有争议事实认定 |
| 🟡 低风险 | 涉案金额<100万元 且 案件事实清楚、法律关系简单 | 金额<100万 且 证据充分、对方无异议 |

**分类输出格式：**
```
高风险案件池：[案件列表，按敞口金额降序]
中风险案件池：[案件列表，按敞口金额降序]
低风险案件池：[案件列表，按敞口金额降序]
```

### 步骤3：时间轴提取与截止事项生成

**操作说明：**
- 扫描所有活跃案件的日程安排
- 提取近7日（本周）和近30日（本月的截止事项
- 按日期排序，标注每项截止的事项类型（举证期限/答辩期限/开庭/上诉等）

**截止事项类型定义：**

| 类型标识 | 说明 | 依据 |
|---------|------|-----|
| 举证期限 | 提交证据的截止日期 | 民诉法第68条 |
| 答辩期限 | 提交答辩状的截止日期 | 民诉法第125条 |
| 开庭审理 | 庭审日期 | 民诉法第136条 |
| 上诉期限 | 提起上诉的截止日期 | 民诉法第171条 |
| 执行申请 | 申请强制执行的截止日期 | 民诉法第239条 |
| 保全续期 | 财产保全续期的截止日期 | 民诉法第100条 |

### 步骤4：汇总指标计算

**操作说明：**
- 统计活跃案件总数
- 加总所有案件的敞口金额
- 计算本月预计法律费用（基于案件阶段和平均费用率）
- 计算已消耗预算占总预算的比例

**指标定义：**

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 总案件数 | 活跃案件计数 | 含所有风险等级 |
| 总敞口金额 | Σ(每件案件涉案金额) | 以人民币万元为单位 |
| 本月预计费用 | Σ(高风险案件×1.2 + 中风险×1.0 + 低风险×0.8) | 按风险系数调整 |
| 预算消耗率 | 已消耗预算 / 总预算 × 100% | 反映成本控制状况 |
| 平均案件周期 | Σ(案件持续天数) / 案件数量 | 反映案件处理效率 |

### 步骤5：组合健康度评分

**操作说明：**
- 基于多个维度对组合进行综合评分
- 评分范围0-100分，划分等级

**评分维度与权重：**

| 维度 | 权重 | 评分规则 |
|-----|-----|---------|
| 风险分布 | 30% | 高风险占比>30%：扣20分；20%-30%：扣10分；<20%：满分 |
| 截止日期紧迫度 | 25% | 本周有>3件截止：扣15分；1-3件：扣5分；无：满分 |
| 费用消耗率 | 25% | >80%：扣20分；60%-80%：扣10分；<60%：满分 |
| 案件处理效率 | 20% | 平均周期>18个月：扣15分；12-18个月：扣5分；<12个月：满分 |

**健康度等级划分：**

| 综合评分 | 等级标识 | 颜色 | 说明 |
|---------|---------|------|------|
| 80-100分 | 优秀 | 🟢 | 组合状态健康，风险可控 |
| 60-79分 | 良好 | 🟢 | 组合状态基本正常，需关注个别案件 |
| 40-59分 | 注意 | 🟡 | 存在一定风险，需要主动管理 |
| 20-39分 | 警告 | 🟠 | 风险较高，需要立即关注 |
| 0-19分 | 严重 | 🔴 | 组合状态危急，需要高层介入 |

### 步骤6：建议行动生成

**操作说明：**
- 根据组合状态自动匹配建议行动
- 按优先级排序输出

**行动建议匹配规则：**

| 触发条件 | 建议行动 | 优先级 |
|---------|---------|-------|
| 高风险案件>3件 | 建议召开高风险案件专题讨论会 | P1 |
| 本周截止>5件 | 立即确认各案件截止事项准备状态 | P1 |
| 预算消耗率>80% | 建议重新评估本案预算分配 | P2 |
| 健康度评分<40分 | 建议向管理层提交风险预警报告 | P2 |
| 低风险案件>50% | 建议评估是否可以合并处理以节约成本 | P3 |
| 平均周期>18个月 | 建议对长周期案件进行专项清理 | P3 |

## 六、数据来源与合规说明

**数据来源：**
- 案件信息来源于内部案件管理系统（CMS）
- 风险等级基于民法典及相关司法解释判定
- 费用数据来源于财务系统

**合规要求：**
- 输出内容仅供内部管理使用
- 涉及当事人隐私的信息应进行脱敏处理
- 不得向外部第三方披露

**法律依据：**
- 《中华人民共和国民法典》（2021年1月1日生效）
- 《中华人民共和国民事诉讼法》（2021年12月24日第三次修正）
