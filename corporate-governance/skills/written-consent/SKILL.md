# 股东/董事书面决定合法性审查咨询

## 1. 目的与范围

### 1.1 核心目的
对股东会/董事会书面决定的形式与内容合法性进行审查，识别程序性瑕疵与实体性缺陷，出具审查意见供参考。

### 1.2 适用范围
- 有限责任公司股东书面决定
- 股份有限公司董事会书面决定
- 中外合资经营企业董事会书面决定
- 书面决定的形式合规性审查（召集程序、表决程序、记录存档）
- 书面决定的实质合法性审查（决议事项权限、法律依据、少数股东保护）

### 1.3 本Skill不涵盖
- 股东会/董事会现场会议的组织与主持
- 书面决定的有效性诉讼代理
- 公司登记机关的变更登记手续
- 审计委员会/监事会相关决议审查

## 2. 书面决定类型与法定要求

### 2.1 有限责任公司股东书面决定

**法律依据：《公司法》第59条、第60条**

| 要件 | 要求 | 常见瑕疵 |
|-----|------|---------|
| 适用情形 | 法律或章程规定可采用书面形式的事项 | 误将须召开股东会的重大事项纳入书面决定 |
| 表决权比例 | 按照法律或章程规定的表决比例 | 未达到法定/章程要求的通过比例 |
| 签名要求 | 所有股东签名或盖章 | 缺少部分股东签名/盖章 |
| 书面形式 | 书面文件，记载决定事项 | 口头承诺/聊天记录不构成书面决定 |

**可采用书面形式的典型事项：**
- 股东以书面形式一致同意的决定事项（公司法第59条第2款）
- 日常经营中授权董事长/执行董事决定的事项（公司章程授权范围）
- 法律、章程规定可以简易程序通过的事项

**须召开股东会审议的事项（不得以书面决定替代）：**
- 公司的合并、分立、解散、清算
- 公司形式的变更（有限责任公司→股份有限公司）
- 增加或减少注册资本
- 发行公司债券
- 章程修改
- 公司重大资产处置（单笔或累计达到一定比例）

### 2.2 股份有限公司董事会书面决定

**法律依据：《公司法》第73条、第120条至第124条**

| 要件 | 要求 | 常见瑕疵 |
|-----|------|---------|
| 适用情形 | 全体董事一致同意且章程允许 | 未全体一致或章程未规定书面形式 |
| 最低出席 | 章程规定的最低出席人数 | 凑不够法定人数 |
| 表决通过 | 章程规定的通过比例 | 表决权数未达标 |
| 会议记录 | 书面记录，董事签名 | 缺少签名或日期 |

**股份有限公司董事会特别要求：**
- 董事会会议应当有过半数董事出席方可举行
- 股份有限公司董事会决议的表决，实行一人一票
- 上市公司须遵守证监会关于董事会会议召开的特别规定

### 2.3 中外合资经营企业董事会书面决定

**法律依据：《中外合资经营企业法实施条例》（2023年修订）第30条**

| 要件 | 要求 | 常见瑕疵 |
|-----|------|---------|
| 适用情形 | 合营企业董事会职权范围内事项 | 超出董事会职权（如涉及合营合同变更） |
| 表决要求 | 出席董事会会议的董事一致通过 | 弃权票视为未通过 |
| 法定人数 | 董事会全体董事的三分之二以上出席 | 未达法定人数 |

## 3. 合法性审查清单

### 3.1 程序性审查

```
□ 3.1.1 召集程序
  ├─ 是否具备召集权（董事长、执行董事、监事会/监事）
  ├─ 召集通知是否依法/依章程发出
  └─ 通知期限是否符合法律或章程要求

□ 3.1.2 出席程序
  ├─ 出席人数是否达到法定/章程要求
  ├─ 委托代理是否出具合法授权委托书
  └─ 关联方董事/股东是否依法回避

□ 3.1.3 表决程序
  ├─ 表决权计算是否正确
  ├─ 表决方式是否符合章程规定
  └─ 表决结果是否达到通过比例

□ 3.1.4 记录存档
  ├─ 是否制作书面会议记录
  ├─ 记录是否载明时间、地点、出席人员、议题、表决结果
  └─ 董事签名是否齐全
```

### 3.2 实体性审查

```
□ 3.2.1 决议权限
  ├─ 决议事项是否属于股东会/董事会职权范围
  ├─ 是否越权决定应属其他机关的事项
  └─ 是否违反法律对特定事项的强制性规定

□ 3.2.2 法律依据
  ├─ 决议依据的法律条款是否正确
  └─ 章程规定是否有法律依据

□ 3.2.3 少数股东保护
  ├─ 是否损害了小股东的合法权益
  ├─ 表决比例是否符合法律或章程对特殊事项的要求
  └─ 是否存在滥用资本多数决情形

□ 3.2.4 表决内容合法性
  ├─ 决议内容是否违反法律、行政法规
  ├─ 决议内容是否违反公司章程
  └─ 决议内容是否存在利益冲突未披露情形
```

## 4. 常见瑕疵与法律后果

### 4.1 程序性瑕疵

| 瑕疵类型 | 法律后果 | 可补救性 |
|---------|---------|---------|
| 召集程序瑕疵 | 决议可撤销（公司法第26条） | 30日内可通过重新决议治愈 |
| 出席人数不足 | 决议无效或可撤销 | 须重新召开会议 |
| 表决比例未达标 | 决议无效 | 须重新表决 |
| 未制作会议记录 | 决议可撤销 | 可通过补充记录补救 |

### 4.2 实体性瑕疵

| 瑕疵类型 | 法律后果 | 可补救性 |
|---------|---------|---------|
| 超越职权 | 决议无效 | 须重新作出有效决议 |
| 违反法律强制规定 | 决议无效 | 不可补救 |
| 违反章程 | 决议可撤销 | 可通过章程修订或追认治愈 |

## 5. 免责声明与使用说明

### 5.1 免责声明
本Skill提供的书面决定合法性审查意见仅供参考，不构成法律意见。书面决定的有效性判定具有个案性，最终结论须由执业律师结合具体证据材料作出专业判断。

### 5.2 使用说明
1. 本Skill适用于对已形成的书面决定进行事后审查
2. 审查时应要求当事人提供完整的书面决定文件、公司章程、股东名册等相关材料
3. 如发现重大瑕疵，建议委托执业律师出具正式法律意见书
4. 本Skill审查结论不替代公司登记机关的审查意见
