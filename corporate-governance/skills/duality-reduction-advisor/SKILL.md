# 两权分离与实际控制人问题咨询器 (Duality Reduction Advisor)

## 1. 目的与范围

### 1.1 核心目的
辅助咨询公司股权结构设计中"两权分离"（所有权与控制权分离）安排，以及实际控制人认定相关法律问题，提供监管要求分析、结构方案比较和风险提示。

### 1.2 适用范围
- 有限责任公司/股份有限公司股权结构设计
- 拟上市企业实际控制人认定与持股方案
- 上市公司控制权稳定性分析
- 一致行动人协议安排
- 股权代持清理与披露
- VIE架构拆除
- 对赌协议中的控制权条款

### 1.3 本Skill不涵盖
- 商业条款谈判（估值、定价）
- 交易融资方案设计
- 税务筹划方案（见税务专业人士）
- 会计处理细节（见会计准则）
- 上市申报文件撰写（见保荐机构）

## 2. 实际控制人认定标准

### 2.1 法律定义框架

```
实际控制人认定路径：
│
├── 上市公司标准（证券法体系）
│   ├── 通过投资关系、协议或其他安排
│   │   能够实际支配公司行为
│   │   的自然人、法人或其他组织
│   │
│   └── 认定情形（收购管理办法第84条）：
│       ├── 持股50%以上
│       ├── 持股不足50%但依表决权可主导
│       │   股东大会/董事会半数以上席位
│       ├── 依协议（如一致行动人）可控制
│       └── 其他合法情形
│
└── 拟上市标准（首发注册管理办法）
    ├── 标准同上，但更严格
    ├── 最近3年实际控制人不得变更
    └── 需披露全部关联方与关联交易
```

### 2.2 认定要素 checklist

| 认定要素 | 核查要点 | 常见问题 |
|---------|---------|---------|
| 持股比例 | 直接持股、间接持股、平台持股 | 代持未清理、信托持股 |
| 董事会控制 | 半数以上董事提名/任免 | 过桥董事、名誉董事 |
| 股东大会控制 | 半数以上表决权 | 一致行动人未登记 |
| 协议控制 | 一致行动人协议、委托投票 | 协议效力瑕疵 |
| 经营决策权 | 实际支配公司经营决策 | 口头安排无书面证据 |
| 实际影响 | 对公司重大事项的实际影响力 | 家族企业中亲属关系认定 |

### 2.3 特殊情形认定

#### 2.3.1 股权代持
- **清理要求**：申报前必须完成清理，保留证据链
- **核查要点**：代持原因、还原方式、税务处理、是否存在纠纷
- **监管态度**：不允许隐名代持规避同业竞争或关联交易披露

#### 2.3.2 一致行动人
- **认定标准**：协议约定共同行使表决权/提案权
- **合并计算**：一致行动人持股合并计算超过30%触发要约收购豁免条件
- **信息披露**：上市公司需在权益变动报告书中披露一致行动关系

#### 2.3.3 员工持股平台
- **有限合伙形式**：GP是否为实际控制人
- **信托/资管计划**：受益人认定
- **多层嵌套**：穿透核查原则

## 3. 两权分离结构类型

### 3.1 主要结构对比

| 结构类型 | 控制强度 | 税务效率 | 监管接受度 | 复杂度 |
|---------|---------|---------|-----------|-------|
| 纯股权直接持股 | 强 | 高 | 高 | 低 |
| 有限合伙持股平台 | 中强 | 中等 | 高 | 中 |
| 法人独资公司持股 | 中 | 中 | 中 | 中 |
| VIE架构 | 弱（协议控制） | 低 | 仅境外上市 | 高 |
| 信托持股 | 弱 | 低 | 低（境内） | 高 |
| 一致行动人联盟 | 中 | 高 | 高 | 中 |

### 3.2 有限合伙持股平台

```
典型架构示例：
┌─────────────────────────────────┐
│         创始人/实际控制人          │
│            (GP, 1%)              │
└───────────────┬─────────────────┘
                │ 执行事务合伙人
        ┌───────▼────────┐
        │  有限合伙企业    │
        │  （持股平台）    │
        └───────┬────────┘
                │ 持有股权
        ┌───────▼────────┐
        │    拟上市主体    │
        │   (有限公司)    │
        └────────────────┘
```

**要点说明**：
- GP通过执行事务合伙人身份控制平台
- LP不参与经营，仅享有收益权
- 创始人可将少量财产份额（1%）绑定100%决策权

**风险提示**：
- GP债务风险可能影响持股稳定
- LP退出可能触发股权变动
- 私募基金GP需履行登记义务

### 3.3 一致行动人协议结构

```
适用场景：多名创始股东各自持股但需保持一致行动

协议核心条款：
├── 提案协同：股东大会提案前内部协商
├── 投票协同：按主导方意见统一投票
├── 转让限制：任何一方转让需其他方同意
├── 期限安排：通常3-5年或上市后解锁
└── 争议解决：主导方意见优先

监管关注：
├── 是否存在单一实际控制人
├── 认定是否稳定、清晰
└── 信息披露是否完整
```

## 4. 上市公司监管要求

### 4.1 控制权变更触发机制

| 触发条件 | 具体标准 | 后果 |
|---------|---------|-----|
| 持股30%以上 | 投资者及其一致行动人持股超过30% | 可能触发要约收购 |
| 控制权转移 | 第一大股东变化且主营业务变更 | 重组上市审查 |
| 实际控制人变更 | 证监会认定实际控制人变更 | 需重新申报 |

### 4.2 实际控制人信息披露要求

```
披露时点：
├── 上市申报：招股说明书详细披露
├── 定期报告：年报中"实际控制人情况"
├── 权益变动：5%以上股东变化需披露
└── 控制权变更：及时公告并说明

披露内容要求：
├── 实际控制人直接持股结构图
├── 间接持股/代持情况
├── 一致行动人关系
├── 与公司关联交易情况
└── 最近三年变动情况（拟上市）
```

### 4.3 监管典型问询关注点

- 实际控制人认定依据是否充分、证据是否完整
- 是否存在股权代持或潜在代持
- 一致行动人协议是否稳定、是否有期限安排
- 控制权结构变化对公司经营稳定性的影响
- 实际控制人与公司同业竞争/关联交易情况
- 家族企业中，亲属关系是否构成共同控制

## 5. 典型问题与处理方案

### 5.1 对赌协议中的控制权条款

| 对赌条款类型 | 控制权影响 | 处理建议 |
|------------|----------|---------|
| 股权回购 | 可能触发控制权变动 | 明确触发条件与比例 |
| 优先清算权 | 影响控制权收益分配 | 约定优先顺序 |
| 反稀释 | 可能摊薄控制权 | 设置防稀释条款 |
| 董事会席位 | 影响董事会控制 | 明确席位数量与任免 |
| 一票否决权 | 实质影响经营决策 | 审慎设置否决事项范围 |

### 5.2 VIE架构拆除

```
拆除流程要点：
│
├── 第一步：梳理存量协议（控制协议/独家选择权等）
│   │
├── 第二步：确定新主体（境内WFOE或境内运营主体）
│   │
├── 第三步：股权/资产平移
│   │   ├── 创始人回购境外投资人股权
│   │   └── 或由境内主体承接业务与资产
│   │
├── 第四步：解除VIE协议安排
│   │   ├── 终止控制协议
│   │   └── 注销境外壳公司（如需）
│   │
└── 第五步：税务处理与外汇登记
    ├── 37号文登记（境外投资）
    ├── 顾问意见（税务成本评估）
    └── 外汇管理局说明

监管关注：
├── 拆除过程是否合规、是否存在纠纷
├── 实际控制人是否变更
├── 业务连续性是否受影响
└── 境外投资人退出安排是否合理
```

### 5.3 股权激励与控制权稳定

| 激励工具 | 控制权影响 | 建议 |
|---------|----------|-----|
| 员工持股平台 | GP控制，激励对象为LP | 通过合伙协议约定退出机制 |
| 股票期权 | 行权后稀释控制权 | 设置期权池上限（如10%） |
| 限制性股票 | 授予时即影响股权结构 | 明确解锁条件与转让限制 |
| 虚拟股权 | 不影响工商登记 | 仅适合非上市企业 |

## 6. 风险提示与escalation事项

### 6.1 高风险情形（必须转介律师）

- 拟上市企业实际控制人认定存在争议或模糊地带
- 涉及国有资产的实际控制人界定问题
- 跨境架构中境外壳公司的实际控制人认定
- 股权代持存在纠纷或潜在纠纷
- 控制权变更涉及上市公司重大资产重组
- 对赌协议触发可能导致控制权实质性转移
- 家族企业亲属关系认定共同控制存在分歧

### 6.2 合规检查要点

- [ ] 股权结构图是否完整、是否穿透至最终受益人
- [ ] 代持清理是否完成、证据是否充分
- [ ] 一致行动人协议是否有效、是否有期限
- [ ] 董事会构成是否与股权比例匹配
- [ ] 是否存在未披露的关联方
- [ ] 实际控制人最近三年是否稳定（拟上市）
- [ ] 特殊权利安排（如一票否决权）是否影响控制权认定
