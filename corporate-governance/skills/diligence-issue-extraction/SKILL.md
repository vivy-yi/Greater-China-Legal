# /diligence-issue-extraction — China Mainland

## CN尽职调查重点领域

### 1. 公司主体资格

| 调查项 | 关注风险 |
|---|---|
| 营业执照有效性 | 过期、未年检、吊销 |
| 经营范围合规性 | 须审批的经营范围是否已取得许可 |
| 注册资本实缴情况 | 是否存在虚假出资、抽逃出资 |
| 股东资格 | 外商投资准入、国有企业特殊规定 |

### 2. 股权结构

| 调查项 | 关注风险 |
|---|---|
| 股权代持 | 代持协议效力、实际控制人认定 |
| 优先购买权 | 转让时是否通知其他股东 |
| 股权质押 | 是否存在质押权人 |
| 一致行动人 | 是否有相关协议 |

### 3. 重大合同

| 调查项 | 关注风险 |
|---|---|
| 合同期限 | 即将到期的重要合同 |
| 合同限制 | 转让限制、优先购买权条款 |
| 关联交易 | 定价公允性、利益输送 |
| 担保责任 | 对外担保、连带保证 |

### 4. 劳动人事

| 调查项 | 关注风险 |
|---|---|
| 劳动合同签订 | 是否全员签订 |
| 社会保险 | 是否足额缴纳 |
| 住房公积金 | 是否开户缴纳 |
| 竞业限制 | 高级管理人员的竞业限制协议 |
| 劳务派遣 | 超比例使用、 同工同酬 |

### 5. 知识产权

| 调查项 | 关注风险 |
|---|---|
| 权属清晰 | 商标、专利、著作权是否登记在公司名下 |
| 许可使用 | 是否存在许可协议、许可范围 |
| 侵权风险 | 是否存在侵权纠纷 |
| 商业秘密 | 保密措施是否到位 |

### 6. 税务合规

| 调查项 | 关注风险 |
|---|---|
| 纳税申报 | 是否按时申报 |
| 税收优惠 | 是否符合优惠条件 |
| 欠税情况 | 是否存在欠税 |
| 转让定价 | 关联交易定价是否合理 |

### 7. 诉讼和行政处罚

| 调查项 | 关注风险 |
|---|---|
| 未决诉讼 | 涉案金额、败诉风险 |
| 行政处罚 | 近年处罚情况 |
| 合规整改 | 是否已完成整改 |

## 本技能不做什么

- 不代替律师尽调。只提取和整理问题。
- 不提供法律意见。只描述风险现象。
- 不评估交易价值。只评估法律风险。
