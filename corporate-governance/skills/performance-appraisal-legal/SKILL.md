# 绩效考核与不胜任退出法律合规咨询
# Performance Appraisal & Incompetence Termination Legal Compliance

## 1. 目的与范围

### 1.1 核心目的

为企业在绩效管理全流程中提供法律合规保障，重点解决以下核心问题：
- 绩效考核制度的合法性与合规性设计
- 不胜任工作的认定标准与证据要求
- 以不胜任为由解除劳动合同的操作合规
- 避免违法解除的法律风险

### 1.2 适用范围

- 试用期员工绩效评估与转正决策
- 在职员工年度/季度绩效考核
- 不胜任工作的调岗、培训与解除
- 绩效相关的劳动争议应对

### 1.3 本Skill不涵盖

- 薪酬体系设计（薪酬结构、奖金分配）
- 股权激励计划
- 集体合同与工资集体协商
- 竞业限制与商业秘密保护（见专项Skill）
- 外籍员工特殊规定

## 3. 绩效考核制度设计

### 3.1 制度合法性要件

根据《劳动合同法》第4条，有效的绩效考核制度须满足：

**程序性要件：**
1. **民主程序** — 须经职工代表大会或者全体职工讨论，提出方案和意见
2. **协商程序** — 与工会或者职工代表平等协商确定
3. **公示告知** — 公示或告知全体员工（须员工签字确认）

**实体性要件：**
1. 考核标准须合法、合理，不得违反法律强制性规定
2. 考核标准须具有可操作性，不能过于模糊
3. 不得设置明显过高的淘汰标准

### 3.2 考核标准设计原则

| 原则 | 要求 | 风险提示 |
|-----|------|---------|
| **明确性** | 考核指标须清晰、可量化 | 模糊指标导致争议 |
| **可达成性** | 目标须在合理努力下可达成 | 过高目标可能被认定为无效 |
| **可衡量性** | 须有客观的数据或行为支撑 | 主观评价须有充分依据 |
| **相关性** | 考核内容须与岗位职责相关 | 无关考核可能被认定为无效 |
| **时限性** | 须明确考核周期和时限 | 无时限考核无法认定不胜任 |

### 3.3 绩效考核周期

| 考核类型 | 推荐周期 | 适用场景 |
|---------|---------|---------|
| 试用期考核 | 入职当月/转正前 | 试用期员工评估 |
| 月度考核 | 每月度 | 销售、运营等岗位 |
| 季度考核 | 每季度 | 职能类、管理类岗位 |
| 年度考核 | 每年 | 综合评价、奖金发放 |

**重要提示：** 考核周期过短（如每周考核）可能导致管理成本过高且易引发争议；考核周期过长（如每年一次）则不利于及时发现和纠正绩效问题。

## 4. 不胜任工作的认定

### 4.1 "不能胜任工作"的法律界定

根据《劳动部关于<劳动法>若干条文的说明》（劳办发〔1994〕289号）：

> "不能胜任工作"是指劳动者不能按要求完成劳动合同中约定的任务或者同工种、同岗位人员的工作量。

**认定要素：**
1. 劳动合同或岗位职责中明确约定了工作任务
2. 劳动者未能在合理时间内完成约定任务
3. 用人单位有客观、可衡量的评估标准
4. 用人单位已履行告知和辅导义务

### 4.2 不胜任认定的证据要求

| 证据类型 | 具体要求 | 证据效力 |
|---------|---------|---------|
| **岗位说明书** | 明确岗位职责和任职要求 | 基础证据 |
| **目标约定** | 双方签字确认的工作目标 | 核心证据 |
| **考核记录** | 每次考核结果及员工签字 | 核心证据 |
| **面谈记录** | 绩效面谈的时间、内容、员工意见 | 重要证据 |
| **改进计划** | PIP（绩效改进计划）及执行记录 | 重要证据 |
| **同事/客户评价** | 360度评价（须事先告知） | 辅助证据 |
| **工作成果** | 任务完成情况统计 | 辅助证据 |

### 4.3 常见不胜任认定误区

**❌ 常见误区：**
1. 仅有主观评价，无客观数据支撑
2. 考核标准未事先明确告知员工
3. 考核结果未书面通知员工
4. 未给员工申辩机会
5. 考核周期过短，无法真实反映能力
6. 以"潜在能力"而非"实际表现"评判

**✅ 正确做法：**
1. 制定书面考核标准，经员工签字确认
2. 保留每次考核的书面记录
3. 及时记录绩效问题，留存书面证据
4. 给予员工申辩和改进机会
5. 设定合理的改进期限（通常为1-3个月）

## 5. 不胜任处理流程

### 5.1 两阶段处理原则

根据《劳动合同法》第40条第（二）项，不胜任处理须经两阶段：

```
┌──────────────────────────────────────────────────────────────┐
│                    两阶段处理流程                             │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  【第一阶段：第一次不胜任】                                   │
│  触发条件：连续两次或累计三次考核不达标                        │
│  处理方式：                                                   │
│  ├── 方式一：调岗（调整到与员工能力相匹配的岗位）             │
│  └── 方式二：培训（针对岗位职责进行专业技能培训）              │
│  培训/调岗期限：通常1-3个月                                  │
│  证明文件：《绩效改进计划（PIP）》须员工签字                   │
│                                                              │
│  【第二阶段：第二次不胜任】                                   │
│  触发条件：培训/调岗后仍不能胜任工作                          │
│  处理方式：解除劳动合同                                       │
│  前提条件：已提前30日书面通知或支付一个月工资作为代通知金      │
│  经济补偿：N或N+1                                             │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 5.2 调岗合规要点

**调岗合法性判断：**

| 合规调岗 | 违规调岗（可能被认定为无效） |
|---------|---------------------------|
| 与员工原岗位具有相关性 | 跨行业大幅度调岗（如技术岗调至销售岗） |
| 不具有侮辱性或惩罚性 | 明显带有侮辱性（如经理调至保安） |
| 薪酬水平相当 | 薪酬大幅降低（超过30%） |
| 工作地点合理 | 工作地点跨省调动且无补偿 |
| 基于员工能力 | 因员工投诉、举报而报复性调岗 |

**调岗操作建议：**
1. 调岗前与员工充分沟通，说明调岗原因
2. 书面出具调岗通知书，明确新岗位名称、职责、薪酬
3. 保留员工签收记录
4. 给予员工合理适应期

### 5.3 培训合规要点

**有效培训 vs. 无效培训：**

| 有效培训 | 无效培训（不被法律认可） |
|---------|----------------------|
| 与岗位职责直接相关 | 与岗位无关的培训 |
| 具有针对性和实用性 | 形式化、走过场的培训 |
| 给予正常工资待遇 | 安排待岗或发放最低工资 |
| 有明确的培训计划和目标 | 无明确计划的"回家休息" |
| 保留培训记录和考核结果 | 无任何考核记录 |

**培训操作建议：**
1. 制定书面的《绩效改进培训计划》
2. 明确培训内容、时间、方式、考核标准
3. 培训期间正常支付工资
4. 培训后进行考核，保留考核记录

## 6. 解除劳动合同操作

### 6.1 以不胜任为由解除的必要条件

根据《劳动合同法》第40条，以不胜任为由解除须满足：

| 序号 | 必要条件 | 说明 |
|-----|---------|-----|
| 1 | 有明确、可衡量的考核标准 | 考核标准须事先公示 |
| 2 | 有客观的考核记录 | 考核结果须书面通知员工 |
| 3 | 员工签字确认考核结果 | 或有其他证据证明员工知晓 |
| 4 | 第一次不胜任后进行了调岗或培训 | 须有书面记录 |
| 5 | 第二次仍不能胜任 | 须有第二次考核记录 |
| 6 | 提前30日书面通知或支付代通知金 | 二选一 |
| 7 | 支付经济补偿金 | N或N+1 |

### 6.2 经济补偿金计算

| 补偿类型 | 计算标准 | 适用情形 |
|---------|---------|---------|
| **N** | 每满一年支付一个月工资 | 合同到期不续签、协商解除等 |
| **N+1** | N个月工资 + 一个月工资（代通知金） | 提前30日通知解除 |
| **2N** | 双倍经济补偿金 | 违法解除劳动合同 |

**工资标准：** 解除前12个月平均工资（包括奖金、津贴等）

**年限计算：** 6个月以上不满一年的按一年计算；不满6个月的支付半个月工资

### 6.3 解除流程与文书

```
┌─────────────────────────────────────────────────────────────┐
│                   解除劳动合同流程                           │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  Step 1：制作《劳动合同解除通知书》                          │
│  ├── 解除理由：劳动者不能胜任工作，经过培训或调岗后仍不能    │
│  │            胜任工作                                      │
│  ├── 解除日期：提前30日书面通知的日期或支付代通知金后30日     │
│  ├── 经济补偿：N或N+1                                       │
│  └── 落款：用人单位名称、印章、日期                          │
│                                                              │
│  Step 2：向工会送达《解除通知书》                            │
│  ├── 提前通知工会，听取工会意见                              │
│  └── 工会应在收到通知后及时回复                              │
│                                                              │
│  Step 3：向员工送达《解除通知书》                            │
│  ├── 须员工亲自签收或采用公证送达                            │
│  └── 保留送达证明                                            │
│                                                              │
│  Step 4：办理离职手续                                        │
│  ├── 开具《离职证明》                                        │
│  ├── 办理社保和公积金转移                                    │
│  ├── 结算工资和未休年假                                      │
│  └── 完成工作交接                                            │
│                                                              │
│  Step 5：支付经济补偿金                                      │
│  └── 在解除时一次性支付                                      │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 6.4 常见违法解除情形

| 违法解除情形 | 法律后果 |
|------------|---------|
| 考核标准未事先告知员工 | 违法解除，支付2N |
| 未进行培训或调岗直接解除 | 违法解除，支付2N |
| 培训/调岗后未再次考核即解除 | 违法解除，支付2N |
| 绩效考核结果未经员工确认 | 可能被认定为违法解除 |
| 员工在医疗期、三期、退休前5年被解除 | 违法解除，支付2N |
| 未通知工会即解除（须通知工会的情形） | 违法解除，支付2N |

## 7. 特殊保护群体

### 7.1 不得以不胜任为由解除的人群

根据《劳动合同法》第42条，以下人员不得以不胜任为由解除劳动合同：

| 保护群体 | 保护期限 | 法律依据 |
|---------|---------|---------|
| 从事接触职业病危害作业的劳动者未进行离岗前职业健康检查 | 离职前 | 第38条 |
| 疑似职业病病人 | 诊断期/医学观察期 | 第38条 |
| 在医疗机构诊断的患有某种疾病的人 | 医疗期内 | 第42条（一） |
| 患病或非因工负伤在医疗期内 | 医疗期内 | 第42条（一） |
| 孕期、产期、哺乳期女员工 | 三期内 | 第42条（四） |
| 在用人单位连续工作满15年且距法定退休年龄不足5年 | 不足5年期间 | 第42条（五） |
| 法律、行政法规规定的其他情形 | 依规定 | 第42条（六） |

### 7.2 医疗期规定

| 工龄 | 实际工作年限 | 医疗期长度 | 医疗期计算周期 |
|-----|------------|----------|--------------|
| 10年以下 | 5年以下 | 3个月 | 6个月内累计病休 |
| 10年以下 | 5年以上 | 6个月 | 12个月内累计病休 |
| 10年以上 | 5年以下 | 6个月 | 12个月内累计病休 |
| 10年以上 | 5年以上 | 9个月 | 15个月内累计病休 |
| 10年以上 | 10年以上 | 12个月 | 18个月内累计病休 |
| 10年以上 | 15年以上 | 18个月 | 24个月内累计病休 |
| 10年以上 | 20年以上 | 24个月 | 30个月内累计病休 |

**医疗期工资：** 不得低于当地最低工资标准的80%

## 8. 绩效争议应对

### 8.1 常见劳动争议类型

| 争议类型 | 常见争议焦点 | 举证要点 |
|---------|------------|---------|
| 绩效考核争议 | 考核标准是否公示、考核结果是否客观 | 制度文件、签收记录 |
| 不胜任解除争议 | 是否真的不能胜任、程序是否合规 | 考核记录、面谈记录 |
| 违法解除争议 | 解除理由是否合法、程序是否合规 | 解除通知书、工会通知 |
| 经济补偿争议 | 补偿金计算基数、年限 | 工资流水、劳动合同 |

### 8.2 举证责任分配

| 情形 | 举证责任 | 说明 |
|-----|---------|-----|
| 用人单位主张员工不胜任 | 用人单位 | 须提供充分证据 |
| 员工主张违法解除 | 员工 | 须提供初步证据 |
| 考核标准公示 | 用人单位 | 须证明员工知晓 |
| 考核结果通知 | 用人单位 | 须证明员工收到 |
| 培训/调岗安排 | 用人单位 | 须证明已履行义务 |

### 8.3 典型案例裁判要旨

**案例一：考核标准未公示被认定违法解除**

> 某公司以员工绩效考核不达标为由解除劳动合同，但未能举证证明考核标准已向员工公示。法院认定公司构成违法解除，判决支付2N赔偿金。

**裁判要旨：** 用人单位须举证证明考核标准已向员工公示，否则承担不利后果。

**案例二：调岗不合理被认定违法解除**

> 某公司以员工不胜任工作为由，将其从技术总监调至客服岗位，薪酬降低50%。法院认为调岗明显不合理，构成违法解除。

**裁判要旨：** 调岗须具有合理性，薪酬大幅降低可能被认定为惩罚性调岗。

**案例三：未给改进机会径直解除**

> 某公司以员工一次考核不达标为由直接解除劳动合同。法院认定公司未履行"培训或调岗"的前置程序，构成违法解除。

**裁判要旨：** 以不胜任为由解除须经两阶段程序，未经培训或调岗径直解除构成违法解除。

## 9. 合规检查清单

### 9.1 绩效考核制度合规检查

| 序号 | 检查项 | 是 | 否 | 备注 |
|-----|-------|----|----|-----|
| 1 | 考核制度是否经民主程序制定 | □ | □ | 须有职工代表大会记录 |

## 10. 升级决策门

### 10.1 必须升级的情形

以下情况必须升级给执业律师进行判断：

1. **试用期员工评估**
   - 试用期延长或试用期解除的合法性
   - 试用期不符合录用条件的认定

2. **特殊保护群体**
   - 医疗期员工、三期女员工的绩效处理
   - 距法定退休年龄不足5年的老员工处理

3. **集体性绩效问题**
   - 多个员工同时绩效考核不达标
   - 涉及经济性裁员可能性

4. **历史遗留问题**
   - 员工已签署绩效考核确认文件，但事后主张非自愿
   - 员工已有劳动仲裁或诉讼记录

5. **外资/国有企业特殊规定**
   - 外资企业解除劳动合同的额外审批要求
   - 国有企业员工安置的特殊规定

### 10.2 升级格式

当需要升级时，按以下格式输出：

```
【升级咨询】
问题描述：[具体法律问题]
背景信息：[相关事实摘要]
已核实情况：[已确认的信息]
初步判断：[可能的法律风险]
建议行动：[需要律师确认的事项]
优先级：紧急/高/中
```
