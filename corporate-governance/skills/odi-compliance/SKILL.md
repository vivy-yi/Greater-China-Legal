# /odi-compliance — 对外直接投资（ODI）合规

## ODI适用情形

### 须办理ODI的情形

| 情形 | 是否须ODI |
|---|---|
| 境内企业在境外新设公司 | ✅ |
| 境内自然人在境外设立公司（含VIE架构中的香港公司） | ✅（37号文） |
| 境内企业收购境外企业股权 | ✅ |
| 境外企业利润汇回 | ✅（先备案才能汇回） |

---

## 备案/核准流程

### 主管机关

| 投资类型 | 主管机关 | 备注 |
|---|---|---|
| 一般境外投资 | 商务部/省级商务厅 | 备案制 |
| 敏感国家/地区/行业 | 商务部 | 核准制 |
| 金融类境外投资 | 银保监会/证监会 | 金融监管部门核准 |
| 涉及国企 | 国资委 | 须国资委审批 |

### 发改委立项

| 投资规模 | 要求 |
|---|---|
| <3000万美元 | 备案 |
| ≥3000万美元 | 核准 |
| 敏感国家/地区/行业 | 核准（不分金额） |

---

## 37号文登记（境内自然人）

### 须办理的情形

| 情形 | 是否须37号文 |
|---|---|
| 中国自然人持境外上市公司股权 | ✅ |
| 中国自然人通过VIE架构控制境外公司 | ✅ |
| 中国自然人取得境外公司股权 | ✅ |

### 登记流程

1. **办理特殊目的公司外汇登记**（在外汇管理局或其授权银行）
2. 登记内容包括：境内居民身份、境外投资路径、境外公司信息

### 未登记后果

| 后果 | 说明 |
|---|---|
| 境外投资收益无法汇回 | 外汇管制限制 |
| 境内银行贷款用途受限 | 银行贷前审查 |
| 行政处罚 | 罚款 + 通报批评 |
| 影响后续融资 | 投资人尽调会发现 |

---

## 外汇登记

### 银行层面

- ODI备案完成后，在银行办理外汇登记
- 登记后即可购汇汇出投资款

### 资金汇出限制

| 要求 | 说明 |
|---|---|
| 外汇登记 | 须先完成ODI备案 |
| 资金来源 | 须说明合法资金来源 |
| 汇出额度 | 根据备案金额 |
