# 公司治理判断框架 — 中国大陆

> 本框架用于 corporate-governance 28 个 skill 的场景识别与判断逻辑，作为公司治理场景的统一依据。
> corporate-governance 是**公司全生命周期 + 治理 + 融资**场景，覆盖企业从设立到上市/退出的核心法律事务。
> 各 skill 内部已有判断逻辑，本文件描述跨 skill 的整体决策路由。

## 三、公司类型判断

```
[节点 1] 组织形式
├─ 有限责任公司 → 公司法第 1-83 条
│   ├─ 普通有限责任公司
│   ├─ 一人有限责任公司（公司法第 63 条）
│   └─ 国有独资公司
├─ 股份有限公司 → 公司法第 84-145 条
│   ├─ 上市公司（特殊规定）
│   └─ 非上市股份公司
├─ 有限合伙企业 → 合伙企业法
│   ├─ 普通合伙（GP/LP）
│   └─ 特殊普通合伙
├─ 个人独资企业 → 个人独资企业法
└─ 外商投资企业 → 外商投资法（2020.01.01）+ 公司法

[节点 2] 是否上市
├─ 上市公司 → 证监会 + 交易所规则
│   ├─ 信息披露（证券法第 78-85 条）
│   ├─ 关联交易（上市公司治理准则）
│   └─ 内幕交易（证券法第 50-53 条）
├─ 新三板挂牌 → 全国中小企业股份转让系统
└─ 非上市 → 公司法为主

[节点 3] 行业
├─ 金融 → 央行/金监总局/证监会监管
├─ 房地产 → 住建部监管
├─ 医药 → 药监局监管
├─ 互联网 → 网信办/工信部监管
└─ 一般行业 → 市场监管总局
```

## 四、股权架构设计（equity-split-design）

### 4.1 创始人股权分配

```
[节点 1] 创始团队构成
├─ 单一创始人 → 100% 持股（设立时）
├─ 多人创始人 → 按贡献分配
│   ├─ 出资比例（资金贡献）
│   ├─ 技术/资源贡献
│   ├─ 运营贡献
│   └─ 谈判调整
└─ 多轮融资 → 预留股权池

[节点 2] 股权分配考虑
├─ 决策权（67% 绝对控股 / 51% 相对控股 / 34% 一票否决）
├─ 退出权（清算优先权 / 反稀释）
├─ 经济权（分红 / 增资 / 减资）
├─ 成熟期（vesting）
└─ 限制（锁定 + 优先购买权）

[节点 3] 创始人控制权工具
├─ 投票权委托（Voting Proxy）
├─ 持股平台（有限合伙 / 有限责任公司）
├─ AB 股（公司法不直接支持，但可通过协议实现）
├─ 一致行动人协议
└─ 董事会席位安排
```

### 4.2 ESOP 设计（esop-incentive-plan）

```
[节点 1] 激励工具
├─ 股票期权（option）
├─ 限制性股票（RSU）
├─ 虚拟股权（phantom）
├─ 员工持股计划（ESOP）
└─ 股票增值权（SAR）

[节点 2] 授予要素
├─ 授予对象（4 类员工）
├─ 授予数量（一般 10-20%）
├─ 行权价格（平价/折价）
├─ 成熟期（4 年 + 1 年 cliff）
├─ 归属期（vesting schedule）
└─ 退出机制

[节点 3] 实施程序
├─ 公司章程修订
├─ 股东会决议
├─ 激励计划草案
├─ 签署授予协议
├─ 行权登记
└─ 上市后行权/转让
```

## 五、融资判断（financing-cycle）

### 5.1 融资轮次

```
阶段 1：天使轮（Angel）
  ├─ 金额：50-500 万
  ├─ 来源：天使投资人/孵化器
  └─ 估值：500-3000 万

阶段 2：Pre-A
  ├─ 金额：300-1000 万
  └─ 来源：天使 + FA

阶段 3：A 轮
  ├─ 金额：1000-5000 万
  ├─ 来源：VC 机构
  └─ 估值：5000 万 - 3 亿

阶段 4：B 轮
  ├─ 金额：5000 万 - 数亿
  └─ 来源：VC + 战投

阶段 5：C/D/E/Pre-IPO
  └─ 后期融资

阶段 6：IPO
  ├─ A 股：证监会 + 交易所
  ├─ 港股：联交所
  └─ 美股：SEC
```

### 5.2 关键条款

```
[节点 1] 估值调整
├─ pre-money / post-money
├─ 估值方法（DCF / 可比 / 风险调整）
└─ 反稀释条款

[节点 2] 投资人权利
├─ 优先购买权（ROFR）
├─ 共同出售权（Tag-along）
├─ 强制随售权（Drag-along）
├─ 优先清算权（Liquidation Preference）
├─ 保护性条款（Protective Provisions）
├─ 反稀释条款（Anti-dilution）
└─ 董事提名权

[节点 3] 公司治理
├─ 董事会构成
├─ 投资人保护性条款
├─ 一票否决权范围
├─ 信息披露义务
└─ 财务审计
```

## 六、董事会与股东会

### 6.1 董事会（board-minutes / board-resolution-reviewer）

```
[节点 1] 召集程序
├─ 召集通知（提前 10 日，章程可另行规定）
├─ 通知内容（时间/地点/议程）
├─ 董事出席
└─ 委托出席（书面委托）

[节点 2] 议事程序
├─ 法定人数（过半数）
├─ 表决方式（多数决/一致决）
├─ 关联董事回避
├─ 决议方式（书面/现场）
└─ 会议记录（签字）

[节点 3] 决议事项
├─ 普通决议（过半数）
├─ 特别决议（2/3 多数）
├─ 法定事项（公司法第 49 条）
└─ 章程规定的其他事项
```

### 6.2 股东会（shareholder-meeting-advisor）

```
[节点 1] 召集程序
├─ 召集人（董事会/监事会/少数股东）
├─ 通知（提前 15 日，章程可另行规定）
├─ 通知内容
└─ 临时股东会（提议 10% 表决权）

[节点 2] 议事程序
├─ 表决权（按出资比例，章程可另行规定）
├─ 法定人数
├─ 决议方式
└─ 累积投票制（董事/监事选举）

[节点 3] 法定职权（公司法第 59 条）
├─ 决定经营方针和投资计划
├─ 选举/更换董事/监事
├─ 审议批准董事会报告
├─ 审议批准年度财务预算/决算
├─ 审议批准利润分配/弥补亏损
├─ 对增减资/合并/分立/解散作出决议
└─ 修改章程
```

## 七、股权转让（transfer-restriction-advisor）

### 7.1 法定限制

```
[节点 1] 有限责任公司
├─ 股东之间 → 可自由转让（章程可另有规定）
├─ 股东向外转让 → 须其他股东过半数同意
│   ├─ 其他股东有优先购买权（同等条件）
│   ├─ 30 日内未答复视为同意
│   └─ 章程可另行规定
├─ 一人公司 → 可自由转让
└─ 国有股权 → 须评估 + 进场交易

[节点 2] 股份有限公司
├─ 发起人 → 1 年内不得转让
├─ 董监高 → 1 年内不得转让
├─ 上市前股东 → 锁定期（交易所规定）
├─ 上市后 → 减持规则
└─ 国有股 → 特殊规定
```

### 7.2 优先购买权冲突处理

```
Step 1：通知其他股东
  - 书面通知 + 转让条件

Step 2：其他股东表态（30 日内）
  ├─ 同意 → 进入转让
  ├─ 行使优先购买权 → 内部转让
  └─ 弃权 → 外部转让

Step 3：如有冲突
  - 两个以上股东主张优先购买权 → 协商 / 按出资比例
  - 部分股东主张 → 多重处理
```

## 八、关联交易（related-party-transaction）

### 8.1 关联人认定

```
[节点 1] 关联自然人
├─ 实际控制人
├─ 董监高
├─ 5% 以上股东
└─ 上述人员关系密切家庭成员

[节点 2] 关联法人
├─ 同受一方控制/重大影响
├─ 联营/合营企业
├─ 主要投资者个人控制的其他企业
└─ 关键管理人员控制的其他企业

[节点 3] 上市公司额外
├─ 关联董事 12 个月内的所有关联方
├─ 持股 5% 以上股东
└─ 其他
```

### 8.2 关联交易决策程序

```
[节点 1] 非上市公司
├─ 章程规定程序
├─ 关联股东/董事回避
└─ 章程未规定 → 股东会决议

[节点 2] 上市公司（更严格）
├─ 关联董事回避表决
├─ 独立董事事前认可
├─ 董事会决议
├─ 关联股东在股东会回避
└─ 累计 12 个月关联交易额度限制

[节点 3] 信息披露
├─ 上市公司：及时披露
├─ 重大关联交易：股东会审议
└─ 业绩预告关联事项
```

## 九、VIE 结构（vie-red-flag）

```
[节点 1] VIE 适用场景
├─ 外资禁止/限制行业
├─ 互联网/教育/媒体（典型）
├─ 境外上市需求
└─ 短期政策不变预期

[节点 2] VIE 架构核心
├─ 境外上市主体（开曼/BVI 公司）
├─ 境外 WFOE（外商独资企业）
├─ 境内运营公司（持牌 + VIE 协议控制）
└─ VIE 控制协议（9 类）

[节点 3] VIE 控制协议
├─ 独家服务协议
├─ 独家购股权协议
├─ 借款协议
├─ 股权质押协议
├─ 委托表决协议
├─ 配偶同意函
├─ 知识产权许可
├─ 不竞争协议
└─ 资产运营托管

[节点 4] VIE 风险评估
├─ 政策风险（外商投资法实施后）
├─ 协议可执行性（最高法第 33 号指导案例）
├─ 外汇风险
├─ 税务风险
├─ 监管态度（按行业）
└─ 红筹回归路径（境外上市备案 + A 股回归）

升级触发：
- VIE 搭建 → 律师 + 投行专项
- VIE 拆除 → 律师 + 投行 + 税务专项
```

## 十一、收购/合并（diligence-issue-extraction / closing-checklist）

### 11.1 收购方式

```
[节点 1] 资产收购 vs 股权收购
├─ 资产收购：购买目标公司部分或全部资产
│   ├─ 优点：风险隔离
│   └─ 缺点：需逐项转让
└─ 股权收购：购买目标公司股权
    ├─ 优点：保留主体
    └─ 缺点：承担负债

[节点 2] 交易对价
├─ 现金
├─ 股权（换股）
├─ 混合
└─ 收益法/资产法/市场法估值

[节点 3] 交易结构
├─ 直接收购
├─ 间接收购（通过子公司）
├─ 一致行动
└─ 联合投资
```

### 11.2 尽调维度

```
[节点 1] 法律尽调
├─ 主体资格
├─ 主要资产
├─ 主要合同
├─ 知识产权
├─ 诉讼仲裁
├─ 合规
└─ 关联方

[节点 2] 财务尽调
├─ 三大报表
├─ 收入确认
├─ 成本费用
├─ 应收账款
└─ 现金流

[节点 3] 业务尽调
├─ 商业模式
├─ 客户结构
├─ 供应商
├─ 竞争格局
└─ 团队

[节点 4] 合规尽调
├─ 反垄断申报
├─ 行业许可
├─ 数据合规
├─ 环保合规
└─ 劳动合规
```

### 11.3 交割清单（closing-checklist）

```
[节点 1] 交割前
├─ 各项同意取得（股东会/董事会/政府）
├─ 反垄断申报完成
├─ 重组完成（如有）
├─ 资金到位
└─ 过渡期事项

[节点 2] 交割日
├─ 股东变更登记
├─ 章程修订
├─ 印鉴/证照交接
├─ 资料交接
├─ 资金支付
└─ 交割确认书

[节点 3] 交割后
├─ 整合管理启动
├─ 业绩对赌执行
├─ 未披露事项处理
├─ 尾款支付
└─ 协同效应实现
```

## 十二、关联保护与小股东权利（minority-rights-protection）

### 12.1 少数股东权利

```
法定权利：
├─ 知情权（查阅账簿，公司法第 33 条）
├─ 召集股东会权（10% 表决权）
├─ 股东会决议撤销权（公司法第 22 条）
├─ 异议股东回购请求权（公司法第 74 条）
├─ 司法解散请求权（公司法第 182 条）
├─ 派生诉讼权（公司法第 151 条）
└─ 直接诉讼权（侵权时）

章程可特别规定：
├─ 累积投票制
├─ 一票否决权（章程设计）
└─ 类别股东保护
```

### 12.2 股东诉讼路径

```
[节点 1] 直接诉讼
├─ 股东个人权利受侵害
├─ 股东会/董事会决议违法
└─ 章程修订损害股东权利

[节点 2] 派生诉讼
├─ 公司利益受损
├─ 公司未起诉
├─ 股东代位起诉（1% 持股或章程规定）
└─ 监事会/类似机构起诉

[节点 3] 异议股东回购
├─ 法定情形（公司法第 74 条）
├─ 程序（股东会决议通过 60 日内）
└─ 价格协商或评估
```

## 十三、公司治理运作的特殊问题

### 13.1 公司僵局（judicial dissolution）

```
认定：
├─ 股东会/董事会持续无法召开
├─ 股东会/董事会决议无法形成
├─ 经营管理严重困难
├─ 公司继续存续会使股东利益受到重大损失
└─ 其他途径无法解决

救济路径：
├─ 调解
├─ 强制股权收购
├─ 司法解散（公司法第 182 条）
└─ 破产清算
```

### 13.2 董事责任与责任保险

```
董事义务：
├─ 忠实义务（公司法第 147 条）
├─ 勤勉义务（公司法第 147 条）
├─ 避免利益冲突
└─ 不得泄露秘密

责任类型：
├─ 民事赔偿（违反义务致公司/股东损失）
├─ 行政处罚（违法行为）
└─ 刑事责任（如职务侵占、挪用）

董事责任保险：
├─ 覆盖范围：董事因履职引发的索赔
├─ 不覆盖：故意违法/欺诈/重大过失
└─ 保额/保费
```

## 十四、利润分配（profit-sharing-plan）

```
[节点 1] 分配程序
├─ 弥补亏损（公司法第 166 条）
├─ 提取法定公积金（10%）
├─ 提取任意公积金（章程规定）
└─ 股东分配（按出资比例）

[节点 2] 分配限制
├─ 弥补亏损前不得分配
├─ 公积金转增资本须依法
└─ 上市前分配须考虑上市影响

[节点 3] 股东借款 vs 分红
├─ 借款：股东与公司借贷关系
├─ 分红：必须符合法定程序
└─ 视同分配（个人借款视同分红）

升级触发：
- 大额分红 → 律师 + 财务顾问
- 跨境分红 → 外汇 + 税务
- 上市前分配 → 律师 + 投行
```

## 十六、与其他 plugin 的差异

| 维度 | corporate-governance | ip-infringement | contract-review |
|---|---|---|---|
| 关注重点 | 公司治理 + 融资 + 股权 | IP 侵权 | 合同条款 |
| 输出类型 | 治理方案 + 股权设计 + 融资条款 | 侵权认定 | 合同审查 |
| 数据需求 | 工商 + 公司法 + 案例 | 专利商标数据库 | 法规 + 行业惯例 |
| 时效要求 | 低（治理稳定） | 中 | 中 |
| 风险等级 | 中-高 | 高 | 中-高 |
| 涉外因素 | VIE/ODI/红筹 | 涉外 IP | 涉外合同 |

**核心差异**：corporate-governance 是**公司全生命周期**视角（设立→治理→融资→上市/退出），与 IP 维权、合同审查有显著差异。
