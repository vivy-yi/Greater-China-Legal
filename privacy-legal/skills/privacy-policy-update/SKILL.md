# /privacy-policy-update — China Mainland

## CN法律依据

**PIPL 第17条 — 隐私政策必须包含的内容：**

```
(一) 个人信息处理者的名称或姓名和联系方式；
(二) 个人信息的处理目的、处理方式，处理的个人信息种类、保存期限；
(三) 个人行使权利的方式和程序；
(四) 法律、行政法规规定的其他事项。
```

**PIPL 第7条：** 处理个人信息应当遵循公开、透明原则，公开个人信息处理规则，明示处理的目的、方式和范围。

## 隐私政策必须包含的内容

### 法定内容（PIPL 第17条）

| 内容 | 说明 | 对应PIPL条款 |
|---|---|---|
| 个人信息处理者名称/联系方式 | 公司全称、地址、联系方式 | 第17条(一) |
| 处理目的和方式 | 为什么要处理、怎么处理 | 第17条(二) |
| 个人信息种类 | 收集哪些类型的数据 | 第17条(二) |
| 保存期限 | 数据保留多长时间 | 第17条(二) |
| 权利行使方式和程序 | 用户如何查询/删除/更正 | 第17条(三) |
| 处理规则公开 | 在明显位置公开 | 第7条 |

### PIPL额外要求

| 内容 | 说明 | 对应PIPL条款 |
|---|---|---|
| 敏感个人信息告知 | 生物识别/医疗等须说明处理必要性 | 第29条 |
| 未成年人信息处理 | 处理不满14周岁须取得监护人同意 | 第31条 |
| 委托处理 | 须说明委托处理的情况 | 第21条 |
| 共同处理 | 须明确各方权利义务 | 第20条 |
| 跨境传输 | 须说明境外接收方及保障措施 | 第39条 |
| 自动化决策 | 须告知并提供便捷拒绝选项 | 第24条 |
| 拒绝提供个人信息的权利 | 告知不提供信息的后果 | 第29条 |

## 审查清单

| 检查项 | 说明 |
|---|---|
| □ 个人信息处理者信息完整 | 名称、地址、联系方式、保护负责人 |
