## 常用查询场景

### 场景1：经济补偿金基数是否含加班费/奖金

**查询语句：**
```
法律精灵："[城市] 经济补偿金基数 加班费" 或 "[城市] 经济补偿金基数 奖金" 或 "[城市] 解除劳动合同 经济补偿金 计算基数"
```

**判断标准（见下表）：**

| 城市 | 是否计入加班费 | 是否计入奖金 | 依据 |
|---|---|---|---|
| 北京 | 否 | 否 | 北京高院解答 |
| 上海 | 否 | 否 | 上海高院解答 |
| 广东 | 否 | 否 | 广东高院解答 |
| 江苏 | 否 | 否 | 江苏高院解答 |
| 浙江 | 否 | 否 | 浙江高院解答 |
| 四川 | 否 | 否 | 四川高院解答 |
| 重庆 | 否 | 否 | 重庆高院解答 |
| 全国一般 | 否 | 否 | 劳动合同法实施条例第27条 |

**⚠️ 注意：口径可能变化，查询结果须与最新官方文件核对。**

### 场景4：违法解除赔偿金2N计算

**查询语句：**
```
法律精灵："[城市] 违法解除 赔偿金 2N" 或 "[城市] 赔偿金 计算 方法"
```

**判断标准：**

| 判断要素 | 标准 |
|---|---|
| N的计算方式 | 工龄×月工资 |
| 月工资标准 | 离职前12个月平均工资（含奖金/补贴，不含加班费） |
| 上限 | 职工月平均工资3倍（超过则按3倍计算） |
| 年资上限 | 12年（超过12年按12年计算） |
| 2N | 2×N（违法解除） |

## 查询结果处理

### 情形一：多个数据源结论一致
→ 高置信度，直接使用

### 情形二：只有单个数据源有结论
→ 中置信度，建议用户向当地仲裁委（12333热线）确认

### 情形三：多个数据源结论矛盾
→ 呈现差异，优先采用：
  1. 官方文件（人社局官网/仲裁委文件）
  2. 法院判例（中院/高院）
  3. 律师分析文章
  4. 学术研究

### 情形四：完全查不到
→ 使用全国一般标准，注明"无本地数据"
→ 明确告知用户这是风险点

## 常用查询入口

### [YD] 法律精灵
```
https://fasheng.cn/ （劳动法综合检索）
关键词格式：[城市] + [争议类型] + [具体问题]
```

### [GOV] 裁判文书网
```
https://wenshu.court.gov.cn/
检索方式：案由 + 法院级别 + 裁判年份
建议：中院/高院判例优先
```
