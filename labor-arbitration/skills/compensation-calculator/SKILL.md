# 劳动争议赔偿计算器

## 核心能力

根据输入的案件参数，自动计算各类劳动赔偿金额。

## 支持类型

| 类型 | 说明 |
|------|------|
| 工伤赔偿 | 一次性伤残补助金、医疗补助金、就业补助金 |
| 违法解除赔偿 | 2N赔偿金 |
| 代通知金 | 一个月工资 |
| 经济补偿金 | N或N+1 |
| 加班费 | 工作日/休息日/法定节假日 |

---

## 计算公式

### 一、工伤赔偿

```
一次性伤残补助金 = 本人工资 × 月数（等级决定）
  • 一级：27个月
  • 二级：25个月
  • 三级：23个月
  • 四级：21个月
  • 五级：18个月
  • 六级：16个月
  • 七级：13个月
  • 八级：11个月
  • 九级：9个月
  • 十级：7个月

一次性工伤医疗补助金 = 当地社平工资 × 月数
一次性伤残就业补助金 = 当地社平工资 × 月数
```

**月数标准**（各地略有差异，以当地规定为准）

| 等级 | 医疗补助金 | 就业补助金 |
|------|-----------|-----------|
| 五级 | 36个月 | 36个月 |
| 六级 | 30个月 | 30个月 |
| 七级 | 20个月 | 20个月 |
| 八级 | 15个月 | 15个月 |
| 九级 | 10个月 | 10个月 |
| 十级 | 6个月 | 6个月 |

### 二、违法解除赔偿金（2N）

```
赔偿金 = 本人月工资 × 工作年限 × 2
```

- **本人月工资**：解除前12个月平均工资
- **工作年限**：不满6个月按0.5年计算，超6个月按1年计算
- **上限**：月工资不超过当地社平工资3倍；年限最高12年

### 三、代通知金

```
代通知金 = 上个月工资（提前30天通知可免）
```

或支付一个月工资替代提前通知。

### 四、经济补偿金（N或N+1）

```
经济补偿金 = 本人月工资 × 工作年限（N）
  + 代通知金（+1，如适用）
```

**N计算规则**：
- 每满1年计1个月
- 6个月以上不满1年按1年计算
- 不满6个月计0.5个月
- 上限：月工资不超过社平工资3倍，年限最高12年

### 五、加班费

```
工作日加班：时薪 × 1.5倍 × 时数
休息日加班：日薪 × 2倍 × 天数（或安排补休）
法定节假日：日薪 × 3倍 × 天数
```

## 使用流程

```
用户输入：赔偿类型 + 关键参数（工资、工龄、伤残等级等）
    ↓
识别赔偿类型
    ↓
提取/计算各项参数
    ↓
应用公式计算
    ↓
输出赔偿明细 + 合计金额
```
