## 一、核心判断框架

### 1.1 执法文书类型识别

| 文书类型 | 法律效力 | 企业应对重点 |
|---|---|---|
| 检查通知书 | 程序性告知 | 合规资料准备、人员安排 |
| 整改通知书 | 行政命令 | 查明问题根源、制定整改方案 |
| 行政处罚告知书 | 拟处罚决定 | 陈述申辩、听证申请 |
| 行政处罚决定书 | 最终处罚 | 执行处罚、评估救济途径 |
| 强制执行决定 | 执行依据 | 配合执行或申请复议 |

### 1.2 违规情节认定标准

| 情节级别 | 认定因素 | 典型后果 |
|---|---|---|
| 轻微 | 初次违规、已及时纠正、无危害后果 | 责令改正/警示 |
| 一般 | 存在一定危害但可控 | 罚款/没收违法所得 |
| 严重 | 主观故意、大规模危害、持续时间长 | 重大罚款/停业整顿 |
| 特别严重 | 涉及重大公众利益、恶意违法 | 吊销许可证/追究刑责 |

### 1.3 从轻/减轻情节

- 主动消除或减轻危害后果
- 主动供述监管机构尚未掌握的违法行为
- 积极整改取得明显效果
- 初次违法且危害后果轻微
- 配合监管调查有立功表现

## 三、分析流程

### 3.1 执法应对步骤

```
第一步：文书审阅 → 仔细阅读执法文书，明确事实依据和法律依据
第二步：事实核查 → 内部调查核实违规事实是否属实
第三步：法律分析 → 评估执法程序合法性和处罚适当性
第四步：策略制定 → 确定应对策略（配合整改/陈述申辩/听证）
第五步：材料准备 → 准备申辩材料或整改方案
第六步：正式提交 → 在法定期限内提交材料
第七步：跟进执行 → 跟踪处理结果并执行决定
```

### 3.2 陈述申辩要点清单

| 要点类型 | 内容 |
|---|---|
| 事实层面 | 对违规事实的澄清或异议 |
| 法律层面 | 适用法律是否正确、程序是否合法 |
| 量罚层面 | 处罚幅度是否与违规情节相适应 |
| 从轻情节 | 主动消除后果、配合调查等情形 |
| 历史表现 | 过往无违规记录、合规体系建设情况 |

### 3.3 整改方案要素

| 要素 | 说明 |
|---|---|
| 事实承认 | 对违规事实的准确认定 |
| 原因分析 | 产生问题的深层次原因 |
| 整改措施 | 具体可操作的纠正行动 |
| 制度完善 | 防止同类问题再发生的制度改进 |
| 完成时间 | 明确的整改期限承诺 |

## 升级决策门

以下情形应向上级 Skill 升级：

- 涉及重大行政处罚（罚款超一定金额或可能被停业整顿）
- 涉及许可证被吊销或业务被暂停的重大风险
- 需要提起行政复议或行政诉讼
- 涉及刑事风险（案件移送公安）
- 案件引起媒体广泛关注或舆情危机

---

## 本Skill不涵盖

- 代理企业参加听证或诉讼
- 提供正式法律意见书（需由执业律师出具）
- 直接与监管机构进行沟通协调
- 具体的会计处理和财务报告调整
- 企业内部的纪律处分决策
## 本 Skill 不涵盖

- 行政处罚决定书的正式申辩文本
- 行政复议/行政诉讼的代理
- 整改方案的执行监督
- 刑事责任的辩护
