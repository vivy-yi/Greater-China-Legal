## 一、核心判断框架

### 1.1 许可证类型识别

| 许可证类型 | 典型行业 | 核心判断要素 |
|---|---|---|
| 业务准入许可 | 金融、医疗、教育 | 注册资本、人员资质、场地设施 |
| 产品/服务许可 | 药品器械、食品 | 备案/注册、GMP/GSP认证 |
| 经营资质许可 | 建筑、矿产、烟草 | 资质等级、安全生产许可 |
| 人员执业许可 | 律师、会计师、医师 | 资格考试、继续教育、执业注册 |

### 1.2 核查逻辑

1. **业务定性**：将业务描述映射到监管分类
2. **资质要求**：识别该业务需要的许可证类型
3. **现状比对**：核对现有证照与要求的差距
4. **风险评级**：无证/超范围/即将到期三级风险

### 1.3 常见许可证失效情形

- 有效期届满未续期
- 法定代表人/地址变更未备案
- 重大违规被吊销
- 达不到最低维持条件（资本、人员）被注销

## 三、分析流程

### 3.1 许可证完备性核查步骤

```
第一步：业务拆解 → 将主营业务拆解为具体可核查的业务单元
第二步：资质匹配 → 查找对应监管法规中的资质要求
第三步：现状比对 → 收集现有证照与要求逐项核对
第四步：差距分析 → 识别缺失、超范围、临期证照
第五步：风险评级 → 根据风险等级出具整改建议
```

### 3.2 许可证有效期管理矩阵

| 状态 | 风险等级 | 建议操作 |
|---|---|---|
| 有效期>1年 | 低 | 常规监控 |
| 有效期6-12个月 | 中 | 启动续期准备 |
| 有效期<6个月 | 高 | 立即启动续期申请 |
| 已过期 | 极高 | 立即停止相关业务+申请重新核发 |

### 3.3 超范围经营判断标准

- **形式超范围**：许可证载明业务与企业实际从事业务不一致
- **实质超范围**：业务实质落入需许可范围但许可证未载明
- **穿透判断**：即使营业执照经营范围包含，仍需核查专项许可证

## 升级决策门

以下情形应向上级 Skill 升级：

- 涉及跨行业、跨监管领域的综合资质评估（→ 行业特定合规）
- 涉及许可证被吊销后的行政复议或诉讼（→ enforcement-response-advisor）
- 涉及境外资质互认或跨境业务许可证问题
- 涉及IPO、重大资产重组中的资质核查需要律师意见

## 本Skill不涵盖

- 许可证申请材料的准备与提交（由代理机构或企业法务处理）
- 许可证审批结果的行政复议程序
- 具体的场地设施合规验收（消防、环保等）
- 人员执业资格考试报名与备考指导
## 本 Skill 不涵盖

- 许可证申请的具体格式文本
- 许可证审批的行政诉讼
- 许可证被吊销后的司法救济
- 特定行业（如金融/医药）的现场检查
