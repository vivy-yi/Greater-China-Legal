## 一、核心判断框架

### 1.1 监管政策分级

| 级别 | 制定主体 | 效力 | 典型文件 |
|---|---|---|---|
| 法律 | 全国人大 | 最高效力 | 《证券法》《商业银行法》 |
| 行政法规 | 国务院 | 较高效力 | 《证券公司监督管理条例》 |
| 部门规章 | 部委/监管机构 | 规范性文件 | 《上市公司信息披露办法》 |
| 规范性文件 | 监管机构内设司局 | 具体执行 | 窗口指导意见、问答 |
| 自律规则 | 交易所/行业协会 | 行业自律 | 交易所上市规则、行业公约 |

### 1.2 政策变动类型及影响

| 变动类型 | 对存量业务影响 | 典型场景 |
|---|---|---|
| 新增许可要求 | 可能要求补办资质 | 某业务被纳入持牌管理 |
| 提高准入门槛 | 存量机构需达标 | 注册资本要求提高 |
| 强化信息披露 | 需调整披露系统 | 信息报送格式变化 |
| 限制业务范围 | 存量业务需压缩 | 互联网存款业务整顿 |
| 加重处罚力度 | 合规风险上升 | 违规成本显著提高 |

### 1.3 政策追踪频率建议

| 行业监管强度 | 建议追踪频率 |
|---|---|
| 强监管行业（金融、医药） | 实时/每日 |
| 中等监管行业（教育、建筑） | 每周 |
| 一般监管行业 | 每月 |

## 三、分析流程

### 3.1 政策影响评估步骤

```
第一步：政策识别 → 通过多渠道发现新规/修订
第二步：内容摘要 → 提取核心条款与关键数字
第三步：范围判定 → 评估是否涉及本企业业务范围
第四步：影响程度 → 定性（重大/一般/轻微）+定量（成本/收入）
第五步：过渡期规划 → 梳理从颁布到生效的时间窗口
第六步：合规调整 → 制定整改计划并跟踪执行
```

### 3.2 政策过渡期管理矩阵

| 过渡期时长 | 企业应对策略 |
|---|---|
| <6个月 | 立即启动合规调整，优先处理系统性改动 |
| 6-12个月 | 制定详细实施计划，分阶段推进 |
| >12个月 | 纳入年度合规计划，适时调整 |

### 3.3 重大政策变动预警指标

- 征求意见稿阶段即开始影响企业估值
- 监管机构主要领导公开表态
- 行业协会召集企业座谈
- 媒体大规模报道但尚未正式颁布

## 升级决策门

以下情形应向上级 Skill 升级：

- 政策变动涉及跨多个监管领域（→ compliance-gap-assessor 综合评估）
- 政策解读存在重大争议且影响核心业务模式
- 政策变化需要企业进行重大业务调整或转型
- 涉及 IPO、重组等资本运作中的监管政策适用问题
- 政策变动引发重大处罚风险需制定应对策略（→ enforcement-response-advisor）

## 本Skill不涵盖

- 代替企业向监管机构提交反馈意见
- 法律法规原文的逐条翻译与详细解读
- 政策变动的市场影响和投资建议
- 具体的会计处理和财务系统调整
- 政策变动相关的政府公关和媒体应对
## 本 Skill 不涵盖

- 新规的具体解读（需要律师意见）
- 过渡期合规方案的具体设计
- 征求意见稿的正式提交意见
- 监管政策的立法背景分析
