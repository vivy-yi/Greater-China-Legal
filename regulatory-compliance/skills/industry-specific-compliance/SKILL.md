## 一、核心判断框架

### 1.1 重点行业监管体系概览

| 行业 | 主要监管机构 | 核心法规 | 许可证类型 |
|---|---|---|---|
| 商业银行 | 国家金融监督管理总局 | 《商业银行法》《银行业监督管理法》 | 金融许可证 |
| 证券公司 | 证监会 | 《证券法》《证券公司监督管理条例》 | 证券经营许可证 |
| 保险公司 | 国家金融监督管理总局 | 《保险法》 | 保险经营许可证 |
| 基金管理 | 证监会 | 《基金法》《证券投资基金法》 | 基金管理人登记 |
| 信托公司 | 国家金融监督管理总局 | 《信托法》《信托公司管理办法》 | 信托业务许可证 |
| 上市公司 | 证监会/交易所 | 《证券法》《上市公司信息披露办法》 | IPO/再融资核准 |
| 医药制造 | 国家药监局 | 《药品管理法》《GMP》 | 药品生产许可证/GMP认证 |
| 医药流通 | 国家药监局 | 《药品管理法》《GSP》 | 药品经营许可证/GSP认证 |
| 医疗器械 | 国家药监局 | 《医疗器械监督管理条例》 | 医疗器械注册证 |
| 民办教育 | 教育部/地方教育厅 | 《民办教育促进法》 | 办学许可证 |
| 房地产开发 | 住建部/地方住建部门 | 《城市房地产管理法》 | 房地产开发资质 |
| 烟草专卖 | 国家烟草专卖局 | 《烟草专卖法》 | 烟草专卖许可证 |

### 1.2 行业合规重点领域

**金融行业**
- 资本充足性（净资本/资本充足率）
- 公司治理（股东资格、高管资质）
- 内部控制（风险隔离、合规管理）
- 反洗钱（客户身份识别、大额可疑交易）
- 信息披露（定期报告、重大事项）

**医药行业**
- 产品质量（GMP/GSP合规）
- 研发合规（临床试验、注册申报）
- 销售合规（两票制、商业贿赂）
- 广告合规（适应症宣传、价格广告）
- 数据合规（患者数据保护）

**教育行业**
- 办学资质（许可证、教师资质）
- 教学内容（课程标准、教材选用）
- 收费合规（收费备案、退费机制）
- 安全管理（校园安全、师资审查）
- 培训合规（培训资质、超前教学）

### 1.3 行业监管趋势

| 趋势 | 说明 |
|---|---|
| 准入趋严 | 行业准入门槛提高，新进入者减少 |
| 检查常态化 | 监管检查成为常态而非运动式 |
| 处罚力度加大 | 违规成本显著上升 |
| 科技监管 | 监管科技应用，大数据监控 |
| 跨境监管 | 境外上市、跨境业务监管加强 |

## 三、分析流程

### 3.1 行业合规分析步骤

```
第一步：行业定位 → 确定企业所属细分行业及监管分类
第二步：法规梳理 → 收集该行业相关的全部监管法规
第三步：企业映射 → 将监管要求与企业业务逐项对应
第四步：风险识别 → 识别该行业的特有合规风险点
第五步：案例分析 → 研究同行业处罚案例作为镜鉴
第六步：建议提出 → 结合企业实际提出改进建议
```

| 大股东资质 | 股权管理办法 | □ |
| 高管任职资格 | 董事监事任职资格管理办法 | □ |

| 反洗钱制度 | 《反洗钱法》 | □ |
| 信息安全 | 网络安全法、个人信息保护法 | □ |

## 本Skill不涵盖

- 代替企业执行具体的合规整改措施
- 提供各行业从业人员的执业资格考试指导
- 具体的场所设施合规验收（消防、环保等）
- 行业相关的税务合规和财务处理
- 产品的具体技术标准合规（技术标准问题）
## 本 Skill 不涵盖

- 行业特定许可证的申请流程
- 行业标准的专业判断
- 跨境行业监管的差异分析
- 行业仲裁与争议解决
