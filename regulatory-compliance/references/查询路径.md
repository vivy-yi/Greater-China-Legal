# 行业监管合规查询路径 — 中国大陆

> 本文档提供监管合规场景实际检索入口和操作流程，按数据类型分类整理。
> regulatory-compliance 关注**跨行业监管**，对法规时效性要求极高。
> 数据源标注规则见 `references/数据源清单.md`。

## 一、监管法规检索

### 1.1 中央法规

```
查询目标：合规相关法律法规

步骤 1：[GOV] 国家法律法规数据库
  URL: https://flk.npc.gov.cn
  检索字段：标题（如"中华人民共和国商业银行法"）
  输出：现行有效版本 + 历次修正记录

步骤 2：[BD] 版本时效验证
  工具：beidalu MCP（如果已连接）
  验证项：是否有最新修订/部分失效条款

步骤 3：[YD] 交叉验证
  检索同一法条，确认与 [GOV]/[BD] 一致
  不一致 → 标注 ⚠️ 来源冲突 + 合规顾问复核
```

### 1.2 行业法规（按行业）

```
查询目标：金融/医药/电信/环境/教育等行业法规

URL 模板：
  {行业}监管部门官网 → 政策法规
  例：国家金融监督管理总局 → 政策法规
  例：国家药品监督管理局 → 法律法规
  例：生态环境部 → 政策法规

[Y D] 检索："{行业} + 法规 + {关键词}"
```

### 1.3 监管动态

```
查询目标：监管政策最新动向

URL 模板：
  {监管机构}官网 → 新闻动态/公告通知
  例：中国人民银行 → 公告通知
  例：证监会 → 公告

[web] 时效核验（每日更新）
  技巧：设 RSS 订阅 + 邮件通知

[Y D] 检索："{监管机构} + {年份} + 公告"
```

## 二、监管处罚案例检索

### 2.1 各监管机构处罚公示

```
查询目标：行政处罚案例

URL 模板：
  {监管机构}官网 → 行政处罚 → 处罚信息公示

各机构处罚公示入口：
- 中国人民银行：https://www.pbc.gov.cn → 行政处罚
- 国家金融监督管理总局：https://www.nfra.gov.cn → 行政处罚
- 证监会：https://www.csrc.gov.cn → 行政处罚
- 市场监管总局：https://www.samr.gov.cn → 行政处罚
- 国家药监局：https://www.nmpa.gov.cn → 飞行检查
- 生态环境部：https://www.mee.gov.cn → 行政处罚
- 应急管理部：https://www.mem.gov.cn → 安全生产违法行为
- 工信部：https://www.miit.gov.cn → 行政处罚
```

### 2.2 处罚案例检索技巧

```
[Y D] 检索："{行业} + 行政处罚 + {违法情形}"
[WKL] 综合检索

关键检索项：
- 处罚类型（罚款/吊销/警告）
- 违法情形（具体违反的法律条款）
- 处罚金额
- 处罚机关
- 处罚时间
- 当事人
```

### 2.3 处罚案例分析维度

```
Step 1：识别处罚类型
  - 警告 vs 罚款 vs 责令停业 vs 吊销
  - 个人处罚 vs 单位处罚
  - 财产罚 vs 行为罚 vs 资格罚

Step 2：分析违法情形
  - 直接违法（违反某条具体法规）
  - 间接违法（违反原则性规定）
  - 失职违法（未尽管理义务）
  - 共谋违法（与第三方串通）

Step 3：评估影响
  - 业务影响（是否影响业务资格）
  - 声誉影响（公开处罚的负面声誉）
  - 后续法律风险（是否触发刑事）

Step 4：整改方向
  - 制度层面（修订内控制度）
  - 执行层面（加强合规执行）
  - 系统层面（升级合规管理系统）
  - 人员层面（培训 + 考核）
```

## 四、关键合规时效

### 4.1 行政检查应对时效

| 事项 | 时效 | 法条 |
|---|---|---|
| 行政检查配合 | 即时 | 行政机关要求 |
| 材料提供 | 检查机关指定期限 | 行政机关决定 |
| 陈述申辩 | 收到通知 5 日内 | 行政处罚法第 64 条 |
| 听证申请 | 收到处罚事先告知 5 日内 | 行政处罚法第 64 条 |
| 处罚决定履行 | 决定书指定期限 | 行政处罚法第 72 条 |

### 4.2 行政复议/诉讼时效

| 事项 | 时效 | 法条 |
|---|---|---|
| 复议申请 | 知道作出具体行政行为 60 日内 | 行政复议法第 60 条 |
| 复议机关决定 | 60 日内（可延长 30 日） | 行政复议法第 63 条 |
| 行政起诉 | 知道作出 6 个月内 | 行政诉讼法第 46 条 |
| 复议后起诉 | 收到复议决定书 15 日内 | 行政诉讼法第 46 条 |

### 4.3 行政许可时效

| 事项 | 时效 | 法条 |
|---|---|---|
| 申请办理 | 即时 | 行政许可法 |
| 受理决定 | 5 日内 | 行政许可法第 32 条 |
| 审查决定 | 20 日内（可延长 10 日） | 行政许可法第 38 条 |
| 听证 | 5 日前通知 | 行政许可法第 48 条 |
| 变更/延续 | 届满 30 日前 | 行政许可法第 50 条 |

## 五、合规差距评估方法

### 5.1 评估流程

```
Step 1：合规范围确认
  - 业务范围
  - 监管层级（国家/地方）
  - 适用法规清单
  - 关键合规要求

Step 2：现状收集
  - 内部制度文件
  - 历史合规检查报告
  - 历史处罚记录
  - 业务合同（合规条款）

Step 3：差距比对
  - 法规要求 vs 现状
  - 行业最佳实践 vs 现状
  - 同行做法 vs 现状

Step 4：差距分级
  - 🔴 重大差距（违反法律强制性规定）
  - 🟡 中等差距（违反管理性规定/制度不完善）
  - 🟢 轻微差距（流程瑕疵）

Step 5：整改规划
  - 整改优先级
  - 整改时间表
  - 责任部门/人
  - 整改预算
```

### 5.2 合规检查清单模板

```
□ 公司治理
  □ 公司章程符合最新法规
  □ 股东会/董事会运作合规
  □ 合规负责人已配备

□ 业务合规
  □ 营业执照经营范围 vs 实际业务
  □ 业务许可证有效性
  □ 业务合同条款合规性
  □ 业务模式符合监管要求

□ 财务合规
  □ 税务申报合规
  □ 财务报告合规
  □ 资金使用合规
  □ 反洗钱合规

□ 数据合规
  □ 个人信息保护合规
  □ 数据安全合规
  □ 数据跨境合规（如适用）
  □ 网络安全等级保护（如适用）

□ 行业特定
  □ {行业}特殊许可证
  □ {行业}专项制度
  □ {行业}应急机制
  □ {行业}自查制度

□ 应急准备
  □ 应急预案
  □ 应急演练
  □ 应急物资
  □ 应急联系人
```

## 九、数据源故障回退

| 故障 | 回退方案 |
|---|---|
| [YD] yuandian-mcp 未连接 | 标注 `not connected` + [GOV] 监管机构官网替代 + **合规顾问人工核实** |
| [WKL] weiken-mcp 未连接 | [YD] 替代 + 合规顾问复核 |
| [BD] beidalu API 未连接 | [GOV] 国家法律法规数据库替代 |
| [GOV] 政府网站不可访问 | [web] 替代 + 时效核验 |
| 全部数据库不可用 | **不允许**生成结论 → 合规顾问人工介入 |

**监管合规场景特殊要求**：
- 数据库故障时**不可直接输出结论**
- 必须由合规顾问人工核实
- 监管动态必须 [web] 时效核验
