# 行业监管合规 — 数据源清单

## 一、内部合规文件

| 文件类型 | 用途 |
|---|---|
| 营业执照 | 经营范围核查 |
| 许可证/资质证书 | 证照完备性核查 |
| 公司章程 | 治理结构核查 |
| 内部控制制度 | 合规管理有效性 |
| 合规报告/审计报告 | 年度合规评估 |

## 二、公开数据源

| 数据源 | 核查内容 |
|---|---|
| 国家企业信用信息公示系统 | 工商登记/动产抵押 |
| 监管机构行政处罚公示 | 是否有处罚记录 |
| 中国裁判文书网 | 涉诉情况 |
| 行业协会黑名单 | 行业自律惩戒 |
| 征信机构 | 企业信用报告 |

## 三、行业数据

| 数据源 | 用途 |
|---|---|
| 行业协会统计 | 行业规模/竞争格局 |
| 监管统计口径 | 合规指标行业对标 |
