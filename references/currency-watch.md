## 检查执行流程

```
1. Skill 涉及 currency-watch 清单中的主题
       ↓
2. 检查 references/verification-log.md 中是否有未过期的验证记录
       ↓
3. 如无记录或已过期：
   → 执行 web search 或调用 [GOV]/[YD]
   → 验证结果写入 verification-log.md（标注有效期）
       ↓
4. 输出时标注来源和验证日期
   → 如已过期，标注 ⚠️ 提示
```

---

## 清单维护

- **新增**：当发现某个主题频繁需要验证最新动态时，加入清单
- **移除**：当某个主题已稳定（如法律已修订完成进入稳定期），可移出清单
- **更新频率**：每季度审查一次清单是否需要调整
