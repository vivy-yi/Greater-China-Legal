# 法律数据源注册表
*Greater China Legal — Data Source Governance*

---

## 核心理念

**数据源治理原则：**
- Skill 输出用 `[legal_db]` 标注来源，而非绑定单一 MCP
- 实际数据源通过注册表动态路由，而非硬编码
- 关键结论须多源交叉验证
- 数据源质量持续评分，不合格则降级

## 各数据源详情

### YD — 元典 (Yuandian)

**用途：** 劳动法领域首选数据源
**覆盖：**
- 《劳动合同法》及其司法解释全文
- 各省市劳动仲裁委裁判口径
- 典型劳动争议案例（最高法/省高院指导案例）
- 地方性法规（上海/北京/广东等）
- 社保/公积金最新缴纳基数（部分城市）

**数据格式：**
```
[YD] 检索结果
来源：《中华人民共和国劳动合同法》第47条
内容：经济补偿金的计算方式...
时效：现行有效（2024年最新修订）
```

**调用方式：** 通过 `yuandian-mcp` 工具
**质量评分维度：**
- 法规准确性：★★★★☆
- 案例覆盖：★★★★☆
- 时效性：★★★★☆
- 地方差异：★★★★★（覆盖最全）

---

**调用方式：** 通过 `weiken-mcp` 工具
**质量评分维度：**
- 法规准确性：★★★★☆
- 案例覆盖：★★★★☆
- 时效性：★★★★☆
- 劳动法专项：★★★☆☆（不如YD专注）

- 案例覆盖：★★☆☆☆（偏法规少案例）
- 时效性：★★★★☆
- 地方差异：★★☆☆☆（偏中央少地方）

**覆盖：**

- 案例覆盖：★☆☆☆☆（无案例）
- 时效性：★★★★★（官方即时更新）
- 地方差异：★★★★★（各省市独立更新）

## 质量评分标准

### 评分维度（1-5星）

| 维度 | 说明 | 权重 |
|---|---|---|
| 准确性 | 法规条款是否准确，有无错误 | 30% |
| 时效性 | 是否及时更新现行有效版本 | 25% |
| 覆盖度 | 是否覆盖全国+地方差异 | 20% |
| 可用性 | 检索结果的相关性和可读性 | 15% |
| 权威性 | 来源是否官方/半官方 | 10% |

### 质量监控触发条件

**自动降级：**
- 连续3次检索无结果 → 降1星
- 发现1处明显错误 → 降1星
- 时效性滞后超过6个月 → 降1星

**降级后的处理：**
- 评分低于3星 → 从注册表移除
- Skill输出时标注降级原因

## 多源交叉验证规则

### 必须交叉验证的情形

1. **关键结论**：补偿金计算/违法解除认定/竞业限制有效性
2. **新型问题**：平台经济用工/灵活用工等新兴领域
3. **地方差异**：涉及非北上广深地区

### 交叉验证方法

```
关键结论 = 2个以上数据源一致确认
         ↓
         如不一致，输出：
         "⚠️ 来源冲突：[YD]说是A，[WKL]说是B。请律师复核。"
```
