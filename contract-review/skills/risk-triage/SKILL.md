# /risk-triage — 合同风险等级初筛

## 三维评估矩阵

### 第一维：合同类型风险

| 类型 | 基础风险 | 备注 |
|---|---|---|
| 股权转让/增资 | 🔴 极高 | 涉及公司控制权变更 |
| 借款/担保/保理 | 🔴 极高 | 涉及金钱给付义务 |
| 建设工程/房地产 | 🔴 高 | 金额大、周期长 |
| 软件开发/技术服务 | 🟡 中 | IP归属风险 |
| 采购/销售（标准品） | 🟢 低 | 标准条款、争议少 |
| 租赁（短期） | 🟢 低 | 期限短、退出成本低 |

### 第二维：合同金额风险

| 金额区间 | 风险权重 |
|---|---|
| <10万 | 低权重 |
| 10-100万 | 中权重 |
| 100-1000万 | 高权重 |
| >1000万 | 极高权重 |

### 第三维：合作方信用

| 信用状况 | 风险调整 |
|---|---|
| 上市公司/国企 | 降低风险0.5级 |
| 知名民企 | 维持原级 |
| 中小民营企业 | 提升风险0.5-1级 |
| 初次合作/不了解 | 提升风险1级 |
| 有诉讼/执行记录 | 提升风险1-2级 |

## 风险等级判断

### 综合风险等级

```
综合风险 = f(合同类型风险, 金额风险, 合作方信用风险)
```

| 综合评分 | 风险等级 | 审查深度 |
|---|---|---|
| 0-3 | 🟢 低风险 | 标准模板审查，1-2天 |
| 4-6 | 🟡 中风险 | 重点条款审查，3-5天 |
| 7-9 | 🔴 高风险 | 深度审查+法务介入，5-10天 |
| 10+ | ⚫ 极高风险 | 原则上不签，或上级审批 |

---

## 快速判断标准

### 直接判定高风险的情形

- 对方有失信/限高/被执行记录（企查查/中国执行信息公开网查询）
- 对方涉及重大诉讼（裁判文书网查询）
- 合同涉及担保/连带责任
- 合同涉及知识产权转让/排他许可
- 合同涉及公司控制权变更
- 合同金额>公司注册资本50%
- 合同期限>3年无明确退出机制

### 直接判定低风险的情形

- 标准采购/销售（金额<10万，有长期合作关系）
- 标准租赁（金额<5万，标准条款）
- 格式合同（对方提供，我方仅接受与否）
