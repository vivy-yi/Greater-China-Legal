# /negotiation-redlines — 合同修改建议生成

## 修改原则

### 高风险条款修改策略

| 风险类型 | 修改策略 | 优先级 |
|---|---|---|
| 违约金过高 | 设定上限（不超过损失的130%） | 必改 |
| 单方解除权 | 增加对等解除权/增加补偿 | 必改 |
| 无限连带 | 限定范围/设定上限 | 必改 |
| 知识产权归属 | 争取合理归属/保留改进权 | 必改 |
| 争议解决地 | 争取我方所在地 | 优先 |
| 保密范围过宽 | 明确边界/排除公知信息 | 优先 |

---

## 常见条款修改模板

### 违约金条款

**原条款（高风险）：**
```
任何一方违约，应向守约方支付合同总金额20%的违约金。
```

**修改建议：**
```
任何一方违反本合同约定，应向守约方支付因违约造成的实际损失，但违约金金额最高不超过合同总金额的[X]%。
```

**谈判话术：**
> "20%违约金对于我们这种中小企业来说太高了，如果因为不可抗力导致部分履行，我们也承担不起。建议参照民法典的规定，以实际损失为限，最高不超过合同金额的30%。"

---

### 知识产权归属条款

**原条款（高风险）：**
```
合同履行过程中产生的所有知识产权归甲方所有。
```

**修改建议：**
```
1. 合同履行前已存在的知识产权归原所有方。
2. 合同履行过程中就甲方既有产品/服务的改进归甲方所有。
3. 合同履行过程中专门为甲方开发的新技术/新成果，知识产权归甲方所有，但乙方保留用于展示的署名权。
4. 任何一方独立开发的技术成果归各自所有。
```

---

### 单方解除权条款

**原条款（高风险）：**
```
甲方有权在提前[X]日书面通知乙方后，无理由解除本合同。
```

**修改建议：**
```
1. 甲方因乙方违约行使解除权时，须书面说明解除原因。
2. 甲方因业务调整行使解除权时，须提前[X]日书面通知乙方，并支付乙方已发生成本及合理利润补偿。
3. 乙方行使解除权时，须提前[X]日书面通知甲方，并完成已承接工作的交接。
```

---

### 争议解决条款

**原条款（高风险）：**
```
因本合同引起的争议，提交[对方所在地]仲裁委员会仲裁。
```

**修改建议：**
```
因本合同引起的争议，提交本合同签订地有管辖权的人民法院管辖。
```

**谈判话术：**
> "对方所在地仲裁委对我们来说太远了，成本太高。建议改为合同签订地法院管辖，或者我们各让一步，选择一个中间地点的仲裁委。"

---

### 保密条款

**原条款（高风险）：**
```
合同期内及合同终止后[X]年内，乙方对因合同获知的信息承担保密义务。
```

**修改建议：**
```
1. 保密信息不包括：（a）公知信息；（b）乙方独立开发的信息；（c）非因乙方过错成为公知的信息。
2. 保密义务的存续期间：合同期内及合同终止后2年。
3. 保密义务不适用于：（a）法律要求披露；（b）向有必要知悉的员工/顾问披露（须同样保密）。
```
