## 判断树

### 第一步：交易实质判断

**核心问题：转移的是什么？**

| 转移对象 | 合同类型 |
|---|---|
| 有形货物所有权 | 买卖合同 |
| 货币使用权 | 借款合同 |
| 物的使用权（收取租金） | 租赁合同 |
| 工作成果 | 承揽合同/建设工程 |
| 运输服务 | 运输合同 |
| 保管服务 | 保管合同/仓储合同 |
| 代理服务 | 委托合同/中介合同 |
| 技术成果/使用权 | 技术合同 |
| 经营资产使用权 | 融资租赁合同 |
| 应收账款 | 保理合同 |

---

### 第二步：特殊类型识别

**混合合同判断**

| 情形 | 合同类型 | 适用规则 |
|---|---|---|
| 附带安装的设备买卖 | 买卖合同（附条件） | 安装属于从给付义务 |
| 附带培训的服务合同 | 服务合同 | 培训不构成独立合同 |
| 附带知识产权许可的软件买卖 | 买卖+许可（混合） | 参照买卖+许可 |

**框架合同（长期合作）**

- 形式：框架协议 + 订单/任务书
- 性质：多个合同关系的集合
- 风险：单次订单是否构成合同

**名为投资实为借贷**

- 固定回报（还本付息）→ 借贷合同
- 不承担经营风险 → 借贷合同
- 风险共担 → 投资/合伙

**名为合作实为劳动关系**

- 接受考勤/管理 → 劳动关系
- 固定分成 → 借贷关系

---

### 第三步：合同效力初步审查

| 情形 | 效力 | 后果 |
|---|---|---|
| 主体不合格（无资质） | 效力待定/无效 | 可能无效 |
| 意思表示不真实（欺诈/胁迫） | 可撤销 | 撤销后无效 |
| 违反强制性规定 | 无效/效力待定 | 须具体分析 |
| 违反公序良俗 | 无效 | — |
| 无处分权（无权处分） | 效力待定 | 追认后有效 |
