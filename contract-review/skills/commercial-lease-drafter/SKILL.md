# 商业租赁合同起草 / 审查

## 目的

为商业物业（商铺 / 办公 / 厂房 / 产业园区）起草租赁合同，或审查已有合同草案。覆盖中国商业租赁中最常协商的条款：租金递增机制、免租装修期、优先购买权、续租权、转租条件、物业费分摊、押金条款、违约金。

本技能负责起草合同文本。**不负责**物业尽职调查（产权核查、权利负担排查、规划用途核实），这些需要另行调查 `[SME 核查]`。

## 法域假设

默认中国大陆法域，适用《民法典》。涉及"商住两用"物业可能适用消费者保护相关规则 `[SME 核查]`。出租方或承租方为外商投资企业的，可能有额外合规要求 `[SME 核查]`。

## 加载信息

起草或审查前须确认以下要素：

- **物业类型** — 商铺 / 办公 / 厂房 / 产业园区 / 长租公寓
- **当事人** — 出租方和承租方的全称（营业执照名称）
- **物业地址及面积**（平方米）
- **租赁期限**（年）
- **租金结构** — 固定年/月租金，或营业额提成模式
- **租金递增机制** — 固定比例、CPI联动，还是阶梯式
- **押金金额及退还条件**
- **允许用途**（经营用途）
- **物业费安排** — 是否含在租金内
- **免租装修期**（如有）
- **优先购买权和续租权** — 是否需要约定
- **转租条件** — 是否允许，需否出租方同意
- **争议解决** — 法院管辖或仲裁

## 工作流程

### 第一步：核实基本条款完备性

确认以下必要条款已涵盖或待确定：

| 必要条款 | 状态 | 备注 |
|---------|------|------|
| 物业地址 + 面积 | 待确认 | 须与产权文件一致 `[SME 核查]` |
| 允许用途 | 待确认 | 须符合规划用途 `[SME 核查]` |
| 租赁期限 | 待确认 | 商业用途最长20年 `[SME 核查]` |
| 租金金额及支付周期 | 待确认 | 月付 / 季付 / 年付 |
| 押金金额及退还条件 | 待确认 | 通常为2-3个月租金 |
| 免租装修期（如有） | 待确认 | 通常15-30天 |
| 物业费 | 待确认 | 含在租金还是另行收取 |
| 争议解决 | 待确认 | 法院管辖或仲裁 |

### 第二步：审查已有合同草案

**审查时**逐条对照以下清单：

#### 🔴 高风险条款（即时标注）

| 条款 | 风险 | 建议 |
|------|------|------|
| 租赁期限超过20年 | 超法定最长期限，部分无效 | 最高20年，续租另签协议 `[SME 核查]` |
| 优先购买权未定义 | 承租方失去优先购买机会 | 明确触发条件、行权期限、价格机制 `[SME 核查]` |
| 转租禁止且无例外 | 承租方无法处置资产 | 改为"经出租方书面同意后可转租" `[SME 核查]` |
| 租金递增机制未约定 | 续租期租金争议隐患 | 明确递增方式（比例/CPI/固定金额）`[SME 核查]` |
| 押金退还条件不清 | 押金退还争议 | 列明可扣除情形，结清后X日内退还 |
| 无不可抗力条款 | 疫情等情形无处理依据 | 参照《民法典》第180条补充 `[SME 核查]` |

#### 🟡 中风险条款

| 条款 | 风险 | 建议 |
|------|------|------|
| 物业费无上限 | 运营成本不确定 | 添加年度上限或涨幅限制 |
| 续租条件未约定 | 续租时争议隐患 | 明确提前通知期（通常租期届满前3-6个月）|
| 逾期付款利率未约定 | 拖欠租金无明确罚则 | 约定按法定利率或约定利率计算 `[SME 核查]` |
| 提前解约违约金未约定 | 任一方提前解约无据 | 明确通知期和违约金计算方式 |
| 维修责任约定模糊 | 维修费用争议 | 明确出租方负责主体结构，承租方负责室内装修 |

#### 🟢 建议补充的标准条款

- **交付标准** — 毛坯 / 简装 / 精装交付状态
- **用途变更** — 变更经营用途须经出租方同意
- **合规义务** — 承租方负责办理经营所需的营业执照和许可
- **转让条件** — 租赁权转让的条件和程序
- **损毁灭失** — 物业损毁时租金减免或合同解除 `[SME 核查]`
- **通知方式** — 正式通知的送达地址和方式
- **完整协议** — 本合同取代此前所有协议

### 第三步：从头起草合同结构

新合同建议采用以下章节顺序：

```
1. 合同双方
2. 租赁物业（地址、面积、用途许可）
3. 租赁用途（经营用途限制）
4. 租赁期限（起止日期）
5. 租金及支付方式（金额、周期、递增机制）
6. 押金（金额、退还条件、扣除情形）
7. 物业费（金额、包含项目）
8. 免租装修期（期限、条件）
9. 交付标准（交付时物业状态）
10. 维修责任（出租方vs承租方责任划分）
11. 优先购买权（触发条件、行权期限）`[SME 核查]`
12. 续租权（通知期、续租条件）`[SME 核查]`
13. 转租（须经出租方同意）`[SME 核查]`
14. 提前解约（通知期、违约金）
15. 违约金（逾期利息、其他罚则）`[SME 核查]`
16. 不可抗力 `[SME 核查]`
17. 争议解决（法院或仲裁）
18. 其他（通知方式、完整协议、合同变更）
```

### 第四步：升级决策门

**合同签署前：** 确认操作人身份。若为非律师用户：

> "商业租赁合同一旦签署即产生法律约束力，涉及较大财务风险。签署前建议由律师审查以下事项：（1）出租方的产权证明和出租授权；（2）物业规划用途和经营许可合规性；（3）租金递增和提前解约条款；（4）优先购买权和续租权条款。需要我整理一份关键风险摘要供您带去咨询律师吗？"

未经 `[SME 核查]` 的结论，不应声称合同合法有效。

## 本技能不涵盖

- **物业尽职调查** — 产权核查、抵押查封、规划核实 `[SME 核查]`
- **营业执照办理** — 承租方自行负责经营许可
- **税务建议** — 印花税、增值税、个人所得税等 `[SME 核查]`
- **产权受限物业** — 已抵押物业的出租授权需单独核实 `[SME 核查]`
- **续租协议起草** — 本技能起草合同框架，续租时另签新协议
