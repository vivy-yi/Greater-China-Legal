# 商业合同审查数据源清单 — 中国大陆

> 本清单整理合同审查场景常用数据源，按 [YD]/[WKL]/[GOV]/[BD]/[model] 五级标注分类。

## 三、案例库（合同审查场景特化）

### 3.1 最高人民法院指导案例 / 公报案例

```
URL: 人民法院案例库 https://rmfyalk.court.gov.cn
[Y D] 检索："指导案例 + 合同类型"

合同审查必学案例：
  - 合同效力（阴阳合同/虚假意思表示）
  - 违约金过高调整（30% 标准）
  - 格式条款效力（民法典第 496-498 条）
  - 情势变更（民法典第 533 条）
  - 解除条件成就（根本违约）
  - 表见代理/代表（公司签约权限）
```

### 3.2 各省高院典型案例

```
[Y D] 检索："{省市} + 合同纠纷 + {争议焦点}"

常见争议焦点：
  - 违约金调整比例（各省市略有差异）
  - 表见代理认定
  - 情势变更适用
  - 格式条款效力
  - 不可抗力认定
```

## 八、来源标注规范（合同审查特化）

每个结论/数字必须挂载来源标签：

| 标注 | 含义 | 合同审查示例 |
|---|---|---|
| `[YD]` | 元典检索结果 | `[YD] 民法典第 585 条` |
| `[WKL]` | 无合力检索结果 | `[WKL] 类案 XX 号` |
| `[BD]` | 北达/法天使/合同范本 | `[BD] 法天使/SaaS MSA 范本` |
| `[GOV]` | 政府平台数据 | `[GOV] 民法典合同编通则解释（2023.05.25 施行）` |
| `[web]` | 网络搜索结果 | `[web] 商务部 跨境电商 2024 新规` |
| `[model]` | 模型知识（须核实） | `[model] 一般行业惯例...` |
| `[user]` | 用户提供的文件 | `[user] 客户提供的合同草稿` |
| `[settled — last confirmed YYYY-MM-DD]` | 已验证的稳定引用 | 已确认的法条引用 |

**合同审查特殊规则**：
- **条款引用**：`[YD] 民法典第 X 条 + 司法解释`
- **范本引用**：`[BD] 法天使/{范本名}`
- **案例引用**：`[YD]（2023）最高法民申 XX 号 + 要旨`
- **行业惯例**：`[model] 行业惯例 + 建议核实`

## 十、合同审查风险等级说明

contract-review 风险等级主要由 skill 内部设定，按合同类型与金额：

| 合同类型 | 风险等级 | 原因 |
|---|---|---|
| review（通用） | medium | 标准审查 |
| nda-review | medium | 风险条款识别 |
| saas-msa-review | high | 数据 + 知识产权 |
| commercial-lease-drafter | medium | 财产租赁 |
| land-lease-agreement | high | 涉及不动产 |
| loan-agreement-checker | high | 涉及金融 |
| international-sales-advisor | high | 涉外 + 跨境 |
| mutual-fund-bylaws | high | 基金合规 |
| vendor-agreement-review | medium | 采购标准 |
| labor-contract-audit | high | 涉及劳动者权益 |

**触发升级**：
- 标的 ≥ 100 万 → 律师专项
- 涉外 → 律师主导
- 重大资产/股权 → 律师专项
