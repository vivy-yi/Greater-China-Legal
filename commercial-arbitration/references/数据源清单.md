# 商事仲裁 — 数据源清单

## 一、内部数据

| 数据类型 | 内容 | 核查要点 |
|---|---|---|
| 仲裁协议原文 | 合同中的仲裁条款 | 机构名称/范围/语言/地点 |
| 主合同 | 产生争议的合同 | 争议焦点/履约情况 |
| 往来函件 | 争议产生过程 | 权利主张/抗辩依据 |
| 证据材料 | 支持各自主张的文件 | 真实性/关联性/合法性 |

## 二、公开数据源

| 数据源 | 说明 |
|---|---|
| 贸仲官网 | 仲裁规则、仲裁员名册、费用标准 |
| SCIA官网 | 仲裁规则、费用计算器 |
| HKIAC官网 | 仲裁规则、费用表 |
| SIAC官网 | 仲裁规则、费用计算 |
| ICC官网 | 国际仲裁规则 |
| 最高人民法院官网 | 仲裁司法审查批复/案例 |
| 北大法宝 | 仲裁案例库 |

## 三、费用估算参数

| 机构 | 立案费 | 仲裁员费（参考） | 管理费 |
|---|---|---|---|
| 贸仲 | 按争议金额0.1% | 按争议金额与费率 | 按争议金额0.5-1% |
| SCIA | 按争议金额0.05% | 国际标准 | 按争议金额0.5-1% |
| ICC | 预付管理费 | 高（美元计算） | 按争议金额阶梯 |
