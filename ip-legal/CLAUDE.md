# ip-legal — China Mainland

## Legal Framework

| 类型 | 法规依据 | 主管机关 |
|---|---|---|
| 商标 | 《商标法》2024修订、《商标法实施条例》 | 国家知识产权局（CNIPA） |
| 专利 | 《专利法》2020修订、《专利法实施细则》2023 | 国家知识产权局（CNIPA） |
| 著作权 | 《著作权法》2020修订、《计算机软件保护条例》 | 国家版权局 |
| 商业秘密 | 《反不正当竞争法》第9条、《民法典》1232-1235条 | 市场监管总局（SAMR） |
| 域名 | 《网络安全法》《电子商务法》 | 工业和信息化部（MIIT） |

## CN-Specific Risk Triggers

### 商标风险
- 抢注风险（恶意抢注他人已有一定影响商标）
- 驳回风险（与他人在先权利冲突）
- 侵权风险（销售产品包装/标识侵权）

### 专利风险
- 发明专利实质审查时间较长（2-3年）
- 实用新型/外观设计无实质审查，稳定性较低
- 职务发明奖励纠纷

### 著作权风险
- 软件/字体/图片盗版使用
- UGC内容平台侵权责任（通知-删除规则）
- 数据库保护争议

### 商业秘密风险
- 员工离职带走商业秘密
- 竞业限制协议执行
- 技术合同中的保密条款

## CN Dispute Resolution

| 类型 | 路径 |
|---|---|
| 商标侵权 | 行政执法（市场监督管理局）→ 民事诉讼 |
| 专利侵权 | 知识产权法院 → 民事诉讼 |
| 著作权侵权 | 著作权侵权纠纷（法院）或行政执法 |
| 商业秘密侵权 | 民事诉讼（民法典1232条）+ 刑事报案（刑法219条） |
| 域名纠纷 | 仲裁（CNNIC争议解决）或诉讼 |

## Portfolio Management

| 类型 | 保护周期 | 续展/维持 |
|---|---|---|
| 商标 | 10年 | 到期日前12个月续展 |
| 发明专利 | 20年 | 每年缴纳年费 |
| 实用新型 | 10年 | 每年缴纳年费 |
| 外观设计 | 15年 | 每年缴纳年费 |
| 著作权 | 作者终身 + 死后50年 | 无需续展 |
