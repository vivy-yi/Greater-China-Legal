# 判断框架 — 跨境贸易合规决策树

## 概述
本框架用于系统化评估跨境交易涉及的合规领域，确定应使用的Skill并识别需要升级的复杂情形。

---

## 一、交易前预审判断流程

### Step 1: 交易性质识别

```
这是一笔跨境贸易交易吗？
    │
    ├─ 是 → 进入 Step 2
    └─ 否 → 本框架不适用（国内贸易）
```

### Step 2: 流向判断

```
交易方向是什么？
    │
    ├─ 出口（美国/中国 → 境外）→ export-control-reviewer
    ├─ 进口（境外 → 美国/中国）→ import-tariff-adviser + customs-compliance-assessor
    └─ 过境/转运 → customs-compliance-assessor
```

### Step 3: 交易相对方筛查

```
交易相对方是否在制裁名单上？
    │
    ├─ 是（SDN/Entity List/Denied Persons）→ trade-sanctions-checker（立即停止交易）
    ├─ 不确定 → trade-sanctions-checker（需实时数据库核查）
    └─ 否 → 进入 Step 4
```

### Step 4: 物项管制判断

```
交易的商品/技术/软件是否受出口管制？
    │
    ├─ USML（ITAR）→ export-control-reviewer + trade-sanctions-checker
    ├─ CCL（EAR）→ export-control-reviewer
    └─ 不受控（EAR99）→ 记录保存，继续其他合规检查
```

### Step 5: 关税及贸易救济判断

```
进口商品是否涉及以下情形？
    │
    ├─ Section 301对华关税 → import-tariff-adviser
    ├─ AD/CVD税令 → trade-dispute-advisor
    ├─ FTA优惠税率申请 → customs-compliance-assessor
    └─ 无上述情形 → 标准HTS分类程序
```

## 三、升级决策门快速索引

### 必须立即升级的情形（红色警戒）

1. **涉及禁运国家**：Cuba, Iran, North Korea, Syria, Crimea → trade-sanctions-checker → 持牌律师
2. **SDN List命中** → 立即停止交易 → trade-sanctions-checker → 持牌律师
3. **ITAR受控物项出口** → export-control-reviewer → DDTC咨询
4. **刑事风险（故意违规）** → 相关Skill → 刑事辩护律师

### 建议升级的情形（黄色预警）

| 情形 | 阈值 | 升级方向 |
|------|------|----------|
| HTS分类争议税额 | >$100,000 | Licensed Customs Broker |
| 潜在AD/CVD税令 | 已有调查公告 | 贸易救济应诉事务所 |
| OFAC许可证申请 | 涉及禁运交易 | OFAC专业法律顾问 |
| 海关审计 | CBP发出Information Request | Licensed Customs Broker |
| 多国管辖冲突 | 美+欧+中同时适用 | 国际贸易法律师 |
| 高价值Incoterms争议 | >$500,000 | ICC仲裁员 |

## 四、交叉覆盖场景处理

### 场景：向俄罗斯出口含有美国技术的设备

**问题分解：**
1. 出口管制（EAR）— 是否需要BIS许可？
2. OFAC制裁（SSI）— 是否在禁止范围内？
3. 外国直达产品规则（FDPR）— 美国技术是否限制第三方使用？

**适用Skill顺序：**
1. export-control-reviewer（确定ECCN和许可要求）
2. trade-sanctions-checker（确定OFAC SSI是否适用）
3. 若两者均需许可 → 升级至专业法律顾问

### 场景：中国商品通过台湾加工后进口美国

**问题分解：**
1. HTS分类（原产地改变？）
2. Section 301税率（是否仍为中国原产？）
3. FTA优惠（台湾是否与美国签FTA？）

**适用Skill顺序：**
1. import-tariff-adviser（确定HTS和税率）
2. customs-compliance-assessor（原产地规则）
3. 若涉及FTA → customs-compliance-assessor

## 五、合规优先级排序

当一笔交易同时触发多个合规领域时，按以下优先级处理：

```
优先级 1: 制裁筛查（最优先，刑事风险最高）
    ↓
优先级 2: 出口管制许可（如需许可，未经批准不得交易）
    ↓
优先级 3: 进口关税（含Section 301，正确分类）
    ↓
优先级 4: Incoterms条款（合同层面的风险转移约定）
    ↓
优先级 5: 海关合规（FTA、保税仓库等）
    ↓
优先级 6: 贸易救济（AD/CVD等事后争议）
```
