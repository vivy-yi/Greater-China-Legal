| 领域 | 核心法规 | 主管机关 |

| 数据/互联网 | 《网络安全法》《数据安全法》《个人信息保护法》 | 国家网信办（CAC）|
| 电子商务 | 《电子商务法》 | 市场监管总局（SAMR）|

## CN Regulatory Monitoring Priority Areas

### 高优先级（须持续监控）
- **网信办执法动态**：数据安全/个人信息处罚案例
- **市场监管总局**：反垄断执法、消费者保护执法
- **央行/金融监管**：金融数据合规（征信、支付）
- **工信部**：App个人信息保护专项整治

### 中优先级
- **行业监管**（卫健委、药监局、教育部等）：行业特定合规
- **地方立法**：北上广深先行先试的地方性法规

## CN Compliance Gap Surfacing

监管新规发布后的CN合规落地路径：

```
1. 法规解读 → 2. 差距分析 → 3. 整改方案制定 → 4. 内部政策修订 → 5. 培训与执行 → 6. 持续监控
```

**CN特殊合规节点：**
- 网信办安全评估（数据出境、关键信息基础设施）
- 等保测评（网络安全等级保护）
- App违法违规收集使用个人信息检测
