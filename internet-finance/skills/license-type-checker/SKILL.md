# /license-type-checker — 牌照类型判断

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取牌照类型/监管机构。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取牌照类型/监管机构
2. 分析业务类型
3. 判断须哪类牌照
4. 输出牌照清单和建议

## 牌照判断树

```
第一步：分析业务本质
  → 放贷 → 网络小额贷款 or 消费金融公司
  → 发放贷款并撮合 → 联合贷款
  → 吸收公众存款 → 银行/存款类金融机构
  → 代销理财产品 → 基金销售牌照
  → 保险产品销售 → 保险经纪/代理牌照
  → 聚合支付 → 支付牌照
  → 融资担保 → 融资担保牌照

第二步：判断牌照要求
  → 全国性业务 → 全国性牌照（门槛高）
  → 区域性业务 → 区域性牌照（门槛较低）
```

## 牌照类型一览

| 业务 | 牌照 | 监管机构 | 注册资本要求 |
|---|---|---|---|
| 网络小额贷款 | 省级金融监管局批 | 金融监管总局 | 全国性≥50亿，区域性≥3亿 |
| 消费金融公司 | 金融监管总局批 | 金融监管总局 | ≥3亿 |
| 第三方支付 | 央行批 | 央行 | ≥3亿 |
| 基金销售 | 证监会批 | 证监会 | ≥2000万 |
| 保险代理/经纪 | 金融监管总局批 | 金融监管总局 | ≥500万 |
| 融资担保 | 省级金融监管局批 | 金融监管总局 | ≥1亿 |
