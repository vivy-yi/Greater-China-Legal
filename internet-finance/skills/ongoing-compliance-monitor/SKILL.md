# /ongoing-compliance-monitor — 持续合规监控

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取监控频率/预警指标。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取监控框架/预警指标
2. 建立监控机制
3. 设定监控频率
4. 输出监控清单

## 监控频率

| 监控项 | 频率 | 负责人 |
|---|---|---|
| 监管政策动态 | 实时/每周 | 合规部 |
| 投诉数据 | 每日/每周 | 客服部 |
| 利率合规 | 每笔贷款 | 风控部 |
| 数据安全 | 每季度 | 技术部 |
| 牌照有效期 | 到期前6个月 | 合规部 |

## 监管政策监控

```
重点关注渠道：
1. 金融监管总局官网：https://www.cbirc.gov.cn/
2. 央行官网：https://www.pbc.gov.cn/
3. 证监会官网：https://www.csrc.gov.cn/
4. 中国互联网金融协会：https://www.nifa.org.cn/
5. 各地金融监管局官网

关键词监控：
- "互联网贷款" / "消费金融" / "小额贷款"
- "年化利率" / "综合资金成本"
- "数据安全" / "个人信息"
- "备付金" / "断直连"
```

## 合规预警指标

| 指标 | 预警阈值 | 处理方式 |
|---|---|---|
| 贷款加权平均利率 | >年化36% | 立即整改 |
| 投诉量 | 环比增长>20% | 启动调查 |
| 备付金缺口 | 出现缺口 | 立即补充 |
| 数据泄露事件 | 发生 | 72小时内报告 |
| 高管变动 | 发生 | 10日内报告 |
