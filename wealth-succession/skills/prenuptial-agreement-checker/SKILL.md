## 一、协议类型识别与形式要件

### 1.1 夫妻财产约定类型
| 协议类型 | 签订时间 | 法律依据 | 效力等级 |
|---------|---------|---------|---------|
| 婚前财产协议 | 结婚登记前 | 《民法典》第1063条 | 高（附生效条件） |
| 婚内财产协议 | 婚姻关系存续期间 | 《民法典》第1065条 | 高 |
| 离婚协议 | 离婚过程中 | 《民法典》第1076-1077条 | 高（民政部门备案） |
| 析产协议 | 婚姻解体时 | 《民法典》第1154条 | 中 |

### 1.2 形式要件审查
**有效协议必备要素：**
1. ✓ 双方亲自签署（不得代理）
2. ✓ 真实意思表示（无欺诈、胁迫）
3. ✓ 书面形式（《民法典》第1065条明确要求）
4. ✓ 不得违反法律强制性规定
5. ✓ 不得违背公序良俗

**签署注意事项：**
- 签署时应保留视频或见证人证明签署过程
- 避免在醉酒、疾病急性期等非正常状态下签署
- 财产清单应作为附件详列
- 建议进行公证（公证不是生效要件，但增强证明力）

### 1.3 协议公示与备案
| 方式 | 效力 | 建议程度 |
|-----|------|---------|
| 公证 | 最强证明力 | 强烈建议 |
| 律师见证 | 中等证明力 | 建议 |
| 双方自行保管 | 证明力较低 | 仅限简单协议 |

## 二、协议内容合规性校验

### 2.1 财产归属约定规则
**可约定事项（《民法典》第1063条）：**
- 各自婚前财产归一方所有
- 婚姻关系存续期间所得财产归各自所有
- 婚姻关系存续期间所得财产归一方所有
- 约定财产制优先于法定财产制

**财产类型与归属认定：**
| 财产类型 | 法定归属 | 可约定空间 |
|---------|---------|-----------|
| 婚前存款 | 婚前取得方所有 | 可约定婚后收益归属 |
| 婚前房产 | 出资方所有（婚后还贷部分除外） | 可约定婚后还贷及增值归属 |
| 婚后工资奖金 | 夫妻共同所有 | 可约定各自所有 |
| 婚后经营所得 | 夫妻共同所有 | 可约定按比例所有 |
| 婚后继承/受赠 | 夫妻共同所有（遗嘱除外） | 可约定归一方所有 |
| 知识产权收益 | 夫妻共同所有 | 可约定归属 |

### 2.2 常见条款合规性判断
**条款类型A：有效条款**
| 条款示例 | 效力认定 | 理由 |
|---------|---------|-----|
| "婚前房产归甲方所有，婚后该房产出租收益归甲方所有" | 有效 | 婚前财产+孳息归属 |
| "各自名下的债务归各自承担" | 有效（内部） | 夫妻财产约定 |
| "男方名下公司股权归男方所有" | 有效 | 婚前财产确认 |

**条款类型B：效力待定条款**
| 条款示例 | 效力风险 | 风险来源 |
|---------|---------|---------|
| "一方出轨则放弃全部财产" | 可能无效 | 违反公序良俗或强制性规定 |
| "禁止与特定亲属来往" | 可能无效 | 侵犯人身自由 |
| "约定子女抚养权归属" | 无效 | 抚养权依子女最佳利益确定 |
| "一方不工作则财产归另一方" | 可能无效 | 限制人身自由 |

**条款类型C：无效条款**
| 条款示例 | 无效理由 |
|---------|---------|
| "净身出户"（放弃全部财产） | 违反公序良俗 |
| "不准离婚" | 违反婚姻自由原则 |
| "对外债务由一方承担"（恶意串通损害第三人） | 恶意串通，损害第三人利益 |

### 2.3 债务条款特别注意事项
**内部约定 vs 外部效力：**
```
核心规则：
- 夫妻财产约定对内有效，对外不能对抗善意第三人
- 债权人知道夫妻财产约定的，以约定确定清偿责任
- 债权人不知道约定的，以夫妻共同财产清偿

风险场景：
┌────────────────────────────────────────────────────┐
│ 场景：夫妻约定婚后财产归各自所有                   │
│ 风险：一方对外举债，债权人主张为夫妻共同债务       │
│ 后果：另一方需证明债权人知晓约定，否则共同清偿     │
└────────────────────────────────────────────────────┘
```

**债务隔离建议：**
1. 大额债务由一方单独签字，避免被认定为共同债务
2. 借款合同注明"个人消费/经营使用，与家庭无关"
3. 夫妻一方为有限责任公司股东时，注意公司债务与个人债务隔离
4. 定期梳理家庭债务情况，及时调整协议

## 三、协议风险矩阵与修改建议

### 3.1 主要风险类型
| 风险类型 | 触发情形 | 后果 | 建议措施 |
|---------|---------|-----|---------|
| 显失公平 | 条款极端倾斜 | 诉讼中可能被撤销 | 平衡双方利益 |
| 意思表示不真实 | 受欺诈/胁迫签署 | 协议可撤销 | 保留签署过程证据 |
| 内容违法 | 约定事项违反强制性规定 | 部分或全部无效 | 事先由律师审核 |
| 公示不足 | 未公证且无其他证据 | 难以对抗第三人 | 建议公证 |
| 条款模糊 | 财产范围、价值约定不清 | 执行争议 | 附财产清单附件 |

### 3.2 典型问题与修改方案

**问题A：财产范围不明确**
```
原条款："甲方婚前财产归甲方所有"
风险：未列明具体财产，离婚时可能就婚后还贷/增值产生争议
修改建议：
"甲方于20XX年购买的位于[地址]房产（房产证号：XXXX）归甲方所有，
该房产婚后共同还贷部分及对应增值由甲方对乙方进行补偿，
具体金额参照[计算方式]"
```

**问题B：知识产权收益归属模糊**
```
原条款："各自知识产权收益归各自所有"
风险："知识产权收益"定义不清，可能产生争议
修改建议：
"婚姻关系存续期间，一方取得的知识产权产生的经济收益，
包括但不限于许可费、转让费、版税等，归该方所有；
但该知识产权在婚姻期间完成且主要利用了另一方资源的除外"
```

**问题C：离婚时财产分割预先约定**
```
原条款："未来离婚时财产全部归甲方"
风险：该条款可能因违反公序良俗被认定无效
修改建议：
避免在婚前协议中约定离婚时的财产分割
如需约定，应在离婚协议中根据当时的实际情况确定
```

### 3.3 协议履行保障
| 保障措施 | 实施方式 | 效果 |
|---------|---------|-----|
| 财产登记 | 将财产登记在协议约定的所有人名下 | 物权公示 |
| 公证 | 双方到公证机构办理 | 增强证明力 |
| 律师见证 | 律师审核并见证签署 | 中等证明力 |
| 财产清单 | 附件详细列明财产明细 | 减少争议 |
| 定期更新 | 婚姻重大事项后重新确认或修订 | 适应变化 |

## 升级决策门

### 触发条件（满足任一即升级）
1. **离婚诉讼**：夫妻已进入离婚诉讼程序
2. **财产争议进入诉讼**：就协议效力或财产分割产生纠纷并进入诉讼
3. **涉及境外资产**：协议涉及的财产位于境外或涉及境外身份
4. **家庭暴力/虐待**：存在家庭暴力或虐待情形的证据
5. **重婚/同居**：存在重婚或与他人同居的情形
6. **一方为上市公司股东**：协议涉及上市公司股份分割
7. **隐匿财产争议**：一方怀疑另一方隐匿财产
8. **债务纠纷**：债权人主张夫妻共同债务

### 升级路径
```
升级决策门触发 → 停止自动生成协议审核建议 →
输出"本案件涉及[具体情形]，建议委托婚姻家庭法律师代理"通知 →
提供专业律师查询路径（参见 references/查询路径.md）
```

### 升级后本skill责任终止情形
- 进入离婚诉讼程序后，本skill不提供诉讼策略建议
- 涉及婚姻效力认定（重婚、无效婚姻）的司法程序，不属于本skill范围
- 子女抚养权、抚养费诉讼，不属于本skill范围

## 本Skill不涵盖

### 明确排除事项
1. **协议公证代理**：代理当事人到公证机构办理公证手续
2. **离婚诉讼代理**：代理当事人提起离婚诉讼或应诉
3. **财产分割代理**：代替当事人进行离婚财产分割谈判或诉讼
4. **婚姻效力认定**：重婚、无效婚姻的认定程序
5. **子女抚养权争议**：子女抚养权、抚养费纠纷的处理
6. **家庭暴力保护**：申请人身安全保护令等家庭暴力应对
7. **跨境婚姻财产**：涉及境外财产、境外身份的夫妻财产约定
8. **遗产继承规划**：一方去世后的遗产继承安排（参见 testament-drafting-checker）
9. **税务申报**：夫妻财产约定涉及的税务申报
10. **信托设立**：将夫妻财产置入信托的架构设计（参见 trust-structure-advisor）
