## 一、家族企业治理结构设计

### 1.1 家族企业治理要素
```
治理框架金字塔：
┌────────────────────────────────────────┐
│            家族宪法/家族协议            │  ← 顶层制度
├────────────────────────────────────────┤
│      家族委员会  │  董事会  │  股东会   │  ← 决策机构
├────────────────────────────────────────┤
│    家族办公室    │  执行团队 │  监事会   │  ← 执行/监督
├────────────────────────────────────────┤
│           家族价值观 + 传承理念          │  ← 文化基础
└────────────────────────────────────────┘
```

### 1.2 家族委员会设计
**职能定位：**
- 家族成员利益协调平台
- 家族重大事项决策机构
- 家族宪法/协议的制定与修改机构
- 家族企业股权管理的最高决策机构

**人员构成建议：**
| 角色 | 建议配置 | 任期 | 备注 |
|-----|---------|-----|-----|
| 委员会主席 | 家族长辈或德高望重者 | 3-5年 | 可连任 |
| 成员 | 各支系代表 | 3年 | 轮换制 |
| 顾问 | 外部专业人士 | 1-2年 | 无投票权 |
| 秘书 | 专职人员 | — | 负责会议记录 |

**决策机制：**
- 简单多数：日常事项
- 绝对多数（≥3/4）：章程修改、家族协议重大变更
- 一致同意：涉及家族根本利益的重大事项

### 1.3 家族宪法/家族协议框架
| 章节 | 主要内容 | 约束力 |
|-----|---------|-------|
| 总则 | 制定目的、家族使命、核心价值观 | 强 |
| 家族成员权利义务 | 家族成员身份认定、权利清单、退出机制 | 强 |
| 股权管理 | 股权分类、转让限制、优先购买权 | 强 |
| 家族企业参与 | 家族成员任职资格、薪酬规则、利益冲突 | 强 |
| 争议解决 | 内部调解程序、仲裁条款 | 强 |
| 附则 | 生效条件、修改程序、解释权 | 强 |

## 二、股权传承架构设计

### 2.1 股权结构规划原则
**核心原则：**
1. **控制权集中**：保持家族对核心企业的控制权
2. **流动性分离**：非核心资产可适度开放流动性
3. **代际递减**：通过信托或控股架构实现多代际传承
4. **退出通道**：为不愿参与经营的家族成员提供合理退出机制

### 2.2 常见股权传承模式
| 模式 | 适用情形 | 优势 | 风险 |
|-----|---------|-----|-----|
| 直接传承 | 单一继承人、小型企业 | 结构简单 | 集中风险 |
| 控股公司传承 | 多业务板块、多继承人 | 隔离保护 | 税负增加 |
| 家族信托传承 | 避免分家析产、代际传承 | 专业管理 | 控制权弱化 |
| 双层股权结构 | 引入外部投资但保持控制 | 融资+控制 | 监管要求高 |
| 有限合伙传承 | GP/LP分离，家族担任GP | 税务优化 | 法律复杂 |

### 2.3 股权传承工具选择
```
工具比较矩阵：
┌──────────────────────────────────────────────────────┐
│ 工具        │ 控制力 │ 税务效率 │ 灵活性 │ 复杂度  │
├──────────────────────────────────────────────────────┤
│ 直接转让    │ 高     │ 中      │ 低      │ 低      │
│ 家族信托    │ 中     │ 高      │ 高      │ 高      │
│ 控股公司    │ 高     │ 低      │ 中      │ 中      │
│ 人寿保险    │ 中     │ 高      │ 中      │ 低      │
└──────────────────────────────────────────────────────┘
```

### 2.4 股权转让限制条款
**必要限制条款：**
1. **优先购买权**：家族成员转让股权时，其他成员有优先购买权
2. **随售权（Tag-along）**：外部收购时，家族成员有权按比例共同出售
3. **拖售权（Drag-along）**：多数股东有权强制少数股东一同出售
4. **锁定期**：家族成员入职后若干年内不得转让股权
5. **竞业限制**：离职后若干年内不得从事同业竞争

## 三、家族协议与冲突解决

### 3.1 家族协议类型
| 协议名称 | 核心内容 | 签署主体 |
|---------|---------|---------|
| 家族宪法 | 家族治理根本大法 | 全体家族成员 |
| 家族协议 | 家族企业具体安排 | 家族成员+企业 |
| 股权代持协议 | 股权代持安排 | 代持人与被代持人 |
| 一致行动人协议 | 表决权协同安排 | 家族成员股东 |
| 竞业禁止协议 | 限制同业竞争 | 家族成员+企业 |
| 退出协议 | 退出价格与程序 | 退出成员+企业 |

### 3.2 典型家族冲突类型与预防
| 冲突类型 | 触发情形 | 预防机制 | 解决路径 |
|---------|---------|---------|---------|
| 经营权争夺 | 继承人不愿退让 | 明确任职资格+绩效考核 | 调解→仲裁→诉讼 |
| 股权继承争议 | 多个继承人主张 | 遗嘱+信托安排 | 调解→诉讼 |
| 退出价格争议 | 退出成员与留任成员 | 事先约定定价公式 | 评估→协商 |
| 利益冲突 | 家族成员从事同业竞争 | 竞业禁止+信息披露 | 调解→诉讼 |
| 婚姻财产纠纷 | 家族成员离婚 | 婚姻财产协议+股权限制 | 调解→诉讼 |

### 3.3 家族企业传承时间表
```
典型传承规划（以企业主60岁为起点）：
T-10年（50岁）：
- 启动传承规划
- 建立家族治理机制
- 培养下一代继承人

T-5年（55岁）：
- 明确继承人
- 开始股权调整
- 继承人进入企业

T-3年（57岁）：
- 继承人担任高管
- 董事会逐步改组
- 家族协议正式签署

T-0（60岁）：
- 完成股权传承
- 创始人退出日常管理
- 担任家族委员会主席（荣誉职务）

T+5（65岁）：
- 完全退出
- 担任家族委员会顾问
- 传承完成
```

## 升级决策门

### 触发条件（满足任一即升级）
1. **股权纠纷进入诉讼**：家族成员之间就股权归属、转让等产生诉讼
2. **涉及公众公司**：家族企业为上市公司或拟上市公司
3. **跨境架构**：涉及境外持股平台、离岸公司等跨境架构
4. **金融机构介入**：债权人或金融机构主张权利（如抵押物处置）
5. **继承人之争**：多个继承人争夺企业控制权
6. **婚姻财产纠纷**：家族成员婚姻财产争议涉及企业股权
7. **刑事风险**：家族企业涉及非法集资、职务侵占等刑事风险
8. **监管介入**：企业面临行业监管机构的调查或处罚

### 升级路径
```
升级决策门触发 → 停止自动生成治理建议 →
输出"本案件涉及[具体情形]，建议委托以下专业律师"通知 →
提供公司法专业律师、家族财富律师查询路径（参见 references/查询路径.md）
```

### 升级后本skill责任终止情形
- 进入司法程序后，本skill不提供诉讼策略建议
- 上市公司治理涉及的监管合规，不属于本skill范围
- 跨境架构设计的具体操作，不属于本skill范围

## 本Skill不涵盖

### 明确排除事项
1. **公司章程起草代理**：代替企业起草或修改公司章程
2. **公司登记代理**：代理办理工商变更登记手续
3. **股权纠纷诉讼代理**：代理家族成员提起或应对股权纠纷诉讼
4. **上市公司治理咨询**：涉及上市公司信息披露、关联交易等合规咨询
5. **跨境股权架构设计**：离岸公司设立、BVI架构等
6. **继承税务计算**：遗产税（如开征）的具体计算
7. **刑事辩护**：涉及家族企业的刑事犯罪辩护
8. **婚姻财产诉讼代理**：代理离婚诉讼中的财产分割
9. **劳动法律服务**：家族企业员工的劳动纠纷处理
10. **知识产权诉讼代理**：商标、专利侵权的诉讼代理
