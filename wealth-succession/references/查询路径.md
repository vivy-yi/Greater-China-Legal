# 查询路径 — 财富传承法律服务检索指引

## 一、法律检索路径

### 1.1 法律法规检索
**首选平台：**
| 平台 | 网址 | 适用场景 |
|-----|-----|---------|
| 国家法律法规数据库 | https://flk.npc.gov.cn | 法律法规原文检索 |
| 最高人民法院官网 | https://www.court.gov.cn | 司法解释、典型案例 |
| 中国裁判文书网 | https://wenshu.court.gov.cn | 案例检索 |
| 北大法宝 | https://www.pkulaw.com | 综合法律检索（需订阅） |

**路径示例：**
```
查询"遗嘱效力"相关法规：
1. 国家法律法规数据库 → 搜索"民法典 继承编"
2. 筛选"司法解释"子分类
3. 下载《最高人民法院关于适用〈中华人民共和国民法典〉继承编的解释（一）》
4. 检索"遗嘱"关键词
```

### 1.2 司法案例检索
**案例检索路径：**
```
中国裁判文书网 → 关键词检索 → 筛选条件
├─ 案件类型：民事案件
├─ 案由：继承纠纷
├─ 文书类型：判决书
└─ 时间范围：近三年

常用检索表达式：
- "遗嘱" AND "效力" AND "继承纠纷"
- "遗嘱执行人" AND "继承纠纷"
- "必留份" AND "遗嘱继承"
```

## 二、专业机构查询路径

### 2.1 公证机构查询
**全国公证机构信息查询：**
```
司法部政务服务 → 公证服务 → 公证机构查询
https://www.12348.gov.cn

查询要素：
- 被继承人住所地公证机构
- 遗产主要所在地公证机构
- 具备遗产继承公证资质的机构
```

### 2.2 律师协会推荐
**查询路径：**
| 层级 | 查询入口 | 适用场景 |
|-----|---------|---------|
| 全国 | 中华全国律师协会官网 | 跨省律师推荐 |
| 省级 | 各省律师协会官网 | 省内专业律师 |
| 市级 | 各地司法局/律师协会 | 本地律师推荐 |

**专业方向筛选：**
- 继承法律服务
- 婚姻家庭法律服务
- 公司法/股权传承
- 信托与资产管理
- 税务法律服务

### 2.3 信托公司查询
**持牌信托公司查询：**
```
银保监会官网 → 金融许可证信息查询 → 信托公司
https://www.cbirc.gov.cn

筛选条件：
- 机构类型：信托公司
- 业务范围：家族信托
```

## 三、特定问题查询路径

### 3.1 遗嘱效力鉴定查询
**司法鉴定机构查询：**
```
司法部政务服务 → 司法鉴定机构查询
https://www.12348.gov.cn

关注方向：
- 法医临床鉴定（遗嘱能力）
- 文书鉴定（签名真伪）
- 痕迹鉴定（笔迹比对）
```

### 3.2 资产评估机构查询
**不动产评估：**
```
住房和城乡建设部 → 房地产估价机构查询
https://www.mohurd.gov.cn

关注资质：
- 一级/二级房地产估价机构
- 全国A级以上评估机构
```

**企业股权评估：**
```
财政部/证监会 → 证券评估资质机构
https://www.cas.org.cn

关注资质：
- 证券期货相关业务评估资格
- 矿业权评估资格
```

### 3.3 跨境服务查询

## 四、时效性确认路径

### 4.1 法条时效性检查
**检查频率：**
- 每年1月：年度法规清理后的首次检查
- 重大立法变动后：立即检查受影响法条
- 用户询问时：对引用法条进行即时确认

**时效性标记格式：**
```
[法规名称] 第[条款号]
- 生效日期：YYYY-MM-DD
- 最近修订：YYYY-MM-DD
- 效力状态：现行/废止/部分废止
- 注意事项：[时效性说明]
```

### 4.2 重要时效节点追踪
| 事项 | 关键时限 | 来源 |
|-----|---------|-----|
| 继承开始后遗产分割 | 分割期限无硬性规定，但债权人可请求 | 《民法典》第1147条 |
| 受遗赠人表示接受 | 知道受遗赠后60日内 | 《民法典》第1124条 |
| 继承权纠纷诉讼时效 | 3年（继承开始后超过20年） | 《民法典》第188条 |
| 遗嘱执行人资格争议 | 无固定时效 | 《民法典》第1133条 |
