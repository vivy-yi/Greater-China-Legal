# 基金绩效报告技能

## 技能说明

本技能用于生成私募股权基金的定期绩效报告，支持向LP、监管机构提供规范化、可比的业绩报告。

## 核心功能

### 1. 业绩指标计算
- 已投资本金（Paid-in Capital）
- 呼叫资本（Called Capital）
- 分配金额（D distributions）
- 净资产值（NAV）
- 投资回报倍数（MOIC / TVPI）
- 内部收益率（IRR）
- 内部收益率倍数（MOIC vs IRR对比）

### 2. 投资组合报告
- 被投企业估值变动
- 持股比例变化
- 重大事项记录
- 退出进展跟踪

### 3. 基金运营报告
- 投资进度
- 现金管理情况
- 费用支出情况
- 人员变动情况

### 4. 合规披露
- 信息披露报告
- 重大事项报告
- 关联交易披露
- 投资者年会材料

### 5. 行业对标
- 同类基金业绩对比
- 行业基准指数参照
- 策略有效性分析

## 输入要求

请提供：
- 基金账务数据
- 投资组合明细
- 估值报告
- 基金合同约定的披露模板

## 输出内容

报告包含：
1. 业绩表现摘要
2. 投资组合明细
3. 基金运营概览
4. 风险提示（如有）
5. 合规确认声明

## 适用场景

- 定期LP报告（季度/年度）
- 基金业协会信息披露
- 投资者年会演示
- 监管检查配合材料
