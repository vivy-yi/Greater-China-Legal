# 退出策略顾问技能

## 技能说明

本技能用于为私募股权基金提供退出策略分析与建议，综合考虑法律合规、税务优化、市场时机等因素。

## 核心功能

### 1. 退出方式评估

#### IPO退出
- 上市地点选择（A股、港股、美股）
- 锁定期合规
- 减持规则适用
- 上市条件准备

#### 股权转让退出
- 协议转让流程
- 进场交易要求（国有股权）
- 优先购买权处理
- 税务筹划

#### 回购退出
- 回购条款有效性
- 公司法合规性
- 履行程序检查
- 税务处理

#### 清算退出
- 清算程序启动条件
- 清算组组成
- 债权债务处理
- 剩余财产分配

### 2. 时机判断
- 行业周期分析
- 退出市场环境评估
- 估值水平分析
- 基金期限匹配

### 3. 税务筹划
- 股权转让所得税计算
- 地方性优惠政策申请
- 合伙企业税务穿透处理
- 跨境退出税务规划

### 4. 特殊情形处理
- S基金交易
- 二手份额转让
- 棘轮条款触发
- 拖带权行使

## 输入要求

请提供：
- 基金基本情况（存续期限、投资阶段）
- 投资组合明细
- 各被投企业情况
- 目标退出时间窗口
- 退出方式偏好

## 输出内容

退出策略报告包含：
1. 各投资组合退出路径建议
2. 最佳退出时机建议
3. 税务筹划方案
4. 操作流程指引
5. 风险提示

## 适用场景

- 基金临近退出期
- 被投企业触发回购条件
- 市场出现退出窗口
- LP要求提前退出
