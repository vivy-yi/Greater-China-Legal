# 数据源清单

## 一、内部数据源

### 1.1 基金文件
- 有限合伙协议（LPA）
- 认购协议/申购确认书
- 基金招募说明书
- 托管协议
- 管理协议
- 投资顾问协议

### 1.2 投资者文件
- 投资者适当性证明材料
- 风险测评问卷
- 身份证明文件（法人/自然人）
- 资金来源证明
- 反洗钱调查文件

### 1.3 投资组合文件
- 被投企业工商登记信息
- 投资协议全套文件
- 股东名册及持股证明
- 历次估值报告
- 退出相关文件

### 1.4 财务数据
- 银行流水对账单
- 托管账户报告
- 估值计算底稿
- 税务申报材料
- 审计报告

## 二、外部数据源

### 2.1 政府公开数据
| 数据源 | 获取方式 | 更新频率 |
|-------|---------|---------|
| 中基协公示信息 | 在线查询 | 实时 |
| 企业工商信息 | 公示系统 | 定期 |
| 税务登记信息 | 实名认证后查询 | 按需 |
| 土地/房产信息 | 不动产登记中心 | 按需 |

### 2.2 商业数据库
- **Wind资讯**：金融市场数据
- **Bloomberg**：国际金融市场数据
- **CVSource**：私募股权市场数据
- **企查查/天眼查**：企业工商信息

### 2.3 第三方机构
- 律师事务所：法律意见书
- 会计师事务所：审计报告、税务意见
- 评估机构：资产评估报告
- 托管银行：资金保管报告

## 三、实时数据接口

### 3.1 行情数据
- 交易所行情：沪深交易所
- 新三板行情：全国股转系统
- 境外行情：相应交易所

### 3.2 监管信息
- 证监会公告
- 交易所监管动态
- 中基协通知公告

## 四、数据使用规范

### 4.1 保密要求
- 投资人信息不得对外披露
- 投资组合信息需脱敏处理
- 商业数据需遵守使用协议

### 4.2 合规留存
- 文件保存期限不少于10年
- 电子数据需备份
- 敏感信息需加密存储
