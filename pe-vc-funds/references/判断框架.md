# 判断框架

## 一、基金设立合规性判断

### 1.1 法律主体资格
- 基金管理人是否完成登记
- 基金是否完成备案（私募基金需在中基协备案）
- 组织形式是否符合规定（公司型/有限合伙型/契约型）

### 1.2 投资者适当性
- 投资者是否为合格投资者
- 投资者人数是否符合上限（单只基金不超过200人）
- 投资者风险测评是否完成

### 1.3 募集程序合规
- 是否履行特定对象确定程序
- 风险揭示是否充分
- 冷静期安排是否合规
- 回访确认是否完成

## 二、LP尽职调查判断

### 2.1 主体资格审查
| 审查项目 | 判断标准 |
|---------|---------|
| 法人资格 | 有效存续的企业法人 |
| 出资能力 | 认缴金额与净资产匹配 |
| 合规记录 | 无重大违法违规 |
| 关联关系 | 与GP无利益冲突 |

### 2.2 反洗钱审查
- 是否在制裁名单中
- 资金来源是否合法
- 是否符合FATF规定

## 三、投资协议合规判断

### 3.1 必备条款检查
- 估值调整条款（对赌条款）
- 优先清算条款
- 共同出售权条款
- 拖带权条款
- 保护性条款

### 3.2 禁止性规定
- 不得承诺最低收益
- 不得变相保本保收益
- 不得违反投资者适当性

## 四、退出合规判断

### 4.1 退出方式选择
- IPO退出：需符合锁定期规定
- 股权转让：需履行优先购买权程序
- 回购退出：需符合公司法规定
- 清算退出：需履行清算程序

### 4.2 税务合规
- 股权转让所得税计算
- 合伙企业税务穿透
- 地方性税收优惠政策适用
