# 中国法律域分类体系

## 核心能力

将用户的法律问题归类到正确的法律域，选择对应的处理流程。

---

## 法律域分类

| 域代码 | 法律域 | 核心法规 | 对应Scene |
|--------|--------|---------|----------|
| L-INVEST | 投资 | 《私募投资基金监督管理条例》《外商投资法》 | corporate-governance |
| L-FINANCE | 金融 | 《证券法》《基金法》《银行业监督管理法》 | internet-finance |
| L-DATA | 数据/隐私 | 《个人信息保护法》《数据安全法》《网络安全法》 | data-compliance |
| L-LABOR | 劳动 | 《劳动合同法》《工伤保险条例》《劳动争议调解仲裁法》 | labor-arbitration |
| L-CONTRACT | 合同/商事 | 《民法典》合同编《商法》 | contract-review |
| L-IP | 知识产权 | 《专利法》《商标法》《著作权法》《反不正当竞争法》 | ip-infringement |
| L-CORP | 公司/股权 | 《公司法》《企业国有资产法》 | corporate-governance |
| L-LITIGATION | 诉讼/仲裁 | 《民事诉讼法》《仲裁法》《劳动争议调解仲裁法》 | litigation-support |
| L-TAX | 税务 | 《企业所得税法》《增值税暂行条例》 | tax-compliance |
| L-CRIMINAL | 刑事 | 《刑法》《刑事诉讼法》 | （独立司法程序） |
| L-ADMIN | 行政 | 《行政处罚法》《行政复议法》《行政诉讼法》 | litigation-support |

---

## 路由规则

```
用户法律问题 → 关键词识别 → 归类到法律域 → 选择Scene Skill

示例：
"公司要开除一个员工" → L-LABOR → labor-arbitration
"签了一个采购合同" → L-CONTRACT → contract-review
"客户数据泄露了" → L-DATA → data-compliance
"竞争对手用了我们的商标" → L-IP → ip-infringement
"对方欠钱不还" → L-LITIGATION → litigation-support
"要投资一个私募基金" → L-INVEST → corporate-governance
```

## 域内子分类

### L-CONTRACT（合同/商事）

| 子类 | 关键词 | 典型场景 |
|------|--------|---------|
| 采购/销售 | 采购合同、销售合同、买卖合同 | 供应链合同审查 |
| 服务 | 服务合同、委托合同、居间合同 | 咨询服务、外包 |
| 租赁 | 租赁合同、融资租赁 | 设备/房产租赁 |
| 投资 | 投资协议、股权转让协议、对赌条款 | 股权投资 |
| 劳动 | 劳动合同、劳务协议、保密协议、竞业限制 | 雇佣关系 |
| 借贷 | 借款合同、贷款合同、借条 | 民间借贷、金融借贷 |
| 建筑工程 | 建设工程合同、施工合同 | 房地产/基建 |
| 知识产权许可 | 专利许可、商标许可、软件许可 | IP交易 |

### L-LABOR（劳动）

| 子类 | 关键词 |
|------|--------|
| 解除/终止 | 违法解除、经济性裁员、合同到期不续 |
| 工伤 | 工伤认定、工伤赔偿、工伤等级 |
| 工资/加班 | 拖欠工资、加班费、未休年假 |
| 社会保险 | 社保缴纳、公积金 |
| 保密/竞业 | 商业秘密、竞业限制、保密协议 |
| 劳务派遣 | 派遣员工、用工单位责任 |

### L-DATA（数据/隐私）

| 子类 | 关键词 |
|------|--------|
| 个人信息收集 | 隐私政策、用户同意、Cookie |
| 数据跨境 | 跨境传输、出境评估、数据本地化 |
| 数据泄露 | 数据安全事件、泄露通知、应急响应 |
| 商业数据 | 商业秘密、数据权属、大数据权益 |
| 监管合规 | 网信办处罚、数据安全评估 |
