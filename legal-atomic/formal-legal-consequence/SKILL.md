# /formal-legal-consequence — 形式化推导法律后果

## 四层推导模型

```
第一层：基础后果确定
  R(规范) + F(事实) → E₀(基础法律后果)

第二层：调整因素适用
  E₀ + A₊(加重因素) + A₋(减轻因素) + A₀(免除因素) → E₁(调整后)

第三层：竞合与聚合处理
  E₁ + 竞合规则/聚合规则 → E₂(最终后果)

第四层：量化计算
  E₂ + 计算标准/裁量基准 → E_final(具体量化结果)
```

## 法律后果分类

### 民事法律后果

| 类型 | 具体后果 |
|------|---------|
| 合同效力 | 有效/无效/可撤销/效力待定 |
| 违约责任 | 继续履行/赔偿损失/违约金/定金罚则 |
| 侵权责任 | 停止侵害/赔偿损失/赔礼道歉/精神损害赔偿 |
| 物权变动 | 所有权转移/抵押权设立/善意取得 |
| 不当得利/无因管理 | 返还/补偿 |

### 刑事法律后果

| 类型 | 具体内容 |
|------|---------|
| 主刑 | 管制/拘役/有期徒刑/无期徒刑/死刑 |
| 附加刑 | 罚金/剥夺政治权利/没收财产/驱逐出境 |
| 量刑调整 | 从重/从轻/减轻/免除处罚 |
| 非刑罚 | 免予刑事处罚/缓刑/假释 |

### 行政法律后果

| 类型 | 具体内容 |
|------|---------|
| 行政处罚 | 警告/罚款/没收/吊销许可证/行政拘留 |
| 行政强制 | 查封/扣押/冻结/强制执行 |
| 行政许可 | 准予/不予/撤销/注销 |

### 程序性后果

| 类型 | 说明 |
|------|------|
| 诉讼时效 | 丧失胜诉权 |
| 举证责任 | 承担不利后果 |
| 管辖权 | 移送/驳回 |

## 工作流程

```
步骤1：确认输入要素
  - 已认定的事实
  - 已匹配的法律规范
  - 构成要件满足情况
  - 调整因素（加重/减轻/免除）

步骤2：第一层推导（基础后果）
  将事实涵摄于法律规范

步骤3：第二层推导（调整因素）
  叠加量刑/责任调整因素

步骤4：第三层推导（竞合聚合）
  处理罪名竞合/责任聚合

步骤5：第四层推导（量化计算）
  计算具体金额/刑期区间/罚款幅度

步骤6：标注置信度
  高/中/低，说明不确定性因素
```

## 赔偿金额计算

```
损害赔偿 = 直接损失 + 间接损失（预期利益）+ 精神损害（符合条件的）

违约金：
- 约定违约金 > 实际损失30% → 可申请法院酌减
- 约定的定金 > 标的额20% → 超过部分无效

刑期计算：
- 从轻：基准刑 × (1 - 减刑比例)
- 减轻：降档处理
- 数罪并罚：总和刑期以下，最高刑期以上
```
