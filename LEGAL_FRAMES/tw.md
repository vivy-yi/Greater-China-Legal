# Legal Frame: Taiwan (台湾)

## 法律体系

- **法系：** 大陆法系（承袭日本/德国法律体系，国民党时期法律现代化）
- **法律渊源：** 制定法为主（民法/刑法/行政法等法典），判例仅为参考（非正式渊源）

## 核心法规体系

### 民法
- **民法总则**（2023年修正）— 权利主体/客体/法律行为/代理/时效
- **民法债编**（2023年修正）— 契约/无因管理/侵权行为/不当得利
- **民法物权编**（2023年修正）— 物权通则/所有权/用益物权/担保物权
- **民法亲属编**（2022年修正）
- **民法继承编**（2023年修正）

### 商法
- **公司法**（2024年修正）
- **票据法**
- **海商法**
- **保险法**

### 劳动法
- **劳动基准法**（2024年修正）
- **劳动事件法**（2019年制定）
- **职业安全卫生法**
- **大量解雇劳工保护法**
- **劳动检查法**

### 知识产权
- **专利法**
- **商标法**
- **著作权法**
- **营业秘密法**

### 个人信息保护
- **个人资料保护法**（2015年修正）

### 诉讼与仲裁
- **民事诉讼法**
- **刑事诉讼法**
- **行政诉讼法**
- **仲裁法**

## 关键概念差异（vs 大陆）

| 大陆概念 | 台湾对应 | 说明 |
|---|---|---|
| 民法典 | 民法（分编独立法典） | 总则/债编/物权编/亲属/继承 |
| 劳动合同法 | 劳动基准法 | 雇主与劳工权利义务基准 |
| 给付 | 给付 | 术语相同，含义相近 |
| 物权 | 物权 | 相同概念 |
| 债权 | 债权 | 相同概念 |
| 经济补偿金 | 资遣费 | 按劳工保护法计算 |
| 竞业限制 | 竞业禁止条款 | 须有补偿否则无效（法院判例） |
| 格式条款 | 定型化契约条款 | 民法 §247-1保护 |

## 司法体系

- **最高法院**（民事/刑事/行政诉讼的第三审）
- **高等法院**（区域第二审）
- **地方法院**（第一审，含简易诉讼程序）
- **智慧财产法院**（专门管辖IP案件）
- **劳工法院**（劳动事件法2019年设立专门法院）

## 合同法核心原则

1. **契约自由原则**（民法 §71）
2. **诚信原则**（民法 §148、§219）
3. **有偿等价原则**（有偿双务契约 §241-第247条）
4. **债权保全**（民法 §242-第243条）
5. **定型化契约条款控制**（民法 §247-1）

## 劳动法核心原则

1. **劳动条件书面化**（劳动基准法 §70）
2. **最低劳动基准**（劳动基准法第1章）
3. **积欠工资垫偿**（劳动基准法第28条）
4. **职业灾害补偿**（劳动基准法 §59）
5. **劳工请假权利**（劳动基准法 §43-第50条）
