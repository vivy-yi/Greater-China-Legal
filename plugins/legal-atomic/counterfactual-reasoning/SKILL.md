# /counterfactual-reasoning — 反事实推理

## 核心概念

```
现实世界：A发生 → B发生
反事实世界：假设A未发生 → B是否仍然发生？

若B不再发生 → A是B的原因（具有因果贡献）
若B仍然发生 → A不是B的原因（无因果贡献）
```

### 关键术语

| 术语 | 含义 |
|---|---|
| **目标变量** | 被假设改变/移除的事实/行为/条件 |
| **结果变量** | 需要观察是否变化的法律后果 |
| **反事实世界** | 目标变量被改变后的假设情境 |
| **But-for检验** | "若非"检验——若非A，则B不会发生 |
| **NESS检验** | 不可或缺条件检验 |

---

## 工作流程

### 步骤1：确定分析框架

```
1. 明确目标变量：哪个事实/行为/条件要被假设改变
2. 明确结果变量：哪个法律结果要被评估
3. 确定背景条件：反事实世界中保持不变的其他事实
```

### 步骤2：构建反事实世界

```
构建一个与现实世界完全相同的世界，
除了目标变量被改变/移除。

关键问题：
- 其他所有背景条件是否保持不变？
- 改变目标变量后，现实世界的时间线如何调整？
```

### 步骤3：执行But-for检验

```
现实世界：A发生 → B发生
反事实世界：假设A未发生
  → 若B不发生：But-for因果关系成立
  → 若B仍然发生：无But-for因果关系

注意：必须考虑反事实世界中其他条件是否需要调整
（替代原因/阻断原因的问题）
```

### 步骤4：NESS检验（补充）

```
NESS = 必要充分条件集合

对结果B的每个必要条件X：
- 如果X的缺失会导致B不发生 → X是B的必要条件
- 如果X的集合中每个成员的缺失都会导致B不发生 → 该集合是B的NESS

NESS检验比简单But-for更严格，
可排除替代原因造成的虚假因果。
```

### 步骤5：因果贡献评估

```
如果But-for成立：
  → 评估目标变量对结果的贡献程度
  → 在多因一果的情况下，评估各原因的相对贡献
  → 考虑因果中断的可能性（介入原因）

如果But-for不成立：
  → 目标变量不是法律上的原因
  → 可能需要考虑其他因果理论（预见性因果、重要性因果）
```

## 常见应用场景

### 侵权法：因果关系认定

```
问题：若无被告的过失行为，原告的损害是否仍会发生？

现实：被告超速 → 碰撞 → 原告受伤
反事实：被告未超速 → 碰撞是否仍会发生？
  → 如果正常速度足以避免碰撞 → 因果关系成立
  → 如果碰撞仍会发生（不可避免）→ 因果关系不成立
```

### 合同法：违约损失计算

```
问题：如果合同未被违约，债权人本可获得的利益是多少？

现实：违约方违约 → 债权人损失X
反事实：违约方履行 → 债权人本可获得Y
损失 = Y - X（可得利益损失）
```

### 损害赔偿：可预见性限制

```
问题：如果违约方违约但非违约方未遭受损害，是否需要赔偿？

反事实检验：若无违约方的违约行为，损害是否仍会发生？
  → 如果损害无论如何都会发生 → 违约方的因果贡献有限
  → 如果损害因违约才发生 → 因果关系成立
```
