# /conflict-resolution — 冲突解决和优先级判定

## 三大冲突类型

```
冲突类型
├── A. 法条竞合（规范冲突）
│   ├── A1. 同一法律内部条文竞合
│   ├── A2. 不同法律之间规范冲突
│   ├── A3. 不同位阶法源冲突
│   └── A4. 请求权竞合
├── B. 证据矛盾（事实冲突）
│   ├── B1. 同类证据之间矛盾
│   ├── B2. 不同类型证据之间矛盾
│   └── B3. 直接证据与间接证据矛盾
└── C. 争点优先级（程序/逻辑冲突）
    ├── C1. 程序问题与实体问题排序
    ├── C2. 前提性争点与派生性争点排序
    └── C3. 主要争点与次要争点排序
```

---

## 冲突识别流程

```
步骤0.1：扫描输入材料，识别是否存在冲突
  ├── 检查是否有多个可适用法条 → A类法条竞合
  ├── 检查证据之间是否存在不一致 → B类证据矛盾
  ├── 检查是否有多个争议焦点 → C类争点排序
  └── 若无冲突 → 退出本skill，转入常规分析

步骤0.2：冲突分类
  ├── 判断冲突属于A/B/C
  └── 一个案件可能同时存在多种冲突类型

步骤0.3：冲突严重性评估
  ├── 高：冲突结论将根本改变案件走向
  ├── 中：冲突影响部分争点处理
  └── 低：冲突仅影响细节
```

## A类：法条竞合解决流程

```
步骤A1：列出所有可能适用的法律规范
  - 逐一列出条文全文
  - 标注每个条文的法源位阶（宪法>法律>行政法规>地方性法规>规章）
  - 标注制定/修订时间
  - 标注适用范围（一般/特别）

步骤A2：判断竞合类型
  ├── 不同位阶冲突？→ 上位法优于下位法
  ├── 同一位阶一般法vs特别法？→ 特别法优于一般法
  ├── 同一机关新旧法冲突？→ 新法优于旧法
  ├── 请求权竞合？→ 分析各请求权利弊，告知当事人选择权
  └── 新一般法vs旧特别法冲突？→ 报请有权机关裁决，标注低置信度

步骤A3：形成解决结论
  - 明确最终适用的法律条文
  - 说明排除其他条文的理由
  - 标注置信度
```

### 民法典竞合常见情形

| 竞合情形 | 解决规则 |
|---|---|
| 违约责任 vs 侵权责任（《民法典》第186条） | 当事人可选择 |
| 合同编通则 vs 分则 | 分则优先（特别优于一般） |
| 《民法典》vs《消保法》 | 《消保法》优先 |
| 《民法典》vs《公司法》 | 《公司法》优先 |
| 新法 vs 旧法（同一机关） | 新法优于旧法 |

## B类：证据矛盾解决流程

```
步骤B1：识别和描述矛盾
  - 列出相互矛盾的证据
  - 明确矛盾的具体内容
  - 判断矛盾是根本性还是细节性

步骤B2：证据能力审查（前置过滤）
  ├── 是否属于非法证据应予排除？
  ├── 是否符合法定形式要求？
  └── 若一方无证据能力 → 排除该证据，矛盾消除

步骤B3：证明力比较
  ├── 国家机关公文书证 > 其他书证
  ├── 公证文书 > 一般书证
  ├── 原始证据 > 传来证据
  ├── 直接证据 > 间接证据
  ├── 与当事人无利害关系的证人证言 > 有利害关系的
  └── 数个证据形成的证据链 > 孤证

步骤B4：综合认定
  ├── 将证明力较强的证据作为认定事实的基础
  ├── 检验该认定是否与经验法则一致
  ├── 适用证明标准：
  │   民事一般：高度盖然性（>75%）
  │   民事欺诈/胁迫：高于高度盖然性
  │   刑事：排除合理怀疑
  │   行政：明显优势
  └── 矛盾无法解决时 → 适用举证责任分配规则

步骤B5：形成结论
  - 明确采信/不予采信的证据及理由
  - 标注置信度
```

## C类：争点优先级排序流程

```
步骤C1：争点识别与列举
  - 从诉讼请求中提取争点
  - 从答辩意见中提取争点
  - 形成完整争点清单

步骤C2：争点分类
  ├── 按性质：程序性/事实争点/法律争点
  ├── 按逻辑：前提性/独立/派生性
  └── 按重要：核心/重要/辅助

步骤C3：建立争点依赖关系图
  例：合同是否成立 → 合同是否有效 → 违约责任如何承担

步骤C4：排序规则
  ├── 第一优先：程序性争点（管辖/时效/主体资格）
  ├── 第二优先：前提性事实争点（法律关系是否成立）
  ├── 第三优先：核心法律争点（法律适用/解释）
  ├── 第四优先：派生性争点（责任范围/赔偿数额）
  └── 第五优先：辅助性争点（诉讼费用/执行方式）

步骤C5：效率优化——识别"短路"机会
  ├── 若诉讼时效已过 → 可能无需审理实体争点
  ├── 若合同不成立 → 无需审理违约责任
  └── 若不构成侵权 → 无需审理损害赔偿
```

### 争点排序速查

| 优先级 | 争点类型 | 示例 |
|---|---|---|
| 1（最高） | 程序性争点 | 管辖权/诉讼时效/主体资格 |
| 2 | 前提性事实争点 | 合同成立/法律关系存在 |
| 3 | 核心法律争点 | 法律适用/法律解释 |
| 4 | 派生性争点 | 责任范围/赔偿数额 |
| 5（最低） | 辅助性争点 | 诉讼费用/执行方式 |

## 多类冲突并存处理

```
步骤D1：冲突交叉分析
  - 检查法条竞合的解决是否影响争点排序
  - 检查证据矛盾的解决是否影响法条选择

步骤D2：迭代解决
  ├── 先解决最基础的冲突（通常C类争点排序）
  ├── 再解决影响范围最大的冲突
  ├── 最后解决派生性冲突
  └── 必要时进行多轮迭代

步骤D3：一致性检验
  - 检查所有冲突解决结论之间是否一致
  - 检查最终结论是否符合法律体系内在逻辑
```

## 输出格式

### 冲突识别报告

```
## 冲突识别

| 冲突编号 | 类型 | 内容摘要 | 严重性 |
|---|---|---|---|
| 冲突1 | A类-法条竞合 | 《民法典》第X条 vs 《消保法》第Y条 | 高 |
| 冲突2 | B类-证据矛盾 | 报告A（不合格）vs 报告B（合格） | 高 |
```

### 法条竞合解决报告

```
## 法条竞合解决

### 竞合类型：[类型]
### 涉及条文：
- 《X法》第X条（[位阶]）：[内容摘要]
- 《Y法》第Y条（[位阶]）：[内容摘要]

### 解决规则：[适用规则]
### 最终适用：[条文]（理由：[理由]）
### 置信度：[高/中/低]
```

### 证据矛盾解决报告

```
## 证据矛盾解决

### 矛盾证据：
- 证据A（[提交方]）：[结论]
- 证据B（[提交方]）：[结论]

### 证据能力：两者均✓ / 证据X已被排除（原因：[原因]）
### 证明力比较：[分析]
### 采信结论：[证据A/B]（理由：[理由]）
### 置信度：[高/中/低]
```

### 争点优先级报告

```
## 争点优先级

| 优先级 | 争点编号 | 争点内容 | 类型 | 理由 |
|---|---|---|---|---|
| 1 | 争点3 | 诉讼时效是否经过 | 程序性 | 时效抗辩成立则无需审理实体 |
| 2 | 争点1 | 合同是否成立 | 前提性 | 是后续争点的前提 |
| ... | | | | |

### 短路路径：
- 若争点3结论为"已过时效" → 争点1、2无需审理
```

## 置信度标注

| 等级 | 含义 | 典型场景 |
|---|---|---|
| 🟢 高 | 规则明确，适用无争议 | 明确的上位法vs下位法冲突 |
| 🟡 中 | 规则基本明确，存在解释空间 | 特别法关系认定存在争议 |
| 🔴 低 | 规则不明确，存在重大争议 | 新一般法vs旧特别法冲突；证据势均力敌 |
