# /judicial-value-judgment — 司法价值判断

## 六步框架

### Step 1：判断是否需要调用本Skill

```
需要调用的情况：
- 法律规则存在不确定性
- 存在权利/利益/价值之间的冲突
- 存在开放性法律概念/裁量空间/规范漏洞
- 需要审查权利限制/利益牺牲/公共目的措施是否适度
- 机械适用规则可能导致明显不公

不需要调用的情况：
- 单纯事实查明
- 证据采信
- 明确规则适用
```

### Step 2：锁定裁量问题

```
四类问题区分：
1. 事实问题 → 本skill不处理
2. 规范问题 → 本skill不处理
3. 价值问题 → 本skill重点处理
4. 裁量问题 → 本skill重点处理
```

### Step 3：识别价值冲突结构

| 冲突类型 | 说明 |
|---------|------|
| 个体之间 | 一方自由 vs 另一方人格/财产/隐私/名誉 |
| 共同体之间 | 不同公共利益/制度目标/区域利益冲突 |
| 个体vs共同体 | 个人自由/财产权/隐私权 vs 公共秩序/安全/效率 |

### Step 4：定位相关法律价值

| 核心价值 | 具体表现 |
|---------|---------|
| 秩序 | 社会关系稳定、权力运行有序 |
| 自由 | 依法自主选择、免受非法干涉 |
| 平等 | 排除特权和歧视 |
| 人权 | 尊重独立人格、平等尊严、基本权利 |
| 正义 | 利益/负担/责任/程序的正当分配 |
| 效率 | 资源配置、活动便利、社会财富增长 |

### Step 5：确认规范基础与裁量边界

```
检查项：
- 是否存在明确法律规则？
- 是否存在法律原则/制度目的/立法目的？
- 是否存在宪法/法律/司法解释/指导性案例？
- 法官是否具有裁量空间？
- 价值判断是否可能突破法律文本/制度目的/程序限制？
- 是否存在不得触碰的基本权利/法定程序/权限边界？
```

### Step 6：适用价值位阶原则

```
高优先级 > 低优先级：
- 基本价值通常优先于非基本价值
- 人权/人格尊严通常优先于财产利益
- 实质平等 > 形式平等（在特殊情形下）

均衡原则：
- 不同价值不可直接比较时，采用个案平衡
- 比例原则：限制手段与追求目的必须相称
```
