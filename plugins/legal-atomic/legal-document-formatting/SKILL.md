# /legal-document-formatting — 法律文书格式适用

## 第一步：判断文书类型

```
判断规则：
- 起诉主体是人民检察院 → 刑事
- 核心争议是是否构成犯罪 → 刑事
- 起诉主体是自然人/法人 → 民事
- 核心争议是民事权利义务 → 民事

若材料不足 → 暂停，要求补充
```

---

## 民事判决书七段式

### A-1. 标题
```
XX省XX市中级人民法院
民事判决书
（2024）X民初字第XXX号
```

### A-2. 首部——当事人与审理经过
```
原告：XXX，男/女，XXXX年XX月XX日出生，...
被告：XXX，...
委托诉讼代理人：XXX，XX律师事务所律师

案件由来：...
审理经过：立案日期、适用程序、开庭方式、出庭人员
```

### A-3. 事实部分

**（1）诉辩意见**
```
原告诉称：...
被告辩称：...
第三人述称：...
```

**（2）举证质证概述**
```
本案当事人围绕诉讼请求依法提交了证据，本院组织当事人进行了证据交换和质证。
```

**（3）证据认定与事实认定**
```
- 无争议证据：直接确认
- 争议证据：调用 evidence-evaluation，逐项评估三性与证明力
- 事实认定：按时间顺序叙述
```

### A-4. 理由部分（"本院认为"）

```
本院认为，[核心法律问题]：

1. 关于[争点一]
   调用 dispute-issue-identification → 确定纠纷性质
   调用 legal-article-retrieval → 援引法律条文

2. 关于[争点二]
   调用 deductive-reasoning → 演绎推理
```

### A-5. 判决主文
```
判决如下：
一、...
二、...
三、...
```

### A-6. 尾部
```
审判长：XXX
审判员：XXX
人民陪审员：XXX
XXXX年XX月XX日
书记员：XXX
```

## 刑事判决书格式

### B-1. 首部
```
XX人民法院
刑事判决书
（2024）X刑初字第XXX号
```

### B-2. 正文
```
公诉机关：XX人民检察院
被告人：...

辩护人：...

本院受理后，依法组成合议庭/独任审判，公开/不公开开庭审理了本案。现已审理终结。
```

### B-3. 事实部分
```
经审理查明：...
```

### B-4. 理由部分
```
本院认为：...
```

### B-5. 判决主文
```
判决如下：
一、被告人XXX犯XX罪，判处...
```
