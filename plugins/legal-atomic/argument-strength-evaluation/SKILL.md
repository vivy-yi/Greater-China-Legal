# /argument-strength-evaluation — 论证强度评估

## 核心概念

```
论证强度 ≠ 结论对错，而是"从前提到结论的支撑力度"
置信度 ≠ 论证强度，而是"AI对自身评估的确信程度"

两者可能不一致：
- 高置信度地判断某论证为"中等强度"（对评估本身有信心，但论证本身有不确定性）
```

### 六维度评估体系

| 维度 | 含义 | 默认权重 |
|---|---|---|
| 法律依据确定性 | 援引规范的明确程度、效力层级、现行有效性 | 25% |
| 事实基础充分性 | 事实是否充分、证据是否确凿 | 25% |
| 逻辑推理严密性 | 推理过程是否有逻辑跳跃或谬误 | 20% |
| 反驳抗辩力 | 能否有效抵御可预见的反对意见 | 15% |
| 司法实践一致性 | 是否与主流司法裁判倾向一致 | 10% |
| 价值判断合理性 | 利益衡量、目的解释的价值取向是否合理 | 5% |

### 薄弱环节类型

| 类型 | 严重程度 | 示例 |
|---|---|---|
| 致命缺陷 | ★★★★★ | 援引已废止法条 |
| 重大弱点 | ★★★★☆ | 关键要件缺乏直接证据 |
| 中度风险 | ★★★☆☆ | 类案裁判不统一 |
| 轻度瑕疵 | ★★☆☆☆ | 论述顺序不佳 |
| 潜在隐患 | ★☆☆☆☆ | 法律修订趋势 |

## 工作流程

### 阶段一：论证解构

```
步骤1：识别论证结构
  1.1 提取核心结论 → 该论证最终要证明什么
  1.2 提取主要前提 → 结论依赖哪些事实/法律前提
  1.3 识别推理规则 → 演绎/归纳/类比/目的/体系
  1.4 识别隐含假设 → 未明示但必须成立的前提
  1.5 绘制论证链条图

步骤2：标记评估节点
  [F] 事实节点 → 评估事实基础
  [L] 法律节点 → 评估法律依据
  [R] 推理节点 → 评估逻辑关系
  [A] 假设节点 → 评估隐含假设
  [V] 价值节点 → 评估价值判断
```

### 阶段二：逐维度评估

#### 维度1：法律依据确定性

```
检查清单：
□ 所援引法律规范是否现行有效？
□ 效力层级是否足够？（宪法>法律>行政法规>规章）
□ 法条含义是否明确？
□ 是否存在特别法/一般法、新法/旧法冲突？
□ 司法解释是否明确支持？
□ 是否存在指导性案例对该法条的权威解释？

评分：5分（明确无争议）→ 1分（缺乏直接依据）
```

#### 维度2：事实基础充分性

```
检查清单：
□ 关键事实是否有充分证据支撑？
□ 证据的证明力如何？（直接证据vs间接证据）
□ 证据链是否完整？
□ 是否存在事实不明或争议？
□ 举证责任分配是否有利？

评分：5分（事实清楚）→ 1分（核心事实缺乏证据）
```

#### 维度3：逻辑推理严密性

```
检查清单：
□ 是否存在逻辑跳跃？
□ 是否存在循环论证？
□ 类比推理中比较对象是否具有实质相似性？
□ 是否混淆充分条件与必要条件？
□ 是否存在滑坡谬误？
□ 三段论的大小前提是否都成立？

评分：5分（严密无瑕疵）→ 1分（根本性逻辑错误）
```

#### 维度4：反驳抗辩力

```
检查清单：
□ 是否已识别所有可预见的反对意见？
□ 对方最强的反驳论点是什么？
□ 论证能否有效回应这些反驳？
□ 是否存在"致命反驳"？
□ 是否考虑了免责事由、抗辩权、时效等？

评分：5分（充分预见并有效回应）→ 1分（存在致命反驳）
```

#### 维度5：司法实践一致性

```
检查清单：
□ 是否有相同或类似问题的裁判案例？
□ 主流裁判倾向是否支持该结论？
□ 最高法/最高检的态度如何？
□ 是否存在同案不同判？
□ 是否有指导性案例或公报案例直接相关？

评分：5分（与主流高度一致）→ 1分（与主流明显矛盾）
```

#### 维度6：价值判断合理性

```
检查清单：
□ 论证是否涉及利益衡量？衡量标准是否合理？
□ 目的解释是否符合立法目的？
□ 价值取向是否符合社会主义核心价值观？
□ 是否兼顾了各方利益的平衡？

评分：5分（价值判断合理）→ 1分（价值判断明显不当）
```

### 阶段三：综合评定

```
加权计算公式：
综合评分 = 法律依据×0.25 + 事实基础×0.25 + 逻辑严密性×0.20
         + 反驳抗辩力×0.15 + 司法实践×0.10 + 价值判断×0.05

注意：存在致命缺陷时，无论综合评分如何，整体评级降至"弱"或"极弱"。
```

### 论证强度等级

| 综合评分 | 等级 | 颜色 |
|---|---|---|
| 4.5-5.0 | 极强 | 🟢深绿 |
| 3.5-4.4 | 强 | 🟢绿 |
| 2.5-3.4 | 中等 | 🟡黄 |
| 1.5-2.4 | 弱 | 🟠橙 |
| 1.0-1.4 | 极弱 | 🔴红 |
