# /legal-concept-comprehension — 法律概念理解

## 三层语义结构

```
生活语义（日常语言中的通常含义）
    ↓  法律化转换
法律语义（法律规范、司法解释、判例中的规范含义）
    ↓  学术深化
教义学语义（法学理论、学说争议中的精细界定）
```

### 示例：\"占有\"

| 语义层次 | 来源 | 含义 |
|---------|------|------|
| 生活语义 | 日常语言 | 手里拿着东西 |
| 法律语义 | 《民法典》物权编 | 对物具有事实上的管领力 |
| 教义学语义 | 法学教科书 | 直接占有/间接占有、自主占有/他主占有 |

## 张力结构

```
        边缘地带（可争议）
      ┌─────────────────┐
      │    中心含义     │
      │  （确定无疑）    │
      └─────────────────┘
        边缘地带（可争议）
```

- **中心含义**：最典型的适用情形，无争议
- **边缘地带**：部分要件匹配，存在解释空间

## 构成要件理论

```
法律概念
├── 构成要件 A
│   └── 事实要素 A1、A2、A3...
├── 构成要件 B
│   └── 事实要素 B1、B2...
├── 构成要件 C
│   └── ...
└── 法律效果（要件全部满足后产生的法律后果）
```

## 工作流程

```
步骤1：识别目标概念
  确定用户想理解/辨析/判断的法律概念

步骤2：定位语义层次
  根据语境判断需要哪个层次的解释

步骤3：拆解构成要件
  将抽象概念转化为可检验的事实要素

步骤4：边界辨析
  区分易混概念，明确适用范围

步骤5：检验法律效果
  要件全部满足后产生什么法律后果

步骤6：结合具体事实（如有）
  判断具体事实是否落入概念范畴
```
