# /evidence-argument-chain — 组织证据论证链

## 核心概念

```
证据论证链 = 主张 → 要件 → 证据 → 证明力

┌─────────────────────────────────────┐
│         最终法律主张                    │
│      （如：被告应赔偿损失）              │
├──────────┬──────────┬──────────┬────┤
│  要件A    │  要件B    │  要件C    │ 要件D│
│ 违约事实   │ 损害结果   │ 因果关系   │归责  │
├──────────┼──────────┼──────────┼────┤
│ 证据A1    │ 证据B1    │ 证据C1    │证据D1│
│ 证据A2    │ 证据B2    │ 证据C2    │待补  │
│ 证明力:强  │ 证明力:强  │ 证明力:中  │证明力弱│
└──────────┴──────────┴──────────┴────┘
```

### 六层结构

```
第一层：总主张
  └── 第二层：子主张/争议焦点
        └── 第三层：法律构成要件
              └── 第四层：待证事实
                    └── 第五层：证据材料
                          └── 第六层：证明力评估
```

---

## 工作流程

### Phase 1：主张梳理与要件分解

```
步骤1：识别全部法律主张
  - 从诉讼请求/抗辩理由出发
  - 区分主要主张与备位主张
  - 明确每个主张的法律基础

  主张编号：C-01
  主张内容：被告应向原告支付违约金50万元
  法律基础：《民法典》第577条、第585条
  主张类型：□主要 / □备位
  举证责任方：原告

步骤2：分解法律构成要件
  - 根据请求权基础规范提取构成要件
  - 区分积极要件与消极要件
  - 标注举证责任分配

  ┌──────┬────────────┬────────┬────────┐
  │要件编号│  要件内容   │举证责任方│  性质  │
  ├──────┼────────────┼────────┼────────┤
  │ E-01 │ 合同有效成立 │   原告  │ 积极要件│
  │ E-02 │ 被告存在违约 │   原告  │ 积极要件│
  │ E-03 │ 违约金条款有效│  原告   │ 积极要件│
  │ E-04 │ 损害实际发生 │   原告  │ 积极要件│
  └──────┴────────────┴────────┴────────┘
```

### Phase 2：证据收集与组织

```
步骤3：收集全部证据材料
  - 按证据类型分类（书证/物证/证人证言/电子数据等）
  - 标注每项证据的基本信息

  证据编号：EV-01
  证据类型：书证
  证据名称：《购销合同》（编号：2023-001）
  证明目的：该证据用于证明合同关系成立...
  来源：[原告/被告]于[日期]提交

步骤4：按争议焦点分组证据
  - 将证据分配到不同争议焦点下
  - 识别证据与主张的对应关系
```

### Phase 3：构建映射关系

```
步骤5：建立要件-证据对应表
  对每个构成要件，列出支撑它的全部证据

  E-02（被告存在违约行为）
    证据EV-01：《购销合同》第5条 → 证明被告应于X日交货
    证据EV-03：快递签收单 → 证明被告实际交货时间为Y日（逾期）
    证据EV-07：微信聊天记录 → 证明原告催告被告履约

步骤6：标注每项证据的证明目的
  - 一项证据可有多个证明目的
  - 避免重复计算
```

### Phase 4：评估与缺口识别

```
步骤7：评估证明力
  - 强：直接证据充分，相互印证
  - 中：间接证据为主，需其他证据补强
  - 弱：孤证或证明力受限

  ┌──────┬────────────┬────────┬──────────┐
  │证据编号│  证据名称   │证明目的 │  证明力  │
  ├──────┼────────────┼────────┼──────────┤
  │EV-01 │ 购销合同   │合同成立 │   强     │
  │EV-03 │ 快递签收单  │逾期事实 │   强     │
  │EV-07 │ 微信记录   │催告事实 │   中     │
  └──────┴────────────┴────────┴──────────┘

步骤8：识别证据缺口
  - 哪些构成要件缺乏充分证据？
  - 缺口是否致命？
  - 是否需要补充证据？
```
