# /legal-interpretation-argument — 法律解释论证

## 触发条件

满足任一即可：
1. 法条文本含义模糊、存在多种理解可能
2. 法条适用于具体案件事实时产生争议
3. 需要论证某一法条应当/不应当适用于特定情形
4. 当事人对同一法条的理解存在分歧
5. 法条之间存在表面冲突，需要通过解释协调
6. 新型案件事实无法直接对应既有法条文本

---

## 解释方法体系

```
第一层级：文义解释
  └── 通常含义/专业术语/语境限定

第二层级：体系解释
  └── 同一法律内部/关联法律之间/上位法合致

第三层级：目的解释
  └── 主观目的（立法者意图）/客观目的（法律规范目的）

第四层级：历史解释
  └── 立法资料/法律沿革

第五层级：社会学解释
  └── 社会效果/社会需要/价值取向
```

---

## 工作流程

### 步骤1：接收解释对象

```
输入：
- 待解释法条
- 具体案件事实/法律问题
- 解释目标

确定解释的焦点问题：
"《X法》第X条中的'Y'在本案中应理解为A还是B？"
```

### 步骤2：文义解释

```
操作：
- 确定法条的通常含义
- 确定法律专业术语的特定含义
- 根据语境限定可能的含义范围

输出：
- 文义的射程范围
- 是否存在多种可能含义
- 是否在射程范围内
```

### 步骤3：体系解释

```
操作：
- 检查相关法条的协调关系
- 消除法条间的表面冲突
- 确定法条在体系中的位置

输出：
- 与其他法条是否协调
- 是否需要选择特定法条
- 体系视角下的解释结论
```

### 步骤4：目的解释

```
操作：
- 识别立法者意图（主观目的）
- 识别法律规范的当下目的（客观目的）
- 根据目的选择最合理的解释

输出：
- 主观目的
- 客观目的
- 目的导向的解释结论
```

### 步骤5：历史解释

```
操作：
- 分析立法资料（草案说明/审议报告）
- 分析法律沿革（旧法→新法）
- 确定历史解释对当前解释的影响

输出：
- 立法意图的历史证据
- 法律变更对解释的影响
```

### 步骤6：综合论证

```
操作：
- 汇总各层级解释结论
- 处理可能的冲突（何层级优先）
- 得出最终可辩护的解释结论

优先规则：
- 文义射程内 → 文义解释优先
- 文义射程外 → 目的解释为主
- 涉及历史变更 → 历史解释补充
```
