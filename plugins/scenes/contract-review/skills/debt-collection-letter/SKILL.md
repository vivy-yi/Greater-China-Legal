# 催款函 / 律师函起草

## 目的

为债权人起草正式的催款函或律师函，用于催促债务人履行付款义务。覆盖：欠款确认、付款期限设定、逾期利息计算、法律后果提示。本技能起草函件文本，不代理诉讼。

## 法域假设

默认中国大陆法域。涉及金融机构的催收须符合监管要求（不得暴力催收、不得骚扰）`[SME 核查]`。涉及刑事诈骗或合同诈骗的（伪造身份、虚假交易），不适用本技能，须升级至刑事辩护相关技能。

## 加载信息

起草前确认：

- **债权人** — 全称
- **债务人** — 全称、联系人、地址
- **欠款金额** — 本金、已逾期天数
- **拖欠事由** — 合同/发票/送货单等依据
- **合同依据** — 合同编号、付款条款
- **逾期利息约定** — 合同约定的利率或法定利率 `[SME 核查]`
- **发送方式** — 邮寄EMS、邮箱、传真

## 工作流程

### 第一步：确认欠款事实

确认以下信息已完整：

| 信息项 | 状态 | 备注 |
|--------|------|------|
| 债权人全称 | 待确认 | 与合同签章一致 |
| 债务人全称 + 地址 | 待确认 | 用于送达 |
| 欠款本金金额 | 待确认 | 须有合同/发票/对账单支撑 |
| 逾期天数 | 待确认 | 自应付日起算 |
| 合同约定的逾期利率 | 待确认 | 有约定从约定；无约定按法定利率 `[SME 核查]` |
| 相关凭证 | 待确认 | 合同/发票/送货单/对账单 |

### 第二步：计算逾期利息

- **有约定利率**：按合同约定计算（不超过合同成立时一年期LPR的4倍为上限）`[SME 核查]`
- **无约定利率**：按起诉时一年期LPR计算（2024年约为3.45%，实际以央行公布为准）`[SME 核查]`
- **注意**：利息总额不超过本金金额（防止利息侵蚀本金原则）`[SME 核查]`

### 第三步：起草催款函

结构：

```
[标题：催款函 / 律师函]

致：[债务人全称]（地址）

事由：[合同/订单编号]项下欠款催告

正文：
1. 合同依据和欠款事实
2. 欠款本金金额 + 逾期利息（截至[日期]）
3. 要求付款期限（通常7-15个工作日）
4. 逾期不付的法律后果（诉讼/仲裁/财产保全）
5. 联系方式

签署：[债权人全称] / [代理律师（如有）]
日期：[YYYY-MM-DD]
```

**语气提示：**
- 催款函语气坚定但不过激，不威胁刑事报案（除非确有合同诈骗依据）
- 律师函使用更正式措辞，可盖律所章

### 第四步：升级决策门

**发送前：** 确认操作人身份。若为非律师用户：

> "催款函一旦发送即产生法律证据效力（中断诉讼时效）。如涉及金额较大或债务人有争议，建议由律师审核后发送。需要我整理一份关键行动摘要吗？"

**涉及以下情形须升级：**
- 债务人已失联或拒绝收货
- 存在多个债权人（清偿顺序问题）
- 债务人已进入破产程序
- 疑似合同诈骗

## 输出格式

```
# 催款函草稿 — [债务人名称]

## 欠款信息
| 要素 | 内容 |
|------|------|
| 债务人 | [___] |
| 债权人 | [___] |
| 欠款本金 | [___]元 |
| 逾期天数 | [___]天 |
| 逾期利息 | [___]元（截至[日期]）|
| 本息合计 | [___]元 |
| 合同依据 | [合同编号/发票编号] |

## 函件草稿
[全文]

## 发送建议
- 方式：EMS邮寄 + 邮箱发送
- 留存：邮寄单据 + 函件副本 + 送达回证
- 下一步：如债务人未在期限内付款，建议律师介入或提起诉讼
```

## 本技能不涵盖

- **代理诉讼或仲裁**
- **刑事报案**（合同诈骗须另行处理）
- **债务人财产调查**（使用 litigation-support 中的 asset-search 相关技能）
- **确认诉讼时效** — 须由律师确认（不同合同类型诉讼时效不同）`[SME 核查]`
