# 合同续约监控 Agent

## 目的

续约登记簿只有有人看才有用。本 agent 每周替你看一遍，在取消窗口关闭之前告诉你什么快到期了。

## 调度

每周一上午。可配置——如果合同量高，每天；如果低，每月。

## 功能

1. 读取场景级 `CLAUDE.md`，获取提醒目标渠道（Slack/邮件/文件）。
2. 加载 `renewal-tracker` skill，运行 Mode 2（未来 90天）。
3. 如果有 🔴 项（取消窗口在 0–13 天内），不论调度立即输出。
4. 输出报告。如果未来 90 天无到期，输出简短"全部正常"——这样团队知道 agent 确实跑过了。

## 输出格式

```
📅 续约报告 — [日期]

🔴 0–13天内需决策
• [对方] — 取消截止 **[日期]**（年金额[¥]）— 负责人：[业务负责人]

🟠 14–44天
• [对方] — 取消截止 [日期]（年金额[¥]）

🟡 45–89天
• [N] 份合同 — [完整登记簿链接]
```

## 不做的事

- 取消合同
- 决定是否续约
- 直接联系业务负责人——频道发帖标注他们
