# Capital Call Calculator

## 技能说明

本Skill用于支持私募股权基金的资金募集计算工作，包括认缴出资管理、实缴资金调配、费用计算等。适用场景包括：GP进行资本账户管理、LP核对出资记录、管理费/业绩报酬计算、基金清算分配计算。

## 核心功能

### 1. 出资计算

| 计算类型 | 计算公式 | 参数说明 |
|----------|----------|----------|
| 认缴出资总额 | Σ各LP认缴金额 | |
| 实缴出资进度 | 实缴金额/认缴总额 | 百分比 |
| 出资催缴金额 | 认缴余额 × 催缴比例 | |
| 逾期违约金 | 逾期金额 × 利率 × 天数 | 日息 |

```
出资计算示例：
LP1认缴：1000万
已实缴：600万
本次催缴：30%
本次应缴 = (1000-600) × 30% = 120万
逾期违约金 = 120万 × 0.05% × 天数
```

### 2. 费用计算

#### 2.1 管理费计算

| 计提方式 | 计算公式 | 常见费率 |
|----------|----------|----------|
| 按认缴 | 认缴总额 × 管理费率 | 2%/年 |
| 按实缴 | 实缴总额 × 管理费率 | 1.5-2%/年 |
| 按投资成本 | 投资成本 × 管理费率 | 1-2%/年 |

**计算参数**：
- 计提基础：认缴出资额
- 年费率：1%-2%
- 计提期间：投资期（通常3-5年）后为管理期

#### 2.2 业绩报酬计算（Carried Interest）

```
瀑布式分配机制：
├── 第一层：返还LP本金（100%给LP）
├── 第二层：返还LP优先收益（8%门槛收益率）
├── 第三层：GP与LP按20:80分配
└── 第四层：超额收益按更高比例分配

计算示例：
门槛收益率：8%
分配比例：GP 20% / LP 80%
```

| 计算项 | 说明 |
|--------|------|
| 门槛收益率 | 通常8%/年 |
| 分配比例 | 通常80/20或75/25 |
| 回拨机制 | GP需返还已收业绩报酬 |

### 3. 收益分配计算

| 分配阶段 | LP份额 | GP份额 | 说明 |
|----------|--------|--------|------|
| 返还本金 | 100% | 0% | 先返还LP本金 |
| 门槛收益 | 100% | 0% | 8%优先收益 |
| 剩余分配 | 80% | 20% | 超额部分 |

### 4. 基金财务模型

| 指标 | 计算方法 | 说明 |
|------|----------|------|
| IRR | 现金流折现 | 内部收益率 |
| MOIC | 期末价值/投入成本 | 投资回报倍数 |
| TVPI | (分配+DVI)/实缴 | 总回报倍数 |
| DPI | 分配/实缴 | 已分配回报 |

## 输入要求

用户需提供：

1. **LPA协议**（含费用和分配条款）
2. **出资认缴表**（各LP认缴/实缴明细）
3. **投资凭证**（投资决策文件、付款凭证）
4. **账务数据**（银行流水、估值报告）
5. **费用规则**（如有特殊约定）

## 本Skill不涵盖

本Skill专注于资本账户计算，**不涵盖**以下内容：

- 基金投资决策（→ 投资委员会）
- 被投企业估值（→ fund-performance-reporter）
- 基金销售合规（→ 合规Skill）
- 税务申报（→ tax-compliance）
- LP法律纠纷（→ lawyer-handoff）
