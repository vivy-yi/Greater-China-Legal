# Fund Formation Checker

## 技能说明

本Skill用于检查私募股权基金/创业投资基金设立过程中的合规性，确保基金设立符合监管要求。适用场景包括：新基金筹备设立、存量基金合规自查、监管检查前预检。

## 核心功能

### 1. 基金管理人资格审查

| 检查项 | 合规标准 | 不合规处置 |
|--------|----------|------------|
| 登记状态 | 已完成私募基金管理人登记 | 不得开展私募业务 |
| 高管资质 | 法定代表人有从业资格 | 整改或更换人员 |
| 内控制度 | 制度完备且实际执行 | 要求补充完善 |

### 2. 基金产品备案核查

- 备案材料完整性检查
- 产品类型认定（股权类/证券类/其他类）
- 投资范围合规性审查
- 托管安排核查

### 3. 募集程序合规性

```
合规检查节点：
├── 特定对象确定程序 ──── 是否完成投资者适当性匹配
├── 风险揭示合规性 ───── 风险揭示书是否完整签署
├── 冷静期确认 ───────── 24小时冷静期是否执行
└── 合格投资者认定 ────── 资产/收入证明是否有效
```

### 4. 组织形式合规

| 组织形式 | 必检文件 | 关键条款 |
|----------|----------|----------|
| 公司型 | 公司章程 | 注册资本≥1000万 |
| 有限合伙型 | 合伙协议 | GP无限连带责任 |
| 契约型 | 基金合同 | 三方权责清晰 |

## 输入要求

用户需提供以下文件进行审查：
1. 基金管理人登记证明（电子版PDF）
2. 基金产品备案证明（中基协备案截图）
3. 基金合同/合伙协议/公司章程（清洁版）
4. 招募说明书（如有）
5. 投资者名单及资质证明

## 本Skill不涵盖

本Skill专注于基金设立合规性检查，**不涵盖**以下内容：

- 基金估值建模与分析（→ fund-performance-reporter）
- 投资条款谈判与设计（→ investment-agreement-reviewer）
- 基金销售合规性（→ compliance-assessor）
- 税务申报与筹划（→ tax-compliance）
- 基金清算程序（→ exit-strategy-advisor）
