# /ship-financing — 船舶融资

## 工作流程
```
Step 1: 评估融资方式(抵押 / 优先权 / 留置)
Step 2: 评估船舶登记 + 抵押登记
Step 3: 评估违约情形
Step 4: 评估处置方案(拍卖 / 变卖)
Step 5: 涉外因素
```

## 船舶融资方式 4 类

| 方式 | 适用 |
|------|------|
| **船舶抵押** | 主要融资方式 |
| **船舶优先权** | 海事请求(海商法第 22 条) |
| **船舶留置** | 造船人 / 修船人 |
| **光船租赁** | 实质融资 |

## 船舶抵押登记(《海商法》第 13 条)

**登记机关:**

| 船旗 | 登记机关 |
|------|----------|
| 中国旗 | 中国海事局 |
| 外国旗 | 中国海事局(中国境内抵押) |

**登记效力:**

```
✅ 登记:对抗第三人
✅ 未登记:仅约束双方
✅ 抵押顺序:按登记时间
```

## 船舶优先权(《海商法》第 22 条)

**5 类优先权(法定):**

| 顺序 | 类别 |
|------|------|
| 1 | 救助报酬 + 共同海损分摊 |
| 2 | 海事请求(船员工资 / 港口费 / 货物损失) |
| 3 | 船舶抵押权(已登记) |
| 4 | 船舶留置权 |

**优先权时效:**

| 类别 | 时效 |
|------|------|
| 优先权 | 1 年(自产生之日起) |

## 违约处置

**违约情形:**

| 情形 | 处理 |
|------|------|
| 逾期还款 | 通知 + 催收 |
| 多次逾期 | 主张加速到期 |
| 严重违约 | 申请实现担保物权 |

**实现担保物权程序:**

```
Step 1: 通知债务人
Step 2: 申请法院拍卖 / 变卖
Step 3: 优先受偿
Step 4: 不足部分继续追偿
```

## 涉外因素

**特别考虑:**

| 情形 | 处理 |
|------|------|
| 船旗国法 | 适用船旗国法 |
| 抵押合同约定 | 准据法 |
| 拍卖地点 | 中国 / 船旗国 |
