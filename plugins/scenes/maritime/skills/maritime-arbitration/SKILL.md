# /maritime-arbitration — 海事仲裁

## 工作流程
```
Step 1: 评估仲裁机构选择(CMAC / LMAA / HKIAC)
Step 2: 评估仲裁条款有效性
Step 3: 评估涉外因素
Step 4: 仲裁程序
Step 5: 裁决承认与执行(纽约公约)
```

## 主要仲裁机构 4 类

| 机构 | 适用 | 特点 |
|------|------|------|
| **CMAC**(中国海事仲裁委员会) | 国内海事 | 国内案件 |
| **LMAA**(伦敦海事仲裁员协会) | 涉外海事 | 国际主流 |
| **HKIAC**(香港国际仲裁中心) | 涉外海事 | 亚洲中心 |
| **SCMA**(上海海事仲裁委员会) | 上海 + 长三角 | 上海区位优势 |

**仲裁机构选择:**

| 因素 | 评估 |
|------|------|
| 合同约定 | ✅ 优先 |
| 当事人国籍 | 双方中国 → CMAC;涉外 → LMAA/HKIAC |
| 标的所在地 | 中国 / 国际 |
| 仲裁成本 | 涉外成本高 |
| 执行便利 | 中国 / 仲裁地 |

## 仲裁条款 5 要件

```
✅ 仲裁机构明确(CMAC / LMAA)
✅ 仲裁规则明确(机构规则)
✅ 仲裁地点明确
✅ 仲裁员人数 / 程序明确
✅ 适用法律明确(实体法)
```

**无效情形:**

```
❌ 仲裁机构不明
❌ 仲裁事项超出法律允许范围
❌ 违反中国专属管辖
```

## 仲裁程序 6 阶段

```
Step 1: 申请 + 受理
Step 2: 仲裁庭组成(3 人 / 1 人)
Step 3: 答辩 + 反请求
Step 4: 证据交换 + 质证
Step 5: 开庭审理
Step 6: 裁决(终局)
```

**时限:**

| 程序 | 时限 |
|------|------|
| CMAC 受理 | 15 日 |
| 答辩 | 收到通知 30 日 |
| 裁决 | 受理后 6 个月内 |
| 可延长 | 3 个月(当事人约定) |

## 涉外因素

**裁决承认与执行(《纽约公约》):**

| 情形 | 处理 |
|------|------|
| 中国 + 缔约国 | 申请中国法院承认 + 执行 |
| 拒绝承认 | 违反公共政策 / 超裁 / 程序不公 |

**特别注意:**

```
✅ 中国 1958 年加入《纽约公约》
✅ 中国 1986 年加入《承认及执行外国仲裁裁决公约》
✅ 涉外海事仲裁裁决跨境执行顺畅
```

## 和解 vs 仲裁

**和解优于仲裁:**

| 情形 | 建议 |
|------|------|
| 双方有意愿 | ✅ 和解 |
| 涉商业关系 | ✅ 和解 |
| 时效紧迫 | ✅ 快速和解 |
| 金额不大 | ✅ 和解 |
| 责任不清晰 | ✅ 仲裁 |
