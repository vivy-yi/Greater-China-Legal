# /general-average-salvage — 共同海损 + 救助

## 工作流程
```
Step 1: 评估共同海损构成 / 救助事实
Step 2: 评估理算机构(中国 / 国际)
Step 3: 评估分摊 / 报酬金额
Step 4: 评估 1 年时效
```

## 共同海损(《海商法》第 193 条)

**构成要件:**

```
✅ 同一海上航程
✅ 船舶 + 货物 + 运费共同危险
✅ 主动 + 合理措施
✅ 特殊牺牲 + 额外费用
```

**典型共同海损:**

| 情形 | 性质 |
|------|------|
| 船舶搁浅 / 触礁 | ✅ 视情况 |
| 弃船 | ✅ |
| 货物被抛入海 | ✅ |
| 救助费用 | 部分 |
| 避难港费用 | 部分 |
| 额外燃料 | 部分 |

## 海难救助(《海商法》第 171-192 条)

**救助报酬构成:**

```
✅ 救助成功(无效果无报酬原则)
✅ 救助金额:获救财产价值 × 适当比例
✅ 不超过获救财产价值
```

**救助金额考虑因素:**

| 因素 | 注意 |
|------|------|
| 救助效果 | 决定报酬 |
| 危险程度 | 影响金额 |
| 救助人努力 | 影响因素 |
| 救助人支出 | 影响因素 |
| 获救财产价值 | 决定上限 |

## 理算机构

| 机构 | 适用 |
|------|------|
| **中国国际海损理算中心(北京)** | 国内案件 |
| **伦敦劳合社理算师** | 涉外案件 |
| **纽约理算师** | 美国案件 |

## 1 年时效

```
✅ 共同海损分摊请求权 1 年(《海商法》第 263 条)
✅ 救助报酬请求权 1 年
```
