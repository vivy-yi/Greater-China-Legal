## 破产原因认定

### 破产原因（《企业破产法》第2条）

```
不能清偿到期债务 + 同时满足以下任一：
1. 资产不足以清偿全部债务
2. 明显缺乏清偿能力
```

### 明显缺乏清偿能力的认定

```
1. 因资金严重不足不能清偿
2. 法定代表人下落不明且无其他人员管理财产
3. 经强制执行无法清偿
4. 长期亏损且经营扭亏困难
5. 导致债务人丧失清偿能力的其他情形
```

## 债权人申请破产

### 申请条件

```
单个债权人：债权总额≥50万元（部分地区差异）
须提交：债权证据 + 债务人不能清偿证据
```

### 债权人会议

```
第一次债权人会议由法院召集（债权申报期满后15日内召开）
债权人会议职权（第61条）：
- 核查债权
- 通过财产管理方案
- 通过变价方案
- 通过分配方案
```

## 破产财产分配顺序

| 顺序 | 项目 | 说明 |
|---|---|---|
| 1 | 破产费用+共益债务 | 诉讼费/管理人报酬/变价费用 |
| 2 | 职工债权 | 工资/社保/补偿金 |
| 3 | 税款 | 欠缴税款 |
| 4 | 普通债权 | 按比例分配 |

### 有担保债权

担保物变价款优先于上述顺序受偿，超出部分纳入破产财产。

## 重整计划

### 重整计划内容

```
1. 债务人经营方案
2. 债权分类
3. 债权调整方案
4. 债权受偿方案
5. 重整计划执行期限
6. 监督期限
```

### 重整计划表决

| 表决组 | 通过条件 |
|---|---|
| 有担保债权组 | 出席会议2/3以上+债权占比2/3以上 |
| 职工债权组 | 同上 |
| 税款债权组 | 同上 |
| 普通债权组 | 同上 |

### 法院强制批准（《企业破产法》第87条）

```
至少有一组表决通过 + 重整计划满足：
- 对未通过组的清偿不低于破产清算时的清偿比例
- 重整计划公平对待同一组债权人
```

| 风险等级 | 条件 | 处理方式 |

| ⚠️ MEDIUM | 普通破产清算/重整方案需调整 | 建议审核 |
| ✅ LOW | 简易破产/债权清晰 | 快速处理 |

## 升级决策门

出现以下情形，立即升级至专业律师：
- 涉及跨境破产（内地与香港/境外互认）
- 上市公司/金融机构破产
- 涉众型债权（人数众多）
- 涉及刑事犯罪（虚假破产/隐匿财产）

### 追问规则（关键）

legal-element-extraction 的输出包含 `## 待补充事实` 节。如果该节非空：

1. **暂停当前分析流程**
2. 向用户逐一提问待补充事实，例如：
   > "请问合同中关于[知识产权归属/数据存储位置/价格调整机制]的条款是什么？这会影响后续判断。"
3. 用户补充后，**回到 Step 0 重新执行 legal-element-extraction**，将新信息并入结构化事实
4. 当待补充事实清空后，继续后续分析

**不得在待补充事实未清空的情况下输出最终结论。** 缺失关键事实的结论标注为「推定结论，须在事实补全后复核」。
