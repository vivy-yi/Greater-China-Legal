## 一、核心判断框架

### 1.1 跨境破产管辖权确定

|| 管辖权连接因素 | 中国法院管辖 | 外国法院管辖 |
|---|---|---|---|
| 注册登记地 | 在中国注册 | 在外国注册 |
| 主要利益中心 | 中国为主要利益中心 | 外国为主要利益中心 |
| 营业地 | 在中国有营业 | 在中国无营业 |
| 财产所在地 | 财产主要在中国 | 财产主要在外国 |

### 1.2 跨境破产主要模式

```跨境破产处理模式：
│
├── 统一破产（Unitary Bankruptcy）
│   ├── 一个破产程序管辖全部资产
│   ├── 依主要利益中心确定管辖
│   └── UNCITRAL模型法采用此模式
│
├── 双重破产（Dual Bankruptcy）
│   ├── 债务人注册地破产程序
│   ├── 资产所在地辅助程序
│   └── 需协调程序之间的关系
│
└── 地域性破产（Territorial Bankruptcy）
    ├── 各资产所在地分别破产
    ├── 程序相互独立
    └── 目前较少采用
```

### 1.3 中国跨境破产法律框架

```
中国跨境破产法律依据：
│
├── 企业破产法
│   ├── 第5条：域外效力（管理人请求承认）
│   └── 第21条：集中管辖（涉跨境破产案件）
│
├── 最高人民法院司法解释
│   ├── 关于审理破产案件工作规范
│   └── 关于跨境破产协助的纪要
│
└── 国际条约
    └── 中国尚未加入跨境破产国际公约
```

## 三、分析流程

### 3.1 跨境破产分析步骤

```
第一步：跨境因素识别 → 识别涉及的境外债权人、资产、营业地
第二步：管辖权分析 → 确定主要利益中心与管辖法院
第三步：程序选择 → 选择在中国或境外启动破产程序
第四步：债权申报指引 → 境外债权如何申报与确认
第五步：资产处置方案 → 跨境资产如何变现与分配
第六步：法律风险评估 → 跨境因素带来的特殊风险
```

### 3.2 境外债权申报流程

```
境外债权申报流程：
│
├── 申报准备
│   ├── 准备债权证明文件（经公证认证）
│   ├── 翻译成中文（如需）
│   └── 委托中国律师代理
│
├── 申报提交
│   ├── 向破产管理人提交申报材料
│   ├── 填写债权申报表格
│   └── 提交授权委托书
│
├── 债权审查
│   ├── 管理人初步审查
│   ├── 债权人会议核查
│   └── 法院裁定确认
│
└── 受偿程序
    └── 按债权顺位参与分配
```

### 3.3 境外破产程序在中国承认

|| 承认条件 | 具体要求 | 拒绝情形 |
|---|---|---|---|
| 互惠关系 | 存在双边司法协助条约或实际互惠 | 无互惠先例 |
| 管辖权适当 | 外国法院具有合格管辖权 | 管辖权瑕疵 |
| 程序正当 | 破产程序合法进行 | 程序明显不公 |
| 不违反中国公共秩序 | 不损害中国债权人利益 | 损害公共利益 |
| 真实债权 | 债权真实有效 | 虚假债权 |

## 升级决策门

以下情形应向上级 Skill 升级：

- 涉及外国破产程序在中国承认与执行（正式司法协助请求）
- 涉及中国破产程序在境外承认（需通过外交途径）
- 涉及重大跨境破产案件协调（涉及多国法院）
- 涉及外汇管制与跨境资金转移（需外汇管理部门批准）
- 涉及境外资产的变现在地法律（需当地律师介入）
- 涉及国家豁免问题（国有企业跨境破产）

## 本Skill不涵盖

- 境外破产程序的启动与推进（由当地律师处理）
- 跨境破产诉讼代理（需跨境诉讼 skill）
- 外国法院判决的承认与执行（需国际司法协助程序）
- 境外资产的税务规划（需国际税务 skill）
- 跨境破产中的劳动法问题（涉及外国雇员）
- 反垄断审查与安全审查（跨境并购相关）
## 本 Skill 不涵盖

- 外国破产程序的代理
- 跨境破产判决的相互承认诉讼
- 外国债权人会议的参加程序
- 跨境资产追索的执行程序
