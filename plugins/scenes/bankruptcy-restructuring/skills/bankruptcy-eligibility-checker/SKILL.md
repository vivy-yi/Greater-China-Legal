## 一、核心判断框架

### 1.1 破产原因类型

|| 破产原因 | 适用程序 | 核心判断标准 |
|---|---|---|---|
| 不能清偿到期债务 | 清算/重整/和解 | 债务履行期限届满且未清偿 |
| 资产不足清偿全部债务 | 清算/重整 | 资产价值低于负债总额 |
| 明显缺乏清偿能力 | 重整/和解 | 财产不足以清偿全部债务或资不抵债 |

### 1.2 破产申请主体

```申请主体资格：
│
├── 债务人（企业法人）
│   ├── 必须具备法人资格
│   ├── 不能清偿到期债务
│   └── 明显缺乏清偿能力
│
├── 债权人
│   ├── 须为到期债权人
│   ├── 债务人不能清偿到期债务
│   └── 无财产担保或财产担保不足
│
└── 清算组（清算程序中）
    └── 清算组在清算程序中发现资不抵债
```

### 1.3 程序选择判断

```
破产程序选择决策树：
│
├── 债务人是否已经停止清偿到期债务？
│   ├── 是 → 进入破产原因判断
│   └── 否 → 暂不满足破产申请条件
│
├── 债务人资产是否大于负债？
│   ├── 是 → 审查"明显缺乏清偿能力"
│   └── 否 → 满足"资产不足清偿"要件
│
├── 是否有重整价值？
│   ├── 是 → 优先考虑重整程序
│   └── 否 → 优先考虑清算程序
│
└── 债权人/债务人态度？
    ├── 债务人申请 → 可选择清算/重整/和解
    └── 债权人申请 → 只能选择清算
```

## 二、数据来源

### 2.1 内部文件

|| 文件 | 核查内容 |
|---|---|
| 财务报表（资产负债表、利润表） | 资产状况、负债状况、偿债能力 |
| 银行对账单 | 现金流状况、账户冻结情况 |
| 债务清单 | 债权人名单、债务金额、到期时间 |
| 资产清单 | 不动产、动产、应收账款、知识产权等 |
| 诉讼/仲裁案件清单 | 或有负债、争议债务 |
| 劳动合同清单 | 职工债权金额 |

### 2.2 外部核查路径

|| 数据源 | 查询内容 |
|---|---|
| 国家企业信用信息公示系统 | 企业登记信息、股东信息、经营状态 |
| 中国裁判文书网 | 未决诉讼、执行案件 |
| 人民法院诉讼资产网 | 被执行财产情况 |
| 工商登记档案 | 企业设立、变更、注销记录 |
| 税务部门 | 欠税情况、税务稽查状态 |

## 三、分析流程

### 3.1 Eligibility 核查步骤

```
第一步：主体资格审查 → 是否为企业法人？是否具备申请资格？
第二步：破产原因核查 → 满足何种破产原因（不能清偿/资不抵债/明显缺乏清偿能力）
第三步：申请类型确定 → 清算/重整/和解哪种程序更合适
第四步：材料完备性检查 → 申请材料是否充分
第五步：风险评估 → 申请可能面临的障碍与风险
```

### 3.2 财务指标分析

|| 指标 | 计算方式 | 破产预警线 |
|---|---|---|
| 资产负债率 | 负债总额/资产总额×100% | >100% |
| 流动比率 | 流动资产/流动负债×100% | <100% |
| 速动比率 | （流动资产-存货）/流动负债×100% | <50% |
| 到期债务偿还率 | 已偿还到期债务/到期债务总额×100% | <30% |

### 3.3 重整价值评估因素

- 核心资产是否存在运营价值
- 产业前景与市场地位
- 职工安置可行性
- 债权人受偿比例预期
- 是否有战略投资人意向

## 升级决策门

以下情形应向上级 Skill 升级：

- 涉及上市公司破产申请（需遵守上市公司监管要求）
- 涉及国有企业破产（需履行国有资产监管程序）
- 涉及跨境破产或外国债权人利益保护
- 涉及重大争议债务或诉讼（需律师评估诉讼风险）
- 重整程序中引入战略投资人（需 lawyer-negotiator 支持）
- 涉及职工安置人数超过500人（需HR顾问介入）

## 本Skill不涵盖

- 破产申请材料的准备与提交（由破产管理人或代理律师处理）
- 破产债权确认诉讼（需诉讼类 skill 支持）
- 破产程序中的具体法律救济（复议、撤销等）
- 破产重整计划的具体条款设计（见 restructuring-plan-reviewer）
- 跨境破产承认与执行（见 cross-border-insolvency-advisor）
## 本 Skill 不涵盖

- 破产申请的具体格式文本（需律师起草）
- 债权人会议的召集与主持
- 破产程序的司法审查
- 国有企业破产的特殊审批程序
