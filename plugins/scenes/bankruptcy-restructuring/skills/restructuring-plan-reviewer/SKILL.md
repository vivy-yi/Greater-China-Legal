## 一、核心判断框架

### 1.1 重整计划法定要素

|| 要素 | 法律要求 | 审查要点 |
|---|---|---|---|
| 债权分类 | 企业破产法第82条 | 担保债权/职工债权/税务债权/普通债权分组 |
| 债权调整方案 | 各类债权清偿比例 | 清偿比例不得低于清算清偿率 |
| 受偿期限 | 不超过重整期限 | 分期偿付安排合理性 |
| 债务人经营方案 | 可行性分析 | 现金流预测、扭亏为盈措施 |
| 出资人权益调整 | 股东权益调整 | 稀释或转让股权是否合理 |
| 重整期限 | 不超过6个月（可延长） | 期限是否足够完成重整 |

### 1.2 表决通过条件

```表决机制：
│
├── 担保债权组
│   └── 需1/2以上债权人同意，且债权额超过2/3
│
├── 职工债权组
│   └── 需1/2以上债权人同意，且债权额超过2/3
│
├── 税务债权组
│   └── 需1/2以上债权人同意，且债权额超过2/3
│
├── 普通债权组
│   └── 需1/2以上债权人同意，且债权额超过2/3
│
└── 出资人组（如涉及出资人权益调整）
    └── 需2/3以上出资人同意
```

### 1.3 可行性判断标准

```
重整计划可行性评估框架：
│
├── 经营方案可行性
│   ├── 市场分析合理性
│   ├── 现金流预测可靠性
│   └── 成本控制措施有效性
│
├── 偿债能力分析
│   ├── 正常经营现金流
│   ├── 资产处置变现计划
│   └── 外部融资安排
│
├── 外部支持因素
│   ├── 地方政府支持（如有）
│   ├── 战略投资人意向
│   └── 行业周期判断
│
└── 主要风险因素
    ├── 市场风险
    ├── 融资风险
    └── 执行风险
```

## 三、分析流程

### 3.1 重整计划审查步骤

```
第一步：形式审查 → 法定要素是否完备，格式是否规范
第二步：实体合法性审查 → 是否符合企业破产法规定
第三步：债权分类审查 → 分类是否正确，清偿比例是否合理
第四步：表决可行性分析 → 各组是否能通过，障碍在哪
第五步：执行可行性评估 → 经营方案、偿债方案是否可行
第六步：风险提示 → 列出主要风险点及建议
```

### 3.2 债权清偿标准审查

|| 债权类型 | 最低清偿标准 | 审查要点 |
|---|---|---|---|
| 担保债权 | 担保物评估价值全额受偿 | 评估方法是否合理 |
| 职工债权 | 100%清偿 | 是否有资金来源保障 |
| 税务债权 | 全额清偿 | 滞纳金部分如何处理 |
| 普通债权 | 不低于清算清偿率 | 清算清偿率如何计算 |

### 3.3 重整计划变更条件

- 债权总额变动超过20%
- 担保物价值发生重大变化
- 经营方案因市场变化需重大调整
- 引入新投资人导致出资人权益重大变化
- 重整期限变更

## 升级决策门

以下情形应向上级 Skill 升级：

- 涉及重整计划强制批准（法院强制批准情况复杂）
- 涉及上市公司重整（信息披露、证监会审批）
- 涉及跨境资产或外国债权人（法律适用复杂）
- 涉及出资人权益重大调整（需股东会决议）
- 涉及重大争议债权（影响表决组构成）
- 重整计划执行发生重大变化（需变更程序）

## 本Skill不涵盖

- 重整计划的谈判策略（需 lawyer-negotiator 支持）
- 债权人会议组织与表决程序
- 重整计划草案的具体条款起草
- 战略投资人的引入与谈判
- 重整计划执行期间的日常经营管理
- 重整程序转清算程序的操作
## 本 Skill 不涵盖

- 重整计划的起草（需专业律师）
- 债权人会议的表决代理
- 重整计划执行中的诉讼
- 上市公司重整的特殊审批程序
