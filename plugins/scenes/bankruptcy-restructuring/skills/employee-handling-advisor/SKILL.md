# /employee-handling-advisor — 职工安置顾问

## 核心概念

职工安置是破产程序中最具敏感性也最能体现法律执行力的环节之一。职工债权在《企业破产法》第113条中位列第二顺位（仅次于破产费用和共益债务），优先于税款和普通债权。职工债权具体包括：欠付工资（含奖金和津贴）、社保费用（基本养老保险和医疗保险的单位欠缴部分）、经济补偿金（N+1、2N等）、以及因工伤/工亡产生的抚恤费用。

劳动关系的解除方式取决于破产程序类型。在破产清算中，管理人接管后可与职工解除劳动关系——此属于《劳动合同法》第44条第4款规定的"用人单位被宣告破产"情形，劳动合同自动终止，无需额外协商。在破产重整中，如企业继续经营，可能保留部分或全部职工（第44条第5款"用人单位决定提前解散"不适用），但可根据重整需要依法裁员（《劳动合同法》第41条的经济性裁员程序）。

## 工作流程

### 第一步：职工信息核查与债权预计算

- 收集全部职工名册：姓名、入职日期、岗位、最近12个月平均工资、合同类型（无固定/固定期限/劳务派遣/临时用工）
- 核查欠付工资明细：最后一个月正常工作期间的工资、加班费、奖金（合同约定部分）
- 核查社保缴纳状态：向社保经办机构发函查询欠缴明细；区分单位应缴部分（属于职工债权）与个人应缴部分（从工资中代扣但未代缴的，实质属于职工债权）
- 预计算每位职工的经济补偿金：
  - 一般规定：每满一年支付一个月工资（N）；六个月以上不满一年的按一年算；不满六个月支付半个月工资
  - 高薪封顶：月工资高于当地上年度职工月平均工资三倍的，按三倍计算，补偿年限不超过12年
  - 违法解除：如适用2N标准（前提是破产前存在违法解除情形）

### 第二步：与工会/职工代表的沟通与提前预案

- 根据《企业破产法》第8条，债务人申请破产时应当提交职工安置预案
- 召开职工代表大会或职工代表座谈会，通报破产程序的进展和职工安置的原则
- 明确告知职工债权的法定期限和金额——职工债权不要求职工主动申报（管理人主动调查确认），但需职工配合提供相关材料
- 对离职意向做排摸，识别可能闹访/诉讼的高风险职工群体
- 制定《群体性事件应急预案》报当地劳动监察部门和街道维稳机构备案

### 第三步：劳动关系解除手续办理

- 准备《解除劳动合同通知书》，注明法律依据（《劳动合同法》第44条第4款或第41条）
- 对破产清算的情形，向全体职工统一出具通知书，载明最后工作日和补偿金支付安排
- 对破产重整的情形，依法执行经济性裁员程序：提前30日向工会或全体职工说明情况、听取意见，向劳动行政部门报告
- 确定解除日期——经济补偿金的计算截止日期为职工劳动关系实际解除之日

### 第四条：社保和档案转移

- 向社保经办机构办理减员手续，转移职工的社保关系
- 协助职工办理失业保险金申领（非因本人意愿中断就业）
- 移交职工档案至当地公共就业服务机构或职工个人（可选择）
- 如职工在破产前发生工伤但未评残，协调工伤认定部门加快认定程序

### 第五步：分配执行与争议处理

- 根据职工债权确认金额，从破产财产中按法定顺位优先分配
- 对职工债权有异议（工资标准/经济补偿年限/社保基数存在分歧）的，管理人出具《职工债权审查结果通知书》，告知复议或诉讼权利
- 对可能提起的劳动争议仲裁，准备相关证据材料（考勤记录、工资单、社保记录）
- 对群体性劳动争议，联系法律援助机构为职工提供法律帮助

## 升级决策门

出现以下情形须移交专业劳动法律师/破产律师处理：
- 涉及集体合同或工会集体行动（罢工/示威/信访）
- 高管/核心技术人员补偿金争议金额重大（单个超过当地均工资3倍）
- 存在工伤认定未完成或职业病诊断有争议的职工
- 欠薪涉及建设工程领域"农民工工资"优先权与破产顺位的衔接
- 职工人数超过500人，涉及重大社会稳定风险
- 职工在破产受理前已对债务人提起劳动争议仲裁或诉讼且尚未结案
