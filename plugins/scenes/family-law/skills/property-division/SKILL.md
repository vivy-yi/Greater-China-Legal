# /property-division — 夫妻财产分割

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B2 路由表 + § A6 决策姿态(评估值优先)+ § B11 财产类型注册表。

## 工作流程

```
Step 1: 梳理全部财产(共同 + 个人)
Step 2: 认定每项财产性质(共同 / 个人)
Step 3: 评估财产价值(评估机构)
Step 4: 制定分割方案
Step 5: 协议约定 / 法院判决
Step 6: 协助执行(过户 / 支付)
```

## 夫妻财产性质认定(《民法典》1062-1063 条)

### 共同财产(婚后所得,默认共同)

| 类型 | 注意 |
|------|------|
| 工资 / 奖金 / 津贴 | 全部共同 |
| 经营收益 | 一方经营共同财产收益 |
| 投资收益 | 共同财产投资 |
| 知识产权收益 | 婚后取得 |
| 继承 / 受赠财产 | **例外:** 明确只给一方的除外 |
| 其他应当归共同所有的 | 兜底条款 |

### 个人财产(专属,不可分割)

| 类型 | 注意 |
|------|------|
| 婚前财产 | 婚前取得 |
| 人身损害赔偿 | 残疾赔偿金 / 精神损害抚慰金 |
| 遗嘱 / 赠与合同中明确只归一方 | 形式要件明确 |
| 专用生活用品 | 个人衣物等 |
| 其他应当归一方的 | 兜底条款 |

### 特殊财产认定

| 财产 | 认定 |
|------|------|
| **婚前买房,婚后还贷** | 房子归登记方,但**婚后还贷部分 + 增值部分**为共同财产,另一方有权分割 |
| **父母出资买房** | 婚前出资 → 个人财产 / 婚后出资 → 默认共同(除非明确赠与一方) |
| **婚后买的车** | 共同财产 |
| **公司股权** | 看股权来源 + 是否实际参与经营 |
| **婚前股权,婚后增值** | 股权本身个人,增值部分共同(复杂) |
| **数字资产(比特币/NFT)** | 司法实践:按财产类型处理(共同 / 个人) |

## 分割原则

**基本原则:**

```
1. 协议优先(双方合意)
2. 平均分割(共同财产)
3. 照顾子女 / 女方 / 无过错方
4. 生产 / 生活 / 照顾子女需要
```

**特殊调整:**

| 情形 | 调整 |
|------|------|
| 一方隐藏 / 转移 / 变卖 / 毁损共同财产 | 可**少分或不分** |
| 一方伪造债务企图侵占另一方财产 | 可**少分或不分** |
| 离婚后发现上述行为 | 可**再次起诉分割** |

## 分割方案 4 类

| 方案 | 适用 | 注意 |
|------|------|------|
| **实物分割** | 可分物(动产) | 直接分配 |
| **折价补偿** | 不可分物(如房产) | 一方取得,补偿另一方一半 |
| **拍卖分割** | 双方都不要 | 拍卖后分钱 |
| **变价分割** | 股权等 | 转让 / 减资后再分 |

## 涉房产分割(高频)

**情形 1: 婚后购买,登记在一方名下**

```
✅ 共同财产,即使登记一方名下
✅ 原则上平均分割
✅ 考虑出资比例(若有父母出资)
```

**情形 2: 婚前一方首付,婚后共同还贷**

```
✅ 房子归登记方
✅ 共同还贷部分(本金 + 利息)+ 对应增值部分 → 另一方有权补偿
✅ 计算公式:
   补偿金额 = (共同还贷本息 ÷ 房屋总价款) × 房屋现值 ÷ 2
```

**情形 3: 父母全款买房**

| 时点 | 登记 | 性质 |
|------|------|------|
| 婚前 | 一方子女 | 该方个人财产 |
| 婚前 | 双方 | 共同财产(按份共有) |
| 婚后 | 一方子女 | 默认共同(除非明确赠与一方) |
| 婚后 | 双方 | 共同财产 |

## 涉股权分割(复杂)

**关键认定:**

```
1. 股权来源(婚前 / 婚后取得)
2. 是否实际参与经营(分红 / 工资)
3. 是否有公司其他股东同意转让
```

**分割方式:**

| 方式 | 适用 |
|------|------|
| 股权直接分割 | 有限公司其他股东过半数同意 |
| 折价补偿 | 公司不同意外人加入 |
| 拍卖 / 变卖 | 双方都不要 |

## 涉外财产分割

**关键问题:**

```
1. 财产所在地(影响管辖)
2. 财产性质认定(适用法律)
3. 外国法院判决在中国的执行
```

**实务:** 中国法院 + 中国法律(双方均为中国国籍 + 财产在中国境内)为最常见。
