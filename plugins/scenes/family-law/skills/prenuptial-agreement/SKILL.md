# /prenuptial-agreement — 婚姻协议起草

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B6 协议模板 + § B13 公证要求 + § A7-14 不得协助虚假协议。

## 工作流程

```
Step 1: 确认协议类型(婚前 / 婚内 / 离婚)
Step 2: 双方财产梳理 + 意愿沟通
Step 3: 起草协议(套模板 + 个性化)
Step 4: 律师审查 + 修改
Step 5: 公证(必要时)
Step 6: 签字 + 见证 + 备案
```

## 协议 3 类

| 类型 | 时点 | 法条 |
|------|------|------|
| **婚前财产协议** | 结婚登记前 | 《民法典》第 1065 条 |
| **婚内财产协议** | 婚姻存续期间 | 《民法典》第 1065 条 |
| **离婚协议** | 离婚时(协议离婚) | 《民法典》第 1076 条 |

## 协议形式要件

### 形式

| 类型 | 形式 |
|------|------|
| 婚前 / 婚内协议 | **书面**(必须) |
| 离婚协议 | 书面(协议离婚必备) |
| 口头约定 | **无效**(财产约定) |

### 必备条款

**婚前 / 婚内财产协议必备:**

```
1. 双方基本信息(姓名 / 身份证号 / 住所)
2. 协议时点(婚前 / 婚内)
3. 婚前财产明细(双方各自)
4. 婚后财产约定:
   □ 约定共同财产(默认全部共同)
   □ 约定个人财产(部分约定为个人)
   □ 约定部分共同 / 部分个人
5. 债务约定(婚前 / 婚后个人债务 / 共同债务)
6. 投资收益归属(约定 vs 法定)
7. 知识产权收益归属
8. 继承 / 受赠财产归属(明确 / 默认)
9. 协议生效条件
10. 协议变更 / 解除
11. 争议解决(仲裁 / 法院)
12. 双方签字 + 日期
```

**离婚协议必备(《民法典》1076 条):**

```
1. 双方基本信息
2. 婚姻登记日期
3. 离婚原因(简述)
4. **是否自愿**(必备)
5. 子女安排:
   - 抚养权(明确归谁)
   - 抚养费(金额 + 支付方式 + 调整条件)
   - 探视权(频率 + 地点 + 中止条件)
6. 财产分割(全部共同财产清单 + 归属)
7. 债务分担(清单 + 分担)
8. 经济帮助(若有)
9. 损害赔偿(若有)
10. 协议生效(双方签字 + 日期)
11. 备案机构(婚姻登记机关)
```

## 协议核心规则

### 意思自治原则

```
✅ 双方完全自愿
✅ 内容合法
✅ 形式合规
✅ 不违反公序良俗
```

**无效情形:**

| 情形 | 无效 |
|------|------|
| 一方欺诈 / 胁迫 | 协议可撤销 |
| 恶意串通规避债务 | 无效 |
| 违反法律强制性规定 | 无效 |
| 显失公平(离婚时) | 可撤销 |
| 处分他人财产 | 该部分无效 |

### 公平原则(离婚协议特别重要)

**司法审查重点:**

| 审查点 | 风险 |
|--------|------|
| 一方放弃全部财产 | 可能被认定显失公平 |
| 一方放弃抚养权 / 探视权 | 可能被认定显失公平 |
| 子女抚养费过低 | 法院可调整 |
| 一方完全承担债务 | 可能被认定逃避债务 |

**特别注意:**

```
⚠️ 离婚协议中"放弃一切权利"条款 — 司法实践不一,部分法院支持,部分认为可撤销
⚠️ 离婚后发现对方隐瞒财产 — 可另行起诉分割
⚠️ 离婚后不履行协议 — 可起诉强制履行
```

## 特殊协议设计

### 1. 涉及公司股权

```
若婚前一方持有公司股权:
  - 约定股权仍归一方(个人财产)
  - 但婚后股权增值 / 分红约定归属
  - 若一方参与经营,工资归个人 / 公司

若婚后取得股权:
  - 约定股权归属(双方共有 / 一方所有)
  - 其他股东过半数同意是分割前提
```

### 2. 涉及房产

```
婚前一方买房:
  - 约定房产仍归一方(个人财产)
  - 婚后共同还贷部分归属(可约定)

婚后买房:
  - 约定登记方 / 共同共有
  - 出资比例约定

父母出资:
  - 明确赠与一方还是双方
  - 形式要件(书面赠与合同)
```

### 3. 涉及数字资产

```
婚前持有: 个人财产(若明确)
婚后取得: 共同财产(默认)
特殊约定: 明确归属
```

### 4. 涉外协议

```
特别考虑:
  - 适用法律(中国法 / 外国法)
  - 公证认证
  - 外国法院的承认
```

## 公证要求(§ B13)

**必须公证的情形:**

| 情形 | 注意 |
|------|------|
| **房产分割** | 离婚协议涉及房产过户,公证是过户前提 |
| **重大财产约定** | 建议公证(防止纠纷) |
| **涉外协议** | 涉外协议通常需公证认证 |
| **婚前重大财产协议** | 建议公证(双方出资差异大时) |

## 协议变更 / 解除

| 情形 | 处理 |
|------|------|
| 双方协商一致变更 | 重新签订书面协议 |
| 一方欺诈 / 胁迫 | 受欺诈 / 胁迫方可撤销(1 年内) |
| 显失公平(离婚) | 受不利方可撤销 |
| 违反法律强制性规定 | 无效 |

## 律师审查要点

```
□ 双方意思表示真实自愿
□ 内容合法(不违反强制性规定)
□ 形式合规(书面 + 签字)
□ 财产清单完整
□ 子女 / 抚养费条款合理
□ 债务分担明确
□ 生效条件清晰
□ 争议解决条款
□ 公证 / 见证是否需要
□ 涉外条款(如适用)
```
