# /carbon-trading — 碳交易

## 工作流程
```
Step 1: 评估碳配额分配
Step 2: 评估 CCER 抵消
Step 3: 评估交易方式
Step 4: 评估合规义务
Step 5: 跟踪履约
```

## 碳市场 2 类(全国 + 自愿)

| 市场 | 适用 |
|------|------|
| **全国碳排放权交易市场(强制)** | 重点排放单位 |
| **自愿减排市场(CCER)** | 抵消 + 自愿减排 |

**重点排放单位:**

| 行业 | 时点 |
|------|------|
| **发电** | ✅ 首批纳入(2017) |
| **钢铁 / 水泥 / 电解铝** | ✅ 2024 起逐步纳入 |
| **化工 / 石化 / 造纸 / 航空** | 评估中 |

## 碳配额分配

**分配方式:**

| 方式 | 适用 |
|------|------|
| **免费分配(基准法 / 历史强度法)** | ✅ 现阶段 |
| **有偿分配** | 未来逐步 |

**履约义务:**

```
✅ 每年 3-5 月 履约
✅ 上一年度配额 + 实际排放量
✅ 不足部分:购买配额 / CCER
```

## CCER(国家核证自愿减排量)

**CCER 适用:**

| 情形 | 适用 |
|------|------|
| **5% 抵消限制** | 重点排放单位履约 |
| 林业碳汇 | ✅ |
| 可再生能源 | ✅ |
| 甲烷利用 | ✅ |
| 节能项目 | ✅ |

**抵消规则:**

```
✅ 1 单位 CCER = 1 吨 CO2 当量
✅ 抵消不超过 5% 履约量
✅ 须经过审定 + 核证 + 签发
```

## 交易方式

| 方式 | 适用 |
|------|------|
| **公开交易(挂牌 / 协议)** | 全国碳市场 |
| **CCER 抵消** | 履约 |

**参与主体:**

| 主体 | 条件 |
|------|------|
| **重点排放单位** | 强制履约 |
| **其他组织** | 符合条件可参与 |

## 合规义务

| 义务 | 注意 |
|------|------|
| **年度排放报告** | 2024-2025 履约周期 |
| **第三方核查** | 必须 |
| **配额清缴** | 5% 抵消限制 |
| **信息公开** | 重点排放单位 |

**未履约后果:**

| 情形 | 处罚 |
|------|------|
| 未按时清缴 | ¥2万-3万 |
| 虚报 / 瞒报排放 | ¥1万-3万 |
| 拒绝 / 阻碍核查 | 行政处罚 |

## 全国碳市场(2024-2025 履约周期)

**基本要素:**

| 要素 | 注意 |
|------|------|
| 主管 | 生态环境部 |
| 登记 | 中国登记结算有限责任公司(中碳登) |
| 交易 | 上海环境能源交易所 |
| 价格 | 80-100 元/吨(2024) |
