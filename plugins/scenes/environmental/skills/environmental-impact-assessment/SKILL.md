# /environmental-impact-assessment — 环境影响评价

## 工作流程
```
Step 1: 评估是否须环评
Step 2: 编制环评报告
Step 3: 公众参与
Step 4: 报批
Step 5: 评估后续监管
```

## 须环评的情形(《环评法》第 16 条)

| 项目类型 | 环评要求 |
|----------|----------|
| 工业建设 | 报告书 |
| 能源 / 水利 / 交通 | 报告书 |
| 城市基础设施 | 报告书 |
| 社会事业 | 报告表 |
| 农业小型项目 | 登记表 |

**3 类分类:**

| 分类 | 适用 |
|------|------|
| **报告书** | 影响重大 / 复杂 |
| **报告表** | 影响中度 |
| **登记表** | 影响轻微 |

## 环评报告内容(《环评法》第 17 条)

| 内容 | 注意 |
|------|------|
| 建设项目概况 | 项目内容 / 规模 / 地点 |
| 周围环境现状 | 调查 / 监测 |
| 环境影响预测 | 模拟 / 类比 |
| 环境保护措施 | 预防 / 减缓 / 修复 |
| 评价结论 | 总体评估 |

## 公众参与(《环评法》第 21 条)

**参与方式:**

| 方式 | 适用 |
|------|------|
| **信息公示** | 报告书征求意见稿 |
| **公众调查** | 问卷 / 座谈会 |
| **专家论证** | 重大 + 复杂项目 |

**特别项目:**

| 项目 | 强化参与 |
|------|----------|
| 核设施 | ✅ 强化 |
| 危险化学品 | ✅ 强化 |
| 大型基础设施 | ✅ 部分 |
| 涉敏感人群 | ✅ 强化 |

## 报批程序

```
Step 1: 建设单位编制环评文件
Step 2: 行业主管部门预审
Step 3: 报有审批权的生态环境部门
Step 4: 受理 + 审查
Step 5: 批复(30 日内)
Step 6: 建设过程中监督
```

## 违法责任

| 违法 | 后果 |
|------|------|
| 未批先建 | 责令停止建设 + 罚款 |
| 虚假环评 | 行政处罚 + 报告人失信 |
| 重大变化未重新报批 | 责令改正 + 罚款 |
| 公众参与不充分 | 报告书退回 |

**罚款:**

| 情形 | 罚款 |
|------|------|
| 报告书项目未批先建 | ¥5万-20万 + 设备投资额 1%-5% |
| 报告表 / 登记表项目 | ¥2万-10万 |
