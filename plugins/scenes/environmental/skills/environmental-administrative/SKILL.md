# /environmental-administrative — 环境行政处罚

## 工作流程
```
Step 1: 评估违法行为
Step 2: 评估处罚依据(法律 / 标准)
Step 3: 评估程序合法性
Step 4: 申请听证
Step 5: 复议 / 诉讼
```

## 处罚类型 5 类

| 处罚 | 适用 |
|------|------|
| **罚款** | 1 万-100 万(部分无上限) |
| **责令停产 / 停业** | 严重污染 |
| **查封 / 扣押** | 设施 / 物品 |
| **吊销许可证** | 严重违法 |
| **行政拘留** | 拒绝 / 阻碍执法 |

**特别处罚:**

| 处罚 | 适用 |
|------|------|
| 按日连续处罚 | 拒不改正 |
| 查封 / 扣押 | 设施 / 物品 |

## 听证程序(《行政处罚法》)

**听证条件:**

```
✅ 责令停产 / 停业
✅ 吊销许可证
✅ 较大数额罚款
✅ 其他重大处罚
```

**听证申请:**

| 事项 | 时限 |
|------|------|
| 收到告知书 | 5 日内申请 |
| 听证时间 | 申请后 7 日内 |

## 复议 / 诉讼

**救济路径:**

```
处罚决定 → 60 日内申请复议
         → 6 个月内行政诉讼
```

**特别注意:**

```
⚠️ 复议机关:上一级主管部门 / 人民政府
⚠️ 诉讼管辖:中级法院
⚠️ 处罚不停止执行(原则上)
⚠️ 申请停止执行须法院裁定
```

## 抗辩理由

| 抗辩 | 适用 |
|------|------|
| 程序违法 | 告知 / 听证 / 送达 |
| 证据不足 | 监测 / 鉴定 |
| 法律适用错误 | 条款引用 |
| 超越职权 | 处罚机关 |
| 显失公正 | 处罚幅度 |
