# /esg-compliance — ESG 合规

## 工作流程
```
Step 1: 评估 ESG 适用(上市公司 / 拟上市 / 涉外经营)
Step 2: 评估披露义务(年度 / 临时)
Step 3: 评估 ESG 评级风险
Step 4: 评估 greenwashing 风险
Step 5: 准备披露文件 + 内部合规
```

## ESG 3 大支柱

| 支柱 | 注意 |
|------|------|
| **E (Environment)** | 碳排放 / 资源使用 / 污染 |
| **S (Social)** | 员工 / 供应链 / 人权 |
| **G (Governance)** | 反腐 / 董事会 / 风险 |

## 中国 ESG 披露 4 大规则

| 规则 | 适用 |
|------|------|
| **《上市公司可持续发展报告指引》(2024 上交所 / 深交所)** | 沪深上市公司 + 强制 |
| **《企业可持续披露准则》** | 重点行业(财险 + 大型) |
| **《央企控股上市公司 ESG 专项报告》** | 央企 |
| **《金融机构 ESG 披露指南》** | 金融机构 |

## 披露内容 5 类

| 内容 | 注意 |
|------|------|
| 治理(Governance) | 董事会监督 / ESG 委员会 |
| 战略(Strategy) | ESG 风险 / 机会 |
| 影响 / 风险 / 机会管理 | 评估方法 |
| 指标 / 目标 | 量化指标 + 减排目标 |
| 第三方鉴证 | 独立机构验证 |

## greenwashing 风险防范

**greenwashing 5 类行为(2024 严打):**

| 行为 | 风险 |
|------|------|
| **虚假披露** | 行政处罚 + 民事赔偿 |
| **选择性披露** | 误导投资人 |
| **无依据的"零碳"声称** | 重点打击 |
| **第三方鉴证缺失** | 可信度质疑 |
| **数据造假** | 涉刑(欺诈发行) |

**防范建议:**

```
✅ 真实数据 + 真实案例
✅ 第三方独立鉴证
✅ 明确披露范围 / 边界
✅ 引用国际标准(GRI / SASB / TCFD)
✅ 重要信息须量化
```

## ESG 评级风险

**主要评级机构:**

| 机构 | 特点 |
|------|------|
| MSCI | 全球 + A-CCC 评级 |
| Sustainalytics | 风险评级 |
| S&P Global | ESG 评分 |
| 商道融绿 | 国内权威 |
| 中证 ESG | 国内上市公司 |

**降级风险:**

| 后果 | 注意 |
|------|------|
| 股价下跌 | 投资偏好改变 |
| 融资成本上升 | 绿色债券受限 |
| 客户流失 | 跨国公司采购标准 |
| 监管处罚 | 虚假披露 |
