# /application-and-filing — 执行申请立案

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B1 工作流 + § B2 路由表 + § B4 风险等级 + § A12 管辖 + § B11 注册表。

## 工作流程

```
Step 1: 评估法律文书类型 + 是否生效
Step 2: 核查申请执行期限(2 年)
Step 3: 评估管辖法院
Step 4: 评估申请执行人资格
Step 5: 准备执行申请书 + 证据材料
Step 6: 提交申请 + 跟进立案
```

## 申请执行期限(《民诉法》第 246 条)

**2 年期限:**

```
✅ 从法律文书规定履行期间最后一日起算
✅ 法律文书未规定履行期间的:从生效日起算
✅ 分期履行的:从最后一期履行期限届满日起算
```

**期限起算点:**

| 文书类型 | 起算 |
|----------|------|
| 判决 / 裁定 | 履行期届满日(生效日 + 履行期) |
| 调解书 | 履行期届满日 |
| 仲裁裁决 | 履行期届满日 |
| 公证债权文书 | 履行期届满日 |
| 行政处罚决定 | 履行期届满日 |

**超期后果:**

| 情形 | 后果 |
|------|------|
| 超过 2 年 | 丧失申请执行权 |
| 中断情形(对方确认 / 主张) | 期限重新起算 |
| 不可抗力 | 期限中止 |
| 申请执行前达成和解 | 重新起算 |

## 法律文书类型 6 类

| 类型 | 适用 | 注意 |
|------|------|------|
| **生效判决 / 裁定** | 法院判决 | 默认 |
| **调解书** | 法院调解 | 双方签字生效 |
| **仲裁裁决** | 仲裁机构 | 涉外注意 |
| **公证债权文书** | 公证机构 | 赋予强制执行效力的债权文书 |
| **行政处罚决定** | 行政机关 | 法院申请执行 |
| **刑事裁判涉财产部分** | 法院刑事 | 罚金 / 没收财产 |

## 管辖法院(《民诉法》第 231 条)

| 情形 | 法院 |
|------|------|
| **第一审法院** | 默认 |
| **与一审法院同级的被执行财产所在地法院** | ✅ |
| **仲裁裁决** | 被执行人住所地 / 财产所在地中级法院 |
| **涉外执行** | 中级人民法院 |
| **跨省执行** | 共同上级法院指定 |

**特别注意:**

```
✅ 多个法院有管辖权:申请执行人可任选
✅ 财产跨多省:需向共同上级法院申请
✅ 涉外:中级法院专属
```

## 申请执行人资格

| 主体 | 资格 |
|------|------|
| 法律文书确定的权利人 | ✅ |
| 继承人 / 受让人(经法律文书确认) | ✅ |
| 法律文书确认的代理人 | ✅ |
| 不具资格的主体 | ❌ |

**变更申请执行人:**

```
✅ 法人合并 / 分立
✅ 自然人死亡(继承人)
✅ 债权转让(经法律文书确认)
✅ 申请执行人变更需向法院申请
```

## 立案材料

**必备材料:**

```
1. 执行申请书
2. 法律文书原件 + 副本(判决书 + 生效证明 + 送达回证)
3. 申请执行人身份证明(身份证 / 营业执照)
4. 委托代理手续(若委托律师)
5. 财产线索清单
```

## 申请执行内容

**金钱给付(最常见):**

```
✅ 本金 + 利息 + 罚息 + 复利
✅ 迟延履行利息(法定的)
✅ 诉讼费 + 律师费(若法院判决承担)
```

**物的交付 / 行为 / 知识产权:** 详见 § B6 输出模板。

## 实务注意

**特别注意:**

```
⚠️ 申请执行期限 2 年,过期不候
⚠️ 法律文书必须"生效"(经上诉期 + 上诉)
⚠️ 涉外 / 跨省案件须选对管辖法院
⚠️ 财产线索越详细,执行越有效
⚠️ 同时准备终结本次执行的预案
```

**快速回款策略:**

```
✅ 申请执行前评估对方财产
✅ 同步申请财产保全
✅ 评估执行和解空间
✅ 评估失信被执行人名单威慑
✅ 必要时拒不执行判决罪威慑
```
