# /settlement-and-distribution — 执行和解 + 财产分配

## 工作流程
```
Step 1: 评估和解空间 / 财产分配方案
Step 2: 准备和解协议 / 分配方案
Step 3: 履行和解 / 申请分配
Step 4: 终结本次 / 销案
```

## 执行和解(《民诉法》第 235 条)

**和解方式:**

| 方式 | 适用 |
|------|------|
| **和解协议** | 双方私下达成 + 法院确认 |
| **执行和解笔录** | 法院主持 + 双方签字 |
| **和解履行** | 履行后终结本次执行 |

**和解协议必备条款:**

```
1. 当事人信息
2. 履行内容(金额 / 物 / 行为)
3. 履行期限 + 方式
4. 违约责任
5. 和解效力(一次性了结 / 部分履行)
6. 签字 + 日期
```

**和解的效力:**

| 情形 | 后果 |
|------|------|
| 履行完毕 | 终结执行(销案) |
| 不履行 | 申请执行人可申请恢复原执行 |
| 部分履行 | 恢复未履行部分 |

## 财产分配(多债权人)

**参与分配条件(《高法执行规定》):**

```
✅ 已取得执行依据(判决 / 裁定 / 仲裁 / 公证)
✅ 申请执行后,被执行人财产不足清偿全部债务
✅ 在执行程序中
```

**分配顺序:**

```
1. 优先受偿债权(有担保物权 / 工程款优先权)
2. 普通债权按比例分配
3. 刑事罚金 / 行政罚款
```

**分配原则:**

| 原则 | 注意 |
|------|------|
| 平等原则 | 普通债权按比例 |
| 优先原则 | 担保物权 / 法定优先权 |
| 顺序原则 | 优先受偿 > 普通债权 |
| 实际原则 | 实际查封 / 控制财产 |

## 多债权人分配方案

**分配计算公式:**

```
某债权人分配额 = (可分配金额 × 该债权金额) / 普通债权总额
```

**实务考虑:**

| 情形 | 处理 |
|------|------|
| 多轮查封 | 按查封顺序 |
| 优先受偿债权 vs 普通 | 优先受偿先行 |
| 同一财产多查封 | 协调分配 |
| 共有财产 | 分割后执行 |

## 实务注意

```
⚠️ 和解须有可执行性
⚠️ 一次性了结 vs 分期履行须明确
⚠️ 不履行恢复原执行须明确约定
⚠️ 多债权人分配须有书面方案
⚠️ 优先受偿债权优先
```
