# /objection-and-dispute — 执行异议

## 工作流程
```
Step 1: 评估异议类型
Step 2: 评估异议期限(15 日)
Step 3: 准备异议书 + 证据
Step 4: 跟进 + 异议之诉
```

## 异议类型 3 类

| 类型 | 主体 | 时限 |
|------|------|------|
| **案外人异议** | 第三人(对执行标的有权利) | 执行中 15 日内 |
| **申请执行人异议** | 申请执行人(对终结本次 / 中止执行等) | 收到裁定 10 日内 |
| **被执行人异议** | 被执行人(对执行行为) | 执行中 15 日内 |

## 案外人异议(《民诉法》第 238 条)

**异议条件:**

```
✅ 案外人对执行标的有所有权 / 其他足以阻止转让 / 交付的实体权利
✅ 执行过程中提出
✅ 书面 + 证据
```

**常见情形:**

| 情形 | 主张 |
|------|------|
| 查封前已购买 | 排除执行 |
| 共有财产 | 分割后执行 |
| 第三人抵押 / 质押 | 优先受偿 |
| 错误查封 | 解除查封 |

## 异议处理流程

```
1. 异议申请(15 日内)
2. 法院审查(书面)
3. 裁定:支持 / 驳回
4. 不服裁定:
   - 案外人 → 异议之诉(15 日内)
   - 申请执行人 / 被执行人 → 复议(10 日内)
5. 异议之诉(普通程序,6 个月)
```

## 异议之诉(《民诉法》第 239 条)

**案外人异议之诉:**

| 项目 | 注意 |
|------|------|
| 起诉时间 | 异议裁定送达后 15 日内 |
| 管辖 | 执行法院 |
| 审理 | 普通程序(6 个月) |
| 判决 | 支持 / 驳回 |

**申请执行人异议之诉:**

```
✅ 法院裁定终结本次执行后 15 日内
✅ 主张被执行人有财产 / 法院未穷尽措施
```

## 复议(《民诉法》第 232 条)

**对执行行为裁定不服:**

| 情形 | 救济 |
|------|------|
| 查封 / 扣押 / 冻结等 | 复议(10 日内) |
| 拍卖 / 变卖裁定 | 复议(10 日内) |
| 拘留 / 罚款决定 | 复议(10 日内) |

## 实务注意

```
⚠️ 异议期限 15 日,过期不候
⚠️ 异议不影响执行(若提供担保)
⚠️ 异议之诉期间可申请中止执行
⚠️ 多次异议可能损害信誉
```
