# /monopoly-agreement-review — 垄断协议识别

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B2 路由表 + § B4 罚款 + § A7-13 不得同代理多方 + § B15 7 条哲学。

## 工作流程

```
Step 1: 识别协议类型(横向 / 纵向 / 行业协会)
Step 2: 评估协议内容(价格 / 数量 / 市场分割 / 限制)
Step 3: 评估相关市场 + 市场影响
Step 4: 评估是否构成核心卡特尔
Step 5: 评估豁免情形(7 类)
Step 6: 风险评估 + 整改建议
```

## 协议类型 3 类

### 横向协议(《反垄断法》第 13 条)— 核心卡特尔

**风险最高**,相同 / 类似业务竞争者之间的协议。

| 类型 | 风险 |
|------|------|
| **价格垄断** | 🔴 核心卡特尔,直接禁止 |
| **数量限制** | 🔴 核心卡特尔,直接禁止 |
| **市场分割** | 🔴 核心卡特尔,直接禁止 |
| **限制新技术 / 新产品** | 🔴 核心卡特尔,直接禁止 |
| **联合抵制交易** | 🔴 严重限制竞争 |
| **其他反竞争协议** | 视情形 |

### 纵向协议(《反垄断法》第 14 条)— 转售价格维持

**风险中等**,上下游经营者之间的协议。

| 类型 | 风险 |
|------|------|
| **固定转售价格** | 🔴 直接禁止 |
| **限定最低转售价格** | 🔴 直接禁止(2022 修订明确) |
| **独家销售 / 独家采购** | ⚠️ 一般豁免,滥用支配地位时禁止 |
| **搭售** | ⚠️ 一般豁免,滥用支配地位时禁止 |
| **地域 / 客户限制** | ⚠️ 一般豁免,滥用支配地位时禁止 |

### 行业协会协议(《反垄断法》第 22 条)

**风险特殊**,行业协会的"组织 + 协助"行为。

| 类型 | 风险 |
|------|------|
| **组织本行业经营者达成协议** | 🔴 直接禁止 |
| **提供必要信息 / 协助** | 🔴 直接禁止 |
| **制定行业自律公约(限制竞争)** | 🔴 直接禁止 |
| **行业标准(必要 + 中立)** | ✅ 一般允许 |

**特别注意:** 行业协会自身不是反垄断法上的"经营者",但其组织 + 协助行为构成"垄断协议"。

## 核心卡特尔 vs 边缘行为

**核心卡特尔(零容忍):**

| 行为 | 后果 |
|------|------|
| 价格固定(横向) | 🔴 罚款 + 严重者追究刑责 |
| 数量限制(横向) | 🔴 同上 |
| 市场分割(横向) | 🔴 同上 |
| 围标(招投标) | 🔴 同上 |
| 价格协调(通过行业协会) | 🔴 同上 |

**边缘行为(可豁免):**

| 行为 | 处理 |
|------|------|
| 技术合作 | 视情形 |
| 标准制定 | 必要 + 中立 |
| 联合研发 | 不限制竞争 |
| 中小企业联合 | 视情形 |

## 豁免情形 7 类(《反垄断法》第 20 条)

| 情形 | 注意 |
|------|------|
| **1. 改进技术 / 研究开发** | 必须是为改进技术 / 提高效率 / 提高质量 / 降低成本等 |
| **2. 产品质量改进 / 降低成本** | 必须是为共同目的 |
| **3. 经营效率提高** | 不限制竞争 / 不损害消费者 |
| **4. 公共利益** | 节能减排 / 救灾 / 出口贸易 |
| **5. 经济不景气** | 销售量严重下降 |
| **6. 保护对外贸易 / 国际化经营** | 国内市场不影响 |
| **7. 法律 / 行政法规规定** | 法定豁免 |

**豁免限制:**

```
❌ 严重限制相关市场竞争
❌ 消费者不能分享由此产生的利益
❌ 经营者被要求承担不合理的义务
```

## 协议识别 5 步法

**Step 1: 协议类型判断**

| 协议方关系 | 类型 |
|------------|------|
| 同行(竞争者) | 横向 |
| 上下游 | 纵向 |
| 行业协会 + 会员 | 协会组织 |

**Step 2: 协议内容判断**

| 内容 | 风险 |
|------|------|
| 固定价格 / 转售价格 | 🔴 核心 |
| 限制数量 / 产量 | 🔴 核心 |
| 分割市场 / 客户 | 🔴 核心 |
| 限制技术 / 新产品 | 🔴 核心 |
| 联合抵制 | 🔴 严重 |
| 独家 / 搭售 / 限制地域 | ⚠️ 中等 |

**Step 3: 相关市场判断**

```
✅ 商品市场(哪些产品构成同一市场)
✅ 地域市场(中国 / 全球 / 特定区域)
✅ 时间市场(短 / 长期)
```

**Step 4: 竞争影响评估**

| 因素 | 评估 |
|------|------|
| 市场份额 | 高份额 → 风险大 |
| 集中度 | HH 指数 / CR4 指数 |
| 进入壁垒 | 高壁垒 → 风险大 |
| 消费者选择 | 受限 → 风险大 |
| 创新影响 | 限制创新 → 风险大 |

**Step 5: 豁免评估**

| 情形 | 是否适用 |
|------|----------|
| 7 类豁免情形 | 评估 |
| 豁免限制(严重限制 / 消费者不分享) | 排除 |
| 协议必要性 | 必须为实现豁免目的 |

## 罚款风险(《反垄断法》2022 修订)

**核心卡特尔:**

| 情形 | 罚款 |
|------|------|
| 上一年度销售额 | **1%-10%** |
| 严重情节(行业协会组织 + 屡次实施) | **2%-10%** |
| 情节特别严重 | 加重至 10% 上限 |

**罚款基数:**

```
✅ 上一年度销售额
✅ 协议参与方各自计算
✅ 新企业(无上一年度)按协议期内实际销售额
```

**其他处罚:**

| 处罚 | 适用 |
|------|------|
| 没收违法所得 | ✅ |
| 高管个人罚款 | ¥10万-100万 |
| 责令恢复原状 | ✅ |
| 责令停止协议 | ✅ |
| 行业协会罚款 | ¥10万-300万 |

## 协议整改 5 步

**发现协议可能构成垄断:**

```
Step 1: 立即停止协议 / 重新协商
Step 2: 通知反垄断局(主动报告减轻)
Step 3: 评估是否申请宽大(横向卡特尔核心)
Step 4: 配合调查
Step 5: 整改 + 合规体系建立
```

**整改特别考虑:**

| 情形 | 风险 |
|------|------|
| 协议已签未执行 | 可补救 |
| 协议已执行部分 | 高风险 |
| 协议持续执行 | 🔴 主动宽大申请 |
| 已被调查 | 🔴 配合 + 宽大 + 律师 |

## 行业协会特殊考虑

**特别禁止(《反垄断法》第 22 条):**

```
❌ 行业协会组织本行业经营者达成垄断协议
❌ 行业协会提供必要信息 / 协助
❌ 行业协会制定限制竞争的章程 / 行业公约
```

**合规建议:**

```
✅ 行业协会章程不限制竞争
✅ 行业自律公约不涉及价格 / 数量 / 市场分割
✅ 信息公开中立
✅ 行业标准符合"必要 + 中立"
```

## 跨境垄断协议

**多法域并行(高频):**

| 法域 | 关注 |
|------|------|
| 美国 | 谢尔曼法第 1 条(per se 违法) |
| 欧盟 | TFEU 第 101 条 |
| 英国 | Competition Act 1998 第 1 条 |
| 日本 | 独占禁止法第 3 条 |

**特别注意:**

```
✅ 同一行为多法域调查
✅ 主动宽大制度(各法域独立)
✅ 法律义务不同
```
