# /antitrust-investigation — 反垄断调查应对

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B1 Step 4 + § B4 罚款 + § B5 升级 + § A7-14 律师特权 + § B12 严格隔离。

## 工作流程

```
Step 1: 收到调查通知 → 立即启动应对
Step 2: 内部调查(自查 + 证据保护)
Step 3: 评估宽大申请可能性
Step 4: 准备应对方案(配合 / 抗辩)
Step 5: 应对现场调查(进场 + 资料 + 询问)
Step 6: 应对处罚决定(申辩 / 听证 / 复议 / 诉讼)
```

## 调查启动 3 阶段

**启动方式:**

| 启动 | 启动机关 |
|------|----------|
| **举报启动** | 竞争者 / 上下游 / 消费者举报 |
| **依职权启动** | 反垄断局主动发现 |
| **上级交办** | 上级机关交办 |
| **外国机关移送** | 跨境协调 |

**调查阶段:**

```
阶段 1: 初步核查(1-3 月)
  - 内部讨论
  - 收集线索
  - 决定是否立案

阶段 2: 立案调查(3-12 月,重大更长)
  - 现场调查
  - 资料调取
  - 询问当事人
  - 第三方调查

阶段 3: 处罚决定
  - 事先告知
  - 听证(当事人申请)
  - 处罚决定
  - 送达
```

## 接到调查通知后 24 小时行动

**黄金 24 小时行动清单:**

| 行动 | 注意 |
|------|------|
| 1. 启动律师 + 内部调查组 | ✅ 立即 |
| 2. 通知公司董事会 / 高管 | ✅ 立即 |
| 3. **证据保护(严肃)** | ✅ 立即(后述) |
| 4. 暂停涉及"问题协议"的新业务 | ✅ 立即 |
| 5. 评估宽大申请可行性 | ✅ 24 小时内 |
| 6. 评估是否对外披露 | ✅ 由律师决定 |

**证据保护原则:**

```
✅ 立即启动律师特权 / 工作成果保护
  - 内部调查材料由外部律师主持
  - 文件标注"PRIVILEGED & CONFIDENTIAL — ATTORNEY WORK PRODUCT"
  - 内部调查会议的备忘录由律师准备

❌ 不得:
  - 销毁 / 篡改 / 隐匿证据(《反垄断法》2022 加重情节)
  - 阻碍调查
  - 串通其他被调查方
```

**"加重情节"风险:**

```
⚠️ 销毁 / 隐匿 / 伪造证据 → 罚款上限升至 10%
⚠️ 行业协会组织 + 协助 → 加重
⚠️ 屡次实施 → 加重
⚠️ 阻碍调查 → 加重
```

## 调查类型与策略

**类型 1: 涉嫌垄断协议(横向 / 纵向)**

```
应对策略:
  Step 1: 内部自查
    - 是否有相关协议
    - 是否有价格 / 数量 / 市场分割
    - 是否有豁免情形
  Step 2: 评估宽大申请(横向卡特尔)
    - 是否是第一个申请者
    - 证据是否充分
  Step 3: 准备抗辩材料
    - 不构成协议
    - 有豁免情形
  Step 4: 配合调查 + 准备减轻情节
```

**类型 2: 涉嫌滥用市场支配**

```
应对策略:
  Step 1: 自查相关市场 + 市场份额
  Step 2: 评估"市场支配"是否成立
  Step 3: 评估行为是否构成"滥用"
  Step 4: 准备"正当理由"抗辩
  Step 5: 准备效率抗辩
  Step 6: 准备整改方案
```

**类型 3: 涉嫌集中未申报**

```
应对策略:
  Step 1: 确认未申报事实
  Step 2: 立即补救申报
  Step 3: 主动报告 + 配合调查
  Step 4: 申请减轻罚款
```

## 现场调查应对

**反垄断局进场:**

| 阶段 | 客户应对 |
|------|----------|
| 出示证件 | ✅ 律师在场核对 |
| 调查范围 | ✅ 律师协助界定 |
| 资料调取 | ✅ 律师在场,签字确认 |
| 询问 | ✅ 律师在场,谨慎回答 |
| 现场取证 | ✅ 律师在场,记录范围 |

**特别注意:**

```
⚠️ 不可"同意扩大调查范围"
⚠️ 不可"未经律师审阅签字"提供材料
⚠️ 不可"擅自解释"对方问题
⚠️ 不可"事后修改 / 补充"已提供材料
```

## 资料提供原则

**提供范围:**

| 类型 | 原则 |
|------|------|
| 反垄断局正式要求的 | 必须提供(否则可被强制) |
| 与调查相关的 | 配合提供 |
| 律师工作成果 / 客户与律师的通讯 | **绝对不得提供**(律师特权) |
| 第三方的商业秘密 | 申请保护 |

**提供方式:**

```
✅ 律师审阅后提供
✅ 律师在场情况下提供
✅ 律师签字确认的"提供清单"
✅ 重要文件律师准备"提供说明"
```

**特别保护:**

| 类别 | 保护 |
|------|------|
| 律师工作成果 | **绝对不提供** |
| 高管私人通讯 | 部分保护 |
| 第三方商业秘密 | 申请保护(《反不正当竞争》) |
| 员工私人通讯 | 部分保护 |

## 询问应对

**反垄断局询问:**

| 类型 | 应对 |
|------|------|
| 事实问题 | 简短 + 准确 + 不猜测 |
| 解释问题 | 不承认,留待后续 |
| 涉及宽大 | 律师在场 |
| 涉及第三方 | 不评论 |

**特别注意:**

```
⚠️ 任何回答可能被记录 + 作为证据
⚠️ 不猜测 + 不推断 + 不主动展开
⚠️ 不评论其他被调查方
⚠️ 涉及宽大申请(若有)—— 一旦申请,不可改变
```

## 处罚决定应对

**处罚决定阶段(《反垄断法》第 60 条):**

```
Step 1: 收到事先告知
  - 决定书草案
  - 拟处罚内容
  - 事实 + 理由 + 法条

Step 2: 申请听证
  - 7 日内书面申请
  - 听证委员会(独立)
  - 律师 + 当事人参加
  - 陈述 + 质证

Step 3: 申辩材料
  - 事实 + 理由
  - 程序问题
  - 证据异议
  - 减轻情节

Step 4: 收到处罚决定
  - 不服 → 复议 / 诉讼
  - 复议:60 日内,市场监管总局
  - 诉讼:6 个月内,中级法院
```

## 减轻 / 加重情节

**减轻情节 6 类:**

| 情形 | 效果 |
|------|------|
| 主动报告(主动调查前) | 大幅减轻 |
| 配合调查(早期 + 充分) | 减轻 |
| 主动整改(停止违法行为) | 减轻 |
| 证据充分 + 内部举报 | 减轻 |
| 首次违法 | 减轻 |
| 受胁迫 / 诱骗参与 | 减轻 |

**加重情节 6 类(2022 修订):**

| 情形 | 效果 |
|------|------|
| 行业协会组织 + 协助 | 加重 |
| 屡次实施 | 加重 |
| 阻碍调查 | 加重 |
| 销毁 / 隐匿 / 伪造证据 | 加重至 10% 上限 |
| 对举报人打击报复 | 加重 |
| 上一年度销售额特别巨大 | 加重 |

## 配合调查的边界

**强制配合义务(《反垄断法》第 41 条):**

```
✅ 不得拒绝 / 阻碍调查
✅ 不得销毁 / 隐匿 / 伪造证据
✅ 不得提供虚假材料
✅ 不得拒绝提供资料
```

**保护性权利:**

| 权利 | 注意 |
|------|------|
| 律师在场 | ✅ |
| 律师审阅材料 | ✅ |
| 申请听证 | ✅ |
| 申辩 + 质证 | ✅ |
| 复议 / 诉讼 | ✅ |
| 律师特权保护 | ✅ |

## 跨境调查

**多法域并行(高频):**

| 法域 | 主管 |
|------|------|
| 中国 | 市场监管总局 |
| 欧盟 | 欧委会竞争总司 |
| 美国 | DOJ + FTC |
| 英国 | CMA |

**特别考虑:**

| 情形 | 处理 |
|------|------|
| 同一行为多法域调查 | 协调律师团队 |
| 法律义务不同 | 各法域独立 |
| 证据跨境 | 申请法域协助 |
| 商业秘密保护 | 申请保护令 |
