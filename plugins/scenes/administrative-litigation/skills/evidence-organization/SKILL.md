# /evidence-organization — 行政诉讼证据组织

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § A7-14 被告举证责任 + § A16 Large input 处理 + § B10 数据源标注。

## 工作流程

```
Step 1: 评估诉求 + 行政行为合法性焦点
Step 2: 识别举证责任分配(被告 / 原告 / 共同)
Step 3: 原告初步举证(证明行政行为存在 + 损害)
Step 4: 申请调取证据(如须)
Step 5: 质证准备
Step 6: 证据清单编排 + 提交
```

## 举证责任倒置(《行政诉讼法》第 34 条)

**核心原则:**

```
✅ 被告(行政机关)对作出行政行为的证据 + 所依据的规范性文件 + 法定程序 承担举证责任
✅ 原告仅需初步证明:
  - 行政行为存在(收到决定书等)
  - 自己与行政行为有利害关系
  - 受到损害(若涉赔偿)
```

**不提供证据后果(《行政诉讼法》第 67 条):**

```
被告不提供 / 无正当理由逾期提供证据 → 视为没有相应证据
→ 法院可认定行政行为**没有证据** → 判决撤销
```

**特别注意:** 被告在**收到起诉状副本之日起 15 日内**提交证据(《行政诉讼法》第 67 条 + 高法解释)。

## 举证责任分配 6 类

| 主张 | 举证责任 | 法条 |
|------|----------|------|
| 行政行为存在 | 原告 | 第 12 条 |
| 行政行为合法 | **被告** | 第 34 条 |
| 行政行为符合法定程序 | **被告** | 第 34 条 |
| 行政行为依据规范性文件合法 | **被告** | 第 34 条 |
| 行政行为造成损害 | 原告(若涉赔偿) | 第 38 条 |
| 损害与行政行为有因果关系 | 原告(若涉赔偿) | 第 38 条 |

**原告举证(主动):**

```
✅ 行政行为文书(决定书 / 通知书 / 公告等)
✅ 受到损害的证据(财产损失 / 经营损失 / 人身伤害)
✅ 因果关系初步证明
✅ 行政行为存在的事实
✅ 自身主体资格证明
```

**被告举证(主动):**

```
✅ 作出行政行为的事实证据
✅ 作出行政行为的法律依据(规范性文件)
✅ 作出行政行为的程序证据
✅ 自由裁量合理性的证据
✅ 紧急情况 / 不可抗力等免责证据
```

## 行政诉讼证据 8 类

| 类别 | 注意 |
|------|------|
| 书证 | 决定书 / 通知书 / 笔录 / 报告等 |
| 物证 | 物品 / 痕迹 |
| 视听资料 | 录音 / 录像 / 照片 |
| 电子数据 | 邮件 / 微信 / 系统日志 |
| 证人证言 | 工作人员 / 知情人员 |
| 当事人陈述 | 原告 / 被告陈述 |
| 鉴定意见 | 鉴定结论 |
| 勘验笔录 | 现场勘验记录 |

**特别注意:**
- 行政机关工作人员的陈述 = 当事人陈述(对方当事人,真实性需审查)
- 鉴定意见:被告经常使用,原告可申请重新鉴定

## 证据三性审查

**真实性:**

| 风险 | 审查 |
|------|------|
| 文书真实性 | 公章 / 签字 / 形成时间 |
| 物证真实性 | 来源 / 提取程序 |
| 视听资料 | 完整性 / 未剪辑 |

**合法性:**

| 风险 | 审查 |
|------|------|
| 取证程序违法 | 搜查 / 扣押 / 监听等程序 |
| 形式违法 | 应有签字 / 公章而缺 |
| 超期取证 | 是否在法定期限内 |

**关联性:**

| 风险 | 审查 |
|------|------|
| 与待证事实无关 | 直接排除 |
| 间接相关 | 辅助证明 |
| 品格证据 | 不可作证据 |

## 调取证据申请

**法院调取(《行政诉讼法》第 41 条):**

| 情形 | 申请 |
|------|------|
| 原告因客观原因无法自行收集 | 申请法院调取 |
| 涉及国家秘密 / 商业秘密 / 个人隐私 | 申请法院调取 |
| 被告无正当理由拒不提供 | 法院依职权调取 |

**被告不得收集(《高法解释》第 46 条):**

```
❌ 被告及其诉讼代理人在作出行政行为后自行收集的证据
❌ 被告在复议程序中未提交的证据
❌ 被告在行政程序中未作为作出行政行为依据的证据

(但:原告或第三人在诉讼中提出,被告可以在行政程序中未作为依据的理由)
```

## 证据清单编排

**证据清单必备:**

```
1. 证据名称
2. 证据形式(书证 / 物证 / 视听 / 电子等)
3. 证据来源
4. 证明对象(待证事实)
5. 证据页数 / 件数
6. 提交方(原告 / 被告)
7. 提交日期
```

**证据编排顺序:**

| 顺序 | 类型 |
|------|------|
| 1 | 主体资格证据 |
| 2 | 程序证据 |
| 3 | 实体证据(事实) |
| 4 | 损害证据 |
| 5 | 法律依据规范性文件 |
| 6 | 其他辅助证据 |

## 质证准备

**原告对被告证据的质证要点:**

| 证据 | 质证角度 |
|------|----------|
| 决定书 / 文书 | 真实性 / 公章 / 形式 |
| 调查笔录 | 笔录程序 / 见证人 / 是否独立 |
| 物证 | 提取程序 / 见证人 |
| 鉴定意见 | 鉴定机构资质 / 方法 / 材料 |
| 规范性文件 | 是否现行有效 / 是否适用 |

**被告对原告证据的质证要点:**

| 证据 | 质证角度 |
|------|----------|
| 损害证据 | 真实性 / 关联性(损害是否由行政行为造成) |
| 程序证据 | 形式 / 时间 |
| 反驳证据 | 三性 |

## 证据保全

**申请证据保全的情形:**

```
✅ 证据可能灭失
✅ 证据以后难以取得
✅ 涉及第三人利益
```

**申请时点:** 起诉前 / 起诉中均可,法院裁定是否准许。
