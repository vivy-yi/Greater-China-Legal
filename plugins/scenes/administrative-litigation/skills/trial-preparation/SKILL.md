## 行政诉讼庭审程序

**庭审流程:**

```
1. 开庭准备(核对当事人 / 宣布法庭纪律)
2. 法庭调查
   - 原告陈述诉讼请求 + 事实理由
   - 被告答辩
   - 举证质证
   - 法庭询问
3. 法庭辩论
   - 原告发言
   - 被告答辩
   - 互相辩论
4. 最后陈述
5. 休庭 / 宣判
```

**审理方式:**

| 方式 | 适用 |
|------|------|
| **普通程序** | 一般案件 |
| **简易程序** | 基层 / 中级法院审理的第一审案件(事实清楚 / 权利义务关系明确) |
| **独任审理** | 简易程序案件 |
| **合议庭** | 普通程序 + 重大复杂案件 |

## 不停止执行原则(《行政诉讼法》第 56 条)

**核心原则:**

```
诉讼期间,不停止行政行为的执行
```

**例外停止执行(6 类):**

| 情形 | 停止 |
|------|------|
| 被告认为需要停止执行 | ✅(被告主动) |
| 原告申请 + 法院裁定 | ✅(法院审查) |
| 法律 / 法规规定停止执行 | ✅ |
| 行政行为明显不当 | ✅(法院可裁定) |
| 行政行为可能造成难以弥补的损失 + 停止执行不损害公共利益 | ✅(法院可裁定) |
| 法院认为应当停止执行的其他情形 | ✅ |

**原告申请停止执行:**

```
申请时点:起诉时 / 诉讼中
申请材料:申请书 + 证据(证明不停止会造成难以弥补的损失)
法院裁定:48 小时内作出(紧急情况)
效力:停止至法院裁判生效
```

**特别考虑:**

```
✅ 行政强制措施(查封 / 扣押 / 冻结)一般**不停止**(强制力保障)
✅ 行政许可:可能损害第三人利益时慎重
✅ 行政处罚:执行可能造成不可逆损害时,考虑停止
```

## 司法变更权(《行政诉讼法》第 77 条)

**核心原则:** 行政处罚显失公正,法院可以判决变更。

**适用条件:**

```
✅ 必须是行政处罚
✅ 必须显失公正(明显不当 / 畸重畸轻)
✅ 法院可"变更"(直接改判),不撤销发回
```

**不适用司法变更:**

| 情形 | 处理 |
|------|------|
| 其他行政行为(许可 / 强制 / 确认等) | 撤销 / 确认违法 / 责令履行 |
| 行政处罚不"显失公正" | 撤销发回 |
| 自由裁量合理范围 | 不可变更 |

**实务中"显失公正":**

| 情形 | 司法 |
|------|------|
| 处罚幅度超过法定上限 | ✅ 显失公正 |
| 同等情形不同处罚 | ✅ 显失公正 |
| 处罚与违法行为不匹配 | 部分法院支持 |
| 仅程序违法 | 一般不变更 |

## 庭审重点策略

**法庭调查阶段:**

| 阶段 | 策略 |
|------|------|
| 原告陈述 | 简明扼要 + 紧扣诉求 |
| 被告答辩 | 仔细记录 + 寻找反驳点 |
| 被告举证 | 重点质证程序违法 + 证据不足 |
| 质证 | 抓住关键证据,避免枝节 |
| 法庭询问 | 回答精炼,避免猜测 |

**法庭辩论阶段:**

```
1. 总体立场(一句话,清晰)
2. 行政行为违法点(类型化):
   - 主要事实不清
   - 证据不足
   - 适用法律错误
   - 违反法定程序
   - 超越职权
   - 滥用职权
   - 明显不当
3. 法条引用(准确 + 版本)
4. 类案参照(指导案例优先)
5. 结论请求
```

**特别注意:**

```
⚠️ 行政诉讼辩论聚焦"合法性" — 不替代行政机关自由裁量
⚠️ 司法变更权有限 — 仅"显失公正"行政处罚
⚠️ 行政诉讼判决类型有限 — 撤销 / 确认违法 / 责令履行 / 变更(显失公正)
```

## 行政诉讼判决类型 6 类(《行政诉讼法》第 69-78 条)

| 情形 | 判决 |
|------|------|
| 行政行为合法 | 驳回诉讼请求(第 69 条) |
| 主要证据不足 | 撤销(第 70 条第 1 项) |
| 适用法律错误 | 撤销(第 70 条第 2 项) |
| 违反法定程序 | 撤销(第 70 条第 3 项) |
| 超越职权 | 撤销(第 70 条第 4 项) |
| 滥用职权 | 撤销(第 70 条第 5 项) |
| 明显不当 | 撤销 / 变更(第 70 条第 6 项 / 第 77 条) |
| 不履行法定职责 | 责令履行(第 72 条) |
| 行政行为合法但存在轻微违法 | 确认违法(第 74 条) |
| 行政行为违法但撤销无意义 | 确认违法(第 75 条) |

## 行政诉讼 vs 民事诉讼的差异

| 维度 | 行政诉讼 | 民事诉讼 |
|------|----------|----------|
| 当事人 | 原告 vs 行政机关 | 平等主体 |
| 被告 | 只能是行政机关 | 任何主体 |
| 诉讼目的 | 审查行政行为合法性 | 解决民事争议 |
| 举证责任 | 被告负主要举证 | 谁主张谁举证 |
| 调解 | 行政赔偿 / 补偿可调解 | 一般可调解 |
| 撤诉 | 须被告同意 | 双方可撤 |
| 变更判决 | 限于显失公正 | 多种 |
