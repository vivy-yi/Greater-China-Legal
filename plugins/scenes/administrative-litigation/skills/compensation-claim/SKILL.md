# /compensation-claim — 国家赔偿 / 行政补偿

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B1 Step 6 + § B5 重大国家赔偿升级 + § B13 Enforcement posture。

## 工作流程

```
Step 1: 评估赔偿类型(国家赔偿 / 行政补偿)
Step 2: 评估赔偿义务机关
Step 3: 评估赔偿范围 + 赔偿标准
Step 4: 提交赔偿申请(2 个月内处理)
Step 5: 不服处理结果 → 提起赔偿诉讼
Step 6: 跟踪赔偿履行
```

## 国家赔偿 5 类(《国家赔偿法》)

| 类型 | 法条 | 情形 |
|------|------|------|
| **行政赔偿** | 第 3 条 | 违法行政行为造成损害 |
| **刑事赔偿** | 第 17 条 | 错拘 / 错捕 / 错判 |
| **非刑事司法赔偿** | 第 31 条 | 司法行为致害 |
| **精神损害赔偿** | 第 35 条 | 造成严重精神损害 |
| **人身自由 / 生命健康赔偿** | 第 33/34 条 | 侵犯人身权 |

**行政赔偿 7 类情形(《国家赔偿法》第 3/4 条):**

| 情形 | 注意 |
|------|------|
| 违法拘留 / 逮捕 | ✅ |
| 非法拘禁 / 其他限制自由 | ✅ |
| 殴打 / 虐待 / 唆使 | ✅ |
| 违法使用武器 / 警械 | ✅ |
| 违法罚款 / 吊销许可证 / 责令停产停业 | ✅ |
| 违法征收 / 摊派 | ✅ |
| 造成损害的其他违法行为 | ✅ |

## 行政补偿(非赔偿)

**行政补偿 5 类:**

| 类型 | 法条 | 情形 |
|------|------|------|
| **土地征收补偿** | 《土地管理法》 | 集体土地征收 |
| **房屋征收补偿** | 《国有土地上房屋征收与补偿条例》 | 国有土地上房屋 |
| **行政征用补偿** | 单行法 | 紧急情况下征用财产 |
| **生态移民补偿** | 各省规定 | 生态保护 |
| **其他合法行为造成损失** | 司法解释 | 合法行为致害 |

**行政补偿 vs 国家赔偿:**

| 维度 | 国家赔偿 | 行政补偿 |
|------|----------|----------|
| 前提 | 行政行为**违法** | 行政行为**合法** |
| 法理 | 国家责任 | 公共利益 + 公平负担 |
| 依据 | 《国家赔偿法》 | 单行法 |
| 程序 | 赔偿义务机关先行处理 | 协商 / 协议 |
| 标准 | 法定的赔偿标准 | 协商 + 评估 |

## 赔偿义务机关

**确认原则(《国家赔偿法》第 7 条):**

| 情形 | 赔偿义务机关 |
|------|--------------|
| 行政机关作出违法行为 | 该行政机关 |
| 受委托组织 / 个人作出 | 委托机关 |
| 法律 / 法规授权组织作出 | 该组织 |
| 共同作出 | 共同机关 |
| 行政机关被撤销 | 继续行使职权的机关 |
| 复议改变原行为 | 复议机关 |
| 复议维持原行为 | 原机关 + 复议机关(共同) |

## 赔偿范围 + 标准

**人身权损害:**

| 类型 | 标准 |
|------|------|
| **人身自由** | 限制自由期间(按日)× 国家上年度职工日均工资 |
| **生命健康一般伤害** | 医疗费 + 误工费 + 护理费 |
| **生命健康残疾** | 残疾赔偿金(根据等级) + 医疗费 + 误工费 |
| **生命健康死亡** | 死亡赔偿金 + 丧葬费 + 抚养费 |
| **精神损害** | 司法解释(一般 ≤¥5万) |

**财产权损害:**

| 类型 | 标准 |
|------|------|
| 罚款 | 返还 + 利息 |
| 查封 / 扣押 / 冻结 | 解除 + 返还 + 利息 |
| 吊销许可证 | 恢复 |
| 责令停产停业 | 赔偿实际损失 |
| 征收 / 征用 | 按补偿标准 + 利息 |

**计算公式(2024 标准):**

```
人身自由赔偿金 = 限制自由天数 × 国家上年度职工日均工资
  (2024 年日均工资约 ¥365)

残疾赔偿金 = 上一年度职工年平均工资 × 伤残等级对应倍数
  (一级伤残: 20 倍 / 二级: 18 倍 / ... / 十级: 2 倍)

死亡赔偿金 = 上一年度职工年平均工资 × 20
```

## 申请程序

**Step 1: 提交赔偿申请(《国家赔偿法》第 12 条)**

```
申请时点:行政行为被确认违法后
申请对象:赔偿义务机关
申请材料:
  - 赔偿申请书
  - 行政行为被确认违法的证明(判决 / 复议决定)
  - 损害事实证据
  - 因果关系证明
  - 损失清单
  - 申请人身份证明
```

**Step 2: 赔偿义务机关处理(2 个月 / 3 个月)**

| 处理 | 期限 |
|------|------|
| 赔偿义务机关自收到申请之日起 **2 个月内**作出决定 | 《国家赔偿法》第 13 条 |
| 逾期不赔 / 不答复 → 提起赔偿诉讼(3 个月内) | 《国家赔偿法》第 14 条 |

**Step 3: 赔偿诉讼(《国家赔偿法》第 14 条)**

```
✅ 条件:赔偿义务机关逾期不赔 / 不答复 / 决定不予赔偿
✅ 起诉:自期限届满之日起 3 个月内
✅ 管辖:赔偿义务机关所在地中级法院(中级法院专属)
✅ 审理:中级法院(不得向基层法院起诉)
```

## 行政赔偿 vs 行政诉讼中一并赔偿

**两种方式:**

| 方式 | 适用 | 优劣 |
|------|------|------|
| **行政诉讼中一并提起** | 行政诉讼已立案 | 节省时间,一案解决 |
| **单独提起赔偿诉讼** | 行政诉讼已结案 + 行政行为已被确认违法 | 程序独立 |

**优先选择:** 行政诉讼中一并提起(降低诉累)。

## 行政补偿程序

**协商 + 协议:**

```
Step 1: 评估补偿范围(基于评估机构)
Step 2: 协商补偿标准
Step 3: 签订补偿协议
Step 4: 履行协议(支付 / 安置)
Step 5: 不服协议 → 行政复议 / 行政诉讼
```

**特别考虑:**

| 类型 | 特点 |
|------|------|
| 土地征收 | 省级以上政府批准 + 公告 + 补偿安置方案 |
| 房屋征收 | 市县级政府作出 + 评估 + 补偿决定 |
| 房屋补偿 | 评估机构评估 + 协商补偿 |
| 强制拆除 | 一般需先补偿,后搬迁 |

## 赔偿金计算工具

**精神损害抚慰金标准(参考):**

| 损害程度 | 金额(2024 司法实践) |
|----------|---------------------|
| 严重精神损害(死亡 / 重残) | ¥5万-10万 |
| 一般精神损害(错误拘押) | ¥1万-3万 |
| 轻微 | 视情况 |

**特别考虑:**

```
⚠️ 精神损害赔偿须造成"严重"损害,一般拘押不构成
⚠️ 死亡 / 重残 / 严重精神疾病可主张
⚠️ 司法实践不一,部分案件有差异
```

## 涉外国家赔偿

**特别考虑:**

| 情形 | 处理 |
|------|------|
| 外国人 / 无国籍人 | 同等适用(中国法保护) |
| 中国人在国外 | 适用双边 / 多边条约 |
| 国际组织人员 | 视国际组织法 |
| 领事保护 | 必要时通报外交部 |
