### § A8.1 白领犯罪特别注意 5 大块（监察留置 + 贪贿罪 + 自首立功 + 单位犯罪 + 涉案财物）

> **核心原则**：白领犯罪辩护的核心是**程序合规 + 量刑情节 + 单位 / 个人双轨**。任何辩护策略必须先吃透"监察 vs 公安 vs 检察"程序分流。

#### 块 1：监察留置程序（贪贿案件核心入口）

**核心法规：**
- 《中华人民共和国监察法》（2018 年 3 月生效）
- 《中华人民共和国公职人员政务处分法》（2020 年 7 月生效）
- 《监察法实施条例》（2021 年 9 月生效）

**监察程序核心要点：**

| 阶段 | 时限 | 关键权利 | 法条 |
|------|------|---------|------|
| **初步核实** | 6 个月（可延长 1 次） | 不得限制人身自由 | 监察法 § 38 条 |
| **立案调查** | 6 个月（可延长 1 次） | 进入正式调查 | 监察法 § 39 条 |
| **留置（核心）** | **不得超过 3 个月**（特殊情况可延长 1 次，最长 6 个月） | **家属通知 + 律师会见受限** | 监察法 § 43 条 |
| **移送审查起诉** | 30 日 | 进入刑事程序 | 监察法 § 45 条 |

**留置 vs 刑事拘留关键差异：**

| 维度 | 留置（监察） | 刑事拘留（公安） |
|------|------------|------------|
| **法律依据** | 监察法 | 刑事诉讼法 |
| **适用对象** | 涉嫌职务违法 / 犯罪的公职人员 | 一般犯罪嫌疑人 |
| **决定机关** | 监察机关（自决） | 公安机关（检察院批准） |
| **家属通知** | 24 小时内通知（有碍调查除外） | 24 小时内通知 |
| **律师会见** | **❌ 留置期间不得会见律师** | ✅ 律师可会见（除危害国家安全等） |
| **最长时限** | 3 个月（可延至 6 个月） | 37 天（公安 30 + 检察批捕 7） |
| **执行场所** | 留置场所（专门场所） | 看守所 |

**blocks（绝对禁止 — 监察留置期间）**：
- 伪造 / 毁灭证据 → **《刑法》第 307 条妨害作证罪**
- 串供 / 干扰证人 → **《刑法》第 307 条**
- 家属传递案件信息 → **涉嫌阻碍监察调查**
- 律师违规会见 → **《律师法》第 49 条 + 吊销执照**

**work but ships（可补救 — 留置后程序）**：
- 移送检察机关后 → 立即申请会见 + 阅卷
- 监察阶段陈述 → 仔细核对笔录 + 签字前复核

#### 块 2：贪污贿赂 4 大核心罪名（具体法条 + 量刑档次）

**核心罪名体系（按《刑法》第 382-396 条）：**

| 罪名 | 法条 | 立案标准 | 量刑档次 |
|------|------|---------|---------|
| **贪污罪** | § 382 | 3 万 / 20 万 / 300 万 | 3 年以下 / 3-10 年 / 10 年以上至无期 / **死刑** |
| **受贿罪** | § 385 | 3 万 / 20 万 / 300 万 | 同上 |
| **挪用公款罪** | § 384 | 5 万 / 500 万 | 5 年以下 / 5 年以上 |
| **行贿罪** | § 389 | 1 万 / 20 万 / 100 万 / 500 万 | 5 年以下 / 5-10 年 / 10 年以上 / **无期** |
| **利用影响力受贿罪** | § 388 之一 | 同受贿罪 | 同受贿罪 |
| **介绍贿赂罪** | § 392 | 1 万 / 3 万 | 3 年以下 / 3 年以上 |

**blocks（绝对禁止）**：
- 受贿 300 万 + 拒不退还 → **死刑（可判）**
- 行贿 500 万 + 多次行贿 → **无期徒刑**
- 挪用公款 500 万 + 不退还 → **10 年以上**

**work but ships（量刑从宽）**：
- 贪污 3 万 + 自首 + 退赃 → **可能免除处罚**
- 受贿 20 万 + 自首 + 立功 + 退赃 → **3-10 年**
- 共同犯罪中从犯 → **从轻 / 减轻**

#### 块 3：自首 / 立功 / 认罪认罚（量刑从宽关键情节）

**自首认定（《刑法》第 67 条）：**

| 类型 | 条件 | 法律效果 |
|------|------|---------|
| **一般自首** | 犯罪后自动投案，如实供述 | 可以从轻 / 减轻，**犯罪较轻的可免除** |
| **特别自首（准自首）** | 被采取强制措施期间如实供述司法机关还未掌握的本人其他罪行 | 同上 |
| **坦白** | 被动归案后如实供述（不构成自首） | 可以从轻 |

**立功认定（《刑法》第 68 条）：**

| 类型 | 条件 | 法律效果 |
|------|------|---------|
| **一般立功** | 揭发他人犯罪行为查证属实 / 提供重要线索协助破案 | 可以从轻 / 减轻 |
| **重大立功** | 揭发 / 提供线索使 重大案件 得以侦破 / 揭发多人多起犯罪 / 其他对国家和社会有较大贡献 | **应当减轻 / 免除** |

**认罪认罚（《刑事诉讼法》第 15 条 + 《指导意见》2019）：**

| 阶段 | 程序 | 法律效果 |
|------|------|---------|
| **侦查阶段** | 认罪 + 同意量刑建议 | 可从宽 30% 左右 |
| **审查起诉** | 签署认罪认罚具结书 | 可从宽 20-30% |
| **审判阶段** | 认罪认罚 + 同意适用速裁 | 可从宽 10-20% 或当庭宣判 |

**blocks（绝对禁止 — 辩护策略）**：
- 伪造自首情节（自首前已掌握犯罪事实）→ **不能认定为自首**
- 教唆立功（事先与同案犯串通）→ **不能认定为立功**
- 翻供后认罪 → **不能适用认罪认罚**

**work but ships（实务补救）**：
- 及时退赃 → **从轻 / 减轻幅度大**
- 提供赃款下落 → **辅助退赃 + 立功**
- 单位犯罪中个人认罪 → **个人从宽 + 单位罚金**

#### 块 4：单位犯罪双轨辩护（白领犯罪高频）

**单位犯罪认定（《刑法》第 30 条 + 司法解释）：**

| 要件 | 含义 |
|------|------|
| **主体** | 公司 / 企事业单位 / 人民团体 |
| **主观** | 单位意志（集体决策 / 负责人决定） |
| **利益** | 归单位所有（不是个人） |
| **程序** | 经单位决策机构决定 |

**单位 + 个人双罚（《刑法》第 31 条）：**

| 责任主体 | 处罚 |
|---------|------|
| **单位** | 罚金（无上限） |
| **直接负责的主管人员** | 自然人刑罚（如单位行贿罪对负责人） |
| **其他直接责任人员** | 自然人刑罚 |

**辩护要点：**

| 辩护点 | 适用情形 |
|--------|---------|
| **不构成单位犯罪** | 个人行为 / 非单位意志 / 利益归个人 |
| **单位犯罪与个人犯罪区分** | 多人共同犯罪 ≠ 单位犯罪 |
| **减轻处罚** | 单位自首 + 立功 + 退赃 |
| **主管人员责任范围** | 未参与决策 / 未执行 / 不知情 |

**blocks（绝对禁止 — 辩护策略）**：
- 串通单位成员作伪证 → **《刑法》第 307 条妨害作证罪**
- 单位转移资产逃避退赃 → **《刑法》第 313 条拒不执行判决罪**

#### 块 5：涉案财物追缴 + 财产保全

**核心法规：**
- 《刑事诉讼法》第 100-105 条（财产保全）
- 《刑法》第 64 条（犯罪物品处理）
- 《监察法》第 23-26 条（监察机关查封扣押）

**涉案财物处理流程：**

| 阶段 | 程序 |
|------|------|
| **侦查阶段** | 监察机关 / 公安查封扣押冻结 |
| **审查起诉** | 检察机关审查 + 移送时附清单 |
| **审判阶段** | 法院判决主刑 + 附加刑 + 追缴清单 |
| **执行阶段** | 处置（变卖 / 拍卖 / 划拨） |

**财产辩护要点：**

| 辩护点 | 适用情形 |
|--------|---------|
| **不属涉案财物** | 个人合法财产 / 第三人财产 |
| **善意取得** | 第三人善意取得（需证明善意 + 合理对价） |
| **估值异议** | 财物估值过高（申请重新评估） |
| **来源合法** | 涉案财物来源合法（举证责任倒置可能） |

**blocks（绝对禁止）**：
- 转移 / 隐匿涉案财物 → **《刑法》第 313 条 + 妨害司法**
- 提供虚假权属证明 → **《刑法》第 307 条妨害作证罪**

#### 块 6：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **伪造 / 毁灭证据 / 串供** | 《刑法》第 307 条妨害作证罪 | 3-7 年有期徒刑 |
| 2. **转移 / 隐匿涉案财物** | 《刑法》第 313 条拒不执行判决罪 | 3 年以下 + 罚金 |
| 3. **律师违规会见（监察阶段）** | 《律师法》第 49 条 | 吊销执照 + 涉嫌犯罪 |
| 4. **伪造自首 / 立功情节** | 司法解释 | 不予认定 + 不从宽 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时间表
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — 白领犯罪增强版）**：
- 涉案罪名 + 涉案金额 + 涉案人数？（实体辩护基础）
- 程序阶段：监察留置 / 公安侦查 / 检察审查起诉 / 法院审判？（决定辩护策略）
- 是否有自首 / 立功 / 退赃情节？（量刑关键）
- 单位犯罪还是个人犯罪？（主体责任划分）
- 涉案财物范围 + 估值是否合理？（财产辩护）
- 是否涉外 / 引渡 / 红色通缉令？（跨境程序）

## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** `criminal-procedure-guide`(刑事程序是白领犯罪辩护逻辑起点)

**关键节点(白领犯罪 4 阶段):**

```
侦查阶段:
  Step 1: criminal-procedure-navigation / investigation-response-advisor
          → 强制措施 / 讯问应对 / 取保候审

起诉阶段:
  Step 2: criminal-procedure-guide
          → 阅卷 / 会见 / 质证

审判阶段:
  Step 3: 罪名对应 skill(fraud / bribery / tax-crime / etc.)
          → 辩护策略 / 量刑情节 / 庭审应对

执行 / 申诉阶段:
  Step 4: recovery-procedure-advisor
          → 赃款追缴 / 申诉 / 减刑假释
```

### § B5 升级触发(7 类)

1. **死刑 / 无期案件** → 主办 + 所务会 + 刑辩专家
2. **监察留置** → 主办 + 刑辩律师 + 家属
3. **单位犯罪 + 跨境** → 主办 + 涉外律师 + 所务会
4. **涉案金额 ≥¥1亿** → 主办 + 所务会
5. **涉外追逃 / 引渡** → 主办 + 涉外律师 + 多法域
6. **群体性 / 涉众** → 主办 + 协办 + 维稳
7. **未成年人 / 孕妇 / 老人** → 主办 + 特别辩护
