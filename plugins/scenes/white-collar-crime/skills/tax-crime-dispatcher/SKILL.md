# 涉税犯罪分流器 (Tax Crime Dispatcher)

## 一、目的与范围

### 1.1 核心目的
对涉税犯罪进行构成要件分析，区分不同罪名，评估行政处理与刑事追诉的边界，实现案件正确分流。

### 1.2 适用范围
- 逃税罪（刑法第201条）
- 抗税罪（刑法第202条）
- 逃避追缴欠税罪（刑法第203条）
- 骗取出口退税罪（刑法第204条）
- 虚开增值税专用发票罪（刑法第205条）

### 1.3 罪名体系总览

| 罪名 | 刑法条款 | 行为特征 | 立案标准 |
|-----|---------|---------|---------|
| 逃税罪 | 第201条 | 不缴或少缴税款 | 10万元以上且占应纳税额10%以上 |
| 抗税罪 | 第202条 | 暴力/威胁方法拒不缴税 | 无数额要求 |
| 逃避追缴欠税罪 | 第203条 | 转移/隐匿财产 | 1万元以上 |
| 骗取出口退税罪 | 第204条 | 虚构出口骗退税 | 5万元以上 |
| 虚开增值税发票罪 | 第205条 | 虚开专用发票 | 1万元以上或造成损失 |

## 二、核心罪名分析

### 2.1 逃税罪（刑法第201条）核心要件

#### 犯罪主体
- 纳税人（包括自然人和单位）
- 扣缴义务人（不缴或少缴已扣税款）

#### 客观行为
| 行为方式 | 具体表现 |
|---------|---------|
| 伪造账簿 | 伪造记账凭证、账本 |
| 变造账簿 | 篡改账目数据 |
| 擅自销毁账簿 | 故意损毁会计凭证 |
| 进行虚假纳税申报 | 少报收入、多报成本 |
| 经税务机关通知申报而不申报 | 收到责令通知后不报 |

#### 数额计算
- 逃税数额：10万元以上（重大）→ 占应纳税额10%以上
- 两年内因逃税受过行政处罚或刑事处罚 → 不受10%限制
- 扣缴义务人不缴少缴已扣税款 → 无10%比例要求

#### 行政前置程序（重要！）
**逃税罪是"行政优先"的罪名**
- 必须经税务机关依法下达追缴通知
- 收到通知后仍未补缴 → 才构成犯罪
- **初犯补缴可不追究刑事责任**（刑法第201条第4款）

### 2.2 骗取出口退税罪（刑法第204条）核心要件

#### 与逃税罪区分
| 区分点 | 逃税罪 | 骗取出口退税罪 |
|-------|-------|---------------|
| 行为性质 | 不缴税 | 骗取已缴税 |
| 税款归属 | 应缴未缴 | 虚构损失骗回 |
| 立案标准 | 10万+/10%+ | 5万元以上 |

#### 行为方式
- 假报出口
- 虚报出口（货物未出）
- 低报价格
- 重复申报
- 虚增出口数量

### 2.3 虚开增值税专用发票罪（刑法第205条）

#### 行为方式（四种虚开）
| 类型 | 表现 |
|-----|------|
| 为他人虚开 | 无真实交易开具发票 |
| 为自己虚开 | 虚构交易开具发票 |
| 让他人为自己虚开 | 接受虚开发票 |
| 介绍他人虚开 | 居间介绍虚开 |

#### 定罪标准
- 虚开数额1万元以上
- 或致使国家税款损失1万元以上
- **虚开增值税普通发票** → 刑法第209条（普通发票犯罪）

## 三、案件分流机制

### 3.1 行政处理与刑事追诉边界

```
涉税违法
├── 10万元以下 → 行政处罚（补缴+滞纳金+罚款）
├── 10万元以上+10%以下 → 行政处理优先
│   ├── 补缴+滞纳金 → 不追究刑事责任（初犯）
│   └── 拒不补缴 → 移送公安机关
└── 10万元以上+10%以上 → 原则上移送（除非初犯补缴）
```

### 3.2 移送标准

| 罪名 | 移送标准 | 特殊情形 |
|-----|---------|---------|
| 逃税罪 | 10万+/10%+ | 初犯补缴可不移 |
| 抗税罪 | 无数额要求 | 暴力抗税即移 |
| 逃避追缴欠税罪 | 1万元以上 | 转移隐匿财产即移 |
| 骗取出口退税罪 | 5万元以上 | 无初犯豁免 |
| 虚开增值税专用发票罪 | 1万元以上 | 无初犯豁免 |

## 四、升级决策门

### 4.1 必须升级情形

1. **涉及行政程序**
   - 收到税务稽查通知
   - 收到税务处理决定书
   - 收到责令限期改正通知书

2. **涉及刑事移送**
   - 税务机关拟移送公安
   - 涉及刑事拘留/逮捕

3. **涉及罪名认定**
   - 逃税 vs. 骗取出口退税区分
   - 虚开专用发票 vs. 普通发票

### 4.2 升级格式

```
【升级咨询】
罪名判断：[逃税罪/骗税罪/虚开发票罪/其他]
主体身份：[纳税人/扣缴义务人/单位]
涉案金额：[具体金额]
行为方式：[具体行为描述]
程序阶段：[行政处理/稽查/移送]
建议行动：[需律师确认事项]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. 税务稽查程序代理（由税务律师提供）
2. 税务行政复议/诉讼
3. 刑事辩护策略（由刑事辩护律师提供）
4. 涉税犯罪辩护
5. 涉及海关的涉税犯罪（归海关管辖）
6. 跨境涉税犯罪
7. 已进入审判阶段的案件
8. 税务争议的协商与和解
## 本 Skill 不涵盖

- 逃税罪的刑事辩护
- 税务稽查陪同
- 税务行政复议
- 发票管理系统的整改
