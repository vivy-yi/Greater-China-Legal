# 行贿受贿犯罪顾问 (Bribery Crime Advisor)

## 一、目的与范围

### 1.1 核心目的
对中国法项下贿赂类犯罪进行构成要件分析，区分不同罪名，评估法律风险与量刑档次。

### 1.2 适用范围
- 受贿罪（刑法第385条）
- 斡旋受贿罪（刑法第388条）
- 利用影响力受贿罪（刑法第388条之一）
- 行贿罪（刑法第389条）
- 对单位行贿罪（刑法第391条）
- 介绍贿赂罪（刑法第392条）
- 单位行贿罪（刑法第393条）

### 1.3 受贿类罪名体系

| 罪名 | 刑法条款 | 主体 | 客观要件 |
|-----|---------|------|---------|
| 受贿罪 | 第385条 | 国家工作人员 | 利用职务便利索取/收受财物 |
| 斡旋受贿 | 第388条 | 国家工作人员 | 利用本人职权/地位形成的便利条件 |
| 利用影响力受贿 | 第388条之一 | 近亲属/关系密切人 | 利用离职/原职务便利 |
| 非国家工作人员受贿 | 第163条 | 公司企业人员 | 利用职务便利索取/收受财物 |

### 1.4 行贿类罪名体系

| 罪名 | 刑法条款 | 主体 | 对象 |
|-----|---------|------|-----|
| 行贿罪 | 第389条 | 一般主体 | 国家工作人员 |
| 对单位行贿罪 | 第391条 | 一般主体 | 单位 |
| 单位行贿罪 | 第393条 | 单位 | 国家工作人员 |
| 介绍贿赂罪 | 第392条 | 一般主体 | 双向介绍 |

## 二、构成要件分析

### 2.1 受贿罪（刑法第385条）核心要件

#### 犯罪主体（身份犯）
- 国家工作人员（刑法第93条）
- 以国家工作人员论的人员（国有单位委派等）
- **关键区分**：劳务 vs. 公务

#### "利用职务便利"认定
| 情形 | 是否属于"利用职务便利" |
|-----|------------------------|
| 本人职务产生的便利 | ✓ |
| 本人职权/地位形成的便利条件 | ✓（斡旋受贿） |
| 原有职权余温（离职后） | ✗（但可能构成利用影响力受贿） |
| 单纯利用工作关系 | ✗ |

#### "为他人谋取利益"要件
- 承诺为他人谋取利益（无论是否实现）
- 正在实施谋取利益
- 已谋取到利益
- **新型受贿**（2016年解释）：
  - 干股分红（未转让股权）
  - 合作投资（真投资 vs. 名义投资）
  - 委托理财（真实亏损 vs. 名义亏损）
  - 赌博（真假赌博）
  - 房屋、汽车等财产的权属变动

### 2.2 行贿罪（刑法第389条）核心要件

#### "谋取不正当利益"认定（2016年解释）
- 违反法律、法规、规章
- 违反政策
- 违反行业规范
- 违背公平、公正原则
- 在招标投标、政府采购等活动中

#### 行贿罪特殊构成
- 一般行贿：谋取不正当利益
- 感情投资型：排除性条件（入职、婚丧嫁娶等）
- 主动行贿 vs. 被迫行贿（特别自首）

### 2.3 数额与量刑档次

| 数额 | 量刑（受贿/行贿） |
|-----|------------------|
| 3万元以上（数额较大） | 三年以下有期徒刑或拘役，并处罚金 |
| 20万元以上（数额巨大） | 三年以上十年以下有期徒刑，并处罚金或没收财产 |
| 300万元以上（数额特别巨大） | 十年以上有期徒刑或无期徒刑，并处罚金或没收财产 |

**情节加重**（2016年解释）：
- 多次受贿、多人受贿、多次行贿
- 为他人谋取职务提拔调整
- 赃款用于非法活动
- 造成恶劣影响/经济损失

## 三、监察委程序衔接

### 3.1 监察委管辖范围
- 贪污贿赂犯罪（监察委专属管辖）
- 滥用职权、玩忽职守犯罪（部分）
- 徇私枉法等司法工作人员犯罪（部分）
- 关联的职务犯罪

### 3.2 留置措施
| 项目 | 内容 |
|-----|------|
| 期限 | 3个月（可延长至6个月） |
| 场所 | 监察委指定居所 |
| 权利 | 律师会见（监视居住期间不行） |
| 家属通知 | 24小时内（除无法通知） |

### 3.3 从宽情节
| 情形 | 法律依据 |
|-----|---------|
| 特别自首（被追诉前主动交代） | 刑法第390条第2款 |
| 重大立功 | 刑法第68条 |
| 坦白（如实供述） | 刑法第67条第3款 |
| 当庭自愿认罪 | 刑诉法解释 |

## 四、升级决策门

### 4.1 必须升级情形

1. **涉及监察委调查**
   - 接到监察委询问/通知
   - 涉及留置措施
   - 涉及主动投案

2. **涉及具体刑事指控**
   - 罪名是否成立存疑
   - 数额认定争议
   - 不正当利益认定争议

3. **涉及辩护策略**
   - 罪轻辩护
   - 无罪辩护
   - 程序性辩护

### 4.2 升级格式

```
【升级咨询】
罪名判断：[受贿罪/行贿罪/其他]
主体身份：[国家工作人员/非国家工作人员/单位]
案情摘要：[核心事实]
争议焦点：[主要法律问题]
监察阶段：[是否涉及监察委]
建议行动：[需律师确认事项]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. 刑事辩护策略（由刑事辩护律师提供）
2. 监察委调查阶段的具体应对策略
3. 污点证人/污点公司策略
4. 涉及港澳台的贿赂案件
5. 涉外反腐败（FCPA等）
6. 已进入审判阶段的案件
7. 刑事和解程序
8. 具体涉案财物的追缴与处置
## 本 Skill 不涵盖

- 受贿/行贿罪的刑事辩护
- 监察委调查程序
- 违法所得的没收程序
- 单位合规整改方案
