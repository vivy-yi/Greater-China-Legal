# 涉税犯罪风险审查器 (Tax Crime Risk Checker)

## 一、基本信息

### 1.1 核心目的
为企业/高净值个人提供涉税犯罪风险审查与自查自纠指引，覆盖逃税罪、抗税罪、逃避追缴欠税罪、骗取出口退税罪、虚开增值税专用发票罪的构成要件分析与风险等级评估。

### 1.2 适用范围
- 逃税罪（刑法第201条） — 纳税人/扣缴义务人采取欺骗隐瞒手段
- 抗税罪（刑法第202条） — 以暴力威胁方法拒不缴纳税款
- 逃避追缴欠税罪（刑法第203条） — 纳税人转移隐匿财产
- 骗取出口退税罪（刑法第204条） — 以假报出口骗取退税款
- 虚开增值税专用发票罪（刑法第205条） — 为他人/为自己/让他人为自己/介绍他人虚开
- 虚开发票罪（刑法第205条之一） — 虚开普通发票

## 二、分析框架

### 2.1 逃税罪构成要件矩阵

| 要件 | 内容 | 审查要点 |
|------|------|---------|
| 主体 | 纳税人/扣缴义务人（单位+直接责任人员） | 纳税主体资格认定 |
| 行为 | 欺骗隐瞒手段：伪造变造记账凭证/隐匿销毁账簿/多列支出/少列收入/虚假纳税申报/拒不申报 | 行为持续性和故意性 |
| 结果 | 逃避缴纳税款数额较大+占应纳税额10%以上 | 累计计算+连续计征 |
| 主观 | 故意（过失不构成本罪） | 是否出于逃税故意 |
| 阻却事由 | 补缴税款+滞纳金+已受行政处罚（初犯免刑） | 5年内是否因逃税受过处罚 |

**逃税罪数额档次**：

| 档次 | 逃税数额 | 占比 | 量刑 |
|------|---------|------|------|
| 数额较大 | 5万元以上 | 占应纳税额10%以上 | 3年以下有期徒刑或拘役+罚金 |
| 数额巨大 | 10万元以上 | 占应纳税额30%以上 | 3-7年有期徒刑+罚金 |

### 2.2 虚开增值税专用发票罪核心审查

| 审查维度 | 认定标准 | 企业自查要点 |
|---------|---------|-------------|
| "虚开"认定 | 无真实交易/有真实交易但金额或品名不符/让他人为自己虚开 | 核实每笔交易的合同/物流/资金流是否一致 |
| "三流一致" | 货物流+发票流+资金流必须一致 | 检查进项发票与采购合同/入库单/付款凭证是否对应 |
| 主观目的 | 不以税款抵扣为目的的虚开（挂靠/代开）可能不构罪 | 是否有真实货物交易背景 |
| 数额认定 | 虚开税额（非票面金额） | 区分不含税金额计算 |
| 单位犯罪 | 单位判处罚金+直接责任人处刑 | 实际控制人+财务负责人+直接经办人 |

**虚开增值税专用发票罪量刑标准**（法〔2018〕226号）：

| 档次 | 虚开税额 | 量刑 |
|------|---------|------|
| 数额较大 | 5万元以上 | 3年以下有期徒刑或拘役+2-20万罚金 |
| 数额巨大 | 50万元以上 | 3-10年有期徒刑+5-50万罚金 |
| 数额特别巨大 | 250万元以上 | 10年以上有期徒刑或无期徒刑+5-50万罚金或没收财产 |

### 2.3 刑事追诉起点汇总

| 罪名 | 追诉起点 | 升档标准 | 最高刑 |
|------|---------|---------|-------|
| 逃税罪 | 5万元+10%占比 | 10万元+30% | 7年 |
| 抗税罪 | 不以数额论（暴力手段即构成） | 手段特别残忍+重伤/死亡 | 7年 |
| 逃避追缴欠税罪 | 1万元以上 | 10万元以上 | 7年 |
| 骗取出口退税罪 | 10万元 | 50万元/250万元 | 无期 |
| 虚开增值税专用发票罪 | 5万元 | 50万元/250万元 | 无期 |
| 虚开发票罪 | 100份/40万元 | 500份/200万元 | 7年 |

## 三、操作流程

### 3.1 涉税犯罪风险自查五步法

**Step 1：信息收集与初步评估**
- [ ] 收集近3年纳税申报资料+财务报表+发票台账
- [ ] 了解公司经营模式（是否涉及关联交易/跨境交易/平台经济）
- [ ] 初步判断涉税风险领域（企业所得税/增值税/个税/其他）
- [ ] 确定风险等级（高/中/低）

**Step 2：发票合规审查**
- [ ] 进项发票逐笔核查（是否存在无真实交易的发票）
- [ ] 销项发票开票是否与实际交易一致
- [ ] 是否存在"票货分离"情形
- [ ] 是否存在"挂靠经营"虚开风险
- [ ] 个人消费发票冲抵公司成本

**Step 3：纳税申报合规审查**
- [ ] 收入确认是否完整（有无账外收入/私卡收款）
- [ ] 成本费用是否真实（有无虚列成本/虚假费用）
- [ ] 关联交易定价是否公允（转让定价风险）
- [ ] 享受税收优惠是否合规（高新技术企业/研发加计扣除/小微企业）

**Step 4：自查结论与风险分级**

| 风险级别 | 表现 | 建议行动 |
|---------|------|---------|
| 🔴 高风险 | 涉嫌刑事追诉标准 | 立即聘请专业律师+准备自查补报 |
| 🟡 中风险 | 存在较大税务违规 | 自查补税+主动沟通税务机关 |
| 🟢 低风险 | 一般合规瑕疵 | 内部整改+完善内控 |

**Step 5：自查自纠行动**
- [ ] 主动补缴税款+滞纳金（享受初犯免刑保护）
- [ ] 主动向税务机关报告（坦白从宽）
- [ ] 配合税务稽查（不得销毁隐匿证据）
- [ ] 自查报告的书面记录和存档

### 3.2 逃税罪"初犯免刑"条件

根据刑法第201条第4款，逃税罪初犯满足以下条件的，可以不予追诉：

| 条件 | 要求 | 操作注意 |
|------|------|---------|
| 补缴税款 | 全额补缴逃税税款 | 须有税务机关出具的完税证明 |
| 缴纳滞纳金 | 按日万分之五计算 | 补缴滞纳金不得减免 |
| 接受行政处罚 | 接受罚款（逃税税款50%至5倍） | 缴纳后取得行政处罚决定书 |
| 5年内未受过刑事处罚 | 未因逃税受过刑事处罚/行政处罚 | 如有前科则不适用 |
| 申报被追究前补缴 | 在公安机关立案前/立案后判决前 | 立案前补缴的效力最强 |

**注意**：增值税专用发票的虚开犯罪不适用"初犯免刑"保护。

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **已被税务稽查立案** — 税务机关已启动稽查程序
- **已被公安经侦立案或采取强制措施** — 涉及刑事侦查
- **涉嫌虚开增值税专用发票且税额超过50万元** — 可能面临10年以上刑罚
- **涉及"两套账"（账外账/小金库）** — 情节严重
- **涉及跨境逃税/隐瞒境外资产** — CRS信息交换+外汇管理
- **涉及行政负责人/实控人个人刑责** — 实控人+财务负责人均可能被追诉
- **涉及骗取出口退税** — 属严厉打击范围
- **涉及单位犯罪** — 单位+直接责任人双罚制

### 4.2 升级咨询模板

```
【升级咨询】
主体：[企业名称/个人]
涉嫌罪名：[逃税/虚开/骗取出口退税/抗税/其他]
涉案金额/税额：[万元]
税务阶段：[正常申报/稽查中/已立案/已移送公安]
自查发现：[发现的违规情况]
已补税情况：[已补缴金额/未补缴]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. 税务稽查应对策略（须税务律师+会计师配合）
2. 税务行政处罚听证/复议/诉讼代理
3. 涉税犯罪的刑事辩护（须刑事辩护律师）
4. 转让定价调整/预约定价安排（税务专业）
5. 国际税收协定的适用分析
6. 海关进出口的关税/走私犯罪
7. 环保税/资源税等专业税种的特殊规定
