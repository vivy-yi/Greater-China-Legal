# /customs-valuation-checker — 海关估价合规审查

## 核心概念

海关估价决定了进口货物的关税、增值税和消费税基。根据WTO《海关估价协定》（中国2001年加入）和《海关审定进出口货物完税价格办法》（海关总署令第213号），海关估价适用六种方法，且必须按顺序适用：成交价格法 → 相同货物成交价格法 → 类似货物成交价格法 → 倒扣价格法 → 计算价格法 → 合理方法法。成交价格法是首选——就是发票上显示的实付/应付价格，调整后加上运费/保费等入账。

成交价格法适用的最大门槛是"不存在影响价格的限制条件"和"买卖双方之间不存在关联关系"——或即使存在关联，但成交价格"近似于"可比交易价格。根据海关相关规定，关联方之间的转让定价（transfer pricing）在海关估价中常被质疑，核心抗辩策略是证明（1）关联关系未影响价格，或（2）成交价格已向海关充分披露且海关认可。低报价格的直接后果是补税+滞纳金+罚款（低报税款的30%-200%），严重时可追究刑事责任（走私罪）。

## 工作流程

### 第一步：确认适用的估价方法

- 确认交易是否符合"成交价格法"的适用条件（《完税价格办法》第7-13条）：
  - 是否有对货物处置或使用的限制（如限定转售价格）
  - 销售价格是否受某些条件或因素影响（如后续转售收益分成）
  - 买卖双方是否存在关联关系
  - 如是关联交易，成交价格是否"可接受"（通过测试价格或交易环境审查）
- 如成交价格法不适用，按顺序判断下一种方法：
  - 相同货物成交价格法（同一出口国/同一时间/商业水平近似）
  - 类似货物成交价格法（功能近似/可互换）
  - 倒扣价格法（进口后的转售价-利润-税费）
  - 计算价格法（生产成本+利润+费用）
  - 合理方法法（灵活性最大，需与海关协商）

### 第二步：成交价格调整计算

- 确认应计入完税价格的调整项（《完税价格办法》第15条）：
  - 买方承担的佣金和经纪费（购货佣金除外）
  - 容器和包装费用
  - 买方免费或低价提供的协助价值（材料/工具/模具/设计等）
  - 特许权使用费（与进口货物相关且构成销售条件）
  - 转售/处置收益中返还给卖方的部分
  - 运费至输入地点+保险费（CIF条款下的默认构成）
- 确认不应计入完税价格的项目：购货佣金、进口后的建设/安装/维护费、进口关税和国内税
- 制作《价格调整计算表》

### 第三步：关联交易专项审查

- 分析买卖双方的关联关系类型：母子公司、同一控制下的关联企业、董事/高管关联、家族关联
- 如有关联关系，准备如下证据证明价格可接受（四项测试任选其一）：
  1. 关联交易价格与相同/类似货物的非关联交易价格近似
  2. 从相同出口国进口的相同/类似货物的海关估价近似
  3. 进口后的转售价（倒扣法测试）近似
  4. 生产成本+利润+费用（计算法测试）近似
- 如果转让定价调整（TP adjustment）导致进口价格变动，评估是否需要向海关补申报
- 对跨国企业，评估海关估价与税务转让定价之间的"双重合规"策略

### 第四步：海关质疑应对

- 如海关提出估价质疑（发出《价格质疑通知书》），评估海关质疑的合理性：
  - 海关是否提供了具体的价格异常理由
  - 海关是否提出了替代估价方法
  - 海关质疑是否基于关联关系而"自动"否定成交价格法
- 准备抗辩材料：同期可比交易价格、行业价格水平、成本构成分析、关联关系不影响价格的证明
- 如争议无法在质疑阶段解决，准备向海关提出行政复议或后续的行政诉讼
- 评估补税风险，并按《行政处罚法》评估是否构成"主动披露"（主动披露可以减免罚款）

### 第五步：合规建议出具

- 对估价方案出具整体合规评价：是否可安全使用成交价格法、是否需要预裁定（advance ruling）、是否存在补税风险敞口
- 建议企业改进的估价管理措施：完善关联交易定价文档、建立价格备案制度、申请海关AEO认证（高级认证企业享受较低查验率）等

## 升级决策门

出现以下情形须移交专业海关法律师处理：
- 海关已发出《价格质疑通知书》或《价格磋商通知书》
- 涉及跨境关联交易中的转让定价争议，且涉及多个关联实体
- 海关估价与税务转让定价存在直接冲突（如海关不接受TP调整）
- 低报价格金额重大（应缴税款50万元以上）可能构成走私罪
- 涉及特许权使用费的"是否与进口货物相关"复杂判断
- 需要申请海关预裁定（advance ruling on valuation）
