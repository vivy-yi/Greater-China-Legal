# /engagement-and-meeting — 接受委托 + 首次会见

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取委托规范 + 利益冲突规则 + 法定时限。

## 工作流程

```
Step 1: 家属/当事人咨询 → 利益冲突审查
Step 2: 评估案件 → 决定是否接受委托
Step 3: 签署委托书 + 风险告知书
Step 4: 申请会见(48h 内,重大案件 24h 内)
Step 5: 首次会见(看守所现场)
Step 6: 会见笔录归档
```

## 利益冲突审查(必查)

| 冲突类型 | 检查项 | 状态 |
|----------|--------|------|
| 同案犯代理 | 是否同时代理同案其他被告人 | 严禁 |
| 对方当事人代理 | 是否曾代理被害人 | 严禁 |
| 律所冲突 | 律所其他律师是否已代理相关方 | 必查 |
| 前任代理 | 是否曾在同一案件中代理对方 | 严禁 |

## 委托书必备条款

```
1. 委托人(家属)信息
2. 委托事项(具体罪名)
3. 委托阶段(侦查/审查起诉/审判)
4. 委托权限(一般代理 / 全权代理)
5. 律师费 + 收费方式(法援/政府指派/风险代理禁止情形)
6. 双方签字 + 日期
```

## 首次会见流程

**会见前准备:**
- 携带:律师证 + 委托书 + 会见专用介绍信 + 风险告知书
- 核实:关押地点 + 是否变更

**会见中(严禁事项):**
- ❌ 不得携带信件/现金/物品
- ❌ 不得约定虚假口供
- ❌ 不得私自录音录像
- ✅ 充分告知权利(沉默权 + 律师权 + 申诉控告权)
- ✅ 了解案件事实经过
- ✅ 听取当事人意见 + 解答疑问
- ✅ 商定辩护方向

**会见后:**
- 24h 内形成会见笔录
- 笔录双方签字
- 归档至律所案卷系统
