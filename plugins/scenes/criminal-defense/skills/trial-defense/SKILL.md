# /trial-defense — 庭审辩护

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取风险等级 + § B2 路由表 + § A6 决策姿态。

## 工作流程

```
Step 1: 庭前会议(开审前与公诉人/法官沟通)
Step 2: 庭审准备(辩护词定稿 + 质证提纲 + 发问提纲)
Step 3: 法庭调查(质证 + 申请调取新证据)
Step 4: 法庭辩论(辩护人发表辩护意见)
Step 5: 最后陈述(被告人本人)
Step 6: 庭后总结 + 上诉评估
```

## 庭前会议准备(必做)

**提前 3-5 个工作日与法官沟通:**

| 议题 | 律师动作 |
|------|----------|
| 非法证据排除 | 提交书面申请 + 证据清单 |
| 申请调取新证据 | 提交书面申请 + 必要性说明 |
| 申请证人出庭 | 提交证人名单 + 发问提纲 |
| 申请专家辅助人 | 提交专家资质 + 出庭必要性 |
| 管辖异议 / 回避 | 如有理由,书面提出 |
| 庭审直播 / 不公开审理 | 涉隐私 / 商业秘密的,主动申请 |

**与公诉人沟通:**
- 部分地区允许律师查阅公诉人庭审提纲
- 协商证据简化出示(双方无异议的可不出示)
- 协商简化质证(争议证据重点质证)

## 质证策略

**质证三步法:**

```
1. 真实性:此证据是否客观存在?
2. 合法性:此证据取得方式是否合法?
3. 关联性:此证据与本案是否相关?
```

**常见质证要点:**

| 证据类型 | 质证要点 |
|----------|----------|
| 物证 | 来源 / 提取程序 / 是否原物 |
| 书证 | 是否原件 / 形成时间 / 公章真伪 |
| 证人证言 | 亲历性 / 受到诱导 / 与其他证据矛盾 |
| 被告人供述 | 同步录音录像 / 笔录与口供一致性 / 合法性 |
| 鉴定意见 | 鉴定机构资质 / 鉴定方法 / 鉴定材料 |

**质证话术模板:**
```
对证据 X,辩护人有以下质证意见:
1. 关于真实性:...
2. 关于合法性:...
3. 关于关联性:...
综上,辩护人认为该证据不应作为定案依据,请求法庭依法排除。
```

## 法庭发问策略

**对被告人发问(关键):**

| 时机 | 目的 |
|------|------|
| 公诉人发问后 | 补充关键事实 |
| 单独发问 | 强调辩护要点 |

**对证人发问:**

```
询问顺序: 己方证人先问 → 对方证人交叉询问
询问方式: 开放式(避免诱导性问题)
注意事项: 不得威胁 / 诱导 / 故意刁难
```

**对鉴定人发问:**
- 鉴定方法
- 鉴定依据
- 是否考虑反例
- 同类鉴定常见争议

## 法庭辩论

**辩论结构(5 段论):**

```
1. 总体立场(一句话,清晰)
2. 事实认定(争议点 + 证据分析)
3. 定性分析(此罪/彼罪/无罪)
4. 量刑情节(从轻/减轻)
5. 结论(明确请求)
```

**辩论节奏控制:**

| 时间分配 | 占比 |
|----------|------|
| 事实认定 | 40% |
| 定性 + 量刑 | 50% |
| 总结 | 10% |

**特别注意:**
- 紧扣证据,不情绪化
- 公诉人发言错误时,礼貌但坚定指出
- 不得使用侮辱性语言
- 不得直接攻击公诉人个人

## 最后陈述(被告人本人)

**律师提前与被告人沟通:**

```
提示 1:陈述重点是认罪悔罪 / 家庭情况 / 请求从轻
提示 2:避免与辩护策略冲突
提示 3:避免新事实(防止反言)
提示 4:时间 5-10 分钟,言简意赅
```

**最后陈述不得被法官打断**,这是法定权利。

## 庭后总结

**当日完成:**

| 事项 | 时限 |
|------|------|
| 庭审笔录核对签字 | 当日 |
| 庭审情况报告 | 当日 |
| 上诉评估(若判不利) | 5 日内决定 |
| 客户沟通 | 3 日内 |
