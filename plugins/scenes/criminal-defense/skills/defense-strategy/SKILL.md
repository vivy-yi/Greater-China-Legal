# /defense-strategy — 辩护策略制定

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取决策树 + 升级路径 + 7 条设计哲学。

## 工作流程

```
Step 1: 综合阅卷结论 + 当事人意见
Step 2: 评估无罪可能性
Step 3: 选定辩护主线(实体/程序/证据)
Step 4: 量刑情节梳理(从轻/减轻)
Step 5: 认罪认罚评估
Step 6: 形成辩护策略备忘录
Step 7: 主办律师复核 + 当事人确认
```

## 辩护三轨选择

| 辩护方向 | 适用 | 风险 |
|----------|------|------|
| **实体辩护** | 事实清楚 / 罪名无争议 / 主从犯明确 | 主要争取量刑优惠 |
| **程序辩护** | 程序违法线索 / 超期羁押 / 刑讯逼供 | 可能影响定罪 |
| **证据辩护** | 证据矛盾 / 关键证据缺失 / 鉴定异议 | 可能影响事实认定 |
| **混合辩护** | 多条线索并存 | 综合运用 |

## 无罪 vs 罪轻决策树

```
事实清楚 + 证据充分 + 无程序违法
  → 罪轻辩护(争取从轻/减轻)

事实不清 OR 关键证据缺失 OR 重大程序违法
  → 评估无罪辩护可能性

证据充分 + 量刑优惠空间大 + 当事人同意
  → 认罪认罚(从宽 10-30%)
```

**无罪辩护慎重原则:**
- 须有充分证据基础
- 须告知当事人"无罪辩护可能反致从重"风险
- 须同步准备罪轻 fallback

## 量刑情节梳理清单

**法定从轻/减轻情节:**

| 情节 | 法条 | 注意 |
|------|------|------|
| 自首 | 《刑法》第 67 条 | 自动投案+如实供述 |
| 立功 | 《刑法》第 68 条 | 揭发他人犯罪查证属实 |
| 坦白 | 《刑法》第 67 条第 3 款 | 如实供述自己罪行(从轻) |
| 退赃 / 退赔 | 司法解释 | 全部退赃可从轻 |
| 赔偿损失 + 谅解 | 司法解释 | 取得被害人谅解可从轻 |
| 未成年人 | 《刑法》第 17 条 | 必减 |
| 老年人 | 司法解释 | 酌情从轻 |
| 怀孕 / 哺乳 | 《刑法》第 72 条 | 缓刑适用 |
| 从犯 | 《刑法》第 27 条 | 必减 |
| 胁从犯 | 《刑法》第 28 条 | 必减且应当减轻或免除 |
| 预备犯 / 未遂犯 | 《刑法》第 22/23 条 | 可比照既遂从轻/减轻 |
| 精神病人(限定) | 《刑法》第 18 条 | 必减或免责 |
| 又聋又哑 / 盲人 | 《刑法》第 19 条 | 可从轻/减轻/免责 |

**酌定情节:**

| 情节 | 注意 |
|------|------|
| 犯罪动机 | 义愤 / 激愤 vs 谋利 |
| 一贯表现 | 良好 / 劣迹 |
| 犯罪后态度 | 配合 / 抗拒 |
| 被害人过错 | 引发犯罪的因素 |

## 认罪认罚评估

**适用条件(三要件):**

```
1. 犯罪嫌疑人/被告人自愿
2. 承认指控的犯罪事实
3. 同意量刑建议 + 程序适用
```

**量刑优惠:**

| 时点 | 优惠 |
|------|------|
| 侦查阶段 | 可减 30% |
| 审查起诉阶段 | 可减 20% |
| 一审庭审 | 可减 10%(从宽幅度小于早认罪) |

**认罪认罚 + 辩护冲突:** 认罪认罚后仍可作"罪轻辩护"(就量刑幅度辩护),但**不得**作"无罪辩护"。
