# /trademark-infringement-assessment — 商标侵权认定评估

## 加载上下文

首次使用时，读取 `../CLAUDE.md` 获取商标法侵权判断标准、赔偿计算规则和维权路径选择逻辑。

## 核心概念

商标法第57条列举了七种商标侵权行为，核心判断标准是"混淆可能性"——即相关公众是否可能对商品/服务的来源产生混淆。中国商标法下的混淆包括：直接混淆（错认来源）和售后混淆（购买完成后他人看到产品时的来源误认）。另须注意正当使用的抗辩——描述性使用和指示性使用在一定条件下不构成侵权。

## 工作流程

### 第一步：侵权情形初步分类

**商标法第57条七种侵权情形**：

| 条款 | 侵权情形 | 构成要件 |
|------|---------|---------|
| 第57(1)条 | 在同一种商品上使用相同商标（假冒） | ①同种商品 ②相同商标 |
| 第57(2)条 | 在同一种商品上使用近似商标，或在类似商品上使用相同/近似商标，容易导致混淆 | ①同种或类似商品 ②相同或近似商标 ③混淆可能性 |
| 第57(3)条 | 销售侵犯注册商标专用权的商品 | ①销售行为 ②商品属侵权商品 |
| 第57(4)条 | 伪造/擅自制造他人注册商标标识或销售该类标识 | ①制造或销售 ②标识属伪造/擅自制造 |
| 第57(5)条 | 未经同意更换注册商标并投入市场（反向假冒）| ①更换商标 ②投入市场 |
| 第57(6)条 | 故意为他人侵权提供便利（仓储/运输/邮寄/隐匿等）| ①故意 ②为侵权提供便利 |
| 第57(7)条 | 其他损害商标权的行为 | 须符合商标法基本原则 |

### 第二步：混淆可能性分析 `[SME 核查]`

**混淆可能性判断要素**（最高院商标民事纠纷司法解释第10-12条）：

| 要素 | 判断标准 | 对混淆认定的影响 |
|------|---------|----------------|
| 商标近似 | 音（呼叫）、形（外观）、义（含义）整体比对；以相关公众的一般注意力为标准 | 近似度高→混淆可能性高 |
| 商品/服务类似 | 功能、用途、生产部门、销售渠道、消费对象等方面 | 类似度高→混淆可能性高 |
| 商标显著性和知名度 | 显著性越强、知名度越高，保护范围越宽 | 知名度高→混淆判断标准降低 |
| 相关公众的注意程度 | 专业商品的购买者注意程度高于普通消费品 | 注意程度高→混淆可能性低 |

**混淆类型分析**：

| 混淆类型 | 定义 | 中国司法实践态度 |
|---------|------|----------------|
| 直接混淆（来源混淆）| 消费者认为商品来自商标权人或存在关联关系 | ✅ 核心混淆类型 |
| 售后混淆（旁观者混淆）| 购买完成后的第三方看到产品时产生的来源误认 | ⚠️ 少数案例支持，理论有争议 |
| 初始兴趣混淆 | 消费者在购买前被误导产生兴趣，但购买时已清楚来源 | ⚠️ 部分案例参考，非主流 |

**注意**：中国法院对混淆可能性的判断倾向于"综合考虑"原则，不要求同时满足所有要素，而是根据个案情况权衡。

### 第三步：正当使用抗辩分析 `[SME 核查]`

**描述性使用**（商标法第59条第1款）：
```
注册商标中含有的本商品的通用名称、图形、型号，
或者直接表示商品的质量、主要原料、功能、用途、重量、数量及其他特点，
或者含有的地名，注册商标专用权人无权禁止他人正当使用。
```

| 正当使用类型 | 构成要件 | 示例 |
|------------|---------|------|
| 描述性使用 | ①善意使用 ②使用描述性词汇的原意 ③不突出使用 | "苹果"描述水果种类 |
| 指示性使用 | ①必须使用该商标才能说明商品/服务 ②合理范围内使用 ③不暗示与商标权人有特殊关系 | 维修店标明"适用于X品牌" |
| 姓名权使用 | 使用自己的姓名（非恶意）| 法人的名称中的姓名 |

**正当使用的判断标准**：
- 被告是否善意使用（有无搭便车意图）
- 使用方式是否合理（字体大小、位置、是否突出）
- 是否可能导致混淆
- 该使用是否为说明商品/服务的必需

### 第四步：平行进口合法性评估

**平行进口的定义**：
```
商标权人或其被许可人在A国合法投放市场的商品，
被第三方未经许可进口到B国进行销售。
```

**中国对平行进口的态度**（当前司法实践）：

| 类型 | 合法性 | 条件 |
|------|-------|------|
| 相同品牌/相同品质 | ✅ 一般不认定为侵权 | 商品来源合法、品质相同 |
| 商品存在实质性差异 | 🔴 可能构成侵权 | 品质差异导致消费者混淆 |
| 包装/标签不符合中国规定 | ⚠️ 可能有行政合规风险 | 标签/说明书须符合中国法规 |

**平行进口合法性的关键因素**：
- 商品是否由商标权人或其授权方在境外合法投放市场
- 进口商品与境内销售商品是否存在实质性差异
- 进口商是否对商标权人的声誉造成损害
- 进口行为是否违反中国强制性规定（如标签法）

### 第五步：维权路径与策略建议

| 维权路径 | 适用情形 | 优势 | 劣势 |
|---------|---------|------|------|
| 协商和解 | 侵权初期/规模小 | 成本低、速度快 | 约束力弱 |
| 行政投诉（工商/市监局）| 侵权事实清楚 | 速度快、成本低 | 处罚有限、无赔偿 |
| 海关保护（知识产权备案）| 进出口环节 | 可主动扣货 | 须事先备案 |
| 民事诉讼 | 需要赔偿 | 可获赔、可申请禁令 | 周期长、成本高 |
| 刑事报案 | 情节严重（金额大/规模大）| 惩罚力度强 | 程序复杂 |

### 第六步：升级决策门

> "商标侵权认定涉及混淆可能性判断、正当使用抗辩和平行进口等复杂问题。以下情形须移交专业知识产权律师：(1)涉及假冒注册商标罪或销售假冒注册商标商品罪的刑事追诉；(2)涉及法院禁令申请或应对；(3)涉及跨境商标侵权的平行进口问题；(4)涉及驰名商标认定和跨类保护；(5)侵权规模大（多家侵权方/多平台/持续性侵权）；(6)赔偿金额超过500万元或可能适用惩罚性赔偿。本分析不构成正式法律意见。最终侵权认定须由法院或商标行政执法机关作出。"
