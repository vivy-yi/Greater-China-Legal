# /infringement-triage — China Mainland

## Purpose

> **本技能是triage，不是侵权结论。** 侵权分析事实复杂、法律技术性强。未经律师审查即依据triage采取行动（发送警告函、拒绝停止、提起诉讼）可能导致费用承担、不利判决等后果。

---

## 四类权利分析框架

### 商标侵权

**法律依据：**《商标法》第57条、《最高人民法院关于审理商标民事纠纷案件适用法律若干问题的解释》

**混淆可能性因素：**
| 因素 | 说明 |
|---|---|
| 商标近似程度 | 字形/读音/含义/视觉 |
| 商品类似程度 | 同类或类似商品 |
| 商标显著性和知名度 | 独创性越高/知名度越大，保护范围越宽 |
| 相关公众认知 | 目标消费群体识别能力 |
| 实际混淆证据 | 有无实际混淆 |
| 使用意图 | 是否有搭便车故意 |

---

### 著作权侵权

**法律依据：**《著作权法》第52条、《计算机软件保护条例》

**侵权判断步骤：**
1. 权利基础：作品是否享有著作权？是否在保护期内？
2. 接触+实质性相似：被诉方是否有机会接触原作品？是否构成实质性相似？
3. 合理使用抗辩：是否属于《著作权法》第24条的合理使用情形？

**合理使用（须同时满足）：**
- 为个人学习、研究或欣赏
- 为介绍、评论某一作品或说明某一问题
- 为报道时事新闻
- 为学校课堂教学或科学研究
- 其他法定情形

---

### 专利侵权

**法律依据：**《专利法》第11条

**侵权判断（同上fto-triage）：**
1. 确定权利要求保护范围
2. 识别被诉产品技术特征
3. 全面覆盖原则
4. 等同原则

---

### 商业秘密侵权

**法律依据：**《反不正当竞争法》第9条、《民法典》第1235条

**侵权判断：**
1. 秘密性：信息是否为公众所知或容易获取？
2. 保密措施：权利人是否采取了合理保密措施？
3. 侵权方式：非法获取/披露/使用

**CN特殊：员工带走商业秘密的认定**
- 是否签署保密协议
- 是否有竞业限制协议
- 员工离职后的行为是否超出合理范围
