## 侵权判断标准

### 商标侵权（商标法第57条）

| 情形 | 判断标准 |
|---|---|
| 假冒注册商标 | 未经许可在同一种商品上使用相同商标 |
| 销售侵权 | 销售假冒注册商标的商品 |
| 制造侵权 | 伪造/擅自制造他人注册商标 |
| 反向假冒 | 更换他人注册商标后销售 |

### 专利侵权

| 类型 | 判断标准 |
|---|---|
| 发明/实用新型 | 落入权利要求保护范围（全面覆盖/等同特征） |
| 外观设计 | 与外观设计相同或相近似的 |
| 销售/许诺销售/进口 | 未经许可实施专利 |

### 著作权侵权

| 判断标准 | 说明 |
|---|---|
| 实质性相似 | 受保护表达实质性相似 |
| 接触可能性 | 被控侵权人有接触原作品的可能 |

## 维权路径选择

### 按IP类型选择

| 维权路径 | 商标 | 专利 | 著作权 | 商业秘密 |
|---|---|---|---|---|
| 协商和解 | ✅ | ✅ | ✅ | ✅ |
| 行政投诉 | ✅（工商） | ✅（专利行政） | ✅（版权） | ❌ |
| 民事诉讼 | ✅ | ✅ | ✅ | ✅ |
| 刑事报案 | ✅（情节严重） | ✅（情节严重） | ✅（情节严重） | ✅（情节严重） |

### 维权路径选择逻辑

```
第一步：评估侵权类型和严重程度
  → 批量生产/恶意侵权/造成重大损失 → 刑事报案优先
  → 单次/轻微/损失不明确 → 民事诉讼或协商

第二步：评估证据充分性
  → 证据充分 → 直接诉讼
  → 证据不足 → 先行政投诉固定证据，再诉讼

第三步：评估时效
  → 民事诉讼时效3年（知道或应当知道起算）
  → 刑事报案须在追诉时效内
```

## 赔偿计算规则

### 商标侵权赔偿（商标法第63条）

| 顺序 | 计算方式 | 说明 |
|---|---|---|
| 1 | 实际损失 | 权利人实际损失 |
| 2 | 侵权获利 | 侵权人获得的利益 |
| 3 | 参照许可费 | 参照许可费的倍数 |
| 4 | 法定赔偿 | 500元以下→500万元以下（2023年修法前为300万） |

### 专利侵权赔偿（专利法第65条）

| 顺序 | 计算方式 | 说明 |
|---|---|---|
| 1 | 实际损失 | 同上 |
| 2 | 侵权获利 | 同上 |
| 3 | 参照许可费 | 同上 |
| 4 | 法定赔偿 | 1万元以下→500万元以下 |

### 著作权侵权赔偿（著作权法第54条）

| 顺序 | 计算方式 | 说明 |
|---|---|---|
| 1-3 | 同上 | 实际损失/侵权获利/参照许可费 |
| 4 | 法定赔偿 | 500元→500万元 |

### 追问规则（关键）

legal-element-extraction 的输出包含 `## 待补充事实` 节。如果该节非空：

1. **暂停当前分析流程**
2. 向用户逐一提问待补充事实，例如：
   > "请问合同中关于[知识产权归属/数据存储位置/价格调整机制]的条款是什么？这会影响后续判断。"
3. 用户补充后，**回到 Step 0 重新执行 legal-element-extraction**，将新信息并入结构化事实
4. 当待补充事实清空后，继续后续分析

**不得在待补充事实未清空的情况下输出最终结论。** 缺失关键事实的结论标注为「推定结论，须在事实补全后复核」。
