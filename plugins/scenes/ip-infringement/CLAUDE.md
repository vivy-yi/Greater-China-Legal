## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** IP 类型识别 → 专属 skill

**关键节点:**

```
Step 1: 权属确认(注册证书 / 权利登记)
Step 2: 侵权事实认定(产品 / 行为 / 标识)
Step 3: 证据保全(公证 / 鉴定)
Step 4: 评估损失(实际 / 获利 / 法定)
Step 5: 起诉 / 抗辩
```

### § B2 路由表(按 IP 类型)

| 类型 | 主 skill | 关键点 |
|------|----------|--------|
| 商标侵权 | 商标专属 | 近似 + 混淆 |
| 专利侵权 | 专利专属 | 等同原则 + 现有技术 |
| 著作权侵权 | 著作权专属 | 复制权 + 信息网络传播 |
| 不正当竞争 | 反不正当竞争专属 | 反法 2/6/9 条 |
| 商业秘密 | 反法 9 条专属 | 秘密性 / 价值性 / 保密性 |

### § B3 三色风险体系

| 等级 | 案件类型 | 处理 |
|------|----------|------|
| 🔴 HIGH-1 | 驰名商标 + 标准必要 | **主办律师双签** |
| 🔴 HIGH-2 | 涉外 + 大额 | 主办 + 涉外律师 |
| ⚠️ MEDIUM | 一般 IP | 主办律师 |
| ✅ LOW | 简单 IP | 主办律师即可 |

### § B4 风险等级 + 审批路径

| 档位 | 涉案金额 | 须主办律师 | 须所务会 |
|------|----------|-----------|----------|
| 大型 | ≥¥1000万 | 强制 | 强制 |
| 中型 | ¥100万-¥1000万 | 强制 | 建议 |
| 小型 | <¥100万 | 主办即可 | 可选 |

### § B5 升级触发(7 类)

1. **驰名商标 / 标准必要** → 主办 + 律所审批
2. **涉外** → 主办 + 涉外律师
3. **标准必要专利(SEP)** → 主办 + 行业专家
4. **大额** → 主办 + 律所审批
5. **群体侵权** → 主办 + 群体诉讼
6. **涉刑民交叉** → 主办 + 刑事律师
7. **涉外驰名商标** → 主办 + 涉外律师 + 律所审批

### § B14 Risk calibration

**段 1 识别:** 权属风险 / 侵权认定风险 / 损失计算风险 / 涉外风险 / 重复侵权风险
**段 2 量化:** 概率 × 影响 + 缓释
**段 3 响应:** 接受 / 缓释(和解)/ 转移(临时禁令)/ 规避(无效)

### § B15 7 条设计哲学(IP 特化)

1. **权属先行** — 必须有有效权利
2. **侵权认定** — 客观行为 + 主观故意
3. **损失计算** — 实际 / 获利 / 法定
4. **禁令措施** — 及时制止
5. **鉴定评估** — 必要时借助专家
6. **涉外特别** — 国际公约 + 注册国
7. **群体维权** — 成本 / 效率
