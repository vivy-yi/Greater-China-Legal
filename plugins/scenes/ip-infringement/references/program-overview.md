## 1. 商标侵权深度

### 1.1 商标侵权 4 种情形（商标法 57 条）

| 情形 | 判断标准 | 赔偿加重 |
|------|---------|---------|
| **假冒注册商标** | 未经许可在同一种商品上使用相同商标 | 5 倍以下 / 情节严重 5 倍以上 |
| **销售侵权** | 销售假冒注册商标的商品 | 销售不知道+证明合法来源 → 不承担 |
| **制造侵权** | 伪造/擅自制造他人注册商标标识 | 同上 |
| **反向假冒** | 更换他人注册商标后销售（OEM 场景） | 同上 |

### 1.2 商标近似判断 6 因素

| 因素 | 权重 |
|------|------|
| 商标外观 | 高 |
| 商标读音 | 高 |
| 商标含义 | 中 |
| 商品类别 | 高 |
| 销售渠道 | 中 |
| 消费者认知 | 高 |

### 1.3 驰名商标特殊保护（商标法 13 条）

- 不注册也可获得保护
- 跨类别保护（即使未注册类别）
- 5 倍以下赔偿 / 情节严重 5 倍以上

**认定路径**:
1. 商标局/商标评审委员会认定(行政)
2. 法院在个案中认定(司法)
3. 重点考虑：相关公众知晓程度 / 持续时间 / 宣传力度 / 受保护记录

### 1.4 商标赔偿计算（商标法 63 条）

| 顺序 | 计算方式 | 上限 |
|------|---------|------|
| 1 | 实际损失 | — |
| 2 | 侵权获利 | — |
| 3 | 参照许可费倍数 | 1-3 倍 / 合理倍数 |
| 4 | 法定赔偿 | 500 万以下(2023 修法) |
| 5 | 惩罚性赔偿 | 5 倍以下 / 情节严重 5 倍以上 |

## 2. 专利侵权深度

### 2.1 专利侵权判定 2 大原则

| 原则 | 适用 | 说明 |
|------|------|------|
| **全面覆盖原则** | 发明 / 实用新型 | 被控侵权产品/方法包含权利要求全部技术特征 |
| **等同原则** | 发明 / 实用新型 | 等同替换(手段/功能/效果基本相同)→ 仍构成侵权 |
| **外观设计相同/相近似** | 外观设计 | 与外观设计相同或相近似 |

### 2.2 专利侵权 5 种行为（专利法 11 条）

| 行为 | 性质 |
|------|------|
| **制造** | 直接侵权 |
| **许诺销售** | 直接侵权 |
| **销售** | 直接侵权 |
| **进口** | 直接侵权 |
| **使用方法** | 发明专利直接侵权 / 其他专利不构成 |

### 2.3 专利无效宣告（专利法 45 条）

**无效宣告请求人**: 任何单位或个人
**请求时限**: 自专利授权日起任何时间
**无效理由**:
- 不符合新颖性/创造性/实用性
- 公开不充分
- 修改超范围
- 权利要求不清楚/不支持
- 违反法律/公序良俗

### 2.4 标准必要专利(SEP) 特殊规则

**FRAND 原则**（公平、合理、非歧视）：
- SEP 持有人须以 FRAND 条款许可
- 被许可人须善意谈判
- 反垄断风险：拒绝许可 / 搭售 / 价格歧视 → 可能违反反垄断法

**关键案例**:
- 华为 v. 三星（中国/美国）
- OPPO v. Nokia（中国/德国）
- Apple v. Qualcomm（美国）

### 2.5 专利赔偿计算（专利法 65 条）

| 顺序 | 计算方式 | 上限 |
|------|---------|------|
| 1 | 实际损失 | — |
| 2 | 侵权获利 | — |
| 3 | 参照许可费倍数 | 1-3 倍 / 合理倍数 |
| 4 | 法定赔偿 | 1 万 - 500 万 |

## 3. 著作权侵权深度

### 3.1 著作权侵权判定 2 步

| 步骤 | 判断 |
|------|------|
| **接触可能性** | 被告有合理机会接触原告作品（公知/已发表） |
| **实质性相似** | 受保护表达实质性相似 |

### 3.2 著作权侵权 9 种行为（著作权法 53 条）

| 行为 |
|------|
| 未经许可复制、发行、表演、放映、广播、汇编、通过信息网络传播 |
| 歪曲、篡改他人作品 |
| 剽窃他人作品 |
| 制作、出售假冒他人署名的作品 |
| ... |

### 3.3 软件著作权侵权判定

| 维度 | 判定 |
|------|------|
| 代码相似度 | 源程序 / 目标程序 |
| 独创性 | 保护表达，不保护思想 |
| 合理使用 | 学习/研究/教学 |
| 开源协议 | GPL/MIT/Apache 影响 |

### 3.4 开源合规关键

| 协议 | 关键条款 | 商用 | 分发义务 |
|------|---------|------|---------|
| **GPL v3** | Copyleft | ✅ | 必须开源衍生作品 |
| **LGPL** | 弱 Copyleft | ✅ | 库本身开源 |
| **MIT** | 宽松 | ✅ | 保留版权声明 |
| **Apache 2.0** | 宽松 + 专利 | ✅ | 保留声明 + 专利授权 |

**GPL 传染性**:使用 GPL 组件 → 衍生作品也必须 GPL → 商业秘密代码暴露风险

### 3.5 网络传播权特殊规则

| 规则 | 内容 |
|------|------|
| **避风港原则** | 平台不知道侵权 + 未获利 + 接到通知后删除 → 不承担责任 |
| **红旗原则** | 明显侵权 + 平台未删除 → 承担责任 |
| **通知-删除规则** | 权利人通知 + 平台删除 → 平台免责 |

## 4. 商业秘密深度

### 4.1 商业秘密三性测试（反不正当竞争法 9 条）

| 要素 | 含义 |
|------|------|
| **秘密性** | 不为公众所知悉（非公知信息） |
| **价值性** | 具有商业价值（能为权利人带来经济利益） |
| **保密性** | 已采取保密措施（合理保护措施） |

### 4.2 商业秘密侵权 5 种行为（反不正当竞争法 9 条）

| 行为 | 方式 |
|------|------|
| 以盗窃、贿赂、欺诈、胁迫 | 不正当手段获取 |
| 披露、使用或允许他人使用 | 上述获取的商业秘密 |
| 违反保密义务 / 违反权利人要求 | 披露、使用 |
| 教唆、引诱、帮助他人 | 违反保密义务 |
| 第三人明知或应知 | 仍获取/披露/使用 |

### 4.3 商业秘密 vs 专利 vs 著作权

| 维度 | 商业秘密 | 专利 | 著作权 |
|------|---------|------|--------|
| 取得方式 | 自动（采取保密措施） | 申请+授权 | 自动（创作完成） |
| 保护期限 | 永久（持续保密） | 20 年（发明）| 作者终生 + 50 年 |
| 是否公开 | 秘密 | 公开 | 公开 |
| 保护对象 | 技术信息/经营信息 | 技术方案 | 作品表达 |
| 维权举证 | 三性 + 侵权 + 损失 | 权利要求 + 侵权 + 损失 | 作品 + 接触 + 相似 + 损失 |

## 5. 维权路径深度

### 5.1 4 类维权路径对比

| 路径 | 适用 | 优势 | 劣势 |
|------|------|------|------|
| **协商和解** | 所有 IP 类型 | 快速 / 低成本 / 灵活 | 无强制执行力 |
| **行政投诉** | 商标 / 专利 / 著作权 / 不包括商业秘密 | 快速 / 处罚 / 行政保护 | 赔偿有限 / 无终局裁决 |
| **民事诉讼** | 所有 IP 类型 | 赔偿 / 停止侵权 / 终局 | 时间长 / 成本高 / 举证难 |
| **刑事报案** | 情节严重 | 强威慑 / 罚金 | 高标准 / 影响有限 |

### 5.2 行政投诉路径

| IP 类型 | 行政主管机关 | 程序 |
|---------|------------|------|
| **商标** | 工商(市场监管) | 投诉 → 立案 → 调查 → 处罚 |
| **专利** | 知识产权局(地方) | 投诉 → 立案 → 调查 → 处罚 |
| **著作权** | 版权局(文旅) | 投诉 → 立案 → 调查 → 处罚 |
| **商业秘密** | ❌ 无行政路径 | 只能民事/刑事 |

### 5.3 民事诉讼路径

**管辖**:
- 侵权行为地 / 被告住所地
- 网络侵权：服务器所在地 / 原告发现侵权地

**程序**:
1. 起诉 + 证据保全(可选)
2. 诉前禁令(可选)
3. 庭前调解
4. 开庭审理
5. 一审判决
6. 上诉 + 二审
7. 执行

**诉前禁令**:权利人证明如不及时制止将造成难以弥补损害 → 法院 48 小时内裁定

### 5.4 跨境电商 IP 维权

| 平台 | 投诉路径 |
|------|---------|
| **亚马逊** | Brand Registry / Report IP Violation |
| **eBay** | VeRO (Verified Rights Owner) |
| **速卖通** | 阿里巴巴知识产权保护平台 |
| **Wish** | IP 投诉 |
| **Shopify** | DMCA Notice |

**关键**:
- 必须先有目的国商标/专利/著作权
- 投诉需提供权利证明 + 侵权证据
- 平台审核 3-7 个工作日

### 5.5 海运跨境 IP 侵权

| 环节 | 程序 |
|------|------|
| **海关知识产权备案** | 海关总署备案(商标/专利/著作权) |
| **进出境环节扣留** | 海关主动发现侵权 → 扣留 → 调查 |
| **调查处理** | 海关调查 → 没收 + 销毁 + 罚款 |
