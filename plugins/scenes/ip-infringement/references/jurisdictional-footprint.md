# 法域分层与跨境升级规则 — ip-infringement

> 来源:scene-claudemd-curator CLAUDE.md § 9.1 + Pattern 16/17
> 适用:跨境 IP 维权、海外商标/专利申请、SEP/UPC、跨境电商 IP 保护
> **规则:跨境必查 [域外] + 升级路径按金额分层**

---

## 1. 法域分层(5 大法域)

### 1.1 中国大陆（默认法域）

| 维度 | 内容 |
|------|------|
| 主管机关 | 国家知识产权局(CNIPA) / 国家版权局 / 海关总署 |
| 主要 IP 法律 | 商标法 / 专利法 / 著作权法 / 反不正当竞争法 |
| 域外法适用 | 一般不适用,仅跨境场景触发 |
| 升级路径 | 见 § 1.5 跨境 4 档 |

### 1.2 香港

| 维度 | 内容 |
|------|------|
| 主管机关 | 知识产权署(IPD) |
| 主要法律 | 商标条例 / 专利条例 / 注册外观设计条例 / 版权条例 |
| 与内地关系 | 一国两制(独立 IP 体系) |
| 高度关注 | 独立商标/专利体系 / 内地注册不自动延及香港 |

### 1.3 美国

| 维度 | 内容 |
|------|------|
| 主管机关 | USPTO(专利商标)/ Copyright Office / ITC |
| 主要法律 | Lanham Act(商标)/ 35 USC(专利)/ 17 USC(版权) |
| 高度关注 | 美国诉讼成本高 / 陪审团制度 / ITC 337 调查 |
| 关键门槛 | ITC 337 调查:美国国内产业存在 |

### 1.4 欧盟

| 维度 | 内容 |
|------|------|
| 主管机关 | EUIPO(商标外观)/ EPO(专利)/ EU 法院 |
| 主要法律 | EU 商标条例 / 欧洲专利公约(EPC) |
| 高度关注 | 统一专利法院(UPC)2023 年 6 月 1 日生效 |
| 关键门槛 | UPC 管辖:参加 UPC 协议的国家(德/法/意等 17 国) |

### 1.5 日本 / 韩国 / 东盟

| 国家/地区 | 关键制度 |
|---------|---------|
| **日本** | JPO / 特许法 / 实用新案法 / 商标法 / 著作权法 |
| **韩国** | KIPO / 专利法 / 商标法 / 著作权法 |
| **新加坡** | IPOS / 商标法 / 专利法 |
| **东盟** | 东盟知识产权合作框架(ASEAN IP) |

## 2. [域外] 法源清单（Pattern 16 高度关注）

### 2.1 多边/区域

| 法源 | 范围 | 主要内容 |
|------|------|---------|
| **Paris Convention** | 全球 | 工业产权国际保护(1883) |
| **Berne Convention** | 全球 | 文学艺术作品保护(1886) |
| **PCT(Patent Cooperation Treaty)** | 全球 | 国际专利申请(1970) |
| **Madrid Protocol** | 全球 | 国际商标注册(1989) |
| **Hague Agreement** | 全球 | 工业品外观设计国际注册 |
| **UPC(Unitary Patent)** | 欧盟 17 国 | 统一专利 + 统一专利法院(2023) |

### 2.2 重点国家

| 国家 | 法源 | 关键条款 |
|------|------|---------|
| 美国 | Lanham Act (15 USC § 1051) | 联邦商标注册 + 侵权救济 |
| 美国 | 35 USC | 专利法 + 侵权救济 |
| 美国 | 17 USC | 著作权法 + DMCA |
| 美国 | 19 USC § 1337 (ITC 337) | 不公平进口 + IP 侵权 |
| 欧盟 | EU Trademark Regulation 2017/1001 | 欧盟商标 |
| 欧盟 | European Patent Convention | 欧洲专利 |
| 日本 | 特許法 | 日本专利法 |
| 新加坡 | Patents Act / Trade Marks Act | IPOS 注册 |

### 2.3 跨境 IP 协定清单

| 协定对方 | 内容 |
|---------|------|
| 中国-PCT 成员国 | 国际专利申请 → 进入国家阶段 |
| 中国-马德里成员国 | 国际商标注册 → 领土延伸 |
| 中国-伯尔尼成员国 | 著作权自动保护(出版即保护) |
| 中国-巴黎公约成员国 | 优先权(12 个月商标/专利) |

## 3. 跨境升级规则(Pattern 17)

### 3.1 升级触发(命中即 🔴 升级)

| 触发情形 | 升级目标 | 触发条件 |
|---------|---------|---------|
| **ITC 337 调查** | 美国 IP 律师 | 涉及美国进口产品 |
| **UPC 诉讼** | 欧洲专利律师 | 涉及欧洲专利 + UPC 协议国 |
| **跨境电商大规模投诉** | 跨境 IP 律师 + 平台律师 | 涉及亚马逊/eBay 等 |
| **国际专利申请(PCT)** | 国际专利律师 | 多国布局 |
| **国际商标注册(马德里)** | 国际商标律师 | 多国布局 |
| **涉外商业秘密刑事** | 国际 IP 律师 | 涉外员工/跨境窃取 |
| **SEP / FRAND 谈判** | SEP 专业律师 + 反垄断律师 | 标准必要专利 |
| **美国诉讼管辖** | 美国 IP 律师 | 被告在美国/产品销往美国 |

### 3.2 跨境 4 档升级路径

```
Level 1 (junior): 跨境 IP < 10 万赔偿 / 单次侵权
  → senior via 邮件

Level 2 (senior): 跨境 IP 10-100 万 + 批量 / 跨境电商
  → gc via 邮件+会议

Level 3 (gc): 跨境 IP > 100 万 / SEP / 标准必要
  → ceo via 邮件+董事会

Level 4 (ceo): 跨境 IP 重大 / ITC 337 / UPC / 上市公司
  → board via 会议+文件 + 跨境 IP 律师介入
```

### 3.3 跨境决策树

```
跨境 IP 案件 / 海外申请 / 跨境电商
  ↓
[Step 1] 涉及金额？
  ├─ < 10 万 → Level 1
  ├─ 10-100 万 → Level 2
  └─ > 100 万 → Level 3

[Step 2] 是否涉及特殊情形？
  ├─ ITC 337 / UPC → Level 4 + 美国/欧洲 IP 律师
  ├─ SEP / FRAND → Level 4 + SEP 专业律师
  ├─ 跨境电商大规模 → Level 3 + 跨境 IP 律师
  ├─ 上市公司 → Level 3 + 信息披露
  └─ 无 → 按 Step 1

[Step 3] 是否触发 6 大绝对禁止？
  ├─ 是 → 立即停止 + IP 律师 + 风险评估
  └─ 否 → 按 Step 1/2 升级
```

## 5. 域外法源标注规范

### 5.1 标注规则

**所有跨境 IP 分析必须标注 [域外]**：

```
例：
- 中国商标法 57 条 [GOV:商标法]
- 美国 Lanham Act 商标侵权 [域外:15 USC § 1114]
- 欧盟商标条例 [域外:Regulation (EU) 2017/1001]
- ITC 337 调查 [域外:19 USC § 1337]
- UPC 协议 [域外:UPC Agreement 2013/C 175/01]
- PCT 国际申请 [域外:PCT Article 22]
```

## 6. 法域升级自检 5 问

每次跨境任务前 agent 自问：

1. ☐ 是否识别所有相关法域（中国大陆 + 对方国家/地区）？
2. ☐ 是否查询了目标国 IP 注册状态？
3. ☐ 是否标注 [域外] 法源？
4. ☐ 是否考虑国际公约（巴黎/伯尔尼/PCT/马德里）？
5. ☐ 是否按金额 / 特殊情形触发对应升级路径？

**5 问中 ≥ 1 项未完成 → 不输出跨境结论。**
