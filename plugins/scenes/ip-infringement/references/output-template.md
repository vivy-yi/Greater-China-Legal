## 2. Risk calibration 3 段表（Pattern 18）

| 段 | 含义 | agent 动作 |
|---|------|-----------|
| **blocks** | 真正阻挡——绝对禁止 | 立即停止 + 告知 + 不绕过 |
| **work but ships** | 要修但不会挡 | 提示 IP 律师 + 给时间表 |
| **FYI** | 通知但不动作 | 记录不主动告知 |

### 2.1 blocks 段（ip-infringement 专属）

**绝对禁止 + 必须立即停止**：

| 情形 | 法条 | 后果 |
|------|------|------|
| 故意侵犯他人已注册商标专用权（假冒） | 商标法 57 条 + 刑法 214 条 | 3-10 年有期徒刑 + 罚金 |
| 假冒专利（未经许可标注他人专利号 / 伪造） | 专利法 63 条 + 刑法 216 条 | 5 年以下有期徒刑 + 罚金 |
| 侵犯他人著作权且情节严重 | 著作权法 53 条 + 刑法 217 条 | 3-10 年有期徒刑 + 罚金 |
| 以不正当手段获取商业秘密 | 反不正当竞争法 9 条 + 刑法 219 条之一 | 5 年以下 / 特别严重 5-10 年 |
| 恶意注册商标（违反诚信原则 / 抢注） | 商标法 7 条 + 44 条 | 商标无效 + 不予注册 + 损失赔偿 |
| 明知是侵权仍提供便利条件（帮助侵权） | 商标法 57 条 + 专利法 77 条 + 著作权法 53 条 | 共同侵权 + 连带责任 |

### 2.2 work but ships 段（ip-infringement 专属）

**要修但不会挡**——提示 IP 律师 + 给时间表：

| 情形 | 处理 | 时间表 |
|------|------|-------|
| 商标许可超期/超范围 | 协商续签 | 30 日内 |
| 专利年费未缴 | 补缴 + 滞纳金 | 6 个月内 |
| 著作权登记过期 | 不影响实体权利 | 立即登记 |
| FTO 报告中风险提示 | 规避设计 / 许可谈判 | 60 日内 |
| 商标续展期临近 | 申请续展 | 到期前 12 个月 |
| 商标监测发现近似申请 | 异议 / 无效宣告 | 3 个月异议期 |
| 专利被提起无效宣告 | 应诉 + 提交反证 | 3-6 月 |

### 2.3 FYI 段（ip-infringement 专属）

**通知但不动作**——记录不主动告知：

| 情形 | 记录位置 |
|------|---------|
| 行业 IP 政策变化 | memo 备注 |
| 同业 IP 案例 | 内部案例库 |
| 学者评论 / 行业会议 | 知识库 |
| 自媒体分析 | 不记录 |

## 3. 三色编码应用规范

| 颜色 | 触发场景 | 输出格式 |
|------|---------|---------|
| 🔴 HIGH | 触发 11 必升 OR 6 绝对禁止 OR 涉及刑事 | 红色高亮 + 升级路径 + 截止日 |
| 🟡 MEDIUM | 触发 work but ships 段 OR 一般风险 | 黄色提示 + 建议 + 时间表 |
| 🟢 LOW | FYI 段 OR 常规维权 | 绿色标记 + 记录位置 |

**关键**：所有 🔴 必须在 Reviewer note 第 4 行（升级建议）显式给出"升给谁 + 触发条件"。

## 4. 输出自检 8 问

每次输出前 agent 自问：

1. ☐ Reviewer note 5 行是否齐全？
2. ☐ 6 大绝对禁止 checklist 是否全部勾选？
3. ☐ 11 大必升 checklist 是否全部勾选？
4. ☐ Risk calibration 3 段判断是否完成？
5. ☐ 升级路径是否明确（升给谁 + 触发条件）？
6. ☐ 数据源是否 ≥ 2 源（关键结论）？
7. ☐ Decision tree 5 选项是否给出？
8. ☐ 发出前核查清单是否全部完成？

**8 问中 ≥ 1 项未完成 → 不输出最终结论。**
