# /regulatory-approval — 并购审批流程

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取审批事项/主管机关。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取审批事项/时间线
2. 梳理须办理的审批事项
3. 确定办理顺序
4. 估算时间
5. 输出审批清单

## 审批事项清单

### 境内ODI备案（须全部完成方可对外付汇）

| 事项 | 机关 | 时间 | 备注 |
|---|---|---|---|
| 发改委备案/核准 | 省级/国家级发改委 | 约20工作日 | 敏感行业须核准 |
| 商务部备案/核准 | 省级/国家级商务部 | 约15工作日 | 敏感行业须核准 |
| 外管局外汇登记 | 开户行/外管局 | 约5工作日 | 须在ODI备案完成后 |

### 反垄断审查（市监总局）

| 情形 | 是否须申报 |
|---|---|
| 全球营业额>100亿元 + 中国境内营业额>20亿元 + 另一方>4亿元 | ✅ 须申报 |
| 不满足以上条件 | ❌ 不须申报 |

```
申报时间：作出股权交割协议后30日内
审查时间：初步审查30日 + 进一步审查90日（复杂可延长）
审查期间：不得交割（须等审批通过）
```

### 行业特定审批

| 行业 | 须额外审批 |
|---|---|
| 金融 | 金融监管总局审批 |
| 电信 | 工信部/省级通信管理局 |
| 医疗 | 卫健委/药监局 |
| 教育 | 教育厅/教育部 |
| 互联网内容 | 网信办 |

## 办理顺序

```
第一步：交易意向达成后 → NDA + 独家期谈判
第二步：签署SPA前 → ODI备案申请（并行）
第三步：签署SPA → 含生效条件（ODI完成/反垄断通过）
第四步：ODI备案完成 → 外汇登记
第五步：反垄断审查通过 → 交割
第六步：工商变更登记
```
