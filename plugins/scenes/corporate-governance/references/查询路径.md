## 常用查询场景

### 场景1：某行业是否允许外资

**查询语句：**
```
法律精灵："[行业] 外资准入 负面清单" 或 "[行业] 外商投资 限制 禁止"
```

**判断标准（外商投资准入负面清单）：**
```
禁止类：不得外资准入（VIE也无法解决）
限制类：外资不超过特定比例，或须合资
鼓励类：一般无限制
未列入：允许外资准入
```

---

### 场景2：对赌协议效力判断

**查询语句：**
```
法律精灵："对赌协议 效力 九民纪要" 或 "估值调整协议 效力 法院"
```

**判断标准（九民纪要2019第5条）：**
```
有效情形：
- 目标公司与投资方对赌，约定业绩目标 → 有效
- 目标公司股东与投资方对赌 → 实践中一般认定有效（但须具体分析）

无效情形：
- 固定收益回报（名为投资实为借贷）→ 可能被认定借贷关系
- 违反法律法规强制性规定 → 无效
```

---

### 场景3：VIE结构是否可行

**查询语句：**
```
法律精灵："[行业] VIE 结构 可行性" 或 "[行业] 协议控制 外资准入"
```

**判断标准：**
```
第一步：查该行业是否在负面清单（禁止/限制）
第二步：如在禁止类，VIE为目前实践中的通行做法（但法律上存在风险）
第三步：如在限制类，可考虑合资或VIE
第四步：须评估37号文登记合规
```

---

### 场景4：ODI备案流程

**查询语句：**
```
法律精灵："境外投资 ODI备案 流程 商务部" 或 "37号文 登记 外汇"
```

**常用查询入口：**
```
[GOV] 商务部境外投资管理系统：https://femhzs.mofcom.gov.cn/
[GOV] 发改委境外投资管理：http://www.ndrc.gov.cn/
[GOV] 外汇管理局：www.safe.gov.cn
```

## 查询结果处理

### 情形一：查到明确的法规条文
→ 直接引用，法条优先

### 情形二：查到监管机构的具体要求
→ 标注适用范围和生效时间

### 情形三：查到类似案例（VIE诉讼等）
→ 评估风险，呈现不同结果

### 情形四：属于监管灰色地带
→ 明确告知这是风险点，建议咨询专业律师
