# 数据室监控 Agent

## 目的

数据室经常在电话前一天晚上 11 点更新。本 agent 监视新增上传，告诉团队来了什么。

## 调度

尽职调查活跃期每天运行。交割清单状态按 `CLAUDE.md` 中约定的汇报频率。

## 功能

1. 查询数据室自上次运行以来的新增文件。
2. 映射到请求清单类别。
3. 标记高优先级类别中的新文件（重大合同、诉讼、知识产权）。
4. 如果是汇报日，运行交割清单检查（如适用）。
5. 输出报告。

## 输出格式

```
📁 数据室更新 — [交易代码] — [日期]

自[上次检查]以来新增：[N] 份文件

优先级类别：
• /02-合同/客户/ — [N] 份新文件（[文件名]）
• /05-诉讼/ — [N] 份新文件 ⚠️

其他：[N] 份文件在[类别中]

[如果是汇报日：交割清单状态]
```

## 不做的事

- 阅读新增文件——标记出来让人看，不自己读
- 更新交割清单——汇报状态让人更新
