# /minority-rights-protection

中小股东权益保护咨询。工作流程：信息提取 → 法律分析 → 策略建议 → 升级决策门。

---

## 目的

为中小股东提供权益保护相关法律问题的结构化咨询，分析知情权、表决权、回购请求权、派生诉讼等核心权利的法律依据、构成要件及行使路径，并给出策略建议。

## 法域假设

默认中国大陆法域，适用《公司法》（2024年修订）及司法解释。

- 当事人角色（原告/被告/第三人）

### 第二步：法律分析 `[SME 核查]`

#### 问题类型与适用法条

| 问题类型 | 适用法条（2024年《公司法》） |
|----------|------------------------------|
| 知情权 | 第57条（有限责任公司）、第110条（股份有限公司） |
| 查阅权（会计账簿） | 第57条第2款（有限责任公司）、第110条第2款（股份有限公司） |
| 表决权 | 第72条（普通决议）、第73条（特别决议） |
| 异议股东回购请求权 | 第161条 |
| 股份回购请求权（上市公司） | 第162条 |
| 派生诉讼（股东代表诉讼） | 第188条 |
| 损害赔偿请求权 | 第188条、第192条 |

#### 构成要件分析

**知情权（有限责任公司，第57条）**
- 主体：股东
- 客体：公司章程、股东名册、董事会会议记录、监事会会议记录、财务会计报告、会计账簿
- 要件：股东须说明查阅目的；公司有合理根据认为查阅目的不正当时可拒绝

**知情权（股份有限公司，第110条）**
- 主体：股东
- 客体：公司章程、股东名册、股东会会议记录、董事会会议记录、监事会会议记录、财务会计报告
- 注意：股份有限公司股东查阅会计账簿的权利限制更严

**异议股东回购请求权（第161条）**
- 适用情形：公司连续五年盈利且符合分配条件但连续五年不分配红利；公司合并、分立、转让主要财产；公司章程规定的其他情形
- 主体：对该决议投反对票的股东
- 程序：须在股东会上对特定事项投反对票；请求公司按合理价格回购其股份

**派生诉讼（第188条）**
- 主体：股份有限公司中连续一百八十日以上单独或合计持有公司百分之一以上股份的股东
- 前置程序：书面请求监事会或董事会起诉，监事会/董事会拒绝或怠于起诉（有限责任公司无此限制）
- 事由：公司高管、董事、监事、控制股东损害公司利益

### 第三步：策略建议

#### 知情权受阻
- **路径**：先向公司书面提出查阅请求，说明目的；若被拒绝，向法院提起知情权诉讼
- **风险**：公司可能主张查阅目的不正当；需承担初步举证责任
- **注意**：有限责任公司股东诉请查阅会计账簿，胜诉后可直接强制执行

#### 表决权被侵害
- **路径**：确认表决程序违法（如未通知、表决事项超范围）；诉请撤销相关决议
- **时限**：决议作出之日起六十日内
- **风险**：法院通常不干预商业判断，仅审查程序合法性

#### 异议股东回购请求权
- **路径**：出席股东会并投反对票→书面请求公司回购→协商定价→协商不成则起诉
- **定价**：原则为"合理价格"，参考公司净资产、近期交易价格等
- **注意**：公司可在六个月内科处回购，但若六个月内未完成则效力待定

#### 派生诉讼
- **路径**：书面请求监事会/董事会起诉（30日内无结果）→以公司为被告提起诉讼→胜诉利益归公司
- **风险**：诉讼费用自行承担；法院可能要求提供担保
- **注意**：有限责任公司股东无前置程序限制；上市公司股东需满足持股比例和时间要求

### 第四步：升级决策门

> "这是辅助分析，不构成法律意见。中小股东权益保护重大事项建议由公司法专业律师审核。涉及诉讼或仲裁的，须委托律师代理。"

## 本技能不涵盖

- **代理诉讼或非诉业务**
- **确认法律效力** — 须由律师或法院确认 `[SME 核查]`
- **上市公司特别规定** — 上市公司需遵循证券法特别规则
- **跨境股权结构** — 涉及境外主体的，须另行评估
