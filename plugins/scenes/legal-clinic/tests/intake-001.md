# Test Case: 法律援助申请接收与分流

## Query

一位申请人来到法律诊所，请求法律援助。申请人情况如下：
1. 姓名：张某
2. 年龄：65岁，退休人员，月养老金2,000元
3. 案件类型：与邻居的房屋漏水纠纷导致楼下住户起诉其赔偿损失8万元
4. 申请人主张：漏水非其造成，系房屋老化管道问题
5. 申请人未聘请律师
6. 申请人已被法院传唤，一个月后开庭
7. 申请人所在城市为某二线城市，当地最低生活保障标准为每人每月1,200元

请审核：
1. 该申请人是否符合法律援助申请资格
2. 该案件是否在法律援助范围内
3. 如符合，应分流到哪个专业方向
4. 如不符合，说明理由并提供其他建议

## Expected Behavior

1. 应进行经济困难认定：比较月收入2,000元与当地低保标准1,200元
2. 应判断案件类型（相邻关系纠纷/财产损害赔偿）是否在法律援助范围内
3. 应参照《法律援助法》第25-31条分析是否符合法定条件
4. 应给出分流建议（民事组/房产组）
5. 标注论证强度并注明法条依据来源
