### § A8.1 法律援助特别注意 4 大块（援助资格 + 申请流程 + 案件类型 + 经费标准）

> **核心原则**：法律援助的**国家义务**——符合条件的公民有获得免费法律服务的权利。法援律师不得向当事人收费。

#### 块 1：法律援助资格（《法律援助法》2022 修订）

**核心法规：**
- 《中华人民共和国法律援助法》（2022 年 1 月 1 日修订 + 2023 年实施）
- 《法律援助案件指派文书格式》
- 《12348 法律服务热线规范》

**法律援助 11 类受援人（《法律援助法》第 31 条 + 第 41 条）：**

| 类别 | 法律依据 |
|------|---------|
| **刑事** | 盲聋哑 / 未成年 / 可能判无期死刑 / 强制医疗 / 死刑复核 |
| **公诉** | 被害人因经济困难 |
| **民事 / 行政** | 低保 + 经济困难证明 + 请求事项合法 |
| **劳动** | 农民工讨薪 / 工伤（无须经济困难证明） |
| **请求国家赔偿** | 涉人身自由 / 生命健康 |
| **英雄烈士保护** | 检察机关 / 公益组织 |
| **未成年人保护** | 监护缺失 / 监护侵害 |

**经济困难标准（参考）：**

| 标准 | 数值 |
|------|------|
| **一般公民** | 低保标准 2 倍以内（具体由地方确定） |
| **农民工讨薪** | **无须经济困难证明**（2022 修订扩大） |
| **工伤赔偿** | **无须经济困难证明** |
| **请求国家赔偿** | **无须经济困难证明**（人身自由 / 生命健康） |
| **特殊群体** | 老年人 / 妇女 / 残障 / 军人军属优先 |

**blocks（绝对禁止 — 援助资格）**：
- 假冒经济困难骗取援助 → **《刑法》第 380 条**（虚假诉讼罪）
- 法援律师向当事人收费 → **《律师法》第 49 条 + 吊销执照**

**work but ships（可补救）**：
- 经济困难材料不全 → 3 日内补正
- 援助事项部分超范围 → 部分援助 + 部分自办

#### 块 2：法律援助申请流程 + 12348 热线

**核心流程（5 步）：**

| 步骤 | 内容 | 时限 |
|------|------|------|
| **1. 咨询 / 申请** | 到法律援助中心 / 12348 热线 / 网上申请 | 立即 |
| **2. 资格审核** | 经济困难证明 + 案件材料 | 7 日内 |
| **3. 指派律师** | 法律援助机构指派 / 安排值班律师 | 3 日内 |
| **4. 案件办理** | 提供法律咨询 / 代写文书 / 代理案件 | 全程 |
| **5. 结案评估** | 案件质量评估 + 结案归档 | 结案后 30 日 |

**12348 法律服务热线：**

| 项目 | 内容 |
|------|------|
| **电话** | 12348（全国统一） |
| **服务时间** | 7×24 小时 |
| **覆盖** | 法律咨询 + 援助申请 + 律师事务所查询 |
| **特点** | 免费 + 保密 + 普通话 + 多民族语言 |

**法律援助申请材料清单：**

| 材料 | 必备 |
|------|------|
| **身份证 / 户籍证明** | ✅ 必备 |
| **经济困难证明** | ✅ 必备（特殊案件除外） |
| **案件材料** | ✅ 必备（起诉状 / 答辩状 / 证据） |
| **其他** | 视案件类型（低保证 / 残障证 / 工伤认定） |

**blocks（绝对禁止 — 申请流程）**：
- 故意拖延指派 → **《法律援助法》第 51 条**（责任追究）
- 拒绝合格申请 → **同上**

**work but ships（可补救）**：
- 材料不齐 → 3-7 日内补正
- 资格争议 → 申请异议 + 复查

#### 块 3：4 类高频法律援助案件

**4 类高频案件 + 处理要点：**

**1. 农民工讨薪（最常见）：**

| 要点 | 内容 |
|------|------|
| **管辖** | 劳动合同履行地 / 仲裁委员会 / 法院 |
| **时效** | 劳动仲裁 1 年 |
| **流程** | 调解 → 仲裁 → 诉讼（无须经济困难证明） |
| **关键** | 劳动合同 + 工资条 + 工作记录 |
| **法规** | 《保障农民工工资支付条例》（2020） |

**2. 工伤赔偿：**

| 要点 | 内容 |
|------|------|
| **程序** | 工伤认定 → 劳动能力鉴定 → 仲裁 / 诉讼 |
| **时效** | 1 年（工伤认定） / 1 年（仲裁） |
| **赔偿** | 医疗费 / 停工留薪期工资 / 一次性伤残补助金 |
| **法规** | 《工伤保险条例》 + 《社会保险法》 |
| **特别** | 无须经济困难证明 |

**3. 家事 + 妇女儿童保护：**

| 要点 | 内容 |
|------|------|
| **程序** | 调解优先 → 诉讼 |
| **保护** | 人身安全保护令（48-72 小时） |
| **特殊** | 未成年 / 家暴 / 监护权 |
| **法规** | 《反家庭暴力法》 + 《未成年人保护法》 + 《妇女权益保障法》 |
| **援助** | 优先 + 快速通道 |

**4. 行政诉讼（2022 全覆盖）：**

| 要点 | 内容 |
|------|------|
| **覆盖** | 2022 年修订后**行政诉讼全类型法律援助** |
| **管辖** | 作出行政行为的行政机关所在地法院 |
| **时效** | 6 个月 |
| **特殊** | 行政赔偿 / 行政处罚 / 行政强制 / 行政许可 |
| **法规** | 《行政诉讼法》 |

**blocks（绝对禁止 — 案件类型）**：
- 工伤认定后未在 1 年内申请仲裁 → **失权**
- 家暴受害人未及时申请人身安全保护令 → **后续风险**
- 行政诉讼超过 6 个月 → **失权**

**work but ships（可补救）**：
- 农民工讨薪证据不齐 → 30 日内收集
- 工伤认定争议 → 60 日内复议 / 诉讼

#### 块 4：法律援助经费 + 案件补贴

**核心法规：**
- 《法律援助经费管理办法》（财政部 / 司法部 2022）
- 《法律援助案件补贴标准》（地方规定）

**案件补贴标准（参考）：**

| 案件类型 | 补贴范围 |
|---------|---------|
| **刑事辩护** | 1000-3000 元 / 件 |
| **民事 / 行政代理** | 800-2000 元 / 件 |
| **代写法律文书** | 200-500 元 / 件 |
| **法律咨询** | 免费 |

**经费来源：**

| 来源 | 占比 |
|------|------|
| **财政拨款** | 主体 |
| **法律援助基金** | 补充（社会捐赠） |
| **社会救助** | 边缘 |

**blocks（绝对禁止 — 经费）**：
- 律师索要补贴外费用 → **《律师法》第 49 条 + 吊销执照**
- 律师收受当事人额外费用 → **同上**

**work but ships（可补救）**：
- 补贴标准偏低 → 申请调整 + 政府协商
- 经费不足 → 申请基金补充

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **假冒经济困难骗取援助** | 《刑法》第 380 条虚假诉讼罪 | 3-7 年有期徒刑 |
| 2. **法援律师向当事人收费** | 《律师法》第 49 条 | 吊销执照 + 涉嫌犯罪 |
| 3. **律师拒绝合格指派** | 《法律援助法》第 51 条 | 责任追究 + 处分 |
| 4. **指派律师不尽职** | 《法律援助法》第 54 条 | 处分 + 补贴追回 |

**主动问（6 类不确定 — 法律援助版）**：
- 援助对象 + 经济状况 + 案件类型？（核心资格）
- 是否特殊群体（农民工 / 工伤 / 未成年 / 家暴）？（免经济困难）
- 案件管辖 + 援助阶段？（咨询 / 代理 / 辩护）
- 是否需要立即保护令？（家暴 / 紧急）
- 法律援助中心 / 12348？（申请入口）
- 主办律师 + 督导？（法援律师指派）
