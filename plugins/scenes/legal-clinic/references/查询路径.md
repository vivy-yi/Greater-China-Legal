# 法律诊所查询路径 — 中国大陆

> 本文档提供法律诊所场景实际检索入口和操作流程，按数据类型分类整理。
> legal-clinic 是**学生办案+督导审核**模式，查询后必须经督导复核。
> 数据源标注规则见 `references/数据源清单.md`。

---

## 一、法援法规检索

### 1.1 中央法规

```
查询目标：法律援助法 / 民法典 / 劳动合同法 等

步骤 1：[GOV] 国家法律法规数据库
  URL: https://flk.npc.gov.cn
  检索字段：标题（如"中华人民共和国法律援助法"）
  输出：现行有效版本 + 历次修正记录

步骤 2：[BD] 版本时效验证
  工具：beidalu MCP
  验证项：法律援助法（2022.01.01 施行）+ 配套规定

步骤 3：[YD] 交叉验证
  检索同一法条，确认与 [GOV]/[BD] 一致
  不一致 → 标注 ⚠️ 来源冲突 + 督导复核
```

### 1.2 司法解释

```
查询目标：最高法司法解释

URL: https://www.court.gov.cn → 司法解释
或: [YD] 检索"司法解释 + {案由}"

重点关注（法援场景）：
  - 民法典总则编解释（2022.02.25）
  - 民法典合同编通则解释（2023.05.25）
  - 民法典婚姻家庭编解释
  - 民法典继承编解释
  - 劳动争议司法解释（一）（2020.12.29）
  - 人身损害赔偿司法解释
  - 关于办理人身安全保护令案件适用法律若干问题的规定
```

### 1.3 各省市法律援助实施办法

```
查询目标：北京/上海/广东/江苏 等地法律援助实施办法

URL 模板：
  {省市}人民政府官网 → 法规规章 → 法律援助
  例：北京市人民政府 → 法规 → 北京市法律援助条例

[Y D] 检索："{省市} 法律援助实施办法"
[web] 时效核验：发布日期 + 是否有修订

注意：法援经济困难标准由各省市自行规定，必须核实当地标准
```

### 1.4 部门规章

```
查询目标：法律援助相关规章

[GOV] 司法部官网 → 法规规章 → 法律援助
[Y D] 检索："法律援助 + 部门规章"
```

## 二、案例检索（按 4 场景）

### 2.1 最高人民法院指导案例 / 公报案例

```
URL: 人民法院案例库 https://rmfyalk.court.gov.cn
检索字段：关键词 + 案由
筛选：指导性案例 > 公报案例 > 普通案例

[Y D] 检索："指导案例 + {案由}"

学习价值：
  - 必学：指导案例
  - 重要：公报案例
  - 选学：典型案例
```

### 2.2 C1 劳动法援案例检索

```
查询目标：劳动争议类案

[Y D] 检索："{争议类型} + {地域}"
  例："违法解除 上海"
  例："经济补偿金基数 北京"
  例："工伤认定 广东"

[WKL] 综合检索，对比多源

输出格式：
  | 案例 | 裁判日期 | 法院 | 裁判要旨 |
  | --- | --- | --- | --- |
  |（2023）京02民终1234号 | 2023-06 | 北京二中院 | 程序违法即构成违法解除 |
```

### 2.3 C2 消费者维权案例检索

```
查询目标：消费者权益类案

[Y D] 检索："{争议类型} + 消保法"
  例："欺诈 三倍赔偿"
  例："电商纠纷 七天无理由"
  例："预付卡 跑路"

[WKL] 检索典型案例

注意：消费者维权案件通常金额较小，但涉及群体性案件须特别关注
```

### 2.4 C3 婚姻家事案例检索

```
查询目标：婚姻继承类案

[Y D] 检索："{争议类型} + 民法典婚姻家庭编/继承编"
  例："离婚财产分割 共同财产"

  例："遗嘱继承 法定继承"
  例："人身安全保护令"

[WKL] 检索典型案例

```

### 2.5 C4 行政诉讼案例检索

```
查询目标：行政诉讼类案

[Y D] 检索："{争议类型} + 行政诉讼法"
  例："违法强拆 国家赔偿"
  例："复议前置 被告主体"
  例："行政处罚 显失公正"
  例："行政不作为 履行之诉"

[WKL] 检索典型案例

注意：行政诉讼涉及被告为政府，须督导复核被告主体认定
```

## 四、特定数据查询

### 4.1 工伤认定标准

```
查询目标：工伤/视同工伤/不得认定情形

[GOV] 工伤保险条例 https://www.gov.cn
[Y D] 检索："工伤认定 + {情形}"

关键情形：
  - 工作时间工作场所因工作原因受伤
  - 上下班途中非本人主要责任的交通事故
  - 患职业病
  - 视同工伤（工作时间工作岗位突发疾病 48 小时内死亡等）
  - 不得认定：故意犯罪、醉酒或吸毒、自残或自杀
```

### 4.2 工伤保险待遇标准

```
查询目标：伤残等级对应待遇

URL：各省市人社局 → 工伤保险 → 待遇标准

[Y D] 检索："{伤残等级} + 工伤待遇 + {地区}"

查询项：
  - 一次性伤残补助金（1-10 级，几个月工资）
  - 伤残津贴（1-4 级）
  - 一次性工伤医疗补助金（5-10 级解除合同时）
  - 一次性伤残就业补助金（5-10 级解除合同时）
```

### 4.3 经济补偿金/赔偿金计算

```
查询目标：N / N+1 / 2N 计算

[Y D] 检索："经济补偿金计算 + 月工资基数"

计算工具：
  - 用人单位所在地上年度职工月平均工资（[GOV] 各省市人社局）
  - 劳动者前 12 个月平均工资（公司工资台账 + 当事人陈述 [user]）
  - 服务年限（公司人事记录）

公式：
  N = 月平均工资 × 服务年限
  2N = N × 2（违法解除）
  三倍上限：月平均工资 > 当地三倍平均 → 按三倍
  12 年封顶：按三倍计算最高 12 年

⚠️ 法援场景：当事人陈述数据须 [user] 标注 + 督导复核
```

### 4.4 家暴认定 + 人身安全保护令

```
查询目标：家暴认定标准 + 人身安全保护令申请

[GOV] 反家庭暴力法（2016.03.01 施行）
[Y D] 检索："人身安全保护令 + 申请条件"

申请条件：
  - 当事人因遭受家庭暴力或面临家庭暴力现实危险
  - 可向法院申请人身安全保护令
  - 紧急情况下 48 小时内作出

[Y D] 检索："家暴认定 + 证据"

证据类型：
  - 报警记录
  - 医院诊断证明
  - 伤情照片
  - 证人证言
  - 保证书
```

### 4.5 行政诉讼被告主体认定

```
查询目标：行政诉讼被告主体

[GOV] 行政诉讼法（2017 修正）
[Y D] 检索："被告资格 + 行政机关"

认定规则：
  - 作出具体行政行为的行政机关 → 被告
  - 委托执法的组织 → 委托行政机关为被告
  - 复议机关改变原行为的 → 复议机关为被告
  - 多部门联合执法 → 共同被告
  - 授权执法的组织 → 被授权组织为被告
  - 派出机构执法 → 派出机构为被告（除非法律有特别规定）

⚠️ C4 场景：被告主体认定必须督导复核
```

## 六、争议处理程序

### 6.1 劳动仲裁申请流程

```
Step 1：当事人来访（client-intake）
  ├─ 经济困难认定
  ├─ 案件类型识别（C1 劳动法援）
  └─ 创建 client/[id]/ 文件夹

Step 2：案件研究（research-start / memo）
  ├─ 解除类型认定
  ├─ 仲裁时效核查
  ├─ 适用法律识别
  └─ 督导初步审核

Step 3：仲裁申请（draft）
  ├─ 起草仲裁申请书
  ├─ 准备证据材料
  ├─ 督导复核（supervisor-review-queue）
  └─ 提交仲裁委

Step 4：仲裁审理
  ├─ 立案（5 日内决定）
  ├─ 庭前调解（30 日内）
  ├─ 开庭审理（45 日内结案）
  ├─ 裁决（一裁终局 / 不服可起诉）
  └─ 时效管理（deadlines）

Step 5：结案
  ├─ 裁决执行
  ├─ 案件归档
  └─ 当事人回访
```

### 6.2 民事诉讼流程

```
Step 1-3：与劳动仲裁类似
Step 4：诉讼审理
  ├─ 立案（7 日内决定）
  ├─ 庭前调解
  ├─ 开庭审理（简易 3 个月，普通 6 个月）
  ├─ 一审判决
  ├─ 上诉（15 日内，二审 3 个月）
  └─ 终审判决

Step 5：执行
  ├─ 申请执行（2 年内）
  ├─ 执行措施
  └─ 结案
```

### 6.3 行政诉讼流程

```
Step 1：复议前置判断
  ├─ 是 → 先申请行政复议（60 日内）
  └─ 否 → 直接起诉

Step 2：起诉
  ├─ 起诉状
  ├─ 证据材料
  ├─ 督导复核
  └─ 提交法院（6 个月内）

Step 3：审理
  ├─ 立案（7 日内）
  ├─ 开庭审理（3 个月）
  ├─ 判决
  └─ 上诉（15 日内，二审 2 个月）

Step 4：执行 / 国家赔偿
  ├─ 履行判决
  ├─ 国家赔偿（2 年内）
  └─ 结案
```

### 6.4 时效紧迫度判断

```
特急（≤7 日）：立即处理 + 督导实时审核
  - 即将过仲裁时效
  - 行政诉讼 6 个月时效即将届满
  - 人身安全保护令紧急申请

紧急（≤30 日）：优先处理
  - 仲裁时效剩余 30 日内
  - 上诉期
  - 答辩期

普通（>30 日）：标准流程
  - 一般案件
```

## 八、检索效率技巧

### 8.1 类案检索四步法

```
1. 问题定性（C1/C2/C3/C4 + 案由）
   例：C1 劳动法援 → 违法解除 2N

2. 信息收集
   - 当事人陈述 [user]
   - 证据材料 [BD]
   - 法律事实认定

3. 关键词提取
   规范性：违法解除 / 经济补偿金 / 仲裁时效
   非规范性：未通知工会 / 工资拖欠 / 工时超标

4. 数据库检索
   [YD] + [WKL] + [GOV] 交叉
   地域/时间/审级范围确定
   相似度评估
```

### 8.2 多源交叉验证（法援场景严格化）

```
法援场景关键结论须 2-3 个数据源确认：

- 法律援助资格：[GOV]司法部 + 当地标准 + [YD]案例
- 经济补偿金：[YD] 案例 + [GOV] 人社部基数 + [WKL] 综合
- 违法解除：[YD] 类案 + [WKL] 综合 + 督导审核
- 工伤认定：[YD] + [GOV]人社部 + 当地工伤认定机构
- 复议前置：[GOV]行政复议法 + [YD]案例
- 被告主体：[GOV]行政诉讼法 + [YD]案例

不一致 → 标注 ⚠️ 来源冲突 + 督导复核
```

### 8.3 案件策略制定

```
Step 1：法律分析（memo）
  ├─ 争议焦点识别
  ├─ 法律依据检索
  ├─ 证据评估
  └─ 类似案例对比

Step 2：策略选择
  ├─ 调解优先（成本低 + 快速）
  ├─ 仲裁/诉讼（强制力）
  └─ 投诉举报（行政路径）

Step 3：风险评估
  ├─ 胜诉概率
  ├─ 时间成本
  ├─ 经济成本
  └─ 关系成本

Step 4：当事人沟通
  ├─ 通俗解释法律
  ├─ 告知风险
  ├─ 询问当事人意愿
  └─ 书面确认

Step 5：督导复核
  └─ supervisor-review-queue
```

## 九、应急检索

### 9.1 时效紧迫场景

```
场景：仲裁时效即将届满（≤7 日）

操作流程：
  1. 立即确认时效起算点
  2. 启动时效中断（主张权利/对方同意）
  3. 准备仲裁材料（申请书 + 证据）
  4. 督导实时审核
  5. 提交仲裁委

资源：
  [GOV] 仲裁委地址（紧急联系）
  [YD] 仲裁申请书模板
```

### 9.2 当事人紧急情况

```
场景：人身安全保护令紧急申请

操作流程：
  1. 评估危险等级
  2. 收集家暴证据
  3. 起草保护令申请书
  4. 法院紧急申请（48 小时内裁定）
  5. 联动妇联/警方
```
