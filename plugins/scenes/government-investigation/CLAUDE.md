### § A8.1 政府调查特别注意 4 大块（反垄断处罚 + 宽大制度 + 行政诉讼 + FCPA）

> **核心原则**：政府调查的核心是**程序时限 + 实体抗辩 + 跨境协调**。错过时限 = 失权。

#### 块 1：反垄断处罚金额 + 申报标准（《反垄断法》2022 修订）

**核心法规：**
- 《中华人民共和国反垄断法》（2022 年 6 月 24 日修订）
- 《国务院关于经营者集中申报标准的规定》（2008 + 2024 修订）
- 《关于禁止垄断协议的规定》
- 《关于禁止滥用市场支配地位行为的规定》

**经营者集中申报标准（2024 修订）：**

| 标准 | 全球营业额 | 中国营业额 | 备注 |
|------|----------|----------|------|
| **任一标准** | ≥ 100 亿人民币 | ≥ 10 亿人民币 | 任一达到即触发 |
| **特别行业** | — | — | 银行业 / 保险业单独标准 |

**垄断行为 3 类处罚（2022 修订加重）：**

| 行为类型 | 法条 | 罚款标准 |
|---------|------|---------|
| **经营者集中未申报** | § 56 条 | **¥50 万 - ¥500 万**（个人 ¥10 万 - ¥100 万） |
| **达成垄断协议** | § 56 条 | **上一年度销售额 1% - 10%** |
| **滥用市场支配地位** | § 57 条 | **上一年度销售额 1% - 10%** |
| **卡特尔严重情节** | § 63 条 | **2% - 10%**（可加重） |
| **组织帮助垄断** | § 64 条 | **违法所得 1% - 10%** |

**blocks（绝对禁止 — 反垄断）**：
- 经营者集中达到标准未申报 → **50 万 - 500 万罚款 + 强制拆分**
- 核心卡特尔（价格 / 数量 / 市场分割）→ **销售额 1-10% + 民事赔偿 + 刑事移交**
- 滥用必需设施（拒绝交易 / 限定交易）→ **销售额 1-10%**

**work but ships（可补救）**：
- 申报材料不齐 → 30 日内补正
- 附条件批准未履行 → 60 日内补救 + 报告
- 未达申报标准事后发现 → 主动沟通 + 报告

#### 块 2：宽大制度（卡特尔应对核心）

**核心法规：**
- 《反垄断法》第 64 条 + 《国务院反垄断委员会关于卡特尔案件经营者主动报告宽大制度的适用指南》

**宽大制度适用条件：**

| 顺序 | 申请时点 | 减免幅度 |
|------|---------|---------|
| **第 1 名** | 调查启动前主动报告 | **免除 / 全额减免** |
| **第 2 名** | 调查启动前主动报告 | **减免 50% 以上** |
| **第 3 名+** | 调查启动前主动报告 | **减免 50% 以下** |
| **任意** | 调查启动后**关键证据** | 酌情减免 |
| **已启动后** | 仅提交次要证据 | 通常不予减免 |

**blocks（绝对禁止 — 宽大申请）**：
- 销毁 / 隐匿证据 → **不予减免 + 加处 + 刑事移交**
- 胁迫其他经营者不申请 → **不予减免 + 反垄断处罚升级**
- 串供 / 协调口径 → **不予减免**

**work but ships（可补救）**：
- 宽大申请材料不齐 → 7 日内补正
- 多个经营者同时申请 → 第一名优先

#### 块 3：行政诉讼时效 + 程序时限

**核心法规：**
- 《行政诉讼法》（2014 修订 + 2017 修正）
- 《行政复议法》（2023 修订）

**核心时限速查：**

| 程序 | 时限 | 起算点 | 重要警示 |
|------|------|--------|---------|
| **行政复议** | **60 日** | 知道具体行政行为之日 | 先复议再诉讼（一般情况） |
| **行政诉讼** | **6 个月** | 知道具体行政行为之日 | 直接诉讼（复议非前置） |
| **不动产行政诉讼** | **20 年** | 行政行为作出之日 | 最长保护期 |
| **其他行政诉讼** | **5 年** | 行政行为作出之日 | 最长保护期 |
| **行政处罚听证申请** | 5 日 | 收到听证告知书之日 | 错过 = 失权 |
| **处罚决定书送达** | 7 日内 | 作出之日 | 影响起算 |
| **强制执行申请** | 3 个月 | 复议 / 诉讼期满 | 错过 = 失权 |

**blocks（绝对禁止 — 程序应对）**：
- 错过诉讼时效 → **无法救济**
- 不服处罚不复议不诉讼 → **处罚生效 + 强制执行**
- 听证申请错过 → **丧失听证权**

**work but ships（可补救）**：
- 时效临近 → 紧急申请 + 暂缓执行
- 程序违法 → 行政复议 + 行政诉讼双轨

#### 块 4：FCPA 反海外腐败（涉外案件核心）

**核心法规：**
- 《美国反海外腐败法》（FCPA, 1977 + 后续修订）
- 《英国反贿赂法》（Bribery Act 2010）
- 中国《关于禁止商业贿赂行为的暂行规定》

**FCPA 核心要件：**

| 要件 | 内容 |
|------|------|
| **适用主体** | 美国发行人 / 国内相关发行人（SEC 注册）+ 美国境内公司 |
| **管辖** | 美国发行人 + 美国证券 + 美国境内行为 |
| **禁止行为** | 向外国政府官员行贿 / 支付 / 承诺任何有价值物 |
| **"任何有价值物"** | 现金 / 礼品 / 旅游 / 工作机会 / 慈善捐款 |
| **"外国政府官员"** | 政府官员 / 政党 / 国有企业员工 / 王室 / 国际组织 |
| **"知晓"标准** | 实际知晓 / 故意忽视 / 故意视而不见（willful blindness） |

**FCPA 处罚（严厉）：**

| 处罚类型 | 自然人 | 单位 |
|---------|-------|------|
| **刑事罚款** | **最高 $250,000 / 单项**（可累计） | **最高 $2,000,000 / 单项** |
| **民事罚款** | — | **违法所得 + $10,000 - $100,000 / 项** |
| **监禁** | **最长 5 年 / 单项** | — |
| **证券欺诈附加** | — | 退市 + 投资者赔偿 |

**特别注意（中国企业涉外）：**
- 中国企业在美国上市 / 在美国境内有业务 → **可能适用 FCPA**
- 通过中国代理 / 顾问 / 合资方支付 → **代理行为归责于美国发行人**
- 即使高管不知情 + 第三方行为 → **"willful blindness" 标准仍可追责**

**blocks（绝对禁止 — FCPA）**：
- 通过中国代理向外国官员支付 → **FCPA 适用 + 单位罚款 + 监禁**
- 礼品 / 旅游 / 慈善捐款给外国官员 → **即使形式合规也可能违规**
- 故意忽视 "red flag" → **"willful blindness" 仍可追责**

**work but ships（可补救）**：
- 发现可疑支付 → 30 日内自愿披露（DOJ / SEC Voluntary Disclosure Program）
- FCPA 内控缺陷 → 60 日内整改 + 培训
- 历史违规 → 申请缓刑 + 监察官监督（compliance monitor）

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **经营者集中未申报** | 《反垄断法》第 56 条 | 50 万 - 500 万 + 强制拆分 |
| 2. **核心卡特尔** | 《反垄断法》第 63 条 | 销售额 2-10% + 民事 + 刑事 |
| 3. **FCPA 违规（境外）** | FCPA + 中国相关规定 | 单位 $200 万 + 自然人 5 年监禁 |
| 4. **未做信息披露（证券）** | 《证券法》第 193 条 | 30 万 - 1000 万 + 市场禁入 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时限
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — 政府调查增强版）**：
- 调查机关 + 调查类型 + 涉案金额？（核心事实）
- 程序阶段：立案 / 调查 / 听证 / 决定 / 复议 / 诉讼？（决定应对手法）
- 是否达到经营者集中申报标准？（反垄断触发）
- 是否涉及境外政府官员？（FCPA 触发）
- 处罚幅度预估？（销售额百分比）
- 是否错过时限？（时效核查）
