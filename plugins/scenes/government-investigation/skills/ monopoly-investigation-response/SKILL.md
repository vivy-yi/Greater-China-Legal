# 反垄断调查应对

## 数据源与判断框架引用

**数据源标注：** 本技能引用场景级数据源标注规则——`[YD]`元典MCP、`[WKL]`裁判文书网/无讼、`[BD]`北达检索、`[GOV]`政府平台、`[web]`网络搜索、`[model]`模型推理（须核实）。关键结论须多源交叉验证。

**判断框架：** 以《中华人民共和国反垄断法》（2022修订）为核心依据，叠加《国务院反垄断委员会关于知识产权领域的反垄断指南》《禁止滥用市场支配地位行为暂行规定》及市场监管总局发布的垄断案件执法指南。

## 一、基本信息

### 1.1 适用场景
本技能适用于以下情形：
- 企业收到市场监管总局反垄断局或省级市场监管局的反垄断调查通知（现场检查/询问通知/要求提交文件）
- 企业主动评估是否触发反垄断调查风险（垄断协议/滥用市场支配地位/违法实施经营者集中）
- 宽大制度申请（第一个主动报告可减免或大幅降低罚款）
- 调查过程中需要临时应对策略（文件保全、员工面谈辅导、证据开示策略）

### 1.2 用户需提供信息
- 是否已收到调查通知及通知类型
- 涉嫌垄断行为类型（垄断协议/滥用市场支配地位/违法集中）
- 行业及市场份额数据
- 是否有内部合规制度及执行记录
- 是否有境外关联法域的反垄断调查（欧盟/美国/其他法域）

### 1.3 输出成果形态
- 反垄断调查应急预案
- 文件保全与证据开示指引
- 宽大申请可行性评估报告
- 调查各阶段答辩策略备忘录

## 二、分析框架

### 2.1 垄断协议认定框架

```
[是否属于垄断协议]
├── 横向垄断协议（竞争者之间）
│   ├── 固定或变更商品价格 → 绝对禁止
│   ├── 限制商品生产/销售数量 → 绝对禁止
│   ├── 分割销售市场/原材料采购市场 → 绝对禁止
│   ├── 限制购买新技术/新设备 → 绝对禁止
│   └── 联合抵制交易 → 绝对禁止
├── 纵向垄断协议（上下游之间）
│   ├── 固定向第三人转售商品价格 → 原则禁止
│   ├── 限定向第三人转售商品最低价格 → 原则禁止
│   └── 安全港规则（市场份额<15%且无排除限制竞争效果）
└── 轴幅合谋（Hub-and-Spoke）
    ├── 核心企业组织竞争者之间信息交换
    └── 构成垄断协议共谋的间接证据链
```

### 2.2 滥用市场支配地位认定框架

```
[是否具有市场支配地位]
├── 相关市场界定
│   ├── 商品市场（需求替代分析 + 供给替代分析）
│   └── 地域市场（中国/区域市场）
├── 市场份额评估
│   ├── 1个经营者≥1/2 → 可推定
│   ├── 2个经营者合计≥2/3 → 可推定
│   └── 3个经营者合计≥3/4 → 可推定（任一<1/10除外）
└── 滥用行为认定
    ├── 以不公平高价销售/低价购买
    ├── 低于成本销售（掠夺性定价）
    ├── 拒绝交易（关键设施原则）
    ├── 限定交易（独家/排他）
    ├── 搭售或附加不合理条件
    ├── 差别待遇（同等交易条件）
    └── 国务院反垄断执法机构认定的其他滥用行为
```

### 2.3 经营者集中申报审查框架

```
[是否触发申报义务]
├── 营业额门槛（2022年修订标准）
│   ├── 全球合计>100亿人民币且至少两个经营者境内>4亿
│   └── 中国境内合计>20亿人民币且至少两个经营者境内>4亿
├── 符合简易程序条件
│   ├── 横向集中后市场份额<15%
│   ├── 纵向集中各环节市场份额<25%
│   └── 设立合营企业不在集中相关市场内活跃
└── 申报时间
    ├── 必须在实施集中前申报（禁止gun-jumping）
    └── 未申报不得实施集中
```

## 三、操作流程

### 3.1 突查应对流程（0-24小时黄金期）

```
步骤1：确认执法人员身份与调查范围
  ├── 核实执法证件（市场监管总局/省局反垄断执法人员）
  ├── 确认调查通知书载明的调查事项范围
  └── 立即通知法务负责人/外部律师
步骤2：文件保全与证据控制
  ├── 启动内部文件保全预案（邮件/即时通讯/服务器）
  ├── 设置数据访问权限冻结
  └── 分类标记文件：特权文件（Attorney-Client）/商业敏感/无关文件
步骤3：员工面谈辅导与权利告知
  ├── 指定陪同人（法务/外部律师），员工接受询问须有陪同
  ├── 不回答超出调查范围的问题
  └── 告知员工配合义务与保护自身合法权益的边界
步骤4：调查应对策略评估
  ├── 评估是否构成垄断违法行为
  ├── 评估是否可申请宽大/承诺制度
  └── 启动与执法机构的初步沟通
```

### 3.2 宽大制度申请流程

```
步骤1：自评是否符合宽大条件
  ├── 第一个主动报告并提交重要证据的，可减免100%罚款（2022年反垄断法第56条）
  ├── 第二个主动报告的，可减免不低于50%罚款
  └── 后续主动报告的，可减免不高于50%罚款
步骤2：准备宽大申请材料
  ├── 主动报告书（详细描述垄断协议内容、参与方、实施情况）
  ├── 重要证据材料（书面文件、会议记录、通讯记录）
  └── 保全证据的完整性说明
步骤3：提交宽大申请
  ├── 向市场监管总局反垄断局提交
  └── 可申请先予登记（marker system）锁定第一个申报位置
步骤4：配合调查与合规整改承诺
  ├── 全面配合执法机构调查
  └── 提出合规整改计划并承诺执行
```

### 3.3 行政复议与诉讼衔接

```
[收到行政处罚决定后]
├── 行政复议（向市场监管总局或国务院申请，60日内）
│   └── 复议期间一般不停执行
├── 行政诉讼（收到决定之日起6个月内）
│   └── 可同时申请停止执行
└── 行政和解（承诺制度）
    └── 主动承诺停止违法行为并消除竞争影响 → 执法机构可中止调查
```

## 四、升级决策门

出现以下情形，将案件移交专业律师：
- 进入刑事调查程序（涉嫌串通投标罪等）
- 涉及跨国监管执法（美国司法部/欧盟委员会/FTC同时启动调查）
- 涉案金额特别巨大（罚款可能超过企业年营业额10%）
- 涉及核心业务模式或企业生存危机
- 企业高管可能承担个人法律责任（刑事责任/市场禁入）
