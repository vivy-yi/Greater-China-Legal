## 1. 转让定价与跨境税务深度

### 1.1 转让定价方法选择(5 种 OECD 认可)

| 方法 | 英文缩写 | 最适用场景 | 数据需求 | 可靠性 |
|------|---------|---------|---------|-------|
| 可比非受控价格法 | CUP | 商品交易（有形资产） | 需市场上完全可比的价格 | 高（如能找到） |
| 成本加成法 | C+ | 制造/加工/技术服务 | 成本结构+合理加成率 | 中高 |
| 再销售价格法 | RPM | 分销商（买入后转售） | 毛利率对比 | 中 |
| 交易净利润法 | TNMM | 最广泛使用 | 净利润率指标 | 中高 |
| 利润分割法 | PSM | 高价值无形资产/高度整合 | 利润贡献度分析 | 低（争议大） |

### 1.2 TNMM 测试步骤(中国最常用)

```
1. 选择利润水平指标（PLI）：
   ├─ 制造企业：成本利润率（成本加成率 = EBIT/总成本）
   ├─ 分销企业：营业利润率 = EBIT/销售额
   ├─ 服务企业：成本加成率
   └─ 特许使用：贝里比率 = 毛利率/经营费用率

2. 确定可比公司（数据库查询）：
   ├─ 中国可比公司（首选）→ Wind/同花顺/Osiris
   └─ 如不可行 → 区域可比公司

3. 统计四分位区间：
   ├─ 下四分位数（25%）
   ├─ 中位数（50%）
   └─ 上四分位数（75%）

4. 结论：
   ├─ 处于四分位区间内 → 低风险
   ├─ 处于下四分位数以下但接近 → 中风险
   └─ 显著低于下四分位数 → 高风险
```

### 1.3 同期资料 3 档要求(国家税务总局公告 2016 年第 42 号)

| 企业类型 | 文档要求 | 触发条件 |
|---------|---------|---------|
| **主体文档** | 集团全球业务整体说明 | 年度关联交易 > 2 亿 |
| **本地文档** | 本地企业详细功能风险分析 | 有形资产所有权转让 > 2 亿 / 金融资产 > 1 亿 / 无形资产 > 1 亿；或年度关联交易 > 4000 万 且 利润总额为负 |
| **特殊事项文档** | 特定交易(如 APA 申请、成本分摊) | 申请 APA / 涉及成本分摊 |

### 1.3.1 批量场景 4 类分组指南(v3-self-test 修补)

**适用**：30+ 子公司批量转让定价合规场景。

| 分组 | 标准 | 处理深度 | 时限 |
|------|------|---------|------|
| **高风险(避税地)** | 关联方在开曼/BVI/百慕大 | 100% 严格审查 + FAR 分析 + TNMM 测试 + APA 评估 | 主体文档 6 月 30 日前 |
| **中风险(大额)** | 关联交易 > 2 亿 / 利润率偏离四分位 | 详细 TNMM 测试 + 可比公司分析(中国数据库) | 同期资料备查 5 月 31 日 |
| **低风险(常规)** | 关联交易 < 2 亿 / 利润率在四分位内 | 简化测试 + 同期资料简版 | 同期资料备查 5 月 31 日 |
| **微小(可简化)** | 关联交易 < 400 万 且 非避税地 | 仅同期资料备案(无本地文档) | 备查即可 |

### 1.3.2 数据库查询指引

**可比公司查询**：
1. 首选:中国可比公司 → Wind / 同花顺 / CSMAR 数据库
2. 次选:区域可比公司 → Osiris / RoyaltyStat
3. 不可行:集团内比较

**关键**：
- 行业代码:国民经济行业分类(GB/T 4754-2017)
- 年度匹配:可比公司数据应为最近 3 个年度
- 财务指标匹配:资产规模、员工人数、收入水平

### 1.4 预约定价安排(APA)类型

| 类型 | 主体 | 谈签周期 | 适用 |
|------|------|---------|------|
| **单边 APA** | 纳税人与本国税务机关 | 1-2 年 | 中国境内 |
| **双边 APA** | 两国税务机关 + 纳税人 | 2-3 年 | 中外双边税收协定 |
| **多边 APA** | 多国税务机关 + 纳税人 | 3-4 年 | OECD/G20 多边 |

**关键**：APA 是管理转让定价风险的有效工具,但**申请周期长**,需提前 1-2 年规划。

### 1.5 BEPS 双支柱 GloBE 规则

**GloBE(Global Anti-Base Erosion)规则** — 2021 年 OECD/G20 达成:

| 支柱 | 内容 | 影响 |
|------|------|------|
| **支柱一(Pillar One)** | 跨国企业利润重新分配给市场国 | 营收 > 200 亿欧元 + 利润率 > 10% 的跨国集团 |
| **支柱二(Pillar Two)** | 全球最低税 15% | 营收 > 7.5 亿欧元的跨国集团 |

**对中国企业的影响**：
- 走出去的"中国跨国企业"可能需补缴 GloBE 税
- 中国居民企业海外子公司利润在中国补税 15%
- "走出去"架构需重新评估

### 1.6 跨境预提所得税与协定待遇

| 所得类型 | 中国法定税率 | 协定常见税率 |
|---------|-----------|------------|
| 股息 | 10% | 5%(子公司向母公司) / 0%(母公司控股≥25%) |
| 利息 | 10% | 7%(部分协定) / 0%(政府/央行) |
| 特许权使用费 | 10% | 6%-7%(部分协定) |
| 租金（不动产） | 10% | 10%(大多数无降低) |

**享受协定待遇 4 要件**：
1. 对方国家与中国有税收协定
2. 对方企业为协定对方税收居民(须提供税收居民证明)
3. 所得符合协定条款范围
4. 不是为了避税目的(主要目的测试 PPT)

### 1.7 常设机构(PE)风险 4 类

| 类型 | 触发条件 | 后果 |
|------|---------|------|
| **固定场所 PE** | 在华有办公场所/厂房 | 在华纳税 |
| **工程型 PE** | 在华工程连续 > 6 个月（建筑）/ 12 个月（安装） | 在华纳税 |
| **代理型 PE** | 在华有权以企业名义签署合同 | 在华纳税 |
| **人员派遣 PE** | 员工在华提供服务 > 183 天/年 | 在华纳税 |

**BEPS 反 PE 规避**：
- "commissionnaire 安排"被认定为 PE
- "fragmentation rule" 拆分业务避开 PE 阈值被禁止

## 2. 税务争议处理流程

### 2.1 税务争议 4 阶段

```
Stage 1: 税务稽查阶段(主动配合 / 申辩)
  ↓
Stage 2: 税务行政处罚(听证 / 不服)
  ↓
Stage 3: 行政复议(60 日内申请)
  ↓
Stage 4: 行政诉讼(6 个月内起诉,复议前置或自由选择)
```

### 2.2 税务稽查应对步骤

| 步骤 | 时限 | 关键动作 |
|------|------|---------|
| 1. 收到通知书 | 即时 | 立即通知税务律师 + 财务总监 |
| 2. 准备资料 | 7 日内 | 整理账簿/凭证/合同 |
| 3. 自查补缴 | 15-30 日 | 评估是否主动补缴(可减罚) |
| 4. 配合稽查 | 持续 | 安排专人配合 / 避免财务负责人直接回答 |
| 5. 收到稽查结论 | 30-90 日 | 评估是否接受 / 不服则进入处罚阶段 |
| 6. 听证 | 处罚前 | 申请听证(重大处罚前) |
| 7. 复议 | 60 日内 | 复议前置 / 或自由选择 |
| 8. 诉讼 | 6 个月内 | 行政诉讼(复议后或直接) |

### 2.3 滞纳金与罚款

| 情形 | 滞纳金 | 罚款 |
|------|--------|------|
| **偷税** | 日万分之五(年化 18.25%) | 50%-5 倍 |
| **欠税** | 日万分之五 | — |
| **骗税** | — | 1-5 倍 + 取消退税资格 |
| **编造虚假计税依据** | — | 5 万以下 |
| **未申报** | 日万分之五 | 2 千-1 万 |

**关键**：
- 偷税+主动补缴+接受处罚 → 罚款可降至 50% 以下(税收征管法 63 条第 3 款)
- 行政复议不停止税款缴纳(但可申请缓交)

### 2.4 偷税认定 4 要件(税收征管法 63 条)

**偷税 = 伪造/变造/隐匿/销毁账簿 + 虚假申报**

| # | 要件 | 认定标准 |
|---|------|---------|
| 1 | 客观行为 | 伪造/变造/隐匿/销毁账簿凭证/在账簿上多列支出/不列少列收入/虚假申报 |
| 2 | 主观故意 | 明知应纳税而故意不缴/少缴 |
| 3 | 结果 | 不缴或少缴税款 |
| 4 | 数额 | 逃税金额 ≥ 10 万 且 占应纳税额 10% |

**逃税罪 vs 偷税**：刑法 201 条规定逃税罪,**首次**偷税+主动补缴+接受处罚 → 不予追究刑事责任(但仍可行政处罚)。

## 3. 税收优惠资质

### 3.1 高新技术企业认定(国科发火[2016]32号)

| 条件 | 要求 |
|------|------|
| 企业性质 | 在中国境内注册 1 年以上 |
| 知识产权 | 核心自主知识产权(发明/实用新型/软件著作权) |
| 技术领域 | 国家重点支持的高新技术领域(电子信息/生物医药/航空航天等) |
| 科技人员占比 | ≥ 10% |
| 研究开发费用占比 | 销售收入 5,000 万-2 亿: ≥ 4%; 2 亿以上: ≥ 3%; < 5,000 万: ≥ 6% |
| 高新收入占比 | ≥ 60% |
| 研发组织管理 | 管理制度 / 研发设备 / 成果转化能力 |

**认定后**：企所税 15%(标准 25%),亏损弥补延至 10 年。

### 3.2 研发费用加计扣除(财税[2023]7号)

| 企业类型 | 加计扣除比例 |
|---------|------------|
| 一般企业 | 100% |
| 制造业 | 100% |
| 科技型中小企业 | 120% |
| 集成电路/工业母机 | 120% |

**关键**：
- 实际发生研发费用 / 加计扣除 = 税前可扣除额
- 加计扣除部分形成的亏损可向后结转 5 年
- 须保持研发费用辅助账 / 加计扣除备查资料

### 3.3 小型微利企业

| 条件 | 标准 |
|------|------|
| 年度应纳税所得额 | ≤ 300 万元 |
| 从业人数 | ≤ 300 人 |
| 资产总额 | ≤ 5,000 万元 |

**优惠**：
- 应纳税所得额 ≤ 100 万 → 实际税负 2.5%
- 100 万-300 万 → 实际税负 5%

### 3.4 其他常见税收优惠

| 优惠类型 | 法律依据 | 主要优惠 |
|---------|---------|---------|
| 西部大开发 | 财税[2020]23号 | 企所税 15% |
| 海南自贸港 | 财税[2020]31号 | 鼓励类企业 15% + 个人所得税 15% |
| 横琴粤澳深度合作区 | 财税[2022]19号 | 鼓励类企业 15% |
| 软件企业 | 财税[2012]27号 | "两免三减半" |
| 集成电路企业 | 财税[2020]45号 | "五免五减半"或"十免" |

## 4. 发票合规深度

### 4.1 三流合一

```
合同流：货物/服务合同与发票内容一致
发票流：发票开具内容与实际交易一致
资金流：付款方/收款方与合同一致

三流不一致 → 涉嫌虚开
```

### 4.2 虚开 4 种情形(刑法 205 条)

| # | 情形 | 后果 |
|---|------|------|
| 1 | 为他人虚开(无真实交易而开具) | 刑事责任 |
| 2 | 让他人为自己虚开(接受虚开) | 刑事责任 |
| 3 | 介绍他人虚开(居间介绍) | 刑事责任 |
| 4 | 自己为自己虚开(无实际业务) | 刑事责任 |

**虚开量刑**：
- 虚开税款 ≥ 50 万 → 3-10 年
- 虚开税款 ≥ 250 万 → 10 年至无期
- 数额特别巨大 / 情节特别严重 → 无期 + 没收财产

### 4.3 善意取得虚开发票 4 要件(国家税务总局公告 2012 年第 33 号)

**善意取得须同时满足**：
1. 购货方不知道取得的发票是虚开的
2. 销售方使用的是其所在省的发票
3. 购货方已按规定付款(付至销售方账户)
4. 购货方按发票载明的货物名称等申报抵扣进项

**处理结果**：
- 善意取得 → 不得抵扣进项,**须转出**
- 非善意取得 → 须补缴税款 + 滞纳金 + 罚款 + 涉嫌虚开

### 4.4 不得抵扣进项 11 项

1. 用于免税项目
2. 用于集体福利或个人消费
3. 用于简易计税
4. 非正常损失对应进项
5. 购进的贷款服务、餐饮服务、居民日常服务、娱乐服务
6. 购进的房地产(租入后再转租)
7. 国务院规定的其他情形

### 4.5 数电发票(2024 年全面推广)

| 特征 | 旧发票 | 数电发票 |
|------|-------|--------|
| 介质 | 纸质 | 数字化(全国统一电子发票服务平台) |
| 红冲 | 需回收原票 | 数字化红冲 |
| 入账 | 纸质打印 | 全数字(电子凭证) |
| 归档 | 纸质归档 | 电子归档(财政电子凭证规范) |
