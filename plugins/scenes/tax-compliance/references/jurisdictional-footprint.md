## 1. 法域分层(5 法域)

### 1.1 中国大陆（默认法域）

| 维度 | 内容 |
|------|------|
| 主管机关 | 国家税务总局 / 财政部 / 海关总署 |
| 主要税种 | CIT 25% / VAT 13%/9%/6%/3% / IIT 7 级超额累进 |
| 域外法适用 | 一般不适用,仅跨境场景触发 |
| 升级路径 | 见 § 1.5 升级路径表 |

### 1.2 香港

| 维度 | 内容 |
|------|------|
| 主管机关 | 香港税务局(IRD) |
| 主要税种 | 利得税 16.5%(首 200 万 8.25%) / 薪俸税 2%-17% / 印花税 |
| 与内地关系 | 内地与香港税收安排(2006 年) — 股息 5%(持股 25%+)/利息 7% |
| 域外法适用 | 英美法系 + 域外收入免税 |
| 高度关注 | 来源地认定 / 离岸收入免税申报 / 经济实质要求 |

### 1.3 澳门

| 维度 | 内容 |
|------|------|
| 主管机关 | 财政局 |
| 主要税种 | 所得补充税(累进 3%-12%) / 职业税 / 印花税 / 旅游税 |
| 与内地关系 | 内地与澳门税收安排(2009 年) |
| 高度关注 | 博彩业特别税收 |

### 1.4 台湾

| 维度 | 内容 |
|------|------|
| 主管机关 | 财政部台北国税局 |
| 主要税种 | 营利事业所得税 20% / 营业税 5% |
| 与内地关系 | 大陆与台湾无正式税收协定(两岸人民关系条例) |
| 高度关注 | 政治风险 / 转单安排合规 |

### 1.5 新加坡

| 维度 | 内容 |
|------|------|
| 主管机关 | 新加坡税务局(IRAS) |
| 主要税种 | 企业所得税 17% / GST 9%(2024 起) |
| 与内地关系 | 中国-新加坡税收协定(2007 年) |
| 域外法适用 | 普通法系 + 属地原则 |
| 高度关注 | 税收居民认定 / 海外收入免税 |

## 3. 跨境升级规则(Pattern 17)

### 3.1 升级触发(命中即 🔴 升级)

| 触发情形 | 升级目标 | 触发条件 |
|---------|---------|---------|
| **重大转让定价调整** | 转让定价专业律师 | 调整金额 ≥ 5000 万 |
| **特别纳税调查** | 税务律师 + 集团 CFO | 收到税务机关调查通知 |
| **避税地关联交易** | 跨境税务律师 | 涉及开曼/BVI/百慕大 |
| **OECD 双支柱 GloBE** | 国际税务律师 | 营收 > 7.5 亿欧元的跨国集团 |
| **BEPS 反避税调查** | 跨境税务律师 | 集团内任何成员被调查 |
| **协定待遇拒绝** | 跨境税务律师 | 享受协定被税务机关否认 |
| **税收居民证明问题** | 跨境税务律师 | 对方税收居民身份存疑 |
| **常设机构(PE)风险** | 跨境税务律师 | 在华经营活动 > 6 个月 |
| **海外子公司注销/转让** | 国际税务律师 + 公司律师 | 涉及重大资产转让 |

### 3.2 跨境 4 档升级路径

```
Level 1 (junior): 跨境交易 < 100 万人民币
  → senior (跨境税务师) via 邮件

Level 2 (senior): 跨境交易 100-1000 万 + 一般跨境
  → tax_director via 邮件+会议

Level 3 (tax_director): 跨境交易 > 1000 万 / 转让定价 / 避税地
  → cfo via 邮件+董事会

Level 4 (cfo): 重大 / 双支柱 / PE 风险 / 协定拒绝
  → ceo via 会议+文件 + 跨境税务律师介入
```

### 3.3 跨境决策树

```
跨境交易 / 跨境架构 / 海外子公司
  ↓
[Step 1] 涉及金额？
  ├─ < 100 万 → Level 1
  ├─ 100-1000 万 → Level 2
  └─ > 1000 万 → Level 3

[Step 2] 是否涉及特殊情形？
  ├─ 避税地 → Level 3 + 跨境税务律师
  ├─ 双支柱 GloBE → Level 4 + 国际税务律师
  ├─ 协定拒绝 → Level 3 + 跨境税务律师
  ├─ PE 风险 → Level 3 + 跨境税务律师
  └─ 无 → 按 Step 1

[Step 3] 是否触发 6 大绝对禁止？
  ├─ 是 → 立即停止 + 税务律师 + 刑事风险评估
  └─ 否 → 按 Step 1/2 升级
```

## 5. 域外法源标注规范

### 5.1 标注规则

**所有跨境税务分析必须标注 [域外]**：

```
例：
- 中国法定预提所得税税率 10% [GOV:企业所得税法实施条例]
- 中港税收安排下股息税率 5% [域外:内地与香港税收安排 第十条]
- BEPS 行动计划下受益所有人限制 [域外:OECD BEPS Action 6]
- 美国 GILTI 税率 10.5% [域外:US IRC § 951A]
```

## 6. 法域升级自检 5 问

每次跨境任务前 agent 自问：

1. ☐ 是否识别所有相关法域（中国大陆 + 对方国家/地区）？
2. ☐ 是否查询了税收协定（避免双重征税）？
3. ☐ 是否标注 [域外] 法源（OECD / 美国 / 香港等）？
4. ☐ 是否考虑 BEPS / 双支柱影响？
5. ☐ 是否按金额 / 特殊情形触发对应升级路径？

**5 问中 ≥ 1 项未完成 → 不输出跨境结论。**
