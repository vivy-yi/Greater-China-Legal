# /consumption-tax-compliance — China Mainland

## 法律背景与法规框架

消费税是中国对特定消费品征收的流转税，由《消费税暂行条例》（国务院令第539号）及其实施细则规定。消费税实行"一次课征制"（通常仅在单一环节征收），税目覆盖15类消费品，包括烟、酒、高档化妆品、贵重首饰、鞭炮焰火、成品油、摩托车、小汽车、高尔夫球具、高档手表、游艇、木制一次性筷子、实木地板、电池和涂料。税率从3%（如涂料）到56%（如甲类卷烟）不等。

消费税与增值税构成互补关系：增值税对所有货物和劳务普遍征收（13%/9%/6%），消费税仅对特定消费品征收。两者的计税依据通常相同（含消费税不含增值税的销售额）。消费税的合规重点在于正确认定税目、确定纳税环节和准确计算应纳税额。滥用消费税优惠政策或错误适用低税率可能导致补税、罚款和滞纳金。

## 工作流程

### 第一步：税目认定

确定产品是否属于消费税应税消费品，以及属于哪个税目：

```
消费税15个税目快速筛查：
□ 1. 烟（甲类/乙类卷烟、雪茄、烟丝）
□ 2. 酒（白酒、黄酒、啤酒、其他酒）
□ 3. 高档化妆品（>=10元/mL(g)的化妆品）
□ 4. 贵重首饰及珠宝玉石（金银/钻石/珠宝玉石）
□ 5. 鞭炮焰火
□ 6. 成品油（汽油、柴油、石脑油、溶剂油、航空煤油、润滑油、燃料油）
□ 7. 摩托车（气缸容量>250ml）
□ 8. 小汽车（乘用车、中轻型商用客车）
□ 9. 高尔夫球具
□ 10. 高档手表（单价>=10000元）
□ 11. 游艇
□ 12. 木制一次性筷子
□ 13. 实木地板
□ 14. 电池（原电池/蓄电池/太阳能电池等免税）
□ 15. 涂料（部分免VOC定额）

不属于以上任一税目 → 不缴纳消费税
```

**税目认定争议常见问题：**
- 产品是否属于"化妆品"：按价格（>=10元/mL(g)）和功能双标准判断
- "小汽车"vs"非汽车"：按国家标准GB/T 3730.1分类判断
- "电池"的免税范围：太阳能电池、全钒液流电池免税
- 不同包装规格（如白酒按度数区分）的税目认定

### 第二步：纳税环节判断

消费税的纳税环节因产品类别而异：

| 税目 | 纳税环节 | 特殊说明 |
|------|---------|---------|
| 烟（卷烟） | 生产+批发（双环节） | 生产环节征56%/36%，批发环节征11%+0.005元/支 |
| 酒（白酒除外） | 生产 | 啤酒250元/吨或220元/吨 |
| 白酒 | 生产 | 20%+0.5元/500g（复合计税） |
| 高档化妆品 | 生产 | 15% |
| 贵重首饰 | 生产（金基除外） | 金银首饰在零售环节征5% |
| 成品油 | 生产 | 单位税额 |
| 小汽车 | 生产 | 1%-40%按排量分档 |
| 电池 | 生产 | 4% |
| 涂料 | 生产 | 4% |

**特殊场景：**
- 委托加工：受托方是消费税扣缴义务人（加工环节）
- 进口：进口环节由海关代征
- 带料加工：视同销售缴纳消费税

### 第三步：应纳税额计算

**从价定率：**
```
应纳税额 = 销售额 × 比例税率
销售额 = 全部价款 + 价外费用（不含增值税）
```

**从量定额（如啤酒、黄酒、成品油）：**
```
应纳税额 = 销售数量 × 定额税率
```

**复合计税（如卷烟、白酒）：**
```
应纳税额 = 销售额 × 比例税率 + 销售数量 × 定额税率
```

**抵扣规则（外购已税消费品继续加工）：**
```
准予抵扣的消费税 = 外购已税消费品已缴纳的消费税 × 使用比例
不可抵扣：酒类（除葡萄酒外）、小汽车、高档手表、游艇、电池、涂料
```

**常见计算错误：**
- ❌ 销售额未换算为不含增值税金额（销售额应÷(1+增值税税率)）
- ❌ 包装物押金的税务处理不当（逾期未退的并入销售额）
- ❌ 关联交易价格偏低被核定
- ❌ 增值税和消费税混用计税依据

### 第四步：申报和缴纳

**纳税申报期限：**
- 1、3、5、10、15日为一期的：期满后5日内预缴，次月15日内申报
- 1个月为一期的：期满后15日内申报
- 进口消费品：海关填发缴款书之日起15日内

**申报表格：**
- 主表：《消费税纳税申报表》
- 附表：《本期准予扣除税额计算表》
- 附表：《本期减（免）税额明细表》

**出口退税：**
- 出口应税消费品可退还已纳消费税
- 退税依据：从价计征的按出口数量×出厂价；定额计征的按出口数量×单位税额

## 升级决策门

以下情况须移交专业税务律师处理：

- 消费税税目认定存在争议（产品是否属于应税品目）
- 涉及委托加工应税消费品的价格核定争议
- 税务机关进行消费税专项稽查
- 消费税出口退税被质疑或拒绝
- 涉及利用特殊安排（如改变包装规格/销售方式）规避消费税
- 存在消费税补税风险（金额重大）
- 涉及虚开消费税发票或逃税嫌疑的调查
