# /vat-credit-calculator — 增值税进项税额计算

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取抵扣规则/不得抵扣情形。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取抵扣规则
2. 识别进项类型
3. 判断是否可抵扣
4. 计算可抵扣税额
5. 输出清单

## 可抵扣进项类型

| 类型 | 抵扣凭证 | 可抵扣税率 |
|---|---|---|
| 增值税专用发票 | 专票 | 按票面税额 |
| 海关进口增值税专用缴款书 | 海关缴款书 | 按票面税额 |
| 农产品收购发票 | 收购发票 | 9%/10%抵扣 |
| 旅客运输发票 | 航空/铁路/公路水路 | 9%/3% |
| 收费公路通行费 | 通行费发票 | 5%/3% |

## 不得抵扣进项的情形（增值税法第27条）

| 情形 | 说明 |
|---|---|
| 简易计税项目 | 用于简易计税项目的进项不得抵扣 |
| 免征增值税项目 | 用于免征增值税项目的进项不得抵扣 |
| 集体福利/个人消费 | 交际应酬/福利费中的消费品不得抵扣 |
| 非常损失 | 非正常损失的购进货物不得抵扣 |
| 贷款服务 | 贷款服务进项不得抵扣（2023年后部分可） |
| 餐饮服务 | 餐饮服务不得抵扣（2023年后可抵扣） |

## 农产品抵扣规则

```
自2019年4月1日起：
- 取得9%专票：按票面税额抵扣
- 取得10%专票：按9%抵扣，1%做进项转出
- 购进用于生产或委托加工13%税率货物：可加计扣除1%
```
