# /eit-return-reviewer — China Mainland

## 法律背景与法规框架

企业所得税（CIT/EIT）年度汇算清缴是企业每年最重要的税务合规事项之一。企业所得税法第54条规定：企业应当自年度终了之日起五个月内，向税务机关报送年度企业所得税纳税申报表，并汇算清缴，结清应缴应退税款。申报表以A类（一般企业）和B类（核定征收企业）为主，其中A类申报表包括1张主表和37张附表，涵盖收入、成本、费用、纳税调整、税收优惠、弥补亏损等全维度。

汇算清缴的核心在于"纳税调整"——会计利润与企业所得税应纳税所得额之间的差异需要通过纳税调增或调减来纠正。常见的纳税调整事项包括：限额扣除调整、资产减值准备、公允价值变动、关联交易、税收优惠适用等。2025年后，数电发票全面推广和"金税四期"对申报数据的自动比对能力大幅提升，申报表的逻辑一致性和数据真实性受到更强的自动监控。

## 工作流程

### 第一步：申报表完整性检查

**基本要素检查：**
```
□ 申报类型：A类（查账征收）/ B类（核定征收）
□ 税款所属期间：202X年1月1日-202X年12月31日
□ 纳税人识别号与企业名称是否一致
□ 是否加盖企业公章
□ 是否由法定代表人/授权负责人签字
□ 是否附送财务报告（资产负债表+利润表+现金流量表）
```

**必报附表检查：**
- A100000 企业所得税年度纳税申报表主表
- A101010 一般企业收入明细表
- A102010 一般企业成本支出明细表
- A104000 期间费用明细表
- A105000 纳税调整项目明细表（核心附表）
- A106000 企业所得税弥补亏损明细表
- A107010 减免所得税优惠明细表（如有优惠）
- A107020 所得减免优惠明细表（如有）

### 第二步：关键项目逻辑检查

**收入类检查：**
- 营业收入与增值税申报的销售额是否匹配
- 视同销售收入是否已申报（样品赠送/内部使用/资产移送）
- 政府补助收入是否正确处理（应税收入/不征税收入）
- 非货币性资产交换是否确认收入
- 股息红利收入是否享受免税优惠

**成本费用类检查：**
- 营业成本与收入是否配比
- 费用跨期是否正确归集
- 资本性支出与费用性支出是否正确区分
- 资产折旧/摊销年限是否符合税法规定
  - 房屋建筑物：20年
  - 生产设备：10年
  - 电子设备：3年
  - 无形资产：10年（软件可缩短至2年）

**纳税调整类检查（A105000表核心）：**

| 调整项目 | 方向 | 常见原因 |
|---------|------|---------|
| 业务招待费超限额 | 调增 | 超过销售收入的5‰×60% |
| 广告费超限额 | 调增 | 超过销售收入的15%或30% |
| 公益性捐赠超限额 | 调增 | 超过利润总额的12% |
| 资产减值准备 | 调增 | 未经核定的准备金不得扣除 |
| 公允价值变动损益 | 调增/调减 | 未实现的损益须纳税调整 |
| 股东借款利息 | 调增 | 超过金融企业同期同类贷款利率 |
| 不合规发票支出 | 调增 | 未取得合规税前扣除凭证 |
| 免税收入（国债利息、股息） | 调减 | 享受免税收入优惠 |

### 第三步：税收优惠适用审查

**适用条件检查清单：**

```
高新技术企业优惠（15%税率）：
□ 高新技术产品（服务）收入占企业同期总收入的比例不低于60%
□ 研发费用占同期销售收入总额的比例符合要求
□ 核心知识产权情况
□ 科技人员占企业当年职工总数的比例不低于10%

小型微利企业优惠（分段税率）：
□ 年应纳税所得额不超过300万元
□ 从业人数不超过300人
□ 资产总额不超过5000万元

研发费用加计扣除：
□ 研发活动属于《国家重点支持的高新技术领域》
□ 研发支出核算准确（人员人工/直接投入/折旧/设计费用等）
□ 研发费用与其他费用划分清晰

西部大开发优惠（15%税率）：
□ 主营业务为《西部地区鼓励类产业目录》中规定的产业
□ 主营业务收入占企业收入总额的60%以上

软件企业/集成电路企业优惠：
□ 企业资质认定有效
□ 软件产品登记/集成电路布图设计登记
```

### 第四步：申报表数据交叉验证

**表间逻辑关系检查：**
- 主表利润总额 = 利润表利润总额（应完全一致）
- A105000纳税调整后所得 = 主表应纳税所得额
- 已预缴税款 = 各季度预缴申报金额汇总
- 应补退税额 = 应纳税额 - 减免税额 - 已预缴税额

**跨系统数据校验（金税四期自动比对项）：**
- 营业收入 vs 增值税申报表销售额
- 工资薪金 vs 个人所得税代扣代缴总额
- 固定资产折旧 vs 增值税进项发票的设备采购
- 银行账户利息收入 vs 贷款利息支出
- 关联交易金额 vs 关联交易申报表

## 升级决策门

以下情况须移交专业税务律师处理：

- 申报表数据与增值税/个税/银行数据存在重大不一致
- 企业享受税收优惠但适用条件存疑
- 存在以前年度亏损弥补的复杂计算
- 涉及企业重组/合并/分立等特殊税务处理
- 申报表中涉及税务稽查或纳税评估调整事项
- 应补税额巨大（财务影响重大）
- 申报期限临近但申报表仍未定稿
