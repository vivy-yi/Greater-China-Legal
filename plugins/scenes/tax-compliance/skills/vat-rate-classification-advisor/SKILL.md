# /vat-rate-classification-advisor — China Mainland

## 法律背景与法规框架

增值税是中国最大的税种，税率结构经过2018年、2019年两次简并后，形成了13%、9%、6%三档基本税率加零税率和免税的税率体系。增值税暂行条例第2条规定了13%和9%两档基本税率，营改增试点实施办法引入了6%税率适用于现代服务业。《增值税法》已于2024年通过，将于2026年1月1日实施，但税率结构基本维持不变。

税率分类的难点在于"边界判定"——同一种销售行为可能因具体内容不同而适用不同税率。最典型的是混合销售（一项销售行为涉及货物+服务，统一按主业税率）与兼营（多种独立销售行为，分别适用不同税率）的区分。增值税发票须按正确税率开具，历史上税率适用错误的，可能导致购买方进项税额抵扣错误，进而引发补税和滞纳金风险。在"金税四期"系统下，税率适用的自动比对更加严格。

## 工作流程

### 第一步：销售行为分类

**税率快速索引：**

| 税率 | 适用范围 | 典型举例 |
|------|---------|---------|
| **13%** | 货物销售+加工修理修配劳务+有形动产租赁 | 一般货物销售、设备生产、设备租赁 |
| **9%** | 不动产+农产品/食品/书籍+交通运输+邮政+建筑+基础电信 | 商品/农产品销售、建筑服务、不动产租赁 |
| **6%** | 现代服务业+增值电信+金融服务+生活服务+无形资产 | 技术服务、咨询、软件、设计、金融服务 |
| **零税率** | 出口货物+国际运输+境外建筑/设计/研发服务 | 出口货物、国际运输服务 |
| **免税** | 农产品自产自销+避孕药品+古旧图书+个人住房等 | 农业生产者销售自产农产品 |

**9%税率的详细清单（货物部分）：**
```
□ 农产品（种植/养殖/水产/林业/牧业产品）
□ 食用植物油
□ 食用盐
□ 自来水/暖气/冷气/热水/煤气/天然气/沼气/居民用煤炭制品
□ 图书/报纸/杂志/音像制品/电子出版物
□ 饲料/化肥/农药/农机/农膜
□ 国务院规定的其他货物
```

**6%税率的详细清单（服务部分）：**
```
□ 研发和技术服务（研发服务、技术转让、技术咨询）
□ 信息技术服务（软件服务、信息技术服务）
□ 文化创意服务（设计服务、商标著作权转让、广告服务）
□ 物流辅助服务（仓储、装卸、货运代理）
□ 鉴证咨询服务（认证、鉴证、咨询、审计、律师）
□ 广播影视服务（制作、发行、播映）
□ 商务辅助服务（企业管理、经纪代理、人力资源）
□ 增值电信服务
□ 金融服务（贷款、保险、金融商品转让）
```

### 第二步：混合销售与兼营判定

**混合销售（统一按主业税率）：**
```
判定标准：一项销售行为同时涉及货物和服务
├─ 例：销售空调+安装服务（如果安装是销售的组成部分）
├─ 例：电梯销售+安装（电梯生产企业的销售和安装）
└─ 适用税率：按主业税率统一适用
    ├─ 制造企业（主业卖货物）：13%
    └─ 建筑企业（主业建筑服务）：9%
```

**兼营（分别适用税率）：**
```
判定标准：同一纳税人同时经营多种独立的业务
├─ 例：销售商品（13%）+技术服务（6%）+不动产租赁（9%）
├─ 例：软件开发（6%）+软件产品销售（13%）
└─ 处理：分开核算分别适用税率
    ├─ 未分别核算的 → 从高适用税率
    └─ 分别核算的 → 各自适用税率
```

**常见争议边界：**
- 设备销售+安装：混合销售（按主业税率）vs 兼营（分开核算）
  - 2017年11号公告后：纳税人销售自产设备同时提供安装，可按兼营处理（分别核算）
  - 非自产设备：仍按混合销售处理
- 软件+技术：6% vs 13%（关键看是单独提供的技术服务还是软件产品附带的服务）
- 餐饮+外卖：6% vs 13%（堂食餐饮6%，外卖餐饮按餐饮服务6%，中央厨房生产的食品13%）

### 第三步：特殊税率场景

**不动产租赁和销售：**
- 不动产租赁（2016年营改增后）：9%（一般纳税人）/5%（简易计税）
- 不动产销售：9%（一般纳税人）/5%（简易计税/小规模）
- 个人出租住房：5%减按1.5%

**金融商品转让：**
- 金融商品转让：6%
- 差额征收（卖出价-买入价后征税）
- 负差可结转下一纳税期（不可跨年）

**出口退税零税率适用条件：**
```
零税率适用条件：
□ 货物出口（一般货物退税率为13%/9%/6%/0%）
□ 国际运输服务
□ 境外研发/设计服务
□ 向境外单位提供的完全在境外消费的服务

退税计算：
├─ 免抵退税：适用于生产企业
└─ 免退税：适用于外贸企业

不得适用零税率的情形：
× 出口不予退（免）税的货物和劳务
× 出口国家禁止出口的货物
× 未在规定期限内申报退免税
```

### 第四步：发票开具与申报

**税率填写规则：**
- 发票上的税率须与实际适用税率一致
- 数电发票（全面数字化电子发票）系统自动带出税率
- 税率填写错误的后果：
  - 少开税率 → 购买方少抵扣，销售方补税
  - 多开税率 → 购买方多抵扣，双方都可能被要求说明

**申报表填报：**
- 按不同税率分行填报销售额和销项税额
- 简易计税的销售额和税额单独填报
- 免税销售额单独填报

**税率适用错误的处理（发现后）：**
1. 当月发现：作废原发票，重新开具正确税率的发票
2. 跨月发现：开具红字发票（冲回原发票），重新开具
3. 跨年发现：申报更正，补缴或少缴税款
4. 补税须加收滞纳金（每日万分之五）

## 升级决策门

以下情况须移交专业税务律师处理：

- 混合销售和兼营的认定存在争议（可能导致大额补税）
- 出口退税零税率/免税的适用条件被税务机关质疑
- 历史发票税率适用错误涉及金额重大
- 税务机关对税率适用范围进行专项稽查
- 涉及增值税免税政策的适用范围争议
- 复杂交易结构（如设备+服务+融资租赁组合）的税率适用
- 涉及行业特殊政策（如农产品增值税进项税额核定扣除、资源综合利用产品即征即退）的税率适用
