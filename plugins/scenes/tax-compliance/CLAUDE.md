## 发票合规规则

### 增值税发票类型

| 类型 | 适用范围 |
|---|---|
| 增值税专用发票 | 一般纳税人，可抵扣进项 |
| 增值税普通发票 | 不可抵扣进项 |
| 机动车销售统一发票 | 车辆购置税 |
| 电子发票 | 同上，含数电发票 |

### 虚开增值税发票风险

| 情形 | 风险 |
|---|---|
| 为他人开具与实际不符的发票 | 虚开，刑事责任 |
| 自己开具与实际不符的发票 | 虚开，刑事责任 |
| 介绍他人开具与实际不符的发票 | 虚开，刑事责任 |
| 善意取得虚开发票 | 不得抵扣，需补税 |

### 善意取得判断（国家税务总局公告2012年第33号）

```
善意取得须同时满足：
1. 购货方不知道取得的发票是虚开的
2. 销售方使用的是其所在省的发票
3. 购货方已按规定付款
4. 购货方按发票载明的货物名称等申报抵扣进项
```

## 转让定价风险

### 关联交易的类型

| 类型 | 风险 |
|---|---|
| 购销交易（有形资产） | 价格偏离市场价 |
| 劳务提供 | 费用不合理 |
| 资金融通 | 利率偏离市场 |
| 无形资产转让 | 许可费不合理 |

### 转让定价方法

| 方法 | 适用范围 |
|---|---|
| 可比非受控价格法 | 有市场价的交易 |
| 成本加成法 | 制造业/服务业 |
| 利润分割法 | 高价值无形资产 |
| 交易净利润法 | 难以找到可比方 |

### 同期文档要求

| 企业类型 | 文档要求 |
|---|---|
| 年度关联交易>2000万 | 主体文档 |
| 年度关联交易>400万且利润总额为负 | 本地文档 |
| 单一类型关联交易企业 | 可简化 |

## 跨境税务

### 预提所得税

| 所得类型 | 税率 |
|---|---|
| 股息 | 10%（有税收协定可降低） |
| 利息 | 10%（有税收协定可降低） |
| 特许权使用费 | 10%（有税收协定可降低） |
| 租金（不动产） | 10% |

### 税收协定待遇

```
享受条件：
1. 对方国家与中国有税收协定
2. 对方企业为协定对方税收居民
3. 所得符合协定条款范围
4. 对方企业须向中国税务机关提供税收居民证明
```
