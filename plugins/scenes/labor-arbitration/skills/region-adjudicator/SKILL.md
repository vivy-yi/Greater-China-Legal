# /region-adjudicator — 各省市劳动仲裁裁判口径查询

## 核心原则

**本skill不存储静态数据，数据必须从工具实时获取。**

地区差异数据（工资基数/加班费/高溫津贴等）由以下渠道提供：
- [YD] 法律精灵 — 实时检索
- [GOV] 裁判文书网 — 官方判例
- [GOV] 各地人社局官网 — 官方文件

具体查询路径和判断框架见 references/ 目录。

---

## 查询流程

### 第一步：判断数据类型

| 要查询的问题 | 对应的数据类型 |
|---|---|
| 经济补偿金基数是否含加班费/奖金 | 工资基数口径 |
| N还是N+1还是2N | 解除类型认定 |
| 违法解除赔偿金计算 | 赔偿金口径 |
| 未休年假工资计算 | 年休假口径 |
| 高温津贴是否必须支付 | 津贴补贴口径 |
| 劳动合同到期不续约是否有补偿 | 终止续订口径 |

---

### 第二步：按数据类型查询

#### 工资基数口径（加班费/奖金是否计入）

```
查询目标：用人单位解除劳动合同时，加班费和奖金是否计入经济补偿金基数？

查询路径：
1. [YD] 法律精灵搜索："[城市名] 经济补偿金基数 加班费" + 近2年案例
2. [GOV] 裁判文书网：中院/高院相关判例
3. 如无直接数据，查[WKL]律师分析文章

判断逻辑（见 references/判断框架.md）：
- 多源一致 → 高置信度
- 单源权威 → 中置信度，标注建议确认
- 多源矛盾 → 呈现差异，保守处理
```

#### 解除类型认定（N/N+1/2N）

```
查询目标：某解除情形在当地被认定为合法解除、经济性裁员还是违法解除？

查询路径：
1. [YD] 法律精灵搜索："[城市名] 解除劳动合同 类型 认定" + 近2年案例
2. 重点关注：[YD]中是否有当地仲裁委发布的典型案例集

判断逻辑：
- 协商解除 → N（法定标准）
- 员工过错解除 → 无补偿（须单位举证）
- 员工非过错解除 → N或N+1（须符合法定情形）
- 经济性裁员 → N（须符合法定程序）
- 程序/实体任一违法 → 违法解除 → 2N
```

---

### 第三步：置信度评估

| 置信度 | 条件 | 处理方式 |
|---|---|---|
| 高 | 多源一致 / 官方文件明确 | 直接使用 |
| 中 | 单源权威 / 同省数据 | 使用，建议电话确认 |
| 低 | 多源矛盾 / 无本地数据 | 保守处理，明确标注 |
| 无 | 完全无数据 | 使用全国一般标准，标注风险 |
