# 判断框架 — 数据矛盾时怎么办

## 核心原则

**中国劳动争议没有统一的全国性裁判口径，只有各地仲裁委和法院的"自由裁量空间"。**

法条是统一的（劳动合同法/劳动法），但裁量的尺度各地不同。遇到数据矛盾时，需要判断框架来决策。

---

## 矛盾类型与处理方式

### 类型一：不同法院/仲裁委的判例结论矛盾

**典型场景：**
- 北京某中院判例说"加班费计入补偿金基数"
- 上海某中院判例说"加班费不计入补偿金基数"

**处理方式：**
1. **确定目标用户的管辖地**
   - 劳动合同履行地 → 优先适用当地仲裁委/法院的口径
   - 公司注册地 → 注册地仲裁委/法院口径也有管辖权

2. **按管辖地优先级**
   ```
   首选：劳动合同履行地的仲裁委口径
   次选：公司注册地的仲裁委口径
   参考：同类城市的仲裁委口径（同一省份或经济圈）
   ```

3. **引用时注明来源和适用范围**
   ```
   ⚠️ 本结论基于[城市][法院/仲裁委][年份]判例，
      仅适用于[适用城市]，其他地方可能不同
   ```

---

### 类型二：官方文件与法院判例矛盾

**典型场景：**
- 某省人社厅文件规定"高温津贴每人每月300元"
- 当地法院判例认为"高温津贴属于工资，应计入加班费基数"

**处理方式：**

| 优先级 | 来源 | 说明 |
|---|---|---|
| 高 | 法院判例 | 实际裁判结果，比文件规定更能说明裁判口径 |
| 中 | 人大/政府文件 | 法律依据，但执行中可能有裁量空间 |
| 低 | 学者解读/律师文章 | 参考价值，须注意偏见 |

**例外：** 法定标准（如最低工资、社保基数）以官方文件为准，不受判例影响。

---

### 类型三：当年数据与往年数据矛盾

**典型场景：**
- 2022年查到的口径是"加班费不计入基数"
- 2024年有新的判例说"应当计入"

**处理方式：**
1. **时效判断**
   - 近2年内的判例 > 3年前的判例
   - 法规修订后的判例 > 修订前的判例

2. **趋势判断**
   - 注意劳动争议裁判口径的全国性趋同趋势
   - 最高院指导案例影响各地高院，进而影响中院和基层

3. **保守处理**
   ```
   如最近判例与较早判例矛盾，
   → 呈现两种口径
   → 明确告知用户存在不确定性
   → 建议向当地仲裁委电话确认
   ```

---

### 类型四：法律规定与实际操作矛盾

**典型场景：**
- 劳动合同法规定经济补偿金"月工资包括计时工资、计件工资、奖金、津贴、补贴"
- 实际操作中各地均不把加班费计入

**这类矛盾无法通过查询解决，属于"法律实践中的灰色地带"。**

**处理方式：**
1. **以实际裁判口径为准**（法律条文是"书面规则"，裁判口径是"实际规则"）
2. **查询时以判例为主**，官方文件为辅
3. **输出时明确区分"法律规定"和"裁判口径"**

---

## 置信度评估表

| 置信度 | 条件 | 使用建议 |
|---|---|---|
| 高 | 多源一致（判例+文件+律师分析） | 直接使用 |
| 中 | 单源权威（官方文件或高院判例） | 使用，建议电话确认 |
| 低 | 多源矛盾或数据较旧（>3年） | 保守处理，呈现差异 |
| 无 | 完全查不到本地数据 | 使用全国一般标准，明确标注 |

## 输出时的风险提示模板

### 置信度高
```
✅ 查询结果置信度高，可直接使用。
结论：[具体结论]
来源：[来源清单]
```

### 置信度中
```
⚠️ 查询结果置信度中等，建议向当地仲裁委（12333）电话确认。
本结论基于：[主要来源]
建议确认的问题：[具体问题]
```

### 置信度低
```
🔴 查询结果存在不确定性，保守处理。
可能的结论1：[结论A]（依据：[来源]）
可能的结论2：[结论B]（依据：[来源]）
建议：向当地仲裁委（12333）电话确认，或咨询专业律师。
风险提示：[如果按结论A处理可能的风险]
```

### 无数据
```
⚠️ 未查到本地数据，使用全国一般标准。
全国一般标准：[标准内容]
本地可能有差异的方面：[具体方面]
建议：向当地仲裁委（12333）电话确认。
```

## 特殊情形：多地管辖权冲突

**典型场景：**
- 员工在上海工作（劳动合同履行地）
- 公司注册在北京（公司注册地）
- 劳动合同约定管辖为深圳

**处理方式：**

| 情形 | 管辖仲裁委 | 管辖法院 |
|---|---|---|
| 劳动合同履行地 | ✅ 优先 | ✅ 优先 |
| 公司注册地 | ✅ 有权 | ✅ 有权 |
| 合同约定地 | ❌ 一般不接受 | ✅ 如约定法院管辖则有效 |

**注意：** 劳动仲裁管辖权以劳动合同履行地和用人单位所在地为主，约定管辖在仲裁阶段一般不适用。

**建议：**
1. 按劳动合同履行地查询口径（最常用）
2. 如有管辖权争议，在提交仲裁申请前先咨询当地仲裁委
