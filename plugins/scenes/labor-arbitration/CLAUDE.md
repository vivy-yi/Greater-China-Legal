### § A8.1 劳动仲裁特别注意 4 大块（仲裁时效 + 工伤 + 集体争议 + 涉群体性）

> **核心原则**：劳动仲裁的核心是**时效严格 + 工伤分级 + 集体协商**。错过时效 = 失权；工伤分级错误 = 赔偿争议。

#### 块 1：劳动仲裁时效（《劳动争议调解仲裁法》）

**核心法规：**
- 《劳动争议调解仲裁法》（2007 + 2021 修订）
- 《劳动法》（2018 修正）
- 《劳动合同法》（2012 修订）

**仲裁时效关键节点：**

| 类型 | 时效 | 法条 |
|------|------|------|
| **一般劳动争议** | **1 年**（知道或应当知道权利被侵害之日起）| 《调解仲裁法》第 27 条 |
| **劳动关系存续期间欠薪** | 不受 1 年限制（特殊）| 第 27 条 第 4 款 |
| **劳动关系终止** | 1 年（终止之日起）| 第 27 条 第 4 款 |
| **工伤认定** | 1 年（事故伤害发生之日 / 职业病诊断之日）| 《工伤保险条例》第 17 条 |

**时效中止 / 中断：**

| 情形 | 效力 |
|------|------|
| **不可抗力** | 时效中止 |
| **丧失行为能力无代理人** | 时效中止 |
| **主张权利** | 时效中断 |
| **对方同意履行** | 时效中断 |
| **请求救济** | 时效中断 |

**blocks（绝对禁止 — 时效）**：
- 超过 1 年时效 → **失权**
- 工伤超过 1 年未申请 → **失权**
- 劳动关系存续期间欠薪时效已过 → **可主张**

**work but ships（可补救）**：
- 时效临近 → 紧急申请 + 时效中断证据
- 时效已过但有正当理由 → 申请时效中止 / 中断

#### 块 2：工伤认定 + 劳动能力鉴定

**核心法规：**
- 《工伤保险条例》（2010 修订）
- 《工伤认定办法》（人社部）
- 《劳动能力鉴定——职工工伤与职业病致残等级》（GB/T 16180-2014）

**工伤认定 7 类情形（《工伤保险条例》第 14 条）：**

| 情形 | 含义 |
|------|------|
| **工作时间 / 工作场所 / 工作原因** | 三要素同时满足 |
| **工作时间前后** | 工作场所内 + 从事预备性 / 收尾性工作 |
| **因履行工作职责** | 受到暴力等意外伤害 |
| **患职业病** | 工作相关疾病 |
| **因工外出** | 因工外出期间 + 工作原因 |
| **上下班途中** | 合理路线 + 交通事故（非本人主要责任）|
| **法律 / 行政法规规定** | 其他情形 |

**视同工伤 3 类（《工伤保险条例》第 15 条）：**

| 情形 | 含义 |
|------|------|
| **工作时间 / 工作岗位** | 突发疾病 48 小时内死亡 |
| **因公外出** | 因公外出的意外事故 |
| **抢险救灾** | 见义勇为 / 抢险救灾 |

**劳动能力鉴定 10 级（GB/T 16180-2014）：**

| 级别 | 伤残程度 | 主要赔偿 |
|------|---------|---------|
| **1-4 级** | 完全 / 大部分 / 部分 / 小部分生活不能自理 | 一次性 + 长期 |
| **5-6 级** | 长期 / 部分不能工作 | 一次性 + 部分 |
| **7-10 级** | 部分 / 轻微 / 轻微 | 一次性 |

**blocks（绝对禁止 — 工伤）**：
- 工伤后未在 30 日内申请 → 用人单位逾期 → 1 年内仍可申请（职工本人 / 家属）
- 工伤认定被拒绝 → **60 日内复议 / 6 个月内诉讼**
- 拒付工伤保险待遇 → **民事赔偿 + 责令改正**

**work but ships（可补救）**：
- 申请材料不齐 → 30 日内补正
- 鉴定结论不服 → 15 日内申请复鉴

#### 块 3：集体劳动争议 + 集体协商（《集体合同规定》）

**核心法规：**
- 《集体合同规定》（劳动和社会保障部 2004）
- 《工会法》（2021 修订）
- 《劳动争议调解仲裁法》第 7 条（集体争议）

**集体争议 3 种情形：**

| 情形 | 含义 |
|------|------|
| **劳动一方 10 人以上** | 有共同请求的劳动者 ≥ 10 人 |
| **工会代表** | 工会代表全体职工 |
| **集体合同争议** | 集体合同签订 / 履行争议 |

**集体协商 5 步流程：**

| 步骤 | 内容 |
|------|------|
| 1. **提出协商** | 一方提出 + 协商时间 |
| 2. **代表确定** | 职工方代表（工会 / 职工代表）+ 用人方代表 |
| 3. **协商会议** | 协商会议（一般 60 日内达成）|
| 4. **达成协议** | 集体合同草案 + 职代会通过 |
| 5. **报送审查** | 劳动行政部门 15 日内审查 |

**集体争议仲裁 3 类程序：**

| 程序 | 适用 |
|------|------|
| **就地仲裁** | 职工 10 人以上 + 共同请求 |
| **集体协商** | 工会代表 + 用人方协商 |
| **集体诉讼** | 法院集体诉讼 / 代表人诉讼 |

**blocks（绝对禁止 — 集体争议）**：
- 群体性突发事件处理不当 → **维稳 + 民事赔偿**
- 工会代表资格被剥夺 → **《工会法》第 52 条**
- 集体协商被拒绝 → **劳动行政部门责令改正**

#### 块 4：劳动合同解除 / 终止特别保护

**核心法规：**
- 《劳动合同法》第 39-46 条（解除 / 终止）
- 《劳动合同法》第 48 条（违法解除赔偿）
- 《女职工劳动保护特别规定》

**不可解除劳动者 6 类情形（劳动合同法第 39 条 / 第 40 条 / 第 41 条限制）：**

| 情形 | 不可解除 |
|------|---------|
| **三期女职工**（孕期 / 产期 / 哺乳期）| ✅ 不可解除（除第 39 条严重违纪）|
| **医疗期**（6 个月以上病假）| ✅ 不可解除（除第 39 条）|
| **职业病观察期** | ✅ 不可解除 |
| **工伤丧失 / 部分丧失劳动能力** | ✅ 不可解除（5-6 级例外）|
| **连续工作满 15 年** 且距退休 < 5 年 | ✅ 不可解除 |
| **工会工作人员履职期间** | ✅ 不可解除 |

**经济补偿金 N 计算：**

| 工龄 | 月数 |
|------|------|
| **满 1 年** | 1 个月工资 |
| **满半年不满 1 年** | 1 个月工资 |
| **不满半年** | 半月工资 |
| **月工资 > 当地平均 3 倍** | 3 倍封顶（最长 12 年）|

**违法解除赔偿（2N）：**

| 情形 | 赔偿 |
|------|------|
| 违法解除 / 终止 | 2 倍经济补偿金（2N）|

**blocks（绝对禁止 — 解除）**：
- 违法解除三期女职工 → **2N + 恢复劳动关系**
- 违法解除医疗期员工 → **2N + 恢复劳动关系**
- 违法解除工会工作人员 → **2N + 责令改正**

**work but ships（可补救）**：
- 程序瑕疵 → 重新送达 + 完善程序
- 协商解除 → 签署协商解除协议

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **超过仲裁时效** | 《调解仲裁法》第 27 条 | 失权 |
| 2. **违法解除三期 / 医疗期** | 《劳动合同法》第 48 条 | 2N + 恢复劳动关系 |
| 3. **拒付工伤保险** | 《工伤保险条例》第 62 条 | 责令补缴 + 滞纳金 + 罚款 |
| 4. **拒签 / 拒续无固定期限合同** | 《劳动合同法》第 14 条 | 视为订立无固定期限 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时限
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — 劳动仲裁增强版）**：
- 争议类型（解除 / 工资 / 工伤 / 社保 / 竞业）？（仲裁请求基础）
- 时效状态（1 年内 / 临近 / 已过）？（时效核查）
- 主体身份（三期 / 医疗期 / 工伤 / 高管）？（解除合法性）
- 集体争议（10 人以上）？（集体协商 + 工会）
- 涉外（外籍 / 港澳台 / 海外派遣）？（涉外律师）
- 调解优先？（调解 → 仲裁 → 诉讼 3 步）

## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** 争议类型识别 → 阶段 skill

**关键节点:**

```
Step 1: 劳动仲裁申请(1 年时效)
Step 2: 仲裁受理 + 立案
Step 3: 举证 + 庭审
Step 4: 仲裁裁决(终局 / 非终局)
Step 5: 起诉(一审 / 二审)
```

**特别注意:**

```
✅ 劳动争议必须先经仲裁(《劳动法》第 79 条)
✅ 时效 1 年
✅ 部分案件一裁终局(《劳动争议调解仲裁法》第 47 条)
✅ 起诉权:非终局案件可起诉 / 终局案件特定情形可起诉
```
