### § A5 Outputs

**Reviewer note 5 行(劳动仲裁特化):**
1. 案件基本信息:[争议类型 / 劳动关系 / 申请人 / 被申请人]
2. 主要诉求:[恢复劳动关系 / 工资 / 补偿 / 赔偿]

## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** 争议类型识别 → 阶段 skill

**关键节点:**

```
Step 1: 劳动仲裁申请(1 年时效)
Step 2: 仲裁受理 + 立案
Step 3: 举证 + 庭审
Step 4: 仲裁裁决(终局 / 非终局)
Step 5: 起诉(一审 / 二审)
```

**特别注意:**

```
✅ 劳动争议必须先经仲裁(《劳动法》第 79 条)
✅ 时效 1 年
✅ 部分案件一裁终局(《劳动争议调解仲裁法》第 47 条)
✅ 起诉权:非终局案件可起诉 / 终局案件特定情形可起诉
```

| 等级 | 案件 | 处理 |

| ⚠️ MEDIUM | 一般案件 | 主办律师 |

### § B5 升级触发(7 类)

1. **群体性** → 主办 + 集体协商
2. **死亡 / 重伤** → 主办 + 律所审批
3. **涉外** → 主办 + 涉外律师
4. **一裁终局案件** → 主办 + 律所审批
5. **违法解除** → 主办 + 赔偿评估
6. **重大工伤** → 主办 + 工伤律师
7. **集体合同争议** → 主办 + 工会协调

### § B12 Per-matter Side(P7)

劳动仲裁**严格隔离:**

| Side | 注意 |
|------|------|

| 事项 | 力度 |
|------|------|
| 仲裁 | 强 |
| 一裁终局 | 强(可起诉情形有限) |

### § B14 Risk calibration

**段 1 识别:** 时效风险 / 证据风险 / 解除合法性 / 涉外风险 / 群体风险
**段 2 量化:** 概率 × 影响 + 缓释
**段 3 响应:** 接受 / 缓释(和解)/ 转移(一裁终局)/ 规避(执行)

### § B15 7 条设计哲学(劳动特化)

1. **保护劳动者** — 法律倾斜
2. **仲裁前置** — 必须先经仲裁
3. **1 年时效** — 严格
4. **证据规则** — 工资记录举证责任
5. **群体协商** — 重大案件
6. **一裁终局** — 快速结案
7. **调解优先** — 节约成本
