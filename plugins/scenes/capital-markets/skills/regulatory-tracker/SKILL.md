# 监管动态追踪技能 (regulatory-tracker)

## 技能描述
追踪A股IPO相关监管政策动态、审核问询热点、典型案例及法规修订，为业务决策提供信息支持。

## 追踪范围

### 一、证监会及交易所文件
| 文件类型 | 来源 |
|----------|------|
| 部门规章 | 证监会官网 |
| 规范性文件 | 证监会官网 |
| 审核问答 | 交易所官网 |
| 监管规则适用指引 | 证监会官网 |

### 二、窗口指导意见
- 发行监管类申请常见问题
- 科创板/创业板审核动态
- 现场检查安排

### 三、典型问询案例

#### 3.1 科创板问询重点
1. 符合科创板定位的论证
2. 核心技术先进性论证
3. 研发费用归集合理性
4. 供应商及客户依赖
5. 收入确认政策
6. 存货核算方法

#### 3.2 主板问询重点
1. 持续盈利能力
2. 关联交易合理性
3. 财务真实性
4. 内部控制有效性
5. 同业竞争解决措施

### 四、法规更新追踪

#### 4.1 近期重要法规
| 法规名称 | 生效日期 | 主要变化 |
|----------|----------|----------|
| 证券法（2019修订） | 2020.3.1 | 注册制全面推进、信息披露强化 |
| 首发办法（2023修订） | 待确认 | 持续经营期计算优化 |
| 科创板办法 | 持续修订 | 持续追踪科创板评价指标 |

#### 4.2 审核理念变化
- 从"持续盈利"向"持续经营"转变
- 信息披露"以投资者需求为导向"
- "问出一家真公司"审核理念

### 五、行业审核动态
| 行业 | 审核关注点 |
|------|------------|
| 医药 | 核心产品研发阶段、商业化能力 |
| 半导体 | 核心技术自主可控、供应链安全 |
| 新能源 | 产能利用率、市场空间 |
| 互联网 | 数据安全、用户隐私保护 |

### 六、使用方法
1. 输入追踪主题或时间范围
2. 系统自动检索相关监管文件、问询案例
3. 生成监管动态简报

### 七、输出格式
```
监管动态追踪报告
报告期：XXXX年XX月
一、新规速递
   [文件名称] | [核心内容] | [生效日期]
二、审核动态
   [问询热点] | [涉及企业] | [审核关注点]
三、典型案例
   [案例名称] | [问题类型] | [处理结果]
四、趋势研判
   [下一阶段审核重点预判]
```
