# Skill: information-disclosure-monitor（信息披露监控）

## 功能定位

监控债券发行后的持续信息披露义务，帮助发行人及时、完整、准确地履行信息披露职责，同时帮助投资者追踪债券信息动态。

## 适用范围

| 债券类型 | 监管框架 | 主要披露法规 |
|---------|---------|-------------|
| 公司债 | 证监会/交易所 | 《公司债券发行与交易管理办法》 |
| 企业债 | 证监会 | 《企业债券管理条例》 |
| 可转债 | 证监会/交易所 | 《上市公司证券发行管理办法》 |

## 定期报告内容要求

### 年度报告（必须披露）

**披露时间**：每一会计年度结束之日起4个月内

**内容要求**：

| 章节 | 主要内容 |
|-----|---------|
| 债券基本情况 | 债券代码、简称、发行规模、利率、期限、上市地点 |
| 发行人基本情况 | 公司概况、经营范围、主营业务 |
| 债券兑付情况 | 本年度付息次数、金额、是否按时支付 |
| 募集资金使用情况 | 实际使用金额、闲置资金使用情况、是否变更用途 |
| 审计意见 | 审计报告类型（标准无保留/保留/否定） |
| 财务报告 | 资产负债表、利润表、现金流量表 |
| 重大事项 | 诉讼仲裁、资产重组、债务违约等 |
| 信用评级（如有） | 评级机构、评级结论、评级展望 |

### 中期报告（必须披露）

**披露时间**：每一会计年度的上半年结束之日起2个月内

**内容要求**：

| 章节 | 主要内容 |
|-----|---------|
| 债券基本情况 | 同年度报告 |
| 发行人概况 | 上半年经营情况 |
| 债券付息情况 | 上半年付息情况 |
| 募集资金使用 | 截至6月30日使用情况 |
| 财务报告 | 半年度财务报表 |
| 重大事项 | 上半年发生的重大事项 |

### 季度报告（上市公司可转债须披露）

**披露时间**：季度结束后15日内

**内容要求**：简要财务报告 + 重大事项

## 临时报告触发条件

### 重大事项标准（须立即披露）

根据《公司债券发行与交易管理办法》，以下重大事项发生之日起2个工作日内披露：

| 序号 | 重大事项类型 | 说明 |
|-----|------------|------|
| 1 | 债券名称变更 | 发行人名称变更 |
| 2 | 经营方针重大变化 | 主营业务方向、经营模式变化 |
| 3 | 经营范围重大变化 | 新增或取消重要业务 |
| 4 | 重大资产重组 | 购买或出售资产达到一定比例 |
| 5 | 重大投资行为 | 对外投资超过资产一定比例 |
| 6 | 重大诉讼仲裁 | 涉及金额超过净资产一定比例 |
| 7 | 发行债券逾期未兑付 | 银行借款或其他债务违约 |
| 8 | 无法按期支付利息 | 债券利息无法按时支付 |
| 9 | 减资、合并、分立 | 发行人重大组织变更 |
| 10 | 申请破产 | 主动或被动申请破产 |
| 11 | 解散或者吊销执照 | 发行人主体资格变化 |

### 债券特殊事项

| 事项类型 | 披露时点 | 特殊要求 |
|---------|---------|---------|
| 付息公告 | 付息日前5个工作日 | 载明付息金额、付息日期、付息方式 |
| 兑付公告 | 兑付日前15个工作日 | 载明兑付金额、兑付日期 |
| 赎回公告 | 触发赎回条件当日 | 载明赎回价格、赎回程序 |
| 回售公告 | 触发回售条件当日 | 载明回售价格、回售程序 |
| 持有人会议 | 会议召开前10个工作日 | 载明会议议题、召开时间地点 |
| 评级调整 | 评级决定作出后2个工作日 | 载明调整原因、新评级结论 |

## 可转债特殊披露要求

### 转股情况披露

| 披露事项 | 时点要求 |
|---------|---------|
| 转股价格调整 | 调整生效当日 |
| 转股价格下修 | 决议当日 |
| 转股结果 | 每次转股完成后2个工作日内 |
| 赎回触发 | 触发当日 |
| 回售触发 | 触发当日 |

### 下修条款触发披露

当公司股票在任意连续N个交易日中有N个交易日收盘价低于当期转股价格的X%时：
- 董事会应当审议向下修正转股价格
- 决议须经出席股东大会股东所持表决权2/3以上通过
- 修正后转股价格不低于股东大会前N个交易日均价

## 信息披露渠道

| 披露渠道 | 适用范围 |
|---------|---------|
| 交易所信息披露系统 | 上市公司债券、可转债 |
| 证监会指定网站 | 所有债券 |
| 发行人官方网站 | 补充披露 |
|  Bloomberg/Wind | 市场数据 |

## 违规处罚

### 未按规定披露的处罚

| 违规情形 | 处罚措施 |
|---------|---------|
| 未按期披露年报/中报 | 责令改正、公开谴责、限制交易 |
| 披露信息存在虚假记载 | 警告、罚款、市场禁入 |
| 重大事项未及时披露 | 责令改正、警告、罚款 |
| 泄露信息导致内幕交易 | 依法处理 |

### 典型违规案例类型

1. **财务数据造假**：虚增收入、利润
2. **隐瞒重大诉讼**：未披露正在进行的重要诉讼
3. **虚假承诺**：募集说明书中的承诺未履行
4. **延迟披露**：超过规定时限才披露

## 披露合规自检清单

### 年度报告披露前自检

```
□ 财务报表是否经审计
□ 审计意见类型是否为标准无保留
□ 债券兑付是否按时完成
□ 募集资金使用是否与承诺一致
□ 是否存在未披露的重大事项
□ 信用评级是否有效
□ 实际控制人是否变化
□ 主要资产是否重大变化
□ 担保人或担保物是否重大变化
```

### 临时报告触发自检

```
□ 是否发生应当披露的重大事项
□ 事项是否属于2个工作日内披露情形
□ 披露内容是否完整准确
□ 是否履行内部审批程序
□ 是否通知受托管理人
```

## 使用示例

**场景**：发行人某上市公司于3月15日收到法院判决书，判决其赔偿原告5000万元。公司净资产为8亿元。

**判断**：
- 5000万元 ÷ 8亿元 = 6.25%
- 超过净资产的10%标准（如有类似规定）或属于"重大诉讼"
- 属于"重大诉讼仲裁"类型，应在2个工作日内披露

**应披露内容**：
- 诉讼基本情况
- 判决结果
- 对发行人的影响
- 拟采取的应对措施
