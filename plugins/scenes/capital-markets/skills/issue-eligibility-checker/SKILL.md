# 发行资格核查技能 (issue-eligibility-checker)

## 技能描述
核查发行人是否符合A股IPO发行条件，覆盖主体资格、规范运行、财务与会计指标等各维度。

## 核查依据
- 《首次公开发行股票并上市管理办法》第六条、第九条、第十一条
- 《科创板首次公开发行股票注册管理办法（试行）》第九条、第十条、第十二条
- 《证券法》第十二条

## 核查维度

### 一、主体资格
| 项目 | 主板/创业板 | 科创板 |
|------|------------|--------|
| 设立时间 | 股份有限公司成立满3年 | 股份有限公司成立满3年 |
| 注册资本 | 足额缴纳，主要资产无重大权属纠纷 | 足额缴纳，主要资产无重大权属纠纷 |
| 主营业务 | 最近3年无重大不利变化 | 符合科创板定位 |
| 实际控制人 | 最近3年无变更 | 最近3年无变更 |

### 二、规范运行（第9条）
- 建立健全三会一层制度
- 董事、监事、高管了解法定义务
- 不存在禁入情形：
  - 被证监会禁入尚在禁入期
  - 最近36个月受行政处罚
  - 最近12个月受交易所公开谴责
  - 涉嫌犯罪被立案调查尚未有明确结论

### 三、财务指标（第11条）
| 指标 | 要求 |
|------|------|
| 净利润 | 最近3年均为正且累计≥3000万元 |
| 现金流量 | 3年累计经营现金流≥5000万 或 3年累计营收≥3亿 |
| 发行前股本 | ≥3000万元 |
| 无形资产占比 | ≤20%（扣除土地等） |
| 未弥补亏损 | 不存在 |

### 四、科创板特殊条件（第10条）
预计市值满足下列条件之一：
1. ≥10亿：近2年净利润均为正且累计≥5000万，或近1年净利润为正且营收≥1亿
2. ≥15亿：近1年营收≥2亿，且3年研发投入占比≥15%
3. ≥20亿：近1年营收≥3亿，且3年经营现金流≥1亿
4. ≥30亿：近1年营收≥3亿
5. ≥40亿：主要产品经国家有关部门批准，市场空间大

## 使用方法
输入发行人基本信息，系统自动核查各条件满足情况，输出核查结果报告。

## 升级决策门

以下情况须移交专业律师：
- 涉及监管审批
- 存在法律风险
