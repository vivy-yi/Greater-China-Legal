# 资本市场 — 查询路径

## 一、证监会与交易所规则检索

| 数据源 | 路径 | 说明 |
|---|---|---|
| 证监会官网 | https://www.csrc.gov.cn | 法规栏目，分类检索 |
| 证监会令/公告 | https://www.csrc.gov.cn/csrc/c101864/index.html | 部门规章 |
| 交易所规则（沪市） | https://www.sse.com.cn/lawandrules/sserules/ | 科创板/主板规则 |
| 交易所规则（深市） | https://www.szse.cn/marketrules/ | 创业板/主板规则 |
| 北交所规则 | https://www.bse.cn/rule.html | 北交所业务规则 |

## 二、巨潮资讯（信息披露检索）

| 路径 | 用途 |
|---|---|
| http://www.cninfo.com.cn | 招股书/债券募集说明书/定期报告检索 |
| http://www.cninfo.com.cn/new/disclosure/stock | 上市公司公告检索 |

## 三、案例与问询意见

| 数据源 | 路径 | 用途 |
|---|---|---|
| 证监会行政许可批复 | https://www.csrc.gov.cn/csrc/c101889/list_date.shtml | 核准批文 |
| 交易所问询函 | 各交易所官网"问询函"栏目 | 审核关注点 |
| 否决案例库 | https://www.csrc.gov.cn | 审核未通过原因 |

## 四、行业数据

| 数据源 | 路径 | 用途 |
|---|---|---|
| 国民经济行业分类 | GB/T 4754-2017 | 行业归类判断 |
| 证监会行业分类指引 | https://www.csrc.gov.cn | 上市公司行业分类 |
| 交易所行业信息披露 | 各交易所自律规则 | 行业特定披露要求 |

## 五、检索顺序

1. 证监会官网 → 现行有效法规
2. 巨潮资讯 → 同业公司披露文件参考格式
3. 交易所规则 → 具体披露要求
4. 问询函库 → 审核关注重点
5. 国民经济分类 → 行业归类
