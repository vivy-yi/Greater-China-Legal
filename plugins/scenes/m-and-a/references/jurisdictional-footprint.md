## 1. 法域分层(5 大法域)

### 1.1 中国大陆（默认法域）

| 维度 | 内容 |
|------|------|
| 主管机关 | 国家市场监管总局（反垄断）/ 证监会 / 商务部 / 发改委 / 外管局 |
| 主要合规 | 反垄断法 / 外商投资法 / 境外投资管理办法 / 重大资产重组管理办法 |
| 域外法适用 | 仅跨境场景触发 |
| 升级路径 | 见 § 1.5 跨境 4 档 |

### 1.2 香港

| 维度 | 内容 |
|------|------|
| 主管机关 | 证券及期货事务监察委员会(SFC) / 收购及合并委员会 |
| 主要合规 | 公司收购、合并及股份回购守则 / 上市规则 |
| 与内地关系 | 内地与香港关于建立更紧密经贸关系的安排(CEPA) |
| 高度关注 | 反垄断豁免 / 受要约人保护 |

### 1.3 美国

| 维度 | 内容 |
|------|------|
| 主管机关 | DOJ / FTC（反垄断）/ CFIUS（外资审查）/ SEC |
| 主要合规 | HSR Act（反垄断申报）/ Section 721（外资审查）/ Williams Act（公开收购） |
| 高度关注 | CFIUS 强制申报 / 反垄断域外效力 / SEC 13D 披露 |
| 关键门槛 | HSR 申报门槛约 1.2 亿美金（按年调整） |

### 1.4 欧盟

| 维度 | 内容 |
|------|------|
| 主管机关 | 欧盟委员会（反垄断）/ 各成员国 FDI 审查机构 |
| 主要合规 | EU Merger Regulation（合并控制）/ EU FDI Screening Regulation 2019/452 |
| 高度关注 | 欧盟合并控制(EUMR)/ 外资审查协调 |
| 关键门槛 | 全球营收 > 25 亿欧元 + 至少 2 个成员国营收 > 2.5 亿欧元 |

### 1.5 英国 / 新加坡 / 开曼

| 国家 | 关键合规 |
|------|---------|
| **英国** | UK Enterprise Act 2002 / National Security and Investment Act 2021 / CMA 合并控制 |
| **新加坡** | MAS（金融）/ CCCS（反垄断）/ 外商投资审查（金融/敏感） |
| **开曼/百慕大/BVI** | 公司法 / 经济实质要求（避税地） / 上市规则 |

## 3. 跨境升级规则(Pattern 17)

### 3.1 升级触发(命中即 🔴 升级)

| 触发情形 | 升级目标 | 触发条件 |
|---------|---------|---------|
| **CFIUS 强制申报** | 美国并购律师 | 涉及美国业务 + 关键基础设施/技术/数据 |
| **EU FDI 审查** | 欧盟成员国律师 | 涉及欧盟关键基础设施/国防/敏感数据 |
| **UK NSI Act 审查** | 英国并购律师 | 涉及英国 17 个敏感行业 |
| **HSR Act 申报** | 美国反垄断律师 | 满足 HSR 门槛 |
| **EUMR 申报** | 欧盟反垄断律师 | 满足 EUMR 门槛 |
| **ODI 备案/核准** | 国家发改委 + 商务部律师 | 跨境投资 ≥ 3 亿美金或敏感行业 |
| **跨境换股外汇登记** | 外管律师 | 跨境换股 |
| **红筹/VIE 架构** | 跨境架构律师 | 涉及境外上市 |
| **返程投资 37 号文登记** | 外管律师 | 境内居民境外 SPV |
| **重大资产重组跨境** | 证监会沟通律师 | 跨境换股 + 重大资产重组 |

### 3.2 跨境 4 档升级路径

```
Level 1 (junior): 跨境交易 < 1 亿人民币
  → senior via 邮件

Level 2 (senior): 跨境交易 1-10 亿 + 一般跨境
  → gc via 邮件+会议

Level 3 (gc): 跨境交易 > 10 亿 / 反垄断触发 / 上市公司
  → ceo via 邮件+董事会

Level 4 (ceo): 重大 / 国资 / 跨境重大 / 红筹 / CFIUS / EU FDI
  → board via 会议+文件 + 跨境并购律师介入
```

### 3.3 跨境决策树

```
跨境交易 / 跨境架构 / 海外子公司
  ↓
[Step 1] 涉及金额？
  ├─ < 1 亿 → Level 1
  ├─ 1-10 亿 → Level 2
  └─ > 10 亿 → Level 3

[Step 2] 是否涉及特殊情形？
  ├─ 红筹/VIE → Level 3 + 跨境架构律师
  ├─ CFIUS / EU FDI → Level 4 + 跨境并购律师
  ├─ 上市公司 → Level 3 + 证监会沟通律师
  ├─ 国资 → Level 4 + 国资律师
  └─ 无 → 按 Step 1

[Step 3] 是否触发 6 大绝对禁止？
  ├─ 是 → 立即停止 + 律师 + 风险评估
  └─ 否 → 按 Step 1/2 升级
```

## 5. 域外法源标注规范

### 5.1 标注规则

**所有跨境并购分析必须标注 [域外]**：

```
例：
- 中国反垄断法申报 [GOV:反垄断法 26 条]
- 美国 HSR Act 申报门槛 [域外:HSR Act § 18]
- CFIUS 强制申报情形 [域外:31 CFR Part 800.401]
- EU FDI 审查协调 [域外:Regulation (EU) 2019/452]
- UK NSI Act 17 个敏感行业 [域外:UK NSI Act 2021 Schedule 1]
```

## 6. 法域升级自检 5 问

每次跨境任务前 agent 自问：

1. ☐ 是否识别所有相关法域（中国大陆 + 对方国家/地区）？
2. ☐ 是否查询了反垄断申报门槛（HSR / EUMR / 中国）？
3. ☐ 是否查询了外资审查要求（CFIUS / EU FDI / UK NSI）？
4. ☐ 是否标注 [域外] 法源？
5. ☐ 是否按金额 / 特殊情形触发对应升级路径？

**5 问中 ≥ 1 项未完成 → 不输出跨境结论。**
