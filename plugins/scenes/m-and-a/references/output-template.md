## 2. Risk calibration 3 段表（Pattern 18）

| 段 | 含义 | agent 动作 |
|---|------|-----------|
| **blocks** | 真正阻挡——绝对禁止 | 立即停止 + 告知 + 不绕过 |
| **work but ships** | 要修但不会挡 | 提示外部律师 + 给时间表 |
| **FYI** | 通知但不动作 | 记录不主动告知 |

### 2.1 blocks 段（m-and-a 专属）

**绝对禁止 + 必须立即停止**：

| 情形 | 法条 | 后果 |
|------|------|------|
| 经营者集中达到申报标准但未申报即实施 | 反垄断法 26 条 + 48 条 | 限期改正 + 50 万罚款 + 恢复原状 |
| 上市公司重大资产重组未停牌/未披露 | 重大资产重组管理办法 5 条 + 证券法 191 条 | 责令改正 + 100-1000 万罚款 |
| 国有股权转让未进场交易 | 企业国有资产法 54 条 + 国资委 3 号令 | 转让无效 + 责任人处分 |
| 外资准入负面清单禁止领域外商投资 | 外商投资法 28 条 + 负面清单 | 责令关闭 + 罚款 |
| 重大未披露关联方资金占用/担保 | 上市公司治理准则 + 证券法 | 责令改正 + 罚款 + 责任人处分 |
| 跨境换股/返程投资未完成外汇登记 | 外管局 37 号文 + 19 号文 | 限期整改 + 罚款 |

### 2.2 work but ships 段（m-and-a 专属）

**要修但不会挡**——提示外部律师 + 给时间表：

| 情形 | 处理 | 时间表 |
|------|------|-------|
| 商业条款争议（SPA 陈述保证） | 协商或仲裁 | 谈判期内 |
| 对赌协议触发后的争议解决 | 协商 + 仲裁准备 | 60 日内 |
| 整合期间的人事/文化冲突 | 整合管理办公室(IMO) | 100 天内 |
| 估值争议（资产基础法/收益法分歧） | 第三方评估复核 | 30 日内 |
| 反垄断申报材料不齐全 | 补充材料 | 30 日内 |
| 跨境审批延迟 | 沟通 + 备选方案 | 60 日内 |
| 对赌业绩未达成 | 协商补偿 + 启动回购 | 业绩期满 |

### 2.3 FYI 段（m-and-a 专属）

**通知但不动作**——记录不主动告知：

| 情形 | 记录位置 |
|------|---------|
| 行业并购趋势变化 | memo 备注 |
| 同业案例 | 内部案例库 |
| 学者评论 / 行业会议 | 知识库 |
| 自媒体分析 | 不记录 |

## 3. 三色编码应用规范

| 颜色 | 触发场景 | 输出格式 |
|------|---------|---------|
| 🔴 HIGH | 触发 11 必升 OR 6 绝对禁止 OR 交易金额巨大 | 红色高亮 + 升级路径 + 截止日 |
| 🟡 MEDIUM | 触发 work but ships 段 OR 一般风险 | 黄色提示 + 建议 + 时间表 |
| 🟢 LOW | FYI 段 OR 常规交易 | 绿色标记 + 记录位置 |

**关键**：所有 🔴 必须在 Reviewer note 第 4 行（升级建议）显式给出"升给谁 + 触发条件"。

## 4. 输出自检 8 问

每次输出前 agent 自问：

1. ☐ Reviewer note 5 行是否齐全？
2. ☐ 6 大绝对禁止 checklist 是否全部勾选？
3. ☐ 11 大必升 checklist 是否全部勾选？
4. ☐ Risk calibration 3 段判断是否完成？
5. ☐ 升级路径是否明确（升给谁 + 触发条件）？
6. ☐ 数据源是否 ≥ 2 源（关键结论）？
7. ☐ Decision tree 5 选项是否给出？
8. ☐ 发出前核查清单是否全部完成？

**8 问中 ≥ 1 项未完成 → 不输出最终结论。**
