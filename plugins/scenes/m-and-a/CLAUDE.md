### § A5 Outputs

**Reviewer note 5 行(M&A 特化):**
1. 交易基本信息:[类型 / 行业 / 金额 / 阶段 / 主体]
2. 主要风险:[法律 / 财务 / 商业 / 监管]
3. 关键节点:[董事会 / 股东会 / 监管审批 / 反垄断申报]

## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** `deal-structurer`(交易结构是 M&A 起点)

**关键节点:**

```
Step 1: deal-structurer    → 交易结构设计
Step 2: due-diligence       → 法律 + 财务 + 商业尽调
Step 3: disclosure          → 信息披露
Step 4: approval            → 监管审批
Step 5: closing             → 交割
```

**特别考虑:**

```
✅ 反垄断申报(经营者集中)→ cross-border-ma / antitrust
✅ 境外投资(ODI)→ odi-filing
✅ 上市公司重大资产重组 → disclosure + approval
✅ 退市 / 私有化 → going-private-*
```

### § B14 Risk calibration

**段 1 识别:** 估值风险 / 法律风险 / 监管风险 / 整合风险 / 业绩风险
**段 2 量化:** 概率 × 影响 + 缓释
**段 3 响应:** 接受 / 缓释(对赌)/ 转移(VIE/担保)/ 规避(退出)

### § B15 7 条设计哲学(M&A 特化)

1. **结构 > 条款** — 结构错,条款再细也救不了
2. **尽调 > 估值** — 尽调发现隐藏问题
3. **业绩对赌** — 估值与对赌平衡
4. **反垄断优先** — 全球并购必查
5. **VIE 风险** — 效力存争议
6. **境外换汇** — 外汇合规
7. **整合** — 并购成功关键
