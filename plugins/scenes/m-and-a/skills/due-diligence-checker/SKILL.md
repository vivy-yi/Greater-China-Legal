# 尽职调查技能 (Due Diligence Checker)

## 技能定位

本技能用于对目标公司进行系统性法律尽职调查，识别交易风险，出具调查结论。

## 核心能力

### 1. 调查范围确定
- 根据交易类型确定调查深度（完整尽调/简明尽调/专项尽调）
- 识别高风险领域并重点关注
- 协调外部律师、会计师、行业顾问

### 2. 公司主体调查
```
重点审查事项：
□ 设立合法性：注册地址、注册资本、出资方式、存续状态
□ 股权结构：股东名单、出资比例、代持情况、质押冻结
□ 治理结构：董事会构成、议事规则、内部控制
□ 重大变更：历次增减资、股权转让、合并分立
```

### 3. 资产与权益调查
```
不动产：
□ 土地使用权：来源、期限、用途、出让金缴纳
□ 房屋建筑：权属证明、他项权利、违建情况

知识产权：
□ 专利：有效状态、缴费情况、许可状态、归属
□ 商标：注册类别、续展情况、许可备案
□ 著作权：登记情况、职务作品认定

应收账款：
□ 账龄分析、债务人情况、催收记录
```

### 4. 重大合同审查
```
合同分类审查：
□ 采购合同：期限、续约条款、违约责任
□ 销售合同：定价机制、交付条件、退换货
□ 贷款合同：利率、担保、限制性条款
□ 租赁合同：租金水平、租期、优先权

关键条款识别：
□ 控制权变更条款（change of control）
□ 优先购买权条款
□ 竞业禁止条款
□ 保密条款
```

### 5. 劳动合规调查
- 劳动合同签订情况
- 社保公积金缴纳合规性
- 特殊员工处理（工伤、三期女员工）
- 劳务派遣与外包合规性

### 6. 诉讼与合规
```
诉讼风险评估：
□ 已决诉讼：执行情况、潜在影响
□ 未决诉讼：案件性质、金额、败诉风险
□ 潜在诉讼：行业惯例、争议苗头

行政处罚：
□ 工商、税务、环保、安全生产等处罚记录
□ 整改完成情况
```

### 7. 或有负债识别
```
识别方法：
□ 财务指标异常波动
□ 关联交易价格偏离市场
□ 担保圈风险传导
□ 环保复垦义务

量化评估：
□ 预计负债计提
□ 敞口金额评估
□ 风险概率测算
```

## 输出成果

### 尽职调查报告结构
```
第一章 尽职调查概述
  1.1 调查背景与范围
  1.2 调查方法与限制
  1.3 主要发现摘要

第二章 公司主体与股权
  2.1 公司基本情况
  2.2 股权结构分析
  2.3 重大变更记录

第三章 业务与经营
  3.1 主营业务分析
  3.2 主要合同审查
  3.3 业务合规性

第四章 资产与负债
  4.1 主要资产情况
  4.2 知识产权情况
  4.3 债务与担保

第五章 劳动人事
  5.1 用工情况
  5.2 社保合规
  5.3 劳动争议

第六章 诉讼与合规
  6.1 已决诉讼
  6.2 未决诉讼
  6.3 行政处罚

第七章 风险汇总与建议
  7.1 重大风险提示
  7.2 风险缓释建议
  7.3 交易建议
```

## 风险评级标准

| 等级 | 定义 | 影响 | 建议 |
|-----|------|-----|-----|
| **高风险** | 重大法律障碍或导致交易失败 | 实质性影响 | 交易前必须解决或设置有效保障 |
| **中风险** | 可能影响交易价值或增加成本 | 可量化影响 | 通过陈述保证、赔偿条款或价格调整解决 |
| **低风险** | 不影响交易决策 | 微小或可忽略 | 披露并持续关注 |

## 调查配合要点

1. **数据室管理**：建立索引、跟踪更新、记录缺失
2. **访谈记录**：保存访谈笔录，作为免责依据
3. **限制性说明**：如实披露调查限制，不得臆测
4. **交叉验证**：文件审查与访谈、外部调查相结合
5. **时效管理**：注意信息披露的时效性要求

## 升级决策门

**以下情形须向律师升级：**
- 涉及国有资产转让特殊审批
- 外商投资准入存在争议
- 反垄断审查门槛临界
- 交易结构涉及跨境架构
- 存在重大潜在负债未在尽调中发现
- 交易对方存在资信风险
- 涉及特殊监管行业（金融/医药/教育）

## 本 Skill 不涵盖

- 财务审计与估值建模
- 目标公司日常经营管理
- 税务申报与税务筹划
- 劳动仲裁代理与诉讼
- 涉及境外法律的跨境问题
