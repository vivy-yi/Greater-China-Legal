# 技能文件三：外汇管理操作实务技能

## 技能概述

本技能用于指导律师或企业人员理解和操作跨境并购项目中的外汇管理实务，包括资本项目外汇登记、资金跨境汇划、返程投资外汇登记等核心操作。

---

## 一、外汇管理基本框架

### 1.1 外汇管理两大类别

| 类别 | 内容 | 管理原则 |
|-----|-----|---------|
| 经常项目 | 货物贸易、服务贸易、收益分配 | 意愿结汇+真实性审核 |
| 资本项目 | 直接投资、证券投资、外债、跨境担保 | 审批/登记管理 |

**核心原则**：资本项目外汇流入流出须经外汇局核准或登记

### 1.2 管理机关层级

```
国家外汇管理局
     │
 ├── 省级分局
 │      │
 │    ──地市中心支局
 │          │
 │        ──县支局
 │
 └── 授权银行（基层操作主体）
```

## 二、ODI外汇登记操作

### 2.1 登记前置条件

完成以下审批后，方可办理ODI外汇登记：

| 审批事项 | 审批机关 | 文件类型 |
|---------|---------|---------|
| ODI核准/备案 | 商务部/省级商务厅 | ODI证书 |
| 项目核准/备案 | 发改委/省级发改委 | 核准批复/备案证明 |

### 2.2 银行端操作流程

**第一步：选择银行**

须选择具备以下资格的银行：
- 具备资本项目外汇业务经营资格
- 具备ODI外汇业务操作经验
- 当地外汇局已对该银行进行业务授权

**第二步：提交材料**

| 材料名称 | 要求 |
|---------|-----|
| 资本项目外汇业务申请表 | 银行提供格式，按实填写 |
| ODI证书 | 原件+复印件 |
| 发改委核准/备案文件 | 原件+复印件 |
| 投资主体营业执照 | 加盖公章的复印件 |
| 经审计的财务报表 | 复印件 |
| 资金来源说明 | 纸质说明+银行要求模板 |
| 资金使用计划 | 含资金投向、时间安排 |

**第三步：银行端系统录入**

银行操作人员：
1. 登录资本项目外汇业务系统
2. 录入企业基础信息
3. 录入ODI项目信息
4. 生成资本项目信息表
5. 打印业务登记凭证

**第四步：获取登记凭证**

- 《资本项目外汇业务登记凭证》
- 后续资金汇出须凭此凭证

### 2.3 资金汇出操作

**购汇前提**：
- 已完成外汇登记
- 资金来源证明齐备
- 资金用途与ODI项目一致

**汇出流程**：
```
1. 提交汇款申请（电汇/票汇）
2. 附上外汇登记凭证
3. 银行审核真实性
4. 购汇/自有外汇支出
5. 跨境汇出
6. 收取汇出凭证
```

**汇出凭证留存**：
- 水单/汇款回执
- 换汇凭证（如涉及购汇）
- 完整资金流向记录

## 三、37号文登记操作（SPV返程投资）

### 3.1 适用对象

| 主体类型 | 说明 |
|---------|-----|
| 境内自然人 | 持有中国居民身份证的中国公民 |
| 境内法人 | 在中国境内注册的企业 |

### 3.2 登记情形

**必须登记**：
- 境内自然人在境外设立SPV
- 境内自然人通过SPV返程投资
- 境内法人在境外设立SPV

**典型场景**：
- 创始人搭建红筹架构
- 返程收购境内企业
- 境外融资后返程投资

### 3.3 登记流程

**直接到外汇局办理（2015年后简化为银行端办理）**：

| 阶段 | 操作 | 主体 |
|-----|-----|-----|
| 1 | 37号文申请 | 申请人 |
| 2 | 审核+登记 | 外汇局/银行 |
| 3 | 出具备案表 | 外汇局/银行 |

**银行端办理（2015年后推广）**：

1. 境内自然人向银行提交申请
2. 银行审核后在外汇局系统代为登记
3. 外汇局事后核查

### 3.4 补登记操作

**补登记适用情形**：
- 历史上已设立SPV但未办理37号文登记
- 通过SPV已完成返程投资但未登记
- 虚假登记或登记信息不准确

**补登记流程**：
```
1. 申请人向外商投资企业所在地外汇局申请
2. 提交书面说明（含未登记原因、时间、过程）
3. 外汇局审核合理性
4. 审核通过后补办登记
5. 出具备案表
```

**补登记所需材料**：
- 书面申请（含未登记原因说明）
- 境外SPV全套注册文件
- 境内运营公司相关文件
- 资金汇出入凭证
- 近三年财务报表
- 个人完税证明（如为自然人）

## 四、资本项目外汇收入管理

### 4.1 境外投资收入汇回

**合规路径**：
```
境外子公司 → 利润分配 → 汇入境外账户
     ↓
 银行端进行国际收支申报
     ↓
 境内主体收到外汇（保留或结汇）
```

**银行端操作**：
1. 收到境外汇入款项
2. 进行国际收支申报（网上系统）
3. 入账或结汇

### 4.2 股权转让收入

**ODI项下股权转让**：
- 须向外汇局报告
- 所得外汇收入可保留或汇回
- 须提交交易合同、完税证明

## 五、经常项下外汇操作

### 5.1 货物贸易外汇收支

**企业名录分类**：
| 分类 | 要求 |
|-----|-----|
| A类企业 | 便利化收支，凭交易单证直接办理 |
| B类企业 | 须逐笔审核，真实性为主 |
| C类企业 | 严管，逐笔审查 |

**操作要点**：
- 贸易外汇收支须进行国际收支申报
- 货物贸易信贷须在外汇局系统登记
- 超期限或超额须向外汇局报告

### 5.2 服务贸易外汇收支

**单证要求**：
- 合同或发票
- 税务备案表（付汇超过5万美元须备案）
- 其他证明材料

**注意**：
- 利润汇出属于经常项目-收益项目
- 须提交董事会决议、审计报告、完税证明

## 六、违规行为与处罚

### 6.1 常见违规类型

| 违规类型 | 行为描述 | 典型案例 |
|---------|---------|---------|
| 逃汇 | 虚假合同、构造贸易背景 | 虚构进口合同付汇 |
| 套汇 | 伪造单证套取外汇 | 伪造进口发票购汇 |
| 擅自对外借款 | 未登记外债 | 企业直接借外债 |
| 非法汇入外汇 | 未申报的国际收支 | 境外资金未申报入账 |

### 6.2 处罚标准

| 违规类型 | 处罚标准 |
|---------|---------|
| 逃汇 | 责令限期调回外汇，处逃汇金额30%以下罚款；情节严重，30%-等值以下罚款 |
| 套汇 | 没收违法所得，处套汇金额30%以下罚款；情节严重，30%-等值以下罚款 |
| 违规汇兑 | 没收违法所得，处违法金额30%以下罚款 |
| 违规出借外汇 | 警告，处30万元以下罚款 |

### 6.3 合规建议

1. **真实性优先**：所有外汇业务须有真实贸易/投资背景
2. **材料留痕**：所有单证、合同、凭证完整保存5年
3. **主动申报**：国际收支申报、存量权益登记按时完成
4. **定期自查**：企业每年至少一次外汇业务自查

## 七、当前政策要点（2024年参考）

### 7.1 便利化政策

- **意愿结汇**：资本项目外汇收入可意愿结汇
- **简化流程**：直接投资外汇登记下放银行
- **取消事前审批**：部分类别直接投资取消核准改为备案

### 7.2 监管趋势

- **强化真实性审核**：银行加大单证审核力度
- **国际收支申报**：全口径统计，漏报将处罚
- **反洗钱**：大额外汇交易须报送大额交易报告

### 7.3 特别提示

⚠️ 投资移民、 海外房产购置 等资本项下支出，须经外汇局审批，当前政策趋严。

⚠️ 分拆购汇（将大额资金拆分为多笔5万美元以下规避审批）属违规行为。
