# 要约收购技能1：要约收购方案设计与申报

## 技能描述
本技能用于协助用户设计要约收购方案并完成监管申报程序。

## 核心功能
1. 收购方案设计
2. 申报材料准备
3. 合规性审查
4. 进度跟踪

## 方案设计要素

### 确定收购要素
1. **收购比例**
   - 部分收购：满足30%触发线即可
   - 全部收购：私有化目的

2. **要约价格**
   - 定价依据
   - 溢价幅度
   - 价格稳定性

3. **收购期限**
   - 法定范围：30-60日
   - 建议期限：45日

4. **支付安排**
   - 现金/股份/混合
   - 资金证明

## 申报材料清单

### 必备文件
- [ ] 要约收购报告书
- [ ] 财务顾问核查意见
- [ ] 律师法律意见书
- [ ] 收购人身份证明
- [ ] 资金来源说明
- [ ] 收购资金承诺函

### 附件材料
- [ ] 收购人营业执照
- [ ] 收购人最近三年财务报告
- [ ] 一致行动人协议（如有）
- [ ] 资管计划合同（如涉及）

## 申报流程
```
1. 准备阶段（1-2周）
   ↓
2. 材料制作（2-3周）
   ↓
3. 内部审核（1周）
   ↓
4. 向证监会申报
   ↓
5. 审核阶段（约30工作日）
   ↓
6. 补充材料（如需）
   ↓
7. 获得批准
```

## 常见问题
Q：申报材料补正怎么办？
A：按照证监会反馈意见及时补正，通常3-5个工作日完成。

Q：申报期间能否买卖股票？
A：申报期间不得买卖被收购公司股票。

## 升级决策门

- 存在法律风险
