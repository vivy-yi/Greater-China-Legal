## 一、信托类型识别与结构选型

### 1.1 境内主要信托类型
| 信托类型 | 法律依据 | 监管机构 | 委托人限制 | 受托人资质 |
|---------|---------|----------|-----------|-----------|
| 营业信托 | 《信托法》第3条 | 银保监会 | 法人或自然人均可 | 持牌信托公司 |
| 民事信托 | 《信托法》第3条 | 无特定监管 | 自然人/法人均可 | 个人或法人均可 |
| 遗嘱信托 | 《信托法》第13条 | 参照民事信托 | 通过遗嘱设立 | 个人或法人均可 |
| 慈善信托 | 《慈善法》第五章 | 民政部门 | 法人或自然人均可 | 慈善组织或信托公司 |
| 家族信托 | 《信托业务分类通知》 | 银保监会 | 法人或自然人均可 | 持牌信托公司 |

### 1.2 信托设立目的识别
**核心目的分类：**
1. **资产保护型**：隔离委托人债务风险、婚姻财产分割风险
2. **传承规划型**：多代际传承、避免遗产继承程序
3. **税务优化型**：合规前提下的税务递延或减省
4. **慈善目的型**：社会公益、税收优惠
5. **保值增值型**：专业资管、分散投资

### 1.3 信托利益结构设计
```
基本结构要素：
┌─────────────────────────────────────┐
│ 委托人 ──财产转移──▶ 受托人        │
│                        │            │
│                   信托财产          │
│                        │            │
│              ┌───────┴───────┐      │
│              ▼               ▼      │
│        受益人范围       信托目的    │
│        受益条件         存续期限    │
└─────────────────────────────────────┘
```

**中国法下信托财产独立性保障：**
- 信托财产与受托人固有财产分离（《信托法》第16条）
- 信托财产不得强制执行（《信托法》第17条）
- 委托人死亡不影响信托效力（《信托法》第15条）

## 二、信托架构合规性审查

### 2.1 信托目的合法性审查
**禁止设立情形（《信托法》第6条）：**
1. 信托目的违反法律、行政法规强制性规定
2. 信托目的损害社会公共利益
3. 信托财产来源非法

**附停止条件的信托：**
- 条件不得违反公序良俗
- 条件的成就与否须有可信赖的判定机制
- 警惕"永续信托"结构（与我国信托期限制度冲突）

### 2.2 委托人权力保留限制
| 权力类型 | 可保留 | 需谨慎 | 禁止保留 |
|---------|-------|-------|---------|
| 变更受益人 | ✓（保留变更权） | — | — |
| 调整信托财产管理方式 | ✓（知情权） | ✓（需受托人同意） | — |
| 解除信托 | — | ✓（特定情形） | — |
| 干预信托事务执行 | — | — | ✗ | 
| 直接支配信托财产 | — | — | ✗ |

**"假家族信托"识别标准：**
- 委托人保留投资决策权
- 委托人保留随时终止权
- 受益人仅在特定条件下享有受益权（类似债务清偿）
上述情形可能被认定为"通道业务"或实质为借贷关系

### 2.3 信托登记与公示
| 财产类型 | 登记要求 | 登记机构 |
|---------|---------|---------|
| 不动产 | 必须登记 | 不动产登记机构 |
| 股权 | 建议登记 | 市场监督管理部门 |
| 知识产权 | 建议登记 | 知识产权登记部门 |
| 资金 | 无需登记 | — |

**未登记财产的效力：**
- 仅设立内部效力的信托（不得对抗善意第三人）
- 涉及不动产物权变动的，须办理登记否则不生效

### 2.4 营业信托受托人资质
**持牌要求：**
- 营业信托受托人须为银保监会批准设立的信托公司
- 家族信托单笔设立门槛：人民币100万元起（参照2023年监管通知）
- 信托公司资本充足率要求参照《信托公司管理办法》

## 三、信托架构风险矩阵与优化建议

### 3.1 主要风险类型
| 风险类型 | 触发情形 | 法律后果 | 建议措施 |
|---------|---------|---------|---------|
| 信托目的违法 | 设立时意图规避债务 | 信托无效（《信托法》第6条） | 放弃设立 |
| 委托人债权人攻击 | 设立后1年内债权人主张 | 可撤销（《信托法》第12条） | 设定期限缓冲 |
| 婚姻财产分割风险 | 委托人或受益人离婚 | 信托财产可能被认定为共同财产 | 变更受益人结构 |
| 受托人失职 | 受托人违反信托目的管理 | 承担恢复信托财产责任 | 约定监督机制 |
| 跨境合规风险 | 信托财产或受益人涉及境外 | FATCA/CRS申报义务 | 咨询跨境税务专家 |

### 3.2 结构优化方案

**方案A：资产保护型 → 债务隔离优化**
```
问题：委托人担忧未来债务风险
原结构：委托人100%控制权保留
优化结构：
- 引入保护人（自然人或法人）机制
- 约定委托人不得单方面终止
- 设定受益人取得受益权的确定性条件
```

**方案B：传承规划型 → 多代际传承设计**
```
问题：避免子女挥霍、保障孙辈教育
原结构：直接一次性传承
优化结构：
- 设立"激励机制条款"（学业、就业奖励）
- 设置"防火墙条款"（离婚、债务隔离）
- 配置"调整权保留条款"（委托人或保护人可调整细节）
```

**方案C：慈善型 → 税收优惠最大化**
```
合规前提：设立慈善信托
税收优惠路径：
1. 委托人捐赠支出可抵扣企业所得税（≤年度利润总额12%）
2. 个人捐赠可抵扣个人所得税（≤应纳税所得额30%）
3. 信托运营环节税收优惠（参照《慈善法》第82条）
```

### 3.3 信托监察人制度
| 角色 | 职责 | 必要性 |
|-----|------|-------|
| 信托监察人 | 维护受益人利益，必要时向法院起诉受托人 | 强烈建议设置 |

**设置建议：**
- 自然人担任：须具备完全民事行为能力及足够专业性
- 法人担任：建议聘请专业机构（如律师事务所）
- 回避情形：受托人关联方不得担任监察人

## 升级决策门

### 触发条件（满足任一即升级）
1. **跨境因素**：信托财产位于境外、受益人含外籍人士或境外税收居民
2. **监管牌照问题**：涉及信托公司牌照资质争议、信托业务分类边界争议
3. **受托人责任追究**：受托人因管理信托财产致损，受益人拟提起诉讼
4. **信托效力争议**：信托被认定无效或可撤销的诉讼
5. **特殊目的信托**：员工持股信托ESOP、资管计划嵌套、家族慈善联动等复杂结构
6. **委托人债务危机**：债权人已启动司法程序或债务重组
7. **涉及上市公司**：信托持有上市公司股份（涉及信息披露、锁定期要求）

### 升级路径
```
升级决策门触发 → 停止自动生成架构建议 →
输出"本案件涉及[具体情形]，需信托法律专业律师介入"通知 →
提供专业律师查询路径（参见 references/查询路径.md）
```

### 升级后本skill责任终止情形
- 进入司法程序（诉讼/仲裁）后，本skill不提供程序建议
- 信托公司作为当事人的行政监管程序，不属于本skill范围
- 涉及刑事追缴的信托财产冻结/处置程序，不属于本skill范围

## 本Skill不涵盖

### 明确排除事项
1. **信托公司牌照申请**：信托公司设立、变更的行政审批代理
2. **信托产品备案**：单一信托、集合信托产品的金融监管部门备案
3. **投资决策建议**：信托财产的投资方向、投资比例等投资建议
4. **信托税务申报代理**：信托运营环节的增值税、所得税申报代理
5. **跨境信托设立**：离岸信托（BVI、开曼、新加坡等）设立及合规
6. **FATCA/CRS合规咨询**：涉及美国《海外账户税收合规法案》及共同申报准则的跨境税务咨询
7. **信托资金来源合法性论证**：反洗钱合规审查
8. **信托财产评估**：信托财产的市场价值评估
9. **遗嘱执行**：遗嘱执行人的具体职责（参见 probate-procedure-guide）
10. **遗产税规划**：遗产税申报与缴纳（参见 estate-tax-planner）
