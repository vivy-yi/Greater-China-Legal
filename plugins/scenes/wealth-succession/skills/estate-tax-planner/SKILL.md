## 一、遗产与赠与税务全景图

### 1.1 中国现行税制下的财富传承税务
| 税种 | 是否开征 | 适用情形 | 税率/规则 |
|-----|---------|---------|----------|
| 遗产税 | 未全面开征 | 试点城市（深圳等）曾有草案讨论 | 暂未正式立法 |
| 赠与税 | 未单独开征 | 通过个人所得税"偶然所得"或其他方式调节 | 暂未独立立法 |
| 个人所得税 | 征 | 受赠财产、继承财产变现 | 20%税率（受赠所得）/偶然所得20% |
| 企业所得税 | 征 | 以法人为主体传承资产 | 25%（一般企业）/15%（高新技术企业） |
| 土地增值税 | 征 | 不动产传承涉及土地增值 | 超率累进税率30%-60% |

**注：截至2024年，中国尚未全面开征遗产税和赠与税，但相关研究讨论持续进行。**

### 1.2 继承/受赠财产变现税务
| 取得方式 | 变现环节税负 | 免税额度 | 备注 |
|---------|-------------|---------|-----|
| 法定继承 | 无个人所得税（参照免税规定） | 遗产价值全额 | 需继承人到税务机关备案 |
| 遗嘱继承 | 取得时暂免，处置时征 | 参照相关规定 | 各地执行标准不一 |
| 赠与（直系亲属） | 部分地区有免税政策 | 参照当地规定 | 北京、上海等地有差异 |
| 遗赠（非直系） | 20%偶然所得 | 单次≤1万元部分可申请减免 | 需受赠人自行申报 |

### 1.3 资产类型与潜在税负
```
税负评估矩阵：
┌──────────────────────────────────────────┐
│ 资产类型      │ 取得环节 │ 持有环节 │ 转让环节 │
├──────────────────────────────────────────┤
│ 现金/存款     │ 无       │ 无       │ 无       │
│ 房产（住宅）  │ 暂免/低  │ 房产税   │ 增值税+个税 │
│ 房产（非住宅）│ 土地增值税│ 房产税   │ 土地增值税+个税 │
│ 股权          │ 暂免     │ 企业所得税│ 企业所得税+个税 │
│ 股票          │ 暂免     │ 无       │ 印花税+个税   │
│ 艺术品/珠宝  │ 暂免     │ 无       │ 偶然所得税20% │
│ 车辆          │ 暂免     │ 车船税   │ 交易税       │
└──────────────────────────────────────────┘
```

## 二、税务规划路径设计

### 2.1 合规节税核心策略

**策略一：生前赠与 vs 身后继承**
```
比较框架：
┌────────────────────────────────────────────────────┐
│                    生前赠与                        │
├────────────────────────────────────────────────────┤
│ 优点：避免高额遗嘱执行费用；部分税负可递延         │
│ 缺点：赠与人失去财产控制权；赠与税立法趋势不明    │
│ 风险：赠与人债务风险无法隔离                       │
└────────────────────────────────────────────────────┘
┌────────────────────────────────────────────────────┐
│                    身后继承                        │
├────────────────────────────────────────────────────┤
│ 优点：保留控制权至去世；遗产税（若开征）可享有扣除 │
│ 缺点：遗嘱认证程序费用；继承人变现时税负较高       │
│ 风险：继承人间争议可能导致遗产被冻结               │
└────────────────────────────────────────────────────┘
```

**策略二：保险工具的税务优化**
```
税收优惠应用：
1. 人寿保险赔付款：个人所得税免税（《个人所得税法》第4条）
2. 年金保险：领取阶段税负取决于具体产品条款
3. 税优健康险：符合规定的保险赔付款免税

规划要点：
- 投保人设置为被保险人本人或配偶，指定子女为受益人
- 保险赔付款直接支付给受益人，不进入遗产清算程序
- 注意保险合同签订日期早于债务形成日期（债务隔离）
```

**策略三：信托架构节税**
```
税务考量：
- 设立信托时：财产转移是否视同销售（增值税、企业所得税）
- 信托存续期间：信托财产收益的纳税主体认定
- 信托分配时：受益人取得信托利益是否需纳税

合规要点：
- 选择营业信托（持牌信托公司）有利于税务合规性认定
- 慈善信托可享受部分税收优惠（须取得民政部门备案）
```

### 2.2 常见资产传承税务规划
| 资产类型 | 规划路径 | 节税效果 | 风险提示 |
|---------|---------|---------|---------|
| 不动产 | 生前以合理价格出售给子女 | 减少遗产规模 | 须保留合理居所 |
| 股权 | 生前赠与或以增资方式稀释 | 递延资本利得税 | 须注意公司法优先购买权规定 |
| 有价证券 | 开立子女账户逐步转移 | 低税率或暂免 | 须注意短线交易规定 |
| 艺术品 | 捐赠给美术馆或设立基金会 | 税前抵扣 | 须取得评估报告 |

### 2.3 跨境财富传承税务（仅限框架说明）
**已移民家庭的税务考量：**
- 取得境外身份后，原中国税收居民身份的纳税义务（通常有过渡期）
- 境外不动产传承可能涉及当地遗产税/赠与税（如美国遗产税、英国遗嘱认证费等）
- 离岸信托的FATCA/CRS合规申报义务

**注：跨境税务规划属于本skill升级范围，须咨询国际税务专业律师。**

## 三、税务合规与申报指引

### 3.1 继承/赠与税务申报流程
```
申报时间轴：
T+0：被继承人死亡/赠与事实发生
  │
T+30日内：继承人/受赠人向主管税务机关备案
  │
T+90日内：提交财产价值评估申请（如需）
  │
T+6个月内：完成税务清算，缴纳应纳税款
  │
T+12个月内：办理财产过户/变更登记
```

### 3.2 申报材料清单
| 材料类型 | 继承人申报 | 受赠人申报 | 备注 |
|---------|-----------|-----------|-----|
| 身份证明 | ✓ | ✓ | 身份证/护照 |
| 亲属关系证明 | ✓ | — | 户口簿/公证文书 |
| 死亡证明 | ✓ | — | 医疗机构或公安机关出具 |
| 财产权利凭证 | ✓ | ✓ | 房产证/股权证明/存单等 |
| 财产评估报告 | ✓（建议） | ✓（建议） | 不动产、股权等须评估 |
| 遗嘱（若有） | ✓ | ✓ | 遗嘱公证文书 |
| 赠与合同（若有） | — | ✓ | 赠与协议公证文书 |

### 3.3 税务风险点识别
| 风险点 | 触发情形 | 法律后果 | 应对措施 |
|--------|---------|---------|---------|
| 隐匿财产 | 未申报全部遗产 | 补缴税款+滞纳金+罚款 | 全面梳理遗产清单 |
| 价值低估 | 评估价值显著低于市场价 | 税务机关重新核定+补税 | 聘请有资质评估机构 |
| 申报延迟 | 超规定期限未申报 | 按日加收万分之五滞纳金 | 及时申报或申请延期 |
| 跨境资产未申报 | 境外财产未纳入申报范围 | 违反外汇管理规定+税务风险 | 咨询跨境税务专家 |

## 升级决策门

### 触发条件（满足任一即升级）
1. **跨境资产传承**：涉及境外不动产、境外金融账户、境外公司股权
2. **企业股权传承**：传承标的为有限责任公司股份或拟上市公司股份
3. **高净值人士**：个人资产规模预计超过人民币1亿元
4. **移民家庭**：家庭成员已取得或正在申请境外身份（绿卡/护照）
5. **税务稽查**：税务机关已启动稽查程序或发出税务处理决定书
6. **遗产税试点**：涉及已开征遗产税试点地区的传承安排
7. **慈善捐赠安排**：涉及大额公益捐赠及税前抵扣争议

### 升级路径
```
升级决策门触发 → 停止自动生成税务规划建议 →
输出"本案件涉及[具体情形]，税务处理较为复杂，建议委托税务律师全程代理"通知 →
提供税务律师及注册会计师查询路径（参见 references/查询路径.md）
```

### 升级后本skill责任终止情形
- 进入税务稽查程序后，本skill不提供应对策略建议
- 税务行政复议/行政诉讼程序，不属于本skill范围
- 涉及逃税、偷税的刑事责任，不属于本skill范围

## 本Skill不涵盖

### 明确排除事项
1. **税务申报代理**：代替纳税人向税务机关提交任何税种的申报表
2. **税务行政复议代理**：代理纳税人进行税务行政复议
3. **刑事税务辩护**：涉及逃税罪、虚开增值税发票罪等刑事辩护
4. **资产评估**：信托财产、不动产、股权等的市场价值评估（须聘请有资质评估机构）
5. **跨境税务合规**：FATCA、CRS合规及外国税务申报
6. **信托设立税务规划**：信托架构设计与税务规划联动（参见 trust-structure-advisor）
7. **遗嘱执行税务**：遗嘱执行人职责中的具体税务操作（参见 probate-procedure-guide）
8. **保险合同税务咨询**：保险产品的详细税务分析（须咨询保险经纪或税务顾问）
9. **移民税务规划**：移民前的资产剥离方案及移民后的全球税务申报
10. **遗产税立法咨询**：对未立法事项提供预测性意见
