## 一、遗嘱形式要件审查

### 1.1 遗嘱类型识别
| 遗嘱类型 | 法定形式要件 | 见证人要求 | 证明力等级 |
|---------|-------------|-----------|-----------|
| 公证遗嘱 | 遗嘱人向公证机构申请，亲自办理或委托办理 | 公证员主持+记录 | 最高 |
| 自书遗嘱 | 全文亲笔书写、签名、注明年月日 | 无需见证人 | 中高 |
| 代书遗嘱 | 代书人执笔，至少两名见证人在场，遗嘱人/代书人/见证人签名 | 2名以上见证人 | 中 |
| 录音录像遗嘱 | 遗嘱人姓名+图像+声音，至少两名见证人在场 | 见证人参与录制 | 中低 |
| 口述遗嘱 | 危急情况，口述内容完整记录，注明年月日 | 见证人参与记录 | 低 |

### 1.2 形式瑕疵清单
- [ ] 签名缺失或签名字迹不一致
- [ ] 日期标注不完整（缺年/月/日任一）
- [ ] 涂改处未签名确认或未注明修改字数
- [ ] 自书遗嘱中掺入打印件且无亲笔签名
- [ ] 代书遗嘱中仅有一名见证人签名
- [ ] 见证人与遗嘱人或受遗赠人存在利害关系
- [ ] 公证遗嘱未经公证机构保存原件

### 1.3 遗嘱能力评估
**必备要件（全部满足方可继续）：**
1. 立遗嘱时为完全民事行为能力人
2. 遗嘱内容系遗嘱人真实意思表示
3. 无欺诈、胁迫情形
4. 遗嘱人未丧失遗嘱能力（参照《民法典》第1143条）

**风险信号：**
- 遗嘱人曾被诊断为限制民事行为能力或无民事行为能力
- 遗嘱订立前后存在重大医疗事件
- 存在疑似限制人身自由的客观情形

## 二、遗嘱实质条款合规性校验

### 2.1 必留份（特留份）校验
依据《民法典》第1141条：遗嘱应当为缺乏劳动能力又没有生活来源的继承人保留必要的遗产份额。

**算法逻辑：**
```
步骤1：识别第一顺序继承人（配偶、子女、父母）
步骤2：核查是否存在缺乏劳动能力且无生活来源的继承人
步骤3：计算必留份基数 = 遗产总额 ÷ 继承人数量 × 法定比例
步骤4：检验遗嘱中该继承人实得份额 ≥ 必留份
```

**合规阈值：**
- 达标：实得份额 ≥ 必留份基数
- 警告：实得份额 < 必留份基数，但遗嘱人明确表示不知晓该规定
- 违规：实得份额 < 必留份基数，且继承人主张权利

### 2.2 遗嘱内容合法性审查
**禁止性条款（违反则部分或全部无效）：**
1. 遗嘱人无权处分的他人财产（《民法典》第1142条，遗产分割时应先析产）
2. 剥夺缺乏劳动能力又无生活来源的继承人必要份额
3. 遗嘱附有违法条件或不可能条件（如"不许离婚"等）
4. 违反公序良俗的遗赠安排

**有效遗产范围识别：**
- 夫妻共同财产中属于遗嘱人的1/2份额
- 遗嘱人个人合法财产的全部
- 共有财产中遗嘱人的应有份额

### 2.3 遗赠与遗嘱继承区分
| 情形 | 法律适用 | 接受期限 |
|-----|---------|---------|
| 遗嘱继承人 | 《民法典》继承编 | 知道受遗赠后60日内作出表示 |
| 受遗赠人 | 《民法典》第1125条类推适用 | 知道受遗赠后60日内作出表示 |
| 胎儿预留份 | 《民法典》第1155条 | 出生后法定代理人代为表示 |

## 三、遗嘱效力风险矩阵与修改建议

### 3.1 效力风险等级
| 风险等级 | 描述 | 典型情形 | 建议措施 |
|---------|------|---------|---------|
| 🔴 高风险 | 遗嘱可能被认定全部无效 | 完全自书无签名、见证人为利害关系人 | 重新起草，建议公证 |
| 🟡 中风险 | 部分条款无效或效力待定 | 遗漏必留份、处分他人财产 | 修改条款后补充公证 |
| 🟢 低风险 | 形式瑕疵不影响实质效力 | 日期不完整但可核实 | 建议公证确认 |

### 3.2 典型问题与修改方案

**问题A：夫妻共同财产未析产**
```
原条款示例：
"将位于北京市朝阳区XX房屋遗赠给儿子张某"
风险：该房屋为夫妻共同财产，遗嘱人仅有权处分1/2产权
修改建议：
"将本人持有的位于北京市朝阳区XX房屋1/2产权份额遗赠给儿子张某，
并建议在遗产分割前由配偶书面确认其应得份额归属"
```

**问题B：代书遗嘱见证人不适格**
```
风险：见证人与受遗赠人存在姻亲关系，可能被认定为利害关系人
合规见证人范围：邻居、同事、朋友（无血缘、无姻亲、无债权债务）
修改建议：更换见证人，重新制作代书遗嘱
```

**问题C：限制行为能力继承人未保留必留份**
```
步骤1：查询所有继承人现状
步骤2：若有未成年人/丧失劳动能力者，计算必留份
步骤3：在遗嘱中明确保留份额及给付方式
步骤4：建议同步制定信托安排以保障该继承人长期生活
```

### 3.3 遗嘱执行准备度评估
| 检查项 | 标准 | 当前状态 |
|-------|------|---------|
| 遗嘱正本存放安全 | 公证机构或银行保险箱 | □合规 □待改 |
| 遗嘱副本告知可信赖人士 | 至少1人知悉存放位置 | □合规 □待改 |
| 遗产清单完整度 | 不动产、存款、股票、首饰等均有列示 | □合规 □待改 |
| 遗愿执行时间表 | 葬礼、财产过户等有时间节点 | □合规 □待改 |

## 升级决策门

### 触发条件（满足任一即升级）
1. **遗嘱效力争议**：存在继承人或其他利害关系人对遗嘱效力提出异议或诉讼
2. **跨境因素**：遗嘱涉及境外财产、境外身份或需适用外国法律
3. **特殊主体**：遗嘱人/继承人为外籍人士、港澳台居民或无国籍人
4. **复杂财产结构**：遗产包含有限责任公司股权、合伙企业份额、私募基金等需专业估值的资产
5. **精神状态存疑**：曾有精神障碍诊断记录，或遗嘱订立前后有大额财产转移
6. **遗赠负担**：遗嘱附有公益捐赠、抚养义务等复杂条件
7. **遗产金额巨大**：单笔或合计遗产超过人民币5000万元

### 升级路径
```
升级决策门触发 → 停止自动生成审核结论 → 
输出"本案件需专业律师介入"通知 → 
提供就近财富传承专业律师查询路径（参见 references/查询路径.md）
```

### 升级后本skill责任终止情形
- 进入诉讼程序后，本skill不再提供程序性建议
- 涉及遗嘱效力认定的司法鉴定环节，不属于本skill范围
- 跨境遗产纠纷的管辖权确定，不属于本skill范围

## 本Skill不涵盖

### 明确排除事项
1. **遗嘱公证代理**：公证手续需当事人亲自到场或委托公证处，本skill不代理任何公证手续
2. **遗嘱见证人安排**：本skill不作为见证人，不出具见证人证明
3. **遗产价值评估**：不动產、股权等资产的市场价值评估由具有资质的评估机构出具
4. **遗嘱执行人职责履行**：执行人（尤其是遗嘱执行人与继承人非同一人时）的具体职责不属本skill范围
5. **继承诉讼代理**：涉及继承权纠纷、遗嘱效力诉讼的代理服务
6. **跨境遗产规划**：涉及FATCA、CRS、外汇管制等跨境因素的财富传承安排
7. **税务申报代理**：遗产税申报、增值税申报等代理服务
8. **精神鉴定申请**：遗嘱能力司法鉴定程序
9. **家族信托设立**：信托架构设计与信托合同起草（参见 trust-structure-advisor）
10. **婚前/婚内财产协议**：夫妻财产约定（参见 prenuptial-agreement-checker）
