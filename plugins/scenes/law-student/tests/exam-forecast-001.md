# Test Case: 法考考前两个月高频考点预测

## Query

我是一名大三法学生，计划参加今年9月的国家统一法律职业资格考试（客观题）。
目前我的复习进度如下：
1. 民法：已完成一轮复习，真题正确率约70%，合同编和物权编掌握较好，人格权编偏弱
2. 刑法：已完成一轮复习，真题正确率约65%，分则常见罪名尚可，总则共犯理论薄弱
3. 民事诉讼法：已完成一轮，真题正确率约60%，管辖和证据规则需要加强
4. 行政法与行政诉讼法：刚完成50%，行政许可和行政强制混淆
5. 刑事诉讼法：尚未开始
6. 商法：已完成公司法部分，破产法和票据法未开始
7. 理论法、三国法、经济法：尚未开始

距离客观题考试还有约2个月，请帮我预测今年的高频考点分布，
并制定每周冲刺计划。

## Expected Behavior

1. 应根据当前考试大纲和历年真题分布，预测高频考点
2. 应针对该考生的薄弱环节（刑诉、行政法、三国法等）给出优先级建议
3. 应结合时间分配（2个月）制定周度复习计划
4. 应指出每个科目的核心必考知识点
5. 应给出客观题应试技巧（如排除法、时间分配等）
6. 标注预测依据和不确定性范围
