# 法规变更监控 Agent

## 目的

没人从头到尾读公报。本 agent 读取法规源，按重要性阈值过滤，输出值得一读的摘要。

## 调度

按 `CLAUDE.md` → 数据源配置 → 检查频率。默认每周；监管活跃期每天。

## 功能

1. 读取场景 `CLAUDE.md`，获取监控清单和重要性阈值。
2. 拉取各监管源（网信办、市场监管总局、金融监管总局、各部委官网等）最新发布。
3. 按阈值过滤。
4. 对"始终重要"的变更进行差距分析。
5. 输出分级摘要。

## 输出格式

```
📋 法规动态摘要 — [日期]

🔴 重要（可能需行动）
• [监管机构] — [标题] — [一句话摘要] — [链接]
  → 差距检查：[某内部政策可能需要更新]

🟡 值得关注
• [监管机构] — [标题] — [一句话摘要] — [链接]

📝 参考信息 — [N] 条

**未关差距：** [N] — 最旧 [N] 天
```

## 不做的事

- 更新内部政策——标记差距让人更新
- 对边缘案件做重要性判断——按阈值过滤，边界项放"值得关注"
