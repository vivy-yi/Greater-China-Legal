# 建设工程合同审查

## 一、基本信息

### 1.1 核心目的
为发包人/总包人/分包人提供建设工程施工合同全要素审查，覆盖合同价款、付款节点、变更签证、工期延误、质量验收、优先受偿权及质保金等核心条款。

### 1.2 适用范围
- 建设工程总承包合同（EPC/DB/施工总承包）
- 专业分包合同（桩基/幕墙/消防/机电/装修等）
- 劳务分包合同
- 建设工程设计/勘察/监理合同
- 建设工程材料/设备采购合同

## 二、分析框架

### 2.1 建设工程合同效力审查

| 无效情形 | 法律后果 | 补救措施 |
|---------|---------|---------|
| 承包人未取得建筑施工企业资质 | 合同无效，参照合同约定折价补偿 | 取得资质或联合有资质企业投标 |
| 超越资质等级的承包人 | 合同无效（竣工前取得资质可补正） | 竣工前取得相应资质 |
| 没有资质的实际施工人借用资质（挂靠） | 合同无效，发包人知情需承担连带责任 | 禁止挂靠，通过合法分包操作 |
| 建设工程必须招标而未招标/中标无效 | 合同无效 | 重新招标或补充招标程序 |
| 转包/违法分包 | 转包/分包合同无效，转包人/违法分包人受处罚 | 合规分包（主体工程不得分包） |

### 2.2 工程价款核心条款审查

| 审查要点 | 风险提示 | 最佳实践 |
|---------|---------|---------|
| 固定总价 | 工程范围变化导致结算争议 | 明确包干范围和签证调价机制 |
| 单价合同 | 工程量计量基准 | 明确清单计价规范（GB50500-2013） |
| 可调价合同 | 材料价格波动调价公式 | 设置调价触发条件（如波动超±5%） |
| 预付款 | 预付款保函（10%预付款须履约保函） | 约定预付款扣回方式和节点 |
| 进度款 | 节点认定争议 | 明确每节点的完成标准+验收确认 |
| 结算款 | 结算审价周期（发包人延迟审价） | 约定默认认可条款（28天内） |
| 质保金 | 保留3%合同总价，缺陷责任期最长2年 | 明确起算时点和返还条件 |

**建设工程价款优先受偿权**（核心保护机制）：
```
行使条件
├── 发包人未按约定支付工程价款（非因承包人原因）
├── 承包人催告发包人在合理期限内付款
├── 发包人逾期仍不支付
└── 行使期限：自发包人应当给付价款之日起18个月（2021年新解释）

优先受偿范围
├── 工作人员报酬/材料款（直接费用）
├── 工程款（利润部分一般不享有优先权，但实践中存在争议）
├── 垫资款（需有明确约定）
└── 注意：逾期付款利息/违约金不享有优先受偿权
```

### 2.3 工期与延误条款

| 延误原因 | 责任方 | 处理机制 |
|---------|-------|---------|
| 发包人未提供施工条件 | 发包人 | 工期顺延+费用索赔 |
| 设计变更/工程量增加 | 发包人 | 工期顺延+工程款调整 |
| 发包人迟延付款 | 发包人 | 工期顺延+停止施工权利 |
| 不可抗力（疫情/灾害） | 双方分担 | 工期顺延+按公平原则分担费用 |
| 承包人组织管理不善 | 承包人 | 工期延误违约金 |
| 分包人/供应商原因 | 承包人 | 承包人向发包人承担连带责任 |

**工期延误违约金审查重点**：
- 违约金比例：通常为合同价款的万分之几/天（合理区间0.1‰-0.3‰/日）
- 违约金上限：不超过合同总价的5%-10%
- 关键节点：中间工期节点违约金应单独约定
- 封顶条款：工期延误违约金是否设有上限

## 三、操作流程

### 3.1 建设工程合同审查十步法

**Step 1: 合同基本信息审查**
- [ ] 各方主体资质（营业执照+建筑业企业资质证书）
- [ ] 工程概况（名称/地点/规模/工期）
- [ ] 合同金额与计价方式
- [ ] 适用的合同范本

**Step 2: 工程范围条款**
- [ ] 承包范围的精确界定（图纸+清单+技术规格书）
- [ ] 工作范围变更的触发机制
- [ ] 范围排除条款（甲供/甲指分包范围）
- [ ] 配合/协调义务的界定

**Step 3: 价款与支付条款**
- [ ] 预付款比例与支付条件
- [ ] 进度款支付节点与比例
- [ ] 结算程序与时限
- [ ] 背靠背条款审查（总包与分包之间）

**Step 4: 变更签证条款**
- [ ] 变更的范围和类型
- [ ] 签证的确认程序和时限
- [ ] 变更价款的计算方法
- [ ] 发包人不及时确认时的处理

**Step 5: 工期条款**
- [ ] 开工日期和竣工日期
- [ ] 关键节点工期约定
- [ ] 工期顺延的条件和程序
- [ ] 工期延误违约金

**Step 6: 质量与验收条款**
- [ ] 质量标准（国家标准/行业标准/合同约定标准）
- [ ] 分部分项验收程序
- [ ] 竣工验收程序与条件
- [ ] 竣工验收备案义务

**Step 7: 索赔与争议解决**
- [ ] 索赔程序（28天索赔通知+证据要求）
- [ ] 争议解决方式（仲裁/诉讼）
- [ ] 鉴定机构的选定
- [ ] 优先受偿权行使条件

**Step 8: 知识产权与保密**
- [ ] 施工图纸/技术文件的著作权归属
- [ ] 技术成果的归属和使用
- [ ] 保密信息范围和期限

**Step 9: 保险与担保**
- [ ] 建筑工程一切险
- [ ] 第三方责任险
- [ ] 履约保函/预付款保函/质保金保函
- [ ] 独立保函vs从属性保函

**Step 10: 合同终止与清算**
- [ ] 合同解除条件
- [ ] 合同解除后的清算条款
- [ ] 已完工程价款的结算
- [ ] 人员/设备/材料的退场安排

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **合同效力争议** — 未招标/挂靠/转包/违法分包嫌疑
- **优先受偿权行使争议** — 行使期限届满/优先权范围争议
- **建设工程质量事故/安全事故** — 涉及行政处罚/刑事责任
- **涉群体性工程款纠纷** — 农民工工资支付/集体讨薪
- **发包人破产/重整** — 工程款在破产程序中的处理
- **政府类投资项目** — 财政评审/审计结论对抗结算
- **涉外建设工程合同** — FIDIC合同条件适用
- **EPC/总包合同中的设计责任** — 设计缺陷导致的成本超支

### 4.2 升级咨询模板

```
【升级咨询】
合同类型：[总承包/分包/设计/采购]
合同金额：[万元]
工程类别：[房建/市政/工业/其他]
争议焦点：[合同效力/工程款/工期/质量/其他]
已发函件：[催款函/索赔函/解除函等]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. FIDIC合同条款的全面审查（国际工程合同规则复杂）
2. 建设工程安全事故的行政处罚应对
3. 建设工程给排水/暖通等专业工程的特殊标准
4. 政府类工程项目财政评审/审计结算规则
5. 建设工程招标投标阶段的法律服务
6. BT/BOT/PPP等特殊建设模式合同
7. 工程保险理赔的代理谈判
