# 土地使用权获取审查

## 一、基本信息

### 1.1 核心目的
为房地产开发商/产业投资方审查土地使用权获取中的法律风险，覆盖招拍挂出让、协议出让、股权收购、集体经营性建设用地入市等路径的合规性分析与风险排查。

### 1.2 适用范围
- 国有建设用地使用权出让（招拍挂/协议）
- 国有建设用地使用权转让（资产收购/股权收购）
- 集体经营性建设用地入市（试点）
- 城市更新/旧改项目土地获取
- 勾地/带方案出让
- 土地一二级联动开发

## 二、分析框架

### 2.1 土地获取路径对比

| 获取路径 | 适用情形 | 优势 | 风险 |
|---------|---------|------|------|
| 招拍挂出让 | 经营性用地（商业/住宅/工业） | 程序公开透明，物权稳定 | 竞争激烈，价格高 |
| 协议出让 | 特定用途（公益/教育/医疗/工业） | 价格可控 | 适用条件严格，审批复杂 |
| 资产收购 | 直接取得土地使用权+地上建筑物 | 直接取得物业 | 土地增值税高，转让限制 |
| 股权收购 | 通过收购目标公司股权间接取得 | 规避土地转让限制，税务优化 | 隐性债务/或有负债 |
| 集体经营性建设用地入市 | 集体建设用地（试点范围） | 成本较低 | 政策试点不稳定，融资难度大 |
| 城市更新/旧改 | 存量建设用地再开发 | 土地熟化，升值空间 | 拆迁周期长，不确定大 |

### 2.2 土地合规审查清单

**出让前审查**
- [ ] 土地权属清晰（无争议/无查封/无抵押）
- [ ] 土地规划条件明确（用途/容积率/建筑密度/限高）
- [ ] 土地征收/拆迁补偿已到位
- [ ] 出让公告程序合规（发布时间/竞买资格/保证金）
- [ ] 土地出让方案经自然资源局审批
- [ ] 地价评估（须有资质的评估机构）

**出让合同审查**
- [ ] 土地用途（是否与规划一致）
- [ ] 出让年限（住宅70年/商业40年/工业50年）
- [ ] 出让金总额及支付方式
- [ ] 开竣工时间约定（开工/竣工期限）
- [ ] 闲置土地处置条款
- [ ] 违约责任条款（延期支付出让金/延期开发）
- [ ] 政府回购条款（如有）

**股权收购方式特殊审查**
- [ ] 目标公司历史沿革（设立/股权变更/增资）
- [ ] 目标公司负债（银行贷款/应付工程款/担保）
- [ ] 目标公司诉讼/仲裁（土地/M&A相关争议）
- [ ] 目标公司税务（土地增值税/企业所得税/契税）
- [ ] 股权转让备案（如有外资/国有成分）

### 2.3 闲置土地认定与防风险

```
闲置土地认定标准（《闲置土地处置办法》第2条）
├── 已动工开发但开发建设用地面积占应动工开发总面积不足1/3
├── 已投资额占总投资额不足25%
├── 中止开发建设满1年
└── 国有建设用地使用权有偿使用合同/划拨决定书约定的动工日期后满1年未动工开发

处置方式（按认定原因）
├── 因政府原因：延长动工期限/调整土地用途规划/置换土地/协议有偿收回
├── 因企业原因：征缴土地闲置费（出让或划拨价款的20%）
└── 因企业原因且满2年：无偿收回土地使用权

防范措施
├── 签订出让合同时明确开竣工期限
├── 按期开发，避免非实质性开发
├── 如遇政府原因导致闲置，及时取证（发函+申请延期）
├── 定期拍摄现场照片/航拍，建档保存
└── 及时申请延期开发（动工前30天申请）
```

### 2.4 集体经营性建设用地入市要点

| 审查要素 | 具体要求 |
|---------|---------|
| 土地性质 | 土地利用总体规划/城乡规划确定为工矿仓储/商服等经营性用途 |
| 产权主体 | 集体经济组织（村集体/乡镇集体） |
| 入市方式 | 出让/出租/作价出资（入股） |
| 使用年限 | 参照同类国有建设用地（商业40年/工业50年） |
| 收益分配 | 集体经济组织内部民主决策+政府收益调节金 |
| 限制条件 | 不得用于商品住宅开发（试点期间） |

## 三、操作流程

### 3.1 土地获取法律尽职调查流程

**Phase 1: 土地信息收集（1-2周）**
- [ ] 获取地块的规划条件通知书/建设用地规划许可证
- [ ] 查询土地权属（自然资源局不动产登记中心）
- [ ] 获取土地出让公告/出让合同
- [ ] 现场勘查（四至范围/现状/周边设施/配套）

**Phase 2: 合规审查（1-2周）**
- [ ] 土地出让程序合规性审查
- [ ] 土地出让金缴纳情况核实
- [ ] 开竣工约定与执行情况
- [ ] 土地利用现状与规划的一致性
- [ ] 是否存在闲置土地认定风险
- [ ] 是否存在土地使用权抵押/查封

**Phase 3: 隐性风险排查（2-3周）**
- [ ] 土壤污染状况调查（污染地块须修复后方可开发）
- [ ] 历史文化遗产/文物保护排查
- [ ] 地下管线/人防工程影响
- [ ] 地质灾害/地基条件风险
- [ ] 周边居民/企业的环保投诉/诉讼
- [ ] 权利负担（地役权/租赁权/抵押权）

**Phase 4: 交易结构设计**
- [ ] 选择获取路径（招拍挂/股权收购/资产收购）
- [ ] 土地出让金支付计划
- [ ] 税务筹划（契税/土地增值税/企业所得税）
- [ ] 开发节奏与退出安排

### 3.2 土地出让合同签署核查要点

| 核查项 | 建议条款 | 风险提示 |
|-------|---------|---------|
| 用途条款 | 与规划条件完全一致 | 用途不一致无法办证 |
| 容积率 | 具体数值+调整程序 | 超容积率建设=违法建设 |
| 开竣工时间 | 具体日期或条件 | 延期导致闲置认定 |
| 出让金支付 | 分期支付须约定尾款利息 | 逾期=违约金+解除合同风险 |
| 土地交付 | 净地交付标准+时间 | 毛地交付=拆迁未完成风险 |
| 配套建设 | 学校/幼儿园/社区用房移交 | 移交标准/移交时间 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **土地权属存在争议** — 多主体主张土地所有权/使用权
- **闲置土地认定风险** — 已满1年以上未开发，面临收回风险
- **土地出让程序存在瑕疵** — 公告期不足/竞买资格不合规/底价未经评估
- **涉及征地拆迁补偿纠纷** — 原权利人提起行政诉讼
- **涉公益/基本农田** — 土地性质与规划用途不一致
- **股权收购中的重大隐形债务** — 目标公司存在未披露担保/诉讼
- **涉及政府招商引资合同纠纷** — 政府承诺未兑现vs企业违约
- **土地涉及刑事犯罪** — 非法占用农用地/非法转让倒卖土地使用权

## 五、本Skill不涵盖

1. 土地征收/拆迁补偿方案的编制与谈判
2. 违法用地行政处罚的行政复议/行政诉讼
3. 矿产资源压覆/地质灾害评估（地质专业机构出具）
4. 土地增值税清算（税务专业）
5. 在建工程抵押/土地使用权抵押的具体操作
6. 坡地/山地开发的特殊坡地管理规定
7. 军事用地/宗教用地/水利设施用地等特殊用地
