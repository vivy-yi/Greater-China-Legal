# 商品房预售合规审查

## 一、基本信息

### 1.1 核心目的
为房地产开发企业提供商品房预售全流程合规审查，覆盖预售许可证取得条件、广告宣传合规、资金监管、购房者资格审查及按揭贷款对接的全链条风险防控。

### 1.2 适用范围
- 商品住宅预售（期房销售）
- 商业/办公用房预售
- 限价房/共有产权房预售
- 保障性住房配售

## 二、分析框架

### 2.1 预售许可证取得条件

根据《城市房地产管理法》第45条，《城市商品房预售管理办法》第5条，预售必须满足：

| 条件 | 具体要求 | 证明材料 |
|------|---------|---------|
| 已交付全部土地使用权出让金 | 土地出让金缴清+取得国有土地使用证 | 土地使用权证/不动产权证 |
| 持有建设工程规划许可证 | 规划部门审批通过并获证 | 建设工程规划许可证 |
| 投入开发建设资金达到总投资的25%以上 | 施工进度达标（通常要求±0.00或主体结构出地面） | 施工单位证明+监理报告+资金投入审计 |
| 施工进度和竣工交付日期确定 | 施工合同明确开竣工时间 | 施工许可证+施工合同 |
| 已确定施工进度和竣工交付日期 | 同上 | 同上 |
| 已落实前期物业管理（部分地区要求） | 前期物业服务合同+物业用房 | 前期物业合同 |

**禁止预售情形**：
- 未取得预售许可证即收取定金/意向金/购房款
- 以认购/预订/排号等名义变相预售
- 取得预售许可证前发布预售广告
- 捂盘惜售/囤积房源/囤积预售许可证

### 2.2 预售广告宣传合规

根据《广告法》第26条和《房地产广告发布规定》，房地产广告不得：

| 禁止内容 | 典型案例 | 处罚后果 |
|---------|---------|---------|
| 升值/投资回报承诺 | "五年升值50%""年租金回报率8%" | 广告费1-3倍罚款 |
| 以时间表示距离 | "距地铁站5分钟""距国贸15分钟" | 责令停止发布+罚款 |
| 承诺办理户口/入学 | "买房送户口""学区房保障入学" | 行政处罚+虚假宣传诉讼 |
| 违反价格管理规定 | "起价XX元"不含具体房源信息 | 价格法处罚 |
| 未取得预售许可证的广告 | 未取得预售证即发布 | 市场监管处罚 |

**广告合规审查清单**：
- [ ] 预售许可证号码已标注（取得后）
- [ ] 开发企业名称已标注
- [ ] 项目位置描述使用实际距离（km/m）
- [ ] 无"最""第一""首个"等绝对化用语
- [ ] 项目效果图/样板间标注"仅供参考"
- [ ] 学区/落户等不做承诺
- [ ] 不限购宣传须核实政策（尤其一线城市）
- [ ] 面积单位（平方米）与合同一致

### 2.3 商品房预售合同核心条款

| 条款 | 合规要求 | 常见违规 |
|------|---------|---------|
| 房屋基本信息 | 坐落/房号/面积/套型/单价/总价 | 房号不清/面积与实测不符 |
| 交付条件 | 竣工验收备案+两书（质量保证书+使用说明书） | 以"单体竣工"代替"综合验收" |
| 交付时间 | 具体日期或可确定的日期 | 模糊约定（"预计""约"） |
| 面积差异处理 | ±3%以内的多退少补，超过3%可退房 | 不约定面积差异处理 |
| 产权办理 | 房屋交付后90日内办理 | 约定超长办理期限（无期） |
| 装饰装修 | 装修标准/材料品牌/违约赔偿 | "同等档次品牌"差异巨大 |
| 公共配套设施 | 交付时间+运营标准 | 不移交/改为他用 |
| 物业 | 物业服务企业/收费标准/服务内容 | 强制前期物业+收费标准不明 |

### 2.4 预售资金监管

| 监管要素 | 要求 |
|---------|------|
| 监管账户 | 在住建部门指定银行开设专户 |
| 入账资金 | 全部购房款（含定金/首付款/按揭贷款） |
| 资金用途 | 仅用于项目建设（材料/人工/设备/税费） |
| 拨付节点 | 按工程进度核准拨付（±0.00/主体/封顶/竣工/交付） |
| 超额资金 | 超出建设成本的资金可提取 |
| 违规后果 | 暂停拨付+行政处罚+信用惩戒 |

## 三、操作流程

### 3.1 预售许可证办理流程

**Phase 1: 前置条件排查（2-4周）**
- [ ] 土地出让金缴清证明
- [ ] 国有土地使用证/不动产权证已取得
- [ ] 建设工程规划许可证已取得
- [ ] 施工许可证已取得
- [ ] 施工进度达到预售标准（各地要求不同）

**Phase 2: 材料准备（1-2周）**
- [ ] 预售申请表（住建局标准格式）
- [ ] 营业执照/开发资质证书
- [ ] 土地使用权证/规划许可证/施工许可证
- [ ] 施工合同/资金投入证明
- [ ] 预售方案（售价/开盘时间/交付时间/前期物业）
- [ ] 前期物业服务合同
- [ ] 资金监管协议/监管账户证明
- [ ] 预测绘报告（房产测绘部门出具）
- [ ] 项目现场照片/施工进度证明

**Phase 3: 申请与审批（2-4周）**
- [ ] 线上/线下提交申请材料
- [ ] 住建局初审（材料完整性+合规性）
- [ ] 现场核查（施工进度+售楼处+样板间）
- [ ] 审批通过/出具预售许可证
- [ ] 预售许可证公示（住建局官网/现场）

**Phase 4: 开盘前准备**
- [ ] 预售许可证复印件在售楼处公示
- [ ] 一房一价表公示
- [ ] 预售合同示范文本公示
- [ ] 销售人员培训（合规销售话术）
- [ ] 开盘方案（摇号/选房/签约）

### 3.2 购房者资格审查

| 审查类型 | 审查内容 | 审查依据 |
|---------|---------|---------|
| 限购资格 | 本地/外地户籍/社保缴纳/纳税记录 | 城市限购政策 |
| 贷款资格 | 征信报告/收入证明/首付比例 | 央行房贷政策 |
| 税费优惠 | 首套房/二套房认定 | 房产局登记系统 |
| 特殊群体 | 人才购房/军人/拆迁安置 | 各城市特殊政策 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **无证预售/变相预售** — 未取得预售许可证即收取房款
- **逾期交房/烂尾风险** — 开发商资金链断裂/停工
- **虚假宣传/欺诈** — 严重货不对板/学区/配套虚假承诺
- **预售资金挪用** — 未入监管账户/违规提取
- **群体性购房者维权** — 集体退房/集体投诉
- **限购政策违规** — 协助购房者伪造社保证明
- **项目停工/资金链断裂** — 施工单位起诉开发商
- **预售许可证被撤销** — 不符合条件骗取许可

### 4.2 升级咨询模板

```
【升级咨询】
项目名称：[楼盘名称]
预售证号：[证号]

当前阶段：[预售前/预售中/交房前/交房后]
主要风险：[具体描述]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. 商品房买卖合同纠纷的诉讼代理
2. 购房者因质量/延期/虚假宣传的集体诉讼
3. 开发贷款/项目融资的法律服务
4. 建设工程优先受偿权与购房者权利的冲突处理
5. 商品房限购/限售/限价政策的具体解读
6. 现房销售的法律服务（现售程序不同于预售）
7. 房屋交付后的产权办理/税费申报
