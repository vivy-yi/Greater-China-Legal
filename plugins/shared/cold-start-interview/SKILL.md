# /cold-start-interview — China Mainland

## Purpose

中国劳动法是按省份/直辖市细分的地方法律体系。同一问题在上海和广东的答案可能完全不同。本向导绘制你的业务版图——覆盖哪些省市——并构建一个了解各地差异化规则的升级表。

## 管辖版图询问

### Part 0：使用人身份

> 本插件面向处理中国大陆劳动法事务的法务/HR/企业管理者。你在哪里处理劳动关系问题？

**选项：**
1. **公司法务或法律专业人员** — 律师、法务、合规人员，在律师监督下工作。
2. **非律师但有律师资源** — 创始人、业务负责人、法务经理、HR、采购；你有内部或外部律师可咨询。
3. **非律师且无固定律师资源** — 你自行处理劳动关系事务。

### Part 1：管辖版图

> 你在中国哪些省市有员工？（覆盖越多，适配越准确）

**选项：**
- **省份（直辖市/自治区）：** 北京、上海、广东（含深圳）、浙江（含宁波）、江苏、四川、湖北、湖南、福建、山东、河南、河北、辽宁、陕西、安徽、江西、云南、贵州、黑龙江、吉林、山西、海南、内蒙古、广西、西藏、宁夏、青海、新疆
- **特别行政区：** 香港、澳门
- **台湾地区**
- **其他法域：** 新加坡

**风险等级说明：**
- 🔴 高风险：北上广深（仲裁/诉讼量大，裁判口径严）
- 🟠 中风险：浙江、江苏（经济补偿金基数认定较严格）
- 🟡 一般：其他省份

### Part 2：用工类型

> 你公司的用工类型有哪些？（可多选）

**选项：**
- 全日制劳动合同
- 非全日制用工（小时工）
- 劳务派遣（注意同工同酬合规）
- 业务外包
- 实习生
- 外国人就业（注意就业许可）
- 退休返聘

### Part 3：劳动关系风险触发器

> 你公司曾经遇到过哪些劳动关系问题？（帮助识别高风险情形）

**常见CN风险触发器：**
- 解除劳动合同未支付经济补偿金
- 拖欠工资/加班费
- 社保/公积金未足额缴纳
- 工伤认定争议
- 违法解除（2N赔偿）
- 竞业限制纠纷
- 股权激励争议

### Part 4：离职补偿框架

> 你公司目前的离职补偿标准？（建立风险阈值）

**CN补偿框架（默认）：**

| 情形 | 补偿标准 | 法律依据 |
|---|---|---|
| 协商解除 | N个月工资（或N+1）| 《劳动合同法》第36条 |
| 医疗期满解除 | N+1 | 《劳动合同法》第40条 |
| 不胜任解除 | N+1 | 《劳动合同法》第40条 |
| 客观情况重大变化 | N+1 | 《劳动合同法》第40条 |
| 经济性裁员 | N（优先留用）| 《劳动合同法》第41条 |
| 违法解除 | 2N | 《劳动合同法》第87条 |
