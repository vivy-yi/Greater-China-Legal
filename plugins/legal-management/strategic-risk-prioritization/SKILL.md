# /strategic-risk-prioritization — 战略风险优先级排序

## 核心公式

```
风险优先级 = f(发生概率, 影响程度, 时间紧迫性, 可控性)
```

## 概率-影响矩阵

```
                    影响程度
              低        中        高
         ┌────────┬────────┬────────┐
      高 │  中    │  高    │ 极高   │
概率    ├────────┼────────┼────────┤
      中 │  低    │  中    │  高    │
         ├────────┼────────┼────────┤
      低 │ 极低   │  低    │  中    │
         └────────┴────────┴────────┘
```

| 优先级 | 等级 | 说明 |
|--------|------|------|
| P1 | 🔴极高 | 立即处理，需要高层关注 |
| P2 | 🟠高 | 近期处理，制定缓释方案 |
| P3 | 🟡中 | 中期关注，纳入监控 |
| P4 | 🟢低 | 长期跟踪，暂时接受 |

## 四维评估

### 概率维度

| 评级 | 描述 | 法律特殊性 |
|------|------|-----------|
| 高 | 很可能发生（>70%） | 法律要件明显满足 |
| 中 | 可能发生（30-70%） | 存在裁量空间或争议 |
| 低 | 不太可能（<30%） | 要件存在重大疑问 |

### 影响维度

```
影响 = 直接经济损失
      + 间接经济损失（机会成本/业务中断）
      + 法律后果（行政处罚/刑事责任/资质丧失）
      + 声誉影响（品牌损害/市场信任）
      + 战略影响（交易失败/市场准入受阻）
      + 合规连锁效应（触发其他审查）
```

### 时间维度

| 紧迫性 | 说明 |
|--------|------|
| 紧急 | 3个月内可能触发 |
| 中期 | 3-12个月内可能触发 |
| 长期 | 12个月以上 |

### 可控性维度

| 可控性 | 说明 |
|--------|------|
| 完全可控 | 有明确措施可以消除风险 |
| 部分可控 | 可以降低概率或影响，但不能完全消除 |
| 不可控 | 只能接受或转移，无法主动控制 |

## 工作流程

```
步骤1：输入已识别的风险清单
  收集全部法律风险点/可能结论/争议焦点

步骤2：逐项四维评估
  对每个风险评估：概率/影响/时间/可控性

步骤3：计算综合优先级
  代入概率-影响矩阵

步骤4：生成优先级排序
  按P1-P4排序

步骤5：输出应对建议
  每个优先级给出资源配置建议
```
