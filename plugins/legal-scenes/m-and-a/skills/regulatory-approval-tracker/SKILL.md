# 监管审批追踪技能 (Regulatory Approval Tracker)

## 技能定位

本技能用于管理和追踪并购交易涉及的全部监管审批事项，确保合规高效完成审批流程。

## 核心能力

### 1. 审批类型识别

#### 1.1 反垄断审查
```
申报标准（经营者集中）：
- 参与集中的经营者上一会计年度全球营业额合计≥100亿元
- 或中国境内营业额合计≥20亿元
- 且至少两个经营者中国境内营业额≥4亿元

申报程序：
第一阶段：初步审查（30日）
第二阶段：进一步审查（90日）
第三阶段：延长审查（不超过180日）

简易程序：
- 同一合伙控制下企业合并
- 经营者集中涉及知识产权
- 境外集中导致中国境内控制权变化
```

#### 1.2 外商投资审批
```
审批层级：
- 负面清单（全国）→ 商务部/省级商务主管部门
- 负面清单（自贸区）→ 更短时限

审批时限：
- 常规：90个工作日
- 自贸区：30个工作日（负面清单外）

安全审查：
- 涉及国防安全、关键技术、粮食安全等
- 需过安全审查联席会议
```

#### 1.3 行业监管审批
```
金融行业：
- 银行收购：银保监会审批
- 证券公司：证监会审批
- 保险公司：银保监会审批
- 信托公司：银保监会审批

能源矿产：
- 探矿权/采矿权转让：自然资源部/省级厅
- 油气管道：国家管网公司审批
- 电力项目：发改委/能源局

其他行业：
- 医疗机构：卫健委
- 教育机构：教育部/教育厅
- 出版传媒：宣传部/文旅部
```

#### 1.4 上市公司审批
```
重大资产重组：
- 资产总额/营业收入/净资产≥50%
- 需证监会重组委审核

收购上市公司：
- 超过30%：要约收购豁免/全面要约
- 实际控制人变更：详式权益变动报告

发行股份购买资产：
- 需证监会发行审核委员会审核
- 审核周期通常3-6个月
```

### 2. 审批流程管理

#### 2.1 审批申请准备
```
反垄断申报材料：
□ 经营者集中申报表
□ 交易协议
□ 参与方营业执照和财务报表
□ 市场份额说明
□ 经济分析报告
□ 集中对竞争影响的分析

外商投资审批材料：
□ 项目申请报告
□ 可行性研究报告
□ 投资各方证明文件
□ 投资协议/章程
□ 产能说明/能耗说明（如需）
□ 行业许可证明
```

#### 2.2 审批时间线管理
```
甘特图管理要素：
- 各审批事项开始时间
- 各审批事项预计完成时间
- 各审批事项实际完成时间
- 前置审批与后续审批关系
- 并行审批与串行审批识别

关键路径：
□ 反垄断审查（通常最长）
□ 行业监管审批
□ 外商投资审批
□ 外汇登记
```

#### 2.3 审批进度跟踪
```
跟踪维度：
□ 申请材料受理状态
□ 审核进行中阶段
□ 补正通知及回复
□ 初步审查意见
□ 最终审批决定

预警机制：
- 距预计完成时间30日未完成的预警
- 收到补正通知后7日内预警
- 政策变化可能影响审批的预警
```

### 3. 多审批协调

#### 3.1 并行审批管理
```
并行审批事项：
- 反垄断 + 外商投资 + 行业监管
- 各自独立审批，可同步进行

协调要点：
□ 材料共用部分统一准备
□ 时间节点统筹安排
□ 审批机关协调沟通
□ 补正内容相互借鉴
```

#### 3.2 串行审批管理
```
串行审批情形：
- 外商投资审批 → 反垄断审查
- 行业准入 → 工商变更

管理要点：
□ 识别前置审批
□ 前置审批完成前准备后续材料
□ 预留足够时间缓冲
□ 设置终止费与审批绑定
```

### 4. 特殊审批事项

#### 4.1 跨境审批
```
ODI备案（发改委/商务部）：
- 境外直接投资备案
- 需提供投资架构图
- 资金来源说明

外汇登记：
- 37号文登记（自然人）
- 9号文登记（企业）
- ODI外汇登记

东道国审批：
- 美国CFIUS审查
- 欧盟外国补贴审查
- 其他国家安全审查
```

#### 4.2 国有企业审批
```
审批层级：
- 国有资产监管机构审批
- 人民政府审批（重大交易）
- 交易所公开挂牌（产权转让）

特殊要求：
- 净资产评估
- 职工安置方案
- 债权债务处置方案
- 进场交易
```

### 5. 审批问题处理

#### 5.1 补正通知应对
```
收到补正通知后：
□ 仔细分析补正内容
□ 识别补正原因
□ 准备补正说明
□ 准备补充材料
□ 按时提交补正

补正类型：
- 材料不完整
- 内容不清晰
- 需要进一步说明
- 需要补充佐证
```

#### 5.2 审批附条件/附期限
```
常见附条件：
- 剥离特定资产
- 限制市场份额
- 开放技术/渠道
- 持续信息披露

应对策略：
□ 评估条件可执行性
□ 评估条件对交易影响
□ 与审批机关沟通条件合理性
□ 在交易文件中设置相应条款
```

#### 5.3 审批被拒应对
```
风险识别：
□ 申报前评估审批风险
□ 预设审批失败情形
□ 交易文件预留终止条款
□ 终止费安排

替代方案：
□ 调整交易结构
□ 剥离敏感资产
□ 申请豁免/例外
□ 放弃交易
```

### 6. 审批文档管理

#### 6.1 文档归档
```
归档内容：
□ 申报材料原件
□ 补正通知及回复
□ 审批进度记录
□ 审批决定文件
□ 沟通记录

命名规范：
[审批类型]_[项目简称]_[文件类型]_[日期]
例：反垄断_XX收购_申报表_20240115.pdf
```

#### 6.2 审批数据库
```
跟踪要素：
- 审批事项名称
- 受理日期
- 预计完成日期
- 实际完成日期
- 当前状态
- 关键节点
- 问题记录
- 审批附条件
```

### 审批时间轴
```
关键里程碑：
1. 签约日 → 启动审批准备
2. T+X日 → 首批申报材料提交
3. T+Y日 → 受理/补正完成
4. T+Z日 → 审批完成
5. 交割日 → 最终截止

前置事项：
- 申报材料准备完成
- 第三方同意取得
- 先决条件满足
```

## 升级决策门

**以下情形须向律师升级：**
- 涉及国有资产转让特殊审批
- 外商投资准入存在争议
- 反垄断审查门槛临界
- 交易结构涉及跨境架构
- 存在重大潜在负债未在尽调中发现
- 交易对方存在资信风险
- 涉及特殊监管行业（金融/医药/教育）

## 本 Skill 不涵盖

- 财务审计与估值建模
- 目标公司日常经营管理
- 税务申报与税务筹划
- 劳动仲裁代理与诉讼
- 涉及境外法律的跨境问题
