## 1. 5 大法域分层(Pattern 16)

> **关键**——5 大法域不是平级——按 3 标准(业务量/监管严度/历史诉讼处罚)分层

### 1.1 5 大法域速查

| 法域 | 标识 | 法系 | 监管严度 | 默认权重 |
|------|------|------|---------|---------|
| 中国大陆 | `cn-mainland` | 大陆法系 | 高 | 必填 |
| 香港 | `hk` | 普通法系 | 中 | 选填 |
| 澳门 | `mo` | 大陆法系 | 中 | 选填 |
| 台湾 | `tw` | 大陆法系 | 中 | 选填 |
| 新加坡 | `sg` | 普通法系 | 中 | 选填 |

### 1.2 5 大法域对比

| 维度 | cn-mainland | hk | mo | tw | sg |
|------|------------|-----|-----|-----|-----|
| 司法独立 | 党管 | 独立 | 独立 | 独立 | 独立 |
| 律师制度 | 律师法 | 律师条例 | 律师条例 | 律师法 | Legal Profession Act |
| 律所形式 | 合伙/个人 | 合伙/SOLICITORS | 合伙 | 合伙 | LLP/PLLP |
| 民事时效 | 3 年 | 6 年 | 5 年 | 2 年(一般)/ 15 年(最长) | 6 年 |
| 仲裁机构 | CIETAC/BAC/SHIAC | HKIAC | — | 仲裁协会 | SIAC |
| 判决互认(对大陆) | — | 2019 安排 | 2014 安排 | 1993 协议 | 无 |
| 律师执业范围 | 中国境内 | 港内 | 澳内 | 台湾境内 | 新加坡境内 |

### 1.3 "高度关注"判定(3 标准)

> 命中任一标准即标为"高度关注"

| 标准 | 判定方法 | 例子 |
|------|---------|------|
| **业务量** | 该法域案件数 / 收入占比 | 大陆:100% 必标 |
| **监管严度** | 司法解释/法规数量 / 月度 | 大陆:高(必标) |
| **历史诉讼/处罚** | 过去 3 年被诉/被罚次数 | 多次被罚的法域 |

### 1.4 用户配置字段

```yaml
# ~/.claude/plugins/config/greater-china-legal/litigation-support/CLAUDE.md
jurisdictions: ["cn-mainland"]  # 默认
high_attention_jurisdictions: []  # 必填至少 1 个
foreign_jurisdiction: ""  # 美国/欧盟/日韩/其他
```

## 2. 法域特定升级规则(Pattern 17)

> **关键**——同一案件在不同法域,升级链路不同——按法域自动选

### 2.1 升级规则总表

| 触发器 | cn-mainland | hk / sg(普通法系) | mo / tw(大陆法系) | 涉外(美国/欧盟等) |
|--------|------------|------------------|------------------|-----------------|
| **标的 > 1000 万** | GC + 外部律师 | Senior Partner + 跨境律师 | 高级合伙人 | GC + 当地律师 + 中国跨境律师 |
| **涉外** | 法务总监 + 跨境律师 | Senior Partner + 当地律师 | 高级合伙人 + 跨境律师 | 当地律师主导 + 中国法务配合 |
| **上市公司(A 股)** | GC + 董秘 + 外部律师 | n/a | n/a | n/a |
| **上市公司(港股)** | GC + 跨境律师 | Senior Partner + 合规总监 | n/a | n/a |
| **国资/政府客户** | 法务总监 + 外部律师 + 国资委报告 | n/a | n/a | n/a |
| **涉刑事** | 法务总监 + 刑事律师 | 当地律师 + 跨境律师 | 当地律师 | 当地律师主导 |
| **海牙送达** | 跨境律师 + 当地律师 | 当地律师 | 当地律师 | 当地律师 |
| **纽约公约执行** | 跨境律师 + 当地律师 | 当地律师 | 当地律师 | 当地律师 |
| **制裁清单** | 法务总监 + 合规 + 外部律师 | Senior Partner + 合规总监 | 高级合伙人 + 合规 | 当地律师 + 中国跨境律师 |
| **数据跨境** | PIPL 38-40 + 网信办 | PDPO | PDPA | GDPR/CCPA + 中国 PIPL |

### 2.2 cn-mainland 升级 4 档

| 档位 | 阈值 | 升给 | 通道 |
|------|------|------|------|
| junior | <100 万 / 简单 | 高级律师 | 邮件 |
| senior | 100-1000 万 / 标准 | 合伙人 | 邮件+会议 |
| partner | 1000 万+ / 重大 | 管理合伙人 | 邮件+合伙人会议 |
| managing_partner | 上市公司/涉外/国资 | 律所管委会 | 邮件+管理会议 |

### 2.3 普通法系(hk / sg)升级差异

| 差异点 | cn-mainland | hk / sg |
|--------|------------|---------|
| 律师称谓 | 律师 / 主办律师 | Solicitor / Barrister |
| 出庭资格 | 律师均可 | Solicitor 不能出庭,需转聘 Barrister |
| 合伙形式 | 合伙/个人 | LLP / PLLP |
| 律师费 | 发改委指导价 | 市场调节 |
| 涉外管辖 | 中级法院 | 区域法院 / 高等法院 |
| 时效起算 | 知道或应当知道 | 知道或应当知道 + 损害发生时 |
| 证据规则 | 民诉法 + 证据规定 | 普通法证据规则(传闻证据等) |

### 2.4 涉外案件 4 步处理

1. **判断法域**——是海牙公约成员国?是纽约公约成员国?有双边司法协助?
2. **选择律师**——当地律师主导 + 中国跨境律师协同
3. **送达/取证**——按本文件 § 3 / § 4 程序
4. **判决互认**——按本文件 § 5 安排

## 4. 5 类涉外关键程序

### 4.1 海牙送达(1992 加入)

| 方式 | 适用 | 时效 |
|------|------|------|
| 中央机关请求 | 公约成员国 | 6 个月 |
| 邮寄送达 | 部分国家 | 3 个月 |
| 外交途径 | 非常规 | 不定 |
| 公告送达 | 最后手段 | 60 日 |

**中国海牙送达中央机关**:司法部

### 4.2 海牙取证(1997 加入)

- 域外取证:仅能通过中央机关请求
- 领事/特派员取证:需所在国同意

### 4.3 纽约公约(1958 / 中国 1986 加入)

- 外国仲裁裁决在中国执行:中院管辖
- 不予执行情形:纽约公约第 5 条

### 4.4 涉港判决互认(2019 修订)

- 内地与香港相互认可和执行民商事案件判决
- 适用范围:民商事 + 刑事附带民事
- 不适用:婚姻家庭(走 2017 安排) + 部分行政案件

### 4.5 涉澳判决互认(2014)

- 内地与澳门相互认可和执行民商事判决安排
- 适用范围:民商事

## 5. 跨境合规清单(涉外案件必查)

| 维度 | 中国法 | 域外法 |
|------|------|------|
| **数据** | PIPL 38-40 / 数据安全法 21 | GDPR / CCPA / PDPA / LGPD |
| **制裁** | 出口管制法 / 反外国制裁法 | OFAC / EU sanctions / UK sanctions |
| **反垄断** | 反垄断法(2022 修订,域外效力) | Sherman / Clayton / TFEU 101/102 |
| **证券** | 证券法 / 上市公司信息披露 | Securities Act 1933 / Exchange Act 1934 |
| **反洗钱** | 反洗钱法 | Bank Secrecy Act / AMLD |
| **劳动** | 劳动合同法 | FLSA / FMLA / EU Working Time |

## 6. agent 路由前必查

| 案件 | 必查项 |
|------|------|
| **涉外** | 是否海牙成员国 / 纽约公约 / 双边协议 |
| **涉港/澳** | 司法协助安排 + 适用范围 |
| **涉台** | 两岸协议 + 公证书使用 |
| **数据跨境** | PIPL 38-40 路径(安全评估/标准合同/认证) |
| **制裁** | 对方是否制裁清单(OFAC/欧盟/中国) |
| **上市公司** | 披露义务(临时公告/重大事项) |
| **政府客户** | 国资委报告 + 涉外审批 |
