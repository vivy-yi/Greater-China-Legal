# 诉讼程序概览 — 民/刑/行诉 + 仲裁 + 涉外

> 来源:scene-claudemd-curator `learn-patterns.md` § 0.6(Per-system 特殊法)
> 适用:agent 路由前判断案件类型 + 程序阶段
> 5 大程序并立(不是平级),按案件性质选

---

## 0. 程序选择决策树

```
案件类型
  │
  ├─ 民商事纠纷(合同/侵权/家事/商事)
  │   ├─ 涉外因素?
  │   │   ├─ 无 → 民事诉讼(中国法院)
  │   │   └─ 有 → 民事诉讼 + 海牙送达/纽约公约
  │   └─ 双方有仲裁协议?
  │       ├─ 有 → 仲裁(CIETAC/HKIAC/SIAC 等)
  │       └─ 无 → 民事诉讼
  │
  ├─ 行政争议(对行政处罚/许可/不作为不服)
  │   ├─ 先复议?
  │   │   ├─ 是 → 行政复议 → 不服 → 行政诉讼
  │   │   └─ 否 → 直接行政诉讼
  │   └─ 复议机关共同被告? → 行政诉讼
  │
  ├─ 刑事案件
  │   ├─ 公诉(公安侦查) → 审查起诉 → 审判
  │   └─ 自诉(亲告罪) → 直接立案
  │
  └─ 涉港/澳/台
      └─ 司法协助安排(2019 修订) + 一国两制
```

## 1. 民事诉讼程序(2023 修订民诉法)

### 1.1 一审程序

```
起诉 → 立案(7 日内) → 送达(5 日内) → 答辩(15 日内)
  → 举证(可申请延长) → 庭前会议(可不开庭) → 开庭审理
  → 判决(6 个月内,简易程序 3 个月)

注:
- 公告送达 60 日后视为送达
- 简易程序:标的< 法院辖区上年度就业人员年均工资 30% 的简单案件
- 普通程序:6 个月 + 院长批准延 6 个月
```

### 1.2 二审程序

```
上诉(判决书送达 15 日内) → 上诉状提交 → 二审立案
  → 送达 → 审理(3 个月内) → 判决/发回重审
```

### 1.3 再审程序

```
申请(判决生效 6 个月内) → 法院审查 → 裁定再审/驳回
  → 再审审理(按一审/二审程序)
```

### 1.4 关键时效

| 阶段 | 时效 | 法律依据 |
|------|------|--------|
| 起诉立案 | 7 日 | 民诉法 122 |
| 送达 | 5 日 | 民诉法 124 |
| 答辩 | 15 日 | 民诉法 128 |
| 上诉 | 15 日 | 民诉法 171 |
| 申请再审 | 6 个月 | 民诉法 216 |
| 申请执行 | 2 年 | 民诉法 246 |
| 公告送达视为送达 | 60 日 | 民诉法 95 |
| 一审普通程序 | 6 个月 | 民诉法 152 |
| 一审简易程序 | 3 个月 | 民诉法 161 |
| 二审 | 3 个月 | 民诉法 183 |

## 2. 行政诉讼程序(2017 修正行诉法)

### 2.1 起诉条件

```
1. 原告是行政行为的相对人或其他利害关系人
2. 有明确的被告
3. 有具体的诉讼请求和事实根据
4. 属于人民法院受案范围和受诉人民法院管辖
```

### 2.2 起诉期限

| 情形 | 期限 | 法律依据 |
|------|------|--------|
| 直接起诉 | 6 个月(知道作出行政行为之日起) | 行诉法 46 |
| 不知道行政行为内容 | 1 年(知道或应当知道之日起) | 行诉法 46 |
| 不作为案件 | 60 日履行期届满后 6 个月 | 行诉法 47 |
| 紧急情况 | 立即起诉 | 行诉法 48 |
| 涉外 | 6 个月(知道) | 行诉法 46 |

### 2.3 复议前置 vs 自由选择

**复议前置**:
- 纳税争议(税收征管法 88)
- 自然资源权属确认(行政复议法 30)
- 部分政府信息公开

**自由选择**:
- 一般行政处罚
- 政府信息公开
- 行政强制

### 2.4 简易程序(2015 修正新增)

- 法院 1 人独任审理
- 标的< 3 万元或事实清楚
- 45 日内结案

## 3. 刑事诉讼程序(2018 修正刑诉法)

### 3.1 三大阶段

```
侦查阶段(公安/检察院) → 起诉阶段(检察院) → 审判阶段(法院)
       │                       │                    │
       ├─ 拘留(37 日)         ├─ 审查起诉(1 个月)   ├─ 一审
       ├─ 逮捕(2 个月内)       │   可延 15 日         │   普通 2 个月
       ├─ 侦查羁押(2 个月)     │   重大复杂 +15 日    │   可延至 6 个月
       │   复杂 +1 个月        └─ 不起诉/起诉         │   简易 20 日
       └─ 移送审查起诉                                   └─ 上诉 10 日
```

### 3.2 关键时效

| 阶段 | 时效 | 法律依据 |
|------|------|--------|
| 拘留 | 37 日(一般 14+37) | 刑诉法 82/89 |
| 逮捕 | 24 小时内通知家属 | 刑诉法 93 |
| 侦查羁押 | 2 个月 | 刑诉法 154 |
| 审查起诉 | 1 个月 +15 日 | 刑诉法 172 |
| 一审(普通) | 2 个月 | 刑诉法 208 |
| 上诉 | 10 日 | 刑诉法 230 |
| 再审 | 6 个月(判决/裁定) | 刑诉法 258 |
| 申诉 | 2 年(一般) | 刑诉法 264 |

### 3.3 律师介入

- **侦查阶段**:第一次讯问或采取强制措施之日起,可委托辩护律师
- **会见权**:持"三证"(律师证/委托书/家属关系证明)48 小时内安排
- **阅卷权**:审查起诉之日起

## 4. 仲裁程序(仲裁法)

### 4.1 仲裁协议效力(仲裁法 16-18)

```
有效条件:
1. 形式:合同中仲裁条款 / 独立的仲裁协议
2. 内容:有请求仲裁的意思表示 + 仲裁事项 + 选定的仲裁委员会
3. 主体:有民事行为能力

无效情形(仲裁法 18):
- 约定不明确
- 选定的仲裁委员会不存在
- 仲裁事项超出法律规定的范围
```

### 4.2 程序

```
申请仲裁(当事人) → 受理(仲裁委 5 日内) → 组庭(3 名/1 名)
  → 答辩(15 日内) → 举证(开庭前) → 开庭审理
  → 裁决(6 个月内) → 一裁终局

注:
- 一裁终局(仲裁法 9)——不可上诉
- 撤销:6 个月内向中院申请
- 不予执行:被申请人提出 + 中院裁定
```

### 4.3 主要仲裁机构

| 机构 | 适用 | 网址 |
|------|------|------|
| CIETAC 中国国际经济贸易仲裁委员会 | 国际/国内 | cietac.org |
| BAC 北京仲裁委员会 | 国内 | bjac.org.cn |
| SHIAC 上海国际仲裁中心 | 国际/国内 | shiac.org |
| SCIA 深圳国际仲裁院 | 国际/国内 | scia.com.cn |
| HKIAC 香港国际仲裁中心 | 国际 | hkiac.org |
| SIAC 新加坡国际仲裁中心 | 国际 | siac.org.sg |
| ICC 国际商会仲裁院 | 国际 | iccwbo.org |
| LCIA 伦敦国际仲裁院 | 国际 | lcia.org |

## 5. 涉外程序(2012 民诉法 第五编 + 2023 修订)

### 5.1 涉外送达(海牙送达公约 1992 加入)

| 送达方式 | 适用 | 时效 |
|---------|------|------|
| 依国际条约(海牙公约) | 公约成员国 | 6 个月 |
| 外交途径 | 非常规 | 不定 |
| 委托使领馆 | 非常规 | 不定 |
| 邮寄送达 | 有双边协议 | 3 个月 |
| 公告送达 | 最后手段 | 60 日 |

**中国海牙送达中央机关**:司法部

### 5.2 涉外取证(海牙取证公约 1997 加入)

- 域外取证:只能通过中央机关
- 领事取证:有限
- 特派员取证:需所在国同意

### 5.3 判决互认(2019 修订司法协助安排)

- 中国大陆 ↔ 香港:2019 修订《相互执行仲裁裁决安排》 + 2019《相互认可和执行婚姻家庭民事案件判决安排》
- 中国大陆 ↔ 澳门:《相互认可和执行民商事判决安排》2014
- 中国大陆 ↔ 台湾:《两岸共同打击犯罪及司法互助协议》

### 5.4 纽约公约(承认及执行外国仲裁裁决)

- 1958 年公约
- 中国 1986 年加入
- 缔约国 170+
- 外国仲裁裁决在中国执行:中院管辖

### 5.5 反垄断域外效力

- 中国反垄断法(2022 修订)第 2 条:**域外效力**——境外垄断行为对境内市场竞争产生排除、限制影响的,适用本法
- 美国: Sherman Act / Clayton Act / FTC Act
- 欧盟: TFEU 101/102

## 6. 涉港/澳/台特殊程序

### 6.1 香港回归后司法安排

- 1999《关于内地与香港特别行政区法院相互委托送达民商事司法文书的安排》
- 2008《关于内地与香港特别行政区法院相互认可和执行当事人协议管辖的民商事案件判决的安排》
- 2017《关于内地与香港特别行政区法院相互认可和执行婚姻家庭民事案件判决的安排》
- 2019《关于内地与香港特别行政区法院相互执行仲裁裁决的安排》

### 6.2 澳门回归后

- 2001《相互认可和执行民商事判决安排》

### 6.3 台湾

- 1993《两岸公证书使用查证协议》
- 2009《两岸共同打击犯罪及司法互助协议》
