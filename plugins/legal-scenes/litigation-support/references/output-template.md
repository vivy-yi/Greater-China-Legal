## 1. Reviewer note 5 行(Pattern 9)

> agent 必写——5 行内说清"为什么有风险/建议下一步/注意什么"

```
⚠️ Reviewer note

1. 核心风险:[本案最致命的一点 + 法条 + 风险等级 🔴/🟡/🟢]
2. 证据缺口:[缺失的关键证据 + 补救路径 + 截止日]
3. 程序风险:[管辖/时效/送达等程序瑕疵 + 法条]
4. 升级建议:[是否必升 + 升给谁 + 触发条件(标的/涉外/上市公司)]
5. 下一步:[1-3 个具体动作 + 截止日 + 谁负责]
```

**5 行的深度要求**:
- 每行不超过 80 字
- 必须含法条引用(精确到条)
- 必须含具体动作(动词开头)
- 风险等级用三色(🟢/🟡/🔴)标注

## 2. Risk calibration 3 段表(Pattern 18)

> 来自 `upstream-kwp-design-analysis.md` 4g.7
> **关键洞见**——本项目当前没有"风险分级响应"机制——这是缺失的核心结构
> **3 段**:
> 1. **Usually blocks** —— 真正阻挡的事
> 2. **Usually requires work but ships** —— 妥协空间
> 3. **Usually FYI** —— 通知但不动作

### 2.1 litigation-support 专属 3 段表(初版)

> ⚠️ **TODO:本表"具体内容"应由有诉讼实践经验的律师填**——AI 生成的默认值可能不准确

| 段 | 模式 | 为什么 | 解决路径 |
|---|------|------|--------|
| **Usually blocks** | 诉讼时效已过 | 民诉法 188——必须提"时效抗辩",否则视为放弃 | 申请驳回起诉 |
| | 虚假诉讼 | 民诉法 115——罚款/拘留/刑事责任 | 立即停止 + 主动告知违法 |
| | 管辖违法 | 民诉法 127——必须提"管辖异议",否则视为接受管辖 | 提管辖异议 |
| | 律师伪造证据 | 律师法 49——吊销执照 | 立即停止 |
| | 教唆伪证 | 刑诉法 307——伪证罪 | 立即停止 + 报告 |
| | 妨害司法 | 刑诉法 308——贿赂/胁迫证人 | 立即停止 + 报告 |
| | 标的> 1000 万 + 涉外 | GC + 跨境律师必须复核 | 升级 managing_partner |
| | 上市公司重大未披露事项 | 证券法——刑事风险 | 立即暂停 + 报告 GC/董秘 |
| **Usually requires work but ships** | 证据组织不充分 | 缺部分证据但可补救 | 1-2 周补证 |
| | 管辖异议准备不充分 | 异议可能被驳回 | 备选方案:移送 |
| | 反诉请求未完整 | 部分请求需另案起诉 | 评估后决定 |
| | 涉外送达周期长 | 6 个月+ | 同步推进调解 |
| | 多个被告中部分失联 | 公告送达延宕 | 评估分批审理 |
| | 鉴定/评估争议 | 双方对鉴定结论分歧 | 申请重新鉴定 |
| | 律师费报价超出预算 | 客户预算紧张 | 调整代理范围 |
| **Usually FYI** | 新司法解释生效但暂不适用 | 不在业务范围 | 关注扩展 |
| | 同行被处罚但本公司未涉 | 行业警示 | 修订内控 |
| | 行业自律规则更新 | 不具强制力 | 跟踪 |
| | 域外判例 / 学术评论 | 不具约束力 | 参考 |
| | 法官个人偏好(调研) | 审判风格 | 庭审策略微调 |
| | 最高法案例汇编更新 | 不直接约束 | 引用参考 |

### 2.2 使用规则

| 段 | agent 动作 |
|---|------|
| **blocks** | 立即停止 + 告知用户 + 给出解决路径——**不绕过** |
| **requires work but ships** | 提示律师 + 给时间表 + 不阻塞——**不立即停止** |
| **FYI** | 记录到 matter.md 的"风险提示"段——**不主动告知** |

### 2.3 与 5 类必升的关系

| 必升触发器 | 落点 |
|---------|------|
| 标的 > 1000 万 | 升级 managing_partner(在 blocks 段) |
| 涉外 | 升级跨境律师(在 blocks 段) |
| 上市公司 | 升级 GC + 董秘(在 blocks 段) |
| 国资/政府客户 | 升级法务总监(在 blocks 段) |
| 涉刑事 | 升级刑事律师(在 blocks 段) |

### 2.4 持续学习

- 每季度根据新司法解释/法规更新本表
- 律师反馈"agent 误判"→ 调整本表分类
- 跨 scene 通用 pattern(数据跨境/未披露事项)抽到 `learn-patterns.md` § 0
