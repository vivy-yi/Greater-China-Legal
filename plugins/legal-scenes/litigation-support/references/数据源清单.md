## 三、案例库（诉讼场景核心）

### 3.1 最高人民法院指导案例

```
URL: 人民法院案例库 https://rmfyalk.court.gov.cn
[Y D] 检索："指导案例 + 编号"

诉讼必学指导案例：
- 程序类：立案管辖、保全、执行
- 证据类：举证责任、证据规则
- 实体类：合同、侵权、知识产权
- 执行类：执行异议、追加被执行人
```

### 3.2 最高人民法院公报案例

```
URL: 最高人民法院公报
[Y D] 检索："公报 + 案例"

特色：最高法认为具有指导意义的案例，每季度发布
```

### 3.3 经典案例

```
[Y D] 检索："经典案例 + 案由"

类型：
- 合同纠纷
- 侵权责任
- 公司纠纷
- 知识产权
- 婚姻家庭
- 行政诉讼
- 刑事案件
```

### 3.4 类案检索方法

```
Step 1：识别争议焦点
Step 2：检索类似案例
  - 关键词：争议焦点 + 法院层级
  - 时间：近年优先
Step 3：分析裁判规则
  - 同案同判
  - 上级法院指导
Step 4：撰写法律意见
  - 类案参考
  - 风险提示
```

## 十一、来源标注规范

| 标注 | 含义 | 诉讼场景示例 |
|---|---|---|
| `[YD]` | 元典检索结果 | `[YD] 民诉法解释第 XX 条` |
| `[WKL]` | 无合力检索结果 | `[WKL] 类案 XX 号` |

| `[GOV]` | 政府平台数据 | `[GOV] 法释〔2022〕11 号` |
| `[web]` | 网络搜索结果 | `[web] 最高法 2024 公告` |
| `[model]` | 模型知识（须核实） | 不允许直接引用 |
| `[user]` | 用户提供的文件 | `[user] 客户提供的合同` |
| `[settled — last confirmed YYYY-MM-DD]` | 已验证的稳定引用 | 已律师审核 |

**诉讼场景特殊规则**：
- 司法解释引用：`[GOV] 法释〔年份〕XX 号`
