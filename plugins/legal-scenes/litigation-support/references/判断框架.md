# 诉讼支持判断框架 — 中国大陆

> 本框架用于 litigation-support 35 个 skill 的场景识别与判断逻辑，作为诉讼场景的统一依据。
> litigation-support 是**诉讼全流程支持**场景，覆盖从案件接收到结案/执行的整个律师工作流。
> 各 skill 内部已有判断逻辑，本文件描述跨 skill 的整体决策路由。

## 三、诉讼时效判断（statute-of-limitations）

### 3.1 普通诉讼时效

```
民法典第 188 条：3 年普通诉讼时效
起算点：权利人知道或应当知道权利受到损害以及义务人之日
```

### 3.2 特殊诉讼时效

| 案件类型 | 时效 | 法条 |
|---|---|---|
| 国际货物买卖合同 | 4 年 | 民法典第 594 条 |
| 技术进出口合同 | 4 年 | 民法典第 594 条 |
| 人寿保险 | 5 年 | 保险法第 26 条 |
| 海商请求 | 1 年（货损货差）/ 2 年（其他） | 海商法第 257-260 条 |
| 环境污染损害 | 3 年 | 环境保护法第 66 条 |
| 撤销权（民事） | 1 年（重大误解）/ 90 日（胁迫） | 民法典第 152 条 |
| 撤销权（民事，最长） | 5 年 | 民法典第 152 条 |
| 行政复议 | 60 日 | 行政复议法第 60 条 |
| 行政诉讼 | 6 个月 | 行政诉讼法第 46 条 |
| 劳动仲裁 | 1 年 | 劳动争议调解仲裁法第 27 条 |
| 专利侵权 | 3 年 | 专利法第 71 条 |
| 商标侵权 | 3 年 | 商标法第 60 条 |
| 著作权侵权 | 3 年 | 著作权法第 60 条 |
| 人身损害 | 3 年 | 民法典第 188 条 |

### 3.3 时效中止/中断

```
[节点 1] 中断事由（民法典第 195 条）
├─ 权利人向义务人提出履行请求
├─ 义务人同意履行
├─ 权利人提起诉讼/申请仲裁
├─ 与起诉/仲裁具有同等效力的其他情形
└─ 中断后时效重新计算

[节点 2] 中止事由（民法典第 194 条）
├─ 不可抗力
├─ 无/限制民事行为能力人无代理人
├─ 继承开始后未确定继承人/遗产管理人
├─ 权利人被义务人/他人控制
└─ 时效期间最后 6 个月内

[节点 3] 重新起算
├─ 中断：从中断时重新计算
└─ 中止：消除障碍后 6 个月内

升级触发：
- 时效临近届满 → 立即起诉/仲裁
- 时效已过 → 律师评估剩余空间
```

## 四、案由分析（cause-action-analysis）

### 4.1 民事案由（按 GB/T 案由体系）

```
大类：人格权纠纷、婚姻家庭继承纠纷、物权纠纷、合同纠纷、知识产权与竞争纠纷、
      劳动争议、人事争议、海事海商纠纷、与公司有关的纠纷、证券纠纷、期货交易
      纠纷、信托纠纷、保险纠纷、票据纠纷、民事特别程序案件
```

### 4.2 案由判断步骤

```
Step 1：争议法律关系性质
  ├─ 人格权 → 人格权纠纷
  ├─ 婚姻家庭 → 婚姻家庭纠纷
  ├─ 物权（所有权/用益物权/担保物权/占有）
  ├─ 合同（合同性质）
  ├─ 知识产权与竞争
  ├─ 劳动人事
  └─ 特殊（海事/证券/期货/信托/保险）

Step 2：合同性质细分
  ├─ 买卖合同纠纷
  ├─ 借款合同纠纷
  ├─ 租赁合同纠纷
  ├─ 建设工程合同纠纷
  ├─ 承揽合同纠纷
  ├─ 运输合同纠纷
  ├─ 仓储合同纠纷
  ├─ 委托合同纠纷
  ├─ 行纪合同纠纷
  ├─ 居间合同纠纷
  ├─ 担保合同纠纷
  └─ ...

Step 3：选择案由
  - 单一法律关系 → 单案由
  - 多个法律关系 → 多个案由（主从）
  - 不确定 → 律师 + 法官确认
```

### 4.3 案由选择不当后果

```
后果：
├─ 影响管辖（不同案由管辖不同）
├─ 影响诉讼费（按标的额 vs 按件）
├─ 影响审理思路
└─ 可能被驳回起诉（案由错误）
```

## 五、管辖判断

### 5.1 民事诉讼管辖

```
[节点 1] 级别管辖（民事诉讼法第 17-20 条）
├─ 基层法院 → 一般案件
├─ 中级法院 → 重大涉外/本辖区有重大影响
├─ 高级法院 → 本辖区有重大影响
└─ 最高法院 → 全国有重大影响

[节点 2] 地域管辖（民事诉讼法第 23-35 条）
├─ 一般地域 → 被告住所地
├─ 合同 → 被告住所地 + 合同履行地
├─ 侵权 → 被告住所地 + 侵权行为地
├─ 不动产 → 不动产所在地
├─ 公司诉讼 → 公司住所地
├─ 票据 → 票据支付地
└─ 协议管辖 → 合同约定

[节点 3] 专属管辖
├─ 不动产纠纷
├─ 港口作业纠纷
├─ 遗产继承纠纷（不动产部分）
└─ 中外合资/合作企业合同

[节点 4] 协议管辖 + 应诉管辖
├─ 协议管辖：合同明确约定（与争议有实际联系）
├─ 应诉管辖：未提异议应诉
└─ 无效：违反级别/专属管辖

升级触发：
- 涉外管辖 → 律师专项
- 平行诉讼 → 律师专项
- 管辖异议 → 律师专项
```

### 5.2 行政诉讼管辖

```
[节点 1] 级别管辖（行政诉讼法第 14-16 条）
├─ 基层 → 一般
├─ 中级 → 国务院部门/县级以上政府 + 海关 + 知识产权
├─ 高级 → 本辖区重大
└─ 最高 → 全国重大

[节点 2] 地域管辖
├─ 最初作出行政行为的行政机关所在地
└─ 经复议的案件 → 复议机关所在地

[节点 3] 经复议案件的特别规定
├─ 复议维持 → 原行政机关 + 复议机关共同被告
├─ 复议改变 → 复议机关被告
└─ 复议机关不作 → 原行政机关
```

### 5.3 劳动仲裁管辖

```
劳动合同履行地 / 用人单位所在地 仲裁委员会
（两个以上仲裁委员会都有管辖权 → 优先选择合同履行地）
```

## 六、证据规则

### 6.1 证据类型

```
[节点 1] 当事人陈述
[节点 2] 书证
[节点 3] 物证
[节点 4] 视听资料
[节点 5] 电子数据
[节点 6] 证人证言
[节点 7] 鉴定意见
[节点 8] 勘验笔录
```

### 6.2 举证责任

```
[节点 1] 一般原则（民事诉讼法第 67 条）
  - 谁主张谁举证

[节点 2] 特殊举证责任倒置
  ├─ 高度危险作业致害
  ├─ 环境污染致害
  ├─ 产品质量致害
  ├─ 医疗损害
  ├─ 饲养动物致害
  └─ 物件脱落/倒塌致害

[节点 3] 举证不能后果
  └─ 主张事实不成立（败诉风险）
```

### 6.3 证据收集方式

```
[节点 1] 自行收集
  - 合同/票据/记录
  - 公开信息
  - 现场拍照录像

[节点 2] 律师调查令
  - 法院签发
  - 律师持令调查
  - 范围限制

[节点 3] 法院调取
  - 依申请
  - 依职权
  - 国家秘密/商业秘密/个人隐私

[节点 4] 公证
  - 网络证据公证
  - 行为公证
  - 证据保全公证

[节点 5] 鉴定
  - 申请
  - 法院指定
  - 重新鉴定

升级触发：
- 涉外取证 → 律师专项
- 商业秘密取证 → 律师专项
- 鉴定争议 → 律师 + 鉴定人专项
```

## 七、保全（legal-hold / coercive-measure-assessment）

### 7.1 证据保全

```
[节点 1] 申请条件
├─ 证据可能灭失/难以取得
├─ 申请时间：举证期限届满前 7 日（书面申请）
└─ 法院裁定

[节点 2] 保全方式
├─ 查封
├─ 扣押
├─ 冻结
├─ 鉴定
├─ 勘验
└─ 制作笔录
```

### 7.2 财产保全

```
[节点 1] 诉前财产保全（民诉法 104 条）
├─ 情况紧急
├─ 48 小时内裁定
├─ 30 日内提起诉讼
└─ 担保（一般相当于请求数额）

[节点 2] 诉讼中财产保全
├─ 可能因对方行为/其他原因使判决难以执行
├─ 担保（一般相当于请求数额）
└─ 法院裁定

[节点 3] 担保方式
├─ 现金
├─ 银行保函
├─ 保险
├─ 不动产
└─ 股权
```

### 7.3 行为保全（民诉法 100-101 条）

```
[节点 1] 一般行为保全
├─ 对方行为可能使判决难以执行
└─ 法院裁定

[节点 2] 知识产权行为保全
├─ 侵权行为
├─ 即时制止
└─ 申请人提供担保
```

## 八、强制措施（coercive-measure-assessment）

### 8.1 民事强制措施

```
[节点 1] 妨害民事诉讼的强制措施
├─ 拘传（必须到庭的被告）
├─ 训诫
├─ 罚款（个人 10 万以下/单位 5 万-100 万）
├─ 拘留（15 日内）
└─ 限制高消费/限制出境

[节点 2] 刑事强制措施
├─ 拘传
├─ 取保候审
├─ 监视居住
├─ 逮捕
└─ 拘留
```

### 8.2 强制措施适用

```
[节点 1] 主体
├─ 当事人
├─ 诉讼参与人
├─ 案外人
└─ 协助执行人

[节点 2] 情形
├─ 妨害民事诉讼秩序
├─ 拒不履行义务
├─ 转移财产
├─ 暴力抗拒执行
└─ 虚假诉讼
```

## 九、执行（foreclosure-auction-advisor / execution-obstacle-removal）

### 9.1 申请执行

```
[节点 1] 申请条件
├─ 生效法律文书
├─ 履行期届满
├─ 对方未履行
└─ 2 年内申请执行

[节点 2] 执行管辖
├─ 第一审法院/与第一审法院同级的被执行财产所在地
└─ 仲裁裁决 → 仲裁委员会所在地中级法院
```

### 9.2 财产线索（asset-search-strategy）

```
[节点 1] 不动产
├─ 房管所
├─ 国土局
└─ 银行按揭信息

[节点 2] 动产
├─ 车辆管理所
├─ 工商登记
└─ 实地调查

[节点 3] 银行存款
├─ 网络查询（律师）
├─ 法院查询
└─ 调查令

[节点 4] 股权
├─ 国家企业信用信息公示
├─ 证券登记结算
└─ 工商登记

[节点 5] 债权/应收账款
├─ 合同
├─ 银行流水
└─ 业务往来

[节点 6] 知识产权
├─ 专利/商标/著作权登记
└─ 评估价值
```

### 9.3 执行措施

```
[节点 1] 财产报告令
[节点 2] 查封/扣押/冻结
[节点 3] 拍卖/变卖
[节点 4] 扣留/提取被执行人收入
[节点 5] 搜查
[节点 6] 强制迁出房屋/退出土地
[节点 7] 强制办理财产权证照转移手续
[节点 8] 限制高消费
[节点 9] 限制出境
[节点 10] 信用惩戒
```

### 9.4 排除执行障碍（execution-obstacle-removal）

```
[节点 1] 案外人异议
├─ 异议 → 异议审查 → 异议之诉
└─ 案外人主张权利

[节点 2] 执行复议
├─ 向上一级法院申请复议
├─ 10 日内决定
└─ 复议期间不停止执行（一般）

[节点 3] 执行异议之诉
├─ 案外人
├─ 申请执行人
└─ 被执行人

[节点 4] 申请追加被执行人
├─ 股东出资不实
├─ 抽逃出资
├─ 未经清算即注销
├─ 一人公司财产混同
└─ 第三人书面承诺承担债务
```

## 十、行政复议与诉讼

### 10.1 行政复议

```
[节点 1] 受案范围（行政复议法第 11 条）
├─ 行政处罚
├─ 行政强制
├─ 行政许可
├─ 行政确认
├─ 行政征收/征用/补偿
├─ 行政裁决
├─ 未履行职责
└─ 其他侵犯合法权益的具体行政行为

[节点 2] 不予受理
├─ 国防外交
├─ 行政法规/规章
├─ 内部人事处理
├─ 对民事纠纷调解
└─ 刑事司法行为

[节点 3] 复议机关
├─ 上一级主管部门
├─ 本级政府
├─ 国务院部门 → 国务院或部门
└─ 派出机构 → 设立机构

[节点 4] 时效
├─ 60 日内申请复议
└─ 60 日内作出决定（可延长 30 日）
```

### 10.2 行政诉讼

```
[节点 1] 受案范围
├─ 行政处罚
├─ 行政强制
├─ 行政许可
├─ 行政确认
├─ 行政征收
└─ 其他具体行政行为

[节点 2] 复议前置 vs 自由选择
├─ 复议前置（自然资源权属、纳税、海关、价格等）
├─ 自由选择（一般行政行为）
└─ 复议终局（少数）

[节点 3] 起诉条件
├─ 原告是行政相对人或利害关系人
├─ 有明确的被告
├─ 有具体的诉讼请求和事实根据
├─ 属于法院受案范围和受诉法院管辖
└─ 时效内（6 个月）

升级触发：
- 涉外行政诉讼 → 律师专项
- 公益诉讼 → 律师 + 检察院
- 重大行政案件 → 律师专项
```

## 十六、与其他 plugin 的差异

| 维度 | litigation-support | corporate-governance | ip-infringement |
|---|---|---|---|
| 关注重点 | 诉讼全流程支持 | 公司治理 | IP 侵权 |
| 输出类型 | 文书 + 策略 + 案件管理 | 治理方案 | 侵权认定 |
| 数据需求 | 司法解释 + 案例 + 法规 | 工商 + 公司法 | 专利商标数据库 |
| 时效要求 | **极高**（时效期限） | 低 | 中 |
| 风险等级 | 高 | 中-高 | 高 |
| 涉外因素 | 偶尔 | 偶尔 | 频繁 |

**核心差异**：litigation-support 是**律师工作流**视角（35 skill 覆盖律师日常工作），与公司治理/合同/IP 场景有显著差异。**时效要求最高**——任何错误都可能导致客户败诉。
