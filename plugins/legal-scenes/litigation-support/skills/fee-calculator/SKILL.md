# /fee-calculator — 诉讼费用计算

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取费用标准/风险代理规则。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取诉讼费标准/律师费规则
2. 计算诉讼费
3. 计算其他费用（保全费/鉴定费等）
4. 评估律师费
5. 输出费用清单

## 诉讼费计算

### 财产案件

| 标的金额 | 费率 | 速算增加数 |
|---|---|---|
| 1万元以下 | 3% | — |
| 1-10万元 | 2.5% | -50元 |
| 10-20万元 | 2% | +300元 |
| 20-50万元 | 1.5% | +1300元 |
| 50-100万元 | 1% | +3800元 |
| 100-200万元 | 0.9% | +4800元 |
| 200-500万元 | 0.8% | +6800元 |
| 500-1000万元 | 0.7% | +11800元 |
| 1000-2000万元 | 0.6% | +21800元 |
| >2000万元 | 0.5% | +41800元 |
| 上限 | — | 50万元 |

### 计算公式

```
诉讼费 = 标的金额 × 费率 + 速算增加数
```

### 非财产案件

| 案件类型 | 诉讼费 |
|---|---|
| 离婚案件 | 50-300元（涉及财产分割按财产案件） |
| 人格权纠纷 | 100-500元 |
| 其他非财产案件 | 50-100元 |

### 劳动争议案件

```
仲裁阶段：不收费
一审：10元（简易程序减半收取）
```

## 其他费用

| 费用类型 | 标准 |
|---|---|
| 保全费 | 标的0.5%，最高5000元 |
| 鉴定费 | 实际发生，协商确定 |
| 公证费 | 实际发生，按公证处标准 |
| 公告费 | 实际发生，按公告机构标准 |
| 翻译费 | 实际发生 |

## 律师费

### 风险代理限制

```
风险代理上限（国家发改委/司法部规定）：
- 10万元以下部分：≤18%
- 10-50万元部分：≤15%
- 50-100万元部分：≤12%
- 100-500万元部分：≤9%
- 500万元以上部分：≤6%

注：工伤赔偿/劳动仲裁不得风险代理
```

### 律师费参考

| 案件类型 | 简单案件 | 复杂案件 |
|---|---|---|
| 合同纠纷 | 1-3万 | 5-20万 |
| 知识产权 | 2-5万 | 10-50万 |
| 劳动争议 | 0.5-2万 | 3-10万 |

## 费用减免

```
诉讼费减免条件：
- 低保户/特困户 → 可申请免交
- 残疾人无固定收入 → 可申请减交

律师费减免：
- 法律援助受援人 → 免费
- 经济困难证明 → 可申请援助
```
