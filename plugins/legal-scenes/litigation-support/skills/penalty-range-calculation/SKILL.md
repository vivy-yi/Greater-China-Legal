# 行政处罚罚款幅度计算

## 目的

计算行政处罚的罚款幅度，辅助评估罚款区间和减轻策略。覆盖：法定罚款幅度查找、从重减轻情节评估、裁量基准表参照、听证和复议权利提示。本技能不代理听证或复议，但辅助准备材料。

## 法域假设

默认中国大陆法域，涉及《行政处罚法》和各行业监管法规（市场监督管理/税务/环保/安全生产等）`[SME 核查]`。

## 加载信息

- 违规类型和涉及的具体法条
- 违法行为描述和持续时间
- 涉及金额（违法所得/经营额/货值等）
- 当事人情况（初犯/累犯/主动消除后果等）

## 工作流程

### 第一步：查找法定罚款幅度

根据违规类型，查找对应的法定罚款幅度 `[SME 核查]`：

| 违规类型 | 常见罚款幅度 |
|---------|------------|
| 市场监管（广告法）| 广告费用×3-5倍 / 经营额×1%-10% |
| 税务（偷逃税款）| 欠缴税款×50%-5倍 |
| 环保（超标排污）| 10万-100万 / 责令停产整顿 |
| 安全生产（违规）| 1万-10万 / 责任人5000-2万 |
| 食品安全 | 货值×5-10倍 / 吊销许可证 |

### 第二步：评估从重/减轻情节

| 情节类型 | 对罚款的影响 |
|---------|------------|
| 初次违法 | 可从轻或减轻 |
| 主动消除或减轻后果 | 可从轻或减轻 |
| 配合调查如实陈述 | 可从轻 |
| 造成人身伤害/重大财产损失 | 从重 |
| 累犯/拒不改正 | 从重 |
| 涉及婴幼儿/食品安全等 | 从重 |

### 第三步：计算罚款区间

```
法定罚款幅度：[最小值] - [最大值]
适用计算基数：[违法所得/经营额/货值]
从重/减轻系数：[___]
建议罚款区间：[___] - [___]
```

### 第四步：听证和复议权利 `[SME 核查]`

- **听证权利**：罚款金额达到听证标准的（通常较大金额），当事人有权申请听证
- **复议权利**：收到处罚决定书后60日内可申请行政复议
- **诉讼权利**：复议不满意可进一步提起行政诉讼

### 第五步：升级决策门

> "行政处罚的计算涉及法定幅度和裁量基准的具体适用，最终罚款金额取决于执法机关的裁量。申请听证、复议或诉讼时，建议由行政法专业律师代理。这不构成法律意见。"
