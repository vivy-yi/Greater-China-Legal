### § A8.1 法学院学习特别注意 4 大块（IRAC 方法 + 法考 + 论文 + 学说）

> **核心原则**：法学学习的核心是**请求权基础思维 + 规范分析方法 + 学术诚实**。任何结论都必须回到法条 + 判例 + 学说。

#### 块 1：IRAC 案例分析方法（法学思维基础）

**IRAC = Issue + Rule + Application + Conclusion**

| 步骤 | 内容 | 示例 |
|------|------|------|
| **I（Issue 争议点）** | 识别法律问题 | 合同是否成立？违约责任？ |
| **R（Rule 法律规则）** | 适用法律规则 | 《民法典》第 490 条 / 司法解释 |
| **A（Application 法律适用）** | 事实对应法律 | 合同签字 + 实际履行 = 成立 |
| **C（Conclusion 结论）** | 得出结论 | 合同成立 + 违约责任 |

**进阶方法（IRPA / FIRAC）：**

| 方法 | 含义 |
|------|------|
| **CREAC（Conclusion / Rule / Explanation / Application / Conclusion）** | 中国法学教育主流 |
| **IRPA（Issue / Rule / Policy / Application）** | 加入政策考量 |
| **FIRAC（Facts / Issue / Rule / Application / Conclusion）** | 先陈述事实 |

**blocks（绝对禁止 — 案例分析）**：
- 跳过法条直接结论 → **零分**
- 捏造判例 / 法条 → **学术不端 + 零分**
- 结论与论证矛盾 → **不及格**

**work but ships（可补救）**：
- 论证不充分 → 补充 + 引用判例
- 结论偏颇 → 重写 + 多角度分析

#### 块 2：法考（法律职业资格考试）核心要点

**法考结构（2025 改革后）：**

| 阶段 | 科目 | 形式 |
|------|------|------|
| **客观题** | 18 科（18 选 18） | 200 题 + 单选 50 / 多选 50 / 不定项 50 |
| **主观题** | 5 选 1 + 选做题 | 案例分析 + 法律文书 |

**法考 18 科核心分布：**

| 学科 | 客观题占比 | 主观题可能 |
|------|----------|----------|
| **民法** | 高（30+ 分） | 必考 |
| **刑法** | 高（30+ 分） | 必考 |
| **行政法** | 中（20+ 分） | 可能 |
| **民诉 / 刑诉** | 中（各 20+ 分） | 可能 |
| **商法 + 经济法 + 知产** | 中（合计 30+ 分） | 必考（商法） |
| **理论法 + 宪法 + 司法制度** | 中（合计 30+ 分） | — |
| **三国法（国际公私法 + 国际经济法）** | 低（各 10 分） | 选考 |

**主观题 5 选 1（10 大题型）：**

| 题型 | 核心考点 |
|------|---------|
| **民法** | 合同 / 物权 / 侵权综合 |
| **刑法** | 共同犯罪 / 罪数 / 量刑情节 |
| **行政法** | 行政诉讼 / 复议 |
| **商法** | 公司法 / 合伙 / 票据 |
| **民诉** | 第三人撤销之诉 / 再审 / 执行异议 |
| **刑诉** | 非法证据排除 / 证人出庭 |
| **理论法** | 法治理念 + 司法制度 |
| **三国法** | 国际私法 / 国际商事仲裁 |

**blocks（绝对禁止 — 法考）**：
- 法条错误引用 → **零分**
- 主观题少于 500 字 → **不及格**
- 考试作弊 → **禁考 2 年 + 记录**

**work but ships（可补救）**：
- 客观题错题 → 30 日内复盘
- 主观题超时 → 限时训练

#### 块 3：法学论文写作规范（学术规范）

**论文结构（本科 + 研究生通用）：**

| 部分 | 字数占比 | 关键 |
|------|---------|------|
| **摘要** | 5% | 300-500 字 + 关键词 |
| **引言 / 问题的提出** | 10% | 研究问题 + 意义 + 综述 |
| **文献综述** | 15% | 中外文献 + 综述 + 评析 |
| **正文** | 50% | 论证 + 法条 + 判例 + 学说 |
| **结论** | 10% | 总结 + 创新 + 局限 |
| **参考文献** | 5% | 规范格式 |
| **注释 + 致谢** | 5% | — |

**学术规范核心：**

| 规范 | 要求 |
|------|------|
| **查重率** | 本科 ≤ 20% / 硕士 ≤ 15% / 博士 ≤ 10% |
| **引注** | GB/T 7714 / 法学引注规范（脚注 + 文末） |
| **判例引用** | 案号格式（YYYY）法院代码案由代码第N号 |
| **法条引用** | 必须引第N条 + 版本（如"《民法典》2021 第 490 条"） |
| **AI 使用披露** | 部分高校要求披露 AI 辅助 |

**blocks（绝对禁止 — 论文）**：
- 抄袭 / 剽窃 → **学位撤销 + 学术处分**
- 伪造数据 → **学术处分 + 法律责任**
- 伪造引注 → **学术处分**
- 重复发表 → **学术处分**

**work but ships（可补救）**：
- 查重率超标 → 重写 + 改写
- 引注不规范 → 重新引用 + 校对
- 逻辑不严 → 重新组织结构

#### 块 4：法学学说 + 通说 / 少数说

**中国法学常见学说流派：**

| 学科 | 通说 | 少数说 |
|------|------|--------|
| **民法** | 请求权基础 / 法律关系分析 | 关系合同 / 利益衡量 |
| **刑法** | 客观主义（结果无价值） | 主观主义（行为无价值） |
| **行政法** | 控权论 / 平衡论 | 管理论 / 公共利益论 |
| **法理学** | 马克思主义法学 | 社科法学 / 法教义学 |
| **宪法** | 根本法 / 最高法 | 政治宪法学 / 规范宪法学 |

**学说适用规则：**

| 场景 | 处理 |
|------|------|
| **本科** | 介绍通说 + 提及少数说 |
| **硕士** | 深入评析各学说 + 倾向 |
| **博士** | 提出新学说 + 论证 |
| **法考主观题** | 以通说为准 + 必要时提及 |

**blocks（绝对禁止 — 学说）**：
- 混淆学说与现行法 → **考试失分**
- 否定通说无充分论证 → **学术不当**

**work but ships（可补救）**：
- 学说引用不全 → 补充中外学说
- 论述片面 → 多角度对比

#### 块 5：4 大绝对禁止（含具体后果）

| 禁止 | 后果 |
|------|------|
| 1. **考试作弊 / 学术不端** | 学位撤销 + 禁考 + 记录 |
| 2. **伪造引注 / 抄袭剽窃** | 学位撤销 + 学术处分 |
| 3. **捏造判例 / 法条** | 考试零分 + 学术处分 |
| 4. **AI 辅助未披露** | 部分高校学术处分 |

### § B17 跨场景协作（脱敏钩子）

法学院学习辅助业务节奏与脱敏/还原联动：

```
案例改编 → legal-document-redaction（脱敏公开案例库 + 比对文件）
         → 公开教学案例库 / 论文引用
         → 收回教师批改
         → legal-document-restoration（runs 级还原 + round-trip 校验）
         → 入教学档案
```

- **触发时机**：真实案例改编为教学案例、学术论文引用案件时
- **输出用途**：教学案例版 / 论文附录版 / 公开判例库版
- **配置**：调用前 Read `legal-document-redaction/references/config.md`，当事人姓名细粒度+可逆（保留映射表供教师批改）
- **特别注意**：未成年人/性侵/家事案件特殊保护，不可进入公开案例库
