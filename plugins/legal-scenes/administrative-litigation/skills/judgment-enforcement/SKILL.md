# /judgment-enforcement — 行政诉讼判决执行

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B13 Enforcement posture + § B3 风险等级 + § A5 决策树。

## 工作流程

```
Step 1: 确认判决生效(生效日)
Step 2: 评估履行内容(撤销 / 重做 / 履行 / 赔偿 / 变更)
Step 3: 督促被告履行(3 个月内)
Step 4: 不履行应对(申请法院强制执行)
Step 5: 评估国家赔偿(若涉损害)
Step 6: 跟踪履行结果
```

## 履行期限(《行政诉讼法》第 96 条)

**核心期限:**

```
第一审判决 / 裁定:
  被告(行政机关)应当自判决书 / 裁定书送达之日起 15 日内履行
  (若涉及重做具体行政行为) → 3 个月内重新作出

二审 / 终审判决:
  自判决书送达之日起生效
  履行期限同上
```

**3 个月重做(《行政诉讼法》第 71 条):**

```
✅ 撤销判决生效后,行政机关可以依照法定职权重新作出行政行为
✅ 期限:3 个月内
✅ 重新作出的行为不得以同一事实和理由作出与原行为基本相同的行政行为
```

## 履行类型 5 类

| 判决类型 | 履行内容 |
|----------|----------|
| **撤销判决** | 停止执行 + 必要时重做(3 个月内) |
| **确认违法判决** | 停止执行 + 后续补救(恢复原状 / 赔偿) |
| **责令履行判决** | 行政机关在期限内履行法定职责 |
| **变更判决**(显失公正) | 按变更内容执行 |
| **赔偿判决** | 在期限内支付赔偿金 |

## 履行督促 3 步

**Step 1: 提醒被告履行(判决生效后 15 日内)**

```
方式:
  - 律师函
  - 当面沟通
  - 书面通知

内容:
  - 判决主要内容
  - 履行期限
  - 不履行后果
```

**Step 2: 跟进(15-30 日内)**

```
- 与被告执行部门沟通
- 确认履行进展
- 评估是否存在履行困难
```

**Step 3: 必要时申请强制执行(超过履行期限)**

```
- 向第一审法院申请
- 申请材料:
  - 申请执行书
  - 判决书 / 裁定书
  - 被告未履行证明
```

## 不履行应对(《行政诉讼法》第 96 条第 2 款)

**行政机关拒绝履行:**

| 措施 | 内容 |
|------|------|
| **罚款(单位)** | 对单位罚款(通常 ¥5万-100万) |
| **拘留(责任人)** | 对直接负责的主管人员和其他直接责任人员司法拘留(15 日内) |
| **公告** | 法院可在报刊 / 媒体上公告 |
| **向上级机关 / 监察机关通报** | 移送监察机关处理 |
| **继续履行** | 法院可继续强制履行 |
| **赔偿衔接** | 原告可申请国家赔偿 |

**具体罚款标准(《高法解释》):**

| 情形 | 罚款金额 |
|------|----------|
| 拒绝履行判决 | 单位 ¥5万-100万,个人可拘留 |
| 拒不履行赔偿判决 | 单位 ¥5万-100万 |
| 伪造证据 / 阻碍执行 | 单位可罚款,个人可拘留 |
| 暴力抗法 | 追究刑事责任 |

**向上级机关通报(《行政诉讼法》第 96 条):**

```
✅ 法院可向被告的上一级行政机关 / 监察机关提出司法建议
✅ 建议给予被告负责人 / 直接责任人员行政处分
✅ 情节严重的,追究刑事责任
```

## 履行困难情形

**行政机关"无法履行":**

| 情形 | 处理 |
|------|------|
| 撤销后无法恢复原状 | 国家赔偿(替代履行) |
| 责令履行但客观不能 | 终止 + 国家赔偿 |
| 法律法规变更 | 重新审查后处理 |
| 行政机关已被撤销 | 继续行使职权的机关履行 |

**特别注意:**

```
⚠️ 客观不能履行 ≠ 拒绝履行
⚠️ 客观不能履行 → 须提供证明 + 法院认定
⚠️ 拒绝履行 → 直接罚款 / 拘留
```

## 国家赔偿衔接(《行政诉讼法》第 76 条)

**可一并提起国家赔偿的情形:**

| 情形 | 赔偿 |
|------|------|
| 行政行为被撤销 / 确认违法 | 造成损害的可申请赔偿 |
| 行政不作为被责令履行 | 造成损害的可申请赔偿 |
| 行政处罚被变更(显失公正) | 多出部分可申请赔偿 |

**赔偿方式:**

| 类型 | 标准 |
|------|------|
| 财产损害 | 实际损失 |
| 人身自由损害 | 限制自由期间误工费 / 精神损害 |
| 人身损害 | 医疗费 / 误工费 / 残疾赔偿金 |
| 精神损害 | 司法解释(¥5万以内为主) |
| 死亡 | 死亡赔偿金 + 丧葬费 |

**申请方式:**

```
✅ 行政诉讼中一并提起(优先)
✅ 单独提起(确认违法后另行起诉)
✅ 赔偿义务机关先行处理(2 个月)
```

## 强制执行程序

**申请执行期限:**

| 情形 | 期限 |
|------|------|
| 行政诉讼判决 / 裁定 | 申请执行期限 2 年(从履行期届满起算) |

**申请材料:**

```
1. 申请执行书
2. 判决书 / 裁定书
3. 原告身份证明
4. 被告未履行证明
5. 委托代理手续
```

**法院执行措施:**

| 措施 | 适用 |
|------|------|
| 划拨款项 | 履行金钱给付 |
| 强制履行 | 履行行为 |
| 罚款 | 督促履行 |
| 拘留 | 责任人 |

## 跟踪履行

**3 个时点跟踪:**

| 时点 | 事项 |
|------|------|
| 判决生效 15 日 | 确认是否启动履行 |
| 履行期届满 | 评估履行完成度 |
| 履行完毕后 | 评估国家赔偿衔接 |

**履行评估标准:**

| 履行度 | 评估 |
|--------|------|
| **完全履行** | 已按判决内容执行,无需后续行动 |
| **部分履行** | 主要部分已履行,部分未履行(可能涉赔偿) |
| **拒绝履行** | 启动强制执行 + 罚款 / 拘留申请 |
| **客观不能** | 评估 + 国家赔偿 |
| **变通履行** | 法院认定 + 评估接受度 |
