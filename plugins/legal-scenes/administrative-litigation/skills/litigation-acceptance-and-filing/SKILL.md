# /litigation-acceptance-and-filing — 立案受理

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B1 工作流 + § B2 路由表 + § B7 决策树 + § B11 注册表。

## 工作流程

```
Step 1: 核查起诉期限(6 个月 / 15 日 / 最长保护期)
Step 2: 评估受案范围(是否在《行政诉讼法》第 12 条范围)
Step 3: 评估复议前置(是否须先复议)
Step 4: 评估原告资格(是否有利害关系)
Step 5: 评估起诉状内容(被告 / 诉讼请求 / 事实理由)
Step 6: 决定:起诉 / 补正 / 撤回 / 转复议
```

## 起诉期限 3 档(《行政诉讼法》第 46 条)

**直接起诉 — 一般案件:**

```
收到文书之日起 6 个月内
```

**复议后起诉 — 经过复议程序:**

```
收到复议决定书之日起 15 日内
```

**特别长保护期:**

| 情形 | 时限 |
|------|------|
| **不动产相关行政行为** | 最长 **20 年** |
| **其他具体行政行为** | 最长 **5 年** |
| **诉行政机关不作为** | 法定履行期限届满之日起 6 个月 |

**期限起算:**

| 文书送达 | 起算点 |
|----------|--------|
| 直接送达 | 签收日 |
| 邮寄送达 | 邮件签收日 |
| 公告送达 | 公告期满日 |
| 留置送达 | 留置日 |

**超期后果:**
- 原则上**丧失起诉权**(《行政诉讼法》第 46 条)
- 例外:扣除 / 延长 / 不属于自身原因耽搁

## 受案范围(《行政诉讼法》第 12 条)

**受案范围(可诉):**

| 类型 | 注意 |
|------|------|
| 行政处罚(警告 / 罚款 / 吊销等) | ✅ |
| 行政强制 / 行政强制措施 | ✅ |
| 行政许可(拒绝 / 撤销 / 变更) | ✅ |
| 行政确认 | ✅ |
| 行政征收 / 征用决定 | ✅ |
| 行政不作为(拒绝履行 / 拖延) | ✅(须先申请) |
| 侵犯经营自主权 | ✅ |
| 违法要求履行义务 | ✅ |
| 行政协议(政府特许经营等) | ✅(部分) |
| 政府信息公开(拒绝 / 部分拒绝) | ✅ |
| 农业承包合同纠纷 | ✅ |

**不受案范围(不可诉):**

| 类型 | 注意 |
|------|------|
| 国防 / 外交等国家级行为 | ❌ |
| 行政法规 / 规章 / 规定(抽象行政行为) | ❌(部分可附带审查) |
| 行政机关对公务员的奖惩 / 任免 | ❌ |
| 法律规定由行政机关最终裁决的 | ❌ |
| 刑事侦查行为 | ❌(非行政行为) |

**附带审查(《行政诉讼法》第 53 条):**

```
✅ 原告可一并请求审查行政行为所依据的下列规范性文件:
  - 国务院部门规章
  - 县级以上地方政府规章
  - 国务院部门 / 地方政府发布的其他规范性文件
❌ 不可单独起诉规章
```

## 复议前置(《行政诉讼法》第 44 条)

**复议前置情形(典型):**

| 情形 | 法条 |
|------|------|
| 自然资源权属确认 | 《行政复议法》第 30 条 |
| 纳税争议(缴纳税款) | 《税收征管法》第 88 条 |
| 价格违法的处罚 | 《价格法》 |
| 海关处罚(部分) | 《海关法》 |
| 劳动争议(部分) | 单行法 |

**复议 vs 起诉选择:**

| 情形 | 建议 |
|------|------|
| 复议更有利(行政机关自我纠错) | 先复议 |
| 复议是法定前置 | 必须复议 |
| 复议机关与原机关同利益 | 直接起诉 |
| 涉及重大行政行为 | 评估后选择 |
| 已过复议期限 | 直接起诉 |

**复议期限:**

| 复议机关 | 期限 |
|----------|------|
| 一般 | 60 日 |
| 紧急 | 30 日 |
| 单行法特殊规定 | 视规定 |

## 原告资格(《行政诉讼法》第 25 条)

**原告类型:**

| 类型 | 资格 |
|------|------|
| 行政相对人 | ✅ |
| 利害关系人(权益受影响) | ✅ |
| 受害人(被处罚人以外的受害人) | ✅ |
| 公民 / 法人 / 其他组织 | ✅ |
| 外国人 / 无国籍人 / 外国组织 | ✅(中国法保护) |

**不具有原告资格:**

```
❌ 与行政行为无利害关系的人
❌ 仅是"关心公益"的公民(但可检举)
❌ 行政机关(内部上下级)
```

## 立案材料清单

**起诉状必备:**

```
1. 原告基本信息(姓名 / 名称 / 住所 / 联系方式)
2. 被告(作出行政行为的机关全称 + 住所)
3. 诉讼请求(撤销 / 确认违法 / 责令履行 / 赔偿)
4. 事实与理由(行政行为经过 + 违法点 + 法条)
5. 此致法院
6. 起诉人签字 + 日期
```

**附件:**

| 材料 | 来源 |
|------|------|
| 行政行为文书(处罚决定 / 许可决定等) | 当事人 |
| 复议决定书(若经过复议) | 复议机关 |
| 原告身份证明 | 当事人 |
| 证据清单 + 证据副本 | 当事人 / 律师 |
| 委托代理手续(若委托律师) | 律所 |

## 补正与驳回

**补正情形(法院通知补正):**

| 情形 | 期限 |
|------|------|
| 起诉状内容欠缺 | 7 日内补正 |
| 原告不明确 | 补正 |
| 被告不明确 | 补正 |
| 证据不齐 | 补正 |
| 诉讼请求不明确 | 补正 |

**驳回起诉情形:**

| 情形 | 处理 |
|------|------|
| 不在受案范围 | 裁定驳回起诉 |
| 超过起诉期限(无正当理由) | 裁定驳回起诉 |
| 原告不适格 | 裁定驳回起诉 |
| 复议前置未申请 | 裁定驳回起诉 |
| 已生效判决处理过 | 裁定驳回起诉 |

**可上诉:** 驳回起诉裁定可在 10 日内上诉。
