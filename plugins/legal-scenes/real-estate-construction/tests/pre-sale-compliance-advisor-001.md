# Test Case: 商品房预售合规审查

## Query

我公司是一家房地产开发企业（注册地在深圳），现拟对某住宅项目进行预售。
项目情况如下：
1. 项目已取得《国有土地使用证》、《建设用地规划许可证》、《建设工程规划许可证》
2. 正在申请《建筑工程施工许可证》，预计下周取得
3. 项目主体结构已施工至地上5层（共18层），已确定施工进度达到预售条件
4. 预计开盘价为均价5.8万元/平方米
5. 已委托代理销售公司进行营销推广
6. 准备在取得预售许可证前开始意向客户登记和认筹

请审查以下事项的合规性：
1. 预售许可证的申请条件和前置程序是否满足
2. 拟在取得预售证前进行认筹是否合规
3. 预售资金监管要求
4. 认购书和商品房买卖合同的合规要点
5. 广告宣传中的合规红线

## Expected Behavior

1. 应列出商品房预售许可证的法定条件（《城市房地产管理法》第45条）
2. 应指出未取得预售许可证不得进行认筹/收取定金（《商品房销售管理办法》第22条）
3. 应分析深圳市预售资金监管的具体要求
4. 应核查认购书中的关键条款合规性
5. 应指出广告宣传中禁止的用语（如"最优""第一"等）
6. 应标注风险等级，违规预售可能面临行政处罚
