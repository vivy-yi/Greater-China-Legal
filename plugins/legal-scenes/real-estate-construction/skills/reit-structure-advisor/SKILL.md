# REITs架构设计咨询

## 一、基本信息

### 1.1 核心目的
为商业地产/基础设施持有方（原始权益人）提供公募REITs及类REITs架构设计的法律支持，覆盖资产重组、税务优化、专项计划结构搭建、合规审查及投资者穿透核查。

### 1.2 适用范围
- 境内公募基础设施REITs（试点范围仓储物流/产业园区/保障性租赁住房/收费公路/清洁能源等）
- 交易所类REITs/CMBS（商业物业资产证券化）
- 保障性租赁住房REITs
- 香港/新加坡REITs上市
- 私募不动产基金

## 二、分析框架

### 2.1 REITs核心结构要素

```
基础设施REITs结构（典型模式）
┌─────────────────────────────────────┐
│          公募REITs基金                │
│  （上交所/深交所公开交易）            │
│  ┌───────────────────────────────┐   │
│  │  资产支持证券（ABS）专项计划     │  │
│  │  ┌─────────────────────────┐   │  │
│  │  │   项目公司（SPV）         │  │  │
│  │  │   持有底层资产            │  │  │
│  │  │   ┌───────────────────┐ │  │  │
│  │  │   │  基础设施/商业物业  │ │  │  │
│  │  │   └───────────────────┘ │  │  │
│  │  └─────────────────────────┘   │  │
│  └───────────────────────────────┘   │
└─────────────────────────────────────┘
```

| 层级 | 角色 | 法律依据 |
|------|------|---------|
| 公募REITs基金 | 上市主体，向公众投资者募集 | 公募基金运作管理办法 |
| 资产支持专项计划 | 持有项目公司股权/债权 | 资产证券化业务管理规定 |
| 项目公司（SPV） | 持有底层资产产权 | 公司法 |
| 底层资产 | 基础设施/商业物业 | 各有产权证照 |

### 2.2 底层资产合规要求

| 审查要素 | 基础设施REITs（中国） | 类REITs（交易所） | 香港REITs |
|---------|---------------------|-----------------|----------|
| 资产类型 | 仓储物流/产业园/保障房/收费公路/能源/环保 | 商业物业/办公楼/酒店/商场 | 各类商业物业+酒店 |
| 运营年限 | 原则上运营3年以上（保障房除外） | 运营2年以上 | 无硬性要求 |
| 现金流 | 近3年平均净现金流为正且稳定 | 覆盖倍数（DSCR）不低于1.2X | 覆盖倍数不低于1.0X |
| 资产权属 | 产权清晰，无重大纠纷 | 产权清晰 | 产权清晰 |
| 资产估值 | 不低于10亿元（部分行业可低） | 不低于8亿元 | 无明确下限 |
| 合规手续 | 土地/规划/建设/消防/环保全手续齐全 | 主要手续齐全 | 需当地律师确认 |

### 2.3 资产重组税务优化框架

根据2022年REITs试点税收政策（财政部税务总局公告2022年第3号）：

| 交易环节 | 税务处理 | 适用条件 |
|---------|---------|---------|
| 资产划转至项目公司（第一步） | 企业所得税递延（特殊性税务处理） | 按账面净值划转+不确认所得 |
| 项目公司股权转让至专项计划（第二步） | 企业所得税递延至REITs实际出售时 | 原始权益人持股比例不低于20% |
| 原始权益人战略配售 | 自持部分暂不征税（持有期间） | 战略配售+持有期限承诺 |
| 资产重组阶段 | 土地增值税（视同销售/划转）— 各地执行不一 | 划转方式可暂不征土增税（需当地税务确认） |
| 契税 | 资产划转免征（财税〔2021〕29号） | 同一投资主体内部划转 |
| 增值税 | 资产划转不征增值税（同一实控人） | 连续12个月累计不超过规定额 |

### 2.4 发行规模与杠杆限制

| 指标 | 基础设施REITs | 类REITs/CMBS |
|------|--------------|-------------|
| 杠杆上限 | 基金总资产的28%（借款+总资产比） | 通常60%-75%（LTV） |
| 现金流覆盖 | 可供分配金额不低于基金年度可供分配金额的90% | DSCR≥1.2倍 |
| 分红比例 | 可供分配金额的90%以上 | 按ABS计划安排 |
| 发行规模 | 按评估价（非折价） | 按评估价的一定折扣（通常60-75%） |

## 三、操作流程

### 3.1 公募REITs发行流程

**Phase 1：前期准备（3-6个月）**
- [ ] 底层资产筛选与合规审查
- [ ] 资产重组方案设计
- [ ] 财务审计（最近3年+1期）
- [ ] 资产评估（收益法/市场法/成本法）
- [ ] 法律尽职调查
- [ ] 现金流测算与可供分配金额测算

**Phase 2：项目申报（2-4个月）**
- [ ] 省级发改委出具专项意见
- [ ] 国家发改委审核通过（纳入推荐项目）
- [ ] 中国证监会/交易所受理
- [ ] 基金管理人/ABS管理人/财务顾问文件准备

**Phase 3：发行与上市（2-3个月）**
- [ ] 询价/定价
- [ ] 网下发售+公众发售
- [ ] 战略配售（原始权益人20%+战略投资者）
- [ ] 基金合同生效+基金份额登记
- [ ] 上市交易

**Phase 4：存续期管理（持续）**
- [ ] 基金运营（资产运营管理）
- [ ] 季度/年度报告披露
- [ ] 收益分配（每年至少1次）
- [ ] 资产扩募（二级市场融资+收购新资产）
- [ ] 资产处置（重大资产处置须持有人会议通过）

### 3.2 原始权益人重点关注问题

| 问题 | 分析 | 建议 |
|------|------|------|
| 控制权保留 | 原始权益人认购REITs比例不低于20% | 保留对底层资产的运营管理权 |
| 资产出表vs表内 | 并表处理vs出表处理（会计处理差异大） | 按会计准则+原始权益人财务报表需求 |
| 资产隔离 | 真实出售/破产隔离 | 专项计划+项目公司双层隔离机制 |
| 运营管理 | 原始权益人担任外部管理机构 | 签订运营管理协议+管理费 |
| 扩募/退出 | 同比例扩募权/优先收购权 | 上市REITs扩募+原始权益人退出安排 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **底层资产存在重大合规瑕疵** — 土地用途不一致/未取得完整产权/违建
- **资产重组涉及重大税务不确定** — 土地增值税/划转税务暂不明确
- **原始权益人信用风险** — 存在债务违约/担保代偿/失信记录
- **涉及上市公司分拆REITs** — 分拆上市合规+信息披露义务
- **涉及跨境架构（香港/新加坡REITs）** — 跨境税务+外汇管理+法律冲突
- **发改委/证监会问询回复** — 专业法律意见书出具
- **REITs扩募中的新增资产收购** — 关联交易合规+资产评估公允性
- **REITs终止/清算** — 资产处置方案+持有人权益保护

## 五、本Skill不涵盖

1. REITs的公募基金注册/基金管理人合规（基金公司专业负责）
2. REITs发行中的承销/销售/路演安排
3. REITs资产评估/财务审计的具体执行
4. REITs存续期的会计处理（合并报表/投资性房地产计量）
5. 境外REITs市场的具体上市规则（需当地法律顾问）
6. 保障性租赁住房纳入REITs试点的具体认定程序
7. REITs投资者诉讼/集体诉讼的代理
