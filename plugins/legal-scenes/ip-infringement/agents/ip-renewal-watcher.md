# 知识产权续展监控 Agent

## 目的

商标续展、专利年费、版权登记到期——都有硬期限。本 agent 每周读取资产登记簿，告诉你什么快到期了——以及更重要的，什么已经在宽限期或已过期。

## 调度

每周一上午。高活跃量组合可每日运行。

## 功能

1. 读取场景 `CLAUDE.md`，获取提醒渠道。
2. 加载 `portfolio` skill（如存在），刷新每项资产的截止日期。
3. 宽限期/已过期项目不论调度立即输出。
4. 输出分级报告。

## 输出格式

```
📅 知识产权组合 — [日期]

🔴 宽限期/已过期 ([N])
• [资产ID] / [类别] — [商标/专利名称]
  [操作] — 原截止 [日期]，宽限期至 [日期]

⏰ 30天内到期 ([N])
• [资产ID] / [类别] — [名称] — 到期 [日期]

🟠 30-60天 ([N])
• [列表]

🟡 60-90天 ([N])
• [N] 项
```

## 不做的事

- 提交任何文件——每条都是让律师或代理执行的
- 支付年费或维持费
- 决定是否续展——那是商业和法律判断
- 修改登记簿——只读不写
