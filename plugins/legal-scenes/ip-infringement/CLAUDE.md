### § A8.1 知识产权特别注意 4 大块（专利 + 商标 + 著作权 + 商业秘密）

> **核心原则**：IP 维权的核心是**权属清晰 + 证据完整 + 措施及时**。"权利不稳" + "证据不足" + "行动迟缓" = 维权失败。

#### 块 1：专利侵权（《专利法》2020 修订 + 2023 细则）

**核心法规：**
- 《专利法》（2020 年 10 月修订 + 2021 年 6 月生效）
- 《专利法实施细则》
- 《关于审理专利纠纷案件若干问题的规定》

**专利类型 + 保护期：**

| 类型 | 保护期 | 申请条件 |
|------|--------|---------|
| **发明专利** | 20 年 | 新颖性 + 创造性 + 实用性 |
| **实用新型** | 10 年 | 新颖性 + 实用性 + 创造性降低 |
| **外观设计** | 15 年 | 新颖性 + 美感 + 适于工业应用 |

**专利侵权 4 种行为：**

| 行为 | 法条 |
|------|------|
| **制造** | 《专利法》第 11 条 |
| **许诺销售** | 《专利法》第 11 条 |
| **销售** | 《专利法》第 11 条 |
| **使用** | 《专利法》第 11 条（仅发明专利）|
| **进口** | 《专利法》第 11 条 |

**专利侵权救济（2020 修订加强）：**

| 救济 | 标准 |
|------|------|
| **停止侵权** | 立即停止 + 销毁侵权产品 |
| **销毁模具 / 工具** | 主要用于制造侵权产品 |
| **赔偿损失** | 实际损失 / 获利 / 许可费倍数（1-5 倍） |
| **惩罚性赔偿** | 故意 + 情节严重（1-5 倍）|
| **合理费用** | 律师费 / 公证费 / 鉴定费 |

**blocks（绝对禁止）**：
- 假冒他人专利 → **《刑法》第 216 条假冒专利罪（3 年以下 + 罚金）**
- 伪造专利证书 → **《刑法》第 280 条**
- 标准必要专利拒绝许可 → **反垄断风险**

**work but ships（可补救）**：
- 专利侵权警告 → 30 日内协商 / 停止
- 专利布局不完善 → 申请专利 + 商业秘密保护

#### 块 2：商标侵权（《商标法》2019 修订）

**核心法规：**
- 《商标法》（2019 年 4 月修订 + 2019 年 11 月生效）
- 《商标法实施条例》
- 《关于审理商标授权确权行政案件若干问题的规定》

**商标类型 + 保护期：**

| 类型 | 保护期 | 申请条件 |
|------|--------|---------|
| **普通商标** | 10 年（可续） | 显著性 + 不冲突 |
| **驰名商标** | 跨类保护（已注册 10 年 + 实质 5 年） | 广为知晓 + 高知名度 |

**商标侵权 5 类行为（《商标法》第 57 条）：**

| 行为 | 含义 |
|------|------|
| **未经许可在同种商品使用相同商标** | 假冒注册商标 |
| **未经许可在类似商品使用相同商标** | 近似商品混淆 |
| **未经许可在同种商品使用近似商标** | 近似商标混淆 |
| **销售侵犯商标专用权商品** | 销售侵权商品 |
| **伪造 / 擅自制造注册商标标识** | 商标标识伪造 |

**驰名商标特殊保护：**

| 条件 | 保护 |
|------|------|
| 已注册驰名商标 | 跨类保护（不相同 / 不类似商品） |
| 未注册驰名商标 | 在相同 / 类似商品上保护 |
| 恶意抢注他人驰名商标 | 撤销 + 反不正当竞争 |

**商标侵权赔偿（2019 修订）：**

| 赔偿类型 | 标准 |
|---------|------|
| 实际损失 | 权利人因侵权减少的收入 |
| 获利 | 侵权人因侵权获得的利益 |
| 许可费倍数 | 许可费合理倍数（1-5 倍） |
| 法定赔偿 | 500 万以下（情节严重） |
| 惩罚性赔偿 | 故意 + 情节严重（1-5 倍） |

**blocks（绝对禁止）**：
- 假冒注册商标 → **《刑法》第 213 条（情节严重 3-10 年）**
- 销售假冒注册商标商品 → **《刑法》第 214 条（情节严重 3-10 年）**
- 恶意抢注驰名商标 → **撤销 + 反不正当竞争**

**work but ships（可补救）**：
- 商标警告函 → 30 日内协商 / 停止
- 商标布局不完善 → 注册防御商标 + 跨类保护

#### 块 3：著作权侵权（《著作权法》2020 修订）

**核心法规：**
- 《著作权法》（2020 年 11 月修订 + 2021 年 6 月生效）
- 《信息网络传播权保护条例》
- 《关于审理著作权纠纷案件若干问题的规定》

**著作权自动取得：**

| 原则 | 内容 |
|------|------|
| **自动保护** | 作品完成即取得，无需登记 |
| **登记对抗** | 登记可对抗善意第三人 |
| **保护期** | 公民作品终生 + 死后 50 年 |

**著作权侵权 5 类行为（《著作权法》第 53 条）：**

| 行为 | 含义 |
|------|------|
| **未经许可复制** | 复制发行 |
| **未经许可发行** | 公开销售 |
| **未经许可出租** | 有偿许可他人临时使用 |
| **未经许可传播** | 信息网络传播 |
| **未经许可改编 / 翻译 / 注释** | 演绎作品侵权 |

**网络服务提供者责任（《信息网络传播权保护条例》）：**

| 责任类型 | 法源 |
|---------|------|
| **直接侵权** | 未经许可传播 |
| **间接侵权（避风港原则）** | 不知道 + 不应知道 + 接到通知后删除 |
| **红旗原则** | 明显侵权应当知道 → 无避风港保护 |

**blocks（绝对禁止）**：
- 盗版复制发行 → **《刑法》第 217 条（违法所得 3 万以上 3-7 年）**
- 盗版网络传播 → **《刑法》第 217 条**
- 软件破解 / 反编译销售 → **《刑法》第 217 条**

**work but ships（可补救）**：
- 收到删除通知 → 立即删除 + 断开链接
- 来源不明 → 立即核实 + 暂停使用

#### 块 4：商业秘密 + 不正当竞争（《反不正当竞争法》2019 修订）

**核心法规：**
- 《反不正当竞争法》（2019 年 4 月修订）
- 《关于审理侵犯商业秘密民事案件若干问题的规定》

**商业秘密 3 要件：**

| 要件 | 内容 |
|------|------|
| **秘密性** | 不为公众所知悉 |
| **价值性** | 具有商业价值 |
| **保密性** | 采取合理保密措施 |

**商业秘密侵权 4 类行为（《反不正当竞争法》第 9 条）：**

| 行为 | 含义 |
|------|------|
| **盗窃** | 秘密窃取 |
| **贿赂** | 以贿赂获取 |
| **欺诈** | 欺诈获取 |
| **胁迫** | 胁迫获取 |
| **违反约定** | 违约披露 / 使用 |
| **教唆 / 协助** | 教唆 / 帮助他人违反保密义务 |

**商业秘密赔偿（2019 修订）：**

| 赔偿 | 标准 |
|------|------|
| 实际损失 | 权利人因侵权减少 |
| 获利 | 侵权人因侵权获得 |
| 许可费倍数 | 合理倍数（1-5 倍） |
| 法定赔偿 | 500 万以下 |
| 惩罚性赔偿 | 故意 + 情节严重（1-5 倍）|

**blocks（绝对禁止）**：
- 侵犯商业秘密 → **《刑法》第 219 条（情节严重 3-10 年）**
- 通过贿赂 / 欺诈获取 → **同上**
- 违反保密协议 → **民事 + 刑事**

**work but ships（可补救）**：
- 员工离职 → 立即回收保密文件 + 设备
- 保密措施不全 → 立即完善（保密协议 + 物理隔离）

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **假冒专利** | 《刑法》第 216 条 | 3 年以下 + 罚金 |
| 2. **假冒注册商标** | 《刑法》第 213 条 | 情节严重 3-10 年 + 罚金 |
| 3. **盗版复制发行** | 《刑法》第 217 条 | 违法所得 3 万 + 3-7 年 |
| 4. **侵犯商业秘密** | 《刑法》第 219 条 | 情节严重 3-10 年 + 罚金 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时限
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — IP 增强版）**：
- IP 类型（专利 / 商标 / 著作权 / 商业秘密）？（核心）

- 是否涉外？（多法域协调 + 海关备案）
- 是否驰名商标 / 标准必要专利？（特殊保护）

### § B1 工作流(主入口 + 关键节点)

**主入口:** IP 类型识别 → 专属 skill

**关键节点:**

```
Step 1: 权属确认(注册证书 / 权利登记)

Step 3: 证据保全(公证 / 鉴定)
Step 4: 评估损失(实际 / 获利 / 法定)
Step 5: 起诉 / 抗辩
```
