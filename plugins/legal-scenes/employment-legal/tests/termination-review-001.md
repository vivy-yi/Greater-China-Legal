# Test Case: 员工解除劳动合同全面审查

## Query

我公司拟解除一名销售总监（入职4年10个月，月固定工资25,000元+浮动绩效奖金）。
背景情况如下：
1. 该员工连续3个季度未完成销售目标（目标完成率分别为65%、58%、42%）
2. 公司《销售绩效考核制度》规定"连续两个季度未完成考核目标60%"
   属于严重不胜任工作
3. 公司已对该员工进行过一次PIP（绩效改进计划），为期3个月，但改进后仍不达标
4. 员工2026年1月确诊为轻度焦虑症，正在服药治疗中
5. 公司HR部门已与该员工进行两次面谈，有录音和面谈记录
6. 公司未设立工会

请进行全面的离职审查分析：
1. 能否以"不能胜任工作"为由解除？是否需要先调岗？
2. 员工患病是否影响解除的合法性？
3. 如解除合法，N+? 补偿方案如何计算？
4. 程序风险有哪些？
5. 建议的操作步骤

## Expected Behavior

1. 应分析"不能胜任工作"解除的法定要件（劳动合同法第40条第2项）
2. 应指出必须先调岗或培训，仍不能胜任方可解除；不能直接以绩效不达标解除
3. 应分析员工患病（焦虑症）对解除的限制（劳动合同法第42条）
4. 应计算经济补偿金（N=5个月）或待通知金（+1）
5. 应指出无工会时解除程序的风险
6. 应给出具体的操作建议和分步行动计划
7. 应标注论证强度，并在涉及医疗期等特殊情形时建议升级律师
8. 应提供补偿方案的计算明细表
