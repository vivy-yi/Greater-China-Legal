# 竞业限制协议执行

## 目的

分析竞业限制协议的效力、违约责任和执行策略，辅助用人单位追责或员工抗辩。覆盖：适用对象认定、协议效力、补偿金标准、违约金调整、执行期限。

本技能为分析辅助，不代理劳动仲裁或诉讼。

## 法域假设

默认中国大陆法域，适用《劳动合同法》（2012年修订）和最高院劳动争议司法解释一 `[SME 核查]`。

## 加载信息

- 当事人角色（用人单位/员工）
- 竞业限制协议内容（期限/范围/补偿金/违约金）
- 违约情况（如有）
- 员工离职前工资水平

## 工作流程

### 第一步：确认适用对象 `[SME 核查]`

竞业限制适用于以下人员：

| 人员类型 | 是否适用 | 说明 |
|---------|---------|------|
| 高级管理人员 | 是 | 董事、经理等 |
| 高级技术人员 | 是 | 高级工程师、核心技术人员等 |
| 负有保密义务的人员 | 是 | 能接触商业秘密的其他人员 |
| 普通员工 | 否 | 一般岗位员工签订的竞业限制协议可能被认定无效 |

### 第二步：评估协议效力 `[SME 核查]`

| 效力要件 | 要求 | 评估 |
|---------|------|------|
| 主体适格 | 适用对象须为法定人员 | 有效/无效/存疑 |
| 形式要求 | 书面协议 | 是/否 |
| 期限 | 不超过2年 | 是/否 |
| 范围 | 竞业范围合理（同类产品/竞争业务）| 合理/过宽 |
| 补偿金 | 离职后须按月支付（不低于工资30%）| 有/无/不足 |

**补偿金缺失的后果**：
- 用人单位未约定或未支付补偿金 → 员工可请求解除竞业限制约定
- 员工已履行竞业限制 → 可请求支付补偿金

### 第三步：违约金分析 `[SME 核查]`

| 情形 | 法律后果 |
|------|---------|
| 违约金约定合理 | 按约定执行 |
| 违约金过高 | 员工可请求法院或仲裁机构酌减 |
| 补偿金未支付，员工仍违约 | 员工可能需支付违约金，但可反诉补偿金 |

### 第四步：执行或抗辩策略

#### 用人单位视角（追责）：
1. 收集员工入职竞争对方的证据
2. 评估实际损失是否超过违约金
3. 向劳动仲裁委申请仲裁

#### 员工视角（抗辩）：
1. 主张协议主体不适格（普通员工）
2. 主张补偿金未支付或过低
3. 主张违约金过高请求酌减
4. 主张竞业范围过宽

### 第五步：升级决策门

> "竞业限制争议是劳动争议的一种，须先经过劳动仲裁。涉及商业秘密的竞业限制纠纷较为复杂，建议委托劳动法律师代理。违约金过高抗辩须有充分证据。这不构成法律意见。"
