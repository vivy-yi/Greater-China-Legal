## 0. 程序选择决策树

```
劳动争议类型
  │
  ├─ 解除劳动合同争议
  │   ├─ 协商解除(36 条) → 补偿金计算
  │   ├─ 过失性解除(39 条) → 无需补偿
  │   ├─ 非过失性解除(40 条) → N + 代通知金
  │   └─ 经济性裁员(41 条) → 20 人+ / 10%+
  │
  ├─ 工资/福利争议
  │   ├─ 拖欠工资 → 50-100% 赔偿金
  │   ├─ 加班费 → 3 类加班费计算
  │   ├─ 年休假 → 300% 折算
  │   └─ 二倍工资 → 未签合同
  │
  ├─ 社保/公积金争议
  │   └─ 补缴(由劳动监察/税务处理,仲裁不受理)
  │
  ├─ 工伤争议
  │   └─ 30 日内申请 → 工伤认定 → 待遇
  │
  └─ 集体争议
      └─ 10 人以上集体合同/集体协商
```

## 1. 劳动合同解除类型与法律后果

### 1.1 协商解除(劳动合同法第 36 条)

- 用人单位提出:支付经济补偿金(N)
- 劳动者提出:无经济补偿
- 程序:双方协商一致,书面解除

### 1.2 过失性解除(劳动合同法第 39 条)

- 情形:
  1. 在试用期间被证明不符合录用条件
  2. 严重违反用人单位的规章制度
  3. 严重失职,营私舞弊,给用人单位造成重大损害
  4. 劳动者同时与其他用人单位建立劳动关系,经提出拒不改正
  5. 因劳动合同法第 26 条第 1 款第 1 项规定的情形致使劳动合同无效
  6. 被依法追究刑事责任
- 程序:用人单位可随时解除,**无需支付经济补偿金**
- 注意:须有充分证据,程序合法

### 1.3 非过失性解除(劳动合同法第 40 条)

- 情形:
  1. 劳动者患病或非因工负伤,医疗期满后不能从事原工作,也不能从事用人单位另行安排的工作
  2. 劳动者不能胜任工作,经过培训或调整工作岗位,仍不能胜任
  3. 劳动合同订立时所依据的客观情况发生重大变化,致使劳动合同无法履行,经协商未能达成协议
- 程序:提前 30 日书面通知**或**支付一个月工资(代通知金)
- 后果:支付经济补偿金(N)

### 1.4 经济性裁员(劳动合同法第 41 条)

- 条件:
  1. 依照企业破产法规定进行重整
  2. 生产经营发生严重困难
  3. 企业转产、重大技术革新或经营方式调整,经变更劳动合同后,仍需裁减人员
  4. 其他因劳动合同订立时所依据的客观经济情况发生重大变化,致使劳动合同无法履行
- 人数门槛:**20 人以上**或**裁减不足 20% 但占职工总数 10% 以上**
- 程序:
  1. 提前 30 日向工会或全体职工说明情况
  2. 听取工会或职工的意见
  3. 将裁减人员方案向劳动行政部门报告
- 后果:支付经济补偿金(N)
- **优先留用**:
  1. 与本单位订立较长期限的固定期限劳动合同
  2. 与本单位订立无固定期限劳动合同
  3. 家庭无其他就业人员,有需要扶养的老人或未成年人

## 2. 经济补偿金计算框架

### 2.1 计算公式

- **N**:每满一年支付一个月工资(劳动合同法第 47 条)
- **六个月以上不满一年**:按一年计算
- **不满六个月**:支付半个月工资(劳动合同法第 47 条)

### 2.2 月工资定义

- 劳动合同解除或终止前 **12 个月**平均工资
- 包括:基本工资 + 奖金 + 津贴 + 补贴 + 加班费

### 2.3 上限规定

- **三倍上限**:劳动者月工资高于用人单位所在地上年度职工月平均工资 **3 倍**的,按 3 倍支付
- **最高年限**:**12 年**(特殊高收入者)
- 普通员工:不设上限

### 2.4 违法解除赔偿金

- **2N**:违反劳动合同法规定解除或终止劳动合同
- 情形:
  1. 未给补偿
  2. 程序违法
  3. 禁止解除情形(§ 5.3 6 大绝对禁止)

### 2.5 N+1

- 仅当用人单位选择"代通知金"替代提前 30 日通知时适用
- 提前 30 日通知则只需 N,无需加 1

## 3. 劳动争议处理流程

### 3.1 协商(非必经)

- 争议发生后,双方可协商解决
- 协商非必经程序,但推荐优先尝试

### 3.2 调解(非必经)

- 企业劳动争议调解委员会(企业内部)
- 基层人民调解组织(街道/乡镇)
- 行业性、专业性人民调解组织
- 调解协议无强制执行力

### 3.3 劳动仲裁(**必经前置程序**)

- **时效**:**1 年**(劳动争议调解仲裁法第 27 条)
- 管辖:用人单位所在地或合同履行地劳动争议仲裁委员会
- 审理时限:**45 日**内,案情复杂可延长 15 日
- 费用:**免费**
- 不服仲裁裁决 → 15 日内向人民法院起诉

### 3.4 诉讼

- 对仲裁裁决不服,15 日内向人民法院提起诉讼
- 仲裁前置,不得直接起诉

### 3.5 集体劳动争议

- 10 人以上共同请求 → 集体争议
- 可推举 1-3 名代表人参加仲裁
- 仲裁委员会可优先立案

## 4. 各省市劳动法裁判差异

### 4.1 经济补偿金计算基数

| 地区 | 上年度职工月平均工资倍数 | 说明 |
|------|--------------------------|------|
| 全国统一 | 3 倍 | 劳动合同法 47 条 |
| 北京 | 通常按区级平均工资 | 实际操作中高于部分省市 |
| 上海 | 按市级平均工资 | 基数较高 |
| 广东(广州/深圳) | 按市级平均工资 | 深圳有独立标准 |
| 江苏 | 按市级平均工资 | 部分区域有补充规定 |

**注意**:高收入员工(超过 3 倍平均工资)的补偿年限最高不超过 12 年。

### 4.2 加班费计算基数

| 地区 | 计算标准 | 争议风险 |
|------|---------|--------|
| 北京 | 约定工资÷21.75÷8 | 约定工资低于法定最低时须调整 |
| 上海 | 约定工资÷21.75÷8 | 常用"工资清单"确认基数 |
| 广东 | 约定工资÷21.75÷8 | 最低工资标准作为保底 |
| 浙江 | 约定工资÷21.75÷8 | 偏向劳动者解释 |
| 江苏 | 约定工资÷21.75÷8 | 部分地区要求剔除固定津贴 |

### 4.3 年休假未休折算工资

| 地区 | 计算比例 | 常见争议 |
|------|---------|--------|
| 全国统一 | 日工资收入 300%(未休年休假) | 工龄证明是举证关键 |
| 北京 | 同上 | 折算不足 1 天部分四舍五入 |
| 上海 | 同上 | 部分仲裁委要求精确到角 |
| 广东 | 同上 | 寒暑假冲抵问题常见 |

### 4.4 违法解除赔偿金(2N)

| 地区 | 裁量标准 | 倾向性 |
|------|---------|--------|
| 北京 | 程序违法+实体违法 | 用人单位败诉率高 |
| 上海 | 程序违法即构成 | 倾向保护劳动者 |
| 广东 | 实体违法为主 | 部分情形容忍程序瑕疵 |
| 江苏 | 程序+实体综合 | 相对平衡 |

### 4.5 竞业限制

| 地区 | 最长期限 | 补偿金要求 | 法院态度 |
|------|---------|---------|--------|
| 全国 | 2 年(劳动合同法 24 条) | 未明确最低标准 | 司法解释要求合理性 |
| 北京 | 2 年 | 不低于工资 20% | 倾向认定有效 |
| 上海 | 2 年 | 不低于工资 30% | 部分案件以补偿过低无效 |
| 广东 | 2 年 | 参照约定 | 部分不支持高额违约金 |
| 浙江 | 2 年 | 参照约定 | 倾向保护企业 |

### 4.6 社保/公积金补缴

| 地区 | 仲裁/法院受理态度 | 说明 |
|------|----------------|------|
| 全国 | 仲裁委不受理,要求行政部门处理 | 社保补缴通过劳动监察/税务 |
| 上海 | 同上 | 但工伤待遇可仲裁主张 |
| 广东 | 同上 | 住房公积金独立投诉渠道 |

### 4.7 规章制度民主程序

| 地区 | 职代会/工会要求 | 公示方式 |
|------|--------------|--------|
| 北京 | 须经职代会或工会讨论 | 公示+签字确认 |
| 上海 | 须公示 | 强调公示的证据保全 |
| 广东 | 建议经工会 | 无强制要求但有案例支持 |
| 江苏 | 须民主程序 | 认定未履行民主程序的规章制度无效 |

## 5. 高频争议类型

| 争议类型 | 常见诉求 | 举证要点 |
|---------|---------|--------|
| 违法解除 | 继续履行 / 赔偿金 2N | 解除事实依据、程序合规性 |
| 拖欠工资 | 工资+赔偿金(50%-100%) | 工资支付记录、考勤记录 |
| 未缴社保 | 补缴 + 赔偿 | 劳动关系证明、工资记录 |
| 加班费 | 平时/休息日/法定节假日加班费 | 考勤记录、工资台账 |
| 年休假 | 未休年休假工资报酬 | 工龄证明、休假记录 |
| 二倍工资 | 未签书面合同第二倍工资 | 劳动关系起算时间、合同文本 |
| 经济补偿金 | N / N+1 / 2N | 解除原因、服务年限、工资标准 |

## 6. 文件签发要求

### 6.1 解除劳动合同证明书(离职证明)

- 依据:劳动合同法第 50 条
- 内容:期限、工作岗位、在本单位工作年限
- 时限:解除或终止之日起 **15 日内**出具
- 样式:不得设定附加条件,不得扣押证件

### 6.2 社保与档案转移

- 依据:劳动合同法第 50 条
- 时限:**15 日内**办理
- 失业保险:符合条件可申领失业金

### 6.3 工资结算

- 依据:劳动合同法第 50 条
- 时限:**一次性付清**(解除或终止时)
- 不得克扣或者无故拖欠

### 6.4 工伤相关文件

- 工伤认定申请表
- 劳动能力鉴定结论
- 工伤待遇申领表
