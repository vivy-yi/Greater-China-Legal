# Test Case: 供应链应收账款融资方案合规审查

## Query

我公司是一家大型制造企业（核心企业），拟为其上游中小供应商提供供应链融资支持方案。
方案设计如下：
1. 供应商将对我公司的应收账款转让给一家商业保理公司
2. 保理公司向供应商提供融资（融资比例为应收账款的80%）
3. 我公司向保理公司出具《应收账款确认函》
4. 融资利率为年化12%
5. 保理公司持有商业保理牌照（天津注册）

请审查该方案的合规性，特别是：
1. 保理公司的资质是否齐全
2. 应收账款质押/转让是否需要登记
3. 融资利率是否存在合规风险
4. 核心企业出具确认函的法律效力
5. 供应商破产情况下应收账款的归属问题

## Expected Behavior

1. 应核查保理公司是否有商业保理牌照及资质
2. 应指出应收账款转让须在中国人民银行征信中心动产融资统一登记公示系统（中登网）登记
3. 应分析年化12%利率是否超过法定上限（参照民间借贷司法解释）
4. 应分析《应收账款确认函》的法律效力及核心企业承担的责任
5. 应分析供应商破产时应收账款归属问题（已转让vs未转让）
6. 应标注数据来源并建议就具体金额问题咨询律师
