# 刑事程序指南 (Criminal Procedure Guide)

## 一、目的与范围

### 1.1 核心目的
为中国法项下刑事诉讼程序提供阶段化指引，帮助用户了解各阶段程序要点、权利保障及注意事项。

### 1.2 适用阶段
- 立案阶段
- 侦查阶段（公安/监察委）
- 审查起诉阶段
- 一审程序
- 二审程序
- 执行程序
- 特别程序（缺席审判、违法所得没收）

### 1.3 案件类型说明
本Skill主要适用于：
- 企业家犯罪案件
- 职务犯罪案件（监察委管辖）
- 经济犯罪案件
- 涉税犯罪案件

## 二、程序阶段全流程

### 2.1 立案标准

| 条件 | 说明 |
|-----|------|
| 事实条件 | 有犯罪事实 |
| 法律条件 | 需要追究刑事责任 |
| 管辖条件 | 属于本机关管辖 |

**立案标准**（公安管辖案件）：
- 刑法分则规定的犯罪
- 达到各罪立案追诉标准
- 属于公安机关管辖范围

### 2.2 侦查阶段

#### 公安机关侦查程序

| 程序 | 期限 | 要点 |
|-----|------|-----|
| 拘传 | 12小时（可延长至24小时） | 传唤到案后仍可拘传 |
| 拘留 | 24小时内送看守所 | 拘留后24小时内通知家属 |
| 逮捕 | 侦查羁押最长7个月 | 需检察院批准/决定 |
| 监视居住 | 最长6个月 | 指定居所或住所 |
| 取保候审 | 最长12个月 | 非羁押强制措施 |

#### 监察委调查程序

| 程序 | 期限 | 要点 |
|-----|------|-----|
| 留置 | 3个月（可延长至6个月） | 监察委专属措施 |
| 讯问 | 留置期间 | 律师会见受限制 |
| 查询冻结 | 同步推进 | 财产控制措施 |
| 移送审查起诉 | 留置期满后 | 检察院衔接 |

#### 侦查阶段权利清单

| 权利 | 内容 |
|-----|------|
| 委托辩护 | 被侦查起即可委托 |
| 申诉控告 | 对违法侦查行为申诉 |
| 核实证据 | 核对笔录、提出异议 |
| 申请变更强制措施 | 取保/监视居住/保释 |

### 2.3 审查起诉阶段

| 程序 | 期限 | 要点 |
|-----|------|-----|
| 审查起诉 | 1个月（重大复杂可延长） | 检察院审查全部证据 |
| 补充侦查 | 1个月（可退查两次） | 公安补充侦查 |
| 提起公诉 | 法定期间内 | 起诉书+证据目录 |
| 不起诉 | 法定情形 | 无罪/罪轻/证据不足 |

**审查起诉阶段权利**：
| 权利 | 说明 |
|-----|------|
| 委托辩护 | 继续委托或更换律师 |
| 阅卷权 | 查阅全部案卷材料 |
| 听取意见 | 检察官听取辩护意见 |
| 认罪认罚 | 签署认罪认罚具结书 |

### 2.4 一审程序

#### 普通程序

| 程序 | 期限 | 要点 |
|-----|------|-----|
| 受理 | 7日内 | 法院决定是否受理 |
| 送达 | 10日内 | 起诉书送达被告人 |
| 庭前会议 | 需要时 | 非法证据排除等 |
| 开庭审理 | 2个月内审结 | 最长可延3个月+ |
| 宣判 | 10日内 | 当庭或择日宣判 |

#### 认罪认罚程序

| 要素 | 内容 |
|-----|------|
| 适用条件 | 认罪+认罚+同意适用 |
| 量刑建议 | 检察院提出量刑建议 |
| 法院采纳 | 一般应采纳检察院量刑建议 |
| 从宽幅度 | 可从宽20-30% |

#### 审理期限

| 情形 | 期限 |
|-----|------|
| 普通程序 | 2个月（可延3个月） |
| 简易程序 | 20日（可能减刑） |
| 速裁程序 | 10日（可能减刑） |

### 2.5 二审程序

| 程序 | 期限 | 要点 |
|-----|------|-----|
| 上诉 | 10日内 | 上诉权不可剥夺 |
| 抗诉 | 5日内 | 检察院提起 |
| 审查 | 2个月 | 阅卷+书面/开庭 |
| 改判/维持 | 法定情形 | 发回重审最多两次 |

## 三、强制措施详解

### 3.1 羁押型强制措施

| 措施 | 场所 | 期限 | 律师会见 |
|-----|------|------|---------|
| 拘传 | 办案机关 | 12-24小时 | 可 |
| 拘留 | 看守所 | 24小时送押 | 立即可 |
| 逮捕 | 看守所 | 2个月+延长 | 可 |
| 监视居住（指定居所） | 居所/指定地点 | 6个月 | 可 |

### 3.2 非羁押型强制措施

| 措施 | 期限 | 限制 |
|-----|------|------|
| 取保候审 | 12个月 | 离开居住地需批准 |
| 监视居住（住处） | 6个月 | 未经批准不得离开 |

### 3.3 羁押必要性审查

| 要点 | 内容 |
|-----|------|
| 时间 | 检察院主动/依申请 |
| 评估因素 | 证据、罪行、情节、可能判罚 |
| 结果 | 继续羁押/释放/变更措施 |

## 四、证据规则

### 4.1 证据种类

| 种类 | 说明 |
|-----|------|
| 物证书证 | 实物证据 |
| 证人证言 | 证人陈述 |
| 被害人陈述 | 被害人陈述 |
| 被告人供述和辩解 | 口供 |
| 鉴定意见 | 专业鉴定 |
| 视听资料/电子数据 | 录音录像/数据 |
| 勘验/检查/辨认/侦查实验 | 过程记录 |

### 4.2 非法证据排除

| 类型 | 情形 | 后果 |
|-----|------|------|
| 硬性排除 | 刑讯逼供、暴力威胁 | 绝对排除 |
| 软性排除 | 重大违法、可能影响公正 | 应当排除 |
| 毒树之果 | 非法证据衍生证据 | 裁量排除 |

### 4.3 证明标准

| 阶段 | 标准 |
|-----|------|
| 立案 | 有犯罪事实 |
| 逮捕 | 有证据证明犯罪事实 |
| 起诉 | 犯罪事实已查清/证据确实充分 |
| 有罪判决 | 排除合理怀疑 |

## 五、升级决策门

### 5.1 必须升级情形

1. **涉及具体刑事程序**
   - 强制措施的异议
   - 非法证据排除申请
   - 证人出庭申请

2. **涉及辩护策略**
   - 罪轻/无罪辩护
   - 程序性辩护
   - 认罪认罚程序选择

3. **涉及与司法机关沟通**
   - 律师会见
   - 阅卷
   - 正式法律文书

### 5.2 升级格式

```
【升级咨询】
程序阶段：[侦查/审查起诉/一审/二审]
程序问题：[具体程序问题]
当前状态：[当前程序状态]
具体问题：[需要帮助的事项]
建议行动：[需律师确认事项]
优先级：紧急/高/中
```

## 六、本Skill不涵盖

1. 具体案件的法律策略（由辩护律师提供）
2. 刑事辩护意见的撰写
3. 非法证据排除的专项策略
4. 涉黑涉恶案件的特别程序
5. 死刑复核程序
6. 减刑假释程序
7. 涉及港澳台的刑事司法互助
8. 刑事和解程序的具体操作
## 本 Skill 不涵盖

- 刑事辩护
- 羁押必要性审查
- 取保候审的具体申请
- 国家赔偿程序
