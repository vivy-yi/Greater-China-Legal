# 商业合同审查判断框架 — 中国大陆

> 本框架用于 contract-review 28 个 skill 的场景识别与判断逻辑，作为合同审查场景的统一依据。
> contract-review 是**合同全生命周期**视角（起草→审查→谈判→签署→履行→变更/终止→争议），不同场景侧重不同环节。
> 各 skill 内部已有判断逻辑，本文件描述跨 skill 的整体决策路由。

## 三、合同类型判断（核心路由）

```
[节点 1] 合同标的？
├─ 货物（动产）→ 买卖合同（民法典合同编第9章）
├─ 服务（劳务）→ 服务合同（民法典合同编第22章）
├─ 不动产（土地/房屋）→ 租赁合同（第14章）
├─ 资金（借款）→ 借款合同（第12章）
├─ 知识产权 → 技术合同（第20章）
├─ 股权/投资 → 股权投资协议
├─ 知识产权许可 → 许可合同
└─ 其他 → 按"服务合同"一般规则（兜底）

[节点 2] 合同相对方？
├─ 法人/企业 → 主体审查（企查查/天眼查）
├─ 自然人 → 主体审查（身份证）
├─ 政府/事业单位 → 特殊审查
└─ 境外主体 → 涉外合同 + 准据法

[节点 3] 争议解决方式？
├─ 约定仲裁 → 仲裁条款审查（仲裁法第 16-18 条）
├─ 约定法院 → 管辖条款审查（民诉法第 35 条）
├─ 未约定 → 法定管辖（被告住所地/合同履行地）
└─ 仲裁 vs 法院选择 → 成本/效率/执行考虑

[节点 4] 适用法律？
├─ 大陆法 → 民法典 + 司法解释
├─ 香港/澳门/台湾 → 准据法（普通法系/大陆法系）
├─ 境外 → 准据法（外国法适用）
└─ 国际条约 → 优先适用
```

## 四、风险分级判断

### 4.1 风险等级判断树

```
[节点 1] 条款是否违反法律强制性规定？
├─ 是（违反效力性强制性规定）→ 🔴 高风险（合同无效）
├─ 是（违反管理性强制性规定）→ 🟡 中风险（合同有效但可能处罚）
└─ 否 → 进入下一步

[节点 2] 条款是否显失公平？
├─ 是（违约金过高 > 实际损失 30%）→ 🟡 中风险（可请求调整）
├─ 是（格式条款加重对方义务）→ 🟡 中风险（民法典第 496-498 条）
└─ 否 → 进入下一步

[节点 3] 条款是否明确可执行？
├─ 是 → 🟢 低风险
├─ 否（关键条款缺失/模糊）→ 🟡 中风险
└─ 否（条款存在歧义）→ 🟡 中风险

[节点 4] 条款是否保护我方核心利益？
├─ 是 → 🟢 低风险
├─ 部分保护 → 🟡 中风险
└─ 否（让渡过多）→ 🔴 高风险
```

### 4.2 升级触发条件

下列情形**必须**升级至律师/法律顾问：

- 标的金额 ≥ 100 万 → 律师复核
- 涉外合同（含港澳台） → 律师主导
- 涉及股权变更/对赌/反稀释 → 律师专项
- 涉及重大资产收购/转让 → 律师专项
- 涉及数据跨境（PIPL 出境）→ 律师 + 数据合规专项
- 涉及上市公司重大事项 → 律师 + 合规专项
- 涉及反垄断申报 → 律师专项
- 涉及关联交易 → 合规 + 律师
- 行业特殊（金融/医疗/建筑等）→ 行业律师

## 五、合同效力判断

### 5.1 合同无效情形（民法典第 146-154 条）

| 情形 | 法条 | 风险 |
|---|---|---|
| 无民事行为能力人签订 | 第 144 条 | 🔴 无效 |
| 违反法律行政法规强制性规定 | 第 153 条 | 🔴 无效 |
| 违背公序良俗 | 第 153 条 | 🔴 无效 |
| 恶意串通损害他人合法权益 | 第 154 条 | 🔴 无效 |
| 以合法形式掩盖非法目的 | 第 146 条 | 🔴 无效 |
| 虚假意思表示（阴阳合同） | 第 146 条 | 🔴 无效 |

### 5.2 合同可撤销情形（民法典第 147-151 条）

| 情形 | 法条 | 风险 |
|---|---|---|
| 重大误解 | 第 147 条 | 🟡 可撤销（90 日内） |
| 欺诈 | 第 148-149 条 | 🟡 可撤销（1 年内） |
| 胁迫 | 第 150 条 | 🟡 可撤销（1 年内） |
| 显失公平 | 第 151 条 | 🟡 可撤销（1 年内） |

### 5.3 合同效力待定情形

- 限制民事行为能力人超出认知范围签订（需法定代理人追认）

## 六、合同解除判断

### 6.1 法定解除（民法典第 563 条）

```
[节点 1] 是否发生根本违约？
├─ 不可抗力致使不能实现合同目的 → 法定解除
├─ 履行期限届满前一方明确表示不履行主要债务 → 法定解除
├─ 一方迟延履行主要债务经催告后合理期限内仍未履行 → 法定解除
└─ 一方违约致使不能实现合同目的 → 法定解除

[节点 2] 是否符合特别规定？
├─ 借款合同（民法典第 673 条）—— 借款人未按约定用途使用借款
├─ 租赁合同（民法典第 729 条）—— 租赁物危及安全/健康
├─ 服务合同（民法典第 934 条）—— 后果严重违背预期
└─ 其他特别规定

[节点 3] 程序是否合规？
├─ 通知义务（民法典第 565 条）—— 解除合同须通知对方
├─ 异议期限（民法典第 565 条）—— 对方有异议须 3 个月内起诉
└─ 履行情况 —— 解除前已履行部分可能需恢复原状
```

### 6.2 约定解除

- 须在合同中明确约定解除条件
- 约定解除条件不得违反强制性规定
- 通知 + 异议程序同法定解除

### 6.3 解除后果

| 情形 | 法律后果 |
|---|---|
| 尚未履行 | 终止履行 |
| 已履行 | 恢复原状（民法典第 566 条） |
| 违约责任 | 违约条款仍可主张 |
| 损害赔偿 | 可主张赔偿损失 |
| 担保 | 担保条款对担保人仍有效 |

## 七、违约金判断

### 7.1 违约金合理性（民法典第 585 条）

```
[节点 1] 约定违约金是否过高？
├─ 是（> 实际损失 30%）→ 可请求法院调整
├─ 否 → 维持约定
└─ 无法判断 → 收集证据 + 法院裁定

[节点 2] 调整幅度？
├─ 实际损失 → 调整为实际损失
├─ 略高于实际损失 → 可适度调整
└─ 过分高于实际损失 → 大幅调整（不超过实际损失 130%）

[节点 3] 实际损失计算？
├─ 直接损失 → 已支出费用
├─ 可得利益损失 → 合同正常履行可获得的利益
└─ 间接损失 → 因违约造成的其他损失
```

### 7.2 违约金条款常见问题

| 问题 | 风险 | 处理 |
|---|---|---|
| 违约金过高（> 30%） | 🔴 | 标注 + 建议调整 |
| 违约金计算基数不明 | 🟡 | 要求明确计算方式 |
| 违约金与赔偿金重复主张 | 🟡 | 不得重复（民法典第 588 条） |
| 迟延履行违约金与解除并存 | 🟡 | 可同时主张 |
| 一次性 vs 按日累计 | 🟢 | 合同约定 |

## 八、格式条款判断（民法典第 496-498 条）

### 8.1 格式条款无效情形

- 提供格式条款一方不合理地免除或者减轻其责任
- 加重对方责任
- 限制对方主要权利
- 提供格式条款一方排除对方主要权利

### 8.2 格式条款特别要求

- **提示说明义务**：提供方应采取合理方式提示对方注意
- **重大利害关系条款**：应以合理方式提示
- **未履行提示义务**：对方可主张该条款不构成合同内容

### 8.3 格式条款解释

- 通常理解解释（民法典第 498 条）
- 不利解释（提供方不利）
- 合同整体解释

## 九、合同变更与转让

### 9.1 合同变更（民法典第 543-544 条）

| 情形 | 法律要求 |
|---|---|
| 协议变更 | 双方协商一致，书面形式（如合同要求） |
| 法定变更 | 法律规定的变更情形（如情势变更） |
| 重大误解/欺诈 | 可请求变更（撤销权） |

### 9.2 合同转让

| 类型 | 法律要求 |
|---|---|
| 债权转让 | 通知债务人即可（民法典第 546 条） |
| 债务转让 | 须债权人同意（民法典第 551 条） |
| 概括转让 | 须对方同意（民法典第 552 条） |
| 合同权利义务一并转让 | 须对方同意（民法典第 555 条） |

### 9.3 情势变更（民法典第 533 条）

```
适用条件：
1. 合同成立后
2. 发生了当事人在订立合同时无法预见的重大变化
3. 不属于商业风险
4. 继续履行明显不公平

效果：
- 受不利影响方可与对方重新协商
- 协商不成可请求法院或仲裁机构变更/解除
```

## 十、跨境/涉外合同

### 10.1 准据法选择

```
[节点 1] 当事人是否约定准据法？
├─ 是 → 依约定（但不得违反中国强制性规定/公共政策）
├─ 否 → 进入下一步

[节点 2] 最密切联系原则
├─ 合同签订地 → 适用
├─ 合同履行地 → 适用
├─ 标的物所在地 → 适用
└─ 被告住所地 → 兜底

[节点 3] 中国强制性规定
├─ 涉及中国自然资源/基础设施 → 中国法
├─ 涉及外商投资准入 → 中国法（外商投资法）
├─ 涉及数据跨境 → 中国法（PIPL/DSL）
└─ 涉及外汇管制 → 中国法
```

### 10.2 涉外合同特殊要求

- **书面形式**（涉外合同法第 32 条）
- **审批/登记**（如技术进出口）
- **外汇结算**（外管局规定）
- **仲裁条款**（涉外仲裁规则）
- **送达条款**（涉外送达）

## 十一、特殊合同类型

### 11.1 SaaS 主服务协议（saas-msa-review）

```
关键审查点：
- 服务水平协议（SLA）
- 数据所有权（重要！）
- 数据安全（PIPL 合规）
- 数据跨境（如适用）
- 知识产权归属
- 责任限制条款
- 续费/终止条款
- 第三方集成责任
```

### 11.2 保密协议（NDA）

```
关键审查点：
- 保密信息范围（明确）
- 保密期限（一般不超过 5 年）
- 违约责任（违约金合理性）
- 适用法律（中国法 vs 境外法）
- 竞业限制条款（需单独约定）
- 关联公司定义（影响范围）
- 例外情形（公知信息/独立开发等）
```

### 11.3 商业租赁合同（commercial-lease-drafter）

```
关键审查点：
- 租赁标的（明确地址/面积）
- 租赁期限（民法典第 705 条：≤ 20 年）
- 租金及调整机制
- 押金（不超过 6 个月租金）
- 转租/分租（须出租人同意）
- 装修/改造（须明确约定）
- 优先购买权（民法典第 726-728 条）
- 维修责任（民法典第 712-713 条）
- 提前解除（违约金）
```

### 11.4 借款合同（loan-agreement-checker）

```
关键审查点：
- 借款用途（不得用于违法活动）
- 利率（不得超过合同成立时一年期 LPR 的 4 倍）
- 还款方式
- 担保方式（保证/抵押/质押）
- 违约责任（逾期利息）
- 提前还款（须明确）
```

### 11.5 国际货物买卖合同（international-sales-advisor）

```
关键审查点：
- 价格条款（CIF/FOB/EXW 等 Incoterms 2020）
- 货物交付（时间/地点/方式）
- 风险转移（Incoterms 规定）
- 单据（发票/提单/原产地证）
- 付款方式（信用证/汇付/托收）
- 检验/索赔期限
- 不可抗力（援引条款）
- 法律适用（CISG 优先 / 中国法补足）
```

## 十三、与其他 plugin 的差异

| 维度 | contract-review | employment-legal | law-student | legal-clinic |
|---|---|---|---|---|
| 目标用户 | 法务/律师/采购 | 律师/HR | 学生 | 法援学生/律师 |
| 输出类型 | 合同审查意见 + 修改建议 | 法律意见 | 知识传递 | 法律服务 |
| 关注重点 | 条款合规性 + 风险点 | 解除合法性 + 补偿金 | 学科知识 | 当事人权益 |
| 数据需求 | 法规 + 实务案例 + 行业惯例 | 法规 + 实务案例 + 地方差异 | 教材 + 真题 | 案件档案 + 当事人信息 |
| 风险等级 | 中-高（合同涉及金额） | 高（影响案件） | 低（学习） | 中-高（影响当事人） |
| 时效要求 | 中（合同周期长） | 高（仲裁 60 日） | 低 | 极高（法援时效紧迫） |

**核心差异**：contract-review 关注**条款的具体表述和风险点**，而不是法律理论或案件时效。
