## 常用查询场景

### 场景1：违约金是否过高

**法律依据：** 民法典第585条（过高可请求法院调整，超过实际损失30%）

**查询语句：**
```
法律精灵："[城市] 违约金过高 调整" 或 "[城市] 违约金 实际损失 30%"
```

**判断标准：**
```
是否过高：违约金 > 实际损失的130% → 可请求法院调整
调整幅度：法院一般调整为实际损失或略高于实际损失
```

---

### 场景2：合同效力判断

**查询语句：**
```
法律精灵："合同效力 无效 情形 民法典" 或 "[具体情形] 合同效力"
```

**常见无效情形（民法典第146-154条）：**
```
- 违反法律行政法规强制性规定
- 违背公序良俗
- 恶意串通损害他人合法权益
- 以合法形式掩盖非法目的
- 无民事行为能力人签订
- 虚假意思表示（阴阳合同）
```

---

### 场景3：管辖法院/仲裁条款

**查询语句：**
```
法律精灵："仲裁条款效力 认定" 或 "[地区] 法院管辖 确定"
```

**判断标准：**
```
合同未约定管辖 → 被告所在地或合同履行地法院
合同约定仲裁 → 须明确仲裁机构，否则无效
约定法院 → 须符合民事诉讼法规定（被告住所地/合同履行地/合同签订地）
约定与争议无实际联系 → 可能被认定无效
```

---

### 场景4：合同解除条件

**查询语句：**
```
法律精灵："[合同类型] 解除条件 法定" 或 "[合同类型] 解除 违约金"
```

**判断标准：**
```
法定解除（民法典第563条）：
- 不可抗力致使不能实现合同目的
- 履行期限届满前一方明确表示不履行主要债务
- 一方迟延履行主要债务经催告后合理期限内仍未履行
- 一方违约致使不能实现合同目的
- 法律规定其他情形

约定解除：
- 须在合同中明确约定
- 约定解除条件不得违反强制性规定
```

---

## 查询结果处理

### 情形一：多源结论一致
→ 高置信度，直接使用

### 情形二：仅有律师分析
→ 中置信度，建议参考但不完全依赖

### 情形三：多源矛盾
→ 呈现差异，标注不确定性，建议保守处理

### 情形四：查不到类似判例
→ 使用法律规定，注明"无类似判例参考"
