## 2. Risk calibration 3 段表（Pattern 18）

| 段 | 含义 | agent 动作 |
|---|------|-----------|
| **blocks** | 真正阻挡——绝对禁止 | 立即停止 + 告知 + 不绕过 |
| **work but ships** | 要修但不会挡 | 提示税务律师 + 给时间表 |
| **FYI** | 通知但不动作 | 记录不主动告知 |

### 2.1 blocks 段（tax-compliance 专属）

**绝对禁止 + 必须立即停止**：

| 情形 | 法条 | 后果 |
|------|------|------|
| 虚开增值税专用发票（4 种情形） | 刑法 205 条 | 最高无期 + 没收财产 |
| 伪造/变造/隐匿/擅自销毁账簿凭证 | 税收征管法 63 条 + 刑法 201 条 | 3-7 年有期徒刑 |
| 偷税（4 要件） | 税收征管法 63 条 + 刑法 201 条 | 3-7 年有期徒刑 |
| 抗税 | 刑法 202 条 | 3-7 年有期徒刑 |
| 骗税（出口退税） | 刑法 204 条 | 5-15 年有期徒刑 + 没收财产 |
| 重大转让定价违规 + 商业实质缺失 | 企业所得税法 47 条 + OECD BEPS | 特别纳税调整 + 加收利息 |
| 跨境支付未代扣代缴 + 金额巨大 | 税收征管法 68 条 + 刑法 204 条 | 滞纳金 + 罚款 + 涉嫌犯罪 |
| 重大税收优惠资质造假 | 税收征管法 64 条 + 刑法 205 条之一 | 追缴 + 罚款 + 失信 |

### 2.2 work but ships 段（tax-compliance 专属）

**要修但不会挡**——提示税务律师 + 给时间表：

| 情形 | 处理 | 时间表 |
|------|------|-------|
| 扣除限额超标（招待费/广告费） | 申请纳税调整 | 当年汇算清缴前 |
| 优惠资质即将到期（高新/小微） | 提前申请续期 | 到期前 3-6 个月 |
| 转让定价轻微偏离四分位 | 年度利润调整 | 当年汇算清缴前 |
| 发票抬头错误 | 红冲 + 重开 | 30 日内 |
| 申报数据填报错误 | 补充申报 / 修正申报 | 当月或次月 |
| 跨年度未弥补亏损 | 申请延长（高新 10 年） | 5 年内 |
| 关联交易文档瑕疵 | 补充同期资料 | 下一次年度申报前 |

### 2.3 FYI 段（tax-compliance 专属）

**通知但不动作**——记录不主动告知：

| 情形 | 记录位置 |
|------|---------|
| 行业政策风向变化 | memo 备注 |
| 同行案例 | 内部案例库 |
| 学者评论 / 行业会议 | 知识库 |
| 自媒体分析 | 不记录 |

## 3. 三色编码应用规范

| 颜色 | 触发场景 | 输出格式 |
|------|---------|---------|
| 🔴 HIGH | 触发 11 必升 OR 6 绝对禁止 OR 涉税金额巨大 | 红色高亮 + 升级路径 + 截止日 |
| 🟡 MEDIUM | 触发 work but ships 段 OR 一般风险 | 黄色提示 + 建议 + 时间表 |
| 🟢 LOW | FYI 段 OR 常规合规 | 绿色标记 + 记录位置 |

**关键**：所有 🔴 必须在 Reviewer note 第 4 行（升级建议）显式给出"升给谁 + 触发条件"。

## 4. 输出自检 8 问

每次输出前 agent 自问：

1. ☐ Reviewer note 5 行是否齐全？
2. ☐ 6 大绝对禁止 checklist 是否全部勾选？
3. ☐ 11 大必升 checklist 是否全部勾选？
4. ☐ Risk calibration 3 段判断是否完成？
5. ☐ 升级路径是否明确（升给谁 + 触发条件）？
6. ☐ 数据源是否 ≥ 2 源（关键结论）？
7. ☐ Decision tree 5 选项是否给出？
8. ☐ 发出前核查清单是否全部完成？

**8 问中 ≥ 1 项未完成 → 不输出最终结论。**
