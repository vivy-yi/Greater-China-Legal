# /tax-preference-application-advisor — China Mainland

## 法律背景与法规框架

中国税收优惠政策体系覆盖企业所得税、增值税、个人所得税等多个税种，涉及产业优惠（高新技术企业/软件集成电路/西部大开发/海南自贸港）、规模优惠（小型微利企业）、活动优惠（研发费用加计扣除/创业投资）、区域优惠（自贸区/粤港澳/海南/上海临港）等多个维度。税收优惠的适用以"自行判别、申报享受、相关资料留存备查"为原则，不再需要事前审批。

2024年以来，税收优惠管理的趋势是"放管服"改革：企业自行判断是否符合条件，自行在申报表中填写享受，将证明材料留存备查。但这不意味着风险降低——税务机关在后续管理（30%的抽查率和实地核查）中发现企业不符合条件的，将要求补税并加收滞纳金，情节严重的可能涉及罚款。因此，准确判断适用条件和保留完整证明材料是享受税收优惠的关键。

## 工作流程

### 第一步：优惠资格筛查

按企业基本特征筛选可能的优惠：

```
企业所得税优惠筛查：
├─ 高新技术企业（15%税率）
│   ├─ 条件：高新领域+研发投入+高新收入+科技人员
│   └─ 认定：科技部/财政/税务联合认定
├─ 小型微利企业（分段优惠税率）
│   ├─ 条件：资产<5000万+人数<300+应纳税所得额<300万
│   └─ 优惠：≤100万部分约2%，100-300万部分5%
├─ 研发费用加计扣除（100%加计扣除）
│   ├─ 条件：真实性+不属于负面清单行业
│   └─ 2024年起：100%加计扣除（制造业/科技型中小企业）
├─ 西部大开发（15%税率）
│   ├─ 条件：西部省份+鼓励类产业目录+收入占比>60%
│   └─ 至2030年底有效
├─ 软件企业/集成电路企业
│   ├─ "两免三减半"或"五免五减半"
│   └─ 须符合软件企业/集成电路企业认定条件

增值税优惠筛查：
├─ 软件产品即征即退（实际税负>3%部分即征即退）
├─ 资源综合利用即征即退
├─ 技术转让/技术开发免征
├─ 小微企业月销售额10万以下免征
└─ 集成电路企业/软件企业税收优惠
```

### 第二步：条件逐项确认

**高新技术企业认定条件（重点优惠）：**

```
高新认定硬性条件检查（国科发火[2016]195号）：
1. 知识产权：核心知识产权（发明/实用新型/软著/布图设计等）
   □ 1项及以上发明（直接计入满足量）
   □ 6项及以上实用新型/软著/布图设计等
2. 技术领域：属于国家重点支持的高新技术领域（八大领域）
   □ 电子信息/生物与新医药/航空航天/新材料/高技术服务/新能源与节能/
     资源与环境/先进制造与自动化
3. 科技人员占比：企业当年职工总数中科技人员占比≥10%
   □ 科技人员：[人数] / 职工总数：[人数] = [%]
4. 研发费用占比（近三个会计年度）：
   □ 销售收入<5000万：研发费用占比≥5%
   □ 5000万≤收入<2亿：研发费用占比≥4%
   □ 收入≥2亿：研发费用占比≥3%
5. 高新收入占比：高新收入占企业同期总收入比例≥60%
   □ 高新收入：[金额] / 总收入：[金额] = [%]
6. 无重大安全/质量/环境事故
```

**研发费用加计扣除备查材料清单：**
```
□ 研发项目立项决议
□ 研发项目计划书和预算
□ 研发人员名单（姓名/岗位/参与项目）
□ 研发费用归集表（人员人工/直接投入/折旧/设计试验/其他）
□ 研发支出辅助账
□ 研发成果报告
□ 委托研发合同（如涉及委托外部机构）
```

### 第三步：优惠叠加与选择

当多个优惠同时适用时，须注意优惠的叠加规则：

| 优惠A | 优惠B | 叠加规则 |
|-------|-------|---------|
| 高新技术企业（15%） | 小型微利企业（分段优惠） | 可选最优，不能叠加 |
| 高新技术企业（15%） | 西部大开发（15%） | 可选最优 |
| 研发加计扣除 | 高新或小型微利 | 可叠加（先加计扣除再计算应纳税所得额） |
| "两免三减半" | 高新技术企业 | 选其中一个（通常在优惠期结束后转高新） |
| 软件产品即征即退 | 高新技术企业 | 可同时享受（增值税和企业所得税不同税种） |

**选择建议：**
- 如果同时符合高新和小微，计算两个方案的实际税负后择优选择
- 高新+研发加计扣除的组合通常是税负最优方案
- 软件企业优惠期（两免三减半）结束后可转高新

### 第四步：申请/备案流程

**事前备案（需先获得资格的优惠）：**

| 优惠类型 | 认定机构 | 申请周期 | 有效期 |
|---------|---------|---------|-------|
| 高新技术企业 | 科技部门 | 3-6个月 | 3年 |
| 软件企业 | 工信部门 | 2-4个月 | 每年复审 |
| 集成电路企业 | 发改委/工信部 | 3-6个月 | 视资质 |
| 西部大开发鼓励类目录 | 自行判断 | — | 年度适用 |

**自行享受（无需事前审批但须留存资料）：**
- 研发费用加计扣除：申报时自行计算，材料留存备查
- 小型微利企业：在申报表中自动判断并适用
- 技术转让免征增值税：自行判断（建议做技术合同登记）
- 软件产品即征即退：申报后申请退税

**后续管理风险：**
- 税务机关每年对享受优惠企业进行抽查（约30%）
- 如认定不符合条件：
  - 补缴税款
  - 加收滞纳金（每天万分之五）
  - 情节严重的，按税收征管法处罚

## 升级决策门

以下情况须移交专业税务律师处理：

- 高新技术企业认定申请被主管部门退回（须分析原因并重新准备）
- 享受税收优惠后被税务机关质疑条件不符合
- 多个税收优惠叠加适用存在冲突或选择困难
- 企业重组/并购导致税收优惠资格承继问题
- 涉及税收优惠政策的追溯调整或执法口径变化
- 研发费用加计扣除被税务机关核查或调查
- 软件企业/集成电路企业税收优惠的特殊认定问题
