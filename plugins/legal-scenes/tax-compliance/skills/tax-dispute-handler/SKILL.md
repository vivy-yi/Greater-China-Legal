# /tax-dispute-handler — 税务争议处理

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取税务救济途径/复议规则。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取救济途径/程序
2. 判断税务争议类型
3. 制定应对策略
4. 选择救济路径
5. 输出处理建议

## 税务争议类型

| 类型 | 性质 | 触发方式 |
|---|---|---|
| 纳税评估 | 日常管理 | 税局主动发起 |
| 税务稽查 | 重点检查 | 税局主动发起（有证） |
| 税务检查 | 一般检查 | 税局主动发起（日常） |
| 税务处罚 | 行政处罚 | 违反税法 |
| 税务复议 | 行政救济 | 当事人对决定不服 |

## 税务稽查应对

```
稽查程序：
1. 出示证件（税务检查证）+ 下达检查通知书
2. 实施检查（查账/实地检查/询问）
3. 制作笔录
4. 形成稽查报告
5. 审理部门审理
6. 送达处理处罚决定书

应对要点：
1. 核实检查人员身份和文件
2. 配合但不过度披露
3. 律师尽早介入
4. 关注追诉时效（通常3年，重大可延长）
5. 重视听证权利（重大处罚前有听证权）
```

## 税务行政复议

```
前置条件：须先缴清税款或提供担保，方可申请复议

复议机关：
- 同级政府或上级税务机关
- 对国税局决定 → 上级国税局或同级政府

复议时效：60日（自知道决定之日起）

复议优势：
- 不收费
- 审查合法性
- 可调解（赔偿部分可调解）
```

## 税务行政诉讼

```
诉讼时效：15日（复议后不服）或6个月（直接起诉）

管辖：被告所在地中级人民法院

审理内容：合法性审查（不审查合理性）

注：逃税等刑事案件须先经刑事程序
```
