# /tax-treaty-application-advisor — China Mainland

## 法律背景与法规框架

中国已与超过110个国家和地区签署了避免双重征税协定（DTA/税收协定）。税收协定的核心功能是：在跨境交易中合理分配征税权，降低预提所得税税率，并消除双重征税。企业所得税法第58条规定，税收协定优先于国内法适用。这意味着当税收协定规定的税率低于国内法税率时，纳税人有权适用协定待遇。

享受协定待遇的核心机制是"受益所有人"（Beneficial Owner）认定。国家税务总局公告2019年第35号建立了新的"自行判断、申报享受、资料留存备查"的协定待遇管理模式。纳税人自行判断是否符合享受协定待遇的条件，在申报时直接按协定税率纳税（或申请退税），并将证明资料留存备查。税务机关通过后续管理进行核查。如果后续管理中被认定为不符合条件的，将面临补税和滞纳金。

## 工作流程

### 第一步：协定适用性判断

确认所得类型和协定条款适用：

```
协定覆盖的所得类型：
├─ 股息（Dividends）——协定通常将税率降至5%-10%
├─ 利息（Interest）——协定通常将税率降至7%-10%
├─ 特许权使用费（Royalties）——协定通常将税率降至6%-10%
├─ 不动产收益（Immovable Property）——多为来源国征税
├─ 财产转让收益（Capital Gains）——涉及公司股份转让
├─ 独立个人劳务（Independent Personal Services）
├─ 非独立个人劳务（Dependent Personal Services）
├─ 教师和研究人员
├─ 学生和实习人员
└─ 其他所得

判断是否有适用协定：
├─ 是否与中国签署税收协定？→ https://www.chinatax.gov.cn/协定查询
├─ 收款方是否为协定对方税收居民？
├─ 收款方是否为中国税收居民？
├─ 所得类型是否在协定覆盖范围内？
└─ 是否有协定限制条款（如LLC反滥用条款等）
```

### 第二步：享受协定待遇的核心条件

**条件一：税收居民身份**

- 境外收款方须为协定对方国家的税收居民
- 须提供对方主管税务机关出具的《税收居民证明》（CO/TRC）
- 中国境内方也可以申请享受对方国家的协定待遇

**条件二：受益所有人认定（最核心也最具争议性）**

受益所有人是指对所得或所得据以产生的权利或财产具有所有权和支配权的人。国税总局公告2012年第30号列举了不利于受益所有人认定的因素：

```
"打钩"测试——不利于受益所有人认定的情形：
□ 收款方有义务在规定时间内将所得支付给第三国居民
□ 收款方从事的经营活动不构成实质性经营活动
□ 收款方的资产、规模和人员配置与所得金额不匹配
□ 收款方对所得或所得产生的权利没有实际控制权
□ 收款方在申请前12个月内注销或变更控制权
□ 收款方所在国对境外所得不征税或税率极低（避税地）

以上情形越多 → 受益所有人认定风险越高
以上情形均不满足 → 受益所有人认定基本安全
```

### 第三步：协定税率与国内法税率对比

**常用协定税率参考表（以中国与主要贸易伙伴为例）：**

| 协定对方 | 股息（持股门槛） | 利息 | 特许权使用费 |
|---------|---------------|------|-------------|
| 国内法（无协定） | 10% | 10% | 10% |
| 香港 | 5%（持股≥25%）/10% | 7% | 7% |
| 新加坡 | 5%（持股≥25%）/10% | 7%/10% | 6%/10% |
| 美国 | 10% | 10% | 10% |
| 英国 | 5%（持股≥25%）/10% | 10% | 6%/10% |
| 日本 | 10% | 10% | 10% |
| 德国 | 5%（持股≥25%）/10% | 10% | 10% |
| 法国 | 5%（持股≥25%）/10% | 10% | 10% |
| 韩国 | 5%（持股≥25%）/10% | 10% | 10% |
| 荷兰 | 5%（持股≥25%）/10% | 10% | 6%/10% |

注：具体适用税率须以各协定原文为准。部分协定设有持股比例和持股期限要求。

### 第四步：申请流程（2019年第35号公告模式）

**境内支付方扣缴时直接适用协定税率：**
```
支付方扣缴流程：
1. 接收方提供《税收居民证明》（CO/TRC）
2. 接收方提供《非居民纳税人享受协定待遇信息报告表》
3. 支付方按协定税率扣缴（而非国内法税率）
4. 支付方在纳税申报中填报协定待遇信息
5. 双方各自保存资料备查（后续管理期限5年）
```

**退税模式（如已按国内法税率多缴）：**
```
1. 境内支付方原按国内法10%扣缴
2. 接收方向主管税务机关申请退税（差额部分）
3. 须提供：CO/TRC + 完税证明 + 协定待遇报告表
4. 退税期限：自多缴税款之日起三年内
5. 退税处理时间：通常30-60个工作日
```

**留存备查材料：**
- 协定对方居民身份证明（CO/TRC，须为协定年度）
- 受益所有人认定相关资料
- 合同或协议
- 股息/利息/特许权使用费的支付凭证
- 持股证明（股息适用协定优惠税率须证明持股比例和期限）

## 升级决策门

以下情况须移交专业税务律师处理：

- 受益所有人认定存在被质疑风险（收款方缺乏实质性经营活动）
- 税收协定待遇申请被税务机关拒绝
- 涉及"择协避税"（Treaty Shopping）或"协定滥用"问题
- 涉及LLC等特殊实体的税收协定适用
- 涉及税收协定与国内法冲突（如反避税条款优先）
- 涉及税收协定的"利益限制条款"（LOB）适用
- 涉及跨境重组的税收协定影响（如股权对价交易的免税待遇）
- 国家税务总局启动特别纳税调查或一般反避税调查
