### § A8.1 监管合规特别注意 4 大块（行政处罚 + 听证 + 信用 + 重大违法）

> **核心原则**：监管合规的核心是**许可完备 + 持续合规 + 危机响应**。"侥幸过关"不能替代"系统建设"。

#### 块 1：行政处罚程序（《行政处罚法》2021 修订）

**核心法规：**
- 《行政处罚法》（2021 年 1 月 22 日修订 + 2021 年 7 月 15 日生效）
- 《关于规范行政处罚裁量权的指导意见》
- 《行政处罚听证办法》（2021 修订）

**行政处罚 5 类（《行政处罚法》第 9 条）：**

| 类别 | 含义 |
|------|------|
| **警告** | 声誉处罚 |
| **罚款** | 财产处罚 |
| **没收违法所得 / 非法财物** | 财产处罚 |
| **责令停产停业** | 行为处罚 |
| **吊销许可证件** | 资格处罚 |

**行政处罚程序 3 阶段：**

| 阶段 | 程序 |
|------|------|
| **1. 立案 / 调查** | 立案 + 调查取证 + 告知 |
| **2. 决定** | 简易 / 一般 / 听证程序 |
| **3. 执行** | 自觉履行 / 申请复议 / 诉讼 |

**听证适用 4 类情形（《行政处罚法》第 63 条）：**

| 情形 | 罚款下限 |
|------|---------|
| **较大数额罚款** | 5 万 - 50 万（按省级规定）|
| **没收较大数额违法所得** | — |
| **吊销许可证件** | — |
| **责令停产停业** | — |

**blocks（绝对禁止 — 行政处罚程序）**：
- 拒绝 / 阻碍调查 → **《行政处罚法》第 55-56 条 + 罚款升级**
- 伪造 / 隐匿证据 → **罚款升级 + 涉嫌刑事**
- 听证前未告知 → **程序违法 + 决定撤销**

**work but ships（可补救）**：
- 配合调查 → 30 日内补充材料
- 听证申请 → 5 日内申请

#### 块 2：信用监管 + 黑名单

**核心法规：**
- 《社会信用体系建设法（草案）》（2024 公开征求意见）
- 《关于对失信被执行人实施联合惩戒的合作备忘录》
- 《重大税收违法失信主体信息披露办法》

**信用监管 3 类措施：**

| 措施 | 适用 |
|------|------|
| **失信被执行人** | 限制高消费 + 限制出境 |
| **重大税收违法** | 公布 + 联合惩戒 |
| **严重质量违法** | 列入严重违法失信名单 |

**黑名单 5 类后果（《公司法》/ 招投标 / 政府采购）：**

| 后果 | 期限 |
|------|------|
| **限制高消费** | 持续到义务履行 |
| **限制出境** | 持续到义务履行 |
| **限制招投标** | 1-3 年 |
| **限制政府采购** | 1-3 年 |
| **限制融资贷款** | 持续到信用修复 |

**blocks（绝对禁止 — 信用）**：
- 拒不履行生效法律文书 → **列入失信被执行人 + 联合惩戒**
- 偷税漏税 → **重大税收违法失信 + 公布**
- 严重质量违法 → **列入严重违法失信名单 + 联合惩戒**

**work but ships（可补救）**：
- 信用修复 → 履行义务 + 申请修复
- 异议 → 信用异议 + 复议 / 诉讼

#### 块 3：双随机 + 信用风险分类监管

**核心法规：**
- 《关于在市场监管领域全面推行"双随机、一公开"监管的意见》（2019）
- 《企业信用风险分类管理办法》（市监总局 2022）

**双随机监管 5 步：**

| 步骤 | 内容 |
|------|------|
| 1. **建立随机抽查事项清单** | 监管事项清单 |
| 2. **建立执法人员名录库** | 执法人员 + 专家库 |
| 3. **建立检查对象名录库** | 监管对象库 |
| 4. **随机抽取检查对象 + 执法人员** | 双随机 |
| 5. **公开抽查结果** | 一公开 |

**信用风险分类（4 类）：**

| 类别 | 监管措施 |
|------|---------|
| **A 类（低风险）** | 减少抽查频次（年度 1 次）|
| **B 类（一般风险）** | 常规抽查（半年 1 次）|
| **C 类（较高风险）** | 提高抽查频次（季度 1 次）|
| **D 类（高风险）** | 重点监管 + 不定期抽查 |

**blocks（绝对禁止 — 监管应对）**：
- 拒绝 / 阻碍双随机检查 → **信用降级 + 联合惩戒**
- 隐瞒真实情况 → **信用降级 + 处罚升级**
- 拒不整改 → **列入异常经营名录 / 严重违法名单**

**work but ships（可补救）**：
- 配合检查 → 立即准备材料
- 整改 → 30 日内提交整改报告

#### 块 4：互联网 / 数据 / 跨境专项监管

**核心法规：**
- 《电子商务法》（2019 + 2024 修订）
- 《网络安全法》（2017 + 2024 修订）
- 《数据安全法》（2021）
- 《个人信息保护法》（2021）

**互联网平台 5 类责任：**

| 责任 | 法源 |
|------|------|
| **主体登记** | 《电商法》第 9 条 |
| **公示义务** | 主体信息 + 规则 |
| **平台内经营者审核** | 主体资质核验 |
| **消费者权益保护** | 平台连带责任（生命健康 / 其他权益）|
| **数据报送** | 监管机关数据报送义务 |

**数据合规 5 大领域：**

| 领域 | 监管机关 |
|------|---------|
| **网络安全** | 公安部 + 网信办 |
| **数据安全** | 网信办 + 行业主管 |
| **个人信息** | 网信办 + 行业主管 |
| **关键信息基础设施** | 网信办 + 行业主管 |
| **跨境数据** | 网信办（安全评估 / 标准合同 / 认证）|

**blocks（绝对禁止 — 互联网监管）**：
- 未取得许可从事互联网业务 → **责令停业 + 罚款 + 刑事**
- 数据泄露未报告 → **100 万以下 + 责令停业 + 刑责**
- 100 万人+ 出境未评估 → **5000 万以下 + 责令停业**

**work but ships（可补救）**：
- 许可过期 → 30 日内补办
- 数据安全漏洞 → 立即修复 + 报告

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **无证经营 / 超出许可范围** | 《行政许可法》第 81 条 | 责令改正 + 罚款 + 刑事 |
| 2. **拒绝 / 阻碍调查** | 《行政处罚法》第 55-56 条 | 罚款升级 + 信用降级 |
| 3. **失信被执行 + 拒不履行** | 《民事诉讼法》第 255 条 | 列入失信 + 限制高消费 |
| 4. **重大数据泄露未报告** | 《个保法》第 57 条 | 100 万以下 + 责令停业 + 刑责 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时限
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — 监管合规增强版）**：
- 行业 + 监管机关？（核心监管框架）
- 业务类型 + 许可状态？（许可完备性）
- 是否涉外？（多法域协调）
- 重大违规 + 群体性？（危机响应）
- 数据 / 网络 / 平台？（互联网专项监管）
- 信用风险类别（A/B/C/D）？（监管频率）

### § B14 Risk calibration

**段 1 识别:** 合规差距 / 违规历史 / 处罚风险 / 涉外风险 / 声誉风险
**段 2 量化:** 概率 × 影响 + 缓释
**段 3 响应:** 接受 / 缓释(整改)/ 转移(认证)/ 规避(退出)

### § B15 7 条设计哲学(合规特化)

1. **预防 > 救济** — 合规建设成本低
2. **主动合规 > 被动应对** — 主动识别
3. **多领域协同** — 跨部门合规
4. **持续监控** — 静态合规不够
5. **高管层参与** — 自上而下
6. **记录留痕** — 应对检查
7. **危机预案** — 重大违规预案
