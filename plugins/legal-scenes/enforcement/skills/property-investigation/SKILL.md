# /property-investigation — 财产调查

## 加载上下文
读取 `../CLAUDE.md` § B1 Step 2 + § A4 集成 + § B5 升级。

## 工作流程
```
Step 1: 已知财产线索梳理
Step 2: 未知财产线索发现(法院 + 律师调查)
Step 3: 申请法院调查令
Step 4: 调查成果整理 + 提供法院
```

## 财产线索 8 类

| 类别 | 调查方法 |
|------|----------|
| **银行存款** | 法院查询(银行账号) |
| **不动产** | 不动产登记中心查询 |
| **车辆** | 车管所查询 |
| **股权** | 国家企业信用信息公示系统 |
| **证券账户** | 证券公司 / 中证登 |
| **保险** | 保险公司查询 |
| **应收账款** | 第三方债务人 |
| **知识产权 / 应收账款** | 知识产权局 / 商业调查 |

## 法院调查令申请

**适用条件(《高法执行规定》):**

```
✅ 申请执行人无法自行调查
✅ 涉及被执行人财产线索
✅ 律师调查受限
```

**申请材料:**
- 调查令申请书
- 调查线索(被调查财产)
- 调查机关(银行 / 房管 / 车管 / 工商)

## 律师调查(直接)

**律师可独立调查:**

| 方式 | 适用 |
|------|------|
| 国家企业信用信息公示系统 | 工商档案 / 股权 |
| 中国裁判文书网 | 涉诉情况 |
| 中国执行信息公开网 | 失信 / 限高 |
| 国家知识产权局 | 商标 / 专利 |
| 公开新闻 / 公告 | 重大财产线索 |

**律师调查受限:**

| 不可直接 | 须通过法院 |
|----------|------------|
| 银行存款 | ✅ |
| 不动产 | ✅ 部分 |
| 车辆 | ✅ |
| 通讯记录 | ❌ |
| 银行流水 | ❌ |

## 调查成果使用

**法律保护:**

```
✅ 律师调查成果:律师工作成果保护
✅ 法院调查:直接作为执行依据
✅ 第三人提供:可作证据
```

**特别注意:**

```
⚠️ 不得贿赂 / 威胁被调查人
⚠️ 不得伪造财产线索
⚠️ 不得侵犯商业秘密
```
