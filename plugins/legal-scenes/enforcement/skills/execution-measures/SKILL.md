# /execution-measures — 执行措施

## 工作流程
```
Step 1: 评估适用措施(对应执行内容)
Step 2: 评估措施组合(多措施)
Step 3: 申请执行 + 跟踪
```

## 执行措施 9 类

| 措施 | 适用 | 注意 |
|------|------|------|
| **查询 / 冻结 / 划拨** | 银行存款 | 法院依职权 |
| **查封 / 扣押** | 动产 / 不动产 | 法院裁定 |
| **拍卖 / 变卖 / 抵债** | 已查封财产 | 公开拍卖 |
| **提取被执行人收入** | 工资 / 退休金 | 留必要生活费 |
| **搜查** | 隐匿财产 | 法院依职权 |
| **强制迁出 / 退出** | 不动产 | 法院强制 |
| **拘传 / 拘留** | 拒不履行 | 15 日内 |
| **罚款** | 拒不履行 | 法院裁量 |
| **失信被执行人名单** | 长期拒不履行 | 联合惩戒 |

## 措施组合

| 场景 | 推荐组合 |
|------|----------|
| 银行存款 | 查询 → 冻结 → 划拨 |
| 不动产 | 查封 → 评估 → 拍卖 |
| 车辆 | 查封 → 拍卖 |
| 股权 | 冻结 → 评估 → 拍卖 |
| 综合 | 财产调查 + 多措施并用 |

## 失信被执行人名单(《最高法规定》)

**纳入条件:**

```
✅ 有履行能力而拒不履行
✅ 妨碍执行 / 抗拒执行
✅ 规避执行
✅ 违反财产报告制度
✅ 其他失信行为
```

**联合惩戒:**

| 措施 | 影响 |
|------|------|
| 限制高消费(限高) | 飞机 / 高铁 / 星级酒店 |
| 限制出境 | 边检 + 护照 |
| 限制贷款 | 金融机构 |
| 限制任职 | 担任高管 / 法定代表人 |
| 限制招投标 | 政府采购 |
| 限制子女高消费 | 配偶 / 子女 |

## 拒不执行判决罪(《刑法》第 313 条)

**构成要件:**

```
✅ 对人民法院判决有能力执行而拒不执行
✅ 情节严重
```

**情节严重情形:**

| 情形 | 后果 |
|------|------|
| 隐匿 / 转移财产 | ✅ 情节严重 |
| 拒不申报财产 | ✅ 情节严重 |
| 暴力阻碍执行 | ✅ 情节严重 |
| 担保人拒不履行 | ✅ 情节严重 |

**刑罚:** 3 年以下有期徒刑 / 罚金;特别严重 3-7 年。

## 注意事项

```
⚠️ 过度执行(超出执行依据) → 赔偿
⚠️ 错误执行第三人财产 → 赔偿
⚠️ 强制措施须严格依法
⚠️ 拘留须保障被执行人基本生活
```
