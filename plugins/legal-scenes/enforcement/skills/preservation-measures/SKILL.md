# /preservation-measures — 财产保全

## 工作流程
```
Step 1: 评估保全时机(诉前/诉中/执行前)
Step 2: 评估保全财产范围
Step 3: 评估担保(诉前 / 涉外)
Step 4: 申请 + 跟进
```

## 保全 3 类时机

| 时机 | 适用 | 担保 |
|------|------|------|
| **诉前保全**(《民诉法》第 104 条) | 紧急情况 | 须提供担保 |
| **诉中保全**(第 103 条) | 诉讼进行中 | 可不提供(法院裁定) |
| **执行前保全**(第 167 条) | 仲裁前 / 执行前 | 视情况 |

## 保全财产范围

| 类型 | 注意 |
|------|------|
| 银行存款 | 限额(必要金额) |
| 不动产 | 整体查封 / 部分 |
| 股权 | 全部 / 部分 |
| 动产 | 扣押 / 查封 |
| 应收账款 | 冻结债权 |

**过度保全风险:** 保全范围超出请求范围 → 赔偿损失。

## 申请材料

```
1. 财产保全申请书
2. 担保(诉前须提供)
3. 申请执行依据
4. 紧急情况说明(诉前)
```
