### § A8.1 公司治理特别注意 4 大块（公司法 2023 + 上市公司治理 + 股东权益 + 董监高责任）

> **核心原则**：公司治理的核心是**股东会 / 董事会 / 监事会三会分立 + 董监高忠实勤勉义务 + 信息披露合规**。"内部决策"必须程序合规，否则决议可撤销。

#### 块 1：《公司法》2023 修订重大变化

**核心法规：**
- 《公司法》（2023 年 12 月修订 + 2024 年 7 月 1 日生效）
- 《公司法司法解释（一）》（2024 修订）
- 《关于适用〈公司法〉若干问题的规定》

**2023 修订 5 大变化：**

| 变化 | 内容 | 实务影响 |
|------|------|---------|
| **注册资本 5 年实缴** | 有限责任公司股东 5 年内实缴 | 新公司法强制 |
| **股东失权制度** | 未实缴部分股权可失权 | 股东保护 + 债权人保护 |
| **法定代表人责任** | 任职 / 辞任均需登记 | 防止挂名 |
| **资本公积金弥补亏损** | 不得用于弥补亏损 | 资本维持 |
| **类别股制度** | 允许设置类别股 | 创业公司灵活融资 |

**注册资本 5 年实缴要点：**

| 类型 | 实缴期限 |
|------|---------|
| **新设公司** | 5 年内实缴 |
| **存量公司** | 过渡期（2024.7.1 - 2027.6.30）+ 调整期 |
| **存量过渡** | 国务院另行规定 |

**blocks（绝对禁止 — 公司法）**：
- 应实缴未实缴 → **股东失权 + 民事赔偿**
- 虚假出资 / 抽逃出资 → **《公司法》第 252-253 条 + 《刑法》第 159 条**
- 提供虚假登记材料 → **《公司法》第 250 条 + 罚款 + 撤销登记**

**work but ships（可补救）**：
- 实缴期限临近 → 减资 + 重新约定
- 股东失权 → 减资程序 + 通知债权人

#### 块 2：上市公司治理（《上市公司治理准则》2024）

**核心法规：**
- 《上市公司治理准则》（2002 + 2024 修订）
- 《上市公司章程指引》（2024 修订）
- 《上市公司独立董事管理办法》（2023 修订）
- 《上市公司董事会秘书工作规则》

**上市公司治理 5 大结构：**

| 结构 | 职责 |
|------|------|
| **股东大会** | 最高权力机构（修改章程 / 增减资 / 合并 / 分立 / 利润分配）|
| **董事会** | 决策机构（执行股东大会决议 / 决定经营计划）|
| **监事会** | 监督机构（监督董事 / 高管行为）|
| **独立董事** | 独立监督（占比 ≥ 1/3，重大事项独立意见）|
| **董事会秘书** | 协调 / 信息披露 |

**独立董事 5 类重大事项必发表意见：**

| 事项 | 独立意见 |
|------|---------|
| 关联交易 | 事前认可 + 独立意见 |
| 重大资产重组 | 独立意见 |
| 高管薪酬 | 独立意见 |
| 变更募集资金用途 | 独立意见 |
| 会计政策 / 估计变更 | 独立意见 |

**blocks（绝对禁止 — 上市公司）**：
- 关联交易未回避表决 → **决议无效 + 民事赔偿**
- 信息披露虚假 → **100 万 - 1000 万 + 市场禁入**
- 内幕信息交易 → **1-10 倍违法所得 + 5 年禁入 + 刑责**

**work but ships（可补救）**：
- 治理结构不完善 → 30 日内补建
- 信息披露不完整 → 立即补充披露

#### 块 3：股东权益保护 + 股东代表诉讼

**核心法规：**
- 《公司法》第 22 条（决议瑕疵）
- 《公司法》第 188 条（股东代表诉讼）
- 《公司法司法解释（一）》

**股东代表诉讼触发（双重前置）：**

| 步骤 | 内容 |
|------|------|
| 1. **股东书面请求** | 股东向监事会 / 不设监事会的监事 |
| 2. **60 日内未起诉** | 监事会 / 监事拒绝 / 30 日内未起诉 / 情况紧急 |
| 3. **股东自行起诉** | 股东以自己名义为公司起诉 |

**股东代表诉讼资格：**

| 主体 | 资格 |
|------|------|
| **有限责任公司股东** | ✅ 任何股东 |
| **股份有限公司股东** | 连续 180 日持股 ≥ 1% |

**决议瑕疵 3 类：**

| 类型 | 法律后果 |
|------|---------|
| **决议内容违法** | 无效 |
| **召集程序 / 表决方式违法** | 可撤销（60 日内起诉）|
| **决议内容违反章程** | 可撤销 |

**blocks（绝对禁止 — 股东权益）**：
- 控股股东滥用控制权 → **《公司法》第 21 条 + 民事赔偿**
- 关联交易未披露 → **民事赔偿 + 决议可撤销**
- 拒绝小股东查阅会计账簿 → **《公司法》第 57 条 + 民事赔偿**

**work but ships（可补救）**：
- 程序瑕疵 → 重新召集 + 表决
- 信息披露不全 → 立即补充

#### 块 4：董事 / 监事 / 高管忠实勤勉义务

**核心法规：**
- 《公司法》第 180-188 条（董监高义务 + 责任）
- 《公司法》第 188 条（股东代表诉讼针对董监高）
- 《刑法》第 169 条（徇私舞弊低价折股罪）

**忠实义务 5 类禁止（《公司法》第 180 条 / 第 181 条）：**

| 禁止 | 含义 |
|------|------|
| **挪用资金** | 不得挪用公司资金 |
| **违规借款** | 不得违反章程规定向股东 / 实际控制人提供借款 |
| **同业竞争** | 不得未经股东会同意从事同业经营 |
| **关联交易** | 不得未经股东会同意与公司交易 |
| **谋取商业机会** | 不得谋取属于公司的商业机会 |

**勤勉义务 5 类要求：**

| 要求 | 含义 |
|------|------|
| **了解公司经营状况** | 关注公司经营 + 财务 |
| **参加董事会** | 出席 + 表决 + 监督 |
| **做出合理商业判断** | 符合一般理性人标准 |
| **报告重大事项** | 及时披露重大事件 |
| **保守商业秘密** | 不得泄露公司秘密 |

**董事责任 3 类：**

| 责任 | 触发 |
|------|------|
| **民事赔偿** | 违反义务 + 公司损失 |
| **股东代表诉讼** | 60 日内公司未起诉 + 股东自行 |
| **刑事责任** | 徇私舞弊 / 国有公司低价折股 / 受贿 |

**blocks（绝对禁止 — 董监高）**：
- 挪用资金 → **《公司法》第 184 条 + 《刑法》第 272 条**
- 同业竞争 → **《公司法》第 184 条 + 收入归入公司**
- 关联交易未披露 → **民事赔偿 + 决议可撤销**
- 国有公司低价折股 → **《刑法》第 169 条（徇私舞弊 + 国有公司）**

**work but ships（可补救）**：
- 程序瑕疵 → 重新决议
- 信息披露不全 → 立即补充披露
- 同业竞争 → 取得股东会同意

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **虚假出资 / 抽逃出资** | 《公司法》第 252-253 条 + 《刑法》第 159 条 | 责令补足 + 罚款 + 刑责 |
| 2. **关联董事未回避** | 《公司法》第 124 / 185 条 | 决议无效 + 民事赔偿 |
| 3. **董监高挪用资金** | 《公司法》第 184 条 + 《刑法》第 272 条 | 民事赔偿 + 3 年以下 |
| 4. **信息披露虚假** | 《证券法》第 193 条 | 100 万 - 1000 万 + 市场禁入 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时限
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — 公司治理增强版）**：
- 公司类型（有限 / 股份 / 上市 / 国有）？（治理结构）
- 股东 / 董事会决议合法性？（决议程序）
- 关联交易是否回避表决 + 披露？（决议有效性）
- 董监高是否违反忠实 / 勤勉义务？（责任风险）
- 是否触及上市公司信披义务？（合规义务）
- 股东代表诉讼是否双重前置？（程序合法性）
