# /profit-sharing-plan

## 目的

为公司利润分配方案提供咨询支持，涵盖：
1. **分红条件审查** — 是否满足法定及章程约定的分红条件
2. **分配方案设计** — 分红比例、形式（现金/股票）、时间安排
3. **决议程序合规** — 股东会/董事会决议的合法有效性
4. **股东权益保护** — 中小股东分红权救济途径
5. **特殊情形处理** — 亏损、滚存利润、回购配合等情形

---

## 法域假设

默认中国大陆法域，适用《公司法》（2024年修订）及证券监管相关规定。

## 第一部分：分红条件审查

### 法定分红条件

|| 条件 | 条款 | 说明 |
|---|---|---|---|
| 实际盈利 | 公司须有实际利润 | 亏损不得分红（第210条） |
| 累计盈利 | 须弥弥补历年亏损并提取公积金 | 先彆亏、后分配 |
| 公积金足额 | 须提取法定公积金（10%达注册资本50%可不再提） | 第210条 |
| 决议通过 | 须经股东会决议通过 | 董事会无权自行决定 |

### 有限责任公司分红

**《公司法》第210条**：公司分配当年税后利润时，应当提取利润的百分之十列入公司法定公积金。公司法定公积金累计额为公司注册资本的百分之五十以上的，可以不再提取。

。公司法定公积金不足以弥补以前年度亏损的，在依照前款规定提取法定公积金之前，应当先用当年利润弥补亏损。

。公司从税后利润中提取法定公积金后，经股东会决议，还可以从税后利润中提取任意公积金。

。公司弥补亏损和提取公积金后所余税后利润，有限责任公司按照股东实缴的出资比例分配，但是公司章程另有规定的除外。

### 股份有限公司分红

**《公司法》第210条**（股份有限公司）：公司分配当年税后利润时，应当提取利润的百分之十列入公司法定公积金。公司法定公积金累计额为公司注册资本的百分之五十以上的，可以不再提取。

。公司法定公积金不足以弥补以前年度亏损的，在依照前款规定提取法定公积金之前，应当先用当年利润弥补亏损。

。公司弥补亏损和提取公积金后所余税后利润，股份有限公司按照股东所持有的股份比例分配，但是股份有限公司章程另有规定的除外。

### 上市公司特殊规定

|| 规定 | 说明 |
|---|---|---|
| 现金分红优先 | 上市公司应当优先采用现金分红方式 |
| 三年平均现金分红比例 | 最近三年以现金方式累计分配的利润不少于最近三年年均净利润的30% |
| 董事会说明义务 | 董事会应当综合考虑行业特点、发展阶段、自身经营模式、盈利水平等因素 |
| 差异化现金分红 | 不同发展阶段可差异化现金分红政策 |

## 第二部分：分配方案设计

### 分红形式比较

|| 形式 | 优点 | 缺点 | 适用场景 |
|---|---|---|---|---|
| 现金分红 | 直接回报股东，股东自主决定再投资 | 公司现金流压力 | 盈利稳定、现金流充裕的公司 |
| 股票分红（送股） | 不消耗现金，扩大股本 | 稀释股权 | 资本公积金充足、高成长公司 |
| 现金+股票混合 | 兼顾回报与留存 | 结构复杂 | 成熟期企业 |
| 实物分红 | 消化库存，股东实用 | 税务处理复杂 | 特定行业（酒类等） |

### 分红比例设计参考

```
行业分红比例参考（现金分红占净利润比例）：

├── 金融/银行：20-35%（高资本充足率要求）
├── 公用事业：40-70%（稳定现金流）
├── 制造业：20-50%（资本开支差异大）
├── 科技/互联网：0-30%（成长期多留存）
└── 国有企业：30-50%（国有资本收益上缴）

决策因素：
- 公司所处发展阶段（成长期/成熟期）
- 现金流状况与资本开支需求
- 股东回报预期
- 债务融资能力
- 行业惯例
```

### 滚存利润（未分配利润）处理

|| 情形 | 处理原则 |
|---|---|---|
| 当期不分红 | 须股东会决议，可规定具体用途（如扩大再生产） |
| 历年滚存利润 | 可一次性分配，也可分年度分配 |
| 滚存利润用途限定 | 无强制限定，但须符合公司发展规划 |
| 退市/解散时滚存利润 | 须向股东按比例分配 |

## 第三部分：决议程序合规

### 决策权限划分

|| 公司类型 | 分红方案提议 | 审议决定 |
|---|---|---|---|
| 有限责任公司 | 董事会 → 提交股东会 | 股东会决议（过半数表决权） |
| 股份有限公司 | 董事会 → 提交股东大会 | 股东大会决议（出席股份过半数） |
| 上市公司 | 董事会制定预案 | 股东大会审议 |

### 决议程序要点

```
分红决议标准流程：

1. 财务部门测算
   ├── 当期利润金额
   ├── 彆补亏损情况
   ├── 公积金提取计算
   └── 可分配利润金额

2. 董事会审议
   ├── 审议《利润分配方案》
   └── 形成会议决议

3. 股东会/股东大会决议
   ├── 审议《利润分配方案》
   ├── 说明分红比例及理由
   └── 表决通过（注意表决权比例要求）

4. 公告与实施（上市公司）
   ├── 公告分红方案
   ├── 确定股权登记日
   └── 派发红利
```

### 决议无效/可撤销情形

|| 情形 | 后果 | 救济途径 |
|---|---|---|---|
| 未经决议自行分红 | 决议无效，须返还分红 | 股东起诉确认无效 |
| 决议程序瑕疵 | 可撤销（如未提前通知、表决权不足） | 股东自决议作出起60日内申请撤销 |
| 违反章程约定 | 可能被撤销或认定无效 | 股东申请法院介入 |
| 关联交易损害公司利益 | 赔偿公司损失 | 公司或股东代表诉讼 |

## 第四部分：中小股东权益保护

### 强制分红救济

|| 情形 | 法律依据 | 救济途径 |
|---|---|---|---|
| 连续盈利但长期不分红 | 公司法第210条 | 股东可起诉要求分红（法院审查） |
| 大股东滥用控制权不分红 | 公司法第21条（关联交易） | 股东代表诉讼 |
| 章程规定分红条件但公司拒绝 | 章程约定 | 股东按章程约定主张 |
| 解散时仍有未分配利润 | 公司法第232条 | 清算组须向股东分配 |

### 股东分红权诉讼要件

```
股东起诉要求分红的审查要件：

1. 公司存在可分配利润
2. 股东会已作出分红决议
3. 公司无正当理由拒绝执行
4. 分红已届满清偿期限

注意：
- 除非章程另有规定，股东会未作分红决议前，法院一般不支持强制分红
- 特殊情形：大股东通过控制权长期拒绝分红，可能构成权利滥用
```

### 知情权保障

|| 权利 | 说明 |
|---|---|
| 查阅财务会计报告 | 股东有权查阅公司会计报告（第57条） |
| 查阅会计账簿 | 股东可书面申请查阅（第57条） |
| 审计权 | 特定情形下可申请法院指定审计 |

## 第五部分：特殊情形处理

### 亏损情形下的分红

|| 情形 | 是否可分红 | 说明 |
|---|---|---|---|
| 当期亏损 | ❌ 不可 | 须先彆补亏损 |
| 历年亏损未彆补 | ❌ 不可 | 须先彆补历年亏损 |
| 历年亏损已彆补完毕 | ✅ 可 | 提取公积金后可分配 |
| 资本公积金彆补亏损 | ⚠️ 特殊情形 | 须符合严格条件 |

### 股份回购与分红配合

|| 情形 | 处理原则 |
|---|---|---|
| 回购期间分红 | 回购期间不得分红（第142条） |
| 减资配合分红 | 须按减资程序处理 |
| 利润回购股份 | 视为利润分配，缴纳个人所得税 |

### 上市公司特殊限制

|| 限制情形 | 说明 |
|---|---|
| 半年度报告不分红 | 半年报不分红，年度报告为强制分红 |
| 重大资产重组停牌期间 | 分红程序暂停 |
| 股权登记日与除权除息 | 注意除权除息影响 |

| 公积金提取 | 公司法第210条 |

### 分红方案合规审查清单

- [ ] 公司当期是否盈利，是否存在未彆补亏损
- [ ] 是否已按规定提取法定公积金（注册资本50%前）
- [ ] 利润分配方案是否经董事会审议通过
- [ ] 利润分配方案是否经股东会/股东大会决议
- [ ] 股东会召集程序是否符合公司法及章程规定
- [ ] 股份有限公司是否按股份比例分配（章程另有约定除外）
- [ ] 上市公司是否满足现金分红比例要求（三年累计30%）
- [ ] 分红决议内容是否符合章程约定
- [ ] 涉及国有资本须符合国资监管规定
- [ ] 个人股东取得分红是否涉及个人所得税代扣代缴
