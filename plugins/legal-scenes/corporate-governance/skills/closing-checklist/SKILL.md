# /closing-checklist — China Mainland

## Purpose

M&A、股权交易、资产收购等交易的交割前法律合规检查清单。

---

## 交割前通用检查

### 第一类：交易文件签署

| 检查项 | 状态 | 备注 |
|---|---|---|
| □ 股权转让协议/资产收购协议已签署 | ✅/❌ | |
| □ 董事会/股东会决议已签署 | ✅/❌ | |
| □ 目标公司其他股东放弃优先购买权（如适用）| ✅/❌ | 《公司法》第84条 |
| □ 涉及国有资产已履行评估/备案程序 | ✅/❌ | 《企业国有资产法》|
| □ 涉及外商投资企业已履行审批程序 | ✅/❌ | 《外商投资法》|
| □ 涉及上市公司已履行信息披露/要约豁免 | ✅/❌ | 《证券法》|

---

### 第二类：交割条件满足

| 检查项 | 状态 | 备注 |
|---|---|---|
| □ 反垄断审查已完成（达到申报标准）| ✅/❌ | 经营者集中申报 |
| □ 经营者集中申报已完成 | ✅/❌ | 营业额标准：参购方合计>20亿+其中两家>4亿 |
| □ 境外投资审批（如有）| ✅/❌ | 发改委/商务部 |
| □ 外债登记（如有）| ✅/❌ | 外汇局 |

---

### 第三类：工商变更登记

| 检查项 | 状态 | 备注 |
|---|---|---|
| □ 新股东/股权比例已取得变更后的营业执照 | ✅/❌ | |
| □ 法定代表人已变更（如适用）| ✅/❌ | |
| □ 注册资本变更登记（如适用）| ✅/❌ | |
| □ 经营范围变更登记（如适用）| ✅/❌ | |

---

### 第四类：证照和印章

| 检查项 | 状态 | 备注 |
|---|---|---|
| □ 营业执照正副本已更换 | ✅/❌ | |
| □ 公章、财务章、合同章已更换/移交 | ✅/❌ | |
| □ 银行预留印鉴已更新 | ✅/❌ | |

---

### 第五类：债权债务处理

| 检查项 | 状态 | 备注 |
|---|---|---|
| □ 债权人通知（如有须通知的债务）| ✅/❌ | |

| □ 担保责任转移已处理 | ✅/❌ | |

## 输出格式

```markdown
## 交割前检查清单——[交易名称]

**检查日期：** YYYY-MM-DD
**交易类型：** [股权收购/资产收购/合并/其他]
**目标公司：** [名称]
**预计交割日：** YYYY-MM-DD

### 总体状态

**完成度：** XX/X 项（XX%）
**关键未完成项：** [列表]

### 未完成项及后续行动

| 检查项 | 负责方 | 截止日期 | 状态 |
|---|---|---|---|
| [事项] | [人] | [日期] | 待处理 |

### 可关闭确认

□ 所有条件已满足，可以交割
□ 仍有关键未完成项，暂不能交割
□ 等待[具体条件]完成后可交割
```

## 本技能不做什么

- 不代替律师尽职调查。只提供标准检查清单。
- 不代办登记备案。只提供清单。
- 不保证交割无法律风险。
