### § A8.1 商事仲裁特别注意 4 大块（纽约公约 + 临时措施 + 仲裁协议 + 投资仲裁）

> **核心原则**：商事仲裁的核心是**仲裁协议效力 + 临时措施 + 纽约公约执行**。没有有效的仲裁协议 = 没有仲裁管辖。

#### 块 1：纽约公约执行机制（涉外裁决核心）

**核心法规：**
- 《承认及执行外国仲裁裁决公约》（纽约公约，1958 + 中国 1987 加入）
- 《民事诉讼法》第 304 条（涉外仲裁裁决执行）
- 《最高人民法院关于适用〈中华人民共和国民事诉讼法〉的解释》第 540-544 条

**纽约公约核心要点：**

| 要点 | 内容 |
|------|------|
| **适用范围** | 承认 + 执行外国仲裁裁决（缔约国境内作出） |
| **互惠保留** | 中国仅对缔约国领土内作出的裁决适用 |
| **商事保留** | 中国仅对契约 / 非契约商事关系适用 |
| **申请执行期限** | 2 年（自裁决履行期届满起） |

**纽约公约 5 类拒绝执行情形（核心抗辩）：**

| 抗辩 | 含义 |
|------|------|
| 1. **行为能力缺陷** | 当事人无行为能力（依本国法） |
| 2. **协议无效** | 仲裁协议依选定法律无效 / 未选定则依裁决地法无效 |
| 3. **超出范围** | 仲裁员裁决事项超出提交仲裁范围 |
| 4. **程序违反** | 组庭 / 程序违反当事人约定 / 仲裁地法 |
| 5. **未适当通知** | 未能适当通知（指定仲裁员 / 审理） |
| 6. **公共政策** | 执行违反执行地公共政策 |

**承认与执行程序（中法院）：**

| 步骤 | 时限 | 法院 |
|------|------|------|
| **申请** | 2 年内 | 中级人民法院 |
| **受理** | 7 日内 | 中院 |
| **裁定** | 6 个月内（可延长） | 中院 |
| **执行** | 依民事诉讼法 | 中院 |

**blocks（绝对禁止 — 纽约公约）**：
- 已签署有效仲裁协议 → **必须仲裁，不能诉讼**
- 外国裁决已生效 → **承认与执行（除非 5 类抗辩）**
- 公共政策抗辩滥用 → **不予执行 + 外交保护**

**work but ships（可补救）**：
- 申请材料不齐 → 30 日内补正
- 期限临近 → 紧急申请 + 恢复

#### 块 2：临时措施 + 财产保全（紧急救济核心）

**核心法规：**
- 《仲裁法》第 28 条（财产保全）
- 《仲裁法》第 46 条（证据保全）
- 各机构规则（CIETAC / HKIAC / SIAC / ICC 紧急仲裁员规则）

**临时措施 4 类：**

| 措施 | 内容 | 适用 |
|------|------|------|
| **财产保全** | 查封 / 冻结 / 扣押对方财产 | 防止裁决难以执行 |
| **证据保全** | 固定关键证据（防止毁损） | 证据即将灭失 |
| **行为保全** | 禁止被告实施特定行为 | 知识产权 / 反不正当竞争 |
| **临时禁令** | 维持现状 / 恢复原状 | 紧急情况 |

**仲裁前 / 仲裁中临时措施申请：**

| 阶段 | 申请机关 |
|------|---------|
| **仲裁前** | 法院（《仲裁法》第 28 条） |
| **仲裁中** | 仲裁委员会转交法院 |
| **紧急仲裁员（emergency arbitrator）** | 机构规则（CIETAC 2024 / HKIAC / SIAC） |

**紧急仲裁员核心要点：**

| 要点 | 内容 |
|------|------|
| **触发条件** | 紧急 + 不能等待仲裁庭组成 |
| **时限** | 申请后 24-48 小时决定 |
| **管辖** | 临时措施（不处理实体争议） |
| **效力** | 当事人有约束力（多数机构规则） |
| **法院执行** | 可申请法院承认与执行（部分法域） |

**blocks（绝对禁止 — 临时措施）**：
- 对方财产转移后申请保全 → **已无效果（需时提前）**
- 证据毁损后申请保全 → **已无效果**
- 紧急仲裁员作出后不遵守 → **违反仲裁协议 + 法院制裁**

**work but ships（可补救）**：
- 申请材料不齐 → 3 日内补正
- 担保不足 → 法院通知 7 日内补足

#### 块 3：仲裁协议效力 5 类争议

**核心法规：**
- 《仲裁法》第 16-19 条
- 《最高人民法院关于适用〈仲裁法〉若干问题的解释》

**仲裁协议效力 5 类争议：**

| 争议类型 | 法律依据 | 效力 |
|---------|---------|------|
| **1. 主体不适格** | 仲裁协议由无权代理人签订 | **无效** |
| **2. 超出仲裁范围** | 约定事项超出法律允许仲裁范围 | **超出部分无效** |
| **3. 选定的机构不存在** | 约定的仲裁委员会不存在 | **协议无效** |
| **4. 内容不明确** | 仲裁机构 / 仲裁员 / 仲裁规则约定不明 | **可补救 / 无效** |
| **5. 主合同无效** | 仲裁协议独立存在（《仲裁法》第 19 条） | **不影响仲裁协议** |

**blocks（绝对禁止 — 仲裁协议）**：
- 协议无效仍推进仲裁 → **裁决无效**
- 主合同无效 → 主张仲裁协议无效（错误理解独立原则）
- 选定多个仲裁机构（不明确）→ 协议可能无效

**work but ships（可补救）**：
- 协议不明确 → 30 日内补充协议
- 仲裁机构不存在 → 协商选定替代机构

#### 块 4：投资仲裁（ICSID + BIT / FTA）

**核心法规：**
- 《关于解决国家和他国国民之间投资争端公约》（ICSID 公约，1966）
- 中国与 100+ 国家签订双边投资条约（BIT）
- 中国与多国签订 FTA 投资章节（如中欧 CAI、中美 BIT 谈判）

**ICSID 仲裁核心要点：**

| 要点 | 内容 |
|------|------|
| **管辖** | 投资者 vs 东道国（国家同意） |
| **同意方式** | BIT / 投资合同 / 投资法 |
| **仲裁地** | 通常在 ICSID 总部（华盛顿）或机构所在地 |
| **法律适用** | BIT / 东道国法律 / 国际法 |
| **救济** | 损害赔偿 / 恢复原状 / 停止侵害 |

**投资仲裁 4 类救济：**

| 救济 | 适用 |
|------|------|
| **征收补偿（expropriation）** | 国有化 / 间接征收（税收 / 监管） |
| **公平公正待遇（FET）** | 司法拒绝 / 透明度 / 合理期待 |
| **最惠国待遇（MFN）** | 第三方国家更优惠待遇 |
| **国民待遇（NT）** | 类似国内投资者待遇 |

**BIT 触发条件典型：**

| 触发 | 含义 |
|------|------|
| **征收** | 直接征收 / 间接征收（监管导致价值显著下降） |
| **违反 FET** | 司法拒绝 / 程序不公 / 透明度不足 |
| **违反保护伞条款** | 违反投资合同具体承诺 |
| **歧视待遇** | 违反 MFN / NT |

**blocks（绝对禁止 — 投资仲裁）**：
- 直接征收 → **投资者启动 BIT 仲裁 + 巨额赔偿**
- 间接征收（监管措施导致 80% 价值下降）→ **同上**
- 违反 FET（拒绝司法）→ **同上**

**work but ships（可补救）**：
- 监管措施涉嫌间接征收 → 评估 + 协商 + 提前告知
- BIT 条款违反 → 申请友好协商

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **违反有效仲裁协议仍诉讼** | 《仲裁法》第 26 条 | 法院驳回起诉 |
| 2. **伪造 / 毁灭证据** | 《刑法》第 307 条 | 3-7 年有期徒刑 |
| 3. **临时措施后转移财产** | 《仲裁法》第 28 条 + 妨害司法 | 罚款 + 拘留 + 刑责 |
| 4. **裁决执行违反纽约公约** | 《纽约公约》第 5 条 + 公共政策 | 拒绝承认与执行 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时限
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — 商事仲裁增强版）**：
- 仲裁机构 + 仲裁地 + 适用法律 + 语言？（仲裁协议核心）
- 涉案金额 + 争议类型 + 行业？（程序设计）
- 临时措施需求（紧急 / 一般）？（救济设计）
- 是否涉外（纽约公约）？（执行设计）
- 是否投资仲裁（ICSID / BIT）？（特殊规则）
- 是否已有仲裁协议 + 效力？（管辖基础）

## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** `arbitration-clause-reviewer`(仲裁条款是商事仲裁起点)

**关键节点(商事仲裁 5 阶段):**

```
签约 / 仲裁协议:
  Step 1: arbitration-clause-drafter / arbitration-clause-reviewer
          → 仲裁条款设计 + 有效性

立案 / 答辩:
  Step 2: arbitral-procedure-advisor + 各机构 procedure-advisor
          → 仲裁请求 + 答辩

审理:
  Step 3: evidence-rule-guide + interim-measures-application
          → 证据 + 临时措施

裁决:
  Step 4: arbitral-procedure-advisor + jurisdiction-dispute-advisor
          → 管辖 + 实体审理

执行 / 撤销:
  Step 5: award-enforcement-checker
          → 纽约公约执行 + 撤销之诉
```
