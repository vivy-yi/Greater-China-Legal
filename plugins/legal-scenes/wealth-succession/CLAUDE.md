### § A8.1 财富传承特别注意 4 大块（遗嘱形式 + 信托设立 + 遗产税 + 跨境）

> **核心原则**：财富传承的核心是**法律形式完备 + 税务合规 + 跨境协调**。"口头承诺 + 私凭文书"无法对抗法定继承。

#### 块 1：遗嘱形式要件（《民法典》继承编 2021）

**核心法规：**
- 《民法典》第 1133-1144 条（遗嘱形式）
- 《民法典》第 1121-1132 条（继承开始）
- 《遗嘱公证细则》

**6 种遗嘱形式（按法定优先顺序）：**

| 形式 | 见证人数 | 其他要求 | 法条 |
|------|---------|---------|------|
| **自书遗嘱** | 无 | 全文亲笔 + 签名 + 年月日 | § 1134 |
| **代书遗嘱** | 2 人以上见证 | 1 人代书 + 注明年月日 + 见证人签名 | § 1135 |
| **打印遗嘱** | 2 人以上见证 | 打印 + 见证人 + 注明年月日 | § 1136 |
| **录音录像遗嘱** | 2 人以上见证 | 录像中口述姓名 + 年月日 | § 1137 |
| **口头遗嘱** | 2 人以上见证 | 危急情况 + 事后书面补 | § 1138 |
| **公证遗嘱** | 公证员办理 | 公证机构出具 | § 1139 |

**遗嘱形式 6 类无效情形：**

| 无效情形 | 法条 |
|---------|------|
| **无见证人** | § 1135-1138 |
| **见证人无行为能力** | § 1140 |
| **见证人为继承人 / 受遗赠人** | § 1140 |
| **见证人为利害关系人** | § 1140 |
| **缺少签名 / 年月日** | § 1134-1137 |
| **伪造 / 篡改** | § 1144 |

**blocks（绝对禁止 — 遗嘱）**：
- 伪造 / 篡改遗嘱 → **《刑法》第 280 条伪造证件罪 + 民事无效**
- 见证人为继承人 / 受遗赠人 → **遗嘱无效**
- 多份遗嘱冲突时未立新遗嘱 → **法定继承**

**work but ships（可补救）**：
- 遗嘱形式瑕疵 → 重新订立 + 公证
- 遗嘱内容变化 → 立新遗嘱 + 公证 + 撤回旧遗嘱

#### 块 2：家族信托设立流程（信托法 + 银保监）

**核心法规：**
- 《信托法》（2001 年生效）
- 《信托公司集合资金信托计划管理办法》
- 《关于规范信托公司信托业务分类的通知》（2023 银保监）
- 《关于规范金融机构资产管理业务的指导意见》（2018 资管新规）

**家族信托核心要点：**

| 要点 | 内容 |
|------|------|
| **本质** | 委托人将财产委托给受托人，受益人受益 |
| **设立规模** | 1000 万 - 数亿（无最低限制） |
| **期限** | 通常 10 年以上 / 永续 |
| **受益人** | 委托人 + 配偶 + 子女 + 父母 |
| **受托人** | 信托公司（持牌）/ 私人信托公司 |

**家族信托设立 7 步流程：**

| 步骤 | 内容 | 时限 |
|------|------|------|
| 1. **需求分析** | 财富规模 + 传承目标 + 风险隔离 | 1-2 周 |
| 2. **结构设计** | 信托架构 + 受益人 + 保护人 | 1-2 周 |
| 3. **尽调与协议** | 起草信托合同 + 尽职调查 | 1-2 个月 |
| 4. **财产转移** | 资金 / 股权 / 不动产 + 登记 | 1-2 个月 |
| 5. **信托登记** | 银保监 / 中信登 + 财产登记 | 1 个月 |
| 6. **运营管理** | 财产管理 + 收益分配 | 持续 |
| 7. **分配 / 终止** | 按合同分配 + 终止清算 | 按合同 |

**信托财产隔离效力：**

| 情形 | 效力 |
|------|------|
| **委托人破产** | 信托财产不纳入破产财产 |
| **受托人破产** | 信托财产不纳入受托人破产财产 |
| **债权人追偿** | 仅在信托设立前债权可追索 |
| **离婚** | 配偶非受益人则无权分割 |

**blocks（绝对禁止 — 信托）**：
- 信托财产被强制执行 → **违反《信托法》第 17 条**
- 信托财产转移 / 隐藏 → **违反信托义务 + 民事赔偿**
- 受托人混同信托财产 / 自有财产 → **《信托法》第 28 条 + 恢复原状**

**work but ships（可补救）**：
- 信托合同不完善 → 修订 + 补充
- 受托人变更 → 协议 + 银保监备案

#### 块 3：遗产税试点 + 赠与税衔接（未来趋势）

**核心法规：**
- 《关于深化个人所得税改革的决定》（2018 — 暂无遗产税）
- 国务院多次表态"研究论证"（尚未立法）
- 深圳市 / 海南自由贸易港可能率先试点

**遗产税 5 大设计要点（国际比较 + 中国潜在）：**

| 要点 | 内容 | 国际参考 |
|------|------|---------|
| **起征点** | 高（2000 万 - 5000 万人民币） | 美国 545 万美元 / 日本 3000 万日元 |
| **税率** | 累进 10%-50% | 美国 18%-40% / 日本 10%-55% |
| **免税额** | 配偶 / 子女 / 父母基础免征额 | 美国 545 万美元 / 德国 50 万欧元 |
| **扣除项** | 丧葬费 / 债务 / 公益捐赠 | 国际通行 |
| **生前赠与** | 多数国家并入遗产税基数 | 美国 + 中国香港 |

**中国可能试点方案（学界讨论）：**

| 方案 | 内容 |
|------|------|
| **时点** | 2027 - 2030 试点 / 全面开征 |
| **试点城市** | 深圳 / 海口 / 上海（先行） |
| **起征点** | 2000 万 - 5000 万 |
| **税率** | 10% - 50% 累进 |
| **与赠与税衔接** | 生前 5 年内赠与并入基数 |

**blocks（绝对禁止 — 遗产税）**：
- 隐匿遗产 → **逃税罪（《刑法》第 201 条）+ 补缴 + 罚金**
- 虚假赠与逃避遗产税 → **同上**

**work but ships（可补救）**：
- 遗产规划不足 → 提前 10-20 年规划
- 跨境资产 → 跨境架构 + 多法域协调

#### 块 4：跨境资产继承 + 多法域协调

**核心法规：**
- 《涉外民事关系法律适用法》（2010 + 2023 修订）
- 《民法典》第 1128-1131 条（法定继承）
- 1961 年《遗嘱处分方式法律适用公约》（海牙）

**跨境继承 5 类常见情形：**

| 情形 | 处理 |
|------|------|
| **中国公民境外资产** | 依财产所在地法 + 海牙公约 |
| **外籍华人中国资产** | 依中国法（继承编） |
| **跨境不动产** | 不动产所在地法（专属管辖） |
| **跨境金融资产** | 依金融资产所在地 + 协议约定 |
| **跨境股权** | 依股权所在公司法 + 章程 |

**多法域遗产继承协调：**

| 步骤 | 内容 |
|------|------|
| 1. **继承启动** | 死亡证明 + 亲属关系证明 + 遗嘱 |
| 2. **法律适用** | 依国际私法 + 死者住所 / 国籍 |
| 3. **公证认证** | 中国驻外使馆认证 + 翻译公证 |
| 4. **继承过户** | 不动产 / 股权 / 银行账户 |
| 5. **税务申报** | 中国 + 境外双重税务 |

**blocks（绝对禁止 — 跨境继承）**：
- 境外遗嘱未认证 → **中国法院 / 公证处不承认**
- 跨境资产未申报 → **税务 + 外汇违规**

**work but ships（可补救）**：
- 跨境资产继承手续复杂 → 委托境外律师 + 国内律师协同
- 多法域遗产税 → 申请税务抵免

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **伪造 / 篡改遗嘱** | 《刑法》第 280 条伪造证件罪 + 民事无效 | 3-10 年 + 民事赔偿 |
| 2. **隐匿 / 转移遗产逃避继承 / 税务** | 《刑法》第 201 条逃税罪 | 3-7 年 + 补缴 + 罚金 |
| 3. **受托人混同信托财产** | 《信托法》第 28 条 | 恢复原状 + 民事赔偿 |
| 4. **跨境资产未申报** | 中国 + 境外双重处罚 | 双重税务 + 外汇违规 |

**主动问（6 类不确定 — 财富传承版）**：
- 资产规模 + 资产类型？（不动产 / 股权 / 金融）
- 家庭结构（核心 / 再婚 / 多代）？（继承人范围）
- 是否跨境资产 + 外籍继承人？（多法域协调）
- 传承工具（遗嘱 / 信托 / 保险 / 控股）？（综合规划）
- 是否遗产税试点 + 赠与税衔接？（税务规划）
- 配偶 + 子女 + 父母关系？（继承顺序）
