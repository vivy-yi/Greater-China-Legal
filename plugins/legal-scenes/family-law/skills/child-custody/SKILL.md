# /child-custody — 子女抚养

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § A7-13 未成年人利益最大化原则 + § A8 涉未成年人主动建议 + § B2 路由表。

## 工作流程

```
Step 1: 子女信息核实(年龄 / 健康 / 现状)
Step 2: 父母抚养能力评估
Step 3: 抚养权归属建议 / 主张
Step 4: 抚养费计算
Step 5: 探视权安排
Step 6: 协议 / 判决执行跟进
```

## 抚养权归属(《民法典》1084 条)

### 离婚时抚养权归属

| 年龄 | 原则 |
|------|------|
| **< 2 周岁** | 原则上**母亲**(除非母亲有严重不适合抚养情形) |
| **2-8 周岁** | 综合考虑(双方条件) |
| **> 8 周岁** | **尊重子女意愿**(法院会参考) |

### 综合考虑因素

| 因素 | 倾向 |
|------|------|
| 抚养能力(经济 / 时间) | 优者 |
| 抚养意愿 | 强意愿者 |
| 子女现状(主要照料方) | 维持现状优先 |
| 父母品行 | 良好者 |
| 祖父母 / 外祖父母协助能力 | 考虑 |
| 子女意愿(> 8 岁) | 重要参考 |

### 不适合抚养的情形

| 情形 | 注意 |
|------|------|
| 严重疾病(可能传染) | 不宜 |
| 有赌博 / 吸毒 / 酗酒恶习 | 不宜 |
| 有家庭暴力倾向 | **绝对不宜** |
| 服刑中 | 不宜 |
| 其他严重不利于子女身心健康 | 不宜 |

## 抚养费(《民法典》1085 条)

### 计算公式(参考)

```
抚养费 = 月总收入的 20-30%
       (负担两个以上子女 ≤ 50%)
```

**实务标准:**

| 月总收入 | 抚养费比例 |
|----------|----------|
| ≤¥5000 | 一般 20-30% |
| ¥5000-10000 | 20-30% |
| >¥10000 | 可适当降低比例,但有下限 |

**考虑因素:**

| 因素 | 注意 |
|------|------|
| 子女实际需要 | 学费 / 医疗 / 兴趣班 |
| 父母负担能力 | 经济条件 |
| 当地生活水平 | 一线城市 vs 县城 |

### 抚养费 5 类

| 类型 | 标准 |
|------|------|
| 教育费 | 必要 + 合理(高额辅导班须协商) |
| 医疗费 | 实报实销(大病) |
| 生活费 | 月付 / 季付 |
| 特殊费用 | 重大疾病 / 意外 |
| 未来费用 | 大学(部分法院支持) |

### 抚养费调整情形

| 情形 | 调整 |
|------|------|
| 物价上涨 | 可申请增加 |
| 父母收入变化 | 可申请调整 |
| 子女需要变化 | 可申请调整 |
| 给付方丧失劳动能力 | 可申请减免 |

## 探视权(《民法典》1086 条)

### 探视权 4 类设计

| 类型 | 适用 |
|------|------|
| **定期探视** | 周末 / 节假日 |
| **假期探视** | 寒暑假 / 国庆 |
| **远程探视** | 电话 / 视频(异地) |
| **监督探视** | 家暴情形(第三方在场) |

### 探视权细节

```
1. 频率:每周 / 每月 / 每季
2. 时长:几小时 / 过夜 / 几天
3. 地点:一方家 / 公共场所 / 中立场所
4. 交通:谁接送 / 费用谁出
5. 节假日:春节 / 中秋等安排
```

### 探视权中止(《民法典》第 1086 条第 3 款)

| 情形 | 中止 |
|------|------|
| 直接抚养方患严重疾病 | 中止(治好后恢复) |
| 直接抚养方虐待子女 | **永久中止**(变更抚养) |
| 直接抚养方阻碍探视 | 另一方可申请强制执行 |
| 探视方有不利子女行为 | 法院可中止 |

## 变更抚养关系(《民法典》1084 条 + 司法解释)

**可变更情形:**

| 情形 | 注意 |
|------|------|
| 直接抚养方患重病 / 伤残 | 无法继续抚养 |
| 直接抚养方不尽抚养义务 | 虐待 / 遗弃 / 有恶习 |
| 8 岁以上子女愿随另一方生活 | 重要参考 |
| 有其他正当理由 | 法院综合认定 |

**程序:** 单独起诉变更抚养关系,不受 6 个月限制。

## 涉未成年人特别程序

| 情形 | 程序 |
|------|------|
| **< 8 岁** | 必须**家事调查官**评估 |
| **8-14 岁** | 法院**单独询问**子女意愿 |
| **> 14 岁** | 法院充分听取子女意见 |
| **家暴 / 虐待** | 启动**未成年人保护程序** |

## 涉外抚养

**关键问题:**

```
1. 子女国籍 / 居所(影响管辖)
2. 适用法律(海牙公约 / 国内法)
3. 外国法院判决在中国的执行
4. 跨境探视(海牙儿童诱拐公约)
```
