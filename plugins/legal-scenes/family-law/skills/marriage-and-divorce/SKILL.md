# /marriage-and-divorce — 婚姻与离婚

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B7 决策树 + § B3 调解优先原则 + § A6 决策姿态。

## 工作流程

```
Step 1: 评估婚姻状态(有效 / 无效 / 可撤销)
Step 2: 评估诉求(协议离婚 vs 诉讼离婚)
Step 3: 协议离婚路径(双方同意)
Step 4: 调解离婚路径(一方犹豫)
Step 5: 诉讼离婚路径(一方不同意)
Step 6: 财产 / 子女处置(联动其他 skill)
```

## 婚姻效力 3 类

| 类型 | 法条 | 救济 |
|------|------|------|
| **有效婚姻** | 《民法典》第 1049 条 | 需解除(协议 / 诉讼) |
| **无效婚姻** | 《民法典》第 1051 条 | 重婚 / 近亲 / 未到法定婚龄 | 自始无效 |
| **可撤销婚姻** | 《民法典》第 1052/1053 条 | 胁迫 / 重大疾病未告知 | 1 年内可撤销 |

## 协议离婚 5 步流程(《民法典》第 1076-1078 条)

```
Step 1: 双方共同申请 + 提交材料
       - 双方身份证 + 户口簿
       - 结婚证
       - 离婚协议书(一式三份)

Step 2: 婚姻登记机关受理 → 30 日冷静期

Step 3: 冷静期满后 30 日内,双方共同到场确认

Step 4: 婚姻登记机关查明双方确实自愿 + 已妥善处理子女 / 财产

Step 5: 发放离婚证
```

**协议离婚必备条款:**

| 条款 | 注意 |
|------|------|
| 双方基本信息 | 完整 + 准确 |
| 婚姻登记日期 | 准确 |
| 离婚原因 | 简述 |
| **子女安排** | 抚养权 + 抚养费 + 探视 |
| **财产分割** | 全部共同财产 + 个人财产归属 |
| **债务分担** | 共同债务 + 个人债务 |
| 其他约定 | 经济帮助 / 损害赔偿 |
| 协议生效 | 双方签字 + 日期 |

## 诉讼离婚

**起诉条件:**

```
✅ 重婚 / 与他人同居
✅ 实施家暴 / 虐待 / 遗弃
✅ 赌博 / 吸毒等恶习屡教不改
✅ 因感情不和分居满 2 年
✅ 其他导致夫妻感情破裂的情形
```

**第一次起诉不判离的情形:**

| 情形 | 处理 |
|------|------|
| 一方坚决不同意离婚 | 第一次通常不判离 |
| 没有法定破裂情形 | 驳回起诉 |
| 调解和好 | 驳回 |

**第二次起诉:** 第一次判决生效 6 个月后可再次起诉,通常判离。

## 感情破裂认定 5 类(《民法典》第 1079 条)

| 情形 | 证据 |
|------|------|
| 重婚 / 与他人同居 | 照片 / 证人 / 视频 |
| 家暴 / 虐待 / 遗弃 | 报警记录 / 医院诊断 / 妇联记录 |
| 赌博 / 吸毒 | 公安处罚 / 强制戒毒 |
| 分居满 2 年 | 租赁合同 / 证人 / 村委会证明 |
| 其他情形 | 综合认定 |

## 涉外婚姻特殊处理

**管辖:**

| 情形 | 管辖 |
|------|------|
| 一方在国外 | 国内法院管辖(被告住所地) |
| 双方在国外 | 不在国内管辖(在所在国办理) |
| 华侨 | 中级人民法院 |
| 涉港澳台 | 视具体情形 |

**特别注意:**
- 外国法律可能影响财产分割
- 外国法院判决在中国的承认(《民诉法》第 288 条)

## 调解优先原则(§ B3)

| 调解空间 | 动作 |
|----------|------|
| 🟢 强 | 协议离婚 + 律师调解 |
| 🟡 中 | 法院调解 + 律师调解 |
| 🟠 弱 | 直接诉讼 |
| 🔴 无 | 公告送达 + 缺席判决 |
