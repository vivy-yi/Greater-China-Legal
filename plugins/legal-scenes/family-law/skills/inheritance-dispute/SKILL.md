# /inheritance-dispute — 遗产继承

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B1 Step 9 + § A12 涉外继承 + § B5 跨国继承升级。

## 工作流程

```
Step 1: 确认被继承人死亡(死亡证明 / 户籍注销)
Step 2: 梳理遗产范围 + 继承人范围
Step 3: 判断继承方式(法定 / 遗嘱 / 遗赠 / 遗赠扶养协议)
Step 4: 评估继承份额 + 特殊情形
Step 5: 协议分割 / 法院诉讼
Step 6: 协助执行(过户 / 支付)
```

## 继承开始时间(《民法典》1121 条)

**继承从被继承人死亡时开始:**

| 死亡方式 | 继承开始 |
|----------|----------|
| 自然死亡 | 死亡时 |
| 宣告死亡 | 判决作出之日 |
| 失踪宣告 | 同上 |

## 继承方式优先级(《民法典》1123 条)

```
1. 遗赠扶养协议(最高)
2. 遗嘱继承 / 遗赠
3. 法定继承(最低)
```

## 法定继承人顺序(《民法典》1127 条)

```
第一顺序:
  - 配偶
  - 子女(包括婚生 / 非婚生 / 收养 / 有抚养关系的继子女)
  - 父母(包括生父母 / 养父母 / 有抚养关系的继父母)

第二顺序:
  - 兄弟姐妹
  - 祖父母
  - 外祖父母

继承开始 → 第一顺序继承人继承 → 第二顺序不继承
第一顺序无人 → 第二顺序继承
```

## 继承份额(《民法典》1130 条)

**基本原则:** 同一顺序继承人**平均分配**

**特殊调整:**

| 情形 | 调整 |
|------|------|
| 生活有特殊困难 + 缺乏劳动能力 | 适当照顾(可多分) |
| 对被继承人尽主要抚养义务 | 可多分 |
| 有抚养能力 / 条件但不尽抚养义务 | 应当不分或少分 |
| 继承人协商同意 | 可不均等 |

## 遗嘱 6 类(《民法典》1133-1144 条)

| 类型 | 形式 | 见证人 |
|------|------|--------|
| **自书遗嘱** | 全文亲笔 + 签名 + 日期 | 无需 |
| **代书遗嘱** | 2 名以上见证人 | 必须 |
| **打印遗嘱** | 2 名以上见证人 | 必须 |
| **录音录像遗嘱** | 2 名以上见证人 | 必须 |
| **口头遗嘱** | 2 名以上见证人(危急情况) | 必须 |
| **公证遗嘱** | 公证机构办理 | 无需 |

**遗嘱必备:**

```
1. 遗嘱人基本信息
2. 遗产明细(具体到不动产门牌号)
3. 继承人指定(姓名 + 关系)
4. 各继承人份额
5. 遗嘱人亲笔签名 + 日期(自书)
```

**遗嘱无效情形:**

| 情形 | 无效 |
|------|------|
| 无民事行为能力人 | 无效 |
| 受胁迫 / 欺骗 | 无效 |
| 伪造 | 无效 |
| 被篡改 | 篡改部分无效 |
| 形式不符 | 无效(如代书无见证) |
| 处分他人财产 | 该部分无效 |

## 继承权丧失(《民法典》1125 条)

**4 类丧失继承权情形:**

| 情形 | 后果 |
|------|------|
| **故意杀害被继承人** | 丧失(无论既遂 / 未遂) |
| **为争夺遗产而杀害其他继承人** | 丧失 |
| **遗弃被继承人** | 丧失 |
| **虐待被继承人情节严重** | 丧失 |

**注意:** 丧失继承权的,可依遗嘱(如有)取得遗产。

## 遗产范围

| 类型 | 注意 |
|------|------|
| 房产 / 车辆 / 存款 | 全部 |
| 公司股权 | 股东资格继承 |
| 知识产权 | 财产权益部分 |
| 债权 | 遗产组成部分 |
| 债务 | 继承人清偿(以遗产为限) |

**不属于遗产的:**

| 类型 | 注意 |
|------|------|
| 人身损害赔偿金 | 不属于 |
| 抚恤金 | 不属于(但可参照分割) |
| 保险金(指定受益人) | 不属于 |

## 继承债务清偿

**原则:** 继承人**以遗产实际价值**为限清偿(《民法典》1161 条)

```
✅ 继承人可放弃继承(无需清偿债务)
✅ 实际价值范围内清偿
❌ 不需以自己财产清偿(除非自愿)
```

## 涉外继承(《民法典》1124 条 + 外国法律)

**关键问题:**

```
1. 遗产所在地(动产 / 不动产)
2. 适用法律(中国法 / 外国法)
3. 继承人范围(本国 / 外国)
4. 外国判决在中国的执行
```

**国际私法规则:**

| 情形 | 适用法律 |
|------|----------|
| 不动产继承 | 物之所在地法 |
| 动产继承 | 经常居所地法(可协议) |
| 遗嘱方式 | 符合立遗嘱时 / 死亡时任一地法律均有效 |

## 继承权纠纷诉讼

**时效:** 3 年(自知道或应当知道权利被侵犯之日起)

**诉讼请求:**

```
1. 确认继承权
2. 分割遗产(份额 / 实物)
3. 排除妨害(侵占遗产)
4. 赔偿损失
```

**管辖:** 被继承人死亡时住所地 / 主要遗产所在地法院
