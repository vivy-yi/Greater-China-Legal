# /domestic-violence — 家暴维权

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取 § B5 升级路径 + § A8 涉家暴主动建议 + § A7-14 不得协助伪造家暴证据。

## 工作流程

```
Step 1: 评估家暴严重程度 + 紧急性
Step 2: 紧急情况 → 报警 + 申请人身安全保护令
Step 3: 证据固定(伤情 / 报警 / 证人 / 录音录像)
Step 4: 多救济路径协同:
        - 民事:离婚 + 损害赔偿 + 抚养权
        - 行政:报警 + 妇联投诉
        - 刑事:故意伤害 / 虐待 / 遗弃控告
Step 5: 长期安全规划
```

## 家暴 4 类(《反家庭暴力法》)

| 类型 | 表现 | 法律救济 |
|------|------|----------|
| **身体暴力** | 殴打 / 捆绑 / 限制人身自由 | 故意伤害罪 / 虐待罪 + 离婚损害赔偿 |
| **精神暴力** | 辱骂 / 恐吓 / 跟踪 / 骚扰 | 人身安全保护令 + 离婚损害赔偿 |
| **性暴力** | 强制猥亵 / 强奸 | 刑事控告 + 离婚损害赔偿 |
| **经济控制** | 剥夺生活来源 / 财产控制 | 离婚财产分割照顾 |

## 人身安全保护令(《反家庭暴力法》第 23-32 条)

### 申请条件

```
✅ 正在遭受家暴 OR 有遭受家暴的现实危险
✅ 申请人:本人 / 近亲属 / 公安机关 / 妇联 / 居委 / 村委
✅ 申请法院:申请人住所地 / 家庭暴力发生地基层法院
```

### 保护令 6 类措施

```
1. 禁止被申请人实施家庭暴力
2. 禁止被申请人骚扰 / 跟踪 / 接触申请人及其近亲属
3. 责令被申请人迁出申请人住所
4. 保护申请人人身安全的其他措施
5. (涉及子女)禁止被申请人接触未成年子女
6. 责令被申请人支付未成年子女抚养费 / 老人赡养费
```

### 申请 5 步流程

```
Step 1: 准备申请书 + 证据
Step 2: 向法院提交
Step 3: 法院 72 小时内作出裁定(紧急情况 24 小时)
Step 4: 送达被申请人
Step 5: 执行 + 违反可拘留
```

**申请书必备:**

```
1. 申请人 / 被申请人基本信息
2. 家暴事实经过(时间 / 地点 / 方式 / 频次)
3. 证据清单(报警 / 医院 / 证人 / 录音录像 / 妇联记录)
4. 申请具体措施(对应 6 类)
5. 申请人签字 + 日期
```

### 证据类型

| 证据 | 来源 |
|------|------|
| 报警记录 / 出警记录 | 110 + 派出所 |
| 医院诊断书 / 病历 / X 光 | 医院 |
| 伤情照片 / 视频 | 当事人 / 亲友 |
| 妇联投诉记录 | 妇联 |
| 居委 / 村委调解记录 | 居委 / 村委 |
| 证人证言 | 邻居 / 亲友 / 子女 |
| 录音录像 | 当事人(注意合法性) |
| 保证书 / 悔过书 | 被申请人 |
| 微信 / 短信记录 | 通讯记录 |

## 损害赔偿(《民法典》1091 条)

**可主张情形:**

```
✅ 重婚 / 与他人同居
✅ 实施家暴 / 虐待 / 遗弃
✅ 有其他重大过错
```

**赔偿范围:**

| 类型 | 计算 |
|------|------|
| 物质损害 | 实际损失 |
| 精神损害 | 司法解释(一般 ¥5000-50000) |
| 医疗费 / 误工费 / 护理费 | 实际支出 |

## 刑事控告(《刑法》)

| 罪名 | 法条 | 情形 |
|------|------|------|
| 故意伤害罪 | 第 234 条 | 轻伤 / 重伤 / 死亡 |
| 虐待罪 | 第 260 条 | 长期虐待(告诉才处理) |
| 遗弃罪 | 第 261 条 | 拒绝扶养 |
| 强奸罪 / 强制猥亵 | 第 236/237 条 | 性暴力 |
| 非法拘禁罪 | 第 238 条 | 限制人身自由 |

## 多救济路径协同

```
紧急人身危险:
  Step 1: 报警 110
  Step 2: 申请人身安全保护令(72h / 紧急 24h)
  Step 3: 必要时申请人身安全保护令 + 临时安置

非紧急:
  Step 1: 报警 + 医院诊断 + 证据固定
  Step 2: 申请人身安全保护令
  Step 3: 启动离婚程序 + 主张损害赔偿
  Step 4: 评估刑事控告

长期:
  Step 1: 心理咨询
  Step 2: 经济独立规划
  Step 3: 抚养权主张(涉子女)
  Step 4: 安全居所规划
```

## 涉外家暴

**特别考虑:**

```
1. 一方外籍:涉外国法律 / 领事保护
2. 海外华人:涉海牙公约
3. 跨境跟踪:涉引渡 / 国际刑警
4. 受害方外籍居留:涉签证 / 庇护
```

## 升级触发(§ B5)

```
🔴 家暴致伤(轻伤 +)→ 立即报警 + 刑事控告
🔴 跟踪 / 骚扰持续 → 申请人身安全保护令 + 报警
🔴 涉及未成年子女 → 启动未成年人保护程序 + 变更抚养
🔴 一方失联 / 失踪 → 报警 + 公告送达
🔴 涉外 → 涉外律师协助
```
