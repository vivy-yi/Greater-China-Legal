# /ship-collision — 船舶碰撞

## 工作流程
```
Step 1: 评估责任基础(过失)
Step 2: 评估过失比例
Step 3: 1 年时效核查
Step 4: 证据保全(航海日志 / 雷达数据 / 气象)
Step 5: 涉外因素
```

## 责任原则

| 原则 | 适用 |
|------|------|
| **过失责任** | 双方按过失比例分担 |
| **推定过失** | 违反航行规则 / 避碰规则 |
| **1 年时效** | 自碰撞日起算 |

## 过失比例 4 类

| 情形 | 比例 |
|------|------|
| 单方过失 | 100% 责任方 |
| 双方过失 | 按比例(50/50 / 30/70 等) |
| 无法查明 | 50/50 平均(实际) |
| 不可抗力 | 各自承担 |

## 证据清单 6 类

| 证据 | 来源 |
|------|------|
| 航海日志 | 船方(法院调取) |
| 雷达数据 | 船方 + 港口 VTS |
| 气象记录 | 气象局 |
| 船员证言 | 申请法院调取 |
| AIS 数据 | 船方 |
| 碰撞现场勘验 | 法院 |

## 损害赔偿范围

| 项目 | 计算 |
|------|------|
| 船舶损失 | 修复费 / 全损价值 |
| 货物损失 | 实际价值 |
| 救助费 | 实际支出 |
| 营运损失 | 船期损失 |
| 人员伤亡 | 医疗费 + 误工 + 抚恤 |
| 清除残骸费 | 实际支出 |
