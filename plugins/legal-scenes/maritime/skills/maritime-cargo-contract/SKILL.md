# /maritime-cargo-contract — 海运货物合同纠纷

## 工作流程
```
Step 1: 评估责任基础(承运人 / 实际承运人 / 货方 / 保险)
Step 2: 评估免责事项(《海商法》第 51 条)
Step 3: 评估 1 年时效
Step 4: 证据保全(提单 / 检验报告 / 航海日志)
Step 5: 评估涉外因素 + 准据法
```

## 承运人责任 3 阶段

| 阶段 | 责任 |
|------|------|
| **装货前** | 谨慎处理 + 使船舶适航 |
| **装货港到卸货港** | 妥善 + 谨慎管理货物 |
| **卸货后** | 通知 + 交付 |

## 免责事项 12 类(《海商法》第 51 条)

| 类别 | 注意 |
|------|------|
| 火灾(承运人过失除外) | ✅ 免责 |
| 天灾 / 海难 | ✅ 免责 |
| 战争 / 暴乱 | ✅ 免责 |
| 公共当局行为 | ✅ 免责 |
| 罢工 / 停工 | ✅ 免责 |
| 救助 / 企图救助海上人命 / 财产 | ✅ 免责 |
| 货方原因(包装不当 / 标志不清) | ✅ 免责 |
| 货物自然特性 / 内在缺陷 | ✅ 免责 |
| 隐蔽缺陷(承运人谨慎不能发现) | ✅ 免责 |
| 非承运人过失造成(其他) | ✅ 免责 |
| 不可抗力 | 视情况 |

**特别注意:** 火灾免责须承运人证明"无过失"。

## 1 年时效(《海商法》第 257 条)

```
✅ 时效 1 年(不中止 / 不中断)
✅ 货物交付之日 / 应交付之日起算
✅ 超过 1 年:丧失请求权
```

**时效起算:**

| 情形 | 起算 |
|------|------|
| 一般货损 | 货物交付之日 |
| 全部灭失 | 应当交付之日起 60 日 |
| 迟延交付 | 实际交付之日 |

## 证据保全 5 类

| 证据 | 来源 |
|------|------|
| **提单** | 承运人签发 |
| **检验报告** | 第三方检验机构 |
| **航海日志** | 船方(申请法院调取) |
| **气象记录** | 气象局 |
| **港口装卸记录** | 港口 |

**特别注意:** 航海日志必须**法院调取**(船方一般不主动提交)。

## 涉外因素

**准据法选择:**

| 提单约定 | 适用 |
|----------|------|
| 中国法 | ✅(国内案件) |
| 海牙规则 | 涉外 + 提单未明确 |
| 鹿特丹规则 | 涉外 + 提单明确 |
| 其他国家法 | 涉外 + 提单明确 |

**国际公约:**

- 海牙规则(1924)
- 海牙—维斯比规则(1968)
- 鹿特丹规则(2009,中国未加入但国际常用)
