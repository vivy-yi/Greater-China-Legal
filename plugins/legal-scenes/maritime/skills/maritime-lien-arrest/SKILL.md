# /maritime-lien-arrest — 海事请求保全

## 工作流程
```
Step 1: 评估海事请求权(海商法第 22 条)
Step 2: 评估扣押船舶选择
Step 3: 申请扣押 + 担保
Step 4: 拍卖 / 变卖 + 分配
Step 5: 涉外因素
```

## 海事请求权 5 类(海商法第 22 条)

| 顺序 | 类别 |
|------|------|
| 1 | 救助报酬 + 共同海损分摊 |
| 2 | 海事请求(船员工资 / 港口费 / 货物损失 / 油污) |
| 3 | 船舶抵押权(已登记) |
| 4 | 船舶留置权 |
| 5 | 救助人 / 打捞人 / 船坞 / 修船人报酬 |

**特别注意:** 海事请求权不因船舶所有权变更而消灭(跟随船舶)。

## 扣押船舶选择

**可扣押的船舶:**

| 情形 | 适用 |
|------|------|
| 当事船舶 | 引发请求的船舶 |
| 姐妹船 | 同一人所有的其他船舶 |
| 经营船舶 | 船舶所有人变更时 |

**不可扣押:**

```
❌ 国家所有船舶
❌ 用于军事 / 政府非商业用途的船舶
❌ 船员个人工资请求外(可扣押,但优先于其他)
```

## 申请扣押流程

```
Step 1: 准备申请书 + 海事请求权证明
Step 2: 选择管辖法院(船籍港 / 扣押地)
Step 3: 申请扣押(可申请前扣押)
Step 4: 提供反担保(避免错误扣押)
Step 5: 法院裁定 + 执行
```

**反担保:**

| 情形 | 反担保 |
|------|--------|
| 一般海事请求 | 申请保全金额的 30% |
| 错误扣押 | 申请保全金额的 100% |

## 拍卖 / 变卖 / 分配

**拍卖:**

| 阶段 | 时限 |
|------|------|
| 拍卖公告 | 30 日前 |
| 第一次拍卖 | 起拍价 |
| 流拍 | 降价 20% |
| 第二次拍卖 | 再降 20% |
| 变卖 | 协议 |

**分配顺序:**

```
1. 拍卖费用 + 税费
2. 优先权(海商法第 22 条)
3. 抵押权(已登记)
4. 普通债权按比例
```

## 涉外因素

| 情形 | 适用 |
|------|------|
| 船旗国 | 适用船旗国法 |
| 扣押地 | 适用中国法 |
| 国际公约 | 1952 年扣押海船公约(中国未加入) |
