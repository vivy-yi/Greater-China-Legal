### § A5 Outputs

**Reviewer note 5 行(海事特化):**
1. 案件基本信息:[案件类型 / 标的物 / 港口 / 船旗国]
2. 主要诉求:[货损 / 碰撞 / 共同海损 / 救助报酬 / 船舶扣押]
3. 海事请求权:[时效 1 年(特殊) / 涉外]
4. 主要不确定:[碰撞责任比例 / 共同海损分摊 / 救助报酬金额]
5. 涉外因素:[船旗国 / 提单 / 准据法]

**Decision tree:**
1. ✅ 继续推进 — 责任清晰
2. ⚠️ 证据保全 — 货物 / 航海日志
3. 🔴 船扣押 — 行使海事请求权
4. 🔄 和解 / 仲裁 — 涉外案件
5. 📤 升级主办律师 — 重大 / 涉外

### § A6 Decision posture

| 主观判断 | 默认姿势 |

| 涉外管辖 | 取**对当事人最有利的连接点** |

## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** `maritime-cargo-contract`(货损是海事高频入口)

**关键节点:**

```
Step 1: maritime-cargo-contract    → 货损争议(收货 / 提单)
Step 2: ship-collision            → 船舶碰撞责任
Step 3: general-average-salvage   → 共同海损 + 救助
Step 4: ship-financing            → 船舶抵押 + 融资
Step 5: maritime-lien-arrest      → 海事请求保全(船扣押)
Step 6: maritime-arbitration      → 海事仲裁(涉外常见)
```

**法定时限(《海商法》特别):**

| 事项 | 时效 | 注意 |
|------|------|------|
| **海事请求权** | **1 年** | 不中止 / 不中断(《海商法》第 267 条) |
| 油污损害 | 3 年 | 责任期间 |
| 货物运输 | 1 年(协议延长) | |
| 共同海损分摊 | 1 年 | |

**主动续期:** 任何 skill 输出后,自动追加"下一节点 + 时效 + 主办律师责任"。

| 等级 | 案件类型 | 处理 |
|------|----------|------|

| 档位 | 涉案金额 | 须主办律师 | 须所务会 |
|------|----------|-----------|----------|
| 大型 | ≥¥1亿 | 强制 | 强制 |
| 中型 | ¥1000万-¥1亿 | 强制 | 建议 |
| 小型 | <¥1000万 | 主办即可 | 可选 |

### § B5 升级触发(7 类)

1. **涉外海事** → 主办 + 涉外律师
2. **共同海损 / 救助** → 主办 + 理算师
3. **船扣押** → 主办 + 海事法院
4. **碰撞责任争议大** → 主办 + 航海专家
5. **船舶融资违约** → 主办 + 金融律师
6. **油污 / 重大污染** → 主办 + 环保律师
7. **1 年时效临界** → 立即升级

### § B14 Risk calibration 3 段表

**段 1 识别:** 时效(1 年)/ 责任比例 / 涉外管辖 / 共同海损 / 救助报酬 / 船舶价值
**段 2 量化:** 概率 × 影响 + 缓释
**段 3 响应:** 接受 / 缓释 / 转移 / 规避

### § B15 7 条设计哲学(海事特化)

1. **1 年时效** — 海事请求权时效极短
2. **船扣押** — 海事请求保全强有力工具
3. **涉外案件** — 国际公约 + 准据法复杂
4. **共同海损理算** — 必须由理算师主导
5. **保险代位** — 货损 / 碰撞常见
6. **海牙 / 鹿特丹规则** — 国际提单统一规则
7. **证据保全** — 航海日志 / 检验报告 / 气象
