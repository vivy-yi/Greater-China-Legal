### § A8.1 资本市场特别注意 4 大块（注册制 + 信披处罚 + 重大重组 + 内幕交易）

> **核心原则**：资本市场监管的核心是**信息披露 + 禁止内幕 + 程序合规**。任何"商业秘密"或"暂不披露"都不能成为违规借口。

#### 块 1：注册制 vs 核准制（2019 改革）

**核心改革（2019 年科创板 / 2023 年全面注册制）：**

| 板块 | 注册制实施 | 审核机关 |
|------|----------|---------|
| **科创板** | 2019.7.22 | 交易所审核 + 证监会注册 |
| **创业板** | 2020.8.24 | 交易所审核 + 证监会注册 |
| **北交所** | 2021.11.15 | 交易所审核 + 证监会注册 |
| **主板** | 2023.2.17 | **全面注册制** |
| **科创 50** | — | 涨跌幅 20%（一般 10%） |

**注册制 vs 核准制核心差异：**

| 维度 | 核准制（旧） | 注册制（新） |
|------|----------|----------|
| **审核主体** | 证监会发审委 | 交易所（证监会注册） |
| **审核重点** | 实质判断（盈利预测等） | **信息披露** |
| **中介责任** | 部分 | **压实中介机构责任**（保荐 + 律师 + 会计师） |
| **审核期限** | 不定 | 交易所 6 个月 + 证监会 15 个工作日 |
| **发行价格** | 23 倍市盈率隐性上限 | 市场化询价 |

**blocks（绝对禁止）**：
- 注册制下"以信息披露为中心"原则违反 → 退回 + 处罚
- 中介机构未勤勉尽责 → 监管措施 + 业务暂停

#### 块 2：信息披露违法处罚（《证券法》2020 修订 + 2024 实施）

**信息披露违法处罚（2020 修订后加重）：**

| 违法情形 | 法条 | 处罚 |
|---------|------|------|
| **未按规定信披** | § 193 条 | **100 万 - 1000 万**（个人 50 万 - 500 万） |
| **虚假记载 / 误导性陈述 / 重大遗漏** | § 193 条 | **100 万 - 1000 万**（同上） |
| **发行欺诈（IPO / 再融资）** | § 181 条 | **500 万 - 5000 万** |
| **保荐人 / 律师 / 会计师未勤勉尽责** | § 182 条 | **暂停 / 撤销业务许可 + 罚款** |
| **内幕信息知情人未登记** | § 191 条 | **10 万 - 100 万 + 市场禁入** |

**blocks（绝对禁止 — 信披）**：
- IPO 欺诈发行 → **500 万 - 5000 万 + 退市 + 5 年市场禁入 + 刑事移交（160 条欺诈发行罪）**
- 重大资产重组虚假信披 → **100 万 - 1000 万 + 强制复牌 + 民事赔偿**
- 关联交易未披露 → **100 万 - 1000 万 + 市场禁入**

**work but ships（可补救）**：
- 信披不完整 → 立即补充披露 + 致歉
- 信息有误 → 立即更正 + 监管报告
- 披露不及时 → 立即补披露 + 解释

#### 块 3：重大资产重组触发 + 程序

**核心法规：**
- 《上市公司重大资产重组管理办法》（证监会 2014 / 2023 修订）
- 《上市公司证券发行注册管理办法》

**重大资产重组触发标准（任一）：**

| 标准 | 数值 | 法源 |
|------|------|------|
| **资产总额** | ≥ 上市公司最近一期经审计 **30%** | § 12 条 |
| **营业收入** | ≥ 上市公司最近一期经审计 **30%** | § 12 条 |
| **资产净额** | ≥ 上市公司最近一期经审计 **30%** | § 12 条 |
| **交易金额（含承担债务）** | ≥ 上市公司市值 **30%**（创业板）/ 50% | § 12 条 |
| **换股 / 发行股份购买资产** | 任一达到即触发 | § 12 条 |

**重组程序核心时限：**

| 程序 | 时限 | 关键节点 |
|------|------|---------|
| **停牌** | 筹划阶段立即停牌（一般 ≤ 10 个交易日） | 申请继续停牌不超过 25 日 |
| **董事会决议** | 披露重组预案 | 首次董事会 |
| **股东大会决议** | 决议通过 + 中小投资者单独计票 | 关联股东回避 |
| **交易所审核** | 受理后 6 个月（可延长） | 重大重组 |
| **证监会注册** | 15 个工作日 | 注册制下 |

**blocks（绝对禁止 — 重组）**：
- 重大重组达到标准未停牌 → **内幕信息泄露 + 责任人 5 年市场禁入**
- 关联交易未回避表决 → **决议无效 + 民事赔偿**
- 借壳上市（重组上市）达到标准未按 IPO 审核 → **按欺诈发行处罚**

**work but ships（可补救）**：
- 重组材料不齐 → 30 日内补正
- 股东大会未通过 → 修改方案 + 重新表决

#### 块 4：内幕交易 + 市场操纵（《证券法》2020 修订 + 《刑法》182 条）

**核心法规：**
- 《证券法》第 50 / 76 条（内幕信息 + 知情人员）
- 《刑法》第 180 条（内幕交易罪）
- 《刑法》第 182 条（操纵证券、期货市场罪）

**内幕信息 19 项（《证券法》第 80 条）：**

| 类别 | 例子 |
|------|------|
| **重大事件** | 重大资产重组 / 重大合同 / 重大亏损 / 重大诉讼 |
| **财务变化** | 业绩预告 / 利润分配 / 注册资本变化 |
| **股东变化** | 控股股东变化 / 持股 5% 以上股东变化 |
| **其他** | 涉及发行的重大事项 / 停牌 / 退市 |

**内幕交易 3 类处罚（2020 修订加重）：**

| 处罚 | 标准 |
|------|------|
| **行政处罚** | **违法所得 1-10 倍**（无违法所得则 50 万 - 500 万） |
| **市场禁入** | **5 年 / 10 年 / 终身**（按情节严重程度） |
| **刑事处罚** | **5 年以下 / 5-10 年**（情节特别严重） + 违法所得 1-5 倍罚金 |

**操纵市场 7 种情形（《刑法》第 182 条）：**

| 情形 | 含义 |
|------|------|
| 1. 连续买卖 | 影响证券交易价格或交易量 |
| 2. 约定交易 | 与他人串通，以事先约定的时间、价格和方式进行证券交易 |
| 3. 自买自卖 | 在自己实际控制的账户之间进行证券交易 |
| 4. 蛊惑交易 | 传播虚假信息或不确定重大信息 |
| 5. 抢帽子交易 | 公开评价推荐后反向交易 |
| 6. 虚假申报 | 频繁申报并撤销 |
| 7. 跨市场操纵 | 不同市场间操纵 |

**blocks（绝对禁止 — 内幕交易 / 操纵）**：
- 内幕信息知情人交易 → **1-10 倍违法所得 + 5 年禁入 + 刑责**
- 抢帽子交易（持牌分析师 + 反向交易）→ **5 年禁入 + 刑责**
- 蛊惑交易（传播虚假信息）→ **5-10 年有期徒刑 + 罚金**

**work but ships（可补救）**：
- 内幕信息知情人未登记 → 立即补登记 + 报告
- 信息发布延误 → 立即发布 + 致歉

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **IPO 欺诈发行** | 《证券法》第 181 条 + 《刑法》第 160 条 | 500 万 - 5000 万 + 5 年禁入 + 5 年以下有期徒刑 |
| 2. **重大信息披露虚假** | 《证券法》第 193 条 | 100 万 - 1000 万 + 市场禁入 |
| 3. **内幕交易** | 《证券法》第 191 条 + 《刑法》第 180 条 | 1-10 倍违法所得 + 5 年禁入 + 5 年以下 |
| 4. **操纵市场** | 《刑法》第 182 条 | 5-10 年有期徒刑 + 罚金 |

**关键差异（与 v3 标准一致）**：
- **绝对禁止（blocks）** → agent 看到这些**直接停止**——告诉用户"绝对不能做"
- **work but ships** → agent 提示整改 + 给出时限
- **FYI** → 记录不主动告知

**主动问（6 类不确定 — 资本市场增强版）**：
- 产品类型 + 上市地 + 募资金额？（IPO / 再融资 / 债券 / ABS）
- 财务报告有效期（IPO 一般 6 个月）？
- 是否触及重大资产重组标准？（30% 标准）
- 是否内幕信息知情人？（登记 + 隔离交易）
- 是否跨境上市？（多法域监管）
- 是否上市公司？（持续督导义务）

## Part B — Scene-Adaptive Practice Profile

### § B1 工作流(主入口 + 关键节点)

**主入口:** `issuance-procedure-advisor`(发行是资本市场逻辑起点)

**关键节点(资本市场 5 阶段):**

```
筹备阶段:
  Step 1: eligibility-checker + product-eligibility-checker + transaction-structure-advisor
          → 发行条件 + 产品设计 + 交易结构

申报阶段:
  Step 2: prospectus-drafter + prospectus-reviewer + disclosure-reviewer + compliance-assessor
          → 招股书 + 信披 + 合规

注册 / 发行阶段:
  Step 3: issuance-procedure-advisor + bond-eligibility-checker + asset-segregation-advisor
          → 注册批复 + 发行 + 簿记

上市 / 持续督导:
  Step 4: information-disclosure-monitor + compliance-advisor + regulatory-tracker
          → 持续信披 + 合规监控

终止 / 重启:
  Step 5: bond-coupon-calculator + issue-eligibility-checker
          → 产品调整 / 重启
```
