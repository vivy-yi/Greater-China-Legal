# 可转换公司债券法规汇编

## 基本信息

可转换公司债券（以下简称"可转债"）是指发行人依照法定程序发行、在一定期间内依据约定的条件可以转换成股份的公司债券。

## 主要法规依据

| 法规名称 | 发布机关 | 核心内容 |
|---------|---------|---------|
| 《上市公司证券发行管理办法》 | 证监会 | 可转债发行条件与程序 |
| 《证券法》 | 全国人大 | 证券发行基本规定 |
| 《可转换公司债券管理暂行办法》 | 证监会 | 专项管理规定 |

## 发行条件

### 主体资格
上市公司方可发行可转换公司债券。

### 基本条件（2023年修订前/注册制后）
1. 最近三个会计年度加权平均净资产收益率平均不低于6%
2. 扣除非经常性损益后的净利润与扣除前的净利润相比，以低者作为加权平均净资产收益率的计算依据
3. 发行后累计债券余额不超过最近一期末净资产额的50%
4. 最近三个会计年度实现的年均可分配利润不少于公司债券一年的利息
5. 募集资金使用项目符合国家产业政策规定

### 注册制后新规（2023年）
- 取消盈利能力硬性指标要求
- 改为信息披露为核心的审核机制
- 强调"报齐即报"原则
- 强化事中事后监管

## 发行程序

### 普通程序
1. 董事会提出发行方案
2. 股东大会审议通过
3. 编制募集说明书
4. 提交发行注册申请
5. 证监会注册（注册制）或审核（核定制）
6. 发行实施

### 申请文件
- 募集说明书
- 财务报告（近三年及一期）
- 律师法律意见书
- 保荐机构保荐工作报告
- 信用评级报告（如有）
- 可转债发行方案

## 转股条款

### 转股期
自发行结束之日起六个月后，持有人可以行使转股权利。

### 转股价格
- 转股价格不低于募集说明书公告日前二十个交易日该公司股票交易均价
- 不低于前一个交易日股票交易均价
- 向下修正条款：当股价连续N个交易日低于转股价的X%时触发

### 转股数量计算
转股数量 = 持有的可转债张数 × 100 ÷ 转股价格

## 赎回与回售

### 赎回条款（发行人权利）
当以下情形发生时，发行人有权决定按照债券面值加应计利息赎回全部或部分未转股的可转债：
- 股价连续30个交易日高于转股价的130%
- 可转债余额不足3000万元

### 回售条款（持有人权利）
当以下情形发生时，债券持有人有权将其持有的可转债全部或部分按面值加应计利息回售给发行人：
- 股价连续30个交易日低于转股价的70%-80%（具体触发比例由募集说明书约定）
- 募集资金用途变更

## 信息披露

### 发行阶段披露
- 募集说明书：载明可转债的主要条款、转股价格确定方式、赎回回售条款
- 发行公告
- 评级报告

### 持续信息披露
| 披露事项 | 时限要求 |
|---------|---------|
| 季度报告 | 季度结束后1个月内 |
| 半年度报告 | 上半年结束后2个月内 |
| 年度报告 | 年度结束后4个月内 |
| 转股情况 | 每次转股完成后2日内 |
| 赎回/回售触发 | 触发当日 |
| 向下修正转股价 | 决议当日 |

### 重大事项临时报告
- 发行人的重大资产重组、重大投资行为
- 发行人的减资、合并、分立、解散、申请破产
- 保证人或者担保物发生重大变化
- 涉及标的股票的诉讼、仲裁
- 持有发行人5%以上股份的股东的持股变化

## 债券持有人会议

### 应当召开会议的情形
1. 拟变更债券募集说明书的内容
2. 发行人不能按期支付本息
3. 发行人减资、合并、分立、解散或者申请破产
4. 担保人、担保物或增信措施发生重大变化
5. 法律法规或募集说明书规定的其他情形

### 会议决议效力
债券持有人会议依据法定程序通过的决议对全体债券持有人具有同等效力。

## 违约处置

### 违约情形
- 发行人未能按期支付可转债本金或利息
- 发行人实质违反募集说明书约定

### 处置程序
1. 受托管理人依法申请财产保全
2. 召开债券持有人会议
3. 依法申请仲裁或提起诉讼
4. 参与发行人的破产清算或重整

### 受托管理人职责
- 持续关注发行人经营状况
- 召集债券持有人会议
- 采取法律行动维护持有人权益
- 办理质押登记（如有担保）

## 特殊条款示例

### 典型条款组合
```
转股价格：初始转股价不低于募集说明书公告日前20日均价
转股期间：发行结束后6个月至到期日
赎回条款：股价≥转股价×130%，且未转股余额≥3000万
回售条款：股价≤转股价×70%持续30日
特别修正：股价≤转股价×85%持续15日可提议下修
期限：6年
利率：第一年0.3%，第二年0.5%，第三年1.0%...
```
