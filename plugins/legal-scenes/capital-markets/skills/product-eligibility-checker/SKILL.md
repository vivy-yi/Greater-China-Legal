## 一、适用情形

用户提出以下问题时调用本Skill：
- "这个能做ABS吗"
- "判断某类资产能否作为ABS基础资产"
- "基础资产准入核查"
- "某类收入能否做资产证券化"

## 二、核心分析

**第一步：确认资产类型**

| 资产类型 | 是否可入池 | 依据 |
|---------|-----------|------|
| 应收账款 | ✅ | 《管理规定》第9条 |
| 租赁债权 | ✅ | 《管理规定》第10条 |
| 收费权/收益权 | ✅（需稳定现金流） | 交易所挂牌条件确认指南 |
| 小额贷款债权 | ✅（须满足分散度） | 《管理规定》第11条 |
| 股权/债权（负面清单） | ❌ | 资产证券化基础资产负面清单 |
| 地方政府债务 | ❌ | 负面清单 |

**第二步：满足专项条件**

| 条件 | 核查要点 |
|-----|---------|
| 真实存在性 | 合同+发票+付款记录齐全 |
| 无权利负担 | 无质押/抵押/冻结 |
| 可特定化 | 债务人可识别/金额可确定 |
| 现金流稳定 | 历史记录≥3年/无重大负面事件 |

**第三步：排除负面清单**

- 地方政府及其下属部门
- 军队或军工资产
- 没有特许经营资质的基础设施收费权
- 不动产（住宅类为主）
- 关联往来款

## 三、输出

```
## 基础资产准入结论

【合格】/【有条件合格】/【不合格】

如不合格：
- 不合格原因：[具体原因]
- 整改建议：[如可整改，列出条件]
- 升级建议：[如涉及以下情况，移交专业律师]
  · 国有资产转让
  · 跨境资产
  · 单笔>50亿
```

## 升级决策门

出现以下情况须移交专业律师：
- 基础资产涉及国有资产（须法律意见书）
- 涉及跨境资产或外汇管制
- 原始权益人与担保人存在关联关系
- 涉及地方政府融资平台
