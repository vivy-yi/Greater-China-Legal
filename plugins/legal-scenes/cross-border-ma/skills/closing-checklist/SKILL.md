# /closing-checklist — 并购交割核查清单

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取交割条件/变更登记事项。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取交割条件/核查要点
2. 逐项核查先决条件
3. 核查交割须完成的事项
4. 输出交割清单
5. 确认交割日期

## 先决条件（Conditions Precedent）

| 条件 | 状态 | 备注 |
|---|---|---|
| ODI备案完成 | ☐ | 未完成则无法付汇 |
| 反垄断审查通过 | ☐ | 未通过不得交割 |
| 商务部门审批（如须） | ☐ | 须确认审批范围 |
| 其他监管审批 | ☐ | 行业特定审批 |
| SPA约定条件 | ☐ | 如无重大不利变化等 |

## 交割须完成事项

### 股权变更

```
工商变更登记：
1. 准备材料：股东会决议/章程修正案/SPA/新股东身份证明
2. 提交市场监督管理局
3. 领取新营业执照
4. 更新公司章程
5. 更新股东名册

税务变更（如有）：
1. 税务登记变更
2. 发票Credential变更
```

### 外汇登记

```
ODI外汇登记：
1. 商务部门ODI证书
2. 发改委ODI证书
3. 银行外汇登记
4. 完成登记后方可汇出投资款
```

### 证照变更

| 证照 | 变更事项 |
|---|---|
| 营业执照 | 股东变更 |
| 银行基本户 | 法定代表人/股权变更 |
| 税务 | 须同步更新 |
| 社保公积金 | 须同步更新 |
| 行业许可证 | 须变更（如有） |

## 交割后事项

```
30日内须完成：
1. 工商变更登记
2. 外汇登记
3. 银行账户更新
4. 通知合同相对方（按合同约定）

90日内须完成：
1. 董事会/监事会改选
2. 高管变更登记
3. 章程正式版本备案
```
