# Test Case: 境内企业收购境外AI芯片公司架构设计

## Query

我公司是一家在上海注册的AI芯片设计公司（非上市，注册资本1亿元），
拟以现金方式收购一家位于新加坡的AI芯片设计初创公司100%股权，交易金额约8,000万美元。
目标公司拥有多项AI芯片相关专利，其技术涉及可能受美国出口管制（EAR）管制的EDA工具和半导体设计技术。

请帮我设计交易架构方案，包括：
1. 境内审批路径（发改委/商务部/外管局）
2. 是否需要经营者集中申报
3. 是否涉及外商投资负面清单
4. 交易款跨境支付路径
5. 是否需要考虑VIE或中间控股公司架构

## Expected Behavior

1. 应分析ODI备案路径：发改委备案 + 商务部备案 + 外管局外汇登记
2. 应评估经营者集中申报是否触发（须计算营业额是否达到申报标准）
3. 应指出AI芯片涉及外商投资限制/禁止行业的可能
4. 应讨论美国出口管制（EAR）对交易的影响
5. 应建议新加坡或香港的中间控股公司架构
6. 应标注风险等级并在涉及国家安全审查时建议升级外部律师
