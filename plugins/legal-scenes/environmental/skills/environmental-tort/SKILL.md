# /environmental-tort — 环境侵权

## 工作流程
```
Step 1: 评估侵权构成(污染行为 + 损害 + 因果关系)
Step 2: 评估举证责任(污染方负主要举证)
Step 3: 申请环境鉴定
Step 4: 评估损害范围(人身 + 财产 + 生态)
Step 5: 评估连带责任(《民法典》第 1234 条)
```

## 环境侵权 4 要素

| 要素 | 注意 |
|------|------|
| **污染行为** | 排污 / 倾倒 / 泄漏 / 噪声 |
| **损害** | 人身 / 财产 / 生态 |
| **因果关系** | 污染与损害之间的关联 |
| **违法性** | 违反环保法规 / 标准 |

## 举证责任倒置(《民法典》第 1230 条)

```
✅ 污染方需证明:污染行为与损害无因果关系
✅ 受害人只需初步证明:污染 + 损害
✅ 高度盖然性 + 污染方举证不能 → 推定因果关系
```

## 环境鉴定

| 鉴定类型 | 适用 |
|----------|------|
| **因果关系鉴定** | 污染与损害关联 |
| **损害范围鉴定** | 损害程度 |
| **修复方案鉴定** | 生态修复 |
| **费用鉴定** | 修复费用 |

**鉴定申请:**

```
✅ 当事人申请
✅ 法院指定
✅ 双方协商
```

## 损害赔偿范围

| 项目 | 计算 |
|------|------|
| 人身损害 | 医疗费 + 误工 + 残疾 + 死亡 |
| 财产损害 | 实际价值 + 损失 |
| 生态损害 | 修复费用 + 服务功能损失 |
| 合理费用 | 鉴定费 + 律师费 + 差旅 |

**生态损害特别考虑:**

```
✅ 修复费用:实际修复 + 替代修复
✅ 服务功能损失:生态系统服务价值
✅ 期间损害:污染期间
✅ 调查评估费:环境监测 + 鉴定
```

## 连带责任(《民法典》第 1233 / 1234 条)

**多人侵权:**

| 情形 | 责任 |
|------|------|
| 共同污染 | 连带责任 |
| 多个污染源 | 各自责任 |
| 污染方 + 第三人 | 按份 / 连带 |

**特别注意:**

```
✅ 污染方对生态损害负无过错责任
✅ 第三人原因造成 → 污染方可追偿
✅ 多个污染源 → 举证不能分担连带责任
```

## 3 年时效

```
✅ 自知道或应当知道损害之日起 3 年
✅ 持续污染:时效自污染停止之日起算
```
