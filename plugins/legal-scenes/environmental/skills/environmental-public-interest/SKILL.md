# /environmental-public-interest — 环境公益诉讼

## 工作流程
```
Step 1: 评估主体资格(环保组织 / 检察机关)
Step 2: 评估案件范围(污染 / 生态破坏)
Step 3: 评估诉求(停止侵害 / 修复 / 赔偿)
Step 4: 评估管辖(中级法院)
Step 5: 准备起诉 + 跟进
```

## 主体资格 2 类

| 主体 | 条件 |
|------|------|
| **环保组织** | 依法设区市级以上 + 5 年无违法记录 + 业务范围含环保 |
| **检察机关** | 履行公益诉讼职责(《民事诉讼法》第 55 条) |

**特别注意:**

```
⚠️ 环保组织须 5 年无违法记录
⚠️ 业务范围须含环境保护
⚠️ 须设区市级以上
⚠️ 检察机关可支持起诉 / 独立起诉
```

## 案件范围(《环境保护法》第 58 条)

**可起诉行为:**

```
✅ 污染环境
✅ 破坏生态
✅ 损害社会公共利益
```

**不可起诉:**

```
❌ 个人损害(走环境侵权)
❌ 已达成和解
❌ 已履行修复
❌ 超出公益范围
```

## 诉求 3 类

| 诉求 | 适用 |
|------|------|
| **停止侵害** | 持续污染 |
| **排除妨碍** | 妨碍环境权益 |
| **消除危险** | 即将造成污染 |
| **修复生态环境** | 已造成损害 |
| **赔偿损失** | 生态损害 |
| **赔礼道歉** | 重大影响 |

## 管辖

**级别:** 中级人民法院 / 专门法院

**地域:**

| 案件 | 管辖 |
|------|------|
| 污染案件 | 污染行为地 / 结果地 |
| 生态破坏 | 生态破坏地 |

**特别考虑:**

```
✅ 跨行政区案件:共同上级法院指定
✅ 涉外:中级法院专属
```

## 程序流程

```
Step 1: 调查取证
Step 2: 起诉准备(主体证明 + 证据)
Step 3: 提交起诉状
Step 4: 法院审查(7 日内决定)
Step 5: 公告(30 日)
Step 6: 开庭审理
Step 7: 判决(支持 / 驳回)
```

**公告期 30 日:**

```
✅ 公告期间其他环保组织可申请参加诉讼
✅ 公告期满未申请 → 继续审理
```

## 配套机制

| 机制 | 适用 |
|------|------|
| **生态环境损害赔偿** | 国务院授权省级 / 市级政府索赔 |
| **检察机关支持起诉** | 协助举证 |
| **惩罚性赔偿** | 故意 + 严重情节(2022 民法典修订) |
