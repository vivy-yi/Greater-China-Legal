# /consumer-protection-checker — 消费者保护合规检查

## 加载上下文

**首次使用时：** 读取 `../CLAUDE.md` 获取消保规则/七项禁止。

## 工作流程

1. 读取 `../CLAUDE.md` → 获取七项禁止/投诉处理规则
2. 评估产品消保措施
3. 识别违规点
4. 输出整改建议

## 检查维度

### 维度一：信息披露

| 检查项 | 监管要求 |
|---|---|
| 年化利率展示 | 须以显著方式展示APR |
| 费用说明 | 所有费用（含服务费）须明示 |
| 合同条款 | 不得含隐藏条款 |
| 风险提示 | 须对借款人进行风险提示 |

### 维度二：利率合规（141号文）

| 检查项 | 合规标准 |
|---|---|
| 综合年化利率 | 须≤36% |
| 砍头息 | 禁止（不得从本金中预扣） |
| 服务费前置 | 禁止（费用不得前置扣除） |
| 罚息利率 | 不得显著高于合同利率 |

### 维度三：催收规范

| 检查项 | 合规标准 |
|---|---|
| 催收时间 | 不得在22:00-8:00催收 |
| 催收频率 | 不得频繁骚扰 |
| 联系人催收 | 不得向无关联系人催收 |
| 暴力催收 | 绝对禁止 |

### 维度四：个人信息保护

| 检查项 | 合规标准 |
|---|---|
| 同意收集 | 须明示同意，不可默认勾选 |
| 用途说明 | 须说明收集目的 |
| 数据共享 | 须告知并取得同意 |
| 境外传输 | 须符合数据出境规定 |
