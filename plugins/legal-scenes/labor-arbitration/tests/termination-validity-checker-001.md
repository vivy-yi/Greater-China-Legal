# Test Case: 严重违反规章制度解除劳动合同合法性审查

## Query

我公司拟以"严重违反公司规章制度"为由解除一名员工（工龄3年8个月，月均工资15,000元）。
具体情况如下：
1. 该员工在未经审批的情况下，将公司内部技术文档发送至个人邮箱
2. 公司《信息安全管理规定》第12条明确禁止"未经授权复制、传输公司机密信息"，违者可解除劳动合同
3. 该规定在公司OA系统公示，员工入职时签署了《制度确认书》
4. 员工行为发生在2026年5月，公司2026年6月决定解除
5. 员工否认违规，称发送的是非机密信息
6. 公司未设立工会

请分析：
1. 解除是否合法
2. 如合法，补偿方案是什么
3. 如不合法，最可能的法律后果
4. 是否存在程序风险

## Expected Behavior

1. 应审查规章制度的有效性：是否经民主程序、是否已公示、内容是否合法合理
2. 应分析"严重违反"的认定标准，引用司法实践中对类似行为的裁判口径
3. 应指出公司无工会时解除通知须报上级工会或当地总工会的程序要求
4. 应计算经济补偿金（N=4个月）或违法解除赔偿金（2N=8个月）
5. 应标注论证强度（强/中/弱/存疑）
6. 应评估风险等级并在涉及特殊群体时建议升级律师
