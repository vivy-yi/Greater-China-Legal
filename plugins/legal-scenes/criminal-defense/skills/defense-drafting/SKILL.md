# /defense-drafting — 刑事辩护文书起草

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取输出模板 + 对外文档规范 + § A3 quiet mode(对外文档策略过滤)。

## 工作流程

```
Step 1: 选定文书类型(辩护词 / 上诉状 / 不起诉意见 / 取保申请)
Step 2: 套用文书结构模板
Step 3: 填充案件事实 + 法条 + 辩护理由
Step 4: 主办律师复核 + 修改
Step 5: 当事人确认(如适用)
Step 6: 装订 + 提交
```

## 5 类文书结构模板

### 一、辩护词

```
1. 抬头(致 X X 法院)
2. 辩护人身份介绍(姓名+执业证号+律所)
3. 出庭前对管辖/回避的意见(如有)
4. 辩护人总体观点(无罪/罪轻 + 一句话理由)
5. 具体辩护理由:
   - 事实认定(争议事实 + 证据分析)
   - 定性分析(此罪/彼罪/无罪)
   - 量刑情节(从轻/减轻)
   - 程序问题(如有)
6. 结论:明确请求(无罪 / 减轻 / 从轻)
7. 辩护人签字 + 日期
```

### 二、上诉状

```
1. 抬头(致 X X 法院)
2. 上诉人基本信息(被告人/辩护人)
3. 原审法院 + 案号 + 判决日期
4. 上诉请求(撤销原判 / 改判无罪 / 减轻处罚)
5. 上诉理由:
   - 事实认定错误
   - 证据不足 / 矛盾
   - 定性不准
   - 量刑过重
   - 程序违法
6. 此致法院 + 上诉人签字 + 日期
```

### 三、不起诉意见书(致检察院)

```
1. 抬头(致 X X 检察院)
2. 辩护人身份
3. 当事人基本信息
4. 案件来源(公安移送)
5. 不起诉理由:
   - 事实不清 / 证据不足
   - 不构成犯罪(刑法要件缺失)
   - 情节轻微(《刑法》第 37 条)
   - 超过追诉时效
6. 法律依据(《刑诉法》第 177 条)
7. 请求:依法作出不起诉决定
8. 辩护人签字 + 日期
```

### 五、法律意见书(致办案机关)

```
1. 抬头
2. 律师身份
3. 委托方 + 委托事项
4. 案情简述
5. 法律分析(罪名要件 + 司法解释 + 类案)
6. 结论意见
7. 律师签字 + 日期 + 律所盖章
```

## 文书通用规则

### 法条引用规范

```
✅ 正确:《刑法》第 X 条(2017 修订)
✅ 正确:《刑诉法》第 X 条(2018 修订)
✅ 正确:《最高人民法院关于...的解释》(法释〔2018〕XX 号)
❌ 错误:《刑法》第 X 条(不写版本)
❌ 错误:援引废止法条
```

### 证据引用规范

```
✅ 正确:见案卷第 X 卷第 Y 页
✅ 正确:经庭审举证质证的 [证据名称]
❌ 错误:"经审理查明"直接复制(无证据基础)
```

### 论证逻辑规范(P-F-C)

```
P 前提:案件事实 / 法条规定
F 依据:证据 / 司法解释 / 类案
C 结论:请求法院采纳
```

## 对外文档策略过滤(§ A3 quiet mode)

**送交法院 / 检察院的文书:**
- 删除 [ASSUMPTION] 标注
- 删除 P-F-C 推理链
- 删除内部策略说明
- 保留法条 + 证据引用
- 保留 [verify] 标注

**内部工作底稿:** 保留全部。

## 主办律师复核(必走)

| 文书类型 | 复核要求 |
|----------|----------|
| 辩护词 | 主办律师全篇复核 |
| 上诉状 | 主办律师 + 律所审批(若涉及量刑变更) |
| 不起诉意见 | 主办律师 + 协办 + 律所 |
| 法律意见书 | 主办律师 + 律所盖章 |
