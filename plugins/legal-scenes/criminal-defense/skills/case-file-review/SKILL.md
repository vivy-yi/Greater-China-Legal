# /case-file-review — 阅卷分析与证据梳理

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取大文件处理规范 + 证据注册表。

## 工作流程

```
Step 1: 阅卷申请(检察院阶段 / 法院阶段)
Step 2: 现场阅卷(可拍照 / 复印)
Step 3: 证据三性分析(真实性 / 合法性 / 关联性)
Step 4: 矛盾点标注
Step 5: 非法证据排除线索识别
Step 6: 形成阅卷笔录 + 证据清单
```

## 阅卷权关键点

| 阶段 | 阅卷权 | 范围 |
|------|--------|------|
| 侦查阶段 | **有限**(只能查阅涉案的鉴定意见等技术性证据材料) | 不可阅全卷 |
| 审查起诉 | **全面** | 全卷 |
| 一审阶段 | **全面** | 全卷 |

**实务注意:** 侦查阶段阅卷权争议大,实务中部分检察机关允许律师复印全卷,需主动申请。

## 证据三性分析框架

### 真实性(是否客观存在)

| 风险点 | 检查方法 |
|--------|----------|
| 物证来源 | 提取笔录 + 现场照片 + 见证人 |
| 书证原件 | 是否提供原件 / 复印件来源 |
| 证人证言 | 是否亲历 / 是否受到诱导 |
| 鉴定意见 | 鉴定机构资质 + 鉴定人资质 + 鉴定方法 |

### 合法性(取得方式是否合法)

**5 类非法取证情形:**

| 类型 | 表现 | 救济 |
|------|------|------|
| 刑讯逼供 | 身体伤痕 / 同步录音录像缺失 | 启动非法证据排除程序 |
| 威胁引诱欺骗 | 口供与事实明显矛盾 | 同上 |
| 非法拘禁 | 超期羁押 | 申请变更强制措施 + 申诉 |
| 非法搜查扣押 | 无证搜查 / 扣押清单不符 | 申请排除 |
| 鉴定程序违法 | 单方委托 / 鉴定材料污染 | 申请重新鉴定 |

### 关联性(与案件事实是否相关)

- 直接关联(事实直接相关)
- 间接关联(辅助证明)
- **无关联**(必须排除,例如品格证据)

## 矛盾点识别

**重点对比:**
1. **口供之间:** 多次讯问笔录是否一致(变化点 = 突破口)
2. **口供与客观证据:** 物证 / 书证 / 鉴定意见
3. **证人证言之间:** 同事件不同证人
4. **证人证言与客观证据:** 时空逻辑

**标注方式:**
```
矛盾点 C-001:
  证据1: 口供第3卷第15页(承认在场)
  证据2: 监控录像(显示不在场)
  结论: 真实性存疑,可主张排除/重新质证
```

## 非法证据排除线索(必查)

| 线索 | 来源 |
|------|------|
| 同步录音录像缺失 | 《刑诉法》第 123 条 |
| 笔录时间倒签 | 与提讯证 / 看守所登记比对 |
| 身体伤痕 | 会见时询问 |
| 长期连续讯问 | 超过 24h |
| 辨认程序违法 | 单独辨认 / 暗示性辨认 |
