# /bail-and-detention — 强制措施变更申请

## 加载上下文

**首次使用时:** 读取 `../CLAUDE.md` 获取法定时限 + 升级路径 + 风险等级。

## 工作流程

```
Step 1: 评估羁押必要性 → 决定申请类型
Step 2: 选择申请时机(侦查阶段 / 审查起诉 / 审判)
Step 3: 撰写申请书
Step 4: 提交(办案机关 / 检察院 / 法院)
Step 5: 跟进 + 复议/申诉(若被驳回)
```

## 强制措施 4 类对照

| 类型 | 适用 | 律师权 |
|------|------|--------|
| **拘传** | 12h 内,不得超 24h | 律师可陪同 |
| **取保候审** | 可能判 ≤ 3 年 / 怀孕 / 重病 / 哺乳 | 申请律师即可 |
| **监视居住** | 特定情形(无固定住所/可能毁灭证据) | 申请律师 |
| **逮捕** | 严重罪行 / 有社会危险性 | **羁押必要性审查**(检察院) |

## 取保候审申请关键点

**申请书必备:**
```
1. 当事人基本信息
2. 涉嫌罪名 + 案件事实(承认/否认)
3. 申请理由(满足取保条件):
   - 可能判 ≤ 3 年有期徒刑
   - 怀孕 / 哺乳期妇女
   - 重病(医院证明)
   - 羁押期限将超法定
   - 案件证据已固定
   - 不致发生社会危险性
4. 保证人 / 保证金方案
5. 申请人签字 + 日期
```

**保证人 vs 保证金:**

| 方式 | 适用 | 注意 |
|------|------|------|
| 保证人 | 1-2 名,有固定住处/收入,有能力履行 | 保证人违责罚款 ¥1万-3万 |
| 保证金 | 1000 元起 | 涉案金额大时按比例提高 |

## 羁押必要性审查(逮捕后)

**申请时点:** 逮捕后任何时间 + 检察院审查起诉时

**申请书核心:** 列举羁押条件变化(案件事实变化 / 身体健康 / 家庭情况 / 同案犯处理进度)

## 拒收 / 不予收押申诉

当看守所拒收或办案机关不予收押时:
- 立即向同级检察院申诉
- 引用《刑诉法》第 75 条
- 必要时向上级检察院申诉
