# 判决债权投资咨询

## 一、基本信息

### 1.1 核心目的
为特殊机会投资者提供判决债权的投资可行性分析与执行变现策略，覆盖终本案件恢复执行、被执行人财产线索挖掘、执行异议排除等核心场景。

### 1.2 适用范围
- 民事判决债权（金钱给付判决为主）
- 仲裁裁决债权
- 公证债权文书债权
- 法院调解书债权
- 终本（终结本次执行程序）案件的债权恢复执行

### 1.3 数据源与判断框架引用

| 引用来源 | 用途 |
|---------|------|
| [YD] 元典MCP | 执行案件信息、被执行人涉诉情况、终本标准查询 |
| [WKL] 裁判文书网 | 判决书、执行裁定书、终本裁定 |
| [GOV] 政府平台 | 企业信用公示系统、不动产登记信息、车辆登记 |
| [web] 网络搜索 | 被执行人经营状况、公开财产线索 |
| [model] 模型推理 | 执行回收率预测、执行周期预估（须核实） |

## 二、分析框架

### 2.1 判决债权投资可行性评估五步法

```
Step 1: 判决质量审查
├── 判决是否已生效（一审上诉期是否届满）
├── 是否存在再审申请/抗诉程序
├── 判决金额及利息计算是否明确
├── 被告是否为唯一责任人（追加被执行可能性）
└── 判决可执行性评估（给付内容是否明确）

Step 2: 被执行人偿付能力调查
├── 不动产查询（住宅/商业/厂房/土地 — 全国不动产登记联网）
├── 银行账户查询（对公/对私 — 网络执行查控系统已覆盖）
├── 车辆查询（公安部车辆管理所登记）
├── 股权查询（被投资公司—企业信用信息公示系统）
├── 知识产权查询（商标/专利/著作权）
├── 应收款项/到期债权（公开合同信息）
└── 隐蔽资产线索（股东借款/关联交易/代持）

Step 3: 执行障碍评估
├── 终本原因（无财产/无法处置/执行异议）
├── 其他债权人情况（多案轮候查封）
├── 是否存在执行异议/案外人异议
├── 是否存在参与分配（被执行人资不抵债）
└── 执行时效（申请执行期限2年）

Step 4: 执行成本测算
├── 案件受理费（按标的额比例）
├── 执行申请费（按标的额）
├── 评估拍卖费（担保物评估/拍卖佣金）
├── 律师费/调查费
├── 差旅费/公告费
└── 利息损失（折现值）

Step 5: 投资回报推演
├── 最优情形（全额执行+利息）：[计算IRR]
├── 基准情形（部分执行）：[回收率假设]
├── 最差情形（终本维持）：[损失确认]
└── 建议投资策略（折价买入/分期支付）
```

### 2.2 终本案件恢复执行条件

| 恢复条件 | 证据要求 | 操作路径 |
|---------|---------|---------|
| 发现新财产线索 | 银行存款/不动产/车辆线索 | 提交财产线索+恢复执行申请 |
| 发现有可供执行资产且此前未处置 | 权属证明+价值评估 | 申请恢复处置 |
| 原终本后发现被执行人转移资产 | 转账记录/异常交易 | 申请追加执行/撤销权诉讼 |
| 被执行人法人资格恢复/清算后新发现资产 | 工商登记变更 | 申请恢复 |
| 和解协议不履行 | 和解协议+违约证明 | 恢复执行原判决 |

### 2.3 判决债权折价转让特殊机制

```
判决债权转让
├── 法律依据：《民法典》第545-550条（债权转让）
├── 禁止转让情形：
│   ├── 判决中明确禁止转让的债权
│   ├── 专属于债权人自身的债权（如抚养费/赡养费）
│   └── 按性质不得转让的债权
├── 转让通知效力：通知债务人即生效（无需同意）
├── 转让定价依据：
│   ├── 判决本金 × [回收率折价系数] = 转让价格
│   ├── 折价范围：通常为判决金额的10%-60%
│   └── 决定因素：被执行人偿付能力+执行难度
└── 特殊优势：受让人可申请变更申请执行人
    ├── 法律依据：《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第9条
    └── 操作路径：债权转让+申请变更执行人+恢复执行
```

## 三、操作流程

### 3.1 判决债权投资全流程

**Phase 1：债权的获取与审查（1-2周）**
- [ ] 获取判决书/裁定书/调解书全文
- [ ] 审查判决的生效状态和执行时效
- [ ] 在裁判文书网/执行信息公开网查询被执行人信息
- [ ] 初步执行可行性判断

**Phase 2：深度尽调（2-4周）**
- [ ] 被执行人主体存续状态（是否已注销/吊销/破产）
- [ ] 网络执行查控系统（委托律师调取）
- [ ] 不动产中心查询（委托律师持调查令）
- [ ] 企业信用信息公示系统（股东/对外投资/变更记录）
- [ ] 关联方/利害关系人调查（追加被执行可能性）
- [ ] 终本案件卷宗查阅（如有）

**Phase 3：执行策略设计**
- [ ] 财产保全方案（如有未保全财产）
- [ ] 追加被执行人的路径选择
- [ ] 执行异议的应对方案
- [ ] 参与分配策略（多债权人情形）
- [ ] 税务筹划（债权转让涉及的增值税/所得税）

**Phase 4：变现执行（3-12个月）**
- [ ] 申请恢复执行/首次执行
- [ ] 法院查控与资产处置跟进
- [ ] 评估拍卖/以物抵债决策
- [ ] 执行和解谈判（折价即时清偿）

### 3.2 追加被执行人路径

| 追加情形 | 法律依据 | 适用条件 |
|---------|---------|---------|
| 一人有限责任公司的股东 | 变更追加规定第20条 | 财产混同 |
| 未缴纳出资的股东/发起人 | 变更追加规定第17条 | 未实缴/加速到期 |
| 抽逃出资的股东 | 变更追加规定第18条 | 抽逃事实认定 |
| 未经清算即注销的股东 | 变更追加规定第21条 | 未经依法清算 |
| 无偿接受财产的上级单位 | 变更追加规定第22-25条 | 无偿接受/注册资金未到位 |

## 四、升级决策门

出现以下情形，将案件移交专业律师：

### 4.1 必须升级情形
- **被执行人进入破产程序** — 需在破产程序中申报债权
- **判决书存在再审/抗诉风险** — 生效判决被申请再审
- **追加被执行人诉讼** — 需通过诉讼程序追加
- **案外人执行异议之诉** — 需代理诉讼对抗异议人
- **申请人/被执行人涉及刑事犯罪** — 非法吸收存款/集资诈骗/洗钱
- **跨境执行** — 判决需要在境外承认与执行
- **终本案件涉及虚假诉讼** — 原判决可能被撤销
- **执行时效已届满** — 需恢复执行时效

### 4.2 升级咨询模板

```
【升级咨询】
判决/案号：[法院+案号]
判决金额：[本金+利息]
执行状态：[首次执行/终本/恢复执行]
被执行人：[自然人/企业名称]
主要执行障碍：[终本/无可执行财产/执行异议/其他]
预计回收率：[预估%]
优先级：紧急/高/中
```

## 五、本Skill不涵盖

1. 判决文书生效后的上诉/再审代理
2. 刑事附带民事判决中的财产执行
3. 行政判决/行政裁决的执行
4. 劳动仲裁裁决的执行（劳动仲裁特殊程序）
5. 婚姻家庭类判决（抚养费/探视权）的执行
6. 判决执行中的人格权侵害禁令救济
7. 知识产权行为保全令的执行
