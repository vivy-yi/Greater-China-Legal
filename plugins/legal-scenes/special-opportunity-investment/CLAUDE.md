### § A8.1 特殊机会投资特别注意 4 大块（困境债 + 不良资产 + 重整投资人 + S 基金）

> **核心原则**：特殊机会投资的核心是**估值折扣 + 重组能力 + 退出路径**。"买入便宜 + 主动重组 + 退出兑现"是核心策略。

#### 块 1：困境债投资（DIP / Distressed Debt）

**核心法规：**
- 《企业破产法》第 18 / 75 条（破产申请对未到期债权的影响）
- 《企业破产法》第 82-86 条（重整投资人）
- 《商业银行不良贷款转让规则》

**困境债投资 4 种类型：**

| 类型 | 含义 | 典型折扣 |
|------|------|---------|
| **破产重整债权** | 困境企业的破产债权 / 重整债权 | 票面 30-70% |
| **不良贷款** | 银行转让的不良贷款（NPL） | 票面 10-40% |
| **困境企业可转债** | 困境企业的可转债 / 公司债 | 票面 20-50% |
| **困境企业股权** | 困境企业的少数股权 / 优先股 | 大幅折扣 |

**困境债投资流程（5 步）：**

| 步骤 | 内容 |
|------|------|
| 1. **债权识别** | 通过 AMC / 银行转让 / 律所推荐获取 |
| 2. **尽调** | 财务尽调 + 法律尽调 + 商业尽调 |
| 3. **估值** | 破产清算价值 vs 重整价值 |
| 4. **竞价** | 拍卖 / 竞标 / 协议转让 |
| 5. **管理** | 加入债权人委员会 / 推动重整 |

**blocks（绝对禁止 — 困境债）**：
- 隐瞒债权瑕疵（如连带担保 / 诉讼时效）→ **撤销 + 民事赔偿**
- 操纵债权人会议 → **《企业破产法》第 130 条 + 刑责（妨害清算）**

**work but ships（可补救）**：
- 债权估值过低 → 重新评估
- 重组方案分歧 → 协商 + 投票

#### 块 2：不良资产处置（AMC + 批量处置）

**核心法规：**
- 《金融资产管理公司条例》（2000 + 2024 修订）
- 《商业银行不良贷款转让规则》（银保监 2021）
- 《金融资产管理公司开展非金融机构不良资产业务管理办法》

**5 大 AMC（金融资产管理公司）：**

| AMC | 业务范围 |
|-----|---------|
| **中国华融（现中国中信金融资产）** | 全国 |
| **中国长城** | 全国 |
| **中国东方** | 全国 |
| **中国信达** | 全国 |
| **地方 AMC（每省 1-2 家）** | 本地区 |

**不良资产处置方式（5 种）：**

| 方式 | 适用 | 收益 |
|------|------|------|
| **催收** | 有还款能力 | 全额或部分回收 |
| **重组 / 重整** | 困境企业 | 重整后退出 |
| **诉讼 / 仲裁** | 拒还或争议 | 司法强制执行 |
| **转让（打包 / 单户）** | 流动性需求 | 二次折扣 |
| **资产证券化 ABS** | 大批量 / 标准化 | 资本市场退出 |

**不良资产尽调核心要点：**

| 维度 | 内容 |
|------|------|
| **债权合法性** | 合同 / 担保 / 时效 |
| **债务人资产** | 房地产 / 股权 / 应收账款 |
| **优先权** | 抵押权 / 质权 / 建设工程价款优先受偿权 |
| **关联方** | 关联交易 / 抽逃出资 |

**blocks（绝对禁止 — 不良资产）**：
- 转让虚假债权 → **《刑法》第 224 条合同诈骗**
- AMC 工作人员受贿 → **《刑法》第 385 条受贿罪**

**work but ships（可补救）**：
- 债务人无力偿还 → 重组 / 减让
- 担保物价值下降 → 重新评估

#### 块 3：重整投资人（核心进入路径）

**重整投资人核心要点：**

| 要点 | 内容 |
|------|------|
| **类型** | 战略投资人 / 财务投资人 / 产业投资人 |
| **资格** | 资金实力 + 产业背景 + 重整方案可行性 |
| **招募** | 公开招募 / 邀请招募 / 协商指定 |
| **权利** | 取得重整后股权 + 参与经营 + 调整债权 |
| **义务** | 支付重整投资款 + 履行重整计划 |

**重整投资人 5 步路径：**

| 步骤 | 内容 | 时限 |
|------|------|------|
| 1. **尽职调查** | 财务 + 法律 + 商业 | 1-2 个月 |
| 2. **提交意向** | 重整意向书 + 报价 + 方案 | 进入程序后 1 个月内 |
| 3. **竞争遴选** | 公开招募 / 竞争谈判 | 1-2 个月 |
| 4. **签订协议** | 重整投资协议 + 重整计划草案 | 1-3 个月 |
| 5. **重整执行** | 注入资金 + 经营调整 + 退出 | 6 个月 - 2 年 |

**重整投资款支付方式：**

| 方式 | 含义 |
|------|------|
| **现金收购** | 直接支付现金 |
| **债权转股权** | 债权人将债权转为重整后股权 |
| **资产入股** | 重整投资人以资产出资 |
| **混合方式** | 现金 + 股权 + 资产 |

**blocks（绝对禁止 — 重整投资人）**：
- 重整资金不到位 → **重整计划被否决 + 转入清算**
- 操纵债权人会议 → **《企业破产法》第 130 条 + 刑责**
- 隐瞒关联方关系 → **撤销重整资格**

**work but ships（可补救）**：
- 资金未全额到位 → 申请延期 + 担保
- 重整计划不通过 → 修改方案 + 重新表决

#### 块 4：S 基金 / 二级市场退出

**核心概念：**
- **S 基金（Secondary Fund）** — 专门投资 PE 基金份额的基金
- **二级市场转让** — 原 LP 将基金份额转让给新 LP
- **GP 主导交易** — GP 寻找新投资人替代部分 LP

**S 基金退出 4 大优势：**

| 优势 | 说明 |
|------|------|
| **折扣** | 二级市场基金份额通常 10-30% 折扣 |
| **快进快出** | 无需等待底层项目退出 |
| **J 曲线缩短** | 立即进入成熟期 |
| **底层透明** | 已能看到底层项目表现 |

**S 基金核心结构（5 类）：**

| 类型 | 含义 |
|------|------|
| **直接 LP 份额转让** | LP-to-LP 直接交易 |
| **LP 份额买断** | S 基金买断部分 LP 份额 |
| **基金重组** | 重组基金结构 + 部分退出 |
| **要约收购** | 面向所有 LP 的统一报价 |
| **GP 主导延续** | GP 主导的延期 / 重组 |

**S 基金核心合规：**

| 风险 | 处理 |
|------|------|
| **GP 同意权** | 大多数基金合同要求 GP 同意 |
| **优先购买权** | 其他 LP 优先购买 |
| **转让限制** | 投资人资格 / 数量限制 |
| **信息披露** | 底层项目尽调材料 |

**blocks（绝对禁止 — S 基金）**：
- 绕过 GP 同意转让 → **违约 + 基金合同撤销**
- 重大隐瞒 → **GP 拒绝 + 民事赔偿**

**work but ships（可补救）**：
- GP 拒绝 → 协商 + 调整方案
- LP 优先购买 → 协商定价

#### 块 5：4 大绝对禁止（含具体法条 + 后果分级）

| 禁止 | 法条 | 后果 |
|------|------|------|
| 1. **转让虚假债权** | 《刑法》第 224 条合同诈骗罪 | 3-10 年 + 罚金 |
| 2. **操纵债权人会议** | 《企业破产法》第 130 条 + 妨害清算罪 | 5 年以下 |
| 3. **AMC 工作人员受贿** | 《刑法》第 385 条受贿罪 | 5 年以下 / 5-10 年 |
| 4. **重整资金不到位** | 重整计划被否决 + 转入清算 | 失权 + 损害赔偿 |

**主动问（6 类不确定 — 特殊机会投资版）**：
- 投资类型（困境债 / 不良资产 / 重组 / S 基金）？（核心策略）
- 行业 + 规模 + 折扣率？（估值依据）
- 重组能力（自营 / 引入第三方）？（重组方案）
- 退出路径（IPO / 并购 / S 基金）？（收益预期）
- 中基协备案 + 跨境 ODI？（合规状态）
- AMC / 律所 / 评估机构协同？（中介）
