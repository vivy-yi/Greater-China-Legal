# Test Case: 银行不良资产包收购法律审查

## Query

我公司是一家私募基金，计划从某国有大型银行收购一个不良资产包。
资产包基本情况如下：
1. 资产包包含12笔不良贷款，本金余额合计约3.2亿元
2. 担保方式：其中8笔有房产抵押担保，2笔有保证担保，2笔为信用贷款
3. 抵押物分布：深圳4处、广州3处、佛山1处
4. 债务人状态：3家已进入破产程序，5家已停产但未破产，4家仍在经营
5. 大部分贷款已逾期超过2年
6. 银行已对其中6笔提起诉讼，2笔已取得生效判决但未执行到位
7. 转让方式：公开竞价转让

请审查以下事项：
1. 债权转让的合规要求（是否有优先购买权等）
2. 抵押担保的有效性：是否存在顺位抵押、抵押物查封情况
3. 已进入破产程序的债权如何处理
4. 已有生效判决的债权后续执行路径
5. 建议收购价格区间的主要考量因素
6. 尽职调查应重点关注的事项清单

## Expected Behavior

1. 应分析金融不良资产转让的监管要求（银保监会相关规定）
2. 应审查抵押担保的有效性和可执行性
3. 应分析破产程序中债权申报和参与分配的策略
4. 应分析已判决债权的执行难点和路径
5. 应给出收购价格建议的考量因素
6. 应列出法律尽职调查事项清单
7. 应标注风险等级，对涉及破产和执行的债权标注高不确定性
