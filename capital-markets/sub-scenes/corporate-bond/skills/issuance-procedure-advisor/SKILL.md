# Skill: issuance-procedure-advisor（债券发行程序指导）

## 功能定位

为债券发行人提供从准备阶段到发行完成的全流程程序指导，包括各阶段工作内容、时间节点、所需文件及注意事项。

## 债券类型支持

| 债券类型 | 监管主体 | 审核方式 | 主要法规 |
|---------|---------|---------|---------|
| 公司债 | 证监会 | 注册制（2023年后） | 《公司债券发行与交易管理办法》 |
| 企业债 | 发改委→证监会 | 审核制/注册制 | 《企业债券管理条例》 |
| 可转债 | 证监会 | 注册制 | 《上市公司证券发行管理办法》 |

│  └─ 1.3 尽职调查                                          │
├─────────────────────────────────────────────────────────┤

│  └─ 2.3 提交注册/审核                                     │
├─────────────────────────────────────────────────────────┤
│  第三阶段：审核阶段（注册制下约12个工作日）                 │
│  ├─ 3.1 监管机构问询（如有）                              │

## 第一阶段：准备阶段

### 1.1 内部决策

#### 股份有限公司/有限责任公司（公司债/企业债）

| 步骤 | 内容 | 依据 |
|-----|------|-----|
| 董事会决议 | 制定债券发行方案 | 《公司法》第108条 |
| 股东大会决议 | 审议通过发行方案 | 《公司法》第99条 |

#### 上市公司（可转债）

| 步骤 | 内容 | 依据 |
|-----|------|-----|
| 董事会决议 | 制定可转债发行方案 | 《上市公司证券发行管理办法》 |
| 股东大会决议 | 审议通过，决议有效期通常12个月 | 证监会相关规定 |

### 1.2 中介机构选聘

| 中介机构 | 职责 | 资质要求 |
|---------|------|---------|
| 主承销商 | 尽职调查、发行方案设计、承销 | 证监会批准的公司债券承销资格 |
| 律师事务所 | 法律尽职调查、出具法律意见书 | 律师事务所执业许可 |
| 会计师事务所 | 财务审计、出具审计报告 | 会计师事务所执业许可 |
| 评级机构（如需） | 信用评级、出具评级报告 | 证监会认可的评级资质 |

### 1.3 尽职调查

**主要调查内容**：
- 发行人基本情况及历史沿革
- 财务状况及经营成果
- 关联交易及同业竞争
- 重大债权债务
- 诉讼仲裁及行政处罚
- 募集资金投向

## 第二阶段：申报阶段

### 2.1 申请文件清单

#### 公司债申请文件

```
1. 募集说明书（申报稿）
2. 发行公告
3. 发行人最近三年经审计的财务报告
4. 主承销商核查意见
5. 律师法律意见书
6. 评级报告（如需评级）
7. 担保文件（如有担保）
8. 发行人董事会决议、股东大会决议
9. 资产权属证明文件
10. 偿债保障措施说明
```

#### 企业债申请文件

```
1. 发行企业债券的申请书
2. 发行章程（募集说明书）
3. 省级发改委初审意见
4. 会计师事务所审计的财务报表
5. 律师法律意见书
6. 担保函或抵押合同（如有）
7. 评级报告
8. 资金投向证明文件
```

#### 可转债申请文件

```
1. 募集说明书
2. 发行公告
3. 近三年财务报告及审计报告
4. 发行可转债的可行性分析
5. 律师法律意见书
6. 保荐机构保荐工作报告
7. 信用评级报告（如有）
8. 董事会、股东大会决议
9. 转股价格计算依据
```

### 2.2 内部审批流程

```
股东大会审议 → 通过决议 → 授权董事会办理具体事宜
                ↓
         决议有效期：通常12个月内有效
```

### 2.3 提交审核

| 债券类型 | 提交对象 | 提交方式 |
|---------|---------|---------|
| 公司债 | 证监会/交易所 | 电子化注册系统 |
| 企业债 | 证监会 | 企业债券注册审核系统 |
| 可转债 | 证监会 | 证券交易所审核系统 |

## 第三阶段：审核阶段

### 3.1 注册制流程（公司债、可转债）

```
提交注册申请
    ↓
证券交易所在5个工作日内受理
    ↓
交易所审核（通常不超过12个工作日）
    ↓
证监会注册（通常5个工作日内）
    ↓
注册生效
```

### 3.2 审核制流程（存量企业债、部分企业债）

```
提交审核申请
    ↓
省级发改委初审
    ↓
国家发改委审核（存量已转证监会）
    ↓
取得发行批文
```

### 3.3 问询回复

**常见问询问题类型**：
- 财务数据异常波动解释
- 募集资金用途合规性
- 担保或增信措施充分性
- 发行人治理结构
- 关联交易的公允性

**回复要求**：
- 在15个工作日内提交回复
- 如需延期，需提交延期申请并说明理由

## 第四阶段：发行阶段

### 4.1 发行前准备

| 工作内容 | 时间要求 |
|---------|---------|
| 刊登募集说明书 | 发行前2-5个工作日 |
| 刊登发行公告 | 与募集说明书同步 |
| 簿记建档 | T日（发行日）前1-2日 |
| 确定利率/价格 | 簿记结束后 |

### 4.2 路演推介

**路演方式**：
- 网上路演
- 现场推介会
- 一对一沟通

**路演材料**：
- 发行方案简介
- 发行人基本情况
- 财务状况概述
- 募集资金用途
- 风险提示

### 4.3 定价发行

| 发行方式 | 说明 |
|---------|------|
| 招标发行 | 向投招标机构招标确定利率 |
| 簿记建档 | 主承销商簿记，投资者报价 |
| 定价发行 | 固定利率，直接销售 |

## 第五阶段：上市及存续期

### 5.1 上市交易

| 事项 | 说明 |
|-----|------|
| 上市申请 | 发行结束后向交易所申请上市 |
| 上市时间 | 通常在发行结束后5-10个工作日 |
| 代码分配 | 沪市：11开头 / 深市：11开头 |

### 5.2 存续期管理

| 管理事项 | 频率/触发条件 |
|---------|--------------|
| 年度报告披露 | 每年4月30日前 |
| 中期报告披露 | 每年8月31日前 |
| 临时重大事项披露 | 事项发生后2个工作日内 |
| 付息/兑付 | 按募集说明书约定 |
| 债券持有人会议 | 法定情形发生时 |

## 流程时间参考

| 债券类型 | 总体流程时间 |
|---------|-------------|
| 公司债（注册制） | 约3-4个月 |
| 企业债 | 约4-6个月 |
| 可转债 | 约4-6个月 |

## 常见问题

### Q: 股东大会决议有效期多久？
A: 通常为12个月，发行人在此期间内完成发行。

### Q: 债券是否可以分期发行？
A: 可以，需在募集说明书中明确分期发行安排。

### Q: 主承销商可以更换吗？
A: 原则上不可以更换，变更需重新履行程序。

### Q: 发行失败后能否重新发行？
A: 可以，但需重新经过内部决策和监管审核程序。
