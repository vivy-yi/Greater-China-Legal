# Skill: bond-eligibility-checker（债券发行资格核查）

## 功能定位

对拟发行企业债券、公司债券或可转换债券的主体进行资格合规性核查，判断其是否符合发行条件。

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|-----|-----|
| 债券类型 | string | 是 | company-bond/enterprise-bond/convertible-bond |
| 主体类型 | string | 是 | 股份有限公司/有限责任公司/上市公司 |
| 净资产（万元） | number | 是 | 最近一期经审计净资产 |
| 近三年平均可分配利润（万元） | number | 是 | 近三年平均可分配利润 |
| 累计债券余额（万元） | number | 否 | 已有债券余额 |
| 最近三年加权平均净资产收益率 | number | 否 | 可转债必需 |
| 募集资金用途 | string | 是 | 简要说明资金投向 |

## 核查逻辑

### 公司债券（company-bond）

```
条件1: 净资产 >= 6000万元
条件2: 近三年平均可分配利润 >= 债券年利息
条件3: 累计债券余额 <= 净资产 × 40%
条件4: 募集资金用途符合国家产业政策
条件5: 最近三年无债务违约记录
条件6: 最近三年无重大违法违规记录
```

### 企业债券（enterprise-bond）

```
条件1: 发行人具有法人资格
条件2: 财务状况良好，具有到期还本付息能力
条件3: 债券总额不超过自有资产净值
条件4: 募集资金用于固定资产项目或符合产业政策的项目
条件5: 利率不超过同期银行存款利率的40%
条件6: 非机关团体、非金融机构
```

### 可转换债券（convertible-bond）

```
条件1: 发行人为上市公司
条件2: 最近三年加权平均净资产收益率 >= 6%（注册制前要求）
条件3: 发行后累计债券余额 <= 净资产 × 50%
条件4: 近三年年均可分配利润 >= 债券年利息
条件5: 募集资金投向符合国家产业政策
条件6: 转股价格设定符合规定（不低于前20日均价）
```

## 核查流程

```
1. 接收用户输入的债券类型及财务数据
2. 根据债券类型加载对应核查标准
3. 逐项核查财务指标
4. 检查合规性要求（违约记录、违法违规）
5. 评估募集资金用途合规性
6. 生成核查结论及建议
7. 输出详细核查报告
```

## 使用示例

**用户输入**：
```
债券类型：公司债
主体类型：股份有限公司
净资产：8000万元
近三年平均可分配利润：1200万元
累计债券余额：2000万元
募集资金用途：新型材料生产线建设
最近三年债务违约记录：无
```

**Skill处理**：
- 净资产8000万元 > 6000万元 ✓
- 可分配利润1200万元 > 估算年利息（假设债券规模1亿元、利率5%，年利息500万元）✓
- 债券余额2000万/净资产8000万 = 25% < 40% ✓
- 资金用途需进一步核实是否符合产业政策

**输出结果**：
```
核查结论：PASS（条件通过）

通过条件：
✓ 净资产8000万元 ≥ 6000万元
✓ 可分配利润足以支付债券利息
✓ 债券余额占比25% < 40%上限
✓ 无债务违约记录
✓ 无重大违法违规记录

需确认事项：
? 募集资金用途"新型材料生产线"是否属于《产业结构调整指导目录》鼓励类
```

## 常见问题

### Q: 净资产刚好6000万元是否满足条件？
A: 是，满足"不低于6000万元"的最低要求。

### Q: 三年平均可分配利润如何计算？
A: 将近三年的可分配利润（净利润减提取法定公积金后的余额）相加后除以3。

### Q: 已有债券余额如何计算40%上限？
A: 公式：拟发行债券额度 + 已有债券余额 ≤ 净资产 × 40%

### Q: 可转债"最近三年加权平均净资产收益率"如何计算？
A: ROE = 净利润 ÷ 平均净资产 × 100%，取三年平均值。
