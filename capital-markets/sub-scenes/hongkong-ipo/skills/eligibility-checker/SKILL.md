# 资格核查技能（eligibility-checker）

## 功能描述
核查发行人是否满足香港联交所主板或创业板的上市资格条件。

## 调用方式
```
eligibility-checker: <发行人基本信息>
```

## 输入参数
- 公司全称
- 注册地
- 主营业务描述
- 最近三年财务数据（收入、盈利、现金流）
- 股本结构
- 管理层信息

## 核查项目

### 主板核查清单
- [ ] 盈利测试：最近年度盈利≥2,000万港元，前两年累计≥3,000万港元
- [ ] 市值/收入测试：最近年度收入≥5亿港元，市值≥40亿港元
- [ ] 市值/收入/现金流测试：最近年度收入≥5亿港元，市值≥20亿港元，现金流≥1亿港元
- [ ] 营业记录≥3年
- [ ] 管理层最近3年未变动
- [ ] 主营业务明确且受控制
- [ ] 公众持股≥25%，至少300名公众股东
- [ ] 最高3名公众股东持股合计≤50%

### 创业板核查清单
- [ ] 收入及现金流测试：最近年度收入≥1亿港元，现金流≥3,000万港元
- [ ] 仅收入测试：最近年度收入≥3亿港元，市值≥5亿港元
- [ ] 营业记录≥2年
- [ ] 管理层最近2年未变动
- [ ] 公众持股≥25%，至少100名公众股东
- [ ] 已委聘GEM保荐人

### 不同投票权（WVR）额外条件
- [ ] 市值≥400亿港元（或市值≥100亿港元+收入≥10亿港元）
- [ ] 已编备不同投票权架构说明文件
- [ ] 创办人必须为不同投票权受益人

## 输出格式
```
## 资格核查结果

### 适用测试：[盈利测试/市值收入测试/市值收入现金流测试]
### 核查结论：[符合资格/不符合资格/需补充材料]

### 详细分析
1. 财务指标：✓/✗
2. 营业记录：✓/✗
3. 管理层稳定性：✓/✗
4. 公众持股：✓/✗

### 建议
[如不符合，提出改进建议]
```

## 注意事项
- 须核实财务数据是否经审计师审计
- 注意区分持续经营业务与新业务
- 审查是否存在影响控制权稳定的因素

## 升级决策门

以下情况须移交专业律师：
- 涉及监管审批
- 存在法律风险
