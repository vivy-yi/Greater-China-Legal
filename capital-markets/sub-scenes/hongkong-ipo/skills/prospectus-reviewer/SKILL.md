# 招股书审查技能（prospectus-reviewer）

## 功能描述
审查上市招股说明书是否符合披露要求，识别潜在法律风险。

## 调用方式
```
prospectus-reviewer: <招股书文本或关键章节>
```

## 审查维度

### 1. 法定披露完整性
- [ ] 公司资料（历史、股本架构、业务性质）
- [ ] 财务资料（近三年经审计财务报表）
- [ ] 管理层资料（董事、监事、高管名单及资历）
- [ ] 发售详情（股数、价格区间、发售方式）
- [ ] 风险因素披露
- [ ] 募集资金用途
- [ ] 盈利预测（如有）

### 2. 披露质量核查
- [ ] 资料完整、真实、清晰
- [ ] 无误导性陈述
- [ ] 无重大遗漏
- [ ] 前后一致无矛盾

### 3. 财务信息核查
- [ ] 报表编制基准明确
- [ ] 会计政策披露充分
- [ ] 关联交易完整披露
- [ ] 资产估值合理
- [ ] 现金流量健康

### 4. 法律事项核查
- [ ] 重大诉讼披露
- [ ] 知识产权状况
- [ ] 合同风险识别
- [ ] 监管合规状况
- [ ] 股权架构清晰无代持

### 5. 行业特定披露
- [ ] 行业概况及竞争地位
- [ ] 监管环境变化影响
- [ ] 行业主要风险因素

## 风险信号识别

### 红旗指标（Red Flags）
1. 收入确认政策异常或激进
2. 关联交易占比过高
3. 毛利率显著高于行业平均水平
4. 现金流与净利润严重背离
5. 审计意见非无保留
6. 管理层频繁变动
7. 专利技术存在纠纷
8. 环保或监管处罚历史

### 需重点说明事项
- 业务模式创新导致无对标公司
- 单一客户或供应商依赖
- 创始团队退出或继承问题
- 境外经营涉及特殊监管

## 输出格式
```
## 招股书审查报告

### 总体评价
[符合披露要求/需修改/存在重大法律风险]

### 风险等级
[低风险/中风险/高风险/无法审查]

### 详细发现
1. 披露完整性：✓/✗/需补充
2. 财务真实性：✓/✗/需说明
3. 法律合规性：✓/✗/需关注

### 具体问题
| 章节 | 问题描述 | 严重程度 | 建议 |
|------|----------|----------|------|

### 修改建议
[按优先级列出需修改或补充的内容]
```

## 注意事项
- 仅基于文本内容进行形式审查
- 不对财务数据真实性进行独立核实
- 重大判断需咨询持牌律师意见

## 升级决策门

以下情况须移交专业律师：
- 涉及监管审批
- 存在法律风险
