# 信息披露审查技能 (disclosure-reviewer)

## 技能描述
审查A股IPO申报文件中的信息披露质量，识别虚假记载、误导性陈述、重大遗漏，保障披露合规。

## 法规依据
- 《证券法》第七十八条至第八十条
- 《首次公开发行股票并上市管理办法》相关条款
- 《公开发行证券的公司信息披露内容与格式准则》
- 《上市公司信息披露管理办法》

## 审查要点

### 一、完整性审查
| 披露事项 | 依据条款 |
|----------|----------|
| 招股说明书 | 准则第1号全文 |
| 发行人的主体资格 | 首发办法第6条 |
| 财务信息 | 首发办法第11条 |
| 公司治理 | 首发办法第9条 |
| 同业竞争与关联交易 | 披露准则第21条 |

### 二、真实性审查
- 核查财务报表是否真实反映经营状况
- 核实重大合同的真实性
- 验证发明专利、商标等无形资产权属
- 确认诉讼仲裁事项披露完整性

### 三、准确性审查
- 核对数字一致性（招股说明书与审计报告一致）
- 验证行业数据来源及权威性
- 检查行业地位表述是否夸大
- 核实募集资金投资项目测算合理性

### 四、重大性判断标准
以下情形构成重大信息：
1. 涉及发行条件或上市条件的内容
2. 可能对投资者决策产生重大影响
3. 涉及重大诉讼、仲裁、行政处罚
4. 关联交易金额或比例重大
5. 主要资产重大变化

### 五、常见信息披露问题清单

#### 5.1 财务方面
- 虚构收入或利润
- 提前确认收入
- 推迟费用确认
- 遗漏重要负债
- 关联交易非关联化

#### 5.2 业务方面
- 隐瞒竞争对手
- 夸大市场份额
- 虚假核心技术描述
- 专利技术权属纠纷未披露

#### 5.3 法律方面
- 重大诉讼未披露
- 行政处罚隐瞒
- 实际控制人变化未及时披露
- 关联交易未履行审议程序

### 六、审查报告格式
```
信息披露审查报告
一、基本信息
   发行人：XXX
   申报板块：科创板/主板/创业板
   审查范围：招股说明书（申报稿）
二、审查发现
   问题类型 | 具体描述 | 严重程度 | 整改建议
三、结论意见
```

## 使用方法
输入待审查文件，系统自动进行多维度审查，输出问题清单及整改建议。

## 升级决策门

以下情况须移交专业律师：
- 涉及监管审批
- 存在法律风险
