# 债券票面利率与付息计算

## 核心公式

### 一、应付利息计算

```
年应付利息 = 债券票面总额 × 票面利率
每次付息额 = 年应付利息 ÷ 每年付息次数
```

### 二、利息覆盖倍数

```
利息覆盖倍数 = EBITDA（或可用作利息支付的现金流）/ 当期应付利息总额
```
**监管要求：** 公募公司债利息覆盖倍数通常要求≥1.5倍

### 三、折溢价摊销

```
折价摊销 = (面值 - 发行价) ÷ 剩余期限
溢价摊销 = (发行价 - 面值) ÷ 剩余期限
实际利率 = (面值×票面利率 + 折溢价) / 发行价

债券摊余成本 = 初始账面价值 ± 累计摊销额
```

## 计算示例

### 示例一：平价发行公司债

| 参数 | 数值 |
|---|---|
| 债券面值 | 10亿元 |
| 票面利率 | 4.5% |
| 期限 | 5年 |
| 付息频率 | 每年一次 |

| 计算项 | 结果 |
|---|---|
| 年应付利息 | 10亿 × 4.5% = 4500万元 |
| 每次付息额 | 4500万元 / 1 = 4500万元 |
| 本金偿还 | 到期一次偿还 |

### 示例二：折价发行

| 参数 | 数值 |
|---|---|
| 债券面值 | 100元 |
| 发行价 | 97元 |
| 票面利率 | 3.5% |
| 期限 | 5年 |

| 计算项 | 结果 |
|---|---|
| 折价金额 | 100 - 97 = 3元 |
| 每年摊销额 | 3 / 5 = 0.6元 |
| 实际利率 | (100×3.5% + 0.6) / 97 = 4.2% |

## 偿债能力指标核查

| 指标 | 计算公式 | 监管要求 | 核查结果 |
|---|---|---|---|
| 资产负债率 | 负债总额 / 资产总额 | 通常≤70% | [✅/❌] |
| 流动比率 | 流动资产 / 流动负债 | 参考行业均值 | [✅/❌] |
| 利息保障倍数 | EBIT / 利息支出 | ≥1.5 | [✅/❌] |
| 现金利息保障倍数 | 经营现金流 / 利息支出 | ≥1.0 | [✅/❌] |

## 可转债特殊计算

| 参数 | 说明 |
|---|---|
| 转换比例 | = 债券面值 / 转换价格 |
| 转换价值 | = 正股价格 × 转换比例 |
| 转换溢价率 | = (债券价值 - 转换价值) / 转换价值 |

**触发赎回条件（通常）：**
- 连续30个交易日中有15个交易日正股收盘价≥转换价格的130%
- 未转股余额不足3000万元

**触发回售条件（通常）：**
- 连续30个交易日正股收盘价低于转换价格的70-85%

## 升级决策门

**以下情形须向律师升级：**
- 债券结构涉及特殊条款（永续债、可赎回、可转换）需法条确认
- 利息覆盖倍数接近临界值
- 涉及国有资产发行人有特殊财务要求
- 可转债转股价格低于净资产触发监管要求
