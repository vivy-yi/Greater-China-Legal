# 监管政策动态追踪

## 数据源

| 数据源 | 路径/方式 | 优先级 |
|---|---|---|
| 证监会官网 | https://www.csrc.gov.cn → 政策法规/公告 | 高 |
| 交易所官网（沪/深/北） | 各所"规则"栏目 | 高 |
| 巨潮资讯 | http://www.cninfo.com.cn → 问询函 | 中 |
| 彭博/路透 | 监管动态国际视野 | 低（参考） |

---

## 追踪内容

### 一、法规政策

| 类型 | 内容 | 来源 |
|---|---|---|
| 新规发布 | 证监会令/公告/规范性文件 | 证监会官网 |
| 规则修订 | 交易所业务规则修订 | 各交易所官网 |
| 征求意见 | 正在征求意见的法规草案 | 证监会/交易所征求意见栏目 |
| 废止失效 | 失效法规清单 | 证监会官网 |

### 二、审核动态

| 类型 | 内容 | 关注点 |
|---|---|---|
| 审核状态 | 在审企业审核进度 | 否决/通过/撤回统计 |
| 问询意见 | 发审委/上市委问询问题 | 问询高频问题分类 |
| 否决原因 | 审核未通过具体原因 | 关注共性问题 |
| 窗口指导 | 非书面指导的监管口径 | 口头/隐性要求 |

### 三、行业监管

| 行业 | 适用规则 | 特殊要求 |
|---|---|---|
| 金融 | 《商业银行理财子公司》《证券公司监督管理》 | 资本充足率/净资本 |
| 房地产 | 证监会涉房企业融资政策 | 募集资金不得用于拿地 |
| 教育 | 《民办教育促进法》 | 义务教育阶段不得营利 |
| 医药 | 《药品管理法》 | GMP/GSP合规 |

## 升级决策门

**以下情形须向律师升级：**
- 发现与现行法规明显冲突的窗口指导
- 征求意见稿可能对在审项目产生重大影响
- 涉及行业准入政策重大变化
- 涉及外资准入政策调整

## 本 Skill 不涵盖

- 审计报告真实性核查（须持证会计师事务所）
- 行业准入具体审批流程
- 税务处理（请咨询税务顾问）
- 发行保荐业务（须持证保荐代表人）
