# 发行资格核查

## 一、主体资格核查

| 条件 | 法条依据 | 核查结果 | 说明 |
|---|---|---|---|
| 依法设立且存续满3年 | 《首发办法》第8条 | [✅/❌/⚠️] | |
| 注册资本已足额缴纳 | 《首发办法》第9条 | [✅/❌/⚠️] | |
| 生产经营合法合规 | 《首发办法》第10条 | [✅/❌/⚠️] | |
| 最近36个月无重大违法行为 | 《首发办法》第10条 | [✅/❌/⚠️] | |
| 股权清晰，无重大权属纠纷 | 《首发办法》第13条 | [✅/❌/⚠️] | |

## 二、财务指标核查

### A股IPO（主板）

| 指标 | 标准 | 核查结果 |
|---|---|---|
| 净利润 | 最近3年均为正，且累计≥8000万元（以净利润计） | [✅/❌] |
| 营业收入 | 最近3年累计≥3亿元 | [✅/❌] |
| 经营活动现金流 | 累计≥1亿元 | [✅/❌] |
| 净资产 | 最近期末≥5000万元 | [✅/❌] |

### 科创板（预计市值10亿+）

| 指标 | 标准 | 核查结果 |
|---|---|---|
| 科创属性 | 满足《科创属性评价指引》 | [✅/❌] |
| 研发投入 | 最近3年研发投入占比≥5%或≥6000万元 | [✅/❌] |
| 发明专利 | ≥5项（主营业务相关） | [✅/❌] |

### 公司债发行

| 指标 | 标准 | 核查结果 |
|---|---|---|
| 净资产（股份有限公司） | ≥3000万元 | [✅/❌] |
| 累计债券余额 | 不超过净资产40% | [✅/❌] |
| 可分配利润 | 最近3年平均可覆盖债券1年利息 | [✅/❌] |

## 四、升级决策门

**以下情形须向律师升级：**
- 主体资格存在瑕疵但有解释空间
- 财务指标勉强达标（±10%以内）
- 存在潜在重大诉讼或行政处罚
- 同业竞争解决方案不明确
- 关联交易金额重大且定价不公允
- 国有资产转让涉及特殊审批
