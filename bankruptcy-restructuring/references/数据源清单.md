# 破产重整 — 数据源清单

## 一、内部数据

| 数据类型 | 内容 |
|---|---|
| 债务人财产状况说明 | 资产负债表/资产清单/负债清单 |
| 债权人名单 | 债权性质/金额/担保情况 |
| 职工情况 | 在职/离职人数/工资/社保欠缴 |
| 合同清单 | 待履行/已履行/需变更终止的合同 |
| 诉讼仲裁情况 | 已决/未决案件 |

## 二、公开数据源

| 数据源 | 说明 |
|---|---|
| 全国企业破产重整案件信息网 | 案件公告/进度查询 |
| 国家企业信用信息公示系统 | 债务人工商信息/动产抵押 |
| 中国裁判文书网 | 涉诉涉执信息 |
| 淘宝/京东司法拍卖平台 | 破产资产处置 |
| 产权交易所 | 不良资产转让 |
